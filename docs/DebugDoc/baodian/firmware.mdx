# 固件说明

    :::danger 须知

    * Klipper固件最好使用你的上位机固件进行编译
    * 使用其他人编译或者固件版本不一样可能导致各种问题
    * 尽量不要使用魔改的上位机

    :::

## 固件编译说明

    :::tip

    * FLY主板使用Katapult与内存卡烧录的bootloader
    * Katapult固件优点是可无接触直接更新固件
    * bootloader固件优点是不用担心刷错固件
    * 最好使用`MobaXterm`进行固件配置，经过群友反馈一些特别的SSH软件会导致编译失败
    
    :::

    * 请注意不管是哪个固件都需要确保固件编译正确否则会导致无法找到正常打印甚至设备都无法找到

## 固件编译说明

    :::tip
    * 需要注意将`Enable extra low-level configuration options`此选项选择上否则不会显示具体内容
    :::

### STM32

    * 如果STM32固件编译时候需要配置`GPIO pins to set at micro-controller startup`请注意一定要大写

### RP2040

    * 如果RP2040固件编译时候需要配置`GPIO pins to set at micro-controller startup`请注意一定要小写

## 固件烧录说明

    :::danger 须知

    * 请不要使用教程外的方法烧录固件
    * 请检查固件有没有编译错误，编译错误则无法正常烧录固件

    :::

### Katapult固件

    * 使用Katapult固件刷了编译错误的固件解决方法
    * 点击两次主板重置即可
    * 清除全部固件，然后重新刷Katapult固件

### bootloader固件

    * 使用bootloader固件刷了编译错误的固件解决方法
    * 重新使用TF卡刷写即可
    * 刷不进去请先断电，将内存卡安装到主板后在上电，五秒钟即可刷可
