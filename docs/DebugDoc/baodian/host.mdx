# 上位机篇

## 新买的上位机我要如何才能跑起来

  1. 首先得确定你的上位机是什么型号是Lite 2还是PI V2等等上位机
  2. 有没有准备下面的东西
      * 烧录用的软件
      * 对应的系统镜像
      * 读卡器，内存卡，带数据传输功能的TYPE-C

  系统烧录前注意事项:
        * 请保证内存卡容量必须不小于`16GB`或不大于`128G`TF卡，并且传输速度必须为`class10`级或`class10`级以上，建议使用闪迪等品牌的TF卡
        * 请不要随意更新系统！！服务器在国外很容易更新出问题
        * 尽量使用我们推荐的软件来操作
        * 请尽量学一点LINUX知识我个人推荐《鸟哥的Linux私房菜 基础学习篇》等

  3. [系统下载链接](https://mellow-next.klipper.cn/docs/category/%E7%B3%BB%E7%BB%9F%E9%95%9C%E5%83%8F)
  4. [辅助工具下载链接](https://mellow-next.klipper.cn/docs/ResDownload/auxiliary_software/)

## 下载好各类工具与对应的系统镜像后

  * 解压系统，解压后文件后缀名为`.img`
  * 需要使用SDFormatter重新格式化内存卡
  * 使用`win32DiskImager`烧录到系统里面
  * 如果是Lite 2或者MINI PAD需要打开BOOT盘下的`FLY-Config.conf`[进行配置](https://mellow.klipper.cn/#/board/fly_mini_pad/config?id=_34-klipperscreen%e7%9a%84%e9%85%8d%e7%bd%ae)
  * 如果是其他FLY上位机需要打开`FlyOS-Env.txt`[进行配置](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os#%E7%B3%BB%E7%BB%9F%E9%85%8D%E7%BD%AE)

## 启动系统

  * 这个时候系统已经烧录配置完毕，需要使用带数据传输功能的TYPE-C连接电脑使用[串口连接](https://mellow-next.klipper.cn/docs/DebugDoc/BasicTutorial/)
  * 请注意Lite2与MINI PAD系统在启动时候不会显示日志可以先等一会（5分钟内）如果还没有显示需要按两下回车或者重新刷系统
  * 请注意其他上位机正常情况会显示日志加载，如果还没有显示需要按两下回车或者重新刷系统
  * 启动好后就可以[连接WIFI](https://mellow-next.klipper.cn/docs/DebugDoc/BasicTutorial/Internet)
