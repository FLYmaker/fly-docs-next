# 外置驱动使用教程

## 注意事项

:::danger 须知

* 本教程基于Klipper进行控制
* 模块使用共阳极接线

:::


## 驱动接线


        <ImageView image={require('./img/tmc/servo-drive.png').default} size="80%" align="center" />


## 驱动细分跳线

    * 无需任何驱动跳线！！！


## klipper参考配置

    * 需要屏蔽驱对应动配置比如`[tmc5160 stepper_x] `或者`[tmc2209 stepper_x] `
    * 将`microsteps:`设置成`16`
    * 添加`step_pulse_duration: 0.000004`
        <ImageView image={require('./img/tmc/cfg.png').default} size="50%" align="center" />

## 驱动脉冲说明

    :::info

    * 配置都正确情况下，发现打印模型发现偏移可修改`step_pulse_duration: 0.000004`
    * 加大此值`0.000004`比如`0.000005`或者`0.000006`等

    :::
