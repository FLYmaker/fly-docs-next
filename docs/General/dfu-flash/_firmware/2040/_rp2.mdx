* 短接或者长按BOOT，使用type c连接到上位机后输入`lsusb`会出现`2e8a:0003` 属于RP2040是处于固件烧录状态，只能烧后缀名为uf2的固件，当固件烧录进去后会消失
* 弹出`RPI-RP2`后不需要在短接BOOT或者一直按按钮

<ImageView image={require('@site/docs/General/get-id/img/lsusb.png').default} size="100%" align="left" />

* 通过下方命令将固件烧录到RP2040

```bash
cd && cd ~/klipper/
make flash FLASH_DEVICE=2e8a:0003
```

* 也可以使用下方命令烧录其他目录下的固件，如预先编译好的固件
* 需要将`<firmware>`替换成固件所在地址，如:`~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2`
* 请注意只能烧录后缀为`UF2`的固件

```bash
sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/<firmware>
```
