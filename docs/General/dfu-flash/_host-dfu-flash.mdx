1. 安装烧录工具

* FLY上位机无需执行安装烧录工具

```bash
sudo apt update
sudo apt install dfu-util -y
```

2. 执行下面的命令查看是否连接成功。

```bash
lsusb
```

    * 如果没有DFU设备请尝试重新进入**烧录模式**
    * 因系统版本问题可能显示其他名字，如果有`0483:df11`则代表进入烧录模式

    <ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.png').default} size="100%" align="left" />

4. 烧录固件(烧录前确保已经编译过固件)

    * 如果执行不了请在前面加`sudo`请注意加入sudo后需要输入你当前用户的密码，并且密码看不到
    * 其中`-D ~/klipper/out/klipper.bin`中的`~/klipper/out/klipper.bin`可以替换成你的固件存放路径如: `~/FLY_Katapult/BL/D/D5.bin`                               
    * 路径需要自己替换！！！
    * 路径需要自己替换！！！
    * 路径需要自己替换！！！
    ```bash
    sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/klipper/out/klipper.bin
    ```

5. 出现下图箭头处所示内容``File downloaded successfully``则烧录成功


:::warning

红色箭头所指这一行**后面的报错**请忽略！！！

:::
