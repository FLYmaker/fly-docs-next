# FLY_Config 的使用

    * FLY-Config用来简化系统配置等，适用于Klipper全家桶，免去进入ssh写命令的麻烦
    * 只能在FLY定制Armbian系统运行
    * 对初学者友好
    * 直接配置Klipper及周边

## 打开 FlyOS-Env.txt

    * 如果您已经制作好TF卡，请使用读卡器连接到电脑。此时电脑应该会弹出资源管理器并且有一个“可移动磁盘BOOT” 如果未出现，请重新拔下TF卡，再插入电脑
    * 如果是M2WE，请重新进入`BOOT`此时电脑应该会弹出资源管理器并且有一个“可移动磁盘BOOT” 如果未出现，请重新新进去BOOT。

    <ImageView image={require('./img/boot.png').default} size="100%" align="left" />

    * 打开BOOT盘下的`FlyOS-Env.txt`

    <ImageView image={require('./img/env.png').default} size="100%" align="left" />

    * 指定上位机型号

    <ImageView image={require('./img/doard.png').default} size="100%" align="left" />

    * 将`board=none`中的`none`改成你的上位机型号，如下面几个

    ```cfg
    fly-pi, fly-pi-v2, fly-gemini-v1, fly-gemini-v2, fly-gemini-v3, fly-c8
    ```

    * 参考

    ```cfg
    board=fly-c8
    ```

:::tip
其他功能请通过FLY-TOOLS进行配置
:::
