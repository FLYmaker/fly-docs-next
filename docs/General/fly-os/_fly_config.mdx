# FLY_Config 的使用

    * FLY-Config用来简化系统配置等，适用于Klipper全家桶，免去进入ssh写命令的麻烦
    * 只能在FLY定制Armbian系统运行
    * 对初学者友好
    * 直接配置Klipper及周边

## 配置方法

### WIFI

    :::tip 提示

    * WIFI配置方法复杂，建议使用串口连接上位机后进去进行WIFI连接

    :::

### 其他配置说明

    :::tip 提示

    * 3.3.4系统开始可以直接使用FLY-TOOLS进行系统配置无需进去boot修改配置
    * 请注意，老系统并不支持直接使用FLY-TOOLS进行配置
    * <Button variant="contained" disableElevation href="/docs/ToolsDoc/fly-tools/">FLY_TOOLS使用教程</Button>

    :::
