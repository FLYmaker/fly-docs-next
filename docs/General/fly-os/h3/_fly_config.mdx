# FLY_Config 的使用

    * FLY-Config用来简化系统配置等，适用于Klipper全家桶，免去进入ssh写命令的麻烦
    * 只能在FLY定制Armbian系统运行
    * 对初学者友好
    * 直接配置Klipper及周边

## 打开 FLY_Config.conf

    * 如果您已经制作好TF卡，请使用读卡器连接到电脑。此时电脑应该会弹出资源管理器并且有一个“可移动磁盘BOOT” 如果未出现，请重新拔下TF卡，再插入电脑

    <ImageView image={require('@site/docs/General/fly-os/h3/img/boot.png').default} size="100%" align="left" />

    * 打开BOOT盘下的`FlyOS-Env.txt`

    <ImageView image={require('@site/docs/General/fly-os/h3/img/env.png').default} size="100%" align="left" />

## WIFI配置

    * 找到WIFI配置，将WIFI处的flase改成true，并将WiFi名称和密码填写上，保存，插回上位机。等待几分钟后，进入路由器后台管理界面查看IP地址。

:::warning 注意事项
系统启动后会将此wifi保存到系统，并且将`WiFi=true`修改为`WiFi=flase`
:::

    <ImageView image={require('@site/docs/General/fly-os/h3/img/wifi.png').default} size="100%" align="left" />
