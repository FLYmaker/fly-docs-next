* 开始获取ID前，请先[连接到SSH](/docs/DebugDoc/BasicTutorial/index.mdx "点击即可跳转") 
* 请确保下位机连接上位机的数据线具有数据传输功能
* SSH连接连接到上位机后输入`lsusb`并且回车，将可能出现下面几种设备
* 如果系统不支持`lsusb`可以执行下方指令进行安装
* 如果`lsusb`后没有设备，请更换上位机！


    <ImageView image={require('./img/rs232.png').default} size="100%" align="left" />

* `1a86:7523`属于RS232设备如果设备损坏或者USB线不带数据传输功能不会显示设或。

:::warning 注意事项
有1a86:7523`时候才能使用下方命令搜索RS232 ID
:::

* 使用下方命令搜索id

```
ls /dev/serial/by-path/*
```

    <ImageView image={require('./img/rs232-id.png').default} size="100%" align="left" />


* 如果输入`ls /dev/serial/by-path/*`后没有ID出来，但是`lsusb`有设备请更换上位机或者更换上位机系统
* 请注意使用UTOR会输出三个id，但是如果你过另外一个UTOR系统会记录下来导致搜索时候出现多个id
* 请确保id后面带有`-port0`否则无法使用

:::warning 注意事项

请注意工具板与UTOR一定要供地！！！否则无法连接上工具版！！！

:::
