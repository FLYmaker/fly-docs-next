## DSI使用方法

    * 镜像烧写完成后，进入/boot/config.txt中添加下方指令

    ```cfg
    dtoverlay=vc4-kms-dsi-7inch
    ```

## 触摸使用不了解决方法

:::warning 注意事项
树莓派无法使用触摸时候可以按此方法修改配置
:::

    * 进入/boot/config.txt中找到

    ```bash
    dtoverlay=vc4-kms-v3d
    ```

    * 修改
        
    ```bash
    dtoverlay=vc4-fkms-v3d,f
    ```

    * 保存并且退出
