## 注意事项

:::danger 注意事项

* M2WE出厂已经预刷好FLYOS可以直接启动
* M2WE只能使用在FLY的产品上，不支持其他设备
* M2WE的天线端子是`IPEX1`
* M2WE不支持2.4G与5G双频合一、中文、特殊字符、默认隐藏的WIFI

:::
