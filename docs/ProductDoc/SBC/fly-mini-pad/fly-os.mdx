# 系统烧录

## 系统烧录前准备

    * 读卡器
    * TF卡一张，容量必须不小于16GB 或 不大于128G TF 卡，TF 卡的传输速度必须为 class10 级或 class10 级以上
    * 下载对应的[系统镜像](/docs/category/系统镜像 "点击即可跳转") 
    * 下载需要的[烧录工具](/docs/ResDownload/auxiliary_software/ "点击即可跳转")
