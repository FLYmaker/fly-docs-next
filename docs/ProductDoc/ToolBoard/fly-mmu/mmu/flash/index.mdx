# 固件烧录说明

    * MMU可以使用RS232，CAN，USB三种连接方式与上位机连接
    * 其中RS232与CAN需要使用使用`XT30(2+2)-F`进行连接，USB需要使用带数据传输功能的`TYPE-C`数据线进行连接
    * 下方是`XT30(2+2)-F`数据线样式

## 如何判断你需要的哪一种固件

    * CAN固件
        * 好处：如果刷了`Katapult`固件可以无需触碰主板情况下更新固件，并且支持总线连接
        * 坏处：可能因各种问题导致报错，例如与`KlipperScreen`一起使用可能导致归位超时
    * RS232固件
        * 好处：不需要特殊软件即可使用，通信可靠，稳定
        * 坏处：重新刷固件需要进入手动DFU才可以更新固件或者双击RST从可以进去`Katapult`来更新固件
    * USB固件
        * 好处：如果刷了Katapult固件可以无需触碰主板情况下更新固件，通信可靠，稳定
        * 坏处：如果多个USB设备可能会影响数据传输
