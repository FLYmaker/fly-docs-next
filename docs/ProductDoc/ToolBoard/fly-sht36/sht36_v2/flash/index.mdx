# 固件烧录说明

    * SHT36 V2可以使用CAN连接方式与上位机连接
    * CAN需要使用使用`XT30(2+2)-F`进行连接
    * 下方是`XT30(2+2)-F`数据线样式
    <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

## 如何判断你需要的哪一种固件

    * SHT36 V2只能使用CAN方式连接上位机
    * APM 072与STM 072  请看[072CAN固件烧录](./can072.mdx "点击即可跳转")
    * STM 103   请看[103CAN固件烧录](./can103.mdx "点击即可跳转")
        * 好处：如果刷了`Katapult`固件可以无需触碰主板情况下更新固件，并且支持总线连接
        * 坏处：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
        * 提示：如果需要使用CAN请确定上位机是否支持CAN，并且需要搭配UTOC或者刷了桥接固件的主板
