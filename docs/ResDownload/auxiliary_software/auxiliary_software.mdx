# 辅助工具与软件

## **上位机系统烧录**

### SSH软件：MobaXterm
* 选择右边的`Installer edition`即可下载
* SSH软件下载链接:[下载链接](https://mobaxterm.mobatek.net/download-home-edition.html)

### 内存卡格式化软件：SDFormatter
* SD卡格式化软件:[下载链接](https://www.waveshare.net/w/upload/d/d7/Panasonic_SDFormatter.zip)

### 系统烧录软件: Win32 Disk Imager

* 烧录系统软件:[下载地址](https://www.waveshare.net/w/upload/d/d7/Panasonic_SDFormatter.zip)
