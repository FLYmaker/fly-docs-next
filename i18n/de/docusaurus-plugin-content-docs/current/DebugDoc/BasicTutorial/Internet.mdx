# Verbindung zum Netzwerk über SSH

* Bitte stellen Sie sicher, dass das System gestartet ist und über SSH mit einer seriellen Verbindung zum Hostcomputer verbunden ist, empfohlen wird die Nutzung von **MobaXterm_Personal**
* Bitte stellen Sie sicher, dass der Hostcomputer **M2WE** oder ein **2.4G_WiFi-Modul** installiert hat, für **Lite2** und **MiniPad** ist keine Modulinstallation erforderlich
* Bitte beachten Sie, dass **2.4G_WiFi-Modul**, **Lite2** und **MiniPad** nur 2.4G_WiFi verwenden können und nicht 5G_WiFi
* Beachten Sie, dass das Netzwerk nicht dual-band sein darf, da sonst keine normale Verbindung möglich ist
* Bitte stellen Sie sicher, dass die Antenne installiert ist！！！

1. Geben Sie in der SSH-Software den Befehl **nmtui** ein und drücken Sie die Eingabetaste, es erscheint die unten gezeigte Oberfläche

<ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.png').default} size="80%" align="center" />

2. Drücken Sie die Eingabetaste und wählen Sie **Activate a connection**, dann erneut die Eingabetaste

<ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.png').default} size="80%" align="center" />

3. Wählen Sie das Netzwerk, mit dem Sie sich verbinden möchten, drücken Sie die Eingabetaste, geben Sie das WiFi-Passwort ein und **wählen Sie OK** zum Speichern der Verbindung

<ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.png').default} size="80%" align="center" />

4. Nach der Verbindung zum WIFI, suchen Sie auf der Tastatur die **Esc**-Taste und drücken Sie diese mehrmals, um die aktuelle Oberfläche zu verlassen, dann geben Sie **ip a** ein und drücken die Eingabetaste

* Die IP-Adresse des Hostcomputers im Bild ist 192.168.2.136

<ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.png').default} size="80%" align="center" />
