# USB-Kamera hinzufügen


:::danger Wichtig

* Der `crowsnest`-Dienst muss installiert sein, Systeme wie lite2 und mini pad haben ihn nicht installiert
* Es wird eine Plug-and-Play-Kamera benötigt, die unter Linux, Armbian und anderen Klipper unterstützten Systemen ohne zusätzliche Treiber funktioniert.
* Dieses Tutorial verwendet die neuesten Versionen von Klipper, Crowsnest und Fuidd als Beispiel. Bei anderen Methoden müssen Sie selbst recherchieren.

:::

## Konfiguration öffnen

:::tip

* Es gibt viele verschiedene Kamera-IDs je nach System, wir müssen nur die unveränderliche ID bestimmen.

:::

    * Suchen Sie in Fuidd nach der Konfigurationsdatei und öffnen Sie `crowsnest.conf`
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf.png').default} size="100%" align="center" />

    * Finden Sie `device: ` und löschen Sie die ID hinter dem Doppelpunkt
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf1.png').default} size="100%" align="center" />

## Kamera-ID suchen

:::tip

* Stellen Sie sicher, dass die Kamera an den Hostrechner angeschlossen ist

:::

    * Klicken Sie oben auf `DEVICES`, dann auf `VIDEO` und schließlich auf Aktualisieren
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf2.png').default} size="100%" align="center" />
    * Finden Sie `path_by_id` und kopieren Sie die Zeile darunter
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf3.png').default} size="100%" align="center" />

## ID hinzufügen

    * Fügen Sie die ID hinzu und speichern Sie die Konfiguration
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf4.png').default} size="100%" align="center" />

## Kamera aktivieren

    * Klicken Sie auf Einstellungen, wählen Sie Kamera und klicken Sie auf Kamera hinzufügen
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf5.png').default} size="100%" align="center" />
    * Geben Sie einen beliebigen Namen ein und klicken Sie auf Speichern
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf6.png').default} size="100%" align="center" />

## Überprüfen, ob aktiviert

    * Klicken Sie auf das Dashboard, normalerweise sollte das Kamerabild angezeigt werden
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf7.png').default} size="100%" align="center" />
    * Wenn nichts angezeigt wird, starten Sie das System neu
