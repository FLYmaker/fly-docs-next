# Verbindung zur SSH der Hostmaschine

:::danger

* Dieses Dokument gilt nur für die FLY Hostmaschine, andere Geräte dienen nur als Referenz.
* Wenn Sie Firmware kompilieren und flashen möchten, sehen Sie bitte das Netzwerkverbindungstutorial an, nicht das serielle Verbindungstutorial.

:::

* Sie können über eine serielle Schnittstelle oder WIFI eine Verbindung zum Gerät herstellen.
* Bitte bereiten Sie im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminaltool** vor.
* Bitte stellen Sie sicher, dass auf der SD-Karte oder dem M2WE das **entsprechende Betriebssystem für die Hostmaschine** geflasht ist.
* **Wenn die Hostmaschine nicht normal startet, entfernen Sie bitte die Treiber, Endschalter, Lüfter usw.!!!**
* **Verbinden Sie sich zuletzt über das Netzwerk mit SSH, um die Firmware zu kompilieren!!**
* **Für die Kompilierung der Firmware, sehen Sie bitte das Tutorial zur Verbindung der Hostmaschine über WIFI oder Netzwerkkabel.**

        ## Verbindung zur SSH der Hostmaschine über Netzwerk-IP

        * Stellen Sie sicher, dass das System **normal gestartet** ist.
        * Melden Sie sich im **Router-Backend** an, zu dem die Hostmaschine verbunden ist, es sei denn, Sie kennen die IP-Adresse der Hostmaschine.
        * Stellen Sie sicher, dass **Sie den Benutzernamen und das Passwort der Hostmaschine kennen**, Standardbenutzername für das FLY-System ist **fly**, Standardpasswort ist **mellow**.
        * Wenn die Hauptplatine gestartet und mit dem Netzwerk verbunden ist, können Sie die folgenden Methoden verwenden, um sich mit SSH zu verbinden.
        * Öffnen Sie MobaXterm und nehmen Sie die folgenden Einstellungen vor, geben Sie die IP-Adresse ein, die Sie zuvor überprüft haben, oder überprüfen Sie sie im Router-Backend.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Hinweis: Hier im 4. Schritt wird der Benutzername der Hostmaschine eingegeben.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch wird dieses Fenster angezeigt, klicken Sie auf `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Es wird nach dem Passwort gefragt, geben Sie das Passwort der Hostmaschine ein und drücken Sie die Eingabetaste.

        **Hinweis: Hier eingegebenes Passwort wird nicht angezeigt, drücken Sie einfach nach dem Eingeben die Eingabetaste.**

        Standardmäßig für das FLY-System, Benutzername: `fly` , Passwort: `mellow`
