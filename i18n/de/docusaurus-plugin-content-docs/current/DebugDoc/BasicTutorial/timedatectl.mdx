# Ändern der Systemzeitzone

    Bitte beachten Sie, dass FLY HMI, wenn Sie FLY OS 3.3.4 verwenden, direkt mit FLY-Tools die Zeitzone ändern kann.
    Für andere HMI, bitte sehen Sie die untenstehenden Schritte.


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    Ersetzen Sie `<TIMEZONE>` durch den Zeitzonencode, den Sie verwenden möchten. Sie können die Zeitzonencodes auf der Website [iana.org](https://www.iana.org/time-zones) nachschlagen. Zum Beispiel, wenn Sie die Systemzeitzone auf Beijing (Asia/Shanghai) ändern möchten, geben Sie den folgenden Befehl ein:

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    Jetzt wurde Ihre Systemzeitzone erfolgreich auf Shanghai-Zeit geändert. Sie können die Zeitzoneneinstellung mit folgendem Befehl überprüfen:

```bash
date
```
