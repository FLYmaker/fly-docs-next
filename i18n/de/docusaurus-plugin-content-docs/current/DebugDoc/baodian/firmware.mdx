# Firmware-Erklärung

    :::danger Wichtig

    * Es wird empfohlen, die Klipper-Firmware mit der Firmware Ihres Host-Computers zu kompilieren
    * Die Verwendung von von anderen Personen kompilierten oder unterschiedlichen Firmware-Versionen kann zu verschiedenen Problemen führen
    * Vermeiden Sie nach Möglichkeit modifizierte Host-Computer

    :::

## Besondere Hinweise

    :::info Besondere Hinweise
    * Die meisten FLY-Motherboards sind bereits mit der `Katapult`-Firmware ausgeliefert
    * FLY-Motherboards verwenden hauptsächlich die `Katapult`-Firmware zum Flashen der `Klipper`-Firmware
    * Die Motherboards DP5, D5 und D7 müssen manuell mit der entsprechenden `Katapult`-Firmware geflasht werden
    * Aufgrund der vorherigen Firmware-Einflüsse müssen Sie nach dem Flashen der `Katapult`-Firmware zweimal auf das Motherboard-Reset klicken, um die Motherboard-ID zu suchen, sonst kann das Flashen der Firmware nicht durchgeführt werden
    :::

## Firmware-Kompilierungshinweise

    :::tip

    * FLY-Motherboards verwenden Katapult und den Bootloader der Speicherkarte
    * Der Vorteil der Katapult-Firmware ist, dass sie eine kontaktlose Firmware-Aktualisierung ermöglicht
    * Der Vorteil der Bootloader-Firmware ist, dass keine falsche Firmware geflasht wird
    * Es wird empfohlen, `MobaXterm` für die Firmware-Konfiguration zu verwenden. Laut Rückmeldungen von Gruppenmitgliedern können einige spezielle SSH-Programme zu Kompilierungsfehlern führen
    
    :::

    * Bitte beachten Sie, dass unabhängig von der Firmware sichergestellt werden muss, dass sie korrekt kompiliert wurde, da sonst Druckprobleme oder sogar das Gerät nicht gefunden werden können

## Firmware-Kompilierungshinweise

    :::tip
    * Achten Sie darauf, die Option `Enable extra low-level configuration options` zu aktivieren, da sonst bestimmte Inhalte nicht angezeigt werden
    :::

### STM32

    * Wenn bei der Kompilierung der STM32-Firmware `GPIO pins to set at micro-controller startup` konfiguriert werden müssen, achten Sie darauf, dass sie in Großbuchstaben geschrieben sind

### RP2040

    * Wenn bei der Kompilierung der RP2040-Firmware `GPIO pins to set at micro-controller startup` konfiguriert werden müssen, achten Sie darauf, dass sie in Kleinbuchstaben geschrieben sind

## Firmware-Flashen-Erklärung

    :::danger Wichtig

    * Beim Flashen der `Klipper`-Firmware über die `Katapult`-Firmware muss sichergestellt sein, dass das Gerät gefunden werden kann. Wenn das Gerät nicht gefunden wird, klicken Sie zweimal auf das Motherboard-Reset
    * Verwenden Sie bitte keine Methoden zum Flashen, die nicht in den Anleitungen beschrieben sind
    * Überprüfen Sie, ob es Kompilierungsfehler in der Firmware gibt, da dies sonst das normale Flashen der Firmware verhindert

    :::

### Katapult-Firmware

    * Lösung, wenn eine fehlerhafte Firmware mit der Katapult-Firmware geflasht wurde
    * Zweimaliges Klicken auf das Motherboard-Reset, nach dem Reset blinkt normalerweise eine LED auf
    * Löschen Sie die Firmware des Motherboards und flashen Sie die Katapult-Firmware erneut

### Bootloader-Firmware

    * Lösung, wenn eine fehlerhafte Firmware mit der Bootloader-Firmware geflasht wurde
    * Flashen Sie die Firmware erneut mit der TF-Karte
    * Wenn das Flashen nicht klappt, trennen Sie zuerst die Stromversorgung, installieren Sie die Speicherkarte auf das Motherboard und schalten Sie dann ein, innerhalb von fünf Sekunden sollte das Flashen möglich sein
