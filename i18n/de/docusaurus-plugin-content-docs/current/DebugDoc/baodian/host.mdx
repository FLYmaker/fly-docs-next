# Oberflächengerät

## Wie bringe ich mein neues Oberflächengerät zum Laufen

  1. Zuerst musst du feststellen, welches Modell dein Oberflächengerät ist, sei es Lite 2, PI V2 oder ein anderes Oberflächengerät
  2. Hast du folgende Dinge vorbereitet:
        * Software zum Flashen
        * Entsprechendes Systemimage
        * Kartenleser, TF-Karte, TYPE-C mit Datenübertragungsfunktion

  Hinweise vor dem Brennen des Systems:
        * Stelle sicher, dass die Speicherkarte eine Kapazität von mindestens `16GB` oder höchstens `128G` TF-Karte hat und die Übertragungsgeschwindigkeit `class10` oder höher ist, empfohlen werden TF-Karten von Marken wie SanDisk
        * Bitte aktualisiere das System nicht willkürlich!! Da der Server im Ausland ist, kann es leicht zu Problemen bei Updates kommen
        * Verwende nach Möglichkeit die von uns empfohlene Software
        * Versuche, ein bisschen Linux-Wissen zu erlernen. Ich empfehle persönlich <Button variant="contained" disableElevation href="https://item.jd.com/12443890.html">《鸟哥的Linux私房菜 基础学习篇》</Button> oder ähnliches

  3. FLY-Systemimage <Button variant="contained" disableElevation href="/docs/category/系统镜像">Systemimage</Button>
  4. Benötigte Werkzeuge <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Brennwerkzeuge</Button>

## Nachdem du alle Tools und das entsprechende Systemimage heruntergeladen hast

  * Entpacke das System, nach dem Entpacken hat die Datei die Endung `.img`
  * Formatiere die Speicherkarte mit SDFormatter neu
  * Verwende `win32DiskImager`, um das System zu brennen
  * Es wird nicht empfohlen, `balenaetcher` für das Brennen des Systems zu verwenden
  * Falls es sich um ein FLY-Oberflächengerät handelt, öffne `FlyOS-Env.txt` zur Konfiguration
  * Formatiere die Speicherkarte nach dem Brennen des Systems nicht!!

## System starten

  * Nachdem das System gebrannt wurde, muss die Systemkarte in das Oberflächengerät eingebaut und das Gerät eingeschaltet werden
  * Zu diesem Zeitpunkt ist das System bereits gebrannt und konfiguriert, verbinde es mit einem Computer über ein TYPE-C-Kabel mit Datenübertragungsfunktion und verwende eine serielle Verbindung
  * Beachte, dass andere Oberflächengeräte normalerweise das Protokoll laden anzeigen, falls es noch nicht angezeigt wird, drücke zweimal die Eingabetaste oder brenne das System erneut
  * Nach dem Starten kannst du eine WIFI-Verbindung herstellen


## Hinweise zur Verwendung von M2WE und Speicherkarte

  * Wenn das Gerät M2WE und Speicherkarte installiert hat, startet das System standardmäßig von der Speicherkarte
  * Es wird empfohlen, das System mit einer SD-Karte zu brennen, das Brennen über USB auf M2W könnte aufgrund von Treiberkompatibilitätsproblemen fehlschlagen, es wird empfohlen, die Speicherkarte direkt zum Brennen auf M2WE zu verwenden
  * Achte darauf, ob das heruntergeladene Systemimage passt, andernfalls kann das System nicht starten
  * Beachte, dass Lite2 und MINI PAD nur eine Speicherkarte verwenden können


## Hinweise zur täglichen Nutzung

  * Trenne das Gerät nicht direkt vom Strom, schalte es ordnungsgemäß aus, um Datenverlust zu vermeiden
  * Da Server wie GitHud im Ausland sind, wird es nicht empfohlen, selbst Updates oder Plugins zu installieren
