## Bedienungsanleitung

    :::warning Beachten Sie

    * Bitte beachten Sie, dass die Stromkalibrierung vor der Spulenfrequenzkalibrierung durchgeführt werden muss
    * Mehrachsige Z-Maschinen müssen einmal manuell eingeebnet werden, bevor alle Kalibrierungen abgeschlossen sind, und erneut eingeebnet werden, nachdem die Kalibrierung wiederholt wurde
    * Die Firmware des unteren Computers muss vollständig mit der Kliiper-Version übereinstimmen

    :::

    * Bitte beachten Sie, dass die Stromkalibrierung vor der Spulenfrequenzkalibrierung durchgeführt werden muss
    * Mehrachsige Z-Maschinen müssen einmal manuell eingeebnet werden, bevor alle Kalibrierungen abgeschlossen sind, und erneut eingeebnet werden, nachdem die Kalibrierung wiederholt wurde
    * Die Firmware des unteren Computers muss vollständig mit der Kliiper-Version übereinstimmen

    * Bitte verwenden Sie die `FUIDD`-Webseite, um die folgende Konfiguration hinzuzufügen, bevor Sie die nachfolgenden Operationen ausführen
        ```cfg
        [force_move]
        enable_force_move: true
        ```

    ### Kalibrierung des EDDY-Stroms

        * Geben Sie im Konsolenfenster den folgenden Befehl ein
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * Homing von X und Y
            ```bash
            G28 X Y
            ```
        * Nach Abschluss der obigen Schritte muss der Antriebsstrom von Eddy kalibriert werden. Bewegen Sie Eddy zuerst in die Mitte des Heizbetts und heben Sie dann die Düse an, sodass der Abstand zur Heizplatte 20 mm beträgt. Führen Sie dann im Konsolenfenster den folgenden Befehl aus, um den Strom automatisch zu ermitteln und speichern Sie die Parameter mit `SAVE_CONFIG`. Normalerweise sollte der Wert etwa 16 liegen, wenn der Wert 0 oder 30 ist, überprüfen Sie die Verkabelung.
        * Beachten Sie, dass die Position 20 mm über dem Heizbett sein muss
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * Der normale Rückgabewert sollte etwa `15` liegen
        * Nach Abschluss der Kalibrierung `SAVE_CONFIG` speichert die Parameter

    ### Kalibrierung der Eddy-Frequenz


        :::info

        * Beim Installieren von Eddy sollte die Unterseite mindestens 1-2 mm über der Düse positioniert sein
        * Bei der Kalibrierung der Eddy-Frequenz sollte die Spulen-PCB mindestens 5 mm vom Heizbett entfernt sein

        :::

        * Dann bewegen Sie die Düse zur Mitte der Plattform (Achtung: Stellen Sie sicher, dass die Maschine keinen heightmap geladen hat)
        * Beginnen Sie mit der manuellen Z-Versatzkalibrierung [Papier-Test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * Nach Abschluss der Kalibrierung `SAVE_CONFIG` speichert die Parameter
        * Drucker mit `z_tilt` oder `quad_gantry_level（QGL）`-Funktionen sollten zuerst `Z_TILT_ADJUST` oder `QUAD_GANTRY_LEVEL` einmal ausführen und dann die Eddy-Frequenz erneut kalibrieren

    ### Temperaturkompensation

        * Homing aller Achsen
        * Wenn `[quad_gantry_level]` oder `[z_tilt]` vorhanden ist, führen Sie zuerst `Z_TILT_ADJUST` oder `QUAD_GANTRY_LEVEL` aus und homen Sie dann erneut
        * Geben Sie im Konsolenfenster G0 Z5 ein oder bewegen Sie die Z-Achse über die Webseite auf `5 mm` über dem Heizbett
        * Führen Sie den folgenden Code aus, um den `idle timeout` der Maschine etwas länger zu setzen, um Timeouts während des Erwärmungsprozesses zu vermeiden
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=36000
            ```
        * Führen Sie den folgenden Befehl aus, um die Kalibrierung zu starten
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning Beachten Sie

        * Hinweis: In dem obigen Befehl wurde die Zieltemperatur auf `56℃` gesetzt, was für die meisten Geräte geeignet ist.
        * Wenn die tatsächliche Temperatur des Gehäuses Ihrer 3D-Drucker höher ist, können Sie die Zieltemperatur entsprechend anheben.
        * Beachten Sie, dass mit höherer Zieltemperatur die Kalibrierungszeit entsprechend verlängert wird, da Eddy mehr Zeit benötigt, um die Temperaturgleichgewicht zu erreichen.

        :::
    
        1. Nach Ausführung dieses Befehls zeigt die Benutzeroberfläche ein Z-Achsenanpassungsfenster an. Verwenden Sie die oben erwähnte Methode für die manuelle Z-Achsen-Versatzkalibrierung (Paper Test), um das Papier zwischen Düse und Bett zu klemmen und diesen Wert zu bestätigen.
        2. Nach Akzeptanz des Wertes, stellen Sie die Heizbett-Temperatur auf den Höchstwert und die Düse auf 220℃ ein.
        3. Wenn Sie sich in einem Raum mit Klimaanlage oder offenen Fenstern befinden, um sicherzustellen, dass die Temperatur von Eddy steigt, wird empfohlen, die Klimaanlage oder die Fenster zu schließen, da Wind die Temperaturerhöhung beeinflusst.
        4. Mit dem Anstieg der Temperatur von Eddy wird das System Sie automatisch dazu auffordern, alle 4℃ eine manuelle Z-Achsen-Versatzkalibrierung durchzuführen. Die Heizbett-Temperatur ist sehr hoch, Vorsicht vor Verbrennungen!!!
        5. Wiederholen Sie die manuelle Z-Achsen-Versatzkalibrierung (Paper Test), bis die Kalibrierung abgeschlossen ist. Wenn Sie feststellen, dass die Temperatur von Eddy nicht weiter steigt, können Sie die Kalibrierung vorzeitig mit den folgenden Befehlen beenden.
        
        * Zusätzliche gcode-Befehle während der Driftkalibrierung umfassen: TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT`    kann verwendet werden, um vor Erreichen des Schrittwertes neue Daten zu erzwingen.
            * `TEMPERATURE_PROBE_COMPLETE`    kann verwendet werden, um die Kalibrierung vor Erreichen des Ziels abzuschließen.
            * `ABORT`  kann verwendet werden, um die Kalibrierung zu beenden und die Ergebnisse zu ignorieren.
        * Nach Abschluss dieser Schritte wird Eddy in einem breiten Temperaturbereich hervorragende Erstschichtdrucke liefern können!
## Verschiedene Optimierungsmakros

    ### `[quad_gantry_level]` Bett-Scan-Optimierung
        * Wenn `[quad_gantry_level]` konfiguriert ist, können Sie das folgende Makro zur Konfiguration hinzufügen, andernfalls ist es nicht notwendig
        * Dieses Makro reduziert die Sondierhöhe mit abnehmendem Höhenunterschied des Portals
            ```cfg
            [gcode_macro QUAD_GANTRY_LEVEL]
            rename_existing: _QUAD_GANTRY_LEVEL
            gcode:
                SAVE_GCODE_STATE NAME=STATE_QGL
                BED_MESH_CLEAR
                {% if not printer.quad_gantry_level.applied %}
                _QUAD_GANTRY_LEVEL horizontal_move_z=10 retry_tolerance=1
                {% endif %}
                _QUAD_GANTRY_LEVEL horizontal_move_z=2 METHOD=rapid_scan ADAPTIVE=1
                # G28 Z
                RESTORE_GCODE_STATE NAME=STATE_QGL
            ```

    ### `[z_tilt]` Bett-Scan-Optimierung

        * Wenn `[z_tilt]` konfiguriert ist, können Sie das folgende Makro zur Konfiguration hinzufügen, andernfalls ist es nicht notwendig
        * Dieses Makro reduziert die Sondierhöhe mit abnehmendem Höhenunterschied des Portals
            ```cfg
            [gcode_macro Z_TILT_ADJUST]
            rename_existing: _Z_TILT_ADJUST
            gcode:
                SAVE_GCODE_STATE NAME=STATE_Z_TILT
                BED_MESH_CLEAR
                {% if not printer.z_tilt.applied %}
                _Z_TILT_ADJUST horizontal_move_z=10 retry_tolerance=1
                {% endif %}
                _Z_TILT_ADJUST horizontal_move_z=2 METHOD=rapid_scan ADAPTIVE=1
                # G28 Z
                RESTORE_GCODE_STATE NAME=STATE_Z_TILT
            ```

    ### Bett-Scan-Optimierung

        * Wenn ein leistungsstarkes AC-Heizbett (über 500W) verwendet wird
        * Das folgende Makro schaltet das Heizbett beim Scannen aus und heizt es nach Abschluss des Scans wieder auf
            ```cfg
            [gcode_macro BED_MESH_CALIBRATE]
            rename_existing: _BED_MESH_CALIBRATE
            gcode:
                {% set TARGET_TEMP = printer.heater_bed.target %}
                M140 S0
                _BED_MESH_CALIBRATE {rawparams}
                M140 S{TARGET_TEMP}
            ```
    ## Bett-Befehle
    
        * Befehl zum Scannen des gesamten Betts
            ```bash
            BED_MESH_CALIBRATE METHOD=rapid_scan
            ```
        * Befehl für lokales Bett-Scanning, stellen Sie sicher, dass Klipper die Objektauslassfunktion verwenden kann, empfohlen, in den Start-Makro hinzuzufügen
            ```bash
            BED_MESH_CALIBRATE METHOD=rapid_scan adaptive=1 
            ```
