# Druckbeginn: Temperaturinstabilität und zu lange Wartezeit

    * Dieses Makro kann direkt in die Konfiguration eingefügt werden
    * Dieses Makro sorgt dafür, dass normal auf die Erhöhung der Temperatur gewartet wird, aber nach Erreichen der Ziel temperatur sofort weitergemacht wird
    ```cfg title="printer.cfg"
    [gcode_macro M109]
    rename_existing: M109.1
    gcode:
        {% set s = params.S|float %}
        M104 {% for p in params %}{'%s%s' % (p, params[p])}{% endfor %}  ; Set hotend temp
        {% if s != 0 %}
        TEMPERATURE_WAIT SENSOR=extruder MINIMUM={s} MAXIMUM={s+1}   ; Wait for hotend temp (within 1 degree)
        {% endif %}	
    ```
