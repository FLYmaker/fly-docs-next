# Mehrfach-Z-Referenz-Konfiguration

## Doppel-Z mit einem Endschalter

* Beim zweiten Z müssen Sie kein `endstop_pin:` hinzufügen

    ```cfg
    [stepper_z]
    step_pin: PA5
    dir_pin: PA4
    enable_pin: !PA6
    rotation_distance: 4
    full_steps_per_rotation: 200
    microsteps: 16
    endstop_pin: ^!PA9
    position_endstop: 0
    position_max: 300
    homing_speed: 50
    homing_retract_dist: 5

    [stepper_z1]
    step_pin: PB10
    dir_pin:  PB2
    enable_pin: !PB11
    rotation_distance: 4
    full_steps_per_rotation: 200
    microsteps: 16
    step_pulse_duration: 0.000004
    ```

## Doppel-Z mit zwei Endschaltern


* Beim zweiten Z müssen Sie `endstop_pin:` hinzufügen

    ```cfg
    [stepper_z]
    step_pin: PA5
    dir_pin: PA4
    enable_pin: !PA6
    rotation_distance: 4
    full_steps_per_rotation: 200
    microsteps: 16
    endstop_pin: ^!PA9
    position_endstop: 0
    position_max: 300
    homing_speed: 50
    homing_retract_dist: 5
    step_pulse_duration: 0.000004

    [stepper_z1]
    step_pin: PB10
    dir_pin:  PB2
    enable_pin: !PB11
    rotation_distance: 4
    full_steps_per_rotation: 200
    microsteps: 16
    step_pulse_duration: 0.000004
    endstop_pin: ^!PA10
    ```

## Doppel-Z mit Probe oder BL Touch


    * Bei der ersten Z-Konfiguration `endstop_pin:` ändern Sie den Pin auf `probe:z_virtual_endstop  `
    * Sie müssen Probe oder BL Touch konfigurieren

        ```cfg
        [stepper_z]
        step_pin: PA5
        dir_pin: PA4
        enable_pin: !PA6
        rotation_distance: 4
        full_steps_per_rotation: 200
        microsteps: 16
        endstop_pin: probe:z_virtual_endstop  
        # position_endstop: 0
        position_max: 300
        homing_speed: 50
        homing_retract_dist: 5
        step_pulse_duration: 0.000004

        [probe]
        pin: ^PB4                    # Signal-Schnittstelle
        x_offset: 0                  # X-Achse - Versatz des Sensors relativ zur Düse
        y_offset: 0                  # Y-Achse - Versatz des Sensors relativ zur Düse
        #z_offset: 0                 # Z-Achse - Versatz des Sensors relativ zur Düse
        speed: 3.0                   # Nivellierungsgeschwindigkeit
        lift_speed: 5                # Geschwindigkeit beim Anheben des Sensors
        samples: 3                   # Anzahl der Proben
        samples_result: median       # Art der Wertung (Standardwert median - Medianwert)
        sample_retract_dist: 3.0     # Nivellierungsrückzugentfernung
        samples_tolerance: 0.075     # Proben-Toleranz (Beachten Sie, dass ein zu niedriger Wert die Anzahl der Proben erhöhen kann)
        samples_tolerance_retries: 3 # Anzahl der Wiederholungen bei Überschreitung der Toleranz
        ```
