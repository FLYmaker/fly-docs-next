Hier sind die Anweisungen zum Kompilieren der Firmware:

* Verbinden Sie sich über SSH und geben Sie die folgenden Befehle ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte das folgende Fenster erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Zusätzliche Low-Level-Konfigurationsoptionen aktivieren** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **STMicroelectronics STM32** aus und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Prozessormodell**, wählen Sie **STM32F072** aus und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Offset**, wählen Sie: **8KiB Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="Center" />

* Wählen Sie **Kommunikationsschnittstelle**, wählen Sie: **CAN-Bus (auf PB8/PB9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.png').default} size="100%" align="Center" />
