Hier wird erläutert, wie man die Firmware kompiliert:
* Nach dem Verbinden mit SSH geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei ` rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Erweiterte Low-Level-Konfigurationsoptionen aktivieren** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Prozessormodell**, wählen Sie **STM32F072** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Versatz**, wählen Sie: **8KiB Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="Center" />

* **Kommunikationsschnittstelle**, wählen Sie: **USB (auf PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb.png').default} size="100%" align="Center" />
