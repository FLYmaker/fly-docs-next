Hier ist die Anleitung zum Kompilieren der Firmware:

* Nach der Verbindung zum SSH geben Sie die folgenden Befehle ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und Firmware, `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die folgende Benutzeroberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Processor model**, wählen Sie **STM32F072** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset**, und wählen Sie: **(No bootloader)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/no.png').default} size="100%" align="Center" />

* **Communication interface**, wählen Sie: **USB (on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb1.png').default} size="100%" align="Center" />
