Hier ist die Anleitung, wie man die Firmware kompiliert:
* Nachdem Sie sich über SSH verbunden haben, geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei löscht `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` die vorherigen Kompilierungsdaten und Firmware, `make menuconfig` kompiliert die Firmware, und danach sollte die folgende Benutzeroberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Processor model** und wählen Sie **STM32F072** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset**, wählen Sie: **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="Center" />

* Wählen Sie **Communication interface**, wählen Sie: **USB to CAN bus bridge (USB on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.png').default} size="100%" align="Center" />
