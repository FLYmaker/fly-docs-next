Hier ist die Anleitung zur Kompilierung der Firmware:

* Verbinden Sie sich über SSH und geben Sie den folgenden Befehl ein:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und Firmware, und `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die folgende Benutzeroberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Zusätzliche niedrige Konfigurationsoptionen aktivieren** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **Raspberry Pi RP2040**, dann drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Offset**, wählen Sie: **16KiB Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.png').default} size="100%" align="Center" />

* Wählen Sie **Kommunikationsschnittstelle**, wählen Sie: **CAN-Bus**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can.png').default} size="100%" align="Center" />
