    * Kurzschließen oder langes Drücken von BOOT, nach dem Anschließen an den Hostcomputer über Type C und Eingabe von `lsusb` erscheint `2e8a:0003` 
    * Beachten Sie, dass der RP2040 sich derzeit im Firmware-Brennmodus befindet und nur Firmware-Dateien mit der Endung .uf2 brennen kann. Nach dem Brennen der Firmware verschwindet diese.
    * Nach dem Auswerfen von `RPI-RP2` ist es nicht mehr notwendig, BOOT kurz zu schließen oder den BOOT-Button gedrückt zu halten.
        <ImageView image={require('@site/docs/General/get-id/img/2e8a.png').default} size="80%" align="Center" /> 
    * Verwenden Sie den folgenden Befehl, um die Firmware auf den RP2040 zu brennen:

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
