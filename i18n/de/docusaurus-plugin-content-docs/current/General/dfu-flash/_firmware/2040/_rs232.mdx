Hier wird erläutert, wie man Firmware kompilieren kann:
* Nachdem Sie sich über SSH verbunden haben, geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die zuvor kompilierten Daten und die Firmware `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Zusätzliche Low-Level Konfigurationsmöglichkeiten aktivieren** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **Raspberry Pi RP2040** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Versatz**, wählen Sie: **KEIN Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.png').default} size="100%" align="Center" />

* Wählen Sie **Kommunikationsschnittstelle**, wählen Sie: **UART0 auf GPIO0/GPIO1**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/rs232.png').default} size="100%" align="Center" />
