Hier sind die Anweisungen zum Kompilieren des Firmware:

* Nach dem Verbinden zu SSH, geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Erweiterte Low-Level-Konfigurationsmöglichkeiten aktivieren** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **Raspberry Pi RP2040** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Versatz**, wählen Sie: **16KiB Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.png').default} size="100%" align="Center" />

* Wählen Sie **Kommunikationsschnittstelle**, wählen Sie: **USB zu CAN-Bus-Bridge**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/utoc.png').default} size="100%" align="Center" />
