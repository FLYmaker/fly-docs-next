Hier ist die Anleitung zum Kompilieren der Firmware:

*   Verbinden Sie sich über SSH und geben Sie die folgenden Befehle ein:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

*   Dabei ` rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, `make menuconfig` kompiliert die Firmware. Danach sollte das folgende Interface erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

*   Wählen Sie **Enable extra low-level configuration options** und drücken Sie Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

*   Gehen Sie in das Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

*   Gehen Sie in das Menü **Processor model** und wählen Sie **STM32F405** und drücken Sie Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.png').default} size="100%" align="Center" />

*   Wählen Sie **Bootloader offset** und wählen Sie: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.png').default} size="100%" align="Center" />

*   **Communication interface**, wählen Sie: **USB (on PA11/PA12)**

   <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/usb.png').default} size="100%" align="Center" />
