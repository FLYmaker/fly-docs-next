Hier wird erläutert, wie man Firmware kompilieren kann:
* Nach dem Verbinden zu SSH, geben Sie die folgenden Befehle ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, `make menuconfig` kompiliert die Firmware, nach der Ausführung sollte folgendes Interface erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Zusätzliche tiefere Konfigurationsoptionen aktivieren** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Betreten Sie das Menü **Mikrocontroller-Architektur** und wählen Sie dann **STMicroelectronics STM32** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

Bilder werden aktualisiert
    * Betreten Sie das Menü **Prozessormodell**, wählen Sie **STM32F405** und drücken Sie die Eingabetaste

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.png').default} size="100%" align="Center" />

    * Wählen Sie **Bootloader-Offset**, wählen Sie: **32KiB Bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.png').default} size="100%" align="Center" />

    * Wählen Sie **Kommunikationsschnittstelle**, wählen Sie: **USB zu CAN-Bus Bridge (USB an PA11/PA12)**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/utoc.png').default} size="100%" align="Center" />
