Hier ist, wie man Firmware kompilieren kann:
* Nachdem Sie sich über SSH verbunden haben, geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die zuvor kompilierte Firmware und `make menuconfig` beginnt die Firmware-Kompilation. Danach sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Processor model**, wählen Sie **STM32F407** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/407.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset** und wählen Sie: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/32kb.png').default} size="100%" align="Center" />

* Wählen Sie **Communication interface** und wählen Sie: **USB to CAN bus bridge (USB on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/utoc.png').default} size="100%" align="Center" />
