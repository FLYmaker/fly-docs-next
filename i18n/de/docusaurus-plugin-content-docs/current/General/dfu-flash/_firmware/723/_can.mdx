Nachstehend wird erläutert, wie man die Firmware kompilieren kann:
* Verbinden Sie sich über SSH und geben Sie die folgenden Befehle ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, `make menuconfig` kompiliert die Firmware. Nach der Ausführung sollte die untere Oberfläche erscheinen

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Wechseln Sie in das Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Wechseln Sie in das Menü **Processor model** und wählen Sie **STM32H723** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset**, wählen Sie: **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.png').default} size="100%" align="Center" />

* Wählen Sie **Clock Reference (8 MHz crystal)**, wählen Sie: **25 MHz crystal**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Communication interface**, wählen Sie: **CAN bus (on PB8/PB9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/can.png').default} size="100%" align="Center" />
