Hier ist die Anleitung zum Kompilieren des Firmware:

* Nachdem Sie sich über SSH verbunden haben, geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht die vorherigen Kompilierungsdaten und die Firmware, und `make menuconfig` startet die Firmware-Kompilierung. Nach der Ausführung sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie zum Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32**, dann drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie zum Menü **Processor model**, wählen Sie **STM32H723** und drücken Sie die Eingabetaste:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset**, wählen Sie: **NO bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.png').default} size="100%" align="Center" />

* Wählen Sie **Clock Reference (8 MHz crystal)**, wählen Sie: **25 MHz crystal**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Communication interface**, wählen Sie: **Serial (on USART1 PA10/PA9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/uart.png').default} size="100%" align="Center" />
