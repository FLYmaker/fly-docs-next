Hier sind die Anweisungen, wie man die Firmware kompilieren kann:
* Nach dem Verbinden via SSH geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Wobei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` die vorherigen Kompilierungsdaten und Firmware löscht und `make menuconfig` die Firmware kompiliert. Danach sollte die folgende Oberfläche erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Zusätzliche Low-Level-Konfigurationsoptionen aktivieren** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Mikrocontroller-Architektur** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie in das Menü **Prozessormodell**, wählen Sie **STM32H723** und drücken Sie die Eingabetaste

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader-Offset**, wählen Sie: **128KiB Bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.png').default} size="100%" align="Center" />

* Wählen Sie **Referenz der Taktfrequenz (8 MHz Kristall)**, wählen Sie: **25 MHz Kristall**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Kommunikationsschnittstelle**, wählen Sie: **USB (auf PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/usb.png').default} size="100%" align="Center" />
