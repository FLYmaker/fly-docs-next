Hier sind die Schritte zur Firmware-Kompilierung:

* Nach dem Herstellen einer SSH-Verbindung geben Sie den folgenden Befehl ein und drücken Sie die Eingabetaste:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dabei `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` löscht vorherige Kompilierungsdaten und Firmware, `make menuconfig` kompiliert die Firmware. Danach sollte das folgende Bildschirm erscheinen:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Wählen Sie **Enable extra low-level configuration options** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Gehen Sie zum Menü **Micro-controller Architecture** und wählen Sie **STMicroelectronics STM32** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Gehen Sie zum Menü **Processor model**, wählen Sie **STM32H723** und drücken Sie die Eingabetaste.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Wählen Sie **Bootloader offset**, und wählen Sie: **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.png').default} size="100%" align="Center" />

* Wählen Sie **Clock Reference (8 MHz crystal)**, und wählen Sie: **25 MHz crystal**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* Wählen Sie **Communication interface**, und wählen Sie: **USB to CAN bus bridge (USB on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.png').default} size="100%" align="Center" />
