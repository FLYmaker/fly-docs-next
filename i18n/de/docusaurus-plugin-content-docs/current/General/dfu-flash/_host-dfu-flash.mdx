    1. Installieren Sie das Flash-Tool

    * FLY Host-Tool benötigt keine Installation des Flash-Tools
    * Wenn Sie installieren müssen, beachten Sie: Der Host-Tool benötigt eine Internetverbindung zur normalen Installation
    * Wenn es bereits installiert ist, ist keine erneute Installation erforderlich

    ```bash
    sudo apt update
    sudo apt install dfu-util -y
    ```

    2. Führen Sie den folgenden Befehl aus, um zu überprüfen, ob die Verbindung erfolgreich war.

    ```bash
    lsusb
    ```

    * Wenn kein DFU-Gerät angezeigt wird, versuchen Sie bitte, den **Flash-Modus** erneut zu betreten
    * Aufgrund von Systemversionen können andere Namen angezeigt werden. Wenn `0483:df11` vorhanden ist, bedeutet dies, dass der Flash-Modus betreten wurde


        <ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.png').default} size="80%" align="left" />    


    3. Firmware flashen

    * Führen Sie den untenstehenden Code aus, um mit dem Flashen zu beginnen
    
    ```bash
    sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/klipper/out/klipper.bin
    ```

    4. Wenn der untenstehende Pfeil auf die angezeigte Meldung ``File downloaded successfully`` zeigt, war das Flashen erfolgreich

    <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.png').default} size="80%" align="left" />    

    :::tip

    Nach Abschluss des Flash-Vorgangs muss der Stromversorgung der Hauptplatine vollständig unterbrochen werden.

    :::
