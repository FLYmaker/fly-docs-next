1. Installieren Sie das Flash-Tool

* FLY Host-Software benötigt keine Installation des Flash-Tools
* Wenn Sie installieren müssen, beachten Sie bitte: Die Host-Software muss eine Internetverbindung haben, um die Installation durchzuführen
* Wenn es bereits installiert ist, ist keine erneute Installation erforderlich

```bash
sudo apt update
sudo apt install dfu-util -y
```

2. Führen Sie den folgenden Befehl aus, um zu überprüfen, ob die Verbindung erfolgreich ist.

```bash
lsusb
```

* Wenn kein DFU-Gerät angezeigt wird, versuchen Sie, den **Flash-Modus** erneut zu betreten
* Aufgrund von Systemversionen könnte ein anderer Name angezeigt werden, wenn `0483:df11` vorhanden ist, bedeutet dies, dass der Flash-Modus betreten wurde

<ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.png').default} size="100%" align="left" />


3. Laden Sie die vorkompilierte Firmware herunter

* Hier befinden sich alle vorkompilierten Firmwares
* Beachten Sie bitte, dass die Host-Software eine Internetverbindung benötigt, um die Installation durchzuführen
* Die Host-Software muss die folgenden Befehle ausführen, um das Firmware-Paket herunterzuladen

```bash
cd && git clone https://github.com/FengHua741/FLY_Katapult.git
```

* Wenn das Firmware-Paket bereits heruntergeladen wurde, wird empfohlen, das Firmware-Paket zu aktualisieren, bevor Sie die restlichen Schritte durchführen

```bash
cd ~/FLY_Katapult && git pull
```
