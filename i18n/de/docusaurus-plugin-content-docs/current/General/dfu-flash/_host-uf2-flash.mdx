1. Installieren Sie das Brennwerkzeug

    * Geben Sie die folgenden Befehle in SSH ein
    * Nur beim ersten Brennen erforderlich, danach nicht mehr ausführen
    ```bash
    cd ~/klipper/lib/rp2040_flash/ && make
    ```
    <ImageView image={require('@site/docs/General/dfu-flash/img/2040.png').default} size="100%" align="left" />

2. Führen Sie die folgenden Befehle aus, um zu überprüfen, ob die Verbindung erfolgreich war.

    * Geben Sie `lsusb` ein
    * Wenn kein `2e8a:0003` Gerät angezeigt wird, versuchen Sie, den **Brennmodus** erneut zu betreten
    * Aufgrund von Systemversionen kann ein anderer Name angezeigt werden, wenn `2e8a:0003` vorhanden ist, bedeutet dies, dass der Brennmodus aktiviert wurde

        <ImageView image={require('@site/docs/General/get-id/img/2e8a.png').default} size="80%" align="left" /> 

    :::warning Beachten Sie

    Fahren Sie nur fort, wenn `2e8a:0003` vorhanden ist

    :::

3. Rufen Sie die vorab kompilierte Firmware ab

    * Hier sind vorab kompilierte Firmwares
    * Beachten Sie, dass die Host-Maschine eine Internetverbindung benötigt, um die Installation durchzuführen
    * Die Host-Maschine muss die folgenden Befehle ausführen, um das Firmware-Paket herunterzuladen

    ```bash
    cd && git clone https://github.com/FengHua741/FLY_Katapult.git
    ```

    * Wenn das Firmware-Paket bereits heruntergeladen wurde, wird empfohlen, das Paket zu aktualisieren, bevor die restlichen Operationen fortgesetzt werden

    ```bash
    cd ~/FLY_Katapult && git pull
    ```
