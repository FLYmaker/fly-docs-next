    1. Brenntool <Button variant="contained" disableElevation href="https://mellow-next.klipper.cn/docs/ResDownload/auxiliary_software/">Download-Link</Button>
    2. Sie müssen `DFUProgrammer` herunterladen
    3. Kopieren Sie das vorher kompilierte Firmware (klipper.bin) oder die vorkompilierte `Katapult-Firmware` in ein beliebiges Verzeichnis auf Ihrem Computer
    4. Verbinden Sie die Hauptplatine mit Ihrem Computer über ein Type-C-Kabel, stellen Sie sicher, dass Sie den **DFU-Brennmodus** aktiviert haben
    5. Öffnen Sie DFUProgrammer und wählen Sie wie unten gezeigt
        <ImageView image={require('@site/docs/General/dfu-flash/img/apmdfu.png').default} size="100%" align="center" />
        <ImageView image={require('@site/docs/General/dfu-flash/img/apmdfu1.png').default} size="100%" align="center" />
    6. Klicken Sie auf Starten, um die Firmware zu aktualisieren und warten Sie kurz, nach Abschluss können Sie das Type-C-Kabel und den Jumper entfernen
