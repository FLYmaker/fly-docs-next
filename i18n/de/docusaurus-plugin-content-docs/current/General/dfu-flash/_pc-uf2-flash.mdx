    * Bereiten Sie ein Datenkabel mit Datenübertragungsfunktion vor
    * Laden Sie das Firmware-Update auf Ihren Computer herunter
    * Verbinden Sie die BOOT-Jumperleitung oder halten Sie die Taste gedrückt, während Sie den Computer anschließen; es wird der `RPI-RP2`-Datenträger erscheinen
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.png').default} size="100%" align="center" />
    * Entfernen Sie die BOOT-Jumperleitung oder lassen Sie den Knopf los！！！
    * Kopieren Sie die Firmware, die Sie flashen möchten, in den Datenträger. Beachten Sie, dass nur Firmware mit der Endung `UF2` verwendet werden kann！！！ (Bitte ändern Sie die Endung nicht, das hat keinen Sinn)
    * Referenz
        <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.png').default} size="100%" align="center" />
    * Nach dem Kopieren der Firmware wird der `RPI-RP2`-Datenträger automatisch verschwinden, zu diesem Zeitpunkt ist das Flashen der Firmware abgeschlossen！
