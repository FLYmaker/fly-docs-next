## Benutzerhandbuch für die FLY-Oberfläche

:::tip
Öffne deinen Browser mit deiner IP:9999, um auf FLY-Tools zuzugreifen.
:::

:::tip
Beachte beim Kauf eines HDMI-Kabels, dass die HDMI-Schnittstelle auf der FLY-Oberfläche ein Micro HDMI ist!!!
:::

:::tip
Die FLY-Oberfläche unterstützt FPC One-Line-Communication für `FLY-PI-V2`, `FLY-C8` und `FLY-PRO-X10`.
:::

:::tip
Mit dem FPC-zu-HDMI-Modul können auch andere Oberflächen wie Raspberry Pi verwendet werden.
:::
