# Brennen auf M2WE über eine TF-Karte

:::info
Die Systemmigration kann das vorhandene Abbild auf der TF-Karte vollständig auf das M2WE übertragen. Nach Abschluss der Migration entspricht der Inhalt des M2WE dem ursprünglichen Inhalt der TF-Karte.
:::

    * Installieren Sie das M2WE und starten Sie es durch Anschließen der Stromversorgung
    * Bitte verbinden Sie sich zuerst über eine serielle Schnittstelle [mit SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Klicken Sie, um zum Ziel zu springen") 
    * Führen Sie den untenstehenden Befehl aus, es könnte eine Eingabeaufforderung für ein Passwort erscheinen, geben Sie das Passwort `mellow` ein und drücken Sie Enter, beachten Sie, dass das Passwort nicht sichtbar ist

    ```cfg
    sudo nand-sata-install
    ```

    * Wählen Sie die Option `Boot from eMMC - system on eMMC` und drücken Sie Enter (normalerweise ist dies die Standardoption, drücken Sie einfach Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_1.png').default} size="80%" align="left" />

    * Wenn die Nachricht `This script will erase your eMMC. Continue?` erscheint, wählen Sie `Yes` und drücken Sie Enter (normalerweise ist dies die Standardoption, drücken Sie einfach Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_2.png').default} size="80%" align="left" />

    * Wählen Sie das Dateisystem `ext4` und drücken Sie Enter (normalerweise ist dies die Standardoption, drücken Sie einfach Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_3.png').default} size="80%" align="left" />

    * Warten Sie geduldig, bis die Fortschrittsanzeige erscheint (falls nach mehr als 2 Minuten die untenstehende Fortschrittsanzeige nicht erscheint, drücken Sie `CTRL+C`, um den Vorgang abzubrechen, und starten Sie das Gerät neu, bevor Sie es erneut versuchen)
    
    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_4.png').default} size="80%" align="left" />
    * Während des Migrationsprozesses, unterbrechen Sie nicht die Stromversorgung und führen Sie keine anderen Operationen durch, bis die Migration abgeschlossen ist
    * Nach Abschluss der Migration drücken Sie einfach Enter und warten, bis das Gerät vollständig heruntergefahren ist

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_5.png').default} size="80%" align="left" />


    * Nach dem Herunterfahren entfernen Sie die TF-Karte und starten Sie das Gerät erneut, nun kann es über das System in der eMMC gestartet werden
