:::tip
Öffne deinen Browser und gib deine IP:9999 ein, um auf FLY-Tools zuzugreifen.
:::

    * Um den TFT-Bildschirm zu verwenden, müssen die folgenden Optionen aktiviert werden.
    * Beachte, dass es drei Arten von TFT-Bildschirmen gibt:
        * `FLY-TFT-V1` V1 Widerstandsbildschirm, kompatibel mit allen FLY-Hosts
        * `FLY-TFT-V2-R` V2 Widerstandsbildschirm, kompatibel mit C8, PI-V2 und Raspberry Pi mit 40PIN, erfordert jedoch eine G2T-Paarung
        * `FLY-TFT-V2-C` V2 Kapazitiver Bildschirm, kompatibel mit C8, PI-V2 und Raspberry Pi mit 40PIN, erfordert jedoch eine G2T-Paarung
