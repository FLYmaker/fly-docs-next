## Verbindung zum HMI

    * Falls die WiFi-Konfiguration im BOOT-Verzeichnis fehlschlägt, befolgen Sie diese Anleitung zur WiFi-Verbindung, achten Sie darauf, dass Sie eine serielle Verbindung zum HMI herstellen
    * Verbinden Sie sich zuerst per serieller Schnittstelle [mit SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Klicken Sie hier zum Wechseln")
    * Führen Sie den folgenden Befehl aus, es kann sein, dass Sie aufgefordert werden, ein Passwort einzugeben, geben Sie das Passwort `mellow` ein und drücken Sie Enter, beachten Sie, dass das Passwort nicht sichtbar ist

    ```cfg
    sudo nmtui
    ```
