## Verwendung der CSI-Kamera

    * Nach dem Brennen des Images, öffnen Sie /boot/config.txt, suchen Sie nach `camera_auto_detect=1` und ändern Sie es in `camera_auto_detect=0`, dann fügen Sie die Konfiguration gemäß dem Kameramodell hinzu
    * Unten finden Sie eine Beispielkonfiguration

    ```bash
    dtoverlay=imx219,cam1
    ```
