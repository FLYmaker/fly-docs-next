## Raspberry Pi 5 mit MIPI

:::tip
FLY_MIPI ist ein speziell für den Raspberry Pi 5 entwickeltes Adaptermodul, das die direkte Verwendung von DSI- und CSI-Geräten ermöglicht.
:::

### Raspberry Pi 5 mit MIPI Verkabelung

    :::danger Wichtig

    * Die DSI- und CSI-Schnittstellen des Raspberry Pi 5 heißen jetzt MIPI
    * Die MIPI-Schnittstelle kann durch Konfiguration direkt für CSI-Kameras oder DSI-Bildschirme verwendet werden
    * Achten Sie auf die Ausrichtung der Kontakte

    :::

    * Der Raspberry Pi 5 hat jetzt zwei MIPI-Schnittstellen, die beim Betrachten von vorne mit 1 und 0 markiert sind


## DSI-Bildschirm Verkabelung

    * Wenn Sie die Schnittstelle 1 verwenden, konfigurieren Sie den Bildschirm wie folgt:

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * Wenn Sie die Schnittstelle 0 verwenden, konfigurieren Sie den Bildschirm wie folgt:

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## CSI-Kamera Verkabelung

    <ImageView image={require('@site/docs/General/rpi/img/csi1.png').default} size="100%" align="left" />
    * Achten Sie darauf, die Kamera nach dem Modell zu konfigurieren, das Sie gekauft haben. Dieses Tutorial verwendet imx219 als Referenz.

    * Wenn Sie die linke MIPI-Schnittstelle verwenden, konfigurieren Sie die Kamera wie folgt:
    ```bash
    dtoverlay=imx219,cam1
    ```

    * Wenn Sie die rechte MIPI-Schnittstelle verwenden, konfigurieren Sie die Kamera wie folgt:
    ```bash
    dtoverlay=imx219,cam0
    ```
