## Referenzkonfiguration

    ```cfg
    [fan]
    ##    Druckkühlventilator 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Lösung für automatisches Drehen beim Einschalten

    :::info
    * Hier nur eine einfache Erklärung
    * Beim Kompilieren des Firmware die PWM-Signalpin hinzufügen und seinen Zustand umkehren
    * Test mit SUPER8PRO durchgeführt, Pin PF8 verwendet, bei der Firmware-Kompilierung wählen Sie `GPIO pins to set at micro-controller startup` und fügen Sie `!PF8` hinzu
    * Achten Sie darauf, dass das `!` ein englisches Ausrufezeichen ist
    * Andere Hauptplatinen analog
    :::
