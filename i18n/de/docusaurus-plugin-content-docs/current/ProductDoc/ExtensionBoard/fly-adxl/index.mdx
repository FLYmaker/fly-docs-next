## Produktmerkmale

    * Geeignet für 36er Motoren und andere Anwendungen
    * Kann an Raspberry Pi, FLY PI, Orange Pi, Mainboard und andere SPI-Schnittstellen angeschlossen werden

## Produktspezifikationen

    * Eingebaute Schnittstelle: ZH1.5 6P-Stecker
    * Kommunikationsmethode: SPI
    * Betriebsspannung: 5V

## Hinweise

    :::warning Hinweise

    * Betriebsspannung 5V, bitte nicht falsch anschließen
    * Geeignet für verschiedene Firmware, bitte beachten Sie, dass eine SPI-Verbindung möglich sein muss

    :::
