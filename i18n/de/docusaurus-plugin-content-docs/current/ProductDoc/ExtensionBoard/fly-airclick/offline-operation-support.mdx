# Kabelverbindungskonfiguration

:::danger Wichtig

* Nach dem Flashen des STM32F072 mit der Firmware muss eine Verbindung zu Klipper hergestellt werden, sonst wird der Betrieb über eine Kabelverbindung nicht unterstützt.

:::

## Beginnen Sie mit der Kompilierung

    <SSH />

    <MakeOperate/>

    <USB />

    * Drücken Sie die `Q` Taste, um **Save configuration** anzuzeigen, und dann die `Y` Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und die Befehlszeile verlassen sein
    * Geben Sie `make -j4` ein, um mit der Kompilierung zu beginnen, die etwas Zeit in Anspruch nehmen kann
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn die Kompilierung erfolgreich war, wird der folgende Text ausgegeben:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## In den Programmiermodus wechseln

    * Halten Sie beim abgeschalteten AirClick die BOOT-Taste des STM32 gedrückt und verbinden Sie das Programmierinterface des STM32 über Type-C mit dem Host
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.png').default} size="100%" align="center" />
