# FLY_halo

## 1.  Startseite

    Klicken Sie auf das Heizbett oder die Heizdüse und bewegen Sie dann den Finger im Uhrzeigersinn von der oberen Bildschirmkante zur gewünschten Temperatur. Ein erneutes Klicken auf das Heizbett oder die Heizdüse setzt die Temperatur.

    Wenn Sie die MCU-Temperatur und die Temperatur des Host-Computers anzeigen möchten, fügen Sie diese Konfiguration hinzu:

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # Verknüpft mit der MCU (Standard)

    [temperature_sensor host]        # Temperatur des Host-Computers
    sensor_type: temperature_host     # Verknüpft mit dem Host
    ```


    :::warning Hinweis

    * Beachten Sie, dass die Höchsttemperatur aus den Konfigurationseinstellungen gelesen wird

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.png').default} size="50%" align="left" />


## 2.  Extrusion

    * Langes Drücken auf das Symbol ermöglicht das Setzen der Heizdüsentemperatur, der Extrusionslänge und -geschwindigkeit

    :::warning Hinweis

    * Halo unterstützt nur die Steuerung einer einzigen Heizdüse

    :::
