## Installation auf Voron Stealthburner

    * Für die Installation von Halo auf Stealthburner sind unsere Adapterteile erforderlich
    * Laden Sie zunächst die beiden von uns bereitgestellten STL-Dateien herunter und drucken Sie sie aus
        * [Stealthburner_main_body(Adapt_to_Halo).stl(Herunterladen)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(Herunterladen)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(Herunterladen)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Installation der Adapterplatine und Anschlusskabel
    * Der Ausgangspunkt kann nach Ihrem Ermessen gewählt werden, er muss nicht direkt an der Rückseite des Lüfters sein
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.png').default} size="50%" align="left" />


    :::warning Wichtige Hinweise

    * Farbcodierung der Leitungen: **Schwarz ist der negative Pol (GND)**, **Rot ist der positive Pol (VCC) und akzeptiert Spannungen von DC12-24V**, **Weiß ist GPIO47 und kann für die Materialausfallerkennung verwendet werden**, **Gelb ist GPIO48 und kann für das Anhalten nach dem Drucken verwendet werden**

    :::
