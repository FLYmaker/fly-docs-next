# Einführung in die Bedienung der Halo-Seite

## Seitenstruktur

```bash
  Druckdateien-Seite     Startseite        Extruder-Seite      Systemdienste      Einstellungen      Geräteinformationen
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Druckdateien|<-->|   Startseite|<-->|   Extruder  |<-->| Systemdienste|<-->| Einstellungen|<-->| Geräteinfo  |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blockierung  |
                   |             |
                   +-------------+
                     Blockierdetektor-Seite
            (Diese Seite erscheint nur, wenn die Blockierdetektor-Einstellung aktiviert ist)
```

## Extruder-Seite

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.png').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.png').default} size="50%" align="left" />
    * Die bedienbaren Bereiche in Abbildung 1 entsprechen den folgenden Nummern:
    1. Einmal klicken: Zieltemperatur des Extruders +1
    2. Langes Drücken: Zugriff auf die Einstellung der Extrudertemperatur, Anpassung der Temperatur durch Ziehen des Rings, erneut klicken um die Einstellung zu bestätigen
    3. Einmal klicken: Zieltemperatur des Extruders -1
    4. Einmal klicken: Extrudieren
    5. Hoch und Runter wischen: Auswahl der Länge für Extrusion oder Rückzug, Einheit mm
    6. Einmal klicken: Rückzug
    7. Langes Drücken: Zugriff auf die Einstellung der Extrusionsgeschwindigkeit in Abbildung 2, Hoch und Runter wischen zur Auswahl der Geschwindigkeit, Einheit mm/s, danach auf eine leere Stelle klicken um zu verlassen
