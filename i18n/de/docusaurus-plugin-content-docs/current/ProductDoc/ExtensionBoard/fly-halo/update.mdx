## Inkrementelle Aktualisierung (OTA)

    * Auf der Geräteinformationen-Seite von Halo kann man auf die Versionsnummer klicken, um nach Updates zu suchen. Wenn ein Update verfügbar ist, wird eine Nachricht angezeigt. Klicken Sie dann auf "Update", um das Upgrade zu starten. Unterbrechen Sie nicht den Strom während des Prozesses; nach Erreichen von 100% wird das Gerät automatisch neu gestartet.


## Vollständiges Upgrade (USB)

    * Wenn eine vollständige Firmware-Aktualisierung veröffentlicht wird, müssen Sie Halo über USB mit dem Computer verbinden, um das Upgrade durchzuführen.
    :::tip

    * Bei einem vollständigen Upgrade werden Ihre ursprünglichen Einstellungen nicht beibehalten, alle Einstellungen werden auf die Voreinstellungen zurückgesetzt.

    :::

    1. Laden Sie das USB-Brennwerkzeug herunter und entpacken Sie es [flash_download_tool_3.9.4.zip (zum Herunterladen klicken)](https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip)
    2. Laden Sie die Firmware herunter [Halo Firmware-Download](docs/ProductDoc/ExtensionBoard/fly-halo/firmware.mdx)
    3. Öffnen Sie ``flash_download_tool_3.9.4.exe``. Beachten Sie: Nach dem Öffnen des Brennwerkzeugs wird zunächst ein CMD-Fenster angezeigt, danach erscheint die Oberfläche des Brennwerkzeugs. Schließen Sie das CMD-Fenster auf keinen Fall.
    4. Wählen Sie im Brennwerkzeug unter ``ChipType:`` ``ESP32-S3`` aus
    5. Wählen Sie im Brennwerkzeug unter ``WorkMode:`` ``Develop`` aus
    6. Wählen Sie im Brennwerkzeug unter ``LoadMode:`` ``USB`` aus
    7. Klicken Sie abschließend auf ``OK``, siehe Abbildung 1

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/flash-1.png').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/flash-2.png').default} size="50%" align="left" />

    :::tip

    * Das ``grün markierte Gebiet`` in Abbildung 2 muss genau mit dem Bild übereinstimmen, ändern Sie es nicht selbstständig.

    :::

    8. Verbinden Sie Halo mit dem Computer über ein USB-Typec-Kabel
    9. Klicken Sie auf den ``blauen Bereich 1`` in Abbildung 2, um die heruntergeladene Firmware auszuwählen
    10. Klicken Sie auf den ``blauen Bereich 2`` in Abbildung 2, um den COM-Port von Halo auszuwählen (falls nicht vorhanden, halten Sie die ``B``-Taste auf der Rückseite von Halo gedrückt und verbinden Sie dann mit dem Computer)
    11. Klicken Sie auf den ``blauen Bereich 3`` in Abbildung 2, um mit dem Brennen zu beginnen. Nach Abschluss des Brennvorgangs starten Sie Halo neu, um das vollständige Upgrade abzuschließen (Sie dürfen das Kabel erst nach der Bestätigung, dass der Brennvorgang abgeschlossen ist, trennen)
