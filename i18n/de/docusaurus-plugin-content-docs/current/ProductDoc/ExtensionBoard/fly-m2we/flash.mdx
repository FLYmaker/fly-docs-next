## Vorbereitungen vor dem Systemflashing

    * Kartenleser
    * Eine TF-Karte, Kapazität muss nicht kleiner als 16GB und nicht größer als 128GB sein, die Übertragungsgeschwindigkeit der TF-Karte muss mindestens Klassen 10 oder höher sein
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">Systemabbild</Button> herunter
    * Laden Sie die benötigten <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Flashing Tools</Button> herunter

## Systemflashing

:::info
M2WE unterstützt drei Modi, ich empfehle die Migration von TF-Karte auf M2WE und das Flashing auf M2WE mittels M2SD Burner
:::

:::info
* Wenn das Gerät mit M2WE und Speicherkarte ausgestattet ist, startet das System standardmäßig von der Speicherkarte
* Es wird empfohlen, das System mit einer TF-Karte zu flashen, da das Flashing über USB auf M2W aufgrund von Treiberkompatibilitätsproblemen fehlschlagen kann, es wird empfohlen, direkt die [Speicherkarte auf M2WE zu flashen](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Achten Sie darauf, ob das heruntergeladene Systemabbild passt, da es sonst nicht starten kann
:::
