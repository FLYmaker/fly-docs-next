# Eingebaute 5V Ventilatorbuchse

## Das C8-Board hat eine 5V steuerbare Schnittstelle für die Hauptkühlung, die Verkabelung ist wie folgt:

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/corefan.png').default} size="80%" align="center" />

## Referenzkonfiguration

    ```cfg
    [mcu host]                      # FlyOS hat für Sie standardmäßig einen Linux-MCU erstellt, definieren Sie ihn einfach           
    serial: /tmp/klipper_host_mcu 

    [temperature_sensor FLY-π]      # Temperatur des Host-Systems
    sensor_type: temperature_host


    [temperature_fan core_fan]      # Hauptventilator zur Kühlung des Host-Systems
    pin: host:gpiochip1/gpio103     # GPIO-Pin für den Host-Ventilator
    max_power: 1.0
    sensor_type: temperature_host   # Temperatur des Host-Systems
    control:watermark               # Kontrollmethode
    target_temp: 48                 # Temperatur, ab der der Hauptventilator startet
    min_temp: 0                     # Mindesttemperatur, darunter wird ein Fehler gemeldet
    max_temp: 90                    # Maximale Temperatur, darüber wird ein Fehler gemeldet
    off_below: 0.10
    kick_start_time: 0.50
    max_speed: 0.8                  # Maximalgeschwindigkeit, 80% der vollen Leistung
    min_speed: 0.3                  # Minimalgeschwindigkeit, 30% der vollen Leistung
    ```
