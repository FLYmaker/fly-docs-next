# Anleitung zum Firmware-Flashen

    
## Wie erkenne ich, welche Firmware ich benötige

    :::tip

    * Bitte beachte, dass du nur eine der Firmwares flashen musst.

    :::
    
    * C8 kann auf folgende Weise mit der Host-Anwendung verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Bei mehreren USB-Geräten kann die Datenübertragung beeinträchtigt werden
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen ohne USB-Port zu blockieren
        * Nachteile: Kann bei gleichzeitiger Nutzung mit `KlipperScreen` zu Fehlern wie Homing-Timeout führen.
