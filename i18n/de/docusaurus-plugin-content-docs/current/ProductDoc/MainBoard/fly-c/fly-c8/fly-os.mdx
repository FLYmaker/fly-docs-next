## Vorbereitung für das Systemflashing

    * Kartenleser
    * Eine TF-Karte, Kapazität muss mindestens 16GB und höchstens 128GB betragen, TF-Karte muss eine Übertragungsgeschwindigkeit von Class10 oder höher haben
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">Systemabbild</Button>
    * Laden Sie das benötigte <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Flashing-Tool</Button>

## Systemflashing

:::info
Das Flashing auf eine TF-Karte und das Flashing über eine TF-Karte auf ein M2WE erfordern nur eine der beiden Methoden.
:::

:::info
* Wenn das Gerät mit einem M2WE und einer Speicherkarte ausgestattet ist, startet das System standardmäßig von der Speicherkarte
* Es wird empfohlen, das System auf eine TF-Karte zu flashen und über USB auf ein M2WE zu flashen, da dies möglicherweise aufgrund von Treiberkompatibilitätsproblemen fehlschlagen kann. Es wird empfohlen, direkt [die Speicherkarte auf ein M2WE zu flashen](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Achten Sie darauf, ob das heruntergeladene Systemabbild übereinstimmt, da es sonst nicht startet.
:::
