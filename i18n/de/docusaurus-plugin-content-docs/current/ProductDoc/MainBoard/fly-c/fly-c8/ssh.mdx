# Verbindung zur Oberflächenkontrollstation über SSH

* Sie können eine serielle Schnittstelle oder WIFI verwenden, um eine Verbindung zum Gerät herzustellen
* Bitte bereiten Sie im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminaltool** vor
* Stellen Sie sicher, dass auf der SD-Karte oder dem M2WE das **entsprechende System für die Oberflächenkontrollstation** gebrannt ist
* **Falls die Oberflächenkontrollstation nicht ordnungsgemäß startet, entfernen Sie Treiber, Endschalter, Lüfter und andere Peripheriegeräte!!!**
* **Stellen Sie abschließend eine Netzwerkverbindung zu SSH her, um das Firmware-Update zu erleichtern!!!**
* **Für die Firmware-Kompilierung lesen Sie bitte den Leitfaden zur Verbindung der Oberflächenkontrollstation über WIFI oder Netzwerkkabel**

        ## Verwendung einer Netzwerk-IP-Verbindung zur Oberflächenkontrollstation über SSH

        * Melden Sie sich im **Backend des Routers** an, zu dem die Oberflächenkontrollstation verbunden ist
        * Stellen Sie sicher, dass das System **ordnungsgemäß gestartet** ist
        * Stellen Sie sicher, dass **Sie den Benutzernamen und das Passwort der Oberflächenkontrollstation kennen**. Standardmäßig ist der Benutzername **fly** und das Passwort **mellow**
        * Wenn die Hauptplatine gestartet ist und mit dem Netzwerk verbunden ist, können Sie die SSH-Verbindung über die IP-Adresse herstellen
        * Öffnen Sie MobaXterm und nehmen Sie die folgenden Einstellungen vor, füllen Sie die IP-Adresse aus, die Sie vorher überprüft haben, oder schauen Sie im Router-Backend nach

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Hinweis: Hier geben Sie den Benutzernamen der Oberflächenkontrollstation ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch erscheint dieses Fenster, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Es wird zur Eingabe des Passworts aufgefordert, geben Sie das Passwort der Oberflächenkontrollstation ein und drücken Sie die Eingabetaste

        **Hinweis: Das Passwort wird hier nicht angezeigt, geben Sie es ein und drücken Sie die Eingabetaste**

        Standardmäßig ist der Benutzername `fly` und das Passwort `mellow`
