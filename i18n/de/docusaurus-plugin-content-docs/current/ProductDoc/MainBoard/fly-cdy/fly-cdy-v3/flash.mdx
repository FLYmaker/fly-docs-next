# Firmware-Kompilierung und -Flash

    :::tip

    * CDV verwendet standardmäßig eine TF-Karte zum Flashen des Firmware
    * Bereiten Sie eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * No-Name-Speicherkarten können das Flashen fehlschlagen lassen, verwenden Sie bitte möglichst die unten empfohlenen Speicherkarten zum Aktualisieren der Firmware
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte idealerweise `class10` oder höher sein, empfehlenswert sind Marken wie SanDisk

    :::
