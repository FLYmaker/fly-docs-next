## Produktspezifikationen

    * 32-Bit ARM Cortex-M4 Serie 168 MHz, STM32F407ZGT6 Chip
    * Unterstützt CAN-Bus-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * Firmware: Reprap/Klipper
    * Treiberunterstützung: A4988, LV8729, DRV8225, TMC2208/2209/5160 usw.
    * Treibermodus-Unterstützung: TMC: UART/SPI
    * Anzeige: serieller Touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Bei Verwendung der repap-Firmware kann ein erweiterbares ESP32 WiFi-Modul verwendet werden, unterstützt 1.6m/s
    * Einzigartiges PWM-Fan-MOS-Board-Design, kann bei Beschädigung direkt ersetzt werden, um die Wartungskosten zu senken
    * PCB mit 2OZ Kupferdicke, unterstützt höhere Ströme
    * 3 ADC-Schnittstellen, unterstützt bis zu drei Thermisten
    * 6 IO-Schnittstellen, können für Nivellierung und Endschalter verwendet werden
    * 4 Heizstab-Schnittstellen, unterstützt bis zu vier Heizstäbe
