# Firmware-Brennhinweise

:::danger Zu beachten
* Die Hauptplatine-Firmware muss durch Katapult mit Klipper gebrannt werden
* Wenn Sie sicher sind, dass die Hauptplatine bereits mit Katapult gebrannt wurde, ist das Brennen der Katapult-Firmware am Ende nicht erforderlich, dies dient nur als Vorsichtsmaßnahme, falls das Brennen der Katapult-Firmware auf der Hauptplatine nicht erfolgreich war
* Sie können schnell zweimal auf Reset drücken und überprüfen, ob eine LED auf der Hauptplatine aufleuchtet. Wenn ja, bedeutet dies, dass die Hauptplatine in den Katapult-Firmware-Modus gegangen ist.
:::

## Wie Sie feststellen, welche Firmware Sie benötigen

    :::tip

    * Bitte beachten Sie, dass Sie nur eine der Firmwares flashen müssen

    :::
    
    * D5 kann auf folgende Weise mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Wenn mehrere USB-Geräte angeschlossen sind, könnte dies die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Brücken-Firmware</Button>
        * Vorteile: Kann mehrere Geräte verbinden, ohne USB-Anschlüsse zu blockieren
        * Nachteile: Kann zusammen mit `KlipperScreen` verwendet werden, was zu Fehler wie Homing-Timeout führen kann.
