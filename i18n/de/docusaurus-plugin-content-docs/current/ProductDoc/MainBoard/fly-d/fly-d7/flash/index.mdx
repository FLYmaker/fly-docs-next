# Anleitung zur Firmware-Beschreibung

:::danger Wichtig
* Die Hauptplatine-Firmware muss über Katapult geflasht werden
* Wenn Sie sicher sind, dass die Hauptplatine bereits mit Katapult geflasht ist, ist das Flashen der Katapult-Firmware am Ende nicht erforderlich, dies dient nur zur Sicherheit, falls das Flashen der Katapult-Firmware auf der Hauptplatine fehlschlägt
* Sie können zweimal schnell nacheinander zurücksetzen und überprüfen, ob eine LED auf der Hauptplatine blinkt, falls ja, ist die Hauptplatine in den Katapult-Modus gegangen
:::

## Wie man feststellt, welche Firmware benötigt wird

    :::tip

    * Beachten Sie, dass Sie nur eine der Firmwares flashen müssen

    :::
    
    * D7 kann auf folgende Weise mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteil: Zuverlässige und stabile Kommunikation
        * Nachteil: Wenn mehrere USB-Geräte angeschlossen sind, könnte dies die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteil: Kann mehrere Geräte anschließen und nimmt keinen USB-Anschluss in Anspruch
        * Nachteil: Kann durch die gleichzeitige Nutzung von `KlipperScreen` zu Fehlern wie Timeout bei der Homing führen.
