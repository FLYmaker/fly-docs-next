# Anleitung zum Einspielen der Firmware

:::danger Wichtig
* Die Firmware des Mainboards muss über Katapult mit Klipper eingespielt werden
* Wenn das Mainboard bereits mit Katapult versehen ist, ist das Einspielen der Katapult-Firmware am Ende nicht erforderlich, das ist nur eine Vorsichtsmaßnahme, falls die Katapult-Firmware nicht erfolgreich eingespielt wurde
* Sie können zweimal schnell hintereinander auf Reset drücken und überprüfen, ob eine LED auf dem Mainboard aufleuchtet, was bedeutet, dass das Mainboard in die Katapult-Firmware eingetreten ist
:::

## Wie man feststellt, welche Firmware benötigt wird

    :::tip

    * Beachten Sie bitte, dass Sie nur eine der Firmwares einspielen müssen

    :::
    
    * D8 kann auf folgende Weise mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Mehrere USB-Geräte können die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteile: Kann mehrere Geräte verbinden und belegt keinen USB-Anschluss
        * Nachteile: Kann bei gemeinsamer Nutzung mit `KlipperScreen` zu Fehlern wie Homing-Timeouts führen.
