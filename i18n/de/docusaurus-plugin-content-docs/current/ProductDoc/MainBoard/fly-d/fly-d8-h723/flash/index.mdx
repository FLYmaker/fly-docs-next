# Anleitung zur Firmware-Flashung

:::danger Wichtig
* Die Firmware auf dem Hauptmodul muss über Katapult geflasht werden
* Wenn sicher ist, dass die Katapult-Firmware bereits auf dem Hauptmodul installiert ist, ist das Flashen der Katapult-Firmware am Ende nicht erforderlich; es dient nur als Vorsichtsmaßnahme, falls die Flashung der Katapult-Firmware nicht erfolgreich war
* Nach dem schnellen Drücken des Reset-Knopfes zweimal, prüfen, ob eine LED auf dem Hauptmodul blinkt; wenn ja, bedeutet dies, dass das Hauptmodul in den Katapult-Modus gewechselt hat
:::

## Wie ermittelt man, welche Firmware benötigt wird

    :::tip

    * Beachten Sie, dass Sie nur eine der Firmwares flashen müssen

    :::
    
    * D8 kann auf folgende Weise mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteil: Zuverlässige und stabile Kommunikation
        * Nachteil: Wenn mehrere USB-Geräte angeschlossen sind, könnte dies die Datenübertragung beeinflussen
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteil: Kann mehrere Geräte verbinden und nimmt keine USB-Anschlüsse in Anspruch
        * Nachteil: Kann bei gemeinsamer Nutzung mit `KlipperScreen` zu Fehlermeldungen wie "Homing Timeout" führen.
