# Anleitung zur Firmware-Blutung

:::danger Wichtige Hinweise
* Die Firmware des Hauptboards muss über Katapult aufgespielt werden
* Wenn sicher ist, dass das Hauptboard bereits mit Katapult geflasht wurde, ist der letzte Schritt zum Flashen der Katapult-Firmware nicht erforderlich, dies dient nur als Vorsichtsmaßnahme, falls das Flashen der Katapult-Firmware auf dem Hauptboard nicht erfolgreich war
* Man kann zweimal schnell die Reset-Taste drücken und dann überprüfen, ob eine LED auf dem Hauptboard aufleuchtet, was bedeutet, dass das Hauptboard in die Katapult-Firmware eingetreten ist
:::

## Wie man ermittelt, welche Firmware benötigt wird

    :::tip

    * Beachten Sie, dass Sie nur eine der Firmwares aufspielen müssen

    :::
    
    * DP5 kann auf folgende Weise mit dem Host-Computer verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteil: Zuverlässige Kommunikation, stabil
        * Nachteil: Bei mehreren USB-Geräten könnte die Datenübertragung beeinträchtigt werden
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteil: Mehrere Geräte können angeschlossen werden, ohne USB-Ports zu belegen
        * Nachteil: Möglicherweise treten bei der gemeinsamen Nutzung mit `KlipperScreen` Fehler wie Zeitüberschreitungen beim Homing auf.
