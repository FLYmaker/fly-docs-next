# Firmware-Kompilierung und -Einspeisung

    :::tip

    * E3 Pro verwendet standardmäßig eine TF-Karte für die Firmware-Einspeisung
    * Bitte bereiten Sie eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * No-Name-Speicherkarten können zu Fehlern beim Einspeisen führen. Verwenden Sie bitte nach Möglichkeit die unten empfohlenen Speicherkarten für das Firmware-Update
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte idealerweise `class10` oder höher sein. Es wird empfohlen, Marken wie SanDisk zu verwenden

    :::
