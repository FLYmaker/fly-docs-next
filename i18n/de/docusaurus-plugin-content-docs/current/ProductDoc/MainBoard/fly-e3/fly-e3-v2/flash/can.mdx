## Kompilierung starten

    <SSH />

    <CAN />

    * Drücke die `Q`-Taste, es erscheint **Konfiguration speichern**, drücke dann die `Y`-Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und die Rückkehr zur Kommandozeile erfolgt
    * Gib `make -j4` ein, um die Kompilierung zu starten, das dauert etwas länger
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Am Ende erscheint die folgende Ausgabe, was auf eine erfolgreiche Kompilierung hinweist

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
