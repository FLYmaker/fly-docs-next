# Firmware-Beschreibung


## Wie man ermittelt, welche Firmware benötigt wird

    :::tip

    * Bitte beachten Sie, dass Sie nur eine der Firmware-Versionen flashen müssen

    :::
    
    * E3 V2 kann mit der folgenden Methode mit der Host-Maschine verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Mehrere USB-Geräte könnten die Datenübertragung beeinflussen
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen und belegt keine USB-Anschlüsse
        * Nachteile: Kann bei gleichzeitiger Nutzung von `KlipperScreen` zu Home-Timeout-Fehlern führen.
