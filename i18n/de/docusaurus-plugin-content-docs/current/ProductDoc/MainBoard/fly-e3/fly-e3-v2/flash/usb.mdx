# USB-Firmware-Compilation

    :::tip

    * E3 V2 wird werkseitig mit einem TF-Karten-Bootloader ausgeliefert, ein erneutes Flashen ist nicht erforderlich.

    :::

## Kompilierung starten

    <SSH />

    <MakeOperate/>

    <USB />

    * Drücke die `Q` -Taste, es erscheint **Save configuration**, dann drücke die `Y` -Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und die Kommandozeile geschlossen sein
    * Gib `make -j4` ein, um die Kompilierung zu starten, dies dauert etwas länger
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Am Ende wird der folgende Text ausgegeben, wenn die Kompilierung erfolgreich war

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
