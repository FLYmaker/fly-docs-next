# Firmware-Compilation und -Flashing

    :::tip

    * 407ZG verwendet standardmäßig eine TF-Karte für das Firmware-Flashing
    * Bitte bereiten Sie eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * Unbekannte Marken-Speicherkarten können zu Flash-Fehlschlägen führen, verwenden Sie bitte nach Möglichkeit die empfohlenen Speicherkarten zur Aktualisierung der Firmware
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte idealerweise `class10` oder höher sein, es wird empfohlen, Marken wie SanDisk für die TF-Karte zu verwenden

    :::

    * Wenn die folgenden Zeilen ausgegeben werden, war die Kompilation erfolgreich

        ```bash
        Creating hex file out/klipper.bin
        ```
