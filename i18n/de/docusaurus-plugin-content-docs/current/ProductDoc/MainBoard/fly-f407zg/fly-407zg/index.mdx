## Produktparameter

    * 32-Bit ARM Cortex-M4 Serie 168 MHz, STM32F407ZGT6 Chip
    * Unterstützung der CAN-Bus-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * Firmware: Reprap/Klipper
    * Treiberunterstützung: A4988, LV8729, DRV8225, TMC2208/2209/5160 usw.
    * Treibermodusunterstützung: TMC: UART/SPI
    * Display: Serielles Touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Mit der Repap-Firmware kann das erweiterte ESP32 WiFi-Modul verwendet werden, unterstützt 1.6m/s
    * Einzigartiges PWM Fan MOS-Board-Design, bei Beschädigung direkt austauschbar, verringert die Wartungsanforderungen für Kunden
    * PCB verwendet 2OZ Kupferdicke, kann größere Ströme unterstützen
    * 6 ADC-Schnittstellen, unterstützt bis zu sechs Thermistoren
    * 7 IO-Schnittstellen, können für Nivellierung und Endschalter verwendet werden
    * Sieben Heizstab-Schnittstellen, unterstützen bis zu sieben Heizstäbe
