# BL-Firmware

    :::tip

    * Beachten Sie, dass der `Bootloader` bereits im Werk programmiert ist und nicht erneut programmiert werden muss.

    :::

## In den Programmiermodus wechseln


    * Es gibt zwei Methoden, um in den Programmiermodus zu wechseln:
    * Erste Methode: Bei vollständig ausgeschaltetem Hauptbrett, die Pins BOOT und BOOT1 kurzschließen, dann das Datenkabel an das Hauptbrett und das Gerät anschließen.
    * Zweite Methode: Nachdem das Datenkabel an das Hauptbrett und das Gerät angeschlossen wurde, die Pins BOOT und BOOT1 kurzschließen, dann die RST-Taste drücken, RST-Taste loslassen und schließlich die BOOT-Kurzschlussbrücke entfernen.

            4. Firmware programmieren

            :::tip

            Es muss nur eine Firmware programmiert werden.

            :::


            * Hier ist der Befehl zum Programmieren der Bootloader-Firmware:
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/Gemini/GeminiV3.bin
                ```

            5. Wenn der Text wie in der Abbildung gezeigt unter dem Pfeil ``File downloaded successfully`` erscheint, war das Programmieren erfolgreich.

            :::warning

            Bitte ignorieren Sie den **Fehler** nach der roten Pfeilmarkierung in dieser Zeile!!!
