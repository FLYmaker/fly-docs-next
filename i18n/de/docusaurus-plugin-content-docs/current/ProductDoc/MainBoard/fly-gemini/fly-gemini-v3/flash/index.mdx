# Firmware-Brenn-Anleitung

## Wie man feststellt, welche Art von Firmware benötigt wird

    :::tip

    * Beachten Sie bitte, dass Sie nur eine der Firmwares flashen müssen.

    :::
    
    * Gemini V3 kann auf folgende Weise mit dem Host-PC verbunden werden:
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Kann bei mehreren USB-Geräten die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Brücken-Firmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen, ohne USB-Ports zu belegen
        * Nachteile: Kann bei der gleichzeitigen Nutzung mit `KlipperScreen` zu Home-Timeout-Fehlern führen.
