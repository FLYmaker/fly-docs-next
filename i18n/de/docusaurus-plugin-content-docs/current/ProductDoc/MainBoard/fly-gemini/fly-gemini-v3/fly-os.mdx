## Vorbereitung vor dem Systemflashing

    * Kartenzähler
    * Eine TF-Karte, die Kapazität muss mindestens 16 GB und maximal 128 GB betragen, TF-Karte, TF-Karte, die Übertragungsgeschwindigkeit muss mindestens der Klasse 10 entsprechen
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">System-Image</Button> herunter
    * Laden Sie die benötigten <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Flashing-Tools</Button> herunter

## Systemflashing

:::info
Das Flashing auf die TF-Karte und das Flashing auf die M2WE über die TF-Karte erfordert nur eine der beiden Methoden
:::

:::info
* Wenn das Gerät mit M2WE und Speicherkarte ausgestattet ist, startet das System standardmäßig von der Speicherkarte
* Es wird empfohlen, das System über eine TF-Karte auf die M2WE zu flashen. Das Flashing über USB auf die M2W kann aufgrund von Treiberkompatibilitätsproblemen fehlschlagen. Es wird empfohlen, direkt [von der Speicherkarte auf die M2WE zu flashen](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Achten Sie darauf, dass das heruntergeladene System-Image korrekt ist, da es andernfalls nicht startet
:::
