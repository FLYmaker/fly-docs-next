# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.png').default} size="80%" align="center" />

## Produkteigenschaften
    * Integrierte Host-Maschine, kann als Ersatz für den Raspberry Pi als Host-Maschine laufen, mit 1GB eingebautem DDR3-Speicher.
    * Verbesserte 5V-Stromversorgungsschutz, reduziert die Wahrscheinlichkeit von Chipschäden.
    * Entfernung der Speicherkarten-Einstellung für den MCU-Chip, erleichtert das Flashen der Firmware.
    * Vorinstalliertes System mit klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, Resonanzkompensationsplugin numpy und FLY-Tools, keine komplizierten Quellwechsel erforderlich, einfacher Einstieg für Anfänger (Benötigt Kauf einer TF-Karte oder M2WE)

## Produktparameter

    * CPU: Hochleistungs-Chip Allwinner H5, 4-Kern-64-Bit-Cortex-A53
    * GPU: Hochleistungs-6-Kern-Mali 450, Pixel-Füllrate über 2,7 Gpixel/s
    * RAM: 1GB DDR3 (gemeinsam genutzt mit GPU)
    * ROM: Unterstützt maximal 128GB SDCARD
    * Peripheriegeräte: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1
    * Schnittstellen: Integrierte M.2-Schnittstelle, erweiterbar für WIFI, eMMC usw. (Proprietäres Protokoll, verbinden Sie keine nicht für FLY π spezifischen M.2-Geräte)
    * Unterstützt 12-24V Gleichstromversorgung, stabilere Stromversorgung, spart den Spannungswandler
    * Vorbehaltener Anschluss für einen steuerbaren 5V-Lüfter, kann für die Kühlung der Host-Maschine verwendet werden
    * Vorbehaltener CAN-Erweiterungsanschluss
    * Vorinstalliertes System mit klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, Resonanzkompensationsplugin numpy und FLY-Tools, keine komplizierten Quellwechsel erforderlich, einfacher Einstieg für Anfänger (Benötigt Kauf einer TF-Karte oder M2WE)
    * Treiber: Hochspannungstreiber *4
    * MCU: STM32F405
