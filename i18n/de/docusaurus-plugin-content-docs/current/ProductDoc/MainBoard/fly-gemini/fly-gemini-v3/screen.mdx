# Bildschirmanschluss und -konfiguration

:::info

* FLY-TOOLS benötigt eine Verbindung mit der oberen Software, um den IP-Adresszugang `IP+:9999` einzugeben
* Zum Beispiel `192.168.6.110:9999`

:::
