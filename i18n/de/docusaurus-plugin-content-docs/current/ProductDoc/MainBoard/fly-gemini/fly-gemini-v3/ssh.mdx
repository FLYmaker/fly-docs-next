# Verbindung zur Host-Maschine über SSH

* Sie können über eine Seriellverbindung oder WIFI zur Geräte verbinden
* Bereiten Sie bitte im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminaltool** vor
* Stellen Sie sicher, dass auf der SD-Karte oder M2WE das **entsprechende Betriebssystem für die Host-Maschine** installiert ist
* **Falls die Host-Maschine nicht richtig startet, entfernen Sie unbedingt Antrieb, Endschalter, Lüfter usw.!**
* **Verbinden Sie sich zum Schluss über das Netzwerk mit SSH, um das Firmware zu kompilieren!**
* **Für die Firmware-Kompilierung bitte das Tutorial zur Verbindung der Host-Maschine über WIFI oder Ethernetkabel beachten**

        ## Verbindung zur Host-Maschine über Netzwerk-IP über SSH

        * Melden Sie sich im **Router-Backend** an, an das die Host-Maschine angeschlossen ist
        * Stellen Sie sicher, dass das System **korrekt gestartet** ist
        * Stellen Sie sicher, dass Sie **Benutzername und Passwort der Host-Maschine** kennen; das Standardkonto des FLY-Systems ist **fly**, das Standardpasswort ist **mellow**
        * Wenn das Hauptbrett gestartet ist und mit dem Netzwerk verbunden ist, können Sie über die IP-Adresse eine SSH-Verbindung herstellen
        * Öffnen Sie MobaXterm und führen Sie die folgenden Einstellungen durch, geben Sie die zuvor überprüfte IP-Adresse ein oder überprüfen Sie sie im Router-Backend

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Achtung: Geben Sie hier im 4. Schritt den Benutzernamen der Host-Maschine ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch erscheint dieses Fenster, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Es wird nach einem Passwort gefragt, geben Sie das Passwort der Host-Maschine ein und drücken Sie Enter

        **Achtung: Das eingegebene Passwort ist nicht sichtbar, geben Sie es ein und drücken Sie direkt Enter**

        Das Standardkonto des FLY-Systems ist: `fly`, das Standardpasswort ist: `mellow`
