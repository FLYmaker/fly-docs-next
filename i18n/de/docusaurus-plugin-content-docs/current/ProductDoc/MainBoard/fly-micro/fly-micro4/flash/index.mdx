# Anleitung zur Firmware-Installation

:::danger Wichtig
* Die Hauptplatinen-Firmware muss über Katapult Klipper installiert werden
* Wenn sichergestellt ist, dass die Hauptplatine bereits Katapult hat, ist das Installieren der Katapult-Firmware am Ende nicht erforderlich, das dient nur zur Sicherheit, falls die Katapult-Firmware der Hauptplatine nicht erfolgreich installiert wurde
* Man kann zweimal schnell auf Reset drücken und schauen, ob eine LED auf der Hauptplatine blinkt, wenn ja, dann ist die Hauptplatine in der Katapult-Firmware
:::

## Wie man feststellt, welche Art von Firmware benötigt wird

    :::tip

    * Bitte beachten Sie, dass Sie nur eine der Firmware-Optionen installieren müssen

    :::
    
    * E3 V2 kann auf folgende Weise mit dem Host-Computer verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Kann bei mehreren USB-Geräten die Datenübertragung beeinflussen
    * <Button variant="contained" disableElevation href="./can">USB-Brücken-Firmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen und nimmt keinen USB-Anschluss in Anspruch
        * Nachteile: Kann bei Verwendung mit `KlipperScreen` zu Fehler wie Homing-Timeout führen.
