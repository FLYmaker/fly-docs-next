# USB-Firmware-Kompilierung

    :::tip

    * Micro4 aus dem Werk hat bereits das Katapult-Firmware für USB-Verbindungen vorinstalliert, es ist nicht nötig, sie erneut zu installieren

    :::

## Kompilierung beginnen

    <SSH />

    <MakeOperate/>

    <USB />

    * Wählen Sie die GPIO-Pins aus, die beim Start des Mikrocontrollers gesetzt werden sollen, geben Sie ein: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/usb.png').default} size="100%" align="center" />
    * Drücken Sie die `Q`-Taste, es erscheint **Save configuration**, drücken Sie dann die `Y`-Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Nun sollte die Konfiguration gespeichert und die Befehlszeile wiederhergestellt sein
    * Geben Sie `make -j4` ein, um mit der Kompilierung zu beginnen, dies kann etwas Zeit in Anspruch nehmen
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Am Ende sollte der folgende Text angezeigt werden, um den Erfolg der Kompilierung zu bestätigen

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
