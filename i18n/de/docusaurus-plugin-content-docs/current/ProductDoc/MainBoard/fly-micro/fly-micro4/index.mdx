## Produkteigenschaften

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Unterstützt CAN-Brücke
    * Onboard 4x 2209
    * Onboard DIAG-Funktion Anschluss, einfach durch Ein-/Ausstecken eines Jumper-Huts kann die Funktion der unendlichen Homing-Unterstützung verwendet werden

## Produktspezifikationen

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Unterstützt 12-24V DC-Eingang, stabilere Stromversorgung, ohne Spannungsregler-Modul
    * Treibermodus: UART, STEP/DIR
    * Erweiterungsschnittstellen: BLTouch (Servos, Probe), Näherungsschalter X1
    * Lüfter: Zwei kontrollierbare Zweiadrige Lüfter, ein permanenter Lüfter
    * Temperatursensoren: 2x NTC 100K
    * Heizschnittstellen: 2x kontrollierbare Schnittstellen, 10A unterstützt
