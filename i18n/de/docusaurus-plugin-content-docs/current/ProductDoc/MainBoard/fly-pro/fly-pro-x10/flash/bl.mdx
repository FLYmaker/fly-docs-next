# BL-Firmware

    :::tip

    * Beachten Sie bitte, dass das `Bootloader` bereits im Werk vorausgeflasht ist und nicht erneut gebrannt werden muss.

    :::

## Betreten des Brennmodus


    * Es gibt zwei Methoden, um den Brennmodus zu betreten
    * Erste Methode: Bei vollständig ausgeschalteter Hauptplatine, kurzschließen Sie BOOT und verbinden Sie dann das Datenkabel mit der Hauptplatine und Ihrem Gerät.
    * Zweite Methode: Nachdem Sie das Datenkabel mit der Hauptplatine und Ihrem Gerät verbunden haben, kurzschließen Sie BOOT, drücken Sie die RST-Taste, lassen Sie die RST-Taste los und trennen Sie die BOOT-Kurzschlussbrücke.

            4. Brennen der Firmware

            :::tip

            Es muss nur eine der Firmwares gebrannt werden

            * Untenstehend ist der Befehl zum Brennen der Katapult-Firmware
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * Untenstehend ist der Befehl zum Brennen der Bootloader-Firmware
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. Wenn der Text "File downloaded successfully" erscheint, wie im Bild durch den Pfeil markiert, war das Brennen erfolgreich
