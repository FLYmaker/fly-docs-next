# Hinweise zur Hauptplatine

## Stromversorgungsanzeige
    * Beim Betrieb mit Type-C-Stromversorgung ohne Anschluss anderer Hauptplattenstromquellen sollten `SYS_5V` und SYS_3.3V leuchten
    * Nach dem Anschließen der Hauptplattenstromquellen `HVIN`, `VIN`, `BOARO`, `POWER` sollte die Stromversorgungsanzeige der Hauptplatine dauerhaft leuchten

## Treiber

    * TP5160 und TP2209 unterscheiden sich nur im Namen und benötigen keinen Jumper, um den Treibermodus auszuwählen
    * An den Treiberanschlüssen können nur TP-Treiber installiert werden, keine anderen Geräte

## Begrenzungsanschluss
    * ProX10 unterstützt die Auswahl zwischen 5V und 12V Stromversorgung, bitte beachten Sie, dass bei Verwendung eines Näherungsschalters eine Diode zur Spannungsreduktion installiert werden muss

## Oberes Gerät
    * ProX10 unterstützt die Verwendung einer externen oberen Maschine, nach Auswahl der Verbindung über Type-C durch Schalterstellung kann sie verwendet werden

## Firmware-Programmierung

    * Die Hauptplatine unterstützt das Programmieren über eine TF-Karte, standardmäßig jedoch ist `Katapult` vorinstalliert
    * Für das Programmieren über eine TF-Karte muss das BL neu programmiert werden
