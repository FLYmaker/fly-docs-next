    :::tip

    * Super5 Pro verwendet standardmäßig eine TF-Karte zur Firmware-Installation
    * Bitte bereiten Sie eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * Unbekannte Marken von Speicherkarten können zu Installationsfehlern führen. Verwenden Sie bitte nach Möglichkeit die unten empfohlenen Speicherkarten zur Firmware-Aktualisierung
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte mindestens `class10` oder höher sein, es wird empfohlen, TF-Karten von Marken wie SanDisk zu verwenden

    :::

# Anleitung zur Firmware-Installation

    :::tip

    * Das Super5 Pro wird werksseitig mit einem TF-Karten-Bootloader ausgeliefert, es ist keine erneute Installation erforderlich

    :::


## Wie man feststellt, welche Firmware benötigt wird

    :::tip

    * Beachten Sie, dass Sie nur eine der Firmwares installieren müssen

    :::
    
    * Super5 Pro kann auf folgende Weise mit dem Host-Computer verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Mehrere USB-Geräte können die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Brückenfirmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen und blockiert keinen USB-Anschluss
        * Nachteile: Kann bei gleichzeitiger Nutzung mit `KlipperScreen` zu Fehlern wie Timeouts beim Rücksetzen führen.
