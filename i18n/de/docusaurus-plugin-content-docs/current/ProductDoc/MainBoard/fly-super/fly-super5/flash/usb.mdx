    <SSH />

    <USB />

    * Drücke die `Q`-Taste, es erscheint **Konfiguration speichern**, dann drücke die `Y`-Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und die Konsole wieder angezeigt werden
    * Gib `make -j4` ein, um mit der Kompilierung zu beginnen, dies kann etwas dauern
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn die Kompilierung erfolgreich abgeschlossen ist, wird der folgende Inhalt ausgegeben:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
