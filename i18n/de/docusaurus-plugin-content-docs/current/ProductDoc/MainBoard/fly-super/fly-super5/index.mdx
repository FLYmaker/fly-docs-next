# Produktparameter

    * Verwendet einen 32-Bit ARM Cortex-M7 Serien STM32H723ZGT6 Mikrocontroller mit 550 MHz Taktfrequenz
    * Firmware: Marlin 2.0/Reprap/Klipper
    * Treiberunterstützung: A4988、LV8729、DRV8225、TMC2208/2209/5160 usw.
    * Treibermodus Unterstützung: TMC: UART/SPI
    * 5 Hochspannungstriebplätze, unterstützt 5 unabhängige Motorantriebe
    * Anzeige: Serieller Touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 Hochspannungs-Eingang, unterstützt automatische Bettfühler: BLTouch、PL08N und andere Näherungssensoren, Klicky, Voron Tap usw.
    * Wahlweise Endschalterversorgung: 5V/12V/24V
    * Mit der repap-Firmware kann das integrierte ESP32 WiFi-Modul verwendet werden, unterstützt 1.6m/s
    * Einzigartiges PWM Fan MOS-Board-Design, bei Beschädigung direkt austauschbar, reduziert die Wartungsschwierigkeiten für Kunden, zwei 2pin Lüfteranschlüsse können 4pin PWM-Lüfter unterstützen
    * PCB mit 2OZ Kupferstärke, kann höhere Ströme unterstützen
    * 3 ADC-Schnittstellen, unterstützen bis zu drei Thermistoren
    * 7 IO-Schnittstellen, können für Nivellierung und Endschalter verwendet werden
    * 3 Heizstabanschlüsse, unterstützen bis zu drei Heizstäbe
