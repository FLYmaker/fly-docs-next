# BL-Firmware

    :::tip

    * Beachten Sie, dass der `Bootloader` bereits werkseitig installiert ist und nicht erneut geflasht werden muss.

    :::
## Betriebsmodus für das Flashen

    * Es gibt zwei Methoden, um den Betriebsmodus für das Flashen zu betreten
    * Erste Methode: Trennen Sie die Hauptplatine vollständig von der Stromversorgung, verbinden Sie BOOT kurz und verbinden Sie dann das Datenkabel mit der Hauptplatine und Ihrem Gerät.
    * Zweite Methode: Verbinden Sie das Datenkabel mit der Hauptplatine und Ihrem Gerät, verbinden Sie BOOT kurz, drücken Sie die RST-Taste und lassen Sie die RST-Taste los, bevor Sie die BOOT-Verbindung trennen.
