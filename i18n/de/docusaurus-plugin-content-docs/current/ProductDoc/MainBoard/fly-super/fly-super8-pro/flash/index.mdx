    :::tip

    * Super8 PRO verwendet standardmäßig eine TF-Karte zur Firmware-Installation
    * Bereiten Sie bitte eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * Billige Speicherkarten können zu einem Fehler bei der Installation führen. Verwenden Sie bitte nach Möglichkeit die unten empfohlenen Speicherkarten für die Firmware-Aktualisierung
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte idealerweise `class10` oder höher sein, empfohlen werden Marken wie SanDisk

    :::

# Anleitung zur Firmware-Installation

    :::tip

    * Der Super8 Pro wird mit einem vorinstallierten TF-Karten-Bootloader ausgeliefert, eine erneute Installation ist nicht erforderlich

    :::

## Wie man feststellt, welche Firmware benötigt wird

    :::tip

    * Beachten Sie, dass Sie nur eine der Firmwares installieren müssen

    :::
    
    * Super8 PRO kann auf folgende Weise mit dem Host verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Bei mehreren USB-Geräten könnte die Datenübertragung beeinträchtigt werden
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteile: Mehrere Geräte können angeschlossen werden, ohne USB-Anschlüsse zu belegen
        * Nachteile: Durch die gemeinsame Nutzung mit `KlipperScreen` könnten Probleme wie Timeout bei der Homing-Funktion auftreten.
