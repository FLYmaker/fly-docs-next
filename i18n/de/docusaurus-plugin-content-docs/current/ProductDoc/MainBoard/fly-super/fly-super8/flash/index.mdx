    :::tip

    * Super8 verwendet standardmäßig eine TF-Karte für das Firmware-Brennen
    * Bitte bereiten Sie eine TF-Karte mit einer Kapazität von nicht mehr als `32G` vor
    * Markenlose Speicherkarten könnten zu Brennfehlern führen, verwenden Sie bitte nach Möglichkeit die unten empfohlenen Speicherkarten zum Aktualisieren der Firmware
    * Die Übertragungsgeschwindigkeit der TF-Karte sollte idealerweise `class10` oder höher sein, es wird empfohlen, Marken wie SanDisk zu verwenden

    :::

# Firmware-Brenn-Anleitung


## Wie man feststellt, welche Art von Firmware benötigt wird

    :::tip

    * Bitte beachten Sie, dass Sie nur eine Art von Firmware flashen müssen

    :::
    
    * Super8 kann auf folgende Weise mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./usb">USB-Firmware</Button>
        * Vorteile: Zuverlässige Kommunikation, stabil
        * Nachteile: Mehrere USB-Geräte könnten die Datenübertragung beeinträchtigen
    * <Button variant="contained" disableElevation href="./can">USB-Bridge-Firmware</Button>
        * Vorteile: Kann mehrere Geräte anschließen und belegt keine USB-Anschlüsse
        * Nachteile: Kann durch die gleichzeitige Nutzung mit `KlipperScreen` zu Fehler wie Heimzeitenüberschreitung führen.
