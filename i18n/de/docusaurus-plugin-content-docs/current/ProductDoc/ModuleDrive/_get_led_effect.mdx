# Erweitern Sie die Steuerung der LED über Plugins

Sie müssen sich über SSH mit dem Host-PC verbinden und die Installation und Konfiguration nach dem folgenden Link durchführen:

```
https://github.com/julianschill/klipper-led_effect
```

## Installation

Bitte verbinden Sie sich mit **MobaXterm_Personal** oder anderen **SSH-Tools** über **WIFI** mit Ihrem Host-PC und stellen Sie sicher, dass die folgenden Punkte erfüllt sind:

1. **Stellen Sie sicher, dass die Klipper-Firmware auf dem Host-PC installiert ist**
2. **Stellen Sie sicher, dass Sie sich als der Benutzer anmelden, der Klipper installiert hat**
3. **Stellen Sie sicher, dass Ihre Eingabemethode auf Englisch eingestellt ist**
4. **Stellen Sie sicher, dass Ihr Host-PC die Geräte normal suchen kann**
5. **Stellen Sie sicher, dass alle oben genannten Punkte beachtet werden, andernfalls kann der nächste Schritt nicht durchgeführt werden**

Dieses Modul kann mit einem Installationsskript in eine bestehende Klipper-Installation installiert werden.

```
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Referenzkonfiguration

```
[neopixel TP_led]
pin: PB7
chain_count: 25
# Anzahl der LEDs
# Anzahl der LED-Perlen
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
