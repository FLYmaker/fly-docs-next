# TMC 5160 Plus

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/5160plus.png').default} size="100%" align="center" />

## Grundlegende Parameter

    * Eingangsspannung: 12-60V
    * Maximalstrom: 10A
    * Geeignet für: Marlin/Klipper/RRF Firmware
    * Unterstützte Antriebsmodi: TMC: SPI
    * Schritt-/Richtungs-Schnittstelle mit Mikroschrittinterpolation microPlyer
    * Höchste Auflösung: 256 Mikroschritte
    * Unterstützt stealthChop2 für ruhigen Betrieb und flüssige Bewegungen
    * Unterstützt Dämpfung von Resonanzen bei mittleren Frequenzen
    * Unterstützt spreadCycle Hochdynamik-Motorkontroll-Schalter
    * Unterstützt dcStep Lastabhängige Geschwindigkeitskontrolle
    * Unterstützt stallGuard2 hochpräzise sensorlose Motorlastdetektion
    * Unterstützt coolStep Stromkontrolle, spart bis zu 75% Energie

## Hauptplatine Referenzkonfiguration

    :::danger

    Der Antriebs-Sample-Widerstand beträgt 0.033, bitte setzen Sie ihn nicht falsch.

    :::


    <Tabs>
        <TabItem value="Verwendung des Antriebs-EN-Pins zur RGB-Steuerung" default>
        
        * Schalter zum FAN-Anschluss schieben

        ```
        [stepper_x]
        step_pin: 
        dir_pin: 
        # enable_pin: !

        [tmc5160 stepper_x]
        cs_pin: 
        spi_software_mosi_pin: 
        spi_software_miso_pin: 
        spi_software_sclk_pin: 
        run_current: 1.0
        interpolate: False
        sense_resistor: 0.033
        stealthchop_threshold: 0
        driver_DISS2G: 1
        driver_DISS2VS: 1
        ```
        </TabItem>


        <TabItem value="Verwendung des RGB-Pins der Hauptplatine zur RGB-Steuerung" default>
        
        * Schalter zum anderen FAN-Anschluss schieben

        ```
        [stepper_x]
        step_pin: 
        dir_pin: 
        enable_pin: !
        
        [tmc5160 stepper_x]
        cs_pin: 
        spi_software_mosi_pin: 
        spi_software_miso_pin: 
        spi_software_sclk_pin: 
        run_current: 1.0
        interpolate: False
        sense_resistor: 0.033
        stealthchop_threshold: 0
        driver_DISS2G: 1
        driver_DISS2VS: 1
        ```  

        </TabItem>

        <TabItem value="RGB Referenzkonfiguration" default>

        * Steuern der LED des Antriebs durch Plugin-Erweiterung  <Button variant="contained" disableElevation href="/docs/DebugDoc/addon/led.mdx">Hier klicken, um zu springen</Button>

## Stromwarnung

    :::danger

    Der TMC5160 Plus unterstützt 10A, aber übermäßiger Strom kann den Motor beschädigen und erfordert verbesserte Kühlung.

    :::
