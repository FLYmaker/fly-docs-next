# TP 2209

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/tp2209.png').default} size="60%" align="center" />


## Grundlegende Parameter

    * Eingangsspannung: 12V-24V
    * Geeignet für: Marlin/Klipper/RRF Firmware
    * Motor verliert selten Schritte
    * Unterstützt Stille-Modus
    * Große Fläche des Lüfterheizkissens, reduziert die Betriebstemperatur des Treibers
    * Treiber-Modus unterstützt: TMC: UART
    * Unterstützt Motorblockierungserkennung


## FLY-Mutterplatine Jumper

    :::tip

    * TP2209 verwendet UART-Modus
    * TP2209 benötigt keinen Treiber-Jumper, nur eine reine Konfiguration
    
    :::


## TP2209 Referenzkonfiguration

    :::danger

    Der Treibersamplingwiderstand beträgt 0,110, bitte setzen Sie diesen nicht falsch

    :::

    ```cfg
    [tmc2209 stepper_x]
    uart_pin: 
    interpolate: False
    run_current: 0.8                    # Motorbetriebsstromwert (Einheit: mA)
    sense_resistor: 0.110               # Treibersamplingwiderstand nicht ändern
    stealthchop_threshold: 0
    ```



## Unendliche Positionierung Verwendungshinweise

    <DIAG />

    :::tip Hinweise zur unendlichen Positionierung

    * Das ursprüngliche `endstop_pin:` muss deaktiviert oder gelöscht werden, fügen Sie `endstop_pin: tmc2209_stepper_x:virtual_endstop` hinzu
    * Nach der Konfiguration von `diag_pin:` ist der Endschalterstatus auf nicht ausgelöst gesetzt
    * Die Konfiguration von `driver_SGTHRS:` erfordert das Testen eines geeigneten Wertes

    :::


    * Referenzkonfiguration

    ```cfg
    [stepper_x]
    # endstop_pin:PF3
    endstop_pin: tmc2209_stepper_x:virtual_endstop
    homing_retract_dist: 0        # Rückzug nicht auf 0 ändern, kann zu einem Fehlstart führen

    [tmc2209 stepper_x]
    diag_pin: ^
    driver_SGTHRS: 100
    
    ```
