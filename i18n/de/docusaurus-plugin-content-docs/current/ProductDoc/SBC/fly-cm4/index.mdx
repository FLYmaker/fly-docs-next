# Einführung

      Fly-CM4 ist vollständig kompatibel mit Raspberry Pi 3B und 4B und wurde speziell für die Erweiterung des Raspberry Pi CM4 optimiert. Dieses Erweiterungsboard bietet eine komplette Suite an Funktionen, die speziell für 3D-Drucker optimiert sind. Eine 12-24V Gleichstromversorgung gewährleistet Stabilität, was für die präzise Kontrolle und den kontinuierlichen Betrieb eines 3D-Druckers von entscheidender Bedeutung ist. Unterstützung für FLY-TFT und Fly-HDMI Bildschirm-FPC-Verbindungen bietet eine bequemere Verkabelungsmethode. Ein CAN-Port ermöglicht eine direkte Verbindung zwischen Oberflächen- und Unterflächengeräten, was derzeit eine beliebte Methode ist.

## Eigenschaften

    * Unterstützt das Raspberry Pi CM4 Hauptboard
    * Unterstützt 12V-24V Gleichstromversorgung
    * Integriertes UTOC Modul
    * Unterstützt TFT und HDMI Ein-Kabel-Lösung
    * Unterstützt DSI/SPI Bildschirm-Schnittstellen
    * Unterstützt CSI/USB Kamera-Schnittstellen
    * 40PIN Erweiterungsschnittstelle, kompatibel mit Raspberry Pi
