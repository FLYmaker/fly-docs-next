# Einführung

    Fly-Pi-lite2.1 ist eine Einstiegs-Klipper-Hauptsteuerung
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.png').default} size="80%" align="center" />

## Merkmale

    * CPU: Hochleistungs-Chip Allwinner H3, 4-Kern 64-Bit Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Unterstützt bis zu 128GB SDCARD
    * WIFI: Integriertes 2.4G WiFi
    * Peripheriegeräte: Spi x1，URAT x2, USB x2，HDMI x1
    * Schnittstellen: Vorbehaltene FPC, HDMI + USB FPC Einleitung
    * Das System ist vorkonfiguriert mit Klipper, Moonraker, Mainsail, Fluid, Klipper-Screen, Resonanzkompensationsplugin, numpy und FLY-Tools, keine komplexen Quellwechsel nötig, ideal für Anfänger (TF-Karte muss separat erworben werden)


## Aktualisierte Funktionen

    * TFT V1 Schnittstelle auf TFT V2 aktualisiert
    * Hinzufügen von Reset- und Ausschalt-Pins
    * Hinzufügen einer Einschaltanzeige
    * Entfernen des BOOT-Buttons, Austausch gegen Reset-Button
    * Optimierung der Verkabelung
