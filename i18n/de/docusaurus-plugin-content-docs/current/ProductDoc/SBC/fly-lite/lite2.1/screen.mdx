# Bildschirmanschluss und -konfiguration
:::info

* lite2.1 benötigt eine SSH-Verbindung zum System und die Eingabe des folgenden Befehls, um Fly Tools Bildschirmkonfiguration zu nutzen
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools benötigt eine Netzwerkverbindung der Host-Maschine, um über die `IP-Adresse des Hosts +:9999` zugänglich zu sein
* Zum Beispiel `192.168.6.110:9999`

:::
