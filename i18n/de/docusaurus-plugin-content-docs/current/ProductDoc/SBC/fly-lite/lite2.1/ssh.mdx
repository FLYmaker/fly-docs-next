# Verbindung zur Host-Maschine über SSH

* Sie können sich über eine serielle Schnittstelle oder WIFI mit dem Gerät verbinden
* Bereiten Sie bitte im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminal-Tool** vor
* Stellen Sie sicher, dass auf der SD-Karte oder M2WE das **entsprechende Betriebssystem für die Host-Maschine** installiert ist
* **Wenn die Host-Maschine nicht richtig startet, entfernen Sie bitte alle Peripheriegeräte wie Treiber, Endschalter, Lüfter usw.!!!**
* **Verbinden Sie sich zuletzt über das Netzwerk mit SSH, um das Firmware-Update zu erleichtern!!!**
* **Für das Firmware-Update beachten Sie bitte das Tutorial zur Verbindung der Host-Maschine über WIFI oder Ethernet**

        ## Verbindung zur Host-Maschine über IP-Adresse und SSH

        * Melden Sie sich bitte im **Backend Ihres Routers** an
        * Stellen Sie sicher, dass das System **richtig gestartet** wurde
        * Stellen Sie sicher, dass Sie **Benutzername und Passwort der Host-Maschine kennen**, Standardbenutzername im FLY-System **fly**, Standardpasswort **mellow**
        * Wenn das Motherboard gestartet ist und mit dem Netzwerk verbunden ist, können Sie sich über die IP-Adresse mit SSH verbinden
        * Öffnen Sie MobaXterm und konfigurieren Sie wie folgt, geben Sie die IP-Adresse an, die Sie zuvor überprüft haben, oder überprüfen Sie sie im Router-Backend

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Hinweis: Im Schritt 4 geben Sie den Benutzernamen der Host-Maschine ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch erscheint dieses Fenster, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Sie werden aufgefordert, das Passwort einzugeben, geben Sie das Passwort der Host-Maschine ein und drücken Sie die Eingabetaste

        **Hinweis: Das Passwort wird nicht angezeigt, geben Sie es einfach ein und drücken Sie die Eingabetaste**

        Im FLY-System standardmäßig, Benutzername: `fly`, Passwort: `mellow`
