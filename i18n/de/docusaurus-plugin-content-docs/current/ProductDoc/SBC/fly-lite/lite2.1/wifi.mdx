# Verbindung zum WLAN

## Installieren Sie die Antenne auf dem LITE 2.1

    * Der von dem roten Pfeil angezeigte Ort ist die WLAN-Antennenbuchse
    * Die WLAN-Antennenbuchse ist `ipex1`
    * Installieren Sie die Antenne auf dem LITE 2.1, andernfalls könnte aufgrund schlechten Signals keine Internetverbindung hergestellt werden

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.png').default} size="100%" align="left" />  

* Wenn Sie feststellen, dass keine normale WLAN-Verbindung besteht, überprüfen Sie zunächst, ob die Antenne angeschlossen ist, und verwenden Sie dann die untenstehende Methode, um eine Verbindung zum WLAN herzustellen
