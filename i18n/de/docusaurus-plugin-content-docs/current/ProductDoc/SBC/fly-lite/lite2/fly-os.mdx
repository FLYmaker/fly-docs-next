## Vorbereitungen für das System-Brennen

    * Kartenleser
    * Eine TF-Karte, die Kapazität muss mindestens 16GB und maximal 128GB betragen, die Übertragungsgeschwindigkeit der TF-Karte muss mindestens der Klasse 10 entsprechen
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-lite2">System-Image</Button> herunter
    * Laden Sie das benötigte <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Brennprogramm</Button> herunter

## System-Brennen

:::info
* Lite 2 kann nur das System auf die TF-Karte brennen, M2WE kann nicht verwendet werden
* Nach dem Brennen des Systems muss die TF-Karte in den Lite 2 eingebaut werden
:::
