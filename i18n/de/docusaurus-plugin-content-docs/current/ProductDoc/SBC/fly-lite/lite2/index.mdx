# Einführung

    Fly-Pi-lite2 ist ein Einstiegs-Klipper-Hauptgerät

## Merkmale

    * CPU: Hochleistungs-Allwinner-H3-Chip, 4-Kern-64-Bit-Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Unterstützt bis zu 128GB SDCARD
    * WIFI: Integriertes 2.4G WiFi
    * Peripheriegeräte: SPI x1, UART x2, USB x2, HDMI x1
    * Schnittstellen: Reservierte FPC, HDMI + USB FPC One-Wire
    * Das System ist mit klipper, moonraker, mainsail, fulidd, klipper-screen, Resonanzkompensations-Plugin numpy und FLY-Tools vorinstalliert, keine komplexen Operationen wie Quellentausch erforderlich, ideal für Anfänger (eine TF-Karte muss separat erworben werden)
