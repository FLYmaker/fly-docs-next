# Verbindung zum Hostcomputer über SSH

* Sie können über eine serielle Schnittstelle oder WLAN eine Verbindung zum Gerät herstellen
* Bereiten Sie bitte im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminalwerkzeug** vor
* Stellen Sie sicher, dass die SD-Karte oder M2WE mit dem **entsprechenden Betriebssystem für den Hostcomputer** beschrieben ist
* **Wenn der Hostcomputer nicht normal startet, entfernen Sie bitte Treiber, Endschalter, Lüfter und andere Peripheriegeräte!!!**
* **Verbinden Sie sich schließlich über das Netzwerk mit SSH, um das Firmware-Update zu erleichtern!!**
* **Für das Firmware-Update schauen Sie bitte in das Tutorial für die Verbindung des Hostcomputers über WLAN oder Netzwerkkabel**

        ## Verbindung zum Hostcomputer über Netzwerk-IP

        * Melden Sie sich im **Backend des Routers an**, zu dem der Hostcomputer verbunden ist
        * Stellen Sie sicher, dass das System **normal gestartet ist**
        * Stellen Sie sicher, dass Sie **den Benutzernamen und das Passwort des Hostcomputers kennen**, im FLY-System ist der Standardbenutzername **fly** und das Standardpasswort **mellow**
        * Wenn das Motherboard gestartet ist und mit dem Netzwerk verbunden ist, können Sie über die IP-Adresse eine SSH-Verbindung herstellen
        * Öffnen Sie MobaXterm und führen Sie die folgenden Einstellungen durch, geben Sie die IP-Adresse an, die Sie zuvor überprüft haben, oder schauen Sie im Backend des Routers nach

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Achtung: Hier geben Sie im Schritt 4 den Benutzernamen des Hostcomputers ein**
