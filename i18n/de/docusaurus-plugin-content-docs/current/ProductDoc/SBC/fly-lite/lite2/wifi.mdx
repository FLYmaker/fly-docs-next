# WIFI verbinden

## Die Antenne an das LITE 2 anschließen

    * Der mit dem roten Pfeil markierte Ort ist die WIFI Antennenbuchse
    * Der WIFI Anschluss ist `ipex1`
    * Die Antenne muss an das LITE 2 angeschlossen werden, andernfalls könnte eine schlechte Signalverbindung zu Problemen beim Netzwerkzugang führen

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.png').default} size="100%" align="left" />  

* Wenn keine normale WIFI-Verbindung besteht, überprüfen Sie zuerst, ob die Antenne angeschlossen ist, und verwenden Sie dann die unten aufgeführten Methoden, um sich mit dem WiFi zu verbinden
