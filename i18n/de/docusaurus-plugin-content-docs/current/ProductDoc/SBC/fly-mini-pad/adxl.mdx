# Verwendung eines Beschleunigungssensors

    :::tip

    * Mini Pad kann direkt über die eingebaute Schnittstelle mit einem Beschleunigungssensor verbunden oder ein Toolboard mit integriertem Beschleunigungssensor verwendet werden.

    :::

## Externer Beschleunigungssensor über eingebaute Schnittstelle

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.png').default} size="80%" align="left" />

    * Beispielkonfiguration

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # ein Beispiel
    ```
