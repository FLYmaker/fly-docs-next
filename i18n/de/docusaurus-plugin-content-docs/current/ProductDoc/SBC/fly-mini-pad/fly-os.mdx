## Vorbereitungen für das System-Brennen

    * Kartenleser
    * Eine TF-Karte, Kapazität muss mindestens 16GB und maximal 128GB betragen, die Übertragungsgeschwindigkeit der TF-Karte muss mindestens Class10 sein
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-minipad">System-Image</Button> herunter
    * Laden Sie das benötigte <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Brennprogramm</Button> herunter

## System-Brennen

:::info
* MINI PAD kann das System nur auf die TF-Karte brennen, die Verwendung von M2WE ist nicht möglich
* Nach dem Brennen des Systems muss die TF-Karte in das MINI PAD eingesetzt werden
:::
