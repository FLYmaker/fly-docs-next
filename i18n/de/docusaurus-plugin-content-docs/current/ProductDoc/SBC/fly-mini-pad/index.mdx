# Einführung

    Fly-MINI-Pad ist eine Einstiegs-Klipper-Hostmaschine

## Eigenschaften

    * CPU: Hochleistungs-Chip Allwinner H3, 4-Kern-64-Bit Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Unterstützt bis zu 128GB SDCARD
    * WIFI: Integrierter 2.4G WiFi
    * Peripheriegeräte: SPI x1, USB x2, Type-C x1
    * Schnittstellen: Drei XH2.54-2P-Schnittstellen, die jeweils unterstützen, POWER-IN für Stromversorgung bis zu 24V, CAN-BUS für CAN-Bus-Verbindung, LI-PO für 3.7V mit Schutz für Lithium-Ionen-Batterie Stromversorgung/Ladegerät
    * Das System ist mit klipper, moonraker, mainsail, fulidd, klipper-screen, Resonanzkompensationsplugin numpy und FLY-Tools vorinstalliert. Keine komplizierten Quellwechsel nötig, einfacher Einstieg für Anfänger (TF-Karte erforderlich)
    * Bildschirm: Kapazitiver/Touchscreen, Auflösung 320*480
