# Verbindung zur Hostmaschine über SSH

* Sie können über eine serielle Schnittstelle oder WIFI eine Verbindung zum Gerät herstellen
* Bereiten Sie im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminalwerkzeug** vor
* Stellen Sie sicher, dass auf der SD-Karte oder dem M2WE das **entsprechende Betriebssystem der Hostmaschine** gebrannt wurde
* **Wenn die Hostmaschine nicht ordnungsgemäß startet, entfernen Sie bitte Treiber, Endschalter, Lüfter und andere Peripheriegeräte!!!**
* **Stellen Sie schließlich über das Netzwerk eine Verbindung zu SSH her, um das Firmware-Update zu erleichtern!!!**
* **Wenn Sie die Firmware kompilieren müssen, lesen Sie bitte das Tutorial zum Verbinden der Hostmaschine über WIFI oder Netzwerkkabel**

        ## Verbindung zur Hostmaschine über Netzwerk-IP über SSH

        * Melden Sie sich im **Backend des Routers** an, zu dem die Hostmaschine verbunden ist
        * Stellen Sie sicher, dass das System **normal gestartet** ist
        * Stellen Sie sicher, dass Sie **den Benutzernamen und das Passwort der Hostmaschine kennen**. Das FLY-System hat standardmäßig den Benutzernamen **fly** und das Passwort **mellow**
        * Wenn die Hauptplatine gestartet ist und mit dem Netzwerk verbunden ist, können Sie eine SSH-Verbindung über die IP-Adresse herstellen
        * Öffnen Sie MobaXterm und führen Sie die folgenden Einstellungen durch, geben Sie die IP-Adresse ein, die Sie zuvor überprüft haben, oder schauen Sie im Router-Backend nach

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Achtung: Hier geben Sie im 4. Schritt den Benutzernamen der Hostmaschine ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch erscheint dieses Fenster, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Sie werden aufgefordert, ein Passwort einzugeben, geben Sie das Passwort der Hostmaschine ein und drücken Sie die Eingabetaste

        **Achtung: Hier wird das eingegebene Passwort nicht angezeigt, geben Sie es ein und drücken Sie direkt die Eingabetaste**

        Das FLY-System hat standardmäßig den Benutzernamen `fly` und das Passwort `mellow`
