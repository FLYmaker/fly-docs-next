# Verwendung der integrierten UTOC

:::danger Hinweis

Die Verwendung von zwei UTOICs wird nicht empfohlen, da das Brennen und Verbinden sehr umständlich ist.

:::
    * Wenn die integrierte UTOC nicht benötigt wird, führen Sie den folgenden Befehl aus. Es wird keine Rückmeldung gegeben:
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * Zurücksetzen, ebenfalls ohne Rückmeldung:
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * Das MINI Pad hat bereits eine integrierte UTOC
