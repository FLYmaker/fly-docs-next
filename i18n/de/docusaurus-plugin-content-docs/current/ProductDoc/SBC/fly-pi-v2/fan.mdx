# Integrierter 5V Lüfteranschluss

## Pi-V2 verfügt über eine 5V-Steuerungsschnittstelle für die Wärmeableitung des Hauptcontrollers, wie unten abgebildet:

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/fan.png').default} size="80%" align="left" />


## Referenzkonfiguration

    ```cfg
    [mcu host]                      # FlyOS hat bereits eine Linux-MCU für Sie generiert, definieren Sie sie einfach zum Gebrauch           
    serial: /tmp/klipper_host_mcu 

    [temperature_sensor FLY-π]      # Temperatur des Host-Computers
    sensor_type: temperature_host


    [temperature_fan core_fan]      # Lüfter für die Wärmeableitung des Host-Computers
    pin: host:gpiochip1/gpio103     # Pin des Host-Lüfters
    max_power: 1.0
    sensor_type: temperature_host   # Einstellen auf Host-Computertemperatur
    control:watermark               # Kontrollmethode
    target_temp: 48                 # Starttemperatur des Lüfters für den Host-Computer
    min_temp: 0                     # Minimaltemperatur, darunter wird ein Fehler angezeigt
    max_temp: 90                    # Maximaltemperatur, darüber wird ein Fehler angezeigt
    off_below: 0.10
    kick_start_time: 0.50
    max_speed: 0.8                  # Maximale Drehzahl, 80% der maximalen Leistung
    min_speed: 0.3                  # Minimale Drehzahl, 30% der maximalen Leistung
    ```
