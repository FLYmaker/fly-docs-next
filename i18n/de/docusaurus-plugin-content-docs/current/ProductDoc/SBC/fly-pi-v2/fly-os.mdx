## Vorbereitung für das System-Brennen

    * Kartenleser
    * Eine TF-Karte, Kapazität muss mindestens 16GB und maximal 128GB betragen, TF-Kartenübertragungsgeschwindigkeit muss mindestens class10 sein
    * Laden Sie das entsprechende <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">System-Image</Button>
    * Laden Sie das benötigte <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Brennprogramm</Button>

## System-Brennen

:::info
Das Brennen auf die TF-Karte und das Brennen auf die M2WE über die TF-Karte erfordert nur eine Brennvariante
:::

:::info
* Wenn das Gerät M2WE und Speicherkarte installiert hat, startet das System standardmäßig von der Speicherkarte
* Es wird empfohlen, das System über die TF-Karte auf die M2WE zu brennen, da das Brennen über USB auf die M2W aufgrund von Treiberkompatibilitätsproblemen fehlschlagen kann. Es wird empfohlen, direkt die [Speicherkarte auf die M2WE zu brennen](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Achten Sie darauf, ob das heruntergeladene System-Image passt, da es andernfalls nicht starten kann
:::
