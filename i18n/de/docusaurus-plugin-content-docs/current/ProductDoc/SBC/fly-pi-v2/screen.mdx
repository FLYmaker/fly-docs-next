# Bildschirmanschluss und -konfiguration

:::info

* FLY-TOOLS benötigt eine Netzwerkverbindung mit der Host-Anwendung und den Eingabewert der Host-Anwendung `IP+:9999` für den Zugriff
* Zum Beispiel `192.168.6.110:9999`

:::
