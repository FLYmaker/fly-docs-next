# Verbindung zur Host-Maschine über SSH


* Sie können über eine serielle Schnittstelle oder WIFI eine Verbindung zum Gerät herstellen
* Bereiten Sie bitte im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminalwerkzeug** vor
* Stellen Sie sicher, dass auf der SD-Karte oder M2WE das **entsprechende Betriebssystem für die Host-Maschine** gebrannt wurde
* **Wenn die Host-Maschine nicht ordnungsgemäß startet, entfernen Sie bitte alle Treiber, Endschalter, Lüfter usw.!!**
* **Verbinden Sie sich schließlich über das Netzwerk per SSH, um das Firmware-Update zu erleichtern!!**
* **Für das Firmware-Update bitte das Tutorial zur Verbindung der Host-Maschine über WIFI oder Netzwerkkabel beachten**

        ## Verbindung zur Host-Maschine über Netzwerk-IP

        * Melden Sie sich bitte im **Backend des Routers** an, zu dem die Host-Maschine verbunden ist
        * Stellen Sie sicher, dass das System **ordnungsgemäß gestartet** ist
        * Stellen Sie sicher, dass Sie **den Benutzernamen und das Passwort der Host-Maschine kennen**. Der Standardbenutzername des FLY-Systems ist **fly**, das Standardpasswort ist **mellow**
        * Wenn die Hauptplatine gestartet ist und mit dem Netzwerk verbunden ist, können Sie sich über die IP-Adresse per SSH verbinden
        * Öffnen Sie MobaXterm und nehmen Sie die folgenden Einstellungen vor, geben Sie die IP-Adresse ein, die Sie zuvor überprüft haben, oder schauen Sie im Router-Backend nach

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Hinweis: Hier geben Sie den Benutzernamen der Host-Maschine ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsversuch wird dieses Fenster erscheinen, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Es wird nach einem Passwort gefragt, geben Sie das Passwort der Host-Maschine ein und drücken Sie Enter

        **Hinweis: Hier ist das Passwort nicht sichtbar, geben Sie es einfach ein und drücken Sie Enter**

        Das Standard-FLY-System hat den Benutzernamen: `fly` und das Passwort: `mellow`
