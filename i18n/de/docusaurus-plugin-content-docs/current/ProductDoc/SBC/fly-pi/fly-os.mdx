## Vorbereitung vor dem Systemflashing

    * Karteleser
    * Eine TF-Karte mit einer Kapazität von mindestens 16 GB oder nicht mehr als 128 GB TF-Karte, die Übertragungsgeschwindigkeit der TF-Karte muss mindestens Class10 oder höher sein
    * Herunterladen des entsprechenden <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">Systemabbilds</Button>
    * Herunterladen der benötigten <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Flash-Tools</Button>

## Systemflashing

:::info
Es muss nur eine der beiden Methoden zum Flashen der TF-Karte oder zum Flashen über die TF-Karte auf die M2WE verwendet werden.
:::

:::info
* Wenn das Gerät sowohl eine M2WE als auch eine Speicherkarte installiert hat, startet das System standardmäßig von der Speicherkarte.
* Es wird empfohlen, das System auf die M2WE mit einer TF-Karte zu flashen. Das Flashen über USB auf die M2W kann aufgrund von Treiberkompatibilitätsproblemen fehlschlagen. Es wird empfohlen, direkt [die Speicherkarte auf die M2WE zu flashen](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os).
* Bitte achten Sie darauf, dass das heruntergeladene Systemabbild korrekt ist, da es sonst nicht starten kann.
:::
