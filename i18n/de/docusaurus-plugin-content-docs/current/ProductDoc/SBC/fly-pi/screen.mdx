# Bildschirmanschlüsse und Konfigurationen

:::info

* FLY-TOOLS benötigt nach dem Herstellen einer Netzwerkverbindung mit dem Host-PC die Eingabe von `IP:9999` des Host-PCs für den Zugriff
* Zum Beispiel `192.168.6.110:9999`

:::
