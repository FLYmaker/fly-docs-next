# Verbindung zur Host-Maschine über SSH

* Sie können über eine serielle Schnittstelle oder WIFI mit dem Gerät verbinden
* Bereiten Sie bitte im Voraus **MobaXterm_Personal** oder ein anderes **SSH-Terminalwerkzeug** vor
* Stellen Sie sicher, dass auf der SD-Karte oder dem M2WE das **entsprechende Betriebssystem der Host-Maschine** installiert ist
* **Falls die Host-Maschine nicht ordnungsgemäß startet, entfernen Sie bitte Treiber, Endschalter, Lüfter usw.!**
* **Verbinden Sie sich zuletzt über das Netzwerk mit SSH, um die Firmware zu kompilieren!**
* **Für das Kompilieren der Firmware schauen Sie bitte in das Tutorial zur Verbindung über WIFI oder Netzwerkkabel zur Host-Maschine**

        ## Verbindung zur Host-Maschine SSH über Netzwerk-IP

        * Melden Sie sich bitte im **Router-Backend** an, zu dem die Host-Maschine verbunden ist
        * Stellen Sie sicher, dass das System **ordnungsgemäß gestartet** ist
        * Stellen Sie sicher, dass Sie **den Benutzernamen und das Passwort der Host-Maschine kennen**, Standardbenutzername des FLY-Systems ist **fly**, Standardpasswort ist **mellow**
        * Wenn die Hauptplatine gestartet ist und mit dem Netzwerk verbunden ist, können Sie über die IP-Adresse eine SSH-Verbindung herstellen
        * Öffnen Sie MobaXterm und nehmen Sie die folgenden Einstellungen vor, die IP-Adresse füllen Sie mit der zuvor gefundenen aus, oder schauen Sie im Router-Backend nach

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Achtung: Hier geben Sie den Benutzernamen der Host-Maschine im vierten Schritt ein**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Beim ersten Verbindungsaufbau wird dieses Fenster angezeigt, klicken Sie auf `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Es wird aufgefordert, das Passwort einzugeben, geben Sie das Passwort der Host-Maschine ein und drücken Sie die Eingabetaste

        **Achtung: Das Passwort wird nicht angezeigt, geben Sie es ein und drücken Sie dann direkt die Eingabetaste**

        Standardmäßig hat das FLY-System den Benutzernamen: `fly` und das Passwort: `mellow`
