## Produkteinführung

    * Bildschirmgröße: 120.7mm*76.0mm
    * Anzeigefläche des Bildschirms: 108mm*66.2mm
    * Auflösung: 800*480
    * Bildschirmtyp: IPS
    * Kommunikationsart: DSI/Mipi
    * Eingebaute Schnittstelle: 15 pin DSI / 22pin Mipi
    * Stabile Arbeitsbedingungen: 0°-60°C
    * Berührungstyp: Kapazitiv
    * Logikspannung: 3.3V
    * Gewicht: 135.5g

        :::danger Wichtige Informationen

        * Die DSI- und CSI-Schnittstellen des Raspberry Pi 5 heißen jetzt MIPI
        * Die MIPI-Schnittstelle kann durch Konfiguration direkt für CSI-Kameras oder DSI-Bildschirme verwendet werden
        * Beachten Sie die Richtung der Goldfinger

        :::

        * Der Raspberry Pi 5 hat jetzt zwei MIPI-Schnittstellen, wenn man von vorne schaut, kann man die Markierungen 1 und 0 sehen

        * DSI-Bildschirmanschluss


            * Wenn Sie die Schnittstelle Nr. 1 verwenden, müssen Sie den Bildschirm wie folgt konfigurieren

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Wenn Sie die Schnittstelle Nr. 0 verwenden, müssen Sie den Bildschirm wie folgt konfigurieren

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
