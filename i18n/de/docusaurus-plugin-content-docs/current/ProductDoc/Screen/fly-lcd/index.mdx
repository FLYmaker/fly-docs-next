# Einführung

    * Modularer Aufbau, unterstützt mehrere DSI- und HDMI-Kernplatten
    * Verschiedene Größen, 4,3 Zoll, 4 Zoll, 7 Zoll
    * Hochauflösend, kein Detail wird übersehen
    * Unterstützt 10 Stufen der Helligkeitsregulierung
    * Unterstützt 180° Bildschirmumkehr

:::warning Hinweise
Der Sieben-Zoll-Bildschirm kann nicht mit HDMI-Kernplatten verwendet werden, DSI-Kernplatten funktionieren normal
:::
