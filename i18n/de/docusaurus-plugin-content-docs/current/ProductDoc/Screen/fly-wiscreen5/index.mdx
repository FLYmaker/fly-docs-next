# Einführung

      Fly_WiScreen ist ein hochwertiger Klipper-Kontrollbildschirm, entwickelt vom Fly3D-Mainboard-Forschungsteam von Guangzhou Magnesium Lun Electronic Technology Co., Ltd. Der Bildschirm ermöglicht die Steuerung des 3D-Druckers über Wifi, unterstützt die Übertragung von Druckdateien über SD-Karten und USB-Sticks und bietet eine sehr schnelle Bildschirmreaktionszeit für eine ausgezeichnete Benutzererfahrung. Es unterstützt die Einstellung von Master-Slave-Modus für die synchronisierte Steuerung mehrerer Maschinen, was die Verwaltung der Maschinen für den Benutzer erleichtert. Der 5V-Anschluss kann mit einem Lademodul verbunden werden, um eine Batterie für mobile Nutzung zu ermöglichen.

## Produktfunktionen

    * Unterstützt die Verbindung mit anderen Klipper-Hosts über WIFI
    * Unterstützt die WIFI-Suche, nach dem Verbinden mit WIFI kann der Host automatisch gesucht werden
    * Unterstützt die Clusterkontrolle mehrerer Drucker
    * Unterstützt das Hochladen von Drucken über USB und TF-Karte
    * Benutzerdefinierte Makros, die den Betrieb und die Fehlersuche erleichtern

## Produktspezifikationen

    * CPU: Allwinner T113-S3, Dual-Core CORTEX-A7, max. 1.2GHz
    * Arbeitsspeicher: 128M DDR3 integriert
    * Speicher: 128MB Nand Flash, TF-Kartensteckplatz
    * Anzeige: 800x480 Kapazitiver Bildschirm
    * Externe Schnittstellen: Ein USB 2.0, ein TF-Kartensteckplatz
    * WIFI-Protokoll: 802.11b/g/n
    * Bildschirmtyp: IPS
    * Bildschirmgröße: 5 Zoll
    * Mehrpunkt-Touch: Unterstützt
    * Nennspannung: 5V
