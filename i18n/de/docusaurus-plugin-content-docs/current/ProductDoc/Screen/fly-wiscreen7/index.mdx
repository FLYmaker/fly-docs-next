# Einführung

      Fly_WiScreen ist ein High-End-Klipper-Kontrollbildschirm, der vom Fly3D-Motherboard-Entwicklungsteam von Guangzhou Mellow Electronic Technology Co., Ltd. entwickelt wurde. Er kann über WLAN und Netzwerkkabel den 3D-Drucker steuern, unterstützt die Übertragung von Druckdateien über SD-Karten und USB-Sticks. Der Bildschirm reagiert sehr schnell und bietet dem Benutzer eine exzellente Nutzererfahrung. Es unterstützt die Einstellung von Master-Slave-Modus, um die Verwaltung von Maschinen durch den Benutzer zu erleichtern. Der 5V-Eingang kann mit einem Lademodul verbunden werden, um eine Batterie für den mobilen Einsatz zu modifizieren.

## Produktfunktionen

    * Unterstützt die Verbindung mit anderen Klipper-fähigen Hosts über WLAN und Netzwerkkabel
    * Unterstützt WLAN-Suche, nach dem Verbinden mit WLAN kann der Host automatisch gesucht werden
    * Unterstützt die Clustersteuerung mehrerer Drucker
    * Unterstützt das Hochladen von Drucken über USB und TF-Karten
    * Benutzerdefinierte Makros, für eine einfachere Bedienung und Fehlersuche

## Produktspezifikationen

    * CPU: Allwinner T113-S3, Dual-Core CORTEX-A7 bis zu 1,2 GHz
    * Speicher: Integrierter 128M DDR3
    * Speicher: 128MB Nand Flash, TF-Kartensteckplatz
    * Display: 1024X800 Kapazitiver Bildschirm
    * Externe Schnittstellen: Ein USB 2.0, ein TF-Kartensteckplatz
    * WLAN-Protokoll: 802.11b/g/n
    * Bildschirmtyp: IPS
    * Bildschirmgröße: 7 Zoll
    * Mehrfachberührung: Unterstützt
    * Nennspannung: 5V
