## Einstellung der Funktion des Buttons
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide10.gif').default} size="100%" align="left" />

* Senden eines einzelnen Gcode-Befehls mit einer Funktion  
    ```cfg
    gcode_send2("","G28");
    ```
* Senden mehrerer Gcode-Befehle mit einer Funktion:
    ```cfg
    gcode_send2("","G91\\nG1 X10 F3000\\nG90");
    ```
* Oder senden Sie mehrere Gcode-Befehle auf diese Weise:
    ```cfg
    gcode_send2("","G91");
    gcode_send2("","G1 X10 F3000");
    gcode_send2("","G90");
    ```
