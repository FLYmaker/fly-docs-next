## Aktualisieren
    * 1. Benennen Sie die heruntergeladene Datei in `update.img` um.
    * 2. Legen Sie die Datei `update.img` in eine SD-Karte/USB-Stick ein und stecken Sie diese in den Bildschirm. Dies kann auch im eingeschalteten Zustand des Bildschirms erfolgen.
    * 3. Wählen Sie den Bildschirm aus und klicken Sie auf "Upgrade", um das Upgrade abzuschließen.
