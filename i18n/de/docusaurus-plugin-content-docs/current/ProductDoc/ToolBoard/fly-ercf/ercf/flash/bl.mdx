## Betriebsmodus aufrufen

    :::tip

    Der markierte Bereich ist die `BOOT-Taste`

    :::

    * Um in den Betriebsmodus zu gelangen, muss die Hauptplatine vollständig stromlos gemacht werden. Drücken Sie die BOOT-Taste und verbinden Sie dann das Datenkabel mit der Hauptplatine und dem Host.

            4. Firmware aufspielen

            :::tip

            Sie müssen nur eine der Firmwares aufspielen

            :::

            * Unten ist der Befehl zum Aufspielen der Firmware mit 500K CAN-Bitrate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Unten ist der Befehl zum Aufspielen der Firmware mit 1M CAN-Bitrate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Unten ist der Befehl zum Aufspielen der USB-Firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. Wenn die folgende Nachricht erscheint, war das Aufspielen erfolgreich

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            Nach dem Aufspielen der Firmware muss die Hauptplatine einmal vollständig stromlos gemacht werden.

            :::
