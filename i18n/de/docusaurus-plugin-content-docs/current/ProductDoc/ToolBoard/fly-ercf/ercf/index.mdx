# Produkteinführung

    FLY-ERCF ist eine auf RP2040 basierende Steuerplatine für 3D-Drucker mit Mehrfarbensteuerung, hauptsächlich für das Enraged Rabbit Projekt verwendet. Sie kann auch für andere Zweifach-Stapelmotor-Kontrollanwendungen genutzt werden.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.png').default} size="100%" align="left" />

## Produktmerkmale

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Zwei Steuersätze, unterstützt UART und SPI-Treiber
    * Kompatibel mit der originalen ERCF EASY Board
    * Extremer Volumenkompression, 4-Schicht-Plattendesign
    * Integriertes 2.54MM-Doppelreihen-7PIN-Erweiterungsinterface, erweiterbar für Endschalter und andere Zwecke
    * Unterstützt 12-24V Spannung
    * Unterstützt CAN-Bus und RS232-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * CAN-Schnittstelle verwendet XT30 (2+2), unterstützt 15A Strom

## Produktspezifikationen

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Eingangsspannung: DC12V-24V
    * Schnittstellen: 2.54MM-Doppelreihen-7PIN-Erweiterungsinterface, SERVO X1, ENDSTOP X2, EXTRA X1
    * Treiber: Unterstützt UART/SPI und STEP/DIR

    * Unterstützt 12-24V Spannung
    * Unterstützt CAN-Bus, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * CAN-Schnittstelle verwendet XT30 (2+2), unterstützt 15A Strom
