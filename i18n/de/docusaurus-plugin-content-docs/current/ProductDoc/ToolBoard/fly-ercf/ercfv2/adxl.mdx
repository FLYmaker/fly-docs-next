# Verwendung des Beschleunigungssensors


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Wechsel zu klipper

    * Da es noch nicht zusammengeführt wurde, muss klipper gewechselt werden, um es zu verwenden

    ```bash
    sudo cp -r klipper klipperdev
    sudo rm -rf ~/klipper
    git clone https://github.com/FLYmaker/klipper.git
    sudo systemctl restart klipper
    ```

    * Referenzkonfiguration
    ```cfg
    #####################################################################
    #                       LIS3DH-Beschleunigungssensor-Konfiguration (bei Bedarf aktivieren)
    #####################################################################
    [lis2dw]
    is_lis3dh:True
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis2dw
    probe_points:
        125, 125, 20  #Setzen Sie hier die Koordinaten des Düsenplatzes für die Resonanzmessung


```
