## Betreten des Flash-Modus

    :::tip

    Der eingerahmte Bereich ist die `BOOT-Taste`

    :::

    * Zum Betreten des Flash-Modus muss die Hauptplatine vollständig stromlos sein, dann die BOOT-Taste gedrückt halten und das Datenkabel, das Datentransfer unterstützt, an die Hauptplatine und das Host-System anschließen

            4. Firmware flashen

            :::tip

            Es muss nur eine der Firmwares geflasht werden

            :::

            * Untenstehend der Flash-Befehl für die CAN-Bitrate von 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_500K.uf2
                ```
                       
            * Untenstehend der Flash-Befehl für die CAN-Bitrate von 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_1M.uf2
                ```
            * Untenstehend der Flash-Befehl für USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCFV2_CANBOOT_USB.uf2
                ```

            5. Bei diesem Hinweis ist das Flashen erfolgreich abgeschlossen

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            Nach dem Flashen muss die Hauptplatine einmal vollständig stromlos gemacht werden.

            :::
