# RS232-Firmware-Kompilierung

## Kompilierung beginnen

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Wählen Sie GPIO-Pins für den Start des Mikrocontrollers aus, geben Sie ein: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.png').default} size="100%" align="center" />
    * Drücken Sie die `Q`-Taste, es erscheint **Konfiguration speichern**, dann die `Y`-Taste drücken
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Die Konfiguration sollte jetzt gespeichert und die Konsole verlassen sein
    * Geben Sie `make -j4` ein, um mit der Kompilierung zu beginnen, dies dauert etwas
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn am Ende die folgenden Inhalte ausgegeben werden, war die Kompilierung erfolgreich

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## In den Flash-Modus wechseln

    :::tip

    Der umrahmte Bereich ist die `BOOT-Taste`

    :::

    * Um in den DFU-Modus zu wechseln, muss die Hauptplatine vollständig stromlos sein, halten Sie die BOOT-Taste gedrückt und verbinden Sie dann das Datenkabel mit der Hauptplatine und dem Hostcomputer
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.png').default} size="100%" align="center" />
