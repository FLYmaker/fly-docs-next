# Produkteinführung

    FLY-ERCF ist eine 3D-Drucker-Mehrfarben-Kontrollkarte basierend auf RP2040, hauptsächlich für das Enraged Rabbit Projekt verwendet. Es kann auch für andere Anwendungen mit zwei Achsen und Schrittmotorkontrolle eingesetzt werden.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.png').default} size="100%" align="left" />

## Produktmerkmale

    * MCU：Raspberry Pi RP2040, Dual-Core ARM Cortex-M0+@133MHz
    * Unterstützt CAN-Modus, RS232-Modus
    * Zwei Treiberpositionen, unterstützt UART und SPI Treiber
    * Kompatibel mit der Original-ERCF EASY Board
    * Extrem kompaktes Design, 4-Lagen-Board
    * 14-Pin-Erweiterungsanschluss durch XHD2.54 ersetzt, sicherere Verkabelung zur Vermeidung von Unterbrechungen
    * Unterstützt 12-24V Spannung
    * Unterstützt CAN-Bus- und RS232-Verbindungen, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * CAN-Schnittstelle verwendet XT30 (2+2), unterstützt 15A Strom

## Produktspezifikationen

    * MCU：Raspberry Pi RP2040, Dual-Core ARM Cortex-M0+@133MHz
    * Firmware: Klipper CAN/RS232
    * Eingangsspannung: DC12V-24V
    * Schnittstellen: XHD2.54 Doppelreihe 7-Pin Erweiterungsanschluss, SERVO X1, ENDSTOP X2, EXTRA X1
    * Treiber: unterstützt UART/SPI und STEP/DIR
    * Unterstützt 12-24V Spannung
    * Unterstützt CAN-Bus und RS232, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * CAN-Schnittstelle verwendet XT30 (2+2), unterstützt 15A Strom
