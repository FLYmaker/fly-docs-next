## Betreten des Programmiermodus

    :::tip

    Oben ist die `RESET-Taste`, unten die `BOOT-Taste`

    :::

    * Es gibt zwei Methoden, um den Programmiermodus zu betreten
    * Erste Methode: Wenn das Hauptplatine vollständig stromlos ist, drücken Sie die BOOT-Taste und verbinden Sie dann das Datenkabel mit dem Hauptplatine und Ihrem Gerät
    * Zweite Methode: Nachdem Sie das Datenkabel mit dem Hauptplatine und Ihrem Gerät verbunden haben, drücken Sie die BOOT-Taste, dann die RST-Taste, lassen Sie die RST-Taste los und schließlich die BOOT-Taste los
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/boot.png').default} size="100%" align="center" />

            4. Firmware programmieren

            :::tip

            Es muss nur eine Firmware programmiert werden

            :::

            * Hier ist der Befehl zum Programmieren der Firmware mit einer CAN-Geschwindigkeit von 500K
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```
                       
            * Hier ist der Befehl zum Programmieren der Firmware mit einer CAN-Geschwindigkeit von 1M
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```

            * Hier ist der Befehl zum Programmieren der USB-Firmware
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_USB.bin
                ```

            5. Wenn der Pfeil wie unten gezeigt erscheint, ``File downloaded successfully``, war das Programmieren erfolgreich
