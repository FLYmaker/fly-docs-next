    <TabItem value="usb" label="USB-ID abrufen" default>
        
    * Bereiten Sie ein Datenkabel mit Datenübertragungsfunktion vor, bei besonders langen Kabeln wird ein Kabel mit Ferritkern empfohlen.
    
        <GetUsbID />

    <TabItem value="can" label="CANBus UUID abrufen">

    * Vor der ID-Suche, verbinden Sie die Werkzeugplatine mit UTOC oder einer Hauptplatine mit Firmware für Brückenfunktion über ein XT30-Datenkabel, unten ist eine Referenz-Verkabelung.
    * Achten Sie darauf, dass der DIP-Schalter auf die richtige Position gestellt ist.
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Stellen Sie sicher, dass CAN H und CAN L im ausgeschalteten Zustand mit einem Multimeter gemessen einen Widerstand von etwa `60Ω` haben. Wenn der Widerstand `120Ω` beträgt, muss ein `120Ω`-Jumper hinzugefügt werden. Wenn der Widerstand etwa `40Ω` beträgt, muss der Jumper entfernt werden.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.png').default} size="50%" align="center" />

    </TabItem>

    * Achten Sie darauf, dass der DIP-Schalter auf die richtige Position gestellt ist.
