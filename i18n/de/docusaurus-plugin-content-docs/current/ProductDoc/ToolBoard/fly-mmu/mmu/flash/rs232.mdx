# RS232-Firmware-Kompilierung

## Kompilierung starten

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Wählen Sie GPIO-Pins zum Setzen beim Mikrocontroller-Start aus, Eingabe: !PA15 Beachten Sie, dass ! großgeschrieben ist
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/rs232.png').default} size="100%" align="center" />
    * Drücken Sie die `Q`-Taste, es erscheint **Save configuration**, dann die `Y`-Taste drücken
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und die Konsole verlassen sein
    * Geben Sie `make -j4` ein, um die Kompilierung zu starten, dies kann etwas Zeit in Anspruch nehmen
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn die Kompilierung erfolgreich war, wird folgender Inhalt ausgegeben:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## In den Flash-Modus wechseln

    :::tip

    Oben befindet sich die `RESET`-Taste, unten die `BOOT`-Taste

    :::

    * Es gibt zwei Methoden, um in den DFU-Modus zu gelangen:
    * Erste Methode: Bei komplett abgeschaltetem Hauptbrett, die `BOOT`-Taste gedrückt halten und das Datenkabel an das Hauptbrett und den Host anschließen
    * Zweite Methode: Beim angeschlossenen Datenkabel an Hauptbrett und Host, die `BOOT`-Taste gedrückt halten, dann die `RST`-Taste drücken und dann die `RST`-Taste loslassen und schließlich die `BOOT`-Taste
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/boot.png').default} size="100%" align="center" />

## Firmware flashen 
    
    <Katapult />
    
    3. Firmware flashen
    
        ```bash
        sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/kliper/out/klipper.bin
        ```

    4. Wenn der Pfeil wie unten gezeigt erscheint, ``File downloaded successfully``, war das Flashen erfolgreich

        <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.png').default} size="100%" align="center" />

    :::tip

    Nach dem Flashen muss das Hauptbrett einmal vollständig vom Strom getrennt werden.

    :::
