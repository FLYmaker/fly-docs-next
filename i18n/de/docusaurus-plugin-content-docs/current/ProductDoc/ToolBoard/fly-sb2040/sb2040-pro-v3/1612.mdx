# Eddy Verwendung

    :::info

    * Bitte beachten Sie, dass für das Scannen des Bettes die Verwendung einer separaten Z-Begrenzung empfohlen wird, und es wird nicht empfohlen, das Bett als Z-Begrenzung zu verwenden
    * Beim Installieren von Eddy muss der Abstand vom Unterteil zur Düse mindestens 1-2 mm über der Düse liegen
    * Beim Kalibrieren der Eddy-Frequenz muss die Spule mindestens 5 mm vom Heizbett entfernt sein
    * Bitte testen Sie das gesamte Heizbett bei Raumtemperatur und schalten Sie das Heizbett nicht ein, während Sie testen

    :::

## Referenzkonfiguration und Bedienungsanleitung

    ### Bett-Scan-Konfiguration

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Bitte beachten Sie, dass derzeit kein externes 1612-Modul verfügbar ist. Diese Konfiguration kann vorerst nicht verwendet werden
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Temperaturkompensation-Konfiguration

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh-Konfiguration

    * Im Abschnitt [bed_mesh] sollte der Parameter `horizontal_move_z` auf 2 gesetzt werden, damit Eddy beim Scannen der Plattform so nah wie möglich an der Plattform ist

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
