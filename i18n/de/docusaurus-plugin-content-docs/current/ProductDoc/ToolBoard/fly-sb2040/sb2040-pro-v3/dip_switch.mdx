# Einführung in die Onboard-DIP-Schalter

:::warning Hinweise

Aufgrund der hohen Bildauflösung kann das Bild auf der Webseite nicht klar dargestellt werden. Bitte **bewegen Sie den Mauszeiger über das Anschlussdiagramm**, **klicken Sie mit der rechten Maustaste** und wählen Sie **Bild speichern unter**, um das Bild auf den Desktop zu speichern und es dann zu betrachten!!!

:::
