<Tabs>
    <TabItem value="can" label="CANBus UUID abrufen">

    * Verbinden Sie das Toolboard vor der ID-Suche mit UTOC oder einem Mainboard mit Bridge-Firmware über ein XT30-Datenkabel. Hier ist ein Referenz-Verkabelungsdiagramm.
    * Achten Sie darauf, ob der DIP-Schalter korrekt eingestellt ist.
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Stellen Sie sicher, dass der Widerstandswert zwischen CAN H und CAN L bei ausgeschaltetem Strom mit einem Multimeter gemessen etwa `60Ω` beträgt. Wenn es `120Ω` ist, muss der `120Ω`-DIP-Schalter aktiviert werden. Wenn es etwa `40Ω` ist, muss der Schalter deaktiviert werden.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.png').default} size="100%" align="center" />

    </TabItem>

    * Achten Sie darauf, ob der DIP-Schalter korrekt eingestellt ist.
