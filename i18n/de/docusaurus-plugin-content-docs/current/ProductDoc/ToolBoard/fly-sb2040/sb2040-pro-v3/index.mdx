# 1. Produkteinführung

    FLY SB2040 PRO V3 ist eine Werkzeugplatte, die von Guangzhou Mellow Electronic Technology Co., Ltd. speziell für den VORON Stealthburner Druckkopf entwickelt wurde. Mit dieser Werkzeugplatte können Sie die komplizierte Verkabelung zwischen der Hauptplatine und dem Druckkopf durch nur vier Drähte ersetzen und das Layout optimieren. SB2040 ist geeignet für Stealthburner und Afterburner und integriert einen CAN-Transceiver, RS232-Transceiver, USB-Anschluss, TMC2240-Treiber, LIS2DW-Beschleunigungssensor, drei steuerbare Lüfter und andere Funktionen. Diese Werkzeugplatte ist kompatibel mit Klipper- und RRF-Firmware.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/img/sb2040.png').default} size="100%" align="center" />


## 1.1 Produkteigenschaften

    * Vorhandener Boot-Knopf, Benutzer können über TypeC in den Boot-Modus wechseln und die Firmware aktualisieren
    * Unterstützt Klipper-Firmware, RRF-Firmware
    * Unterstützt CAN-Modus, RS232-Modus, RRF CAN-Modus
    * Im RS232-Modus bietet er eine höhere Übertragungsrate und größere Stabilität. Im CAN-Modus unterstützt er CAN-Erweiterungen und bietet eine CAN-Erweiterungsschnittstelle
    * Temperatursensoren: Integrierter MAX31865-Chip, kompatibel mit PT100- und PT1000-Temperatursensoren.
    * Integrierter LIS2DW-Beschleunigungssensor und LDC1612-Sensor
    * Vorhandene RGB-Schnittstelle, integrierter 2240-Treiber, unterstützt verschiedene Nivellierungssensoren wie klicky, Voron Tap, PL08n usw.
    * Unterstützt 12-24V Spannung, 15A Strom, erweiterter Heizstabanschluss, Nennstrom 10A

## 1.2 Neue Funktionen

    * Unterstützt das Wechseln zwischen Klipper CAN-Modus, RS232-Modus, RRF CAN-Modus
    * Neuer RS232-Modus mit höherer Übertragungsrate, größerer Stabilität, einfacherem Debugging
    * Neuer RRF CAN-Modus, kann in eine RRF-Werkzeugplatte umgeschaltet werden, erhöht die Anwendbarkeit
    * Neue CAN-Erweiterungsschnittstelle für einfache Erweiterung von CAN-Geräten
    * Neuer Schalter für Endwiderstand, der Endwiderstand von 120R kann über DIP-Schalter ausgewählt werden, ohne Jumper
    * Neuer 1 Meter Strom-/Signalanschlusskabel, das Anschlusskabel verwendet ein Twist-Design für besseren Störschutz

## 1.3 Produktspezifikationen


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Anschlusschnittstelle verwendet XT30(2+2)-Stecker, unterstützt 15A Strom, Spitzenwert 30A
    * Firmware: Reprap/Klipper
    * Eingangsspannung: DC12V-24V
    * Logikspannung: DC3.3V
    * Integrierte Sensoren: LIS2DW und optional ldc1612 und max31865
    * Heizstabanschluss: Heizstab (HEAT0)
    * Erweiterungsschnittstelle: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN-Anschluss
    * Lüfteranschluss: ein DC-Lüfter, zwei DC- oder PWM-Lüfter
    * Motorantrieb: Integrierter TMC2240
    * Temperatursensoren: ein 100K NTC oder PT1000 (TH0), ein PT100/PT1000 optional
