# Lösung für Unterspannung des Treibers

## Methode eins

    :::danger

    * Stromversorgung austauschen
    * Netzteilfilter hinzufügen
    * Motor auswechseln und überprüfen
    * Motorhaube mit der GND-Leitung der Werkzeugplatte verbinden
    * In der Extruder-Treiberkonfiguration `driver_TPFD: 0` hinzufügen

    :::

## Methode zwei
    * Über SSH mit dem Hostrechner verbinden
    * Folgenden Befehl eingeben
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Nach dem Update von Klipper könnte dieser Schritt rückgängig gemacht werden
    :::
