            4. Firmware programmieren

            :::tip

            Es ist nur eine der Firmwares erforderlich

            :::

            * Hier ist der Befehl zum Programmieren der Firmware mit 500K CAN-Geschwindigkeit
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * Hier ist der Befehl zum Programmieren der Firmware mit 1M CAN-Geschwindigkeit
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. Wenn die folgende Nachricht erscheint, war das Programmieren erfolgreich

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            Nach dem Programmieren muss das Hauptboard einmal vollständig vom Strom getrennt werden.

            :::
