# Firmware-Flash-Anleitung

    * Das SB2040 PRO kann über eine CAN-Verbindung mit dem Host-Computer verbunden werden
    * CAN erfordert die Verwendung von `XT30(2+2)-F` zur Verbindung
    * Hier ist das Aussehen des `XT30(2+2)-F` Datenkabels
    <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Wichtig
* Der Flash der Hauptplatinen-Firmware erfordert das Flashen von Klipper über Katapult
* Wenn feststeht, dass die Hauptplatine bereits mit Katapult geflasht wurde, ist der letzte Flash der Katapult-Firmware nicht erforderlich, dies dient nur als Vorkehrung, falls der Flash der Katapult-Firmware auf der Hauptplatine nicht erfolgreich war
* Man kann zweimal schnell die Reset-Taste drücken und schauen, ob eine LED auf der Hauptplatine zu blinken beginnt, wenn ja, bedeutet das, dass die Hauptplatine bereits in den Katapult-Firmware-Modus eingetreten ist
:::

## Wie man ermittelt, welche Art von Firmware benötigt wird

    * Das SB2040 PRO kann auf folgende Weise mit dem Host-Computer verbunden werden
    * [CAN-Firmware](./can.mdx "Klicken Sie hier, um weiterzuleiten")
        * Vorteile: Wenn die `Katapult`-Firmware geflasht wurde, kann die Firmware ohne Berührung der Hauptplatine aktualisiert werden, und es unterstützt Bus-Verbindungen
        * Nachteile: Kann bei gleichzeitiger Nutzung mit `KlipperScreen` zu Fehlern wie Timeout beim Home-Vorgang führen.
        * Hinweis: Wenn CAN verwendet werden soll, stellen Sie sicher, dass der Host-Computer CAN unterstützt, und es wird UTOC oder eine Hauptplatine mit gebrückter Firmware benötigt.


## Katapult-Firmware-Erklärung

    * Das SB2040 PRO wird standardmäßig mit der CAN-Verbindungsgeschwindigkeit von 1M ausgestattet
