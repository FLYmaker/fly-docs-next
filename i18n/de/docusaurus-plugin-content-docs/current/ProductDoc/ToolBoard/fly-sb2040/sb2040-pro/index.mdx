# Produkteinführung

    FLY SB2040 PRO ist ein Toolboard, das von Guangzhou Mellow Electronic Technology Co., Ltd. speziell für den VORON Stealthburner-Druckkopf entwickelt wurde. Mit diesem Toolboard können Sie die komplexe Verkabelung zwischen dem Hauptboard und dem Druckkopf durch nur vier Kabel ersetzen und das Layout optimieren. SB2040 ist kompatibel mit Stealthburner und Afterburner. Es integriert einen CAN-Transceiver, einen USB-Anschluss, TMC2240-Treiber, ADXL345-Beschleunigungssensor und drei steuerbare Lüfter sowie weitere Funktionen. Dieses Toolboard ist für die Firmware Klipper und RRF geeignet.

## Produkteigenschaften

    * Vorhandener Boot-Button, Nutzer können über TypeC in den Boot-Modus wechseln, um die Firmware zu aktualisieren
    * Unterstützt Klipper-Firmware
    * Temperatursensorunterstützung: Integrierter MAX31865-Chip, kompatibel mit PT100
    * Integrierter ADXL345-Beschleunigungssensor
    * Vorhandene RGB-Schnittstelle, integrierter 2240-Treiber, unterstützt verschiedene Sensoren für die Nivellierung, wie Klicky, Voron Tap, PL08n usw.
    * Unterstützt Spannung von 12-24V, Eingangsstrom von 15A, vergrößerte Heizstab-Anschlüsse, Nennstrom 10A


## Produktspezifikationen


    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Anschluss-Schnittstellen verwenden XT30(2+2)-Stecker, unterstützen Strom von 15A, Spitzenwert 30A
    * Firmware: Klipper
    * Eingangsspannung: DC12V-24V
    * Logikspannung: DC3.3V
    * Integrierte Sensoren: ADXL345 und optional MAX31865
    * Heizstab-Schnittstelle: Heizstab (HEAT0)
    * Erweiterungsschnittstellen: Steuerbare Lüfter*3, RGB*1, Endschalter*3, Heizrohr*1, Thermistor*1
    * Lüfter-Schnittstelle: Drei steuerbare Lüfter
    * Motortreiber: Integrierter TMC2240
