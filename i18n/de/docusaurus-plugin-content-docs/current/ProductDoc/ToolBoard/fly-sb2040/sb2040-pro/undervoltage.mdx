# Lösungsmethoden für Unterspannung des Treibers

## Methode eins

    :::danger

    * Stromversorgung austauschen
    * Netzteilfilter hinzufügen
    * Motor austauschen
    * Motorhülle an die GND der Werkzeugplatte anschließen
    * In der Extruder-Treiber-Konfiguration `driver_TPFD: 0` hinzufügen

    :::

## Methode zwei
    * Über SSH mit dem Hostcomputer verbinden
    * Folgenden Befehl eingeben
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Nach einem Klipper-Update könnte dieser Schritt rückgängig gemacht werden
    :::
