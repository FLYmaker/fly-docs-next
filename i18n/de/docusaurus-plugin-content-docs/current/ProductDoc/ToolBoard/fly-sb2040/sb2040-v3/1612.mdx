# Eddy Nutzung

    :::info

    * Bitte beachten Sie, dass für das Scannen des Bettes eine separate Z-Achsenbegrenzung empfohlen wird, nicht die Verwendung des Bettes als Z-Achsenbegrenzung.
    * Beim Installieren von Eddy sollte der Abstand zwischen der Unterseite und der Düse mindestens 1-2 mm betragen.
    * Bei der Kalibrierung der Eddy-Frequenz sollte die Spule PCB mindestens 5 mm vom Heizbett entfernt sein.
    * Bitte stellen Sie sicher, dass das gesamte Heizbett bei der Testung auf Raumtemperatur ist und das Heizbett während der Testung nicht eingeschaltet wird.

    :::

## Referenzkonfiguration und Nutzungshinweise

    ### Scankonfiguration

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Bitte beachten Sie, dass derzeit kein externes 1612-Modul verfügbar ist, diese Konfiguration kann derzeit nicht verwendet werden.
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Temperaturkompensation-Konfiguration

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh-Konfiguration

    * Der Parameter `horizontal_move_z` in [bed_mesh] sollte auf 2 gesetzt werden, damit Eddy beim Scannen des Bettes so nah wie möglich an der Plattform bleibt.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
