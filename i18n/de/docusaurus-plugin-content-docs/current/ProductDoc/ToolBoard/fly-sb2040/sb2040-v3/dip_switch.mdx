# Einführung in die Onboard-DIP-Schalter

:::warning Hinweise

Aufgrund der hohen Auflösung der Bilder können sie auf der Webseite nicht klar betrachtet werden. Bitte **bewegen Sie den Mauszeiger über das Anschlussdiagramm**, **klicken Sie mit der rechten Maustaste** und wählen Sie **Bild speichern unter**, um das Bild auf den Desktop zu speichern und dann zu betrachten!!!

:::
