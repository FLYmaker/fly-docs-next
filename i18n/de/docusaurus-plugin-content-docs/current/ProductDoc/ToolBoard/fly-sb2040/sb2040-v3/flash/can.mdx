        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />

    * Geben Sie `make -j4` ein, um die Programmierung zu starten, dies dauert etwas länger
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Bei erfolgreicher Programmierung wird am Ende folgendes ausgegeben:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware-Programmierung

    * Vor dem Programmieren der Firmware, verbinden Sie das Werkzeugboard mit einem UTOC oder einer Hauptplatine, die mit der Brückenfirmware programmiert wurde, über ein XT30-Datenkabel. Hier ist das Referenz-Verkabelungsdiagramm
    * Achten Sie darauf, dass der DIP-Schalter in der richtigen Position ist
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Stellen Sie sicher, dass CAN H und CAN L bei ausgeschalteter Stromversorgung mit einem Multimeter gemessen einen Widerstand von etwa `60Ω` haben. Wenn es `120Ω` ist, muss der `120Ω`-DIP-Schalter geöffnet werden. Bei etwa `40Ω` muss der Schalter geschlossen werden.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.png').default} size="100%" align="center" />
