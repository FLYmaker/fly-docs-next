<Tabs>
    <TabItem value="can" label="CANBus UUID abrufen">

    * Verbinden Sie das Werkzeugbrett mit UTOC oder einer Hauptplatine mit Bridge-Firmware vor der ID-Suche mit einem XT30-Datenkabel, unten ist das Bezugsschaltbild
    * Achten Sie darauf, ob die DIP-Schalter auf die richtige Position gestellt sind
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Stellen Sie sicher, dass CAN H und CAN L bei ausgeschalteter Stromversorgung mit einem Multimeter gemessen einen Widerstand von etwa `60Ω` aufweisen. Wenn es `120Ω` sind, öffnen Sie den `120Ω`-Schalter, wenn es etwa `40Ω` sind, schalten Sie den Schalter aus
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.png').default} size="100%" align="center" />   
        <GetCanbusUUID />

    </TabItem>
    <TabItem value="rs232" label="RS232 ID abrufen">

    * Verbinden Sie das Werkzeugbrett mit UTOR oder G2T vor der ID-Suche mit einem XT30-Datenkabel, unten ist das Bezugsschaltbild
    * Achten Sie darauf, ob die DIP-Schalter auf die richtige Position gestellt sind
        <ImageView image={require('@site/docs/General/get-id/img/utor.png').default} size="80%" align="left" />
