# Firmware Flash Anleitung

    * SB2040 V3 kann über RS232 und CAN mit der oberen Maschine verbunden werden
    * RS232 und CAN müssen mit `XT30(2+2)-F` verbunden werden
    * Unten ist das Datenkabel-Stil von `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Beachten Sie

* Das Motherboard-Firmware-Flash erfordert das Brennen von Klipper über Katapult
* Wenn das Motherboard bereits mit Katapult gebrannt wurde, ist das Brennen der Katapult-Firmware am Ende nicht erforderlich, das dient nur als Vorsichtsmaßnahme, falls das Motherboard die Katapult-Firmware nicht erfolgreich gebrannt hat.
* Man kann zweimal schnell die Reset-Taste drücken und schauen, ob eine LED auf dem Motherboard blinkt, wenn ja, bedeutet das, dass das Motherboard in die Katapult-Firmware gegangen ist.
:::

## Wie entscheidet man sich für die richtige Firmware

    * SB2040 V3 kann auf folgende Weise mit der oberen Maschine verbunden werden
    * <Button variant="contained" disableElevation href="./can">CAN-Firmware brennen</Button>
        * Vorteile: Wenn `Katapult`-Firmware gebrannt ist, kann die Firmware ohne Berührung des Motherboards aktualisiert werden, unterstützt auch Bus-Verbindungen
        * Nachteile: Kann bei gleichzeitiger Nutzung mit `KlipperScreen` zu Homing-Timeouts und anderen Fehlern führen.
        * Hinweis: Wenn Sie CAN verwenden möchten, stellen Sie sicher, dass die obere Maschine CAN unterstützt und benötigen UTOC oder ein Motherboard mit gebrannter Brücken-Firmware.
    * <Button variant="contained" disableElevation href="./rs232">RS232-Firmware brennen</Button>
        * Vorteile: Benötigt keine spezielle Software, zuverlässige Kommunikation, stabil
        * Nachteile: Firmware-Updates erfordern manuellen DFU-Eintritt oder Doppelklicken auf RST, um in `Katapult` zu gelangen, um die Firmware zu aktualisieren.
        * Hinweis: Erfordert G2T oder UTOR für den Gebrauch
