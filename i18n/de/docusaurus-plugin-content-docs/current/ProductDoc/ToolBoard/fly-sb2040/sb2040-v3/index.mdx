# 1. Produkteinführung

    FLY SB2040 V3 ist eine Werkzeugplatte, die von Guangzhou Mellow Electronics Technology Co., Ltd. speziell für den VORON Stealthburber-Druckkopf entwickelt wurde. Mit dieser Werkzeugplatte können die komplizierten Verbindungen zwischen der Hauptplatine und dem Druckkopf durch vier Drähte ersetzt werden, wodurch das Layout optimiert wird. SB2040 eignet sich sowohl für stealthburner als auch afterburner. Sie integriert einen CAN-Transceiver, RS232-Transceiver, USB-Anschluss, TMC2209-Treiber, LIS2DW-Beschleunigungssensor, drei steuerbare Lüfter und andere Funktionen. Diese Werkzeugplatte ist kompatibel mit Klipper- und RRF-Firmware.

## 1.1 Produkteigenschaften

    * Vorhandener Boot-Button, Benutzer können über TypeC den Boot-Modus betreten, um die Firmware zu aktualisieren
    * Unterstützung für Klipper-Firmware, RRF-Firmware
    * Unterstützt CAN-Modus, RS232-Modus, RRF CAN-Modus
    * Im RS232-Modus bietet es höhere Übertragungsgeschwindigkeiten und stärkere Stabilität. Im CAN-Modus wird die CAN-Erweiterung unterstützt, CAN-Erweiterungsanschluss ist vorhanden
    * Temperatursensorunterstützung: Integrierter MAX31865-Chip, kompatibel mit PT100- und PT1000-Temperatursensoren.
    * Integrierter LIS2DW-Beschleunigungssensor und LDC1612-Sensor
    * Vorhandenes RGB-Anschluss, integrierter 2209-Treiber, unterstützt verschiedene Nivellierungssensoren, wie klicky, Voron Tap, PL08n etc.
    * Unterstützung für 12-24V Spannung, 15A Stromaufnahme, vergrößerter Heizstab-Anschluss, Nennstrom 10A

## 1.2 Neue Funktionen

    * Unterstützung für das Umschalten zwischen Klipper CAN-Modus, RS232-Modus, RRF CAN-Modus
    * Neuer RS232-Modus mit höherer Übertragungsgeschwindigkeit, stärkerer Stabilität und einfacherer Nutzung beim Debuggen
    * Neuer RRF CAN-Modus, der in eine RRF-Werkzeugplatte umgeschaltet werden kann, um die Anwendbarkeit zu erhöhen
    * Neuer CAN-Erweiterungsanschluss für einfache Erweiterung von CAN-Geräten
    * Neue Endwiderstand-Schalter, der 120R Endwiderstand kann durch Umschalten ausgewählt werden, ohne Jumper
    * Neue 1-Meter-Strom-/Signalverbindungskabel, die eine Zwillingslitze verwenden, für besseren Störwiderstand

## 1.3 Produktparameter


    * MCU：Raspberry Pi rp2040, Dual Core ARM Cortex-M0+@133MHz
    * Anschlussinterface verwendet XT30(2+2)-Steckverbinder, unterstützt 15A Strom, Spitzenwert 30A
    * Firmware: Reprap/Klipper
    * Eingangsspannung: DC12V-24V
    * Logikspannung: DC3.3V
    * Integrierter Sensor: LIS2DW und optional ldc1612 und max31865
    * Heizstab-Anschluss: Heizstab (HEAT0)
    * Erweiterungsanschluss: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN-Anschluss
    * Lüfteranschluss: Ein DC-Lüfter, zwei DC-Lüfter oder PWM-Lüfter
    * Motortreiber: Integrierter TMC2209
    * Temperatursensoren: Ein 100K NTC oder PT1000 (TH0), ein PT100/PT1000 optional
