# Firmware-Flash-Anleitung

    * Der SB2040 kann über eine CAN-Verbindung mit dem Host-PC verbunden werden
    * CAN erfordert die Verwendung von `XT30(2+2)-F` zur Verbindung
    * Unten ist das Aussehen des `XT30(2+2)-F` Datenkabels
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Beachten Sie
* Das Flashen der Hauptplatinenfirmware erfordert das Flashen von Klipper über Katapult
* Wenn sicher ist, dass die Hauptplatine bereits Katapult geflasht hat, ist das Flashen der Katapult-Firmware am Ende nicht nötig, dies dient nur als Sicherheitsmaßnahme für den Fall, dass das Flashen der Katapult-Firmware auf der Hauptplatine nicht erfolgreich war
* Man kann schnell zweimal auf Reset drücken und schauen, ob eine LED auf der Hauptplatine aufblinkt, wenn ja, bedeutet dies, dass die Hauptplatine in die Katapult-Firmware übergegangen ist
:::

## Wie man ermittelt, welche Art von Firmware benötigt wird

    * Der SB2040 kann nur über eine CAN-Verbindung mit dem Host-PC verbunden werden
    * <Button variant="contained" disableElevation href="./can">CAN-Firmware-Flash</Button>
        * Vorteil: Wenn die `Katapult`-Firmware geflasht wurde, kann die Firmware ohne Berührung der Hauptplatine aktualisiert werden und unterstützt Bus-Verbindungen
        * Nachteil: Kann bei gleichzeitiger Verwendung mit `KlipperScreen` zu Home-Timeout-Fehlern führen.
        * Hinweis: Wenn CAN verwendet werden soll, stellen Sie sicher, dass der Host-PC CAN unterstützt und benötigen Sie entweder ein UTOC oder eine Hauptplatine mit gebrückter Firmware
