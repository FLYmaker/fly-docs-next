## Produkteinführung

    FLY SB2040 ist ein Düsentoolboard, das von Guangzhou Magnesium Logic Electronics Co., Ltd. für den VORON Stealthburner-Druckkopf entwickelt wurde. Mit diesem Toolboard können vier Kabel anstelle der komplizierten Verkabelung zwischen Hauptplatine und Druckkopf verwendet werden, um das Layout zu optimieren. SB2040 ist für 36 Schrittmotoren geeignet und integriert einen CAN-Transceiver, einen USB-Anschluss, einen TMC2209-Treiber, einen ADXL345-Beschleunigungssensor, zwei steuerbare Lüfter und weitere Funktionen. Dieses Toolboard ist für die Klipper-Firmware geeignet.

    * Unterstützung für CAN-Bus-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * Integrierter TMC2209-Treiber im UART-Modus, kann für den Extruder verwendet werden
    * Integrierter ADXL345-Beschleunigungssensor
    * Unterstützung für Nivellierungssensoren wie BLTOUCH, PL08N, integrierter XY-Endschalter
    * Unterstützung für Eingangsspannung von 12-24V 
    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Zusätzliche Unterstützung für 4.7K-Ohm-Widerstand (für NTC-Thermistor) oder 1K-Ohm-Widerstand (für PT1000) Umschaltung
    * CAN-Schnittstelle verwendet XT30 (2+2), unterstützt 15A Strom, Spitzenwert 30A
    * Zusätzlicher integrierter NTC100k-Widerstand zur Messung der Kammer-Temperatur
    * Hinzufügen der INT1- und INT2-Anschlüsse des ADXL345 an den MCU für einige Benutzer zur Nivellierung
    * Unterstützung für High-Voltage-Eingangsbegrenzung
    * Inklusive 3 Meter CAN-Verbindungskabel, um Benutzern das Problem zu ersparen, dass sie keine korrekten Terminals pressen können, was zu Signalverlust führen kann
    * Unterstützung für 2/3/4-Linie-Lüfter
    * Geteiltes Design, Stealthburner lässt sich einfach demontieren
    * Kompatibel mit Hartks Stealthburner Toolhead Board (die Lüfterplatine muss nicht ausgetauscht werden)
    * Anschlüsse: Steuerbare Lüfter *3, RGB *1, Endschalter *3, Heizrohr *1, Thermistor *1

## Produktparameter

    * Unterstützung für den Wechsel zwischen Klipper CAN-Modus, RS232-Modus, RRFCAN-Modus
    * Verbindungsanschluss verwendet XT30(2+2)-Steckverbinder, unterstützt 15A Strom, Spitzenwert 30A
    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Eingangsspannung: DC12-24V
    * Maximale Ausgangsstromstärke des Heizstabs: 5A
    * Maximale Ausgangsstromstärke des Lüfteranschlusses: 1A
