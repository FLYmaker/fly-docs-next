            4. Firmware flashen

            :::tip

            Es muss nur eine der Firmwares geflasht werden

            :::

            * Hier sind die Flash-Befehle für die 500K CAN-Geschwindigkeit
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Hier sind die Flash-Befehle für die 1M CAN-Geschwindigkeit
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. Wenn die Meldung ``File downloaded successfully`` wie im Bild mit dem Pfeil angezeigt wird, war das Flashen erfolgreich

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.png').default} size="100%" align="center" />

            :::tip

            Nach Abschluss des Flashvorgangs muss die Hauptplatine einmal vollständig vom Strom getrennt werden.
