    * Drücke die `Q`-Taste, es erscheint **Konfiguration speichern**, dann drücke die `Y`-Taste
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Jetzt sollte die Konfiguration gespeichert und du zurück auf der Kommandozeile sein
    * Gib `make -j4` ein, um die Kompilierung zu starten, das dauert etwas
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn am Ende der folgende Inhalt ausgegeben wird, war die Kompilierung erfolgreich

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware brennen

    * Vor dem Brennen der Firmware, verbinde das Toolboard mit dem UTOC oder einem Mainboard mit Bridge-Firmware über ein XT30-Datenkabel, hier ist ein Beispiel für die Verkabelung
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.png').default} size="80%" align="left" />
    * Stelle sicher, dass bei abgeschaltetem Strom der Widerstand zwischen CAN H und CAN L mit einem Multimeter gemessen etwa `60Ω` beträgt. Wenn es `120Ω` sind, muss ein `120Ω`-Jumper angeschlossen werden; bei etwa `40Ω` muss der Jumper entfernt werden
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.png').default} size="100%" align="center" />
