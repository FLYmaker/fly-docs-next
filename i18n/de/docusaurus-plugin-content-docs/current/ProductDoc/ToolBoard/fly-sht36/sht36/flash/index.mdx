# Firmware-Brenninstruktionen

    * SHT36/42 können über CAN-Verbindung mit dem Host-Computer verbunden werden
    * CAN erfordert das Drücken der MX3.0 Metallklemmen
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.png').default} size="80%" align="left" />

## Wie erkennt man, welche Firmware benötigt wird

    * SHT36/42 können über CAM mit dem Host-Computer verbunden werden
    * <Button variant="contained" disableElevation href="./can">CAN-Firmware-Brenninstruktionen</Button>
        * Vorteile: Bei gebrannter `Katapult`-Firmware kann die Firmware ohne Berührung der Hauptplatine aktualisiert werden und unterstützt Bus-Verbindungen
        * Nachteile: Kann bei der gemeinsamen Nutzung mit `KlipperScreen` zu Fehlern wie Home-Timeout führen.
        * Hinweis: Wenn CAN benötigt wird, stellen Sie sicher, dass der Host-Computer CAN unterstützt und verwenden Sie UTOC oder eine Hauptplatine mit gebrannter Brückenfirmware
