# Produkteinführung

    FLY SHT36/42 ist eine von der Guangzhou Mellow Electronic Technology Co., Ltd. für Extruder-Köpfe von 3D-Druckern entwickelte Düsenwerkzeugplatte. Mit dieser Werkzeugplatte können vier Leitungen anstelle der komplexen Verkabelung zwischen Hauptplatine und Druckkopf verwendet werden, um das Layout zu optimieren. SHT36 eignet sich für 36 Schrittmotoren, SHT42 für 42 Schrittmotoren. Sie integriert einen CAN-Transceiver, USB-Anschluss, TMC2209-Treiber, ADXL345-Beschleunigungssensor, zwei steuerbare Lüfter und andere Funktionen. Diese Werkzeugplatte ist für die Klipper-Firmware geeignet.

    * Extrem kompakte Bauweise, 6-schichtiges Platinendesign
    * Unterstützt CAN-Bus-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * Integrierter TMC2209-Treiber, verwendet UART-Modus, geeignet für Extruder
    * Integrierter ADXL345-Beschleunigungssensor
    * Unterstützt PT100 (MAX31865)
    * Austauschbares Lüfter-MOS-Design
    * Unterstützt BLTOUCH, PL08N und andere Nivellierungssensoren, integrierter XY-Grenzanzeigeschalter
    * Unterstützt 12-24V Eingangsspannung 
    * MCU: STM32F072
    * Kommunikation: CANBUS, USB
    * Schnittstellen: steuerbare Lüfter*2, RGB*1, Endschalter*3, Probe*1, Servo*1, Heizstab*1, Thermist*1, PT100*1

## Produktspezifikationen

    1. Abmessungen: 51,27mm*36,26mm (SHT36), 42mm*42mm (SHT42)
    2. Montageabmessungen: Lochabstand 43,84mm, M3-Montagebohrungen
    3. Prozessor: STM32F072C8T6
    4. Eingangsspannung: DC12-24V 5A
    5. Maximaler Ausgangsstrom des Heizstabs: 5A
    6. Maximaler Ausgangsstrom der Lüfter-Schnittstelle: 1A
