# Verwendung des Beschleunigungssensors


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Klipper aktualisieren

    * Klipper unterstützt seit dem 13. November 2024 den LIS3DH-Beschleunigungssensor. Wenn Sie den LIS3DH-Beschleunigungssensor verwenden möchten, müssen Sie Klipper aktualisieren.
    * Nach dem Herstellen einer Netzwerkverbindung zum Hostcomputer geben Sie den folgenden Code ein (die Aktualisierung kann aufgrund von Netzwerkproblemen fehlschlagen):
        ```bash
        cd ~/klipper 
        git pull
        ```
## Konfiguration hinzufügen


    ```cfg
    #####################################################################
    # LIS3DH-Beschleunigungssensor-Konfiguration (bei Bedarf aktivieren)
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  # Hier setzen Sie die Koordinaten, an denen sich der Druckkopf bei der Resonanzmessung befindet


```
