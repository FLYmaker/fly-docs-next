# Katapult-Firmware

## Beschreibung der Katapult-Firmware

    <CANBOOT />

    :::tip

    * SHT36 LIS3DH unterstützt CAN
    * Hier wird nur die CAN-Firmware in 500K und 1M Bitrate bereitgestellt
    :::

## Betreten des Programmiermodus

    :::tip

    Der markierte Bereich ist der `BOOT-Knopf`

    :::

    * Bei vollständig ausgeschaltetem Hauptboard, kurzschließen Sie den BOOT-Knopf und verbinden Sie dann das Datenkabel mit dem Hauptboard und Ihrem Gerät
    * Dann lösen Sie die Kurzschlussverbindung des BOOT-Knopfes
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/boot.png').default} size="50%" align="center" />

            4. Firmware programmieren

            :::tip

            Sie müssen nur eine Firmware programmieren

            :::

            * Hier ist der Befehl zum Programmieren der Firmware mit 500K CAN-Bitrate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_500K.uf2
                ```
                        
                * Hier ist der Befehl zum Programmieren der Firmware mit 1M CAN-Bitrate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_1M.uf2
                ```


            5. Wenn die folgende Meldung erscheint, war das Programmieren erfolgreich

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            Nach Abschluss des Programmierens muss das Hauptboard einmal vollständig von der Stromversorgung getrennt werden.

            :::
