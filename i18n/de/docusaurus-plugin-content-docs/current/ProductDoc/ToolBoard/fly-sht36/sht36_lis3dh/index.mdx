# Produkteinführung

    FLY-SHT36-LIS3DH ist eine Düsentoolplatte, die von Guangzhou Mellow Electronic Technology Co., Ltd. speziell für 3D-Drucker-Düsen entwickelt wurde. Mit dieser Toolplatte können vier Leitungen anstelle der ursprünglichen komplizierten Verkabelung zwischen Hauptplatine und Druckkopf verwendet werden, um das Layout zu optimieren. FLY-SHT36-LIS3DH eignet sich für 36-Schrittmotoren und integriert einen CAN-Transceiver, USB-Anschluss, TMC2209-Treiber, LIS3DH-Beschleunigungssensor, zwei steuerbare Lüfter und andere Funktionen. Diese Toolplatte ist für die Klipper-Firmware geeignet.

    * Extrem kompaktes Design, 6-Schichten-Platinenkonstruktion
    * Unterstützt CAN-Bus-Verbindung, stabilere Datenübertragung, geringere Latenz, stabilere Verbindung
    * Eingebauter TMC2209-Treiber im UART-Modus, kann für Extruder verwendet werden
    * Eingebauter ADXL345-Beschleunigungssensor
    * Unterstützt PT100 (MAX31865)
    * Austauschbares Lüfter-MOS-Design
    * Unterstützt BLTOUCH, PL08N und andere Nivellierungssensoren, integrierter XY-Endschalter
    * Unterstützt Eingangsspannung von 12-24V 
    * MCU: STM32F072
    * Kommunikation: CANBUS, USB
