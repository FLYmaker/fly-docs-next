## Auf Flaschenmodus wechseln

    :::tip

    Der eingerahmte Bereich ist die `BOOT-Taste`

    :::

    * Schalten Sie die Hauptplatine vollständig ab, kurzschließen Sie die BOOT-Taste und verbinden Sie dann das Datenkabel mit der Hauptplatine und Ihrem Gerät
    * Anschließend entfernen Sie die Kurzschlussverbindung von BOOT

            4. Firmware flashen

            :::tip

            Sie müssen nur eine der Firmwares flashen

            :::

            * Hier ist der Flash-Befehl für die Firmware mit 500K CAN-Bitrate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Hier ist der Flash-Befehl für die Firmware mit 1M CAN-Bitrate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Wenn die folgende Nachricht erscheint, war das Flashen erfolgreich

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            Nach Abschluss des Flash-Prozesses müssen Sie die Hauptplatine einmal vollständig vom Strom trennen.

            :::
