# Produkteinführung

    FLY-SHT36-PRO ist eine Düsenwerkzeugplatte von Guangzhou Magnesium Electron Technology Co., Ltd., die speziell für Extruder-Köpfe von 3D-Druckern entwickelt wurde. Mit dieser Werkzeugplatte können vier Leitungen anstelle der komplexen Verkabelung zwischen Hauptplatine und Druckkopf verwendet werden, um das Layout zu optimieren. FLY-SHT36-PRO ist für 36 Schrittmotoren geeignet und integriert einen CAN-Transceiver, USB-Anschluss, TMC5160-Treiber, ADXL345-Beschleunigungssensor, zwei steuerbare Lüfter und andere Funktionen.

    * Extrem kompaktes Design mit 6 Lagen
    * Unterstützt CAN-Bus-Verbindung für stabilere Datenübertragung, geringere Latenz und stabilere Verbindung
    * Eingebauter TMC5160 für Extruder
    * Eingebauter ADXL345-Beschleunigungssensor
    * Unterstützung für PT100 (MAX31865)
    * Austauschbarer Lüfter-MOS-Design
    * Unterstützt BLTOUCH, PL08N und andere Nivellierungssensoren, integrierte XY-Endschalter
    * Unterstützt Eingangsspannung von 12-24V
    * MCU: STM32F072
    * Kommunikation: CANBUS, USB

## Produktspezifikationen

    1. Unterstützt Wechsel zwischen Klipper CAN-Modus, RS232-Modus und RRFCAN-Modus
    2. Verbindungsschnittstelle verwendet XT30(2+2) Stecker, unterstützt 15A Strom, Spitzenwert 30A
    3. MCU: Raspberry Pi rp2040, Dual Core ARM Cortex-M0+@133MHz
    4. Eingangsspannung: DC12-24V 5A
    5. Maximale Ausgangsstromstärke des Heizstabs: 5A
    6. Maximale Ausgangsstromstärke des Lüfteranschlusses: 1A
