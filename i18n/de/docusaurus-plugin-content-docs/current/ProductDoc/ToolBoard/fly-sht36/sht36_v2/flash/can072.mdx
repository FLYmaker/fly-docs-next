        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware programmieren

    * Verbinden Sie das Werkzeugboard vor der Firmwareprogrammierung mit dem UTOC oder einer Hauptplatine, die eine Brückenfirmware installiert hat, über ein XT30-Datenkabel. Unten finden Sie eine Referenzverdrahtung
    * Achten Sie darauf, ob die Jumper auf der richtigen Position stehen
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.png').default} size="80%" align="left" />
    * Vergewissern Sie sich, dass der Widerstand zwischen CAN H und CAN L bei ausgeschaltetem Strom mit einem Multimeter etwa `60Ω` beträgt. Wenn der Widerstand `120Ω` beträgt, müssen Sie den `120Ω`-Jumper anbringen, wenn er etwa `40Ω` beträgt, muss der Jumper entfernt werden
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.png').default} size="50%" align="center" />
