# Firmware-Flashen Anleitung

    * Firmware-Kompilierung und -Flashen erfordern die Verwendung einer Host-Software
    * SHT36 V2 kann über eine CAN-Verbindung mit der Host-Software verbunden werden
    * CAN erfordert die Verwendung von `XT30(2+2)-F` zur Verbindung
    * Unten ist das Aussehen des `XT30(2+2)-F` Datenkabels
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Beachten Sie
* Das Flashen der Hauptplatinen-Firmware erfordert das Flashen von Klipper über Katapult
* Wenn Sie sicher sind, dass die Hauptplatine bereits Katapult geflasht hat, ist das letzte Flashen der Katapult-Firmware nicht nötig, das dient nur als Vorsichtsmaßnahme, falls das Flashen der Katapult-Firmware auf der Hauptplatine nicht erfolgreich war
* Sie können schnell zweimal auf Reset drücken und dann schauen, ob auf der Hauptplatine eine LED aufleuchtet, wenn ja, bedeutet das, dass die Hauptplatine in die Katapult-Firmware eingetreten ist
:::
