# Eddy Nutzung

    :::info

    * Bitte beachten Sie, dass es empfohlen wird, eine separate Z-Endschaltung für das Scannen zu verwenden und nicht das Scannen als Z-Endschaltung zu nutzen
    * Beim Installieren von Eddy muss der Abstand zwischen dem Boden und der Düse mindestens 1-2 mm betragen
    * Beim Kalibrieren der Eddy-Frequenz muss die Spule PCB mindestens 5 mm von der Heizplatte entfernt sein
    * Bitte stellen Sie sicher, dass die gesamte Heizplatte bei der Überprüfung auf Raumtemperatur ist und die Heizplatte während des Tests nicht eingeschaltet ist

    :::

## Referenzkonfiguration und Nutzungsanleitung

    ### Scankonfiguration

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Bitte beachten Sie, dass es derzeit kein externes 1612-Modul gibt, diese Konfiguration ist momentan nicht verwendbar
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Temperaturkompensation

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh-Konfiguration

    * In [bed_mesh] sollte der Parameter `horizontal_move_z` auf 2 gesetzt werden, damit Eddy bei der Plattformabtastung möglichst nah an der Plattform bleibt

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
