# Einführung in die Onboard-DIP-Schalter

:::warning Wichtiger Hinweis

Aufgrund der hohen Auflösung der Bilder können sie auf der Webseite nicht klar angezeigt werden. Bitte **bewegen Sie den Mauszeiger über das Schaltbild**, **klicken Sie mit der rechten Maustaste** und wählen Sie **Bild speichern unter**, um das Bild auf den Desktop zu speichern und dort zu betrachten!!!

:::
