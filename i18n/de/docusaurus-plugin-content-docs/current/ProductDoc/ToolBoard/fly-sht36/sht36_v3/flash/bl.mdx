# Katapult-Firmware

## Katapult-Firmware-Erklärung

    <CANBOOT />

    :::tip

    * SHT36_V3 unterstützt CAN und RS232
    * Hier bieten wir nur CAN-Firmware in 500K und 1M Geschwindigkeit an
    :::

    :::tip

    Der Pfeil zeigt auf den `BOOT-Knopf`

    :::

            4. Firmware flashen

            :::tip

            Es muss nur eine der Firmwares geflasht werden

            :::

            * Hier ist der Befehl zum Flashen der 500K CAN-Geschwindigkeitsfirmware
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Hier ist der Befehl zum Flashen der 1M CAN-Geschwindigkeitsfirmware
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. Bei dieser Meldung wurde das Flashen erfolgreich abgeschlossen
