# RS232-Firmware-Kompilierung

## Kompilierung starten

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Wählen Sie GPIO-Pins zum Setzen beim Start des Mikrocontrollers, Eingabe: !gpio5 Bitte beachten Sie, dass ! großgeschrieben ist
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.png').default} size="100%" align="center" />
    * Drücken Sie die `Q`-Taste, es erscheint **Save configuration**, dann drücken Sie die `Y`-Taste
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Die Konfiguration sollte jetzt gespeichert und das Terminal verlassen sein
    * Geben Sie `make -j4` ein, um mit der Kompilierung zu beginnen, das dauert etwas
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn die Kompilierung erfolgreich war, wird folgender Text angezeigt:
    
        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```



## Flashing-Modus betreten

    :::tip

    Der Pfeil zeigt auf die `BOOT`-Taste

    :::

    * Wenn das Hauptboard vollständig ausgeschaltet ist, drücken Sie die BOOT-Taste und verbinden Sie dann das Datenkabel mit dem Hauptboard und Ihrem Gerät
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.png').default} size="100%" align="center" />
