# 1. Produkteinführung

FLY SHT36 V3 ist eine von Guangzhou Mellow Electronics Technology Co., Ltd. entwickelte Düsentoolplatte für 3D-Drucker-Extruder. Mit dieser Toolplatte können vier Drähte anstelle der komplexen Verkabelung zwischen Hauptplatine und Druckkopf verwendet werden, um das Layout zu optimieren. SHT36 V3 ist für 36 Schrittmotoren geeignet und integriert einen CAN-Transceiver, RS232-Transceiver, TypeC-Anschluss, TMC2209-Treiber, LIS2DW-Beschleunigungssensor, 2 steuerbare Ventilatoren und weitere Funktionen. Diese Toolplatte ist kompatibel mit Klipper- und RRF-Firmware.

    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/sht36.png').default} size="100%" align="center" />


## 1.1 Produktmerkmale

    * Vorhandener Boot-Button, Benutzer können über TypeC in den Boot-Modus wechseln, um die Firmware zu aktualisieren
    * Unterstützt Klipper-Firmware, RRF-Firmware
    * Unterstützt CAN-Modus, RS232-Modus, RRF CAN-Modus
    * Im RS232-Modus bietet es eine höhere Übertragungsrate und Stabilität. Im CAN-Modus unterstützt es CAN-Erweiterungen, mit vorgesehener CAN-Erweiterungsschnittstelle
    * Temperatursensoren: integrierter MAX31865-Chip, kompatibel mit PT100- und PT1000-Temperatursensoren
    * Unterstützt zwei steuerbare Lüfter, wählbare Spannung, austauschbare Mos-Platine, erleichtert Wartung und Austausch, maximaler Ausgangsstrom bis zu 1A
    * Integrierter LIS2DW-Beschleunigungssensor und LDC1612-Sensor
    * Vorhandene RGB-Schnittstelle, integrierter 2209-Treiber, unterstützt verschiedene Ebnungssensoren, wie BLtouch, klicky, Voron Tap, PL08n usw.
    * Unterstützt 12-24V Spannung, 15A Eingangsstrom, vergrößerte Heizstift-Anschlüsse, Nennstrom 10A

## 1.2 Neue Funktionen

    * Unterstützt Wechsel zwischen Klipper CAN-Modus, RS232-Modus und RRF CAN-Modus
    * Neuer RS232-Modus mit höherer Übertragungsrate, besserer Stabilität und einfacherer Fehlersuche
    * Neuer RRF CAN-Modus, der in eine RRF-Toolplatte umgeschaltet werden kann, erhöht die Anwendbarkeit
    * Neue CAN-Erweiterungsschnittstelle, für einfache Erweiterung von CAN-Geräten
    * Neuer Schalter für Endwiderstand, der 120R Endwiderstand durch DIP-Schalter wählbar ist, ohne Jumper
    * Neue 1 Meter lange Strom-/Signalverbindungskabel, mit verdrillter Gestaltung für bessere Störfestigkeit

## 1.3 Produktparameter

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Verbindungsschnittstellen mit XT30(2+2)-Steckverbinder für 15A Strom, Spitzenwert 30A
    * Firmware: Reprap/Klipper
    * Eingangsspannung: DC12V-24V
    * Logikspannung: DC3.3V
    * Integrierte Sensoren: LIS2DW und LDC1612
    * Heizstift-Anschluss: Heizstift (HEAT0), 10A
    * Erweiterungsschnittstellen: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN-Erweiterungsport
    * Lüfteranschlüsse: zwei DC-steuerbare Lüfter
    * Motorantrieb: Integrierter TMC2209
    * Temperatursensoren: ein 100K NTC oder PT1000 (TH0), ein wählbarer PT100/PT1000
