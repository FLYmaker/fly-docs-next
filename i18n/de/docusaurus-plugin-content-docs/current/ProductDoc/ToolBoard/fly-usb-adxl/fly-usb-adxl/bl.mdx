# Katapult-Firmware

## Flash-Modus betreten

    :::tip

    Der umrahmte Bereich ist die `BOOT-Taste`.

    :::

    * Um in den Flash-Modus zu gelangen, muss das Hauptplatine vollständig vom Strom getrennt sein. Halte die BOOT-Taste gedrückt und schließe dann das Datenkabel an das Hauptplatine und dein Gerät an.
