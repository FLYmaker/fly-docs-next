# USB-Firmware-Kompilierung

    :::tip

    * USB-Beschleunigungssensoren werden mit Katapult-Firmware für USB-Verbindung ausgeliefert
    * Falls diese nicht geflasht sind, kann man sie selbst flashen
    * [Katapult-Firmware-Flash](./bl.mdx "Klicken Sie hier, um zu springen")

    :::

## Kompilierung beginnen

    <SSH />

    <MakeOperate/>

    <USB />
    * Wählen Sie GPIO-Pins zur Festlegung beim Start des Mikrocontrollers, Eingabe: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/usb.png').default} size="100%" align="center" />
    * Drücken Sie die `Q`-Taste, **Save configuration** erscheint, dann die `Y`-Taste drücken
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Nun sollte die Konfiguration gespeichert sein und Sie sollten zurück zur Kommandozeile gelangt sein
    * Geben Sie `make -j4` ein, um mit der Kompilierung zu beginnen, dies dauert etwas länger
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Wenn die Kompilierung erfolgreich war, wird folgender Text angezeigt:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
