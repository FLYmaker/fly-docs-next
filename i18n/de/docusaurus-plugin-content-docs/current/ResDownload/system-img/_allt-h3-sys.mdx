# {props.name} System Download

:::warning

SD-Karten Hinweise:

* Die Kapazität muss nicht kleiner als `16GB` oder größer als `128G` TF-Karte sein
* Die Übertragungsgeschwindigkeit der TF-Karte muss `class10` oder höher sein, es wird empfohlen, TF-Karten von Marken wie SanDisk zu verwenden

:::

:::danger Wichtiger Hinweis

* Aufgrund von Serverproblemen bitte nicht willkürlich das System oder Klipper aktualisieren
* Bitte das FLY OS System herunterladen, die Installation der Originalversion von Armbian ist aufgrund der hohen Schwierigkeit nicht empfohlen

:::

:::info Wichtiger Hinweis

* Beachten Sie, dass das heruntergeladene System entpackt werden muss, anstatt nur das Suffix zu ändern oder direkt zu brennen！！！
* Ohne Entpackung lässt sich das System nicht starten！！！

:::
