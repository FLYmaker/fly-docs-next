# {props.name} System herunterladen

:::warning

SD-Karten-Hinweise:

* Die Kapazität muss mindestens `16GB` und höchstens `128G` TF-Karte sein
* Die Übertragungsgeschwindigkeit der TF-Karte muss `class10` oder höher sein, es wird empfohlen, TF-Karten von Marken wie SanDisk zu verwenden

:::

:::danger Wichtig

* Aufgrund von Serverproblemen, bitte das System oder Klipper nicht willkürlich aktualisieren
* Bitte das FLY OS System herunterladen, die Originalversion von Armbian ist zu kompliziert und wird nicht empfohlen

:::

:::info Wichtiger Hinweis

* Achten Sie darauf, dass das heruntergeladene System entpackt werden muss, nicht einfach das Suffix ändern oder direkt aufzeichnen!!!
* Ohne Entpackung ist ein Start nicht möglich!!!

:::
