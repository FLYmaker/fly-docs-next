# Raspberry Pi Image

  :::warning

  Hinweise zur SD-Karte:

  * Die Kapazität muss nicht kleiner als `16GB` oder größer als `128G` TF-Karte sein
  * Die Übertragungsgeschwindigkeit der TF-Karte muss `class10` oder höher sein, es wird empfohlen, TF-Karten von Marken wie SanDisk zu verwenden

  :::

  :::danger Wichtig

  * Wegen Serverproblemen bitte nicht willkürlich das System oder Klipper aktualisieren
  * Für Raspberry Pi 3B/3B+/4B/CM4/5B geeignetes System-Image
  * Dieses Image basiert auf `bookworm` und installiert verschiedene Dienste
  * Dieses Image installiert und aktiviert standardmäßig den TFT-Bildschirm, wenn dieser nicht benötigt wird, muss manuell `/boot/config.txt` geändert werden

  :::

  :::info Wichtiger Hinweis

  * Bitte beachten Sie, dass das heruntergeladene System entpackt werden muss, nicht einfach das Suffix ändern oder direkt einbrennen!!!
  * Einbrennen ohne Entpacken führt zu einem nicht startenden System!!!

  :::
