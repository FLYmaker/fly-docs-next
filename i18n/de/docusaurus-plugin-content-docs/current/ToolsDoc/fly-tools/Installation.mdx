# FLY-Tools installieren

    1. Neuesten Klipper abrufen

    ```bash
    cd ~/klipper
    git pull
    ```

    * Falls Sie den lokalen Klipper-Quellcode geändert haben und das Abrufen fehlschlägt, verwenden Sie die folgenden Befehle. (Dieser Vorgang verwirft Ihre zuvor geänderten Dateien)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. Neuesten Katapult (ehemals Canboot) abrufen

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Fly-Tools-Dienst installieren

        * Der Installationsprozess hängt vom Git-Dienstleister ab. Falls der Download-Fortschritt hängt, drücken Sie `CTRL+C`, um abzubrechen und erneut zu versuchen
        * Github hat eine maximale Anzahl von 60 Anfragen pro Stunde, Gitee unbekannt
        * Inländische Benutzer sollten das Installationsskript über Gitee abrufen
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Installationsskript über Github abrufen

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
