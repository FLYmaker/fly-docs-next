# Einführung in FLY-Tools

    FLY-Tools ist ein spezielles Web-Debugging-Tool, das von Guangzhou Maglin Electronics Technology Co., Ltd. für Klipper entwickelt wurde. Es ist nicht nur für die Fly-Hosts (Systemeigene Software, keine Installation erforderlich) geeignet, sondern auch für die meisten Hosts auf dem Markt, die den Klipper-Dienst installiert haben.

    :::tip Hinweis

    * FLY-Tools ist nur für Linux-Systeme geeignet, auf denen Klipper installiert ist.

    :::
        

## Eigenschaften von FLY-Tools

    * Unterstützung für mehrere Sprachen, dunkles Thema
    * Unterstützung zum Abfragen und Ein-Klick-Kopieren von USB-Serien-IDs, USB-Geräte-IDs, CANbus-UUIDs, Video-Geräte-IDs
    * Unterstützung zum Ein- oder Ausschalten spezifizierter CAN-Geräte
    * Unterstützung zum Online-Ändern der CAN-Geräte-Rate und der Sendepuffergröße
    * Unterstützung zum Online-Kompilieren von Klipper-Firmware für FLY-Motherboards (Katapult-Firmware-Kompilierung wird angepasst)
    * Unterstützung zum Online-Flashen, unterstützt DFU, HID, CAN und andere Flash-Methoden
    * Unterstützung zum Herunterladen aller kompilierten Firmware-Dateien
    * Unterstützung zum Online-Generieren von Klipper-Lastdiagrammen
    * Unterstützung zum automatischen Messen und Generieren von Resonanzdiagrammen
    * Unterstützung für Web-SSH-Verbindungen
    * Unterstützung für Web-SSH (basierend auf [ttyd](https://github.com/tsl0922/ttyd))
    * Unterstützung für FLYOS Version 3.3.4, konfigurierbare Bildschirme, Web-Anzeigen usw.
