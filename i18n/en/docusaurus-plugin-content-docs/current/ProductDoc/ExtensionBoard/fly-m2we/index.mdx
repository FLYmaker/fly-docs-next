## Precautions

:::danger Precautions

* The M2WE comes pre-flashed with FLYOS and can be started directly
* The M2WE can only be used with FLY products and does not support other devices
* The antenna terminal of the M2WE is `IPEX1`
* The M2WE does not support dual-band (2.4G & 5G), Chinese characters, special characters, or hidden WIFI by default

:::
