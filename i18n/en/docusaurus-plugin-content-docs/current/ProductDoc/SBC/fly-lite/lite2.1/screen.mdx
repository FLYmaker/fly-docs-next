# Screen Wiring and Configuration
:::info

* lite2.1 requires SSH connection to the system, then enter the following command to use Fly Tools screen configuration
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools requires the host computer to be connected to the network, then enter the host computer's `IP+:9999` to access
* For example `192.168.6.110:9999`

:::
