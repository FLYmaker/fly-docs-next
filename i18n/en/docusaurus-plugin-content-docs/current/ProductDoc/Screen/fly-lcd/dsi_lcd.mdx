## Product Introduction

    * Screen Size: 120.7mm*76.0mm
    * Display Area: 108mm*66.2mm
    * Resolution: 800*480
    * Screen Type: IPS
    * Communication Method: DSI/Mipi
    * Onboard Interface: 15 pin DSI / 22pin Mipi
    * Operating Stability: 0°-60°C
    * Touch Type: Capacitive
    * Logic Voltage: 3.3V
    * Weight: 135.5g

        :::danger Notice

        * The DSI and CSI interfaces on the Raspberry Pi 5 have now been renamed to MIPI
        * The MIPI interface can be directly configured to use either a CSI camera or a DSI screen
        * Please pay attention to the orientation of the gold fingers

        :::

        * The Raspberry Pi 5 now has two MIPI interfaces, which are labeled 1 and 0 when viewed from the front

        * DSI Screen Connection


            * If using interface 1, configure the screen as follows:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * If using interface 0, configure the screen as follows:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
