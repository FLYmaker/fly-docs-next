# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3 supports CAN and RS232
    * Only CAN firmware with 500K and 1M rates are provided here
    :::

## Entering Flash Mode

    :::tip

    The left is the `RESET button`, the right is the `BOOT button`

    :::

    * There are two ways to enter flash mode
    * The first method: With the mainboard completely powered off, hold down the BOOT button, then connect the data line to the mainboard and your device
    * The second method: After connecting the data line to the mainboard and your device, hold down the BOOT button, then press the RST button, release the RST button, and finally release the BOOT button
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.png').default} size="100%" align="center" />
