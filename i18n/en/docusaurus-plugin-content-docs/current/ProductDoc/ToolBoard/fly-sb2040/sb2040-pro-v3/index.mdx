# 1. Product Introduction

    FLY SB2040 PRO V3 is a toolboard launched by Guangzhou Mellow Electronic Technology Co., Ltd. for the VORON Stealthburner printhead. Using this toolboard, you can replace the complex wiring between the mainboard and the printhead with just four wires, optimizing the layout. SB2040 is suitable for both stealthburner and afterburner, integrating a CAN transceiver, RS232 transceiver, USB port, TMC2240 driver, LIS2DW accelerometer, three controllable fans, and other features. This toolboard is compatible with Klipper and RRF firmware.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/img/sb2040.webp').default} size="100%" align="center" />


## 1.1 Product Features

    * Reserved Boot button, allowing users to enter Boot mode for firmware updates via TypeC
    * Supports Klipper firmware, RRF firmware
    * Supports CAN mode, RS232 mode, RRF CAN mode
    * In RS232 mode, it offers higher transmission rates and stronger stability. In CAN mode, it supports CAN expansion with reserved CAN expansion interfaces
    * Temperature sensor support: Onboard MAX31865 chip, compatible with PT100 and PT1000 temperature sensors.
    * Onboard LIS2DW accelerometer and LDC1612 sensor
    * Reserved RGB interface, onboard 2240 driver, supports various leveling sensors like klicky, Voron Tap, PL08n, etc.
    * Supports 12-24V voltage, 15A current input, with an enlarged heater port rated at 10A

## 1.2 New Features

    * Supports switching between Klipper CAN mode, RS232 mode, and RRFCAN mode
    * New RS232 mode offers higher transmission rates, stronger stability, and simpler debugging
    * New RRF CAN mode, can switch to RRF toolboard, enhancing compatibility
    * New CAN expansion interface, facilitating the expansion of CAN devices
    * New termination resistor toggle switch, the 120R termination resistor can be selected via toggle switch, no need for jumper wires
    * New 1-meter power/signal connection cable, with twisted pair design for better anti-interference performance

## 1.3 Product Specifications


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Connection interface uses XT30(2+2) connector supporting 15A current, peak 30A
    * Firmware: Reprap/Klipper
    * Input voltage: DC12V-24V
    * Logic voltage: DC3.3V
    * Onboard sensors: LIS2DW and optional ldc1612 and max31865
    * Heater interface: Heater (HEAT0)
    * Expansion interfaces: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN port
    * Fan interfaces: One DC fan, two DC or PWM fans
    * Motor driver: Onboard TMC2240
    * Temperature sensors: One 100K NTC or PT1000 (TH0), one PT100/PT1000 optional
