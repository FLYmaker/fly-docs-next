# Undervoltage Solution for Drivers

## Method One

    :::danger

    * Replace the power supply
    * Add a power filter
    * Replace the motor for diagnosis
    * Connect the motor casing to the tool board's GND
    * Add `driver_TPFD: 0` to the extruder driver configuration

    :::

## Method Two
    * Connect to the host via SSH
    * Enter the following command
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * This step may be reverted after updating Klipper
    :::
