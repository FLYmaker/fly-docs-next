## Katapult Firmware Description

## Entering Flashing Mode

    :::tip

    The one circled is the `BOOT button`

    :::

    * With the mainboard completely powered off, press and hold the BOOT button, then connect one end of the data transfer cable to the mainboard and the other to your device

            :::tip

            You only need to flash one of the firmware files

            :::

            * Below is the command to flash the firmware with a 500K CAN rate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * Below is the command to flash the firmware with a 1M CAN rate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. If the following message appears, the firmware has been successfully flashed
