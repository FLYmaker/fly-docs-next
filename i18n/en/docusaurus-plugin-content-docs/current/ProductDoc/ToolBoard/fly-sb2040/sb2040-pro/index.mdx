# Product Introduction

    FLY SB2040 PRO is a tool board launched by Guangzhou Mellow Electronics Co., Ltd. for the VORON Stealthburner printhead. By using this tool board, you can replace the complex wiring between the mainboard and the printhead with just four wires, optimizing the layout. The SB2040 is suitable for both stealthburner and afterburner, integrating features such as CAN transceiver, USB port, TMC2240 driver, ADXL345 accelerometer, and three controllable fans. This tool board is compatible with Klipper and RRF firmware.

## Product Features

    * Includes a Boot button, allowing users to enter Boot mode for firmware updates via TypeC
    * Supports Klipper firmware
    * Temperature sensor support: Onboard MAX31865 chip, compatible with PT100
    * Onboard ADXL345 accelerometer
    * Reserved RGB interface, onboard 2240 driver, supports various leveling sensors like klicky, Voron Tap, PL08n, etc.
    * Supports input voltage of 12-24V and 15A current, with an enlarged heater port rated for 10A current


## Product Specifications


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Connection interface uses XT30(2+2) connector supporting 15A current, with a peak of 30A
    * Firmware: Klipper
    * Input Voltage: DC12V-24V
    * Logic Voltage: DC3.3V
    * Onboard Sensors: ADXL345 and optional MAX31865
    * Heater Interface: Heater (HEAT0)
    * Expansion Interfaces: 3 controllable fans, 1 RGB, 3 endstops, 1 heating tube, 1 thermistor
    * Fan Interfaces: Three controllable fan ports
    * Motor Driver: Onboard TMC2240
