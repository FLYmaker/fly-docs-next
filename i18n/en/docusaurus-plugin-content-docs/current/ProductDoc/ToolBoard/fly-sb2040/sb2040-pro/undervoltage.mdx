# Solutions for Driver Undervoltage

## Method One

    :::danger

    * Replace the power supply
    * Add a power filter
    * Replace the motor for testing
    * Connect the motor housing to the ground of the tool board
    * Add `driver_TPFD: 0` to the extruder driver configuration

    :::

## Method Two
    * Connect via SSH to the host machine
    * Enter the following command
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * This step might be reverted after updating Klipper
    :::
