## Katapult Firmware Description

## Enter Flashing Mode

    :::tip

    The circled button is the `BOOT button`

    :::

    * With the motherboard completely powered off, press and hold the BOOT button, then connect the data cable to the motherboard and your device

            :::tip

            * Below is the flashing command for 500K CAN rate firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * Below is the flashing command for 1M CAN rate firmware
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. If you see the following prompt, the firmware has been successfully flashed
