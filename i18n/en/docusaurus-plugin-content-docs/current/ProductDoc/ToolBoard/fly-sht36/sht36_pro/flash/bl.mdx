# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36 PRO supports CAN
    * Only CAN firmware with 500K and 1M baud rates are provided here
    :::

## Enter Flash Mode

    :::tip

    The circled one is the `BOOT button`

    :::

    * With the motherboard completely powered off, short the BOOT button and then connect the data cable to the motherboard and your device
    * Then release the BOOT button
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="80%" align="center" />

            4. Flash the firmware

            :::tip

            Only need to flash one of the firmwares

            :::

            * Below is the command for flashing the firmware with 500K CAN baud rate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Below is the command for flashing the firmware with 1M CAN baud rate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. If you see the following prompt, the firmware has been successfully flashed

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            After flashing is complete, the motherboard needs to be completely powered off once.

            :::
