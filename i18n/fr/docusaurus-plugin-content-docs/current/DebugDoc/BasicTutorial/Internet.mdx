# Connexion au réseau via SSH


* Veuillez vous assurer que le système est démarré et connecté à l'ordinateur hôte via SSH en utilisant une connexion série, il est recommandé d'utiliser **MobaXterm_Personal**
* Veuillez vous assurer que l'ordinateur hôte a installé **M2WE** ou le **module 2.4G_WiFi**, si vous utilisez **Lite2** ou **MiniPad**, l'installation de module n'est pas nécessaire
* Veuillez noter que **le module 2.4G_WiFi**, **Lite2** et **MiniPad** ne peuvent utiliser que le WiFi à 2.4G et non le WiFi à 5G
* Veuillez noter que le réseau ne peut pas utiliser une configuration de double bande unifiée sinon la connexion ne fonctionnera pas correctement
* Veuillez vous assurer que les antennes sont bien installées !!!

 1. Dans le logiciel SSH, entrez la commande **nmtui** puis appuyez sur Entrée, vous verrez l'interface suivante :

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. Appuyez sur Entrée puis sélectionnez **Activate a connection** et appuyez à nouveau sur Entrée

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. Sélectionnez le réseau auquel vous souhaitez vous connecter, appuyez sur Entrée, entrez le mot de passe WiFi puis **choisissez OK** pour enregistrer et établir la connexion

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. Une fois le WiFi connecté, sur le clavier, trouvez la touche **Esc** et appuyez plusieurs fois pour quitter l'interface actuelle, puis entrez **ip a** et appuyez sur Entrée

 * L'adresse IP 192.168.2.136 dans l'image est celle de l'ordinateur hôte

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
