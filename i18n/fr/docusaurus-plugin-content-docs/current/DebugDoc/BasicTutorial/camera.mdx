# Ajouter une caméra USB


:::danger Attention

* Il est nécessaire d'avoir installé le service `crowsnest`, comme les systèmes lite2 et mini pad qui ne l'ont pas installé.
* Une caméra USB plug-and-play sans pilote est requise. Elle peut être utilisée sous Linux, Armbian et autres systèmes supportés par Klipper sans installation de pilote supplémentaire.
* Ce tutoriel utilise les dernières versions de Klipper, Crowsnest et Fuidd. Pour d'autres méthodes, il faut faire des recherches par soi-même.

:::

## Ouvrir la configuration

:::tip

* Il existe de nombreux types d'ID de caméra selon le système. Nous devons simplement déterminer un ID qui ne changera pas.

:::

    * Dans Fuidd, trouvez le fichier de configuration, puis ouvrez `crowsnest.conf`
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf.png').default} size="100%" align="center" />

    * Trouvez `device:` et supprimez l'ID qui suit
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf1.png').default} size="100%" align="center" />

## Rechercher l'ID de la caméra

:::tip

* Assurez-vous que la caméra est connectée à l'ordinateur.

:::

    * Cliquez sur `DEVICES` en haut, puis sur `VIDEO`, et enfin sur rafraîchir
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf2.png').default} size="100%" align="center" />
    * Trouvez `path_by_id` et copiez la ligne qui suit
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf3.png').default} size="100%" align="center" />

## Ajouter l'ID

    * Ajoutez l'ID et sauvegardez la configuration
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf4.png').default} size="100%" align="center" />

## Activer la caméra

    * Cliquez sur les paramètres, sélectionnez la caméra, puis cliquez sur ajouter une caméra
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf5.png').default} size="100%" align="center" />
    * Entrez le nom de votre choix et cliquez sur sauvegarder
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf6.png').default} size="100%" align="center" />

## Vérifier l'activation

    * Cliquez sur le tableau de bord, la caméra devrait apparaître
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf7.png').default} size="100%" align="center" />
    * Si elle ne s'affiche pas, redémarrez le système.
