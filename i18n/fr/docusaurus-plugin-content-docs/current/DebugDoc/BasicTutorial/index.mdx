# Connexion à l'hôte SSH

:::danger

* Ce document s'applique uniquement à l'hôte FLY. Les autres dispositifs ne sont donnés qu'à titre indicatif.
* Si vous compilez et flashez le firmware, consultez le tutoriel de connexion réseau plutôt que celui de connexion série.

:::

* Vous pouvez vous connecter à l'appareil via un port série ou par WIFI
* Préparez à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a été flashé avec **le système correspondant à l'hôte**
* **Si l'hôte ne démarre pas correctement, retirez les périphériques externes comme les drivers, les fins de course, les ventilateurs, etc. !!**
* **Enfin, connectez-vous par réseau à SSH pour faciliter la compilation du firmware !!**
* **Pour compiler le firmware, consultez le tutoriel de connexion à l'hôte via WIFI ou câble Ethernet**

        ## Connexion à l'hôte SSH via l'IP réseau

        * Assurez-vous que le système **est démarré correctement**
        * Connectez-vous à l'interface de gestion du **routeur** auquel l'hôte est connecté, sauf si vous connaissez déjà l'adresse IP de l'hôte
        * Assurez-vous de **connaître le nom d'utilisateur et le mot de passe de l'hôte**. Par défaut, pour le système FLY, le nom d'utilisateur est **fly** et le mot de passe **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez utiliser les méthodes suivantes pour vous connecter à SSH
        * Ouvrez MobaXterm et configurez comme suit, en entrant l'IP que vous avez vue précédemment ou que vous pouvez vérifier dans l'interface du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Notez : ici, à l'étape 4, entrez le nom d'utilisateur de l'hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre pop-up apparaîtra, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Un message vous demandera d'entrer le mot de passe, entrez le mot de passe de l'hôte et appuyez sur Entrée

        **Notez : le mot de passe ne sera pas visible à l'écran. Tapez-le et appuyez directement sur Entrée**

        Pour le système FLY par défaut, le nom d'utilisateur est : `fly` et le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion est réussie
