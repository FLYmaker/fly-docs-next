# Fonctions étendues de KlipperScreen

## Méthodes pour activer le curseur de la souris dans KlipperScreen

    * Ouvrez `KlipperScreen.conf` dans la configuration de l'imprimante, ce fichier est généré automatiquement après l'installation de KlipperScreen. Si ce fichier n'existe pas, veuillez le créer vous-même.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.png').default} size="100%" align="center" />
    * Ajoutez le code ci-dessous en haut de la configuration
    ```cfg
    [main]
    show_cursor: True
    ```
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf-1.png').default} size="80%" align="center" />
    * Sauvegardez et redémarrez

## Connexion à distance à d'autres machines hôtes

    * Ouvrez `KlipperScreen.conf` dans la configuration de l'imprimante, ce fichier est généré automatiquement après l'installation de KlipperScreen. Si ce fichier n'existe pas, veuillez le créer vous-même.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.png').default} size="100%" align="center" />
    * Ajoutez le code ci-dessous en haut de la configuration
    * Vous pouvez modifier le nom de l'imprimante `VORON-300` et l'adresse IP de la machine à laquelle vous souhaitez vous connecter `192.168.6.217`.
    * Veuillez noter que le second `[printer KlipperScreen]` est utilisé pour se connecter à distance à une machine étendue, vous pouvez utiliser cette configuration pour le débogage ou pour l'arrêt.
    :::danger Attention

    * Assurez-vous d'être sur le même réseau local

    :::

    ```cfg
    # Define printer and name. Name is anything after the first printer word
    [printer VORON-300]
    # Define the moonraker host/port if different from 127.0.0.1 and 7125
    moonraker_host: 192.168.6.217
    # ports 443 and 7130 will use https/wss
    moonraker_port: 7125
    # Moonraker API key if this is not connecting from a trusted client IP
    moonraker_api_key: False

    # Define printer and name. Name is anything after the first printer word
    [printer KlipperScreen]
    # Define the moonraker host/port if different from 127.0.0.1 and 7125
    moonraker_host: 127.0.0.1
    # ports 443 and 7130 will use https/wss
    moonraker_port: 7125
    # Moonraker API key if this is not connecting from a trusted client IP
    moonraker_api_key: False

    ```
