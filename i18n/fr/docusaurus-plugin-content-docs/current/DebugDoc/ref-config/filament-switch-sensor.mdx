# Capteur de rupture de fil

```cfg title="printer.cfg"
[filament_switch_sensor Capteur de rupture de fil]
pause_on_runout: True
switch_pin: ^PC2
runout_gcode: 
        PAUSE
        M118 Rupture de fil, impression en pause
insert_gcode:
        M118 Filament réinséré

```
