Voici comment compiler le firmware :
* Après vous être connecté à SSH, entrez les commandes suivantes et appuyez sur Entrée :

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

* Dans cette commande, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, `make menuconfig` compile le firmware. Après exécution, vous devriez voir l'interface ci-dessous :

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Architecture du microcontrôleur** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Modèle de processeur**, sélectionnez **STM32F103** et appuyez sur Entrée

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/103/img/103.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du chargeur de démarrage**, choisissez : **8KiB bootloader**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **Bus CAN (sur PB8/PB9)**

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.webp').default} size="100%" align="Center" />
