Voici comment compiler le firmware :

* Connectez-vous à SSH et entrez la commande suivante, puis appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dans cette commande, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données et le firmware précédemment compilés, tandis que `make menuconfig` compile le firmware. Après exécution, vous devriez voir l'écran ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Enable extra low-level configuration options** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Micro-controller Architecture** puis sélectionnez **Raspberry Pi RP2040** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Bootloader offset**, choisissez : **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* **Communication interface**, choisissez : **USBSERIAL**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb.webp').default} size="100%" align="Center" />
