Voici comment compiler le firmware :

* Connectez-vous via SSH et entrez les commandes suivantes, puis appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` efface les données précédemment compilées et le firmware, `make menuconfig` compile le firmware. Après exécution, l'interface suivante devrait apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Allez dans le menu **Modèle de processeur**, sélectionnez **STM32F405** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du bootloader**, choisissez : **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* **Interface de communication**, choisissez : **USB (sur PA11/PA12)**

   <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/usb.webp').default} size="100%" align="Center" />
