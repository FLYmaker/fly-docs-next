Voici comment compiler le firmware :
* Connectez-vous à SSH et entrez les commandes suivantes :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, `make menuconfig` compile le firmware. Après exécution, vous devriez voir l'interface ci-dessous.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Architecture du microcontrôleur** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Modèle de processeur**, sélectionnez **STM32F407** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/407.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du chargeur de démarrage**, choisissez : **Chargeur de démarrage de 32KiB**.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/32kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **Pont USB vers bus CAN (USB sur PA11/PA12)**.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/utoc.webp').default} size="100%" align="Center" />
