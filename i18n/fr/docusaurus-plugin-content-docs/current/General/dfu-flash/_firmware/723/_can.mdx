Voici comment compiler le firmware :

* Connectez-vous via SSH, entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données et le firmware précédemment compilés, `make menuconfig` compile le firmware. Après exécution, vous devriez voir l'interface ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur**, puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Allez dans le menu **Modèle de processeur**, sélectionnez **STM32H723** et appuyez sur Entrée :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Sélectionnez **Décalage du chargeur d'amorçage**, choisissez : **Chargeur d'amorçage de 128KiB**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.png').default} size="100%" align="Center" />

* Sélectionnez **Référence de l'horloge (cristal de 8 MHz)**, choisissez : **Cristal de 25 MHz**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Interface de communication**, sélectionnez : **Bus CAN (sur PB8/PB9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/can.png').default} size="100%" align="Center" />
