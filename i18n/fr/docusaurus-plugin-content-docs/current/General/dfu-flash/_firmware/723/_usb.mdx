Voici comment compiler le firmware :
* Après vous être connecté via SSH, entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Dans cette commande, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware `make menuconfig` compile le firmware. Après exécution, vous devriez voir l'interface ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration supplémentaires de bas niveau** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Entrez dans le menu **Architecture du microcontrôleur** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Entrez dans le menu **Modèle de processeur**, sélectionnez **STM32H723** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Sélectionnez **Décalage du bootloader**, choisissez : **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.png').default} size="100%" align="Center" />

* Sélectionnez **Référence d'horloge (cristal 8 MHz)**, choisissez : **Cristal de 25 MHz**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Interface de communication**, choisissez : **USB (sur PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/usb.png').default} size="100%" align="Center" />
