Voici comment compiler le firmware :
* Connectez-vous à SSH et entrez les commandes suivantes et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Ici, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation et le firmware précédents, `make menuconfig` compile le firmware, et après exécution, l'interface suivante devrait apparaître

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Enable extra low-level configuration options** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Micro-controller Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Processor model**, sélectionnez **STM32H723** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* Sélectionnez **Bootloader offset**, choisissez : **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Clock Reference (8 MHz crystal)**, choisissez : **25 MHz crystal**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* Sélectionnez **Communication interface**, choisissez : **USB to CAN bus bridge (USB on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.webp').default} size="100%" align="Center" />
