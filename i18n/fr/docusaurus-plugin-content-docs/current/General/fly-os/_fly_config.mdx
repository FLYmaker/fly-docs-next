# Utilisation de FLYOS_Env

    :::tip Astuce

    * À partir du système 3.3.4, vous pouvez directement utiliser FLY-TOOLS pour configurer le système sans avoir à modifier la configuration dans le boot
    * Veuillez noter que les anciens systèmes ne supportent pas directement la configuration avec FLY-TOOLS
    * <Button variant="contained" disableElevation href="/docs/ToolsDoc/fly-tools/">Tutoriel d'utilisation de FLY_TOOLS</Button>

    :::
        
### Explication de la configuration WIFI

    :::danger À savoir

    * La méthode de configuration WIFI est complexe, il est nécessaire de se connecter via un port série à l'ordinateur hôte pour établir la connexion WIFI

    :::
