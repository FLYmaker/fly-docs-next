### Installer M2WE ou une carte SD sur M2WE

    :::danger Attention

    * Ce produit supporte USB3.0
    * Ce produit ne supporte pas l'utilisation simultanée de TF et M2WE
    * Après avoir installé M2WE ou une carte SD et connecté à un port USB de l'ordinateur, vous pouvez formater
    * Il est interdit d'installer des dispositifs NVME ou autres, sinon cela pourrait endommager l'appareil !!!

    :::

### Préparation

    * Installez d'abord le logiciel `Win32DiskImager`
    * Il n'est pas recommandé d'utiliser `balenaetcher` pour graver le système
    * Décompressez le système, le nom du fichier après décompression se terminera par `.img`
    * Insérez la carte TF dans un lecteur de carte, puis connectez le lecteur à l'interface USB de l'ordinateur
    * Si la carte TF contient des données, veuillez faire une sauvegarde. Après avoir installé le système, il n'est pas recommandé de retirer la carte TF sauf si vous devez graver sur M2WE

### Formater la carte mémoire

    * Utilisez le logiciel `TFFormatter.exe` pour formater
    * Sélectionnez le disque que vous souhaitez formater

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * Ou utilisez la gestion de disque intégrée à Windows pour supprimer la partition, puis recréer une partition après la suppression.
