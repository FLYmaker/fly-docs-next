# Gravure du système sur une carte TF

### Préparation

    * Installez d'abord le logiciel `Win32DiskImager`
    * Décompressez le système, après décompression le nom de fichier se terminera par `.img`
    * Il n'est pas recommandé d'utiliser `balenaetcher` pour la gravure du système
    * Insérez la carte TF dans un lecteur de cartes, puis connectez le lecteur à un port USB de l'ordinateur
    * Si la carte TF contient des données, sauvegardez-les. Après avoir gravé le système, il n'est pas recommandé de retirer la carte TF sauf si vous devez graver sur M2WE, alors vous pouvez la retirer.

### Formatage de la carte mémoire

    * Utilisez le logiciel `TFFormatter.exe` pour formater la carte TF
    * Sélectionnez le disque que vous souhaitez formater

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * Ou utilisez la gestion des disques intégrée à Windows pour supprimer la partition, puis recréez une partition après la suppression.
