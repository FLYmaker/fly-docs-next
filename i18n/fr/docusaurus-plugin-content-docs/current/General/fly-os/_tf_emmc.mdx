# Graver sur M2WE via une carte TF

:::info
La migration du système permet de transférer intégralement l'image originale de la carte TF vers le M2WE. Une fois la migration terminée, le contenu du M2WE sera identique à celui de la carte TF d'origine.
:::

    * Installez le M2WE puis mettez-le sous tension pour démarrer
    * Veuillez d'abord utiliser le port série pour [vous connecter à SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Cliquez pour être redirigé") 
    * Exécutez la commande ci-dessous, un mot de passe peut être demandé, entrez `mellow` et appuyez sur Entrée, notez que le mot de passe n'est pas visible

    ```cfg
    sudo nand-sata-install
    ```

    * Sélectionnez l'option `Boot from eMMC - system on eMMC` puis appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_1.png').default} size="80%" align="left" />

    * Lorsque `This script will erase your eMMC. Continue?` apparaît, sélectionnez `Yes` et appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_2.png').default} size="80%" align="left" />

    * Sélectionnez le type de système de fichiers `ext4` et appuyez sur Entrée (l'option par défaut est généralement celle-ci, appuyez directement sur Entrée)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_3.png').default} size="80%" align="left" />

    * Attendez patiemment jusqu'à ce que la barre de progression apparaisse (si après 2 minutes la barre de progression ne s'affiche pas, appuyez sur `CTRL+C` pour interrompre l'opération, puis redémarrez l'appareil et réessayez)
    
    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_4.png').default} size="80%" align="left" />
    * Pendant le processus de migration du système, ne coupez pas l'alimentation et n'effectuez aucune autre opération, jusqu'à ce que la migration soit terminée
    * Une fois la migration terminée, appuyez directement sur Entrée, attendez que l'appareil s'éteigne complètement

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_5.png').default} size="80%" align="left" />


    * Une fois l'appareil éteint, retirez la carte TF, puis rallumez l'appareil, vous pourrez maintenant démarrer à partir du système sur l'eMMC
