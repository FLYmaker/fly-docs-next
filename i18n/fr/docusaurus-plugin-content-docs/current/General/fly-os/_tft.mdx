:::tip
Ouvrez votre navigateur à votre IP:9999 pour accéder à FLY-Tools
:::

    * Pour utiliser le TFT, vous devez activer l'option ci-dessous pour activer l'écran
    * Veuillez noter que les écrans TFT se divisent en trois types :
        * `FLY-TFT-V1` Écran résistif V1, compatible avec tous les logiciels FLY
        * `FLY-TFT-V2-R` Écran résistif V2, compatible avec C8, PI-V2 et la série Raspberry Pi avec 40 broches, mais nécessite d'être utilisé avec G2T
        * `FLY-TFT-V2-C` Écran capacitif V2, compatible avec C8, PI-V2 et la série Raspberry Pi avec 40 broches, mais nécessite d'être utilisé avec G2T

    * Une fois dans FLY Tools
