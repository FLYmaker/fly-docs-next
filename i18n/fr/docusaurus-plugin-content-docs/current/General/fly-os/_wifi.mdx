:::danger À noter

    * FLY a besoin de modules M2WE ou 2.4G WIFI pour fonctionner, sinon utilisez le WiFi
    * Veuillez noter que les noms WiFi en chinois ne sont pas supportés ainsi que les WiFi bi-bande 2.4G et 5G combinés
    * Si vous utilisez un mini pad ou lite2, ignorez cet avertissement

    :::

## Connexion à l'ordinateur hôte

    * Si la configuration WiFi échoue dans le disque BOOT, vous pouvez vous référer à cette configuration pour établir la connexion WiFi, notez que vous aurez besoin d'une connexion série à l'ordinateur hôte
    * Veuillez d'abord vous connecter en série [à SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Cliquez pour accéder") 
    * Exécutez les commandes ci-dessous, vous pourriez être invité à entrer un mot de passe, entrez le mot de passe `mellow` et appuyez sur Entrée, notez que le mot de passe est invisible

    ```cfg
    sudo nmtui
    ```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui1.png').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.png').default} size="80%" align="left" />

    * Sélectionnez `Activer une connexion` puis appuyez sur Entrée

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.png').default} size="80%" align="left" />

    * Vous serez dirigé vers l'interface suivante, utilisez les touches `↑ ↓` du clavier pour choisir le WiFi auquel vous souhaitez vous connecter, appuyez sur Entrée, entrez le mot de passe et attendez que la connexion soit établie

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.png').default} size="80%" align="left" />


        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.png').default} size="80%" align="left" />

    * Une fois la connexion établie, l'écran ressemblera à ceci

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.png').default} size="80%" align="left" />

    * Appuyez sur la touche Échap pour quitter, revenez à l'interface de commande, entrez `ip a`, l'adresse IP de l'ordinateur hôte est `192.168.2.126` comme illustré

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui7.png').default} size="80%" align="left" />
