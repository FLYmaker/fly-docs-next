    * Veuillez vous assurer que l'ordinateur hôte peut se connecter normalement à Internet
    * Avant de compiler le firmware, il est nécessaire de se connecter à l'ordinateur hôte via SSH par le réseau
    * Veuillez ne pas utiliser d'outils de communication série pour vous connecter à l'ordinateur hôte

## Explications sur l'opération de la page de configuration de compilation du firmware Klipper

    :::tip

    Veuillez vous assurer que le mode d'entrée du clavier est en mode demi-largeur, c'est-à-dire en mode anglais

    :::

    1. La touche `↑` du clavier, la touche `↓`, sont utilisées pour déplacer le curseur vers le haut et le bas pour sélectionner les éléments de menu
    2. La touche de validation `Enter` ou la touche espace `Space`, sont utilisées pour cocher un menu ou entrer dans un sous-menu
    3. La touche de sortie `ESC`, est utilisée pour revenir au menu précédent
    4. La touche `Q`, est utilisée pour quitter la page de configuration du firmware Klipper
    5. La touche `Y`, lors de la sortie de la page de configuration du firmware Klipper, si un message s'affiche, cliquez sur la touche `Y` pour sauvegarder le fichier de configuration

    :::warning

    Si le nombre d'options de la page de configuration est limité, veuillez d'abord cocher `[ ] Activer les options de configuration de bas niveau supplémentaires`, pour afficher certaines options cachées

    :::
