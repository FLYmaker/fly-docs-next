## Méthode d'utilisation de CSI

    * Une fois la gravure de l'image terminée, allez dans /boot/config.txt, trouvez `camera_auto_detect=1` et modifiez-le en `camera_auto_detect=0`, puis ajoutez la configuration selon le modèle de la caméra
    * Voici la configuration de référence

    ```bash
    dtoverlay=imx219,cam1
    ```
