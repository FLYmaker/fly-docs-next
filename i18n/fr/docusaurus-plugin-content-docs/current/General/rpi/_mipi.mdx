## Raspberry Pi 5 avec MIPI

:::tip
FLY_MIPI est un module d'adaptation spécialement conçu pour le Raspberry Pi 5, permettant d'utiliser directement les appareils DSI et CSI via une adaptation.
:::

### Branchement du Raspberry Pi 5 avec MIPI

    :::danger À savoir

    * Les interfaces DSI et CSI du Raspberry Pi 5 sont maintenant renommées MIPI.
    * L'interface MIPI peut être configurée pour utiliser directement une caméra CSI ou un écran DSI.
    * Veuillez noter l'orientation des doigts d'or.

    :::

    * Le Raspberry Pi 5 dispose maintenant de deux interfaces MIPI, visibles en regardant le devant avec des marquages 1 et 0.

## Branchement de l'écran DSI

    * Si vous utilisez l'interface n°1, configurez l'écran comme suit :

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * Si vous utilisez l'interface n°0, configurez l'écran comme suit :

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## Branchement de la caméra CSI

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * Veuillez noter que le modèle de caméra doit être configuré en fonction de votre achat, ce tutoriel utilise l'imx219 comme référence.

    * Si vous utilisez l'interface MIPI gauche, configurez la caméra comme suit :
    ```bash
    dtoverlay=imx219,cam1
    ```

    * Si vous utilisez l'interface MIPI droite, configurez la caméra comme suit :
    ```bash
    dtoverlay=imx219,cam0
    ```
