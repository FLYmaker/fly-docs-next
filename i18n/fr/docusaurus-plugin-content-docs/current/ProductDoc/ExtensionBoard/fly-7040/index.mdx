## Caractéristiques du produit

    * Vitesse maximale : 45000
    * Débit maximal : 14.7CFM
    * Accessoires en option : Filtre d'admission, réduit le bruit de fonctionnement, empêche les objets étrangers d'entrer dans le ventilateur et de l'endommager, rendant le fonctionnement de l'ensemble plus silencieux et stable

## Précautions

    :::warning Précautions

    * Veuillez noter que l'entrée du ventilateur doit être protégée contre l'intrusion d'objets étrangers pour éviter d'endommager les pales.
    * Veuillez noter que l'alimentation maximale du ventilateur est de 24V.

    :::
