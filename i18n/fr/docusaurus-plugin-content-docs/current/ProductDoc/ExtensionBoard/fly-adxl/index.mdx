# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.png').default} size="100%" align="left" />



## Caractéristiques du produit

    * Adapté aux moteurs 36 et autres
    * Peut être connecté à Raspberry Pi, FLY PI, Orange Pi, carte mère, etc., via une interface SPI

## Spécifications du produit

    * Interface embarquée : connecteur ZH1.5 6P
    * Mode de communication : SPI
    * Tension de fonctionnement : 5V

## Points d'attention

    :::warning Points d'attention

    * La tension de fonctionnement est de 5V, ne pas la connecter incorrectement
    * Adapté à divers firmwares, veuillez vérifier si la connexion SPI est supportée

    :::
