# AirClick Bouton sans fil
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.png').default} size="100%" align="Center" />



## Caractéristiques du produit

    * Haute compatibilité : Conçu pour les machines Klipper, peut être directement connecté à une machine Klipper et envoyer des instructions prédéfinies, offrant une compatibilité et une expérience d'utilisation fluide.
    * Macros personnalisables : Les utilisateurs peuvent personnaliser les instructions selon leurs besoins, rendant l'opération de la machine plus flexible et pratique.
    * Support de connexion sans fil et filaire à Klipper
    * Connexion sans fil : Grâce à la conception ESP32, il peut directement envoyer des instructions personnalisées ou des macros à Klipper, offrant une grande flexibilité et extensibilité.
    * Connexion filaire : Via la puce STM32F072, en cas de défaillance de la connexion réseau à Klipper, des opérations simples peuvent être effectuées. Supporte jusqu'à dix instructions personnalisées ou macros.
    * Facile à utiliser : Possède une interface utilisateur intuitive, permettant aux utilisateurs d'ajouter et de gérer facilement les boutons sans fil, améliorant grandement la facilité d'utilisation.

## Spécifications du produit

    * Chip principal : ESP32
    * Chip supplémentaire : STM32F072
    * Envoi d'instructions via ESP32-WIFI : Nombre illimité
    * Envoi d'instructions via STM32F072-IO : 10
    * Compatibilité : Conçu pour les machines Klipper
    * Alimentation : 5V

## Précautions

    :::danger Précautions

    * Ce produit utilise une alimentation 5V qui peut être fournie via `type c` ou la borne `PH2.0-2P`
    * L'antenne intégrée du produit ne peut être utilisée que pour recevoir les signaux des boutons. Si vous devez vous connecter au WIFI, veuillez installer l'antenne correctement, le connecteur de l'antenne est `ipex3`.
    * Veuillez installer correctement l'antenne !!! Sinon, une mauvaise réception du signal peut entraîner une surchauffe sévère !!!
    * Veuillez noter, pour éviter les déclenchements accidentels, le système désactivera automatiquement le bouton après une pression longue de 15 secondes.

    :::
