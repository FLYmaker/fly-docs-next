# Connexion à Klipper

    :::info Informations importantes

    * L'antenne intégrée du produit ne peut être utilisée que pour la réception, l'antenne d'émission doit être installée séparément, le connecteur d'antenne est de type `ipex3`
    * Veuillez installer l'antenne correctement !!! Sinon, une mauvaise réception du signal peut entraîner une surchauffe sévère !!!
    * La première fois, vous devez vous connecter manuellement au point d'accès AirClick avant de vous connecter au WiFi correspondant

    :::


    :::info Informations importantes

    * AirClick se connecte à Moonraker pour envoyer des commandes de contrôle à la machine Klipper
    * Il permet de définir des macros personnalisées, un arrêt d'urgence, de redémarrer Klipper, de redémarrer le firmware, d'éteindre le système, etc.

    :::

## AirClick se connecte au WiFi

    * Connectez-vous au point d'accès AirClick, le nom du point d'accès est `FLY-WBTN-XXXXXX`
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * Après vous être connecté au point d'accès, accédez à [192.168.4.1](http://192.168.4.1/#/setting) via un navigateur
    * Vous serez redirigé vers l'interface de configuration du WiFi, vous devrez vous connecter manuellement au même routeur que l'imprimante, entrer le mot de passe et cliquer sur le cadre bleu en bas pour vous connecter au WiFi
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * Une fois connecté au WiFi, AirClick affichera son adresse IP, cliquez sur le cadre bleu en bas pour redémarrer l'appareil
        * Le numéro 1 encerclé est l'adresse IP d'AirClick
        * Le numéro 2 encerclé copie l'IP et redémarre l'appareil
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * Veuillez noter que vous devez manuellement basculer votre réseau actuel vers le routeur correspondant, sinon vous ne pourrez pas y accéder

## Changer la langue
    * Cliquez sur l'icône encerclée en haut à droite, sélectionnez la langue que vous souhaitez
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-2.webp').default} size="80%" align="Center" />

## Se connecter à Klipper

    :::info Informations importantes
    * AirClick se connecte à Moonraker pour envoyer des commandes de contrôle à la machine Klipper
    * Il permet de définir des macros personnalisées, un arrêt d'urgence, de redémarrer Klipper, de redémarrer le firmware, d'éteindre le système, etc.
    :::

    * Cliquez sur les paramètres système à gauche
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />
    * Recherchez `Moonraker IP` ci-dessous et entrez l'adresse IP de la machine Klipper
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-1.webp').default} size="80%" align="Center" />
        * Le chiffre 1 entre l'adresse IP de la machine Klipper
        * Le chiffre 2 par défaut est `7125`, si la machine utilise un autre port, entrez le port correspondant
        * Le chiffre 3 confirme et enregistre la configuration
    * Une fois correctement connecté, une notification apparaîtra en haut de la page web
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />
