# Introduction

# Utilisation du tampon LLL

Le tampon de consommable LLL est un dispositif spécialement conçu pour l'alimentation à distance des imprimantes 3D, visant à simplifier les opérations de chargement et de déchargement des consommables. Ce tampon est équipé de deux boutons pour l'avance et le retrait du filament, permettant aux utilisateurs de réaliser ces opérations de manière motorisée.

### Fonction tampon :

Quand l'extrudeuse de l'imprimante 3D avance le filament, si elle tire sur le support de consommable à distance, le tampon détecte cette action et envoie automatiquement du filament vers la buse. À chaque fois que cela se produit, le tampon répète cette opération d'alimentation. Cela réduit efficacement la traction sur le filament par l'extrudeuse, stabilise la force d'extrusion et améliore ainsi la stabilité de la qualité d'impression.

### Fonction de détection de rupture de filament :

Le tampon intègre un détecteur de rupture de filament, qui, lorsqu'il détecte l'absence de filament dans le tampon, envoie un signal à l'imprimante 3D pour qu'elle mette en pause l'impression. Pour utiliser cette fonction, il est nécessaire de connecter les broches de signal et de modifier la configuration de l'imprimante 3D. En revanche, la fonction tampon peut être utilisée sans modification de la configuration.

Grâce à ces conceptions fonctionnelles, le tampon de consommable LLL non seulement simplifie la manipulation des consommables, mais améliore également la stabilité et la qualité du processus d'impression 3D.

* Veuillez noter que pour utiliser les fonctions, il faut d'abord connecter le consommable, sinon le moteur ne peut être contrôlé que par les boutons.

## Référence de configuration

    ```cfg
    [filament_switch_sensor 断料检测]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 缺料
    insert_gcode:
            M118 上料
    ```
