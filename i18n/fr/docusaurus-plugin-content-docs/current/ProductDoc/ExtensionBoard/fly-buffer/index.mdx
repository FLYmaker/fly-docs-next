# Introduction

# Utilisation du tampon LLL

    Le tampon de consommables LLL est un dispositif conçu spécialement pour l'alimentation à distance des imprimantes 3D, visant à simplifier les opérations de chargement et de déchargement des consommables. Cet amortisseur est équipé de deux boutons, respectivement pour l'alimentation et le retrait du filament, permettant aux utilisateurs d'effectuer des opérations d'alimentation et de retrait motorisées.

### Fonction tampon :

    Lorsqu'une imprimante 3D alimente son extrudeur, si elle tire sur le support de consommables à distance, le tampon détecte cette action et envoie automatiquement une section de consommable vers l'extrémité chaude. À chaque traction, le tampon répète cette opération d'alimentation. Cela permet de réduire efficacement la traction sur le consommable par l'extrudeur, de maintenir la poussée stable de l'extrudeur, et d'améliorer ainsi la stabilité de la qualité d'impression.

### Fonction de détection de rupture :

    Le tampon intègre un détecteur de rupture qui, lorsqu'il détecte l'absence de consommable dans le tampon, envoie un signal à l'imprimante 3D pour qu'elle suspende l'impression. Pour utiliser cette fonction, il est nécessaire de connecter les broches de signal et de modifier la configuration de l'imprimante 3D. En revanche, la fonction tampon peut être utilisée directement sans modification de la configuration.

    Grâce à ces conceptions fonctionnelles, le tampon de consommables LLL simplifie non seulement les opérations sur les consommables mais améliore également la stabilité du processus d'impression 3D et la qualité d'impression.

* Veuillez noter que pour utiliser les fonctionnalités, il faut d'abord connecter le consommable, sinon le moteur ne peut être contrôlé que via les boutons.

## Référence de configuration

    ```cfg
    [filament_switch_sensor 断料检测]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 缺料
    insert_gcode:
            M118 上料
    ```
