# FLY-G2T

    La carte d'extension multifonction FLY-G2T est conçue pour étendre les fonctionnalités de la Raspberry Pi et du logiciel de supervision FLY. Elle supporte la communication UART, intègre une interface FPC pour connecter un TFT-V2, et supporte également le contrôle d'un ventilateur 5V. Cela en fait une solution idéale pour améliorer l'interactivité et l'accessibilité de la Raspberry Pi dans diverses applications, telles que l'automatisation industrielle, l'impression 3D, l'automatisation domestique, le contrôle de robots ou toute situation nécessitant une interface homme-machine et une communication externe.

## Caractéristiques du produit

    * Compatibilité : Supporte plusieurs versions de la Raspberry Pi, y compris la Raspberry Pi Zero/3B/3B+/4B.
    * Conversion UART vers RS232 : Fournit une communication série stable, permettant à la Raspberry Pi d'échanger des données avec d'autres équipements via l'interface RS232.
    * Écran tactile capacitif SPI : Connecté via l'interface FPC, l'écran TFT-V2 offre une manipulation tactile intuitive, adaptée aux applications nécessitant une interface utilisateur graphique, par exemple : klipperscreen.
    * Support pour ventilateur 5V : Intègre une interface pour ventilateur 5V, permettant de connecter un ventilateur pour la dissipation thermique, garantissant ainsi la stabilité du logiciel de supervision lors de longues périodes de fonctionnement ou sous haute charge.
    * Design plug-and-play : La carte d'extension est conçue pour être facile à installer, se branchant directement sur le logiciel de supervision sans nécessiter de câblage supplémentaire ou de configuration complexe.
    * Taille compacte : La carte d'extension est de taille raisonnable et n'occupe pas trop d'espace, facilitant son intégration dans divers projets.

## Spécifications du produit

    * Interface de communication : Conversion UART vers RS232
    * Interface d'affichage : Conversion SPI vers interface FPC
    * Alimentation du ventilateur : 5V
    * Compatibilité : Raspberry Pi Zero/3B/3B+/4B, FLY Pi V2, etc.

## Points d'attention

    :::warning Points d'attention

    * Lors de la connexion d'une carte d'outils RS232, assurez-vous que le logiciel de supervision est `mis à la terre` !!!!!!

    :::

    * Assurez-vous que le logiciel de supervision est éteint avant d'installer la carte d'extension.
    * Évitez d'utiliser la carte d'extension dans des environnements humides ou poussiéreux.
    * Lors de l'installation de l'écran, veuillez ne pas le brancher ou le débrancher sous tension et faites attention à l'orientation du câble de l'écran pour éviter tout dommage.
    * Lors de l'installation du ventilateur, veuillez vérifier les spécifications de courant et de tension du ventilateur pour éviter tout dommage.
