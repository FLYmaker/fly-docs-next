# FLY_halo

## 1. Page d'accueil

    Cliquez sur le lit chauffant ou la buse, faites glisser dans le sens horaire à partir du haut de l'écran vers la température souhaitée, puis cliquez à nouveau sur le lit chauffant ou la buse pour définir la température.

    Si vous avez besoin d'afficher la température du MCU et la température de l'hôte, vous devez ajouter cette configuration

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # associé au MCU (par défaut)

    [temperature_sensor host]        # température de l'hôte
    sensor_type: temperature_host     # associé à l'hôte
    ```


    :::warning Notes importantes

    * La température maximale est lue dans la configuration

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.png').default} size="50%" align="left" />


## 2. Extrusion

    * Un appui long sur l'icône permet de définir la température de la buse, la longueur d'extrusion et la vitesse d'extrusion

    :::warning Notes importantes

    * Halo ne supporte que le contrôle d'une seule buse

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/Extrusion.png').default} size="50%" align="left" />
