# Introduction aux opérations des pages Halo

## Structure des pages

```bash
  Page d'impression de fichiers           Accueil             Page d'extrusion            Services système             Paramètres              Informations sur l'appareil
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Print Files |<-->|     Home    |<-->|   Extrude   |<-->| Sys Service |<-->|   Setting   |<-->|  Dev Info   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blocking Det|
                   |             |
                   +-------------+
                     Page de détection de blocage
            (Cette page n'apparaît que si la détection de blocage est activée dans les paramètres)
```

## Page d'extrusion

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.png').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.png').default} size="50%" align="left" />
    * Les zones interactives de l'image 1 ci-dessus correspondent aux numéros ci-dessous
    1. Clic : Augmente la température cible de la tête d'extrusion de 1
    2. Appui long : Accède au réglage de la température d'extrusion, ajustez la température en déplaçant la barre circulaire, cliquez pour confirmer la température
    3. Clic : Diminue la température cible de la tête d'extrusion de 1
    4. Clic : Extrude
    5. Glisser vers le haut ou vers le bas : Permet de choisir la longueur d'extrusion ou de rétraction, en mm
    6. Clic : Rétracte
    7. Appui long : Accède au réglage de la vitesse d'extrusion de l'image 2 ci-dessus, glissez pour choisir la vitesse d'extrusion, en mm/s, puis cliquez sur une zone vide pour quitter
