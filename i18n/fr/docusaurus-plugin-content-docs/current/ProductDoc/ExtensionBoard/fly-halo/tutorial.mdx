# Tutoriel d'utilisation

## Points à noter

    * La première utilisation nécessite une connexion réseau. Si votre WIFI ne peut pas se connecter à Internet, halo ne pourra pas obtenir la configuration initiale au démarrage.
    * Tous les paramètres dans la page de configuration ne prendront effet qu'après avoir cliqué sur **Enregistrer les paramètres** et le redémarrage de l'appareil.
    * La fonction de détection de blocage est désactivée par défaut. Elle ne peut être activée qu'après avoir activé la détection de blocage dans les paramètres et redémarré, puis accéder en faisant défiler depuis l'écran d'accueil.
    * La fonction d'arrêt après impression est désactivée par défaut, mais peut être activée dans les paramètres.
    * Tous les boutons dans les pop-ups nécessitant l'annulation ou la mise en pause de l'impression doivent être maintenus enfoncés, le simple clic n'a aucun effet (pour éviter les erreurs de manipulation).
    * Dans la page des services système, le service Host est l'hôte de la machine hôte.
    * Lorsqu'on utilise le firmware officiel, veuillez ne pas tenter de remplacer le microcontrôleur principal ou la puce FLASH.

    :::tip

    * Notez que si votre fichier de configuration `moonraker.conf` dépasse 4MB, vous pourriez ne pas pouvoir utiliser normalement. En cas de problème, réduisez la taille du fichier de configuration `moonraker.conf`.

    :::

## Mise à jour du firmware du système

    * Les mises à jour de la version halo se divisent en deux types : mise à jour complète et mise à jour OTA incrémentale
        * Mise à jour complète : implique une mise à niveau importante du bas niveau, nécessitant l'utilisation d'un package de firmware complet via USB.
        * Mise à jour incrémentale : mise à niveau OTA normale, vérifiez les mises à jour sur la page des informations de l'appareil.

## Utiliser un smartphone pour scanner et configurer le WIFI

    :::tip

    * Certains appareils mobiles peuvent ne pas avoir la fonction de connexion au WiFi par scan QR, vous pouvez vous connecter au hotspot nommé **FLY-halo-xxxxxxxx** dans les paramètres WiFi de votre téléphone.

    :::

    :::tip

    * Si après connexion au hotspot aucune page ne s'ouvre, allez manuellement à l'adresse dans votre navigateur.

    :::

    :::tip

    * Veuillez noter que HALO ne peut utiliser que le WiFi 2.4G.

    :::

    :::tip

    * L'écran de Halo ne peut pas rechercher les réseaux WiFi cachés.

    :::
