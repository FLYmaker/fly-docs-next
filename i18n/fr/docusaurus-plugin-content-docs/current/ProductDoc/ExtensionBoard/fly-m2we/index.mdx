# FLY-M2WE

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-m2we/img/m2we.png').default} size="40%" align="Center" />

## Points à noter

:::danger Points à noter

* Le M2WE est préchargé avec FLYOS à l'usine et peut être lancé directement
* Le M2WE ne peut être utilisé que sur les produits FLY, il n'est pas compatible avec d'autres appareils
* La borne d'antenne du M2WE est de type `IPEX1`
* Le M2WE ne supporte pas la combinaison des fréquences 2,4G et 5G, les caractères chinois, les caractères spéciaux, ni le WIFI par défaut caché

:::
