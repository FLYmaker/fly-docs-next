# FLY-UTOR

    FLY-UTOR est une solution d'extension d'interface haute performance et polyvalente, conçue pour fournir aux utilisateurs des ports USB supplémentaires et des capacités d'extension d'alimentation, tout en supportant la communication UART vers RS232. Cette carte d'extension est compacte, facile à installer, et convient à diverses applications nécessitant des interfaces et une sortie d'alimentation supplémentaires.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left" />


## Caractéristiques du produit

    * Extension de deux ports USB : La carte d'extension fournit deux ports USB supplémentaires, supportant la norme USB 2.0, assurant la compatibilité avec divers périphériques USB et une transmission de données rapide.
    * Support UART vers RS232 : Intègre un convertisseur UART vers RS232, facilitant l'intégration de périphériques série dans le système, étendant ainsi les capacités de communication des appareils.
    * Extension de deux interfaces d'alimentation : Fournit deux interfaces d'alimentation, permettant aux utilisateurs de connecter deux sources d'alimentation externes simultanément, répondant aux besoins supplémentaires en énergie.
    * Conception d'alimentation stable : La carte d'extension est conçue avec un circuit d'alimentation stable, assurant un fonctionnement stable des périphériques connectés.
    * Facile à installer : Conçu dans des dimensions standards, s'adapte à divers environnements sans nécessiter l'installation de pilotes supplémentaires.
    * Taille compacte : Les dimensions de la carte d'extension sont raisonnables, n'occupant pas beaucoup d'espace, ce qui facilite son intégration dans divers projets.

## Spécifications du produit :

    * Interfaces disponibles : USB X2, UART vers RS232 X2, interface d'alimentation X2
    * Interface d'alimentation : Extension 12V/24V
    * Compatibilité : Ordinateurs hôtes avec interface USB

## Remarques :

    * Débranchez l'alimentation et éteignez tous les appareils avant l'installation.
    * Insérez la carte d'extension USB multifonction dans le port USB de la carte mère ou dans un autre port d'extension USB.
    * Connectez les périphériques USB nécessaires et/ou l'alimentation à la carte d'extension.
    * Pour utiliser la fonction UART vers RS232, connectez les périphériques série correspondants.
    * Rebranchez l'alimentation et démarrez les appareils.
    * Le système devrait automatiquement détecter et configurer la carte d'extension. Si nécessaire, veuillez vous référer aux notes de configuration à la fin de l'opération.
