# Introduction aux commutateurs DIP

:::tip
FLY Gemini se connecte à l'hôte via des commutateurs DIP. Si les commutateurs ne sont pas positionnés correctement, il ne fonctionnera pas correctement.
:::

:::danger Attention
Veuillez noter que les commutateurs DIP doivent être positionnés à l'emplacement spécifié pour se connecter correctement à l'hôte.
:::

## Connexion à l'hôte embarqué

    * Déplacez les commutateurs DIP `1` et `2` vers la droite pour connecter le dispositif inférieur à l'hôte via USB intégré.
    * Déplacez les commutateurs DIP `3` et `4` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'hôte.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * Après avoir déplacé les commutateurs DIP selon cette méthode, le port USB ci-dessous sera occupé par le dispositif inférieur. Veuillez ne pas connecter d'autres dispositifs.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Connexion à un hôte externe

    * Déplacez les commutateurs DIP `3` et `4` vers la droite pour connecter le dispositif inférieur à l'hôte via USB intégré.
    * Déplacez les commutateurs DIP `1` et `2` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'hôte.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
