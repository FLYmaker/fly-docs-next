# Instructions for Flashing Firmware

    
## Comment déterminer le firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware.

    :::
    
    * C8 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : si plusieurs appareils USB sont connectés, cela peut affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : peut se connecter à plusieurs appareils sans occuper les ports USB
        * Inconvénients : peut entraîner des erreurs telles que des timeouts de retour à l'origine si utilisé avec `KlipperScreen`.
