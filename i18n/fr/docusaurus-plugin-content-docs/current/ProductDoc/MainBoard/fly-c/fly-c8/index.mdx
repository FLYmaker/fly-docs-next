## Caractéristiques du produit

    * Utilise un microcontrôleur ARM Cortex-M0+ STM32F407VGT6 avec une fréquence de 168MHz
    * Intègre le mode de travail SPI et UART des drivers TMC
    * Broches DIAG intégrées, permettant une réinitialisation sans fin par simple changement de capuchon de cavalier
    * Ventilateur numérique avec options de tension 24V, 12V, 5V, évitant l'achat d'un module de réduction de tension.
    * Nouvelle méthode de flashage du firmware, mise à jour directe via FLY-Tools
    * Gestion de l'alimentation multi-voie, un court-circuit sur un port, une limite ou un driver n'affecte pas le fonctionnement de l'ordinateur hôte
    * Interface RGB réservée (tension sélectionnable 12V, 5V)
    * Interface SPI réservée, permettant aux clients de connecter un capteur d'accélération pour la compensation d'accélération
    * Écrans TFT et HDMI compatibles avec un câble unique
    * Système préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)

## Spécifications du produit

### Ordinateur hôte
    * CPU : Puce H5 Allwinner hautes performances, 4 cœurs 64 bits Cortex-A53
    * GPU : Mali 450 à 6 cœurs haute performance, taux de remplissage de pixels supérieur à 2.7gpixel/s
    * RAM : 1GB DDR3 (partagée avec le GPU)
    * ROM : Supporte jusqu'à 128GB SDCARD
    * Périphériques : Spi x1, TFT X1, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1, interface audio 3.5mm X1
    * Interfaces : Interface M.2 intégrée, extensible avec WIFI, eMMC, etc. (protocole privé, ne connectez pas de périphériques M.2 non spécifiques à FLY π)
    * Une interface de ventilateur 5V contrôlable ZH1.5 réservée pour le refroidissement de l'ordinateur hôte
    * Supporte les écrans TFT et HDMI FPC avec un seul câble
    * Interface de pont CAN réservée
    * Système préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)

### Ordinateur inférieur

    * MCU : STM32F407VGT6
    * Firmware : KLIPPER/RRF/Marlin
    * Supporte une alimentation CC de 12-24V, plus stable, élimine le besoin d'un module de réduction de tension
    * Courant de sortie maximum du port de lit chauffant : 15A
    * Courant de sortie maximum du port de bâton chauffant : 6A
    * Courant de sortie maximum du port de ventilateur : 1A
    * Modes de travail des drivers supportés : SPI, UART, STEP/DIR
    * Drivers supportés : TMC5160Pro externe, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
    * Interfaces d'extension : BLTouch (Servos, Probe), RGB X2, UART X1, 12864 X1
    * Ventilateurs : Ventilateurs numériques à deux fils X6 (sélectionnables 24V, 12V, 5V)
    * Capteurs de température : 3 voies NTC 100K
    * Structures de machine supportées : Cartesian, Delta, Kossel, Ultimaker, CoreXY
