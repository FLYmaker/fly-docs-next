# Connexion et configuration de l'écran

:::info

* FLY-TOOLS nécessite une connexion en réseau de l'hôte pour entrer l'IP de l'hôte suivi de `:9999` pour y accéder
* Par exemple `192.168.6.110:9999`

:::
