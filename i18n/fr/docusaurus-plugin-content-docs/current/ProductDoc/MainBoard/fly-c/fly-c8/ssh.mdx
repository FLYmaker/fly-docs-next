# Connexion SSH à l'hôte


* Vous pouvez vous connecter à l'appareil via un port série ou le WiFi
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a été flashé avec **le système de l'hôte correspondant**
* **Si l'hôte ne démarre pas correctement, veuillez démonter les périphériques comme les pilotes, les fins de course, les ventilateurs, etc. !!!**
* **Enfin, veuillez vous connecter via le réseau à SSH pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion de l'hôte via WiFi ou câble Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utiliser l'IP réseau pour se connecter à l'hôte SSH

        * Veuillez vous connecter au **panneau d'administration du routeur**
        * Assurez-vous que le système est **correctement démarré**
        * Assurez-vous de **connaître le nom d'utilisateur et le mot de passe de l'hôte**, par défaut pour le système FLY, le nom d'utilisateur est **fly**, le mot de passe est **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez utiliser l'IP pour vous connecter à SSH
        * Ouvrez MobaXterm, configurez-le comme suit, remplissez l'IP que vous avez vue précédemment ou consultez le panneau d'administration du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Remarque : ici, à l'étape 4, entrez le nom d'utilisateur de l'hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * La première connexion affichera cette fenêtre, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Demande de mot de passe, entrez le mot de passe de l'hôte et appuyez sur Entrée

        **Remarque : le mot de passe ne sera pas visible, tapez-le et appuyez directement sur Entrée**

        Par défaut pour le système FLY, le nom d'utilisateur est : `fly`, le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />

        * Connexion réussie

    </TabItem>
