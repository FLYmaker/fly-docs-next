## Installation du pilote

### Jumpers de la carte mère pour le pilote

* **Mode STEP/DIR.** Dans ce mode, la subdivision est réglée à l'aide de jumpers et le courant est ajusté via un potentiomètre. Le plus commun est le pilote A4988, qui nécessite de consulter le tableau de configuration de subdivision fourni par le fabricant et de régler la subdivision à l'aide des jumpers.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilote peuvent communiquer de manière asynchrone en série avec le contrôleur principal via UART, permettant la configuration de la subdivision, du courant de fonctionnement, du mode silencieux, etc., par modification du fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode sont : TMC5160, TMC2230, etc. Ces puces de pilote communiquent de manière synchrone en série avec le contrôleur principal via SPI, permettant également la configuration de la subdivision, du courant de fonctionnement, du mode silencieux, etc., par modification du fichier de configuration.

### Installation du pilote

* Avant l'installation du pilote, il est nécessaire d'effectuer des vérifications appropriées pour éviter d'endommager le pilote ou la carte mère.
* **Pilote FLY :** Si vous n'utilisez pas la fonction de retour à l'origine sans fin, veuillez mettre l'interrupteur DIP à la position 1 ; à l'inverse, si vous souhaitez utiliser cette fonction, veuillez mettre l'interrupteur DIP à la position ON.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.png').default} size="40%" align="center" />

    :::danger

    * **Attention!!! Lors de l'installation du module pilote, assurez-vous que la direction d'insertion est correcte, c'est-à-dire que la broche EN est dans le coin supérieur gauche, sinon le pilote ou même la carte mère pourrait être endommagé!!! Assurez-vous d'appliquer correctement un dissipateur thermique au pilote!!!**

    :::
