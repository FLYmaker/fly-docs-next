# Compilation et programmation du firmware

    :::tip

    * CDV utilise par défaut une carte TF pour la programmation du firmware
    * Veuillez préparer une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec de programmation, veuillez utiliser de préférence les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être de `class10` ou supérieure, il est recommandé d'utiliser des marques comme SanDisk pour les cartes TF

    :::

    * Appuyez sur la touche `Q`, lorsqu'apparaît **Save configuration**, appuyez ensuite sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez revenir à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation se termine avec succès, le message suivant sera affiché :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
