    * Processeur ARM Cortex-M4 32 bits à 168 MHz, puce STM32F407ZGT6
    * Supporte la sélection de la tension d'entraînement, par défaut 5V
    * Supporte le firmware RRF

## Spécifications du produit

    * Processeur ARM Cortex-M4 32 bits à 168 MHz, puce STM32F407ZGT6
    * Supporte la connexion par bus CAN, transmission de données plus stable, latence réduite, connexion plus stable
    * Firmware : Reprap/Klipper
    * Support de drivers : A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Modes de pilotage supportés : TMC : UART/SPI
    * Affichage : écran tactile série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Avec le firmware repap, vous pouvez utiliser un module WiFi ESP32 en extension, supportant 1.6m/s
    * Conception unique de la carte PWM Fan MOS, remplaçable directement en cas de dommage, réduisant la difficulté de maintenance pour le client
    * PCB avec une épaisseur de cuivre de 2OZ, pouvant supporter un courant plus important
    * 3 interfaces ADC, supportant jusqu'à trois thermistances
    * 6 interfaces IO, pouvant être utilisées pour le nivellement et les fins de course
    * 4 interfaces pour les bâtons chauffants, supportant jusqu'à quatre bâtons chauffants
