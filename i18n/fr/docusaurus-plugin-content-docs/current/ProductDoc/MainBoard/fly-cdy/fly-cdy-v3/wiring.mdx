## Installation des pilotes

### Jumper de la carte mère

* **Mode STEP/DIR.** Dans ce mode, utilisez un cavalier pour régler la microstepping et ajustez le courant avec un potentiomètre. Les pilotes les plus courants sont les A4988, pour lesquels il faut consulter le tableau de configuration de microstepping fourni par le fabricant et utiliser les cavaliers pour régler la microstepping.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode incluent : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilote peuvent communiquer de manière asynchrone en série avec le contrôleur principal via UART, permettant de configurer la microstepping, le courant de fonctionnement, le mode silencieux, etc., en modifiant le fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode incluent : TMC5160, TMC2230, etc. Ces puces de pilote communiquent de manière synchrone en série avec le contrôleur principal via SPI, permettant également de configurer la microstepping, le courant de fonctionnement, le mode silencieux, etc., en modifiant le fichier de configuration.

### Installation des pilotes

* Avant d'installer les pilotes, il est nécessaire de les vérifier pour éviter d'endommager les pilotes ou la carte mère.
* **Pilotes FLY :** Si vous n'utilisez pas la fonction de retour à l'origine sans fin, mettez l'interrupteur à bascule à la position 1; sinon, pour utiliser cette fonction, mettez l'interrupteur à bascule à la position ON.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.webp').default} size="40%" align="center" />

    :::danger

    * **Attention !!! Lors de l'installation des modules de pilote, assurez-vous que la direction d'insertion est correcte, c'est-à-dire que la broche EN est dans le coin supérieur gauche, sinon cela pourrait endommager le pilote voire la carte mère !!! N'oubliez pas de coller des dissipateurs thermiques sur les pilotes !!!**

    :::
