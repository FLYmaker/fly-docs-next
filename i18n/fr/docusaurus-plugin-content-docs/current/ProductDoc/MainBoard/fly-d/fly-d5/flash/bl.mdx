# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip

    * D5 utilise une connexion USB
    * D5 est préchargé avec le firmware Katapult à partir du 1er septembre 2024
    :::

## Entrer en mode de flashe

    :::tip

    À gauche se trouve le bouton `BOOT`, à droite le bouton `RESET`

    :::

    * Il y a deux méthodes pour entrer en mode de flashe
    * Première méthode : lorsque la carte mère est complètement déconnectée, appuyez sur le bouton BOOT puis connectez le câble de données d'une part à la carte mère et d'autre part à votre appareil
    * Deuxième méthode : après avoir connecté le câble de données d'une part à la carte mère et d'autre part à votre appareil, appuyez sur le bouton BOOT, puis sur le bouton RST, relâchez le bouton RST et enfin relâchez le bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/img/boot.png').default} size="50%" align="center" />

            :::

            * Voici la commande pour flasher le firmware
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/D/FLY_D5.bin
                ```

            5. Si le message ``File downloaded successfully`` apparaît, le flashe a réussi
