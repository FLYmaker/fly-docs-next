# Instructions pour la programmation du firmware

:::danger À savoir
* La programmation du firmware de la carte mère nécessite de flasher Klipper via Katapult
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, il n'est pas nécessaire de procéder à la programmation du firmware Katapult à la fin, c'est juste au cas où le firmware Katapult de la carte mère n'a pas été programmé avec succès
* Vous pouvez appuyer deux fois rapidement sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est bien entrée dans le firmware Katapult
:::

## Comment déterminer le firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de programmer qu'un seul type de firmware

    :::
    
    * Le D5 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs appareils USB sont connectés, cela peut affecter la transmission de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Peut connecter plusieurs appareils sans occuper un port USB
        * Inconvénients : Peut causer des erreurs comme un dépassement de temps lors de l'utilisation avec `KlipperScreen`.
