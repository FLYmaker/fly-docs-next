# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * D7 utilise une connexion USB
    * D7 est préchargé avec le firmware Katapult à partir du 1er septembre 2024

    :::

## Entrer en mode flash

    :::tip

    À gauche se trouve le bouton `BOOT`, à droite le bouton `RESET`

    :::

    * Il existe deux méthodes pour entrer en mode flash
    * Première méthode : lorsque la carte mère est complètement déconnectée, maintenez enfoncé le bouton BOOT puis connectez le câble de données à la carte mère et à votre appareil
    * Deuxième méthode : après avoir connecté le câble de données à la carte mère et à votre appareil, maintenez enfoncé le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
