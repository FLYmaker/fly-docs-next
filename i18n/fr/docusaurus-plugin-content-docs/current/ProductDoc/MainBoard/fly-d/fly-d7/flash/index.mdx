# Instructions de gravure du firmware

:::danger A savoir
* La gravure du firmware de la carte mère nécessite l'utilisation de Katapult pour graver Klipper
* Si vous êtes certain que la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de graver le firmware Katapult à la fin, c'est juste une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été gravé avec succès
* Vous pouvez rapidement appuyer deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère commence à clignoter, si c'est le cas, cela indique que la carte mère est entrée en mode Katapult
:::

## Comment déterminer le firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Le D7 peut se connecter à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs périphériques USB sont connectés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware de pontage USB</Button>
        * Avantages : Permet de connecter plusieurs appareils sans occuper les ports USB
        * Inconvénients : Peut causer des erreurs de timeout de retour à l'origine si utilisé avec `KlipperScreen`.
