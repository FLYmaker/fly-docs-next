# Introduction

> Fly-D7 est une carte mère pour imprimante 3D haute performance lancée par l'équipe de développement de la carte mère Fly3D de Guangzhou Maillan Electronic Technology Co., Ltd, supportant jusqu'à 7 moteurs pas à pas.


    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.webp').default} size="60%" align="center" />

## **1. Caractéristiques du produit**

    * Utilise un microcontrôleur de la série ARM Cortex-M0+ STM32F072RBT6 à 48 MHz
    * Supporte la connexion par bus CAN, pour une transmission de données plus stable, un retard réduit et une connexion plus fiable
    * Support de drivers : A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Fonction DIAG intégrée, permettant de supporter le retour sans fin par simple insertion et retrait d'un chapeau de cavalier

## 2. Paramètres du produit

    * MCU : STM32F072RBT6
    * Firmware : Klipper/Marlin
    * Support de l'alimentation DC 12-24V, pour une alimentation plus stable, sans besoin de module de réduction
    * Modes de fonctionnement des drivers supportés : SPI, UART, STEP/DIR
    * Drivers supportés : TMC5160Pro externe, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
    * Interfaces d'extension : BLTouch (Servos, Probe), UART X1
    * Ventilateurs : Ventilateurs contrôlables à deux fils X2, ventilateurs en parallèle X2, ventilateurs toujours actifs X2
    * Capteurs de température : 3 voies NTC 100K
    * Interfaces de chauffage : 3 voies contrôlables supportant 10A
    * Structures de machine supportées : Cartesian, Delta, Kossel, Ultimaker, CoreXY
