# Introduction des périphériques embarqués

    * Lorsque le sélecteur est positionné vers la droite, l'interface USB à côté du port Type-C est activée
    * Lorsque le sélecteur est positionné vers la gauche, l'interface RS232 à côté du port Type-C est activée
