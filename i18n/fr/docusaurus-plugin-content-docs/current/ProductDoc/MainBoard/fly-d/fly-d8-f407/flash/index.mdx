# Instructions pour la gravure du firmware

:::danger À savoir
* La gravure du firmware de la carte mère nécessite l'utilisation de Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà Katapult installé, il n'est pas nécessaire de suivre les instructions de gravure de Katapult à la fin, ce n'est qu'une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été gravé avec succès
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est en mode Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * La D8 peut être connectée à l'ordinateur hôte par les moyens suivants
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs dispositifs USB sont connectés, cela peut affecter la transmission de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Peut connecter plusieurs dispositifs sans occuper de port USB
        * Inconvénients : L'utilisation conjointe avec `KlipperScreen` peut provoquer des erreurs telles que des timeouts lors du retour à l'origine.
