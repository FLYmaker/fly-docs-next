# Pont CAN

Le bus CAN est un protocole de communication série utilisé pour des applications en temps réel. Il peut utiliser un câble torsadé pour transmettre des signaux et est l'un des bus de terrain les plus largement utilisés au monde. Le protocole CAN est utilisé pour la communication entre divers composants dans les automobiles, remplaçant ainsi les faisceaux de câbles coûteux et encombrants. La robustesse de ce protocole s'étend à d'autres applications d'automatisation et industrielles. Les caractéristiques du protocole CAN incluent une communication de données sérielles avec intégrité, support en temps réel, une vitesse de transmission allant jusqu'à 1 Mb/s, ainsi qu'un adressage de 11 bits et des capacités de détection d'erreurs.
Les imprimantes 3D utilisent le bus CAN pour réduire le nombre de câbles reliant la tête d'impression à la carte mère. Alors qu'il fallait auparavant une dizaine de câbles, avec le CAN, seulement quatre câbles sont nécessaires, ce qui réduit considérablement le nombre de câbles et simplifie le câblage.
Ce chapitre ne donne qu'un aperçu succinct de l'utilisation du pont CAN.

## Utilisation de la carte outil pour se connecter à la carte mère via CAN

    * Veuillez noter que le câble d'alimentation de la carte outil ne se connecte pas au port CAN, mais à la ligne d'alimentation 12V-24V.

    * Veuillez noter qu'il faut connecter le cavalier de 120Ω.
