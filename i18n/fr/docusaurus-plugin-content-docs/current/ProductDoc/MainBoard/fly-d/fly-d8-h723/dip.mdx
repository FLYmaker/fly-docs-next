# Introduction aux périphériques embarqués

    * Lorsque le commutateur est positionné vers la droite, il ouvre le port USB à côté du port Type-C
    * Lorsque le commutateur est positionné vers la gauche, il ouvre l'interface RS232 à côté du port Type-C
