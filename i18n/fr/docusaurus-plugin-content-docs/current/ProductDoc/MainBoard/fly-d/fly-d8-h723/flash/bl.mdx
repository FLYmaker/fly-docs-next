## Entrer en mode flash

    :::tip

    À gauche se trouve le bouton `RST`, à droite le bouton `BOOT`

    :::

    * Il y a deux méthodes pour entrer en mode flash
    * La première : avec la carte mère complètement débranchée, maintenez le bouton BOOT enfoncé puis connectez un câble USB avec transfert de données d'un côté à la carte mère et de l'autre à votre appareil
    * La deuxième : après avoir connecté le câble USB avec transfert de données d'un côté à la carte mère et de l'autre à votre appareil, maintenez le bouton BOOT enfoncé, puis appuyez sur le bouton RST, relâchez le bouton RST en dernier et enfin relâchez le bouton BOOT
