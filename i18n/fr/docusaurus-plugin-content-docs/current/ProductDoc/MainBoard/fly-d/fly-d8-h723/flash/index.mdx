# Instructions pour la programmation du firmware

:::danger À savoir
* La programmation du firmware de la carte mère nécessite l'utilisation de Katapult pour flasher Klipper
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, vous n'avez pas besoin de suivre les dernières étapes pour programmer le firmware Katapult, c'est juste au cas où la programmation du firmware Katapult sur la carte mère échouerait
* Vous pouvez rapidement appuyer deux fois sur le bouton de réinitialisation pour voir si un LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est passée en mode Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Notez que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * La D8 peut se connecter à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : si plusieurs appareils USB sont connectés, cela peut affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : peut connecter plusieurs appareils sans occuper de port USB
        * Inconvénients : pourrait causer des erreurs comme des timeouts de retour à l'origine lors de l'utilisation avec `KlipperScreen`.
