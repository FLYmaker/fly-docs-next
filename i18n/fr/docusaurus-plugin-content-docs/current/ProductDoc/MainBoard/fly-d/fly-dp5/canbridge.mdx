# Pontage CAN

Le bus CAN est un protocole de communication série utilisé pour des applications en temps réel, qui peut utiliser un câble torsadé pour transmettre des signaux. C'est l'un des bus de terrain les plus largement utilisés au monde. Le protocole CAN est utilisé pour la communication entre divers composants dans les automobiles, remplaçant ainsi des faisceaux de câbles coûteux et encombrants. La robustesse de ce protocole étend son utilisation à d'autres applications d'automatisation et industrielles. Les caractéristiques du protocole CAN incluent une communication de données sérielles complète, le support en temps réel, un débit de transmission allant jusqu'à 1Mb/s, ainsi qu'une capacité d'adressage de 11 bits et de détection d'erreurs.
Les imprimantes 3D utilisent le bus CAN pour réduire le nombre de connexions entre la tête d'impression et la carte mère. Alors qu'il fallait auparavant plusieurs câbles, avec le CAN, seulement quatre câbles sont nécessaires, réduisant considérablement le nombre de câbles et facilitant le câblage.
Ce chapitre donne un aperçu succinct de l'utilisation du pontage CAN.

## Utilisation du bus CAN pour connecter le module d'outils à la carte mère

    * Veuillez noter que le câble d'alimentation du module d'outils ne se connecte pas au port CAN, mais au câble d'alimentation 12V-24V.
