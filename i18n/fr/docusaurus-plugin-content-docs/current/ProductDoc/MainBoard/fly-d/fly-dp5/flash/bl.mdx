## Entrer en mode de flashage

    :::tip

    Le bouton à gauche est le `BOOT`, celui à droite est le `RESET`

    :::

    * Il y a deux méthodes pour entrer en mode de flashage
    * La première : lorsque la carte mère est complètement débranchée, maintenez enfoncé le bouton BOOT puis connectez un câble de données à la carte mère et à votre appareil
    * La seconde : après avoir connecté un câble de données à la carte mère et à votre appareil, maintenez enfoncé le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
