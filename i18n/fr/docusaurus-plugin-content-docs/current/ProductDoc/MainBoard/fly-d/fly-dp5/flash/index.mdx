# Instructions pour la mise à jour du firmware

:::danger À savoir
* La mise à jour du firmware de la carte mère doit se faire via Katapult pour flasher Klipper
* Si vous êtes sûr que la carte mère a déjà été flashée avec Katapult, il n'est pas nécessaire de suivre la dernière étape de mise à jour du firmware Katapult, c'est juste une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été flashé correctement
* Vous pouvez appuyer deux fois rapidement sur le bouton de réinitialisation et vérifier si un LED sur la carte mère commence à clignoter, ce qui indique que la carte mère est en mode Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Le DP5 peut se connecter à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs dispositifs USB sont connectés, cela peut affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Permet de connecter plusieurs dispositifs sans occuper les ports USB
        * Inconvénients : Peut entraîner des erreurs comme des timeouts lors de l'utilisation avec `KlipperScreen`.
