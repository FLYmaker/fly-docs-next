# Compilation et flashage du firmware

    :::tip

    * L'E3 Pro utilise par défaut une carte TF pour le flashage du firmware
    * Veuillez préparer une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marque inconnue peuvent entraîner un échec du flashage, essayez d'utiliser les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être de `class10` ou supérieure, il est recommandé d'utiliser des cartes TF de marques comme SanDisk

    :::

    * Appuyez sur la touche `Q`, puis lorsque **Save configuration** apparaît, appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * La compilation est réussie si la sortie affiche le contenu suivant :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
