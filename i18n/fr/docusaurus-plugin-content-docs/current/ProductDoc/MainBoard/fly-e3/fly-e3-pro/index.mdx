## Caractéristiques du produit

    * Processeur 32 bits ARM Cortex-M4 série à 168 MHz, puce STM32F407ZGT6
    * Supporte le firmware RRF

## Spécifications du produit

    * Processeur 32 bits ARM Cortex-M4 série à 168 MHz, puce STM32F407ZGT6
    * Firmware : Reprap/Klipper
    * Intègre cinq drivers TMC2209
    * Affichage : écran tactile en série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 port IN haute tension, supportant les capteurs de nivellement automatique : BLTouch, capteurs de proximité PL08N, Klicky, Voron Tap, etc.
    * En utilisant le firmware reprap, il peut utiliser un module WiFi ESP32, supportant une vitesse de 1,6 m/s
    * Conception unique de la carte MOS PWM Fan, endommagée peut être directement remplacée, réduisant la difficulté de maintenance pour le client, quatre interfaces de ventilateur 2pin
    * PCB en cuivre 2OZ, capable de supporter un courant plus élevé
    * 2 interfaces ADC, pouvant supporter jusqu'à deux sondes thermiques
    * 4 interfaces IO, pouvant être utilisées pour le nivellement et les fins de course
    * 3 interfaces de bâton chauffant, pouvant supporter jusqu'à trois bâtons chauffants
