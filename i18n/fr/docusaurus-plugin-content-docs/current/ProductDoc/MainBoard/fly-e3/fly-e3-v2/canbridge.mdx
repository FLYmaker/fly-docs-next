# Pontage CAN

Le bus CAN est un protocole de communication série utilisé pour des applications en temps réel. Il peut utiliser un câble torsadé pour transmettre des signaux et est l'un des bus de terrain les plus utilisés au monde. Le protocole CAN est utilisé pour la communication entre divers composants d'une voiture, remplaçant ainsi des faisceaux de câbles coûteux et encombrants. La robustesse de ce protocole lui permet également de s'étendre à d'autres applications d'automatisation et industrielles. Les caractéristiques du protocole CAN incluent la communication de données sérielles avec intégrité, le support en temps réel, un taux de transmission pouvant atteindre 1Mb/s, ainsi qu'une adresse sur 11 bits et des capacités de détection d'erreurs.
Les imprimantes 3D utilisent le bus CAN pour réduire le nombre de connexions entre la tête d'impression et la carte mère. Initialement nécessitant une douzaine de fils, avec CAN, seulement quatre fils sont nécessaires, ce qui réduit considérablement le nombre de câbles et simplifie le câblage.
Ce chapitre donne un aperçu succinct de l'utilisation du pontage CAN.
