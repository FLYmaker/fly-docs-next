# Firmware BL

    :::tip

    * Veuillez noter que le `bootloader` est déjà pré-installé à l'usine, aucune réinstallation n'est nécessaire.

    :::

## Entrer en mode de programmation

    :::tip

    À droite se trouve le bouton `BOOT`, à gauche le bouton `RESET`.

    :::

    * Il y a deux méthodes pour entrer en mode de programmation :
    * La première : avec le plateau complètement déconnecté, maintenez le bouton BOOT enfoncé puis connectez le câble de données d'un côté au plateau et de l'autre à votre appareil.
    * La deuxième : après avoir connecté le câble de données d'un côté au plateau et de l'autre à votre appareil, maintenez le bouton BOOT enfoncé, appuyez sur le bouton RST, puis relâchez le bouton RST et enfin relâchez le bouton BOOT.
