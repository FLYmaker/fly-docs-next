    :::tip

    * E3 V2 utilise par défaut une carte TF pour la gravure du firmware
    * Veuillez préparer une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec de la gravure, veuillez utiliser de préférence les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être de `class10` ou supérieure, il est recommandé d'utiliser des marques comme SanDisk pour les cartes TF

    :::

# Instructions pour la gravure du firmware


## Comment déterminer de quel type de firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * E3 V2 peut se connecter à l'hôte via les méthodes suivantes
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs périphériques USB sont connectés, cela pourrait affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware USB bridge</Button>
        * Avantages : Permet de connecter plusieurs appareils sans occuper les ports USB
        * Inconvénients : Peut entraîner des erreurs comme un dépassement de temps de retour à zéro lorsqu'il est utilisé avec `KlipperScreen`.
