# Compilation et flashage du firmware

    :::tip

    * Par défaut, le 407ZG utilise une carte TF pour le flashage du firmware
    * Préparez une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec du flashage, veuillez utiliser de préférence les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être de `class10` ou supérieure, il est recommandé d'utiliser des cartes TF de marques comme SanDisk

    :::
