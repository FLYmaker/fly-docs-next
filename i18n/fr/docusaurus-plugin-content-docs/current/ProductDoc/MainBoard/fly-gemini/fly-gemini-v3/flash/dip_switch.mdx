# Introduction aux commutateurs DIP

:::tip
FLY Gemini se connecte à l'ordinateur hôte via des commutateurs DIP, une mauvaise position empêchera son fonctionnement normal.
:::

:::danger À savoir
Veuillez noter que les commutateurs DIP doivent être positionnés correctement pour permettre la connexion à l'ordinateur hôte.
:::

## Connexion à l'ordinateur hôte embarqué

    * Positionnez les commutateurs DIP `1`,`2` vers la droite pour connecter le dispositif inférieur à l'ordinateur hôte via USB intégré.
    * Positionnez les commutateurs DIP `3`,`4` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'ordinateur hôte.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * Après avoir positionné les commutateurs DIP de cette manière, le port USB inférieur sera occupé par le dispositif inférieur, veuillez ne pas y connecter d'autres appareils.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## Connexion à l'ordinateur hôte externe

    * Positionnez les commutateurs DIP `3`,`4` vers la droite pour connecter le dispositif inférieur à l'ordinateur hôte via USB intégré.
    * Positionnez les commutateurs DIP `1`,`2` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'ordinateur hôte.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
