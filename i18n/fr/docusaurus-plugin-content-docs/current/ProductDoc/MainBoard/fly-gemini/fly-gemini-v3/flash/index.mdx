# Instructions pour la programmation du firmware

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de programmer qu'un seul type de firmware

    :::
    
    * Gemini V3 peut se connecter à l'hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs appareils USB peuvent affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Peut connecter plusieurs appareils sans occuper de port USB
        * Inconvénients : Peut provoquer des erreurs comme un délai de retour à zéro si utilisé avec `KlipperScreen`.
