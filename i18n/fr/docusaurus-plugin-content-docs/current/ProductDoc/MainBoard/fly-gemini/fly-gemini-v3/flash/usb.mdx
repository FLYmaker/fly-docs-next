# Compilation du firmware USB

## Commencer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, puis sur **Enregistrer la configuration** en appuyant sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Si vous voyez le message suivant, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Programmation du firmware

:::tip
Exécutez la commande suivante pour programmer le firmware
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
