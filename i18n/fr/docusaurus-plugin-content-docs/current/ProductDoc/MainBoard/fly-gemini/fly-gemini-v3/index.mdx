# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.png').default} size="80%" align="center" />

## Caractéristiques du produit
    * Embarque un ordinateur hôte, peut remplacer le Raspberry Pi pour fonctionner comme un ordinateur hôte, il dispose de 1 Go de mémoire DDR3 embarquée.
    * Amélioration de la protection de l'alimentation 5V, réduisant les risques d'endommagement des puces.
    * Suppression du réglage de la carte mémoire du MCU, facilitant le flashing du firmware.
    * Le système est préinstallé avec klipper, moonraker, mainsail, fluidd, klipper-screen, Crowsmest, le plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'effectuer des opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)


## Spécifications du produit

    * CPU : Puce Allwinner H5 haute performance, 4 cœurs 64 bits Cortex-A53
    * GPU : GPU haute performance 6 cœurs Mali 450, taux de remplissage des pixels supérieur à 2,7 Gpixels/s
    * RAM : 1 Go DDR3 (partagé avec le GPU)
    * ROM : Supporte jusqu'à 128 Go SDCARD
    * Périphériques : SPI x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1
    * Interfaces : Interface M.2 embarquée, peut étendre WIFI, eMMC, etc. (protocole privé, ne connectez pas de dispositifs M.2 non spécifiques à FLY π)
    * Supporte une alimentation en courant continu de 12 à 24V, pour une alimentation plus stable, éliminant le besoin d'un module de réduction de tension
    * Un port pour ventilateur contrôlable 5V est prévu pour le refroidissement de l'ordinateur hôte
    * Port d'extension CAN préservé
    * Le système est préinstallé avec klipper, moonraker, mainsail, fluidd, klipper-screen, Crowsmest, le plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'effectuer des opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
    * Drivers : 4 positions de drivers haute tension
    * MCU : STM32F405
