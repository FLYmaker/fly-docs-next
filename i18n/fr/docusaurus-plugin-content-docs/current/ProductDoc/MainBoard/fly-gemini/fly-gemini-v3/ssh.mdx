# Connexion SSH à l'hôte


* Vous pouvez vous connecter à l'appareil via un port série ou WIFI
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a été flashé avec **le système de l'hôte correspondant**
* **Si l'hôte ne démarre pas correctement, veuillez démonter les périphériques tels que les moteurs, les fins de course, les ventilateurs, etc. !!**
* **Enfin, veuillez vous connecter au SSH via le réseau pour faciliter la compilation du firmware !!**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion de l'hôte via WIFI ou câble Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Connexion SSH à l'hôte via IP réseau

        * Veuillez vous connecter à **l'interface administrateur de votre routeur**
        * Assurez-vous que le système est **correctement démarré**
        * Assurez-vous de **connaître le nom d'utilisateur et le mot de passe de l'hôte**, pour le système FLY, le nom d'utilisateur par défaut est **fly**, et le mot de passe par défaut est **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter au SSH via l'IP
        * Ouvrez MobaXterm et configurez-le comme suit, remplissez l'IP que vous avez vue précédemment ou vérifiez-la dans l'interface administrateur du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Remarque : ici, à l'étape 4, vous entrez le nom d'utilisateur de l'hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * À la première connexion, une fenêtre s'ouvrira, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Vous serez invité à entrer le mot de passe, entrez le mot de passe de l'hôte et appuyez sur Entrée

        **Remarque : ici, le mot de passe ne sera pas visible, entrez-le et appuyez directement sur Entrée**

        Pour le système FLY, le nom d'utilisateur par défaut est : `fly` , et le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />
