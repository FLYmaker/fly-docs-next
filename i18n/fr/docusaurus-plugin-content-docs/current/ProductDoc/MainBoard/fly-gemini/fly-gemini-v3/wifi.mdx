# Connexion au WIFI


## Installer l'antenne sur le Gemini

    * L'endroit indiqué par la flèche rouge est l'interface de l'antenne WIFI
    * La prise de l'antenne WIFI est `ipex1`
    * Installez l'antenne sur le Gemini, sinon vous pourriez ne pas pouvoir vous connecter à Internet en raison d'un signal faible.
