## Installation des pilotes

### Sélecteurs de mode de la carte mère

* **Mode STEP/DIR.** Dans ce mode, utilisez des cavaliers pour régler le micro-pas et ajustez le courant via un potentiomètre. Le pilote A4988 est le plus courant, nécessitant la consultation du tableau de configuration des micro-pas fourni par le fabricant pour régler le micro-pas.
* **Mode UART.** Les pilotes couramment utilisés dans ce mode incluent : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilote peuvent communiquer de manière asynchrone en série avec le contrôleur principal, permettant de configurer le micro-pas, le courant de fonctionnement, le mode silencieux, etc., via le fichier de configuration.
* **Mode SPI.** Les pilotes couramment utilisés dans ce mode incluent : TMC5160, TMC2230, etc. Ces puces de pilote communiquent de manière synchrone en série avec le contrôleur principal, permettant également la configuration du micro-pas, du courant de fonctionnement, du mode silencieux, etc., via le fichier de configuration.

### Installation des pilotes

* Avant l'installation, vérifiez les pilotes pour éviter d'endommager les pilotes ou la carte mère.
* **Pilotes FLY :** Si vous n'utilisez pas la fonction de retour à l'origine sans fin, réglez l'interrupteur DIP sur la position 1; à l'inverse, si vous souhaitez utiliser cette fonction, réglez l'interrupteur DIP sur la position ON.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.png').default} size="40%" align="center" />

    :::danger

    * **Attention !!! Assurez-vous que les modules de pilote sont insérés dans le bon sens, c'est-à-dire la broche EN en haut à gauche, sinon cela endommagera le pilote voire la carte mère !!! N'oubliez pas d'appliquer un bon radiateur sur le pilote !!!**

    :::
