# Pontage CAN

Le bus CAN est un protocole de communication série utilisé pour des applications en temps réel, qui peut utiliser un câble torsadé pour transmettre des signaux. Il s'agit de l'un des bus de terrain les plus largement utilisés au monde. Le protocole CAN est utilisé pour la communication entre divers composants dans les véhicules automobiles, remplaçant ainsi les faisceaux de câbles coûteux et encombrants. La robustesse de ce protocole l'a étendu à d'autres applications d'automatisation et industrielles. Les caractéristiques du protocole CAN incluent une communication de données sérielle avec intégrité, support en temps réel, une vitesse de transmission allant jusqu'à 1Mb/s, une adresse de 11 bits et des capacités de détection d'erreurs.
Les imprimantes 3D utilisent le bus CAN pour réduire le nombre de fils connectant la tête d'impression à la carte mère. À l'origine, plusieurs dizaines de fils étaient nécessaires, mais avec le CAN, seulement quatre fils suffisent, ce qui réduit considérablement la complexité du câblage.
Ce chapitre donne un aperçu succinct de l'utilisation du pontage CAN.

## Utilisation du pontage CAN pour connecter la carte mère à la carte outil

    * Veuillez noter que le câble d'alimentation de la carte outil ne se connecte pas au port CAN, mais au câble d'alimentation 12V-24V.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/canbridge.png').default} size="80%" align="center" />
    * Veuillez noter que le cavalier de 120Ω doit être connecté.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/120.png').default} size="80%" align="center" />

## Flasher le firmware de la carte mère

* Assurez-vous que le firmware flashé sur la carte mère D5 est configuré pour `Firmware de pontage CAN via USB`.
