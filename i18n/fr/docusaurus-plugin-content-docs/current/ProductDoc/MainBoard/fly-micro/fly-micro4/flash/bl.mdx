# Firmware Katapult

    :::tip

    * Veuillez noter que `katapult` est déjà pré-installé en usine, il n'est pas nécessaire de le réinstaller.

    :::

## Mode de programmation

    :::tip

    Le bouton entouré est le `Bouton BOOT`.

    :::

    * Pour entrer en mode de programmation, il faut que la carte mère soit complètement débranchée, puis appuyez sur le bouton BOOT et branchez le câble de données d'un côté à la carte mère et de l'autre à votre appareil.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/boot.png').default} size="100%" align="center" />
