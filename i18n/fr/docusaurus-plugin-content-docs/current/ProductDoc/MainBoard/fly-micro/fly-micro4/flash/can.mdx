# Firmware USB vers CAN

## Commencer la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Sélectionnez les GPIO pins à définir au démarrage du microcontrôleur, entrez : gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, puis sur **Save configuration**, et enfin sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prendra un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation se termine avec succès, vous verrez le message suivant :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
