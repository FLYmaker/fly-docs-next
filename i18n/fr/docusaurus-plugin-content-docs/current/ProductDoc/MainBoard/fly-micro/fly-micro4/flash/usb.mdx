# Compilation du firmware USB

    :::tip

    * Le Micro4 est livré avec le firmware Katapult pour la connexion USB déjà préinstallé, donc il n'est pas nécessaire de le reflasher.

    :::

## Commencer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Choisissez les broches GPIO à définir au démarrage du micro-contrôleur, entrez : gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, lorsque **Save configuration** apparaît, appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Maintenant, la configuration devrait être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un peu de temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si le message suivant apparaît, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
