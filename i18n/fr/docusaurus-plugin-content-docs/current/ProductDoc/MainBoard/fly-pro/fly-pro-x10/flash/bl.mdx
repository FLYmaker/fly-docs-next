# Firmware BL

    :::tip

    * Veuillez noter que le bootloader est déjà pré-installé en usine, pas besoin de le flasher à nouveau

    :::

## Mode de programmation

    * Il existe deux méthodes pour entrer en mode de programmation
    * La première : Avec la carte mère complètement débranchée, court-circuitez BOOT puis connectez un câble de données à la carte mère et à votre appareil
    * La deuxième : Après avoir connecté un câble de données à la carte mère et à votre appareil, court-circuitez BOOT, appuyez sur la touche RST, relâchez RST et enfin retirez le court-circuit de BOOT

            4. Flasher le firmware

            :::tip

            Il suffit de flasher un seul firmware

            :::


            * Voici la commande pour flasher le firmware katapult
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * Voici la commande pour flasher le firmware bootloader
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. Si vous voyez apparaître le message indiqué par la flèche ci-dessous ``File downloaded successfully``, le flashage a réussi
