# Instructions pour la gravure du firmware

    :::warning Notes importantes

    * Le PRO X10 est préchargé avec Katapult
    * Une fois le BL gravé, il n'est pas nécessaire de le graver à nouveau
    * Le BL du PRO X10 existe en deux versions : le firmware Katapult et le BL gravé via une carte TF
    * Si nécessaire, vous pouvez choisir de re-graver le <Button variant="contained" disableElevation href="./bl">firmware BL</Button>

    :::
    
    :::danger À savoir
    * La gravure du firmware de la carte mère nécessite l'utilisation de Katapult pour graver Klipper
    * Si vous êtes sûr que la carte mère a déjà Katapult gravé, il n'est pas nécessaire de graver à nouveau le firmware Katapult, c'est juste une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été gravé correctement
    * Vous pouvez rapidement appuyer deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote. Si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult
    :::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de graver qu'un seul type de firmware

    :::
    
    * Le Pro X10 peut se connecter à l'ordinateur via les méthodes suivantes
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs périphériques USB sont connectés, cela peut affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Peut connecter plusieurs périphériques sans occuper les ports USB
        * Inconvénients : Peut causer des erreurs de timeout lors de l'utilisation avec `KlipperScreen`.
