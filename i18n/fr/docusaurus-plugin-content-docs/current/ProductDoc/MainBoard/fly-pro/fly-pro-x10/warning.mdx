# Remarques sur la carte mère

## Voyant d'alimentation
    * Lors de l'utilisation de l'alimentation par Type-C sans connecter d'autres sources d'alimentation de la carte mère, les voyants `SYS_5V` et SYS_3.3V devraient s'allumer
    * Après avoir connecté les sources d'alimentation de la carte mère `HVIN`, `VIN`, `BOARO`, `POWER`, le voyant d'alimentation de la carte mère devrait rester allumé

## Pilotes

    * TP5160 et TP2209 ne diffèrent que par leur nom, et n'ont pas besoin de cavaliers pour choisir le mode de pilote
    * Les ports de pilote ne peuvent installer que les pilotes TP et pas d'autres dispositifs

## Port de limite
    * ProX10 supporte le choix entre une alimentation de 5V et 12V, veuillez noter que si vous utilisez un interrupteur de proximité, il est nécessaire d'installer une diode pour réduire la tension

## Logiciel de contrôle

    * ProX10 supporte l'utilisation d'un logiciel de contrôle externe, après avoir sélectionné la connexion Type-C via les cavaliers, il peut être utilisé

## Programmation du firmware

    * La carte mère supporte la programmation via une carte TF, mais par défaut, elle est livrée avec `Katapult`
    * Si vous avez besoin de programmer via une carte TF, il est nécessaire de reprogammer le BL
