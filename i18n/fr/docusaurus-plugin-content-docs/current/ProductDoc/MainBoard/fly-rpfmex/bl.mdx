# Firmware Katapult



## Entrer en mode de flashage

    :::tip

    Le bouton encerclé est le `bouton BOOT`.

    :::

    * Pour entrer en mode de flashage, il faut que la carte soit complètement déconnectée. Appuyez sur le bouton BOOT et connectez ensuite le câble de données à la carte et à votre appareil.
