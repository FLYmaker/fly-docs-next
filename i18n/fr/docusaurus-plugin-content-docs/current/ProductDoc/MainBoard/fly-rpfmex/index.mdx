## Caractéristiques du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 4 drivers 2209 intégrés


## Spécifications du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware : Klipper
    * Supporte l'alimentation CC de 12 à 24V, pour une alimentation plus stable, éliminant le besoin d'un module de réduction de tension
    * Modes de fonctionnement des drivers : UART, STEP/DIR
    * Interfaces d'extension : ventilateurs contrôlables *2, RGB *1, fins de course *3
