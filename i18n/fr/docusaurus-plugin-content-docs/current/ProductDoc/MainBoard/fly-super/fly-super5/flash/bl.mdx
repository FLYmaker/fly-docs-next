# Firmware BL

    :::tip

    * Veuillez noter que le `bootloader` est pré-flashé en usine, il n'est pas nécessaire de le reflasher.

    :::

## Entrer en mode de programmation

    :::tip

    À droite se trouve le `saut de BOOT`, à gauche le `bouton RESET`.

    :::

    * Il y a deux méthodes pour entrer en mode de programmation
    * Première méthode : avec la carte mère complètement déconnectée, maintenez la touche BOOT enfoncée puis connectez le câble de données à la carte mère et à votre appareil.
    * Deuxième méthode : après avoir connecté le câble de données à la carte mère et à votre appareil, maintenez la touche BOOT enfoncée, appuyez sur la touche RST, puis relâchez la touche RST et enfin relâchez la touche BOOT.

            4. Flasher le firmware

            :::tip

            Il suffit de flasher un seul firmware.

            :::


            * Voici la commande pour flasher le firmware :
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/SUPER/SUPER5PRO.bin
                ```

            5. Si le message ``File downloaded successfully`` apparaît comme indiqué par la flèche, le flashage est réussi.
