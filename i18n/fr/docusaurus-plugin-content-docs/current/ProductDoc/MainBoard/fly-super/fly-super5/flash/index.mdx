    :::tip

    * Le Super5 Pro utilise par défaut une carte TF pour la gravure du firmware
    * Veuillez préparer une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec de la gravure, il est donc préférable d'utiliser les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être au moins de `class10`, il est recommandé d'utiliser des cartes TF de marques comme SanDisk

    :::

# Explication de la gravure du firmware

    :::tip

    * Le Super5 Pro est livré avec un bootloader préinstallé sur la carte TF, pas besoin de le reflasher

    :::


## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Le Super5 Pro peut se connecter à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : si plusieurs dispositifs USB sont connectés, cela peut affecter la transmission de données
    * <Button variant="contained" disableElevation href="./can">Firmware USB pont</Button>
        * Avantages : peut se connecter à plusieurs appareils sans occuper de port USB
        * Inconvénients : peut provoquer des erreurs comme des timeouts de retour à zéro si utilisé avec `KlipperScreen`.
