# Caractéristiques du produit

    * Utilise un microcontrôleur ARM Cortex-M7 STM32H723ZGT6 avec une fréquence de 550 MHz et 32 bits
    * Firmware : Marlin 2.0/Reprap/Klipper
    * Support des pilotes : A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Modes de pilotage supportés : TMC : UART/SPI
    * 5 emplacements pour pilotes haute tension, support de 5 moteurs indépendants
    * Affichage : Écran tactile en série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 entrée haute tension, supporte les capteurs de mise à niveau automatique : BLTouch, capteurs de proximité PL08N, Klicky, Voron Tap, etc.
    * Alimentation des interrupteurs de fin de course optionnelle : 5V/12V/24V
    * Avec le firmware Reprap, utilise le module WiFi ESP32 intégré, supportant 1,6 m/s
    * Design unique de la carte MOS pour ventilateur PWM, peut être remplacée directement en cas de dommage, réduisant la difficulté de maintenance pour le client, deux interfaces de ventilateur 2pin supportant les ventilateurs PWM 4pin
    * PCB avec une épaisseur de cuivre de 2OZ, capable de supporter un courant plus élevé
    * 3 interfaces ADC, supportant jusqu'à trois thermistances
    * 7 interfaces IO, utilisables pour le nivellement et les fins de course
    * 3 interfaces de bâton de chauffage, supportant jusqu'à trois bâtons de chauffage
