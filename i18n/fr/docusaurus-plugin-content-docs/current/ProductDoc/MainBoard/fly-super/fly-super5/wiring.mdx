## Installation du pilote

### Jumelage de la carte mère

* **Mode STEP/DIR.** Dans ce mode, on utilise des cavaliers pour régler la micro-pas et on ajuste le courant via un potentiomètre. Le pilote le plus courant est l'A4988, qui nécessite de consulter le tableau de configuration de micro-pas fourni par le fabricant et de régler les micro-pas avec des cavaliers.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode sont le TMC2208, TMC2209, TMC2226, etc. Ces puces de pilote peuvent communiquer en série asynchrone UART avec le contrôleur principal, permettant de configurer la micro-pas, le courant de fonctionnement, le mode silencieux, etc., par modification du fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode sont le TMC5160, TMC2230, etc. Ces puces de pilote communiquent en série synchronisée SPI avec le contrôleur principal, permettant également de configurer la micro-pas, le courant de fonctionnement, le mode silencieux, etc., par modification du fichier de configuration.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/drive-dip.png').default} size="40%" align="center" />

### Installation du pilote

* Avant l'installation du pilote, une vérification appropriée est nécessaire pour éviter d'endommager le pilote ou la carte mère.
* **Pilote FLY :** Si vous n'utilisez pas la fonction de retour à l'origine sans fin, veuillez placer l'interrupteur DIP sur la position 1 ; inversement, si vous souhaitez utiliser la fonction de retour à l'origine sans fin, placez l'interrupteur DIP sur la position ON.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.png').default} size="40%" align="center" />

    :::danger

    * **Attention !!! Lors de l'installation du module pilote, assurez-vous que la direction d'insertion est correcte, c'est-à-dire que la broche EN est dans le coin supérieur gauche, sinon vous risquez d'endommager le pilote voire la carte mère !!! N'oubliez pas de bien fixer le dissipateur thermique sur le pilote !!!**

    :::
