# Pont CAN

Le bus CAN est un protocole de communication série utilisé pour des applications en temps réel. Il peut utiliser un câble torsadé pour transmettre des signaux et est l'un des bus de terrain les plus répandus au monde. Le protocole CAN est utilisé pour la communication entre divers composants dans les véhicules automobiles, remplaçant ainsi les faisceaux de câbles coûteux et encombrants. La robustesse de ce protocole a permis son utilisation dans d'autres applications d'automatisation et industrielles. Les caractéristiques du protocole CAN incluent la communication de données sérielles avec intégrité, la prise en charge en temps réel, une vitesse de transmission allant jusqu'à 1 Mb/s, une adresse de 11 bits et des capacités de détection d'erreurs.
Les imprimantes 3D utilisent le bus CAN pour réduire le nombre de fils nécessaires pour connecter la tête d'impression à la carte mère. Alors qu'il fallait auparavant une dizaine de fils, avec CAN, seulement quatre fils suffisent, réduisant considérablement le nombre de fils et la complexité de câblage.
Ce chapitre ne fait qu'une brève introduction à l'utilisation du pont CAN.
