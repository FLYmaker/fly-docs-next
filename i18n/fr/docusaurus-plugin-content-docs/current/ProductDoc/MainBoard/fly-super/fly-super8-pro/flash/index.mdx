    :::tip

    * Le Super8 PRO utilise par défaut une carte TF pour la mise à jour du firmware
    * Veuillez préparer une carte TF d'une capacité maximale de `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec de la mise à jour, veuillez utiliser de préférence les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF devrait être au moins de `class10`, il est recommandé d'utiliser une carte TF de marque comme SanDisk

    :::

# Instructions pour la mise à jour du firmware

    :::tip

    * Le Super8 Pro est livré avec le bootloader de la carte TF déjà préinstallé, il n'est pas nécessaire de le réinstaller

    :::

## Comment déterminer le type de firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin d'installer qu'un seul type de firmware

    :::
    
    * Le Super8 PRO peut se connecter à l'ordinateur de contrôle via les méthodes suivantes
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : si plusieurs périphériques USB sont connectés, cela peut affecter le transfert de données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : peut connecter plusieurs appareils sans occuper le port USB
        * Inconvénients : pourrait causer des erreurs comme des timeouts de retour à l'origine si utilisé avec `KlipperScreen`.
