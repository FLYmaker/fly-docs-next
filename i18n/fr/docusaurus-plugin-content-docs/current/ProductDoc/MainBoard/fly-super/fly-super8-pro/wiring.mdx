## Installation du driver

### Jumper de la carte mère

* **Mode STEP/DIR.** Dans ce mode, on utilise des cavaliers pour régler la subdivision et on ajuste le courant avec un potentiomètre. Les plus courants sont les drivers A4988, qui nécessitent de consulter la table de configuration de subdivision fournie par le fabricant et de régler la subdivision avec des cavaliers.
* **Mode UART.** Les drivers les plus courants utilisant ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de driver peuvent communiquer en série asynchrone avec le contrôleur principal via UART, permettant de configurer la subdivision, le courant de fonctionnement, le mode silencieux, etc. par modification du fichier de configuration.
* **Mode SPI.** Les drivers les plus courants utilisant ce mode sont : TMC5160, TMC2230, etc. Ces puces de driver communiquent en série synchrone via SPI avec le contrôleur principal, et peuvent également configurer la subdivision, le courant de fonctionnement, le mode silencieux, etc. par modification du fichier de configuration.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/drive-dip.png').default} size="40%" align="center" />

### Installation du driver

* Avant l'installation, il est nécessaire de vérifier le driver pour éviter tout dommage au driver ou à la carte mère.
* **Driver FLY :** Si la fonction de retour à l'origine sans limite n'est pas utilisée, veuillez mettre l'interrupteur DIP sur la position 1 ; inversement, si vous souhaitez utiliser la fonction de retour à l'origine sans limite, veuillez mettre l'interrupteur DIP sur la position ON.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.png').default} size="40%" align="center" />

    :::danger

    * **Attention !!! Lors de l'installation du module driver, assurez-vous que la direction d'insertion est correcte, c'est-à-dire que la broche EN est dans le coin supérieur gauche, sinon cela pourrait endommager le driver voire la carte mère !!! N'oubliez pas de coller un dissipateur thermique sur le driver !!!**

    :::
