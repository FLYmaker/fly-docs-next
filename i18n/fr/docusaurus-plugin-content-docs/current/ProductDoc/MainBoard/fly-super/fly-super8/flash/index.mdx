    :::tip

    * Super8 utilise par défaut une carte TF pour la gravure du firmware
    * Veuillez préparer une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marques inconnues peuvent entraîner un échec de la gravure, veuillez utiliser de préférence les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF doit être de `class10` ou supérieure, il est recommandé d'utiliser des cartes TF de marques comme SanDisk

    :::

# Instructions de gravure du firmware


## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Super8 peut se connecter à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable
        * Inconvénients : Si plusieurs appareils USB sont connectés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantages : Peut connecter plusieurs appareils sans occuper de port USB
        * Inconvénients : Peut entraîner des erreurs comme le dépassement du temps de retour à zéro lorsqu'il est utilisé avec `KlipperScreen`.
