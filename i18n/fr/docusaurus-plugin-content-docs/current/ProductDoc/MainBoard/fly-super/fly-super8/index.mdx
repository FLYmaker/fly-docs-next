# Spécifications du produit


    * Processeur 32 bits ARM Cortex-M4 série à 168 MHz, puce STM32F407ZGT6
    * Firmware : Marlin 2.0/Reprap/Klipper
    * Support des drivers : A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Mode de pilotage supporté : TMC : UART/SPI
    * 8 emplacements pour drivers haute tension, support pour 8 moteurs indépendants, 5 extrudeuses, 10 ventilateurs PWM
    * Affichage : Écran tactile série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 entrée haute tension, support pour capteur de nivellement automatique : BLTouch, capteurs de proximité PL08N, Klicky, Voron Tap, etc.
    * Alimentation pour interrupteurs de fin de course optionnelle : 5V/12V/24V
    * Avec le firmware reprap, vous pouvez utiliser le module WiFi ESP32 intégré, supportant 1.6m/s
    * Conception unique du MOSFET pour ventilateur PWM, remplaçable en cas de panne, réduit la difficulté de maintenance pour le client, deux interfaces de ventilateur 2pin peuvent supporter des ventilateurs PWM 4pin
    * PCB avec une épaisseur de cuivre de 2OZ, capable de supporter un courant plus élevé
    * 6 interfaces ADC, support jusqu'à six sondes de température
    * 8 interfaces IO, pouvant être utilisées pour le nivellement et les fins de course
    * 5 interfaces pour barres chauffantes, support jusqu'à cinq barres chauffantes
    * 2 interfaces PWM
