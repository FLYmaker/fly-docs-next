# Introduction

    * Quelques points d'attention concernant les pilotes


## Installation du pilote

    * Veuillez noter que l'EN doit être aligné en haut à gauche pour l'installation
    * Les broches EN des connecteurs de pilote sur la carte mère FLY sont toutes en haut à gauche
    * Les FLY DP5, PRO X10, MMU ne peuvent utiliser que le pilote TP
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

## Configuration du pilote

    * La configuration du pilote TP et des pilotes ordinaires est identique
    * Le pilote FLY-2240 supporte la connexion UART, mais nécessite une modification manuelle des résistances
    * Il faut prêter attention à la résistance d'échantillonnage du pilote, une configuration incorrecte peut endommager le moteur et le pilote
