# Introduction

      Fly-CM4 est entièrement compatible avec Raspberry Pi 3B et 4B, et a été spécialement amélioré pour l'extension Raspberry Pi CM4, cette carte d'extension offre un ensemble complet de fonctionnalités optimisées pour les imprimantes 3D. L'alimentation en courant continu de 12-24V assure une stabilité de l'alimentation, cruciale pour le contrôle précis et le fonctionnement continu des imprimantes 3D. La prise en charge des écrans FLY-TFT et Fly-HDMI avec connexion FPC offre un câblage plus pratique. La fourniture d'un port Can permet une connexion directe Can entre la machine hôte et la machine esclave, ce qui est actuellement une méthode très populaire.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-cm4/img/cm4.png').default} size="80%" align="center" />

## Caractéristiques

    * Supporte la carte mère Raspberry Pi CM4
    * Supporte une alimentation en courant continu de 12V-24V
    * Module UTOC intégré
    * Supporte le TFT et HDMI avec un seul câble
    * Supporte les interfaces d'écran DSI/spi
    * Supporte les interfaces de caméra CSI/USB
    * Interface d'extension 40PIN, compatible avec Raspberry Pi
