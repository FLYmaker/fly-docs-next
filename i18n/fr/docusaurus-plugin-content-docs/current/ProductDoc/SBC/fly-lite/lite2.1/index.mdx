# Introduction

    Fly-Pi-lite2.1 est un ordinateur hôte Klipper d'entrée de gamme
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## Caractéristiques

    * CPU : Processeur Allwinner H3 haute performance, 4 cœurs Cortex-A7 64 bits
    * GPU : Mali-400 MP2
    * RAM : 512Mo DDR3
    * ROM : Supporte jusqu'à 128Go SDCARD
    * WIFI : WiFi 2.4G intégré
    * Périphériques : SPI x1, UART x2, USB x2, HDMI x1
    * Interfaces : Connecteur FPC réservé, HDMI + USB FPC en une seule ligne
    * Système préinstallé avec klipper, moonraker, mainsail, fluidd, klipper-screen, plugin de compensation de résonance numpy et FLY-Tools, pas besoin de changer de source ou d'opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF)


## Fonctionnalités mises à jour

    * Interface TFT V1 mise à jour à TFT V2
    * Ajout des broches de réinitialisation et d'arrêt
    * Ajout d'une LED d'indication d'alimentation
    * Suppression du bouton BOOT, remplacé par un bouton de réinitialisation
    * Optimisation du routage
