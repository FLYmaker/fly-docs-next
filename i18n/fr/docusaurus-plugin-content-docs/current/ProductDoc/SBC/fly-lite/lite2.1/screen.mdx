# Connexion et configuration de l'écran
:::info

* lite2.1 nécessite une connexion SSH au système, puis l'entrée de la commande suivante pour utiliser la configuration de l'écran Fly Tools
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools nécessite que la machine hôte soit connectée à Internet pour accéder via l'`IP+:9999` de la machine hôte
* Par exemple `192.168.6.110:9999`

:::
