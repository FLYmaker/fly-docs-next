# Connexion SSH à la machine hôte


* Vous pouvez vous connecter à l'appareil via un port série ou le WiFi
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a bien été flashé avec **le système de la machine hôte**
* **Si la machine hôte ne démarre pas correctement, veuillez retirer les pilotes, les fins de course, les ventilateurs, etc. !!!**
* **Enfin, veuillez vous connecter au SSH via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion de la machine hôte via WiFi ou Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utiliser l'IP réseau pour se connecter au SSH de la machine hôte

        * Veuillez vous connecter à l'interface d'administration de votre **routeur**
        * Assurez-vous que le système est **correctement démarré**
        * Assurez-vous que **vous connaissez le nom d'utilisateur et le mot de passe de la machine hôte**, le nom d'utilisateur par défaut pour le système FLY est **fly**, et le mot de passe par défaut est **mellow**
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter au SSH avec l'IP
        * Ouvrez MobaXterm et configurez-le comme suit, entrez l'IP que vous avez vue précédemment ou vérifiez-la dans l'interface d'administration du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Remarque : Ici, à l'étape 4, entrez le nom d'utilisateur de la machine hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'ouvrira, cliquez sur `Accepter`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Une invite vous demandera d'entrer le mot de passe, entrez le mot de passe de la machine hôte et appuyez sur Entrée

        **Remarque : Le mot de passe ne sera pas visible à l'écran, entrez-le simplement et appuyez sur Entrée**

        Pour le système FLY par défaut, le nom d'utilisateur est : `fly` et le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />
