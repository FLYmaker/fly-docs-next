# Connexion WIFI

## Installer l'antenne sur LITE 2.1

    * L'emplacement indiqué par la flèche rouge est l'interface de l'antenne WIFI
    * La prise de l'antenne WIFI est `ipex1`
    * Installez l'antenne sur LITE 2.1, sinon vous pourriez ne pas pouvoir vous connecter en raison d'un mauvais signal

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.png').default} size="100%" align="left" />  

* Si vous constatez que la connexion WIFI ne se fait pas correctement, vérifiez d'abord si l'antenne est bien connectée, puis utilisez la méthode ci-dessous pour vous connecter au WiFi
