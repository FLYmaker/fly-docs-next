# Introduction

    Fly-Pi-lite2 est un ordinateur hôte Klipper de niveau débutant

## Caractéristiques

    * CPU : Puce Allwinner H3 haute performance, 4 cœurs 64 bits Cortex-A7
    * GPU : Mali-400 MP2
    * RAM : 512MB DDR3
    * ROM : Supporte une carte SD de 128GB maximum
    * WiFi : WiFi 2.4G intégré
    * Périphériques : Spi x1，URAT x2, USB x2，HDMI x1
    * Interfaces : Connecteur FPC réservé, HDMI + USB FPC pour une connexion en une ligne
    * Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, plugin de compensation de résonance numpy et FLY-Tools, pas besoin de changer de source ou d'autres opérations complexes, plus facile pour les débutants (une carte TF doit être achetée)
