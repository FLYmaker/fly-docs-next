# Connexion SSH à l'hôte


* Vous pouvez utiliser la connexion série ou WIFI pour vous connecter à l'appareil
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a été flashé avec **le système correspondant à l'hôte**
* **Si l'hôte ne démarre pas correctement, veuillez retirer les drivers, les butées, les ventilateurs, etc. !**
* **Enfin, connectez-vous via le réseau à SSH pour faciliter la compilation du firmware !**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion à l'hôte via WIFI ou câble Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Connexion SSH à l'hôte via IP réseau

        * Veuillez vous connecter à l'**interface d'administration du routeur** auquel l'hôte est connecté
        * Assurez-vous que le système a **démarré normalement**
        * Assurez-vous **de connaître le nom d'utilisateur et le mot de passe de l'hôte**, par défaut pour le système FLY, le nom d'utilisateur est **fly** et le mot de passe est **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter à SSH via l'IP
        * Ouvrez MobaXterm et configurez comme suit, entrez l'IP que vous avez vérifiée précédemment ou que vous pouvez voir dans l'interface d'administration du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Remarque : Ici, à l'étape 4, entrez le nom d'utilisateur de l'hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * À la première connexion, une fenêtre apparaîtra, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Vous serez invité à entrer le mot de passe, entrez le mot de passe de l'hôte puis appuyez sur Entrée

        **Remarque : Ici, le mot de passe n'est pas visible, après l'avoir entré, appuyez directement sur Entrée**

        Par défaut pour le système FLY, le nom d'utilisateur est : `fly` et le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
