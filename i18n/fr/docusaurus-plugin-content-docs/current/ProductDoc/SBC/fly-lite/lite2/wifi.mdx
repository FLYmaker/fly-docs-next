# Connexion WIFI

## Installer l'antenne sur le LITE 2

    * La flèche rouge indique l'interface de l'antenne WIFI
    * Le port de l'antenne WIFI est `ipex1`
    * Installez l'antenne sur le LITE 2, sinon il pourrait y avoir des problèmes de connexion en raison d'un signal faible

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.png').default} size="100%" align="left" />  

* Si vous constatez que la connexion WIFI n'est pas normale, vérifiez d'abord si l'antenne est bien connectée, puis utilisez la méthode ci-dessous pour vous connecter au WiFi
