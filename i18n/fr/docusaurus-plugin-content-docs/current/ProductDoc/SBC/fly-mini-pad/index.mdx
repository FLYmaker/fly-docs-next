# Introduction

    Fly-MINI-Pad est une machine hôte Klipper d'entrée de gamme

## Caractéristiques

    * CPU : Puce Allwinner H3 haute performance, 4 cœurs 64 bits Cortex-A7
    * GPU : Mali-400 MP2
    * RAM : 512MB DDR3
    * ROM : Supporte jusqu'à 128GB SDCARD
    * WIFI : WiFi 2.4G intégré
    * Périphériques : SPI x1, USB x2, Type-C x1
    * Interfaces : Trois interfaces XH2.54-2P supportant respectivement, POWER-IN pour une alimentation maximale de 24V, CAN-BUS pour la connexion au bus CAN, LI-PO pour une alimentation/charge de batterie lithium-ion de 3.7V avec protection
    * Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, le plugin de compensation de résonance numpy et FLY-Tools, sans nécessiter de changer de source ou d'effectuer des opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF)
    * Écran tactile capacitif/écran résistif, résolution 320*480
