# Connexion SSH à l'hôte


* Vous pouvez vous connecter à l'appareil via un port série ou Wi-Fi
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil terminal SSH**
* Veuillez vous assurer que la carte SD ou M2WE a été flashée avec **le système correspondant à l'hôte**
* **Si l'hôte ne démarre pas correctement, veuillez retirer les périphériques comme les pilotes, les butées, les ventilateurs, etc. !!!**
* **Enfin, veuillez vous connecter à SSH via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion de l'hôte via Wi-Fi ou Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Connexion SSH à l'hôte via l'IP réseau

        * Veuillez vous connecter à **l'interface d'administration du routeur**
        * Assurez-vous que le système est **correctement démarré**
        * Assurez-vous que **vous connaissez le nom d'utilisateur et le mot de passe de l'hôte**, nom d'utilisateur par défaut pour le système FLY **fly**, mot de passe **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter à SSH via l'IP
        * Ouvrez MobaXterm, configurez comme suit, entrez l'IP que vous avez vue précédemment, ou vérifiez-la dans l'interface d'administration du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Remarque : ici, entrez le nom d'utilisateur de l'hôte à l'étape 4**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'ouvrira, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Vous serez invité à entrer le mot de passe, entrez le mot de passe de l'hôte et appuyez sur Entrée

        **Remarque : le mot de passe n'est pas visible lors de la frappe, tapez-le et appuyez directement sur Entrée**

        Pour le système FLY, le nom d'utilisateur par défaut est : `fly`, le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />
