# Utilisation de l'UTOC intégré

:::danger À savoir

Il est déconseillé d'utiliser deux UTOIC, car la gravure et la connexion peuvent être très compliquées.

:::
    * Si vous n'avez pas besoin d'utiliser l'UTOC intégré, exécutez la commande ci-dessous, l'exécution n'aura aucun message :
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * Méthode de restauration, l'exécution n'aura aucun message :
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * Le MINI Pad est déjà équipé d'un UTOC
