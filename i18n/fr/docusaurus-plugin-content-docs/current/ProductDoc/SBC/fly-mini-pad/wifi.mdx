# Connexion WIFI

## Installation de l'antenne sur le MINI PAD

    * L'endroit indiqué par la flèche rouge est l'interface de l'antenne WIFI
    * La prise WIFI est de type `ipex1`
    * Installez l'antenne sur le MINI PAD, sinon il se peut que la connexion Internet soit impossible en raison d'un signal faible
