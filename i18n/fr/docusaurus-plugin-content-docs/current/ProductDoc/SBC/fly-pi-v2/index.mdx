# Introduction

        Nous sommes ravis de lancer le tout nouveau FLY-Pi V2. Cette carte a les mêmes dimensions d'installation que le Raspberry Pi et peut être utilisée comme une machine hôte en remplacement du Raspberry Pi. Elle dispose de 1 Go de mémoire DDR3 embarquée, permettant l'exécution d'un bureau LINUX complet (optimisé sur la base d'Armbian). Cette carte peut être utilisée avec les firmwares Klipper et RRF, et peut même faire fonctionner l'écran Klipper lors de l'utilisation de la carte SHT.
    L'objectif de la conception de cette machine hôte est de rendre l'utilisation des firmwares Klipper ou RRF plus facile pour les clients. Nous avons optimisé l'image du système LINUX pour une utilisation plus adaptée aux imprimantes 3D, rendant l'installation et l'opération plus simples, permettant même aux débutants de l'apprendre et de l'utiliser rapidement. Par rapport aux autres cartes hôtes, cela réduit les coûts d'apprentissage, vous permettant de vous concentrer davantage sur le plaisir de l'impression 3D (en achetant un package avec une carte SD préinstallée avec le firmware Klipper, vous pouvez l'utiliser immédiatement, ce qui est à la fois pratique et rapide).

## Caractéristiques

    * Même format que le Raspberry Pi 3B et 4B
    * CPU : Puce Allwinner H5 haute performance, 4 cœurs, 64 bits Cortex-A53
    * GPU : Haute performance avec 6 cœurs Mali 450, taux de remplissage des pixels supérieur à 2.7 gpixel/s
    * RAM : 1 Go de DDR3 (partagé avec le GPU) (512+512)
    * ROM : Supporte une carte SD jusqu'à 128 Go
    * Périphériques : Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN (module USB vers CAN intégré, c'est-à-dire UTOC intégré)
    * Interfaces : Interface M.2 intégrée, extensible avec WIFI, eMMC, etc. (protocole propriétaire, ne connectez pas d'appareils M.2 non spécifiques à FLY π)
    * Interface 40 broches compatible avec le Raspberry Pi
    * Supporte une alimentation DC de 12 à 24V, pour une alimentation plus stable
    * Prise en charge d'une interface pour ventilateur 5V ZH1.5
    * Le système est préinstallé avec Klipper, Moonraker, Mainsail, Fluidd, Klipper-screen, Crowsnest, plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'opérations complexes, ce qui facilite l'utilisation pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
