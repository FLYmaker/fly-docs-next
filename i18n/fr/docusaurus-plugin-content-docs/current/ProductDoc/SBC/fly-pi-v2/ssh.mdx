# Connexion SSH à l'hôte principal


* Vous pouvez utiliser une connexion série ou WIFI pour vous connecter à l'appareil
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE a bien le **système de l'hôte principal** installé
* **Si l'hôte principal ne démarre pas correctement, veuillez retirer les périphériques externes comme les moteurs, les fins de course, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous via le réseau à SSH pour faciliter la compilation du firmware !!**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion de l'hôte principal via WIFI ou Ethernet**

        ## Utiliser l'IP réseau pour se connecter à SSH de l'hôte principal

        * Veuillez vous connecter à l'interface d'administration de votre **routeur**
        * Assurez-vous que le système est **correctement démarré**
        * Assurez-vous que **vous connaissez le nom d'utilisateur et le mot de passe de l'hôte principal**, le système FLY a par défaut l'utilisateur **fly** et le mot de passe **mellow**
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter via IP à SSH
        * Ouvrez MobaXterm, configurez comme suit, entrez l'IP que vous avez vue précédemment ou vérifiez dans l'interface du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Note : À l'étape 4, entrez le nom d'utilisateur de l'hôte principal**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'ouvrira, cliquez sur `Accepter`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Il vous sera demandé d'entrer le mot de passe, entrez le mot de passe de l'hôte principal et appuyez sur Entrée

        **Note : Le mot de passe ne sera pas visible lors de la saisie, tapez-le et appuyez directement sur Entrée**

        Par défaut, le système FLY a comme nom d'utilisateur : `fly`, et le mot de passe : `mellow`
