## Configuration

    ```
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev0.0

    [resonance_tester]
    accel_chip: adxl345
    probe_points:
        100, 100, 20  # un exemple
    ```
