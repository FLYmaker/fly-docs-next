# Introduction

Nous sommes ravis de présenter la toute nouvelle carte mère FLY-π. Cette carte peut remplacer un Raspberry Pi comme contrôleur principal, avec 1 Go de mémoire DDR3 intégrée, permettant l'exécution d'un bureau Linux complet (optimisé sur la base d'Armbian). La carte peut être utilisée avec les firmwares Klipper et RRF, et même faire fonctionner l'écran Klipper lors de l'utilisation de la carte SHT.

Le but de la conception de ce contrôleur principal est de rendre l'utilisation des firmwares Klipper ou RRF plus facile pour les clients. Nous avons optimisé l'image système Linux pour qu'elle soit plus adaptée à l'impression 3D, rendant l'installation et l'utilisation plus simples, même pour les débutants qui peuvent rapidement apprendre à s'en servir. Par rapport aux autres cartes contrôleurs, cela réduit les coûts d'apprentissage, vous permettant de vous concentrer davantage sur le plaisir de l'impression 3D (achetez le forfait avec carte SD préinstallée avec le firmware Klipper pour une utilisation immédiate, ce qui économise du temps et des efforts).

## Caractéristiques

    * CPU : Puissant chipset Allwinner H5, 4 cœurs Cortex-A53 à 64 bits
    * GPU : Puissant GPU à 6 cœurs Mali 450, taux de remplissage de pixels supérieur à 2.7 Gpixel/s
    * RAM : 1 Go DDR3 (partagé avec le GPU) (512+512)
    * ROM : Supporte jusqu'à 128 Go SDCARD
    * Périphériques : Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1 (nécessite l'achat séparé d'un module CAN)
    * Interfaces : Interface M.2 intégrée, extensible pour WIFI, eMMC, etc. (protocole privé, ne connectez pas d'appareils M.2 non spécifiques à FLY π)
    * Interface 40 broches, compatible avec Raspberry Pi
    * Supporte une alimentation DC de 12-24V, pour une alimentation plus stable
    * Une interface de ventilateur contrôlable à 5V est prévue pour le refroidissement du contrôleur principal
    * Interface d'extension CAN prévue
    * Système préinstallé avec klipper, moonraker, mainsail, fluidd, klipper-screen, Crowsnest, plugin de compensation de résonance numpy et FLY-Tools, sans besoin de changer de source ou d'opérations complexes, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
