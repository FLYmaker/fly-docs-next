# Connexion et configuration de l'écran

:::info

* FLY-TOOLS nécessite que l'ordinateur hôte soit connecté à Internet et que l'adresse `IP+:9999` de l'ordinateur hôte soit entrée pour y accéder.
* Par exemple `192.168.6.110:9999`

:::
