# Connexion SSH à l'hôte

* Vous pouvez vous connecter à l'appareil via un port série ou Wi-Fi
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou le M2WE est gravé avec **le système de l'hôte correspondant**
* **Si l'hôte ne démarre pas correctement, veuillez débrancher les périphériques tels que les pilotes, les fins de course, les ventilateurs, etc. !**
* **Enfin, veuillez vous connecter à SSH via le réseau pour faciliter la compilation du firmware !**
* **Pour compiler le firmware, veuillez consulter le tutoriel de connexion à l'hôte via Wi-Fi ou Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Connexion SSH à l'hôte via IP réseau

        * Veuillez vous connecter à l'**interface d'administration de votre routeur**
        * Assurez-vous que le système a **démarre correctement**
        * Assurez-vous de **connaître le nom d'utilisateur et le mot de passe de l'hôte**, le nom d'utilisateur par défaut du système FLY est **fly**, le mot de passe par défaut est **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter à SSH via l'IP
        * Ouvrez MobaXterm, configurez comme suit, remplissez l'IP que vous avez vue précédemment ou vérifiez-la dans l'interface du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Remarque : ici, à l'étape 4, entrez le nom d'utilisateur de l'hôte**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre pop-up apparaîtra, cliquez sur `Accepter`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * L'invite de mot de passe apparaîtra, entrez le mot de passe de l'hôte et appuyez sur Entrée

        **Remarque : ici, le mot de passe ne sera pas visible, tapez-le et appuyez directement sur Entrée**

        Par défaut pour le système FLY, le nom d'utilisateur est : `fly` et le mot de passe est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />
