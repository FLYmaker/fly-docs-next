# Connexion WIFI


## Installer l'antenne sur le PI

    * L'emplacement indiqué par la flèche rouge est l'interface de l'antenne WIFI
    * Le port de l'antenne WIFI est `ipex1`
    * Installez l'antenne sur le PI, sinon une mauvaise connexion internet pourrait être due à un signal faible
