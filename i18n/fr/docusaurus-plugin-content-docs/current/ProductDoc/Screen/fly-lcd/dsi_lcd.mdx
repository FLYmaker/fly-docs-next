## Présentation du produit

    * Taille de l'écran : 120.7mm*76.0mm
    * Zone d'affichage de l'écran : 108mm*66.2mm
    * Résolution : 800*480
    * Type d'écran : IPS
    * Mode de communication : DSI/Mipi
    * Interface embarquée : 15 pin DSI / 22pin Mipi
    * Stabilité de fonctionnement : 0°-60°C
    * Type de toucher : Capacitif
    * Tension logique : 3.3V
    * Poids : 135.5g

## Système Raspberry Pi
    * Il existe des images système préemballées, testées pour l'affichage et le toucher de l'écran DSI

        :::danger À savoir

        * Les interfaces DSI et CSI du Raspberry Pi 5 sont maintenant appelées MIPI
        * L'interface MIPI peut être configurée pour utiliser directement une caméra CSI ou un écran DSI
        * Veuillez prêter attention à l'orientation des doigts d'or

        :::

        * Le Raspberry Pi 5 dispose maintenant de deux interfaces MIPI, si vous regardez de face, vous pouvez voir les marques 1 et 0

        * Câblage de l'écran DSI


            * Si vous utilisez l'interface n°1, configurez l'écran comme suit :

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Si vous utilisez l'interface n°0, configurez l'écran comme suit :

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
