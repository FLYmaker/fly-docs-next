# Modification de l'image de démarrage
:::tip
    Veuillez strictement suivre la `résolution` ainsi que le `nom de fichier` pour créer l'image, sinon le système ne la reconnaîtra pas.

    Placez l'image créée dans le répertoire racine de la carte SD/clef USB, insérez-la dans l'écran qui est allumé, et cliquez sur mise à jour pour terminer.
:::
## Modification du LOGO de démarrage

    * L'image doit être de `800*480`
    * Le nom doit être `boot_logo.JPG`
    * Ne doit pas dépasser `105KB`




## Modification de l'image de la première interface

    * L'image doit être de `500*400`
    * Le nom doit être `print.png`
    * Ne doit pas dépasser `45KB`



## Modification de l'image de l'interface d'impression

    * L'image doit être de `311*480`
    * Le nom doit être `print_smail.png`
    * Ne doit pas dépasser `45KB`
