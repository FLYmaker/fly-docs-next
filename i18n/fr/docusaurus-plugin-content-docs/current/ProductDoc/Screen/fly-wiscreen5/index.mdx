# Introduction

      Fly_WiScreen est un écran de contrôle Klipper haut de gamme développé par l'équipe de recherche et développement de Fly3D de Guangzhou Mellow Electronic Technology Co., Ltd. Il permet de contrôler une imprimante 3D via le Wifi, supporte la transmission de fichiers d'impression via carte SD et clé USB, offrant une excellente expérience utilisateur grâce à sa rapidité de réponse. Il supporte également la configuration en mode maître-esclave pour une gestion facile de plusieurs machines. Le port d'alimentation 5V peut être connecté à un module de charge pour la modification de batterie, facilitant le déplacement.

## Fonctionnalités du produit

    * Supporte la connexion via WIFI à d'autres machines hôtes équipées de Klipper
    * Supporte la recherche WIFI, permettant une connexion automatique après connexion au WIFI
    * Supporte le contrôle en cluster de plusieurs imprimantes
    * Supporte l'upload d'impression via USB et carte TF
    * Macros personnalisées pour une opération et un débogage plus faciles

## Paramètres du produit

    * CPU: Allwinner T113-S3, double cœur CORTEX-A7 avec une fréquence maximale de 1,2 GHz
    * Mémoire: Intégrée 128M DDR3
    * Stockage: 128MB Nand Flash, emplacement pour carte TF
    * Affichage: Écran capacitif 800x480
    * Interfaces externes: Un port USB 2.0, un emplacement pour carte TF
    * Protocole WIFI: 802.11b/g/n
    * Type d'écran: IPS
    * Taille d'écran: 5 pouces
    * Multitouch: Supporté
    * Tension nominale: 5V
