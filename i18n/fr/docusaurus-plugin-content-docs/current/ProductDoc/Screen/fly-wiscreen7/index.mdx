# Introduction

      Fly_WiScreen est un écran de contrôle Klipper haut de gamme développé par l'équipe de recherche et développement de la carte mère Fly3D de Guangzhou MELEN Electronics Co., Ltd. Il peut contrôler une imprimante 3D via deux modes, le Wifi et le câble Ethernet, et supporte le transfert de fichiers d'impression via une carte SD et une clé USB. L'écran répond très rapidement, offrant une excellente expérience utilisateur. Il supporte également le mode maître-esclave, permettant une gestion facile des machines par des opérations synchronisées sur plusieurs machines. Le port d'alimentation 5V peut être connecté à un module de charge pour la modification de la batterie, facilitant ainsi le déplacement.

## Fonctions du produit

    * Supporte la connexion via WIFI et câble Ethernet à d'autres machines hôtes équipées de Klipper
    * Supporte la recherche WIFI, permettant une connexion automatique à la machine hôte après connexion au WIFI
    * Supporte le contrôle en cluster de plusieurs imprimantes
    * Supporte l'upload d'impression via USB et carte TF
    * Macros personnalisées pour une manipulation et un débogage plus pratiques

## Paramètres du produit

    * CPU: Allwinner T113-S3, double cœur CORTEX- A7, jusqu'à 1.2GHZ
    * Mémoire: Intégrée 128M DDR3
    * Stockage: 128MB Nand Flash, emplacement pour carte TF
    * Affichage: Écran capacitif 1024X800
    * Interfaces externes: Un port USB 2.0, un emplacement pour carte TF
    * Protocole WIFI: 802.11b/g/n
    * Type d'écran: IPS
    * Taille de l'écran: 7 pouces
    * Multi-touch: Supporte
    * Tension nominale: 5V
