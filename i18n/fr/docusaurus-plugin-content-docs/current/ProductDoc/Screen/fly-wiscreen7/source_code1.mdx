# Introduction à la structure du projet

## Introduction
#### Dossier Logic
* Nous programmons 99 % du code de l'interface utilisateur dans le dossier logic, les autres dossiers ne nécessitent pas d'attention.
* gcodeLogic.cc est similaire à main.c, 99 % du code de l'interface utilisateur se trouve ici, les autres fichiers ne nécessitent pas d'attention particulière.

#### Dossier i18n
* Stocke les tables de traduction pour les différentes langues.

#### Dossier resources
* Stocke les ressources d'images utilisées par l'interface utilisateur.

#### Dossier ui
* Stocke les fichiers de l'interface utilisateur, double-cliquez pour ouvrir et afficher.
* Double-cliquez sur la "petite icône œil" indiquée par la flèche en bas à gauche pour ouvrir les interfaces cachées.
