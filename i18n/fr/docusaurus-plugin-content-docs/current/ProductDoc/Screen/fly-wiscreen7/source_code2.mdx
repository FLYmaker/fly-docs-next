## Définir la fonction du bouton
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide10.gif').default} size="100%" align="left" />

* Utiliser une fonction pour envoyer une commande Gcode unique  
    ```cfg
    gcode_send2("","G28");
    ```
* Utiliser une fonction pour envoyer plusieurs commandes Gcode à la fois:
    ```cfg
    gcode_send2("","G91\\nG1 X10 F3000\\nG90");
    ```
* Ou utiliser cette méthode pour envoyer plusieurs commandes Gcode à la fois:
    ```cfg
    gcode_send2("","G91");
    gcode_send2("","G1 X10 F3000");
    gcode_send2("","G90");
    ```
