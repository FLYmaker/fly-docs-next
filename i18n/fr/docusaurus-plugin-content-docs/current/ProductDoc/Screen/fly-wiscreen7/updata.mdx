##   Mise à jour
    * 1. Renommez le fichier téléchargé en `update.img`
    * 2. Placez le fichier `update.img` dans une carte SD/une clé USB et insérez-le dans l'écran, vous pouvez l'insérer pendant que l'écran est allumé.
    * 3. Sélectionnez l'écran, cliquez sur Mettre à jour pour terminer la mise à jour
