# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * ERCF  prend en charge CAN, USB
    * Ici, le firmware CAN est fourni en deux vitesses : 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    Le bouton encerclé est le `BOOT`

    :::

    * Pour entrer en mode flash, il faut couper complètement l'alimentation de la carte mère, appuyer sur le bouton BOOT puis connecter un câble de données d'une extrémité à la carte mère et de l'autre à l'hôte.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.png').default} size="100%" align="center" />

            4. Flasher le firmware

            :::tip

            Il suffit de flasher un seul firmware

            :::

            * Voici la commande pour flasher le firmware à une vitesse CAN de 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Voici la commande pour flasher le firmware à une vitesse CAN de 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Voici la commande pour flasher le firmware USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. Si le message suivant apparaît, le flashage est réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            Après le flashage, il est nécessaire de couper complètement l'alimentation de la carte mère une fois.

            :::
