# Explications sur la gravure du firmware

    * Le ERCF peut se connecter à l'ordinateur via une connexion CAN.
    * La connexion CAN nécessite l'utilisation d'un câble `XT30(2+2)-F`.
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger À savoir
* La gravure du firmware de la carte mère nécessite de graver Klipper via Katapult.
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de graver le firmware Katapult à la fin, c'est juste au cas où la gravure du firmware Katapult de la carte mère aurait échoué.
* Vous pouvez appuyer deux fois rapidement sur le bouton de réinitialisation pour voir si un LED s'allume sur la carte mère, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult.
:::

## Comment déterminer quel firmware vous avez besoin

    * Le ERCF peut se connecter à l'ordinateur de la manière suivante
    * <Button variant="contained" disableElevation href="./can">Gravure du firmware CAN</Button>
        * Avantages : Si vous avez gravé le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et il supporte la connexion en bus.
        * Inconvénients : L'utilisation avec `KlipperScreen` peut entraîner des erreurs de dépassement de temps de retour à l'origine.
        * Astuce : Si vous souhaitez utiliser CAN, assurez-vous que l'ordinateur hôte le supporte et qu'il est équipé de UTOC ou d'une carte mère avec le firmware de pont.
    * <Button variant="contained" disableElevation href="./usb">Gravure du firmware USB</Button>
        * Avantages : Si vous avez gravé le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher à la carte mère, si vous utilisez une carte mémoire ou la gravure DFU, vous devrez toujours mettre à jour le firmware de la manière habituelle, la communication est fiable et stable.
        * Inconvénients : Si plusieurs appareils USB sont connectés, cela peut affecter la transmission des données.
