# Introduction au produit

    FLY-ERCF est une carte de contrôle multicolor pour imprimante 3D basée sur RP2040, principalement utilisée pour le projet Enraged Rabbit. Elle peut également être utilisée pour d'autres situations de contrôle de moteur pas à pas à deux axes.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.webp').default} size="100%" align="left" />

## Caractéristiques du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Deux emplacements de pilote, supportant les pilotes UART et SPI
    * Compatible avec la carte ERCF EASY originale
    * Volume extrêmement réduit, conçu avec une carte à 4 couches
    * Connecteur d'extension à 2 rangées de 7 broches de 2,54 mm pour les fins de butée, etc.
    * Supporte une tension de 12 à 24V
    * Supporte la connexion CAN bus et RS232, transfert de données plus stable, latence réduite, connexion plus stable
    * L'interface CAN utilise XT30 (2+2), supporte un courant de 15A

## Paramètres du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware : Klipper
    * Tension d'entrée : DC12V-24V
    * Interfaces : connecteur d'extension à 2 rangées de 7 broches de 2,54 mm, SERVO X1, ENDSTOP X2, EXTRA X1
    * Pilote : supporte UART/SPI et STEP/DIR

    * Supporte une tension de 12 à 24V
    * Supporte le bus CAN, transfert de données plus stable, latence réduite, connexion plus stable
    * L'interface CAN utilise XT30 (2+2), supporte un courant de 15A
