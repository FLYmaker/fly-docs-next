<Tabs>
    <TabItem value="can" label="Obtention de l'UUID CANBus">

    * Avant de rechercher l'ID, connectez le tableau d'outils à l'UTOC ou à la carte mère avec un firmware de pontage en utilisant un câble de données XT30. Voici le schéma de connexion de référence
    * Veuillez vérifier si le codeur est positionné correctement
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Assurez-vous que la résistance mesurée entre CAN H et CAN L en cas de coupure de courant est d'environ `60Ω`. Si elle est de `120Ω`, vous devez connecter un cavalier de `120Ω`. Si elle est d'environ `40Ω`, vous devez retirer le cavalier
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.png').default} size="100%" align="center" />

    </TabItem>

    * Veuillez vérifier si le codeur est positionné correctement
