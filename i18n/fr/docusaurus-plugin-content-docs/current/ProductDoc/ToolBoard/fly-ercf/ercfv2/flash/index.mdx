# Instructions de Flashage du Firmware

    * L'ERCF V2 peut se connecter à l'ordinateur hôte via RS232 ou CAN.
    * RS232 et CAN doivent être connectés via un câble `XT30(2+2)-F`.
    * Voici le style du câble de données `XT30(2+2)-F` :
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger À Savoir
* Le flashage du firmware de la carte mère nécessite l'utilisation de Katapult pour flasher Klipper.
* Si vous êtes certain que la carte mère a déjà le firmware Katapult, il n'est pas nécessaire de flasher à nouveau le firmware Katapult, c'est juste une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été flashé correctement.
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour vérifier si une LED clignote sur la carte mère, ce qui indiquerait que la carte mère est en mode Katapult.
:::

## Comment déterminer quel firmware vous avez besoin

    * L'ERCF V2 peut se connecter à l'ordinateur hôte de la manière suivante :
    * <Button variant="contained" disableElevation href="./can">Flashage du firmware CAN</Button>
        * Avantages : Si le firmware `Katapult` est installé, il permet la mise à jour du firmware sans toucher à la carte mère et supporte la connexion par bus.
        * Inconvénients : Peut causer des erreurs comme un dépassement de temps de retour à l'origine lorsqu'il est utilisé avec `KlipperScreen`.
        * Conseil : Assurez-vous que l'ordinateur hôte supporte la CAN et qu'il est équipé d'un UTOC ou d'une carte mère avec un firmware de pont.
    * <Button variant="contained" disableElevation href="./rs232">Flashage du firmware RS232</Button>
        * Avantages : Peut être utilisé sans logiciel spécial, communication fiable et stable.
        * Inconvénients : La mise à jour du firmware nécessite une entrée en mode DFU manuel ou un double clic sur RST pour entrer en mode `Katapult` pour la mise à jour.
        * Conseil : Nécessite un G2T ou un UTOR pour fonctionner.
    * <Button variant="contained" disableElevation href="./usb">Flashage du firmware USB</Button>
        * Avantages : Si le firmware Katapult est installé, il permet la mise à jour sans toucher à la carte mère. Si vous utilisez une carte mémoire ou un flashage DFU, la mise à jour reste nécessaire comme avant, communication fiable et stable.
        * Inconvénients : Avec plusieurs périphériques USB, cela peut affecter le transfert de données.
