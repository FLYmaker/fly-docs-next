# Introduction au produit

    FLY-ERCF est une carte de contrôle multicolor pour imprimante 3D basée sur RP2040, principalement utilisée pour le projet Enraged Rabbit. Elle peut également être utilisée dans d'autres situations de contrôle de moteur pas à pas à deux axes.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.webp').default} size="100%" align="left" />

## Caractéristiques du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Support des modes CAN et RS232
    * Deux emplacements de pilote, supportant les pilotes UART et SPI
    * Compatible avec la carte ERCF EASY originale
    * Volume extrêmement réduit, avec une conception de carte à 4 couches
    * Remplacement du port d'extension 14PIN par un XHD2.54, le câblage est plus solide pour éviter les déconnexions
    * Support de la tension 12-24V
    * Support du bus CAN et de la connexion RS232, transmission de données plus stable, latence réduite, connexion plus stable
    * L'interface CAN utilise XT30 (2+2), supportant un courant de 15A

## Spécifications du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware : Klipper CAN/RS232
    * Tension d'entrée : DC12V-24V
    * Interfaces : Port d'extension double rangée 7PIN XHD2.54, SERVO X1, ENDSTOP X2, EXTRA X1
    * Pilotes : Support UART/SPI et STEP/DIR
    * Support de la tension 12-24V
    * Support du bus CAN et RS232, transmission de données plus stable, latence réduite, connexion plus stable
    * L'interface CAN utilise XT30 (2+2), supportant un courant de 15A
