## Description du firmware Katapult

    :::

## Mode de flashage

    :::tip

    Le bouton en haut est `RESET`, celui en bas est `BOOT`

    :::

    * Il existe deux méthodes pour entrer en mode de flashage
    * Première méthode : Avec la carte mère complètement déconnectée, maintenez le bouton BOOT enfoncé puis connectez le câble de données à la carte mère et à votre appareil
    * Deuxième méthode : Après avoir connecté le câble de données à la carte mère et à votre appareil, maintenez le bouton BOOT enfoncé, appuyez sur le bouton RST, relâchez le bouton RST puis relâchez le bouton BOOT

            4. Flashage du firmware

            :::tip

            Il suffit de flasher un seul firmware

            :::

            * Voici la commande pour flasher le firmware à 500K CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```
                       
            * Voici la commande pour flasher le firmware à 1M CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```

            * Voici la commande pour flasher le firmware USB
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_USB.bin
                ```

            5. Si vous voyez le message `File downloaded successfully`, le flashage a réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.png').default} size="100%" align="center" />

            :::tip

            Après le flashage, il est nécessaire de couper complètement l'alimentation de la carte mère.

            :::
