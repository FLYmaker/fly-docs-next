# Compilation du firmware CAN

## Début de la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Sélectionnez les GPIO pins à définir au démarrage du microcontrôleur, entrez : !PA15 Veuillez noter que ! est en majuscule
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can.png').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, puis sur **Save configuration**, ensuite appuyez sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez revenir à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prendra un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Si la compilation est réussie, vous verrez le message suivant à la fin :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Flasher le firmware

    * Avant de flasher le firmware, connectez la carte d'outils à l'UTOC ou à la carte mère avec le firmware de pontage via un câble de données XT30, ci-dessous un schéma de connexion de référence
    * Veuillez vérifier si le commutateur est bien positionné
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can_mode.png').default} size="90%" align="center" />
    * Connexion à l'UTOC pour référence
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Assurez-vous que, lorsque l'appareil est hors tension, la résistance mesurée entre CAN H et CAN L avec un multimètre est d'environ `60Ω`. Si c'est `120Ω`, vous devez connecter le jumper de `120Ω`. Si c'est environ `40Ω`, vous devez retirer le jumper
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.png').default} size="50%" align="center" />
