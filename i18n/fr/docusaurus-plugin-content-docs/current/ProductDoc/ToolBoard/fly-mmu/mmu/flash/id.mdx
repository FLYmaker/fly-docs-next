<Tabs>
    <TabItem value="usb" label="Obtenir l'ID USB" default>
        
    * Veuillez préparer un câble de données avec fonction de transmission de données, si la longueur est particulièrement longue, il est conseillé d'utiliser un câble de données avec anneau magnétique
    
        <GetUsbID />
        
    </TabItem>
    <TabItem value="can" label="Obtenir l'UUID CANBus">

    * Avant de rechercher l'ID, utilisez un câble de données XT30 pour connecter la carte d'outils à UTOC ou à la carte mère avec le firmware de pont installé, ci-dessous un schéma de câblage de référence
    * Veuillez vérifier si les commutateurs DIP sont positionnés correctement
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Assurez-vous que la résistance entre CAN H et CAN L, mesurée avec un multimètre en mode résistance, est d'environ `60Ω` lorsque l'alimentation est coupée, si c'est `120Ω` vous devez connecter le shunt de `120Ω`, si c'est environ `40Ω` vous devez retirer le shunt
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.webp').default} size="50%" align="center" />
        <GetCanbusUUID />

    </TabItem>
    <TabItem value="rs232" label="Obtenir l'ID RS232">

    * Avant de rechercher l'ID, utilisez un câble de données XT30 pour connecter la carte d'outils à UTOR ou G2T, ci-dessous un schéma de câblage de référence
    * Veuillez vérifier si les commutateurs DIP sont positionnés correctement
        <ImageView image={require('@site/docs/General/get-id/img/utor.webp').default} size="80%" align="left" />
