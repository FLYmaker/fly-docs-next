# Introduction au commutateur DIP embarqué

:::warning Attention

En raison de la haute résolution des images, il est impossible de les voir clairement en parcourant la page web. Veuillez **placer le curseur de la souris au-dessus du schéma de câblage**, **cliquer avec le bouton droit**, et sélectionner **Enregistrer l'image sous**, puis enregistrer l'image sur le bureau pour la visualiser!!!

:::

## Sélection du mode de fonctionnement

:::warning Attention

Veuillez noter que le SB2040 PRO V3 prend en charge trois méthodes de connexion. Il est nécessaire de modifier le commutateur DIP et de flasher le firmware pour pouvoir se connecter normalement.
