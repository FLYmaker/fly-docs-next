<Tabs>
    <TabItem value="can" label="Obtention de l'UUID CANBus">

    * Avant de rechercher l'ID, connectez la carte d'outils à UTOC ou à la carte mère avec le firmware de pont installé en utilisant un câble de données XT30, voici le schéma de connexion de référence
    * Veuillez vérifier si le commutateur DIP est positionné correctement
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Assurez-vous que la résistance entre CAN H et CAN L, mesurée avec un multimètre en l'absence de courant, est d'environ `60Ω`. Si elle est de `120Ω`, ouvrez le commutateur DIP à `120Ω`. Si elle est d'environ `40Ω`, fermez le commutateur.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.png').default} size="100%" align="center" />
        <GetCanbusUUID />

    </TabItem>
    <TabItem value="rs232" label="Obtention de l'ID RS232">

    * Avant de rechercher l'ID, connectez la carte d'outils à UTOR ou G2T en utilisant un câble de données XT30, voici le schéma de connexion de référence
    * Veuillez vérifier si le commutateur DIP est positionné correctement
        <ImageView image={require('@site/docs/General/get-id/img/utor.png').default} size="80%" align="left" />              
        <GetRS232ID />
