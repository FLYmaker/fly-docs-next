# 1. Introduction du produit

    FLY SB2040 PRO V3 est une carte d'outils lancée par Guangzhou Mellow Electronic Technology Co., Ltd. pour la tête d'impression VORON Stealthburber. En utilisant cette carte, vous pouvez remplacer les câblages complexes entre la carte mère et la tête d'impression par seulement quatre fils, optimisant ainsi la disposition. Le SB2040 convient pour les têtes d'impression stealthburner et afterburner, il intègre un transceiver CAN, un transceiver RS232, un port USB, un driver TMC2240, un capteur d'accélération LIS2DW, et trois ventilateurs contrôlables, entre autres fonctionnalités. Cette carte d'outils est compatible avec les firmwares Klipper et RRF.

## 1.1 Caractéristiques du produit

    * Bouton Boot préservé, les utilisateurs peuvent entrer en mode Boot via un port TypeC pour mettre à jour le firmware
    * Supporte les firmwares Klipper et RRF
    * Supporte les modes CAN, RS232, et RRF CAN
    * En mode RS232, il offre un taux de transfert plus élevé et une stabilité accrue. En mode CAN, il prend en charge l'extension CAN avec une interface d'extension CAN préservée.
    * Capteurs de température supportés : Puce MAX31865 intégrée, compatible avec les capteurs de température PT100 et PT1000.
    * Capteur d'accélération LIS2DW et capteur LDC1612 intégrés
    * Interfaces RGB réservées, driver 2240 intégré, support de divers capteurs de nivellement comme klicky, Voron Tap, PL08n, etc.
    * Supporte une tension d'entrée de 12-24V, un courant de 15A, une augmentation de la taille du port de la tige de chauffage, avec un courant nominal de 10A

## 1.2 Nouvelles fonctionnalités

    * Permet la commutation entre les modes CAN de Klipper, RS232 et RRF CAN
    * Nouveau mode RS232 offrant un taux de transfert plus élevé, une meilleure stabilité, et un débogage plus simple
    * Nouveau mode RRF CAN, pouvant basculer vers une carte d'outils RRF, augmentant la compatibilité
    * Nouvelle interface d'extension CAN pour faciliter l'ajout de périphériques CAN
    * Nouveau commutateur de sélection de résistance de terminaison, la résistance de terminaison de 120R peut être sélectionnée par bascule, sans besoin de cavaliers
    * Nouveau câble de connexion d'alimentation/signal de 1 mètre, conçu avec des câbles torsadés pour une meilleure résistance aux interférences

## 1.3 Paramètres du produit


    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Les interfaces de connexion utilisent un connecteur XT30(2+2) supportant un courant de 15A, avec un pic de 30A
    * Firmware : Reprap/Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs intégrés : LIS2DW et optionnel ldc1612 et max31865
    * Interface de la tige de chauffage : Tige de chauffage (HEAT0)
    * Interfaces d'extension : RGB, EndStop, PT100/PT1000, port CAN XH2.54-2P
    * Interfaces des ventilateurs : Un ventilateur DC, deux ventilateurs DC ou PWM
    * Driver moteur : Driver TMC2240 intégré
    * Capteurs de température : Un NTC de 100K ou PT1000 (TH0), un PT100/PT1000 optionnel
