# Méthodes de résolution de la sous-tension du pilote

## Méthode une

    :::danger

    * Changer l'alimentation
    * Ajouter un filtre d'alimentation
    * Changer le moteur pour diagnostiquer
    * Connecter l'enveloppe du moteur à la GND de la carte d'outil
    * Ajouter `driver_TPFD: 0` à la configuration de l'extrudeuse

    :::

## Méthode deux
    * Se connecter à l'ordinateur hôte via SSH
    * Entrer la commande ci-dessous
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Cette étape pourrait être annulée après la mise à jour de Klipper
    :::
