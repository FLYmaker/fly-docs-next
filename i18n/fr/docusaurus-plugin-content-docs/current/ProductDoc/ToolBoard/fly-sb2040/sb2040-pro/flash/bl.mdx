# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO supporte le CAN
    * Nous fournissons uniquement le firmware CAN avec des vitesses de 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    Le bouton entouré est le `bouton BOOT`

    :::

    * Lorsque la carte mère est complètement déconnectée, maintenez enfoncé le bouton BOOT puis connectez le câble de données à la carte mère et à votre appareil

            Il suffit de flasher l'un des firmwares

            * Voici la commande pour flasher le firmware à 500K CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * Voici la commande pour flasher le firmware à 1M CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```
