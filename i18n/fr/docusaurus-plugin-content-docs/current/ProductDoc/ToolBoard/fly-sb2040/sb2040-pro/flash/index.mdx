# Instructions pour la programmation du firmware

    * Le SB2040 PRO peut être connecté à la machine hôte via une connexion CAN
    * La connexion CAN nécessite l'utilisation de `XT30(2+2)-F`
    * Voici le style du câble de données `XT30(2+2)-F`
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La programmation du firmware de la carte mère nécessite l'utilisation de Katapult pour programmer Klipper
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, il n'est pas nécessaire de programmer le firmware Katapult à la fin, c'est juste pour prévenir si le firmware Katapult de la carte mère n'a pas été programmé correctement
* Vous pouvez rapidement appuyer deux fois sur le bouton de réinitialisation et vérifier si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * Le SB2040 PRO peut être connecté à la machine hôte de la manière suivante
    * [Firmware CAN](./can.mdx "Cliquez pour accéder")
        * Avantages : Si le firmware `Katapult` est programmé, il permet la mise à jour du firmware sans toucher à la carte mère, et supporte la connexion par bus
        * Inconvénients : Peut provoquer des erreurs comme le dépassement de délai lors de l'homing lorsqu'utilisé avec `KlipperScreen`.
        * Astuce : Si vous avez besoin d'utiliser le CAN, assurez-vous que la machine hôte le supporte et qu'elle est équipée d'UTOC ou d'une carte mère avec un firmware de pont

## Description du firmware Katapult

    * Le SB2040 PRO est livré par défaut avec un firmware Katapult pour la connexion CAN à une vitesse de 1M
