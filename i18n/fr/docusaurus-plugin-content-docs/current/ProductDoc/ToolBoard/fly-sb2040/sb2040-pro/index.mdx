# Présentation du produit

    FLY SB2040 PRO est une carte d'outils lancée par Guangzhou Mellow Electronics Technology Co., Ltd pour la tête d'impression VORON Stealthburber. En utilisant cette carte, on peut remplacer les câblages complexes entre la carte mère et la tête d'impression par seulement quatre fils, optimisant ainsi la disposition. SB2040 est adapté pour Stealthburner et Afterburner, et intègre un récepteur-transmetteur CAN, un port USB, un driver TMC2240, un capteur d'accélération ADXL345, trois ventilateurs contrôlables et d'autres fonctions. Cette carte d'outils est compatible avec les firmwares Klipper et RRF.

## Caractéristiques du produit

    * Bouton Boot pré-installé, l'utilisateur peut entrer en mode Boot via TypeC pour mettre à jour le firmware
    * Prend en charge le firmware Klipper
    * Capteurs de température supportés : puce MAX31865 intégrée, compatible avec PT100
    * Capteur d'accélération ADXL345 intégré
    * Interface RGB réservée, driver 2240 intégré, supporte divers capteurs de nivellement comme Klicky, Voron Tap, PL08n, etc.
    * Supporte une tension d'entrée de 12-24V, un courant de 15A, avec un port de barre chauffante augmenté, courant nominal de 10A


## Spécifications du produit


    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Interface de connexion utilisant un connecteur XT30(2+2) supportant un courant de 15A, pic à 30A
    * Firmware : Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs intégrés : ADXL345 et MAX31865 optionnel
    * Interface de barre chauffante : Barre chauffante (HEAT0)
    * Interfaces d'extension : 3 ventilateurs contrôlables, 1 RGB, 3 fins de course, 1 tube chauffant, 1 thermistance
    * Interfaces de ventilateur : trois voies de ventilateurs contrôlables
    * Driver de moteur : TMC2240 intégré
