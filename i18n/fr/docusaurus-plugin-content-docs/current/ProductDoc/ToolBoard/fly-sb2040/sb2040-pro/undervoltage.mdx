# Méthodes pour résoudre le problème de sous-tension du moteur

## Méthode 1

    :::danger

    * Remplacer l'alimentation
    * Ajouter un filtre d'alimentation
    * Remplacer le moteur pour diagnostiquer
    * Connecter le boîtier du moteur à la GND de la carte
    * Ajouter `driver_TPFD: 0` dans la configuration du driver de l'extrudeuse

    :::

## Méthode 2
    * Se connecter à l'hôte par SSH
    * Entrer la commande ci-dessous
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Cette modification pourrait être annulée après une mise à jour de Klipper
    :::
