# Utilisation d'Eddy

    :::info

    * Veuillez noter que le scan du lit est recommandé d'utiliser une limite Z séparée, il n'est pas recommandé d'utiliser le scan du lit comme limite Z
    * Lors de l'installation d'Eddy, la position du bas par rapport à la buse doit être au moins 1-2 mm au-dessus de la buse
    * Lors de l'étalonnage de la fréquence Eddy, la PCB de la bobine doit être à au moins 5 mm du lit chauffant
    * Veuillez ne pas tester avec le lit chauffant à la température ambiante, et ne pas allumer le lit chauffant pendant le test

    :::

## Configuration de référence et instructions d'utilisation

    ### Configuration du scan du lit

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Veuillez noter qu'il n'y a pas encore de module externe 1612 disponible, cette configuration ne peut pas être utilisée pour le moment
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Configuration de la compensation thermique

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Configuration bed_mesh

    * Le paramètre `horizontal_move_z` dans [bed_mesh], il est recommandé de le régler à 2, pour que Eddy soit le plus proche possible de la plateforme lors du scan

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
