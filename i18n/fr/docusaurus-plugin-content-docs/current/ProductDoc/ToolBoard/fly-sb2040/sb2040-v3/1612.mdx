# Utilisation d'Eddy

    :::info

    * Veuillez noter que pour le balayage du lit, il est recommandé d'utiliser une limite Z séparée, et il n'est pas recommandé d'utiliser le balayage comme limite Z
    * Lors de l'installation d'Eddy, la position de la base par rapport à la buse doit être d'au moins 1-2 mm au-dessus de la buse
    * Lors de l'étalonnage de la fréquence d'Eddy, le PCB de la bobine doit être éloigné du lit chauffant d'au moins 5 mm
    * Veuillez ne pas tester le lit chauffant à la température ambiante, et ne pas allumer le lit chauffant lors des tests

    :::

## Configuration de référence et instructions d'utilisation

    ### Configuration du balayage du lit

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Veuillez noter qu'il n'y a actuellement pas de module externe 1612 disponible, cette configuration n'est donc pas utilisable pour le moment
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Configuration de la compensation de température

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Configuration de bed_mesh

    * Dans le paramètre `horizontal_move_z` de [bed_mesh], il est recommandé de le régler à 2, afin que lors du balayage de la plateforme, Eddy soit le plus proche possible de la plateforme

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
