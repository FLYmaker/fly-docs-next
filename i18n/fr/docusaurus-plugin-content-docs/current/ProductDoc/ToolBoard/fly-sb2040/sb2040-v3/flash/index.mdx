# Explications sur la gravure du firmware

    * Le SB2040 V3 peut se connecter à l'ordinateur via RS232 ou CAN
    * RS232 et CAN nécessitent l'utilisation de `XT30(2+2)-F` pour la connexion
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La gravure du firmware de la carte mère doit être effectuée via Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà le firmware Katapult, vous n'avez pas besoin de graver le firmware Katapult à la fin, c'est juste une précaution en cas d'échec de la gravure du firmware Katapult sur la carte mère
* Vous pouvez appuyer deux fois rapidement sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est entrée dans le mode firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * Le SB2040 V3 peut se connecter à l'ordinateur de la manière suivante
    * <Button variant="contained" disableElevation href="./can">Gravure du firmware CAN</Button>
        * Avantages : Si le firmware `Katapult` est gravé, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et il supporte la connexion par bus
        * Inconvénients : Peut causer des erreurs comme le dépassement du temps de retour à zéro lors de l'utilisation avec `KlipperScreen`.
        * Astuce : Si vous avez besoin d'utiliser CAN, assurez-vous que l'ordinateur le supporte et que vous avez UTOC ou une carte mère avec firmware de pontage
    * <Button variant="contained" disableElevation href="./rs232">Gravure du firmware RS232</Button>
        * Avantages : Aucun logiciel spécial n'est requis, la communication est fiable et stable
        * Inconvénients : Pour mettre à jour le firmware, il faut entrer en mode DFU manuellement ou double-cliquer sur RST pour entrer dans `Katapult` pour mettre à jour le firmware
        * Astuce : Nécessite G2T ou UTOR pour être utilisé
