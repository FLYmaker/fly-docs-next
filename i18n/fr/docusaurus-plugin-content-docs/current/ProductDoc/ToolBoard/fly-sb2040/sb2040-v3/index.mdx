# 1. Introduction du produit

    FLY SB2040 V3 est une carte outil lancée par Guangzhou Mellow Electronic Technology Co., Ltd pour la tête d'impression VORON Stealthburber. En utilisant cette carte outil, il est possible de remplacer le câblage complexe entre la carte mère et la tête d'impression par quatre fils, optimisant ainsi la disposition. Le SB2040 est adapté à Stealthburner et Afterburner, intégrant un récepteur/émetteur CAN, un récepteur/émetteur RS232, un port USB, un driver TMC2209, un capteur d'accélération LIS2DW, trois ventilateurs contrôlables, et d'autres fonctionnalités. Cette carte outil est compatible avec les firmwares Klipper et RRF.

## 1.1 Caractéristiques du produit

    * Bouton de démarrage réservé, permettant aux utilisateurs d'entrer en mode Boot pour mettre à jour le firmware via TypeC
    * Supporte les firmwares Klipper et RRF
    * Supporte les modes CAN, RS232, et RRF CAN,
    * En mode RS232, il offre un débit de transmission plus élevé et une stabilité accrue. En mode CAN, il supporte l'extension CAN avec une interface d'extension CAN réservée
    * Capteurs de température supportés : chip embarqué MAX31865, compatible avec les capteurs de température PT100 et PT1000.
    * Capteur d'accélération LIS2DW embarqué et capteur LDC1612
    * Interfaces RGB réservées, driver 2209 embarqué, supportant divers capteurs de nivellement comme klicky, Voron Tap, PL08n, etc.
    * Supporte une tension d'entrée de 12-24V, un courant d'entrée de 15A, port de barrette chauffante augmenté, courant nominal de 10A

## 1.2 Nouvelles fonctionnalités

    * Supporte le basculement entre les modes CAN de Klipper, RS232, et RRF CAN
    * Nouveau mode RS232 offrant un débit de transmission plus élevé, une meilleure stabilité, et une utilisation en débogage plus simple
    * Nouveau mode CAN RRF permettant de basculer en carte outil RRF, améliorant la compatibilité
    * Nouvelle interface d'extension CAN facilitant l'extension des appareils CAN
    * Nouvelle bascule de résistance terminale, la résistance terminale de 120R peut être sélectionnée via un commutateur sans avoir besoin de cavaliers
    * Nouveau câble d'alimentation/signal de 1 mètre, le câble utilise un design torsadé pour une meilleure résistance aux interférences

## 1.3 Paramètres du produit


    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Les interfaces de connexion utilisent des connecteurs XT30(2+2) supportant un courant de 15A, pic à 30A
    * Firmware : Reprap/Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs embarqués : LIS2DW et optionnels ldc1612 et max31865
    * Interface de barrette chauffante : Barrette chauffante (HEAT0)
    * Interfaces d'extension : RGB, EndStop, PT100/PT1000, port CAN XH2.54-2P
    * Interfaces de ventilateur : un ventilateur DC, deux ventilateurs DC ou PWM
    * Driver de moteur : Driver TMC2209 embarqué
    * Capteurs de température : un capteur NTC 100K ou PT1000 (TH0), un capteur PT100/PT1000 optionnel
