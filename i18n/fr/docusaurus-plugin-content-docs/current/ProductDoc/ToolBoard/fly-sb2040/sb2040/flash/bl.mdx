# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * SB2040 supporte le CAN
    * Ici, nous fournissons uniquement des firmwares CAN à 500K et 1M de taux de transmission
    :::

## Entrer en mode flash

    :::tip

    Le bouton entouré est le `BOOT`

    :::

    * Dans le cas où la carte mère est complètement éteinte, maintenez le bouton BOOT enfoncé puis connectez le câble de données à la carte mère et à votre appareil
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/boot.png').default} size="50%" align="center" />

            4. Flasher le firmware

            :::tip

            Il suffit de flasher un seul firmware

            :::

            * Voici la commande pour flasher le firmware à 500K CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * Voici la commande pour flasher le firmware à 1M CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. Si le message suivant apparaît, le flashage a réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            Après le flashage, il est nécessaire de couper complètement l'alimentation de la carte mère une fois.
