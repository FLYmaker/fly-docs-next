# Instructions pour la mise à jour du firmware

    * Le SB2040 peut se connecter à l'hôte via une connexion CAN
    * La connexion CAN nécessite l'utilisation de `XT30(2+2)-F`
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger À savoir
* La mise à jour du firmware de la carte mère doit se faire via Katapult pour flasher Klipper
* Si vous êtes sûr que la carte mère a déjà Katapult, il n'est pas nécessaire de procéder à la mise à jour du firmware Katapult, c'est juste une précaution au cas où le firmware Katapult de la carte mère n'aurait pas été flashé correctement
* Vous pouvez rapidement appuyer deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote, indiquant que la carte mère est entrée en mode Katapult
:::

## Comment déterminer le firmware dont vous avez besoin

    * Le SB2040 ne peut se connecter à l'hôte que via CAN
    * <Button variant="contained" disableElevation href="./can">Mise à jour du firmware CAN</Button>
        * Avantages : Si le firmware `Katapult` est installé, vous pouvez mettre à jour le firmware sans toucher à la carte mère et cela supporte la connexion par bus
        * Inconvénients : L'utilisation avec `KlipperScreen` peut provoquer des erreurs comme des dépassements de délai lors du retour à la position d'origine
        * Astuce : Si vous avez besoin d'utiliser CAN, assurez-vous que votre hôte le supporte et qu'il est équipé d'UTOC ou d'une carte mère avec un firmware de pont
