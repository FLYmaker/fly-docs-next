# Firmware Katapult

## Explications sur le firmware Katapult

    <CANBOOT />

    :::tip

    * SHT36/42 prend en charge CAN
    * Ici, nous fournissons uniquement le firmware CAN avec des vitesses de 500K et 1M
    :::

## Mode de programmation

    :::tip

    Le bouton encerclé est le `BOOT`

    :::

    * Avec la carte mère complètement débranchée, court-circuitez le bouton BOOT puis connectez le câble de données avec une extrémité à la carte mère et l'autre à votre appareil

            4. Flasher le firmware

            :::tip

            Il suffit de flasher un seul firmware

            :::

            * Voici la commande pour flasher le firmware à 500K CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Voici la commande pour flasher le firmware à 1M CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. Si le message ``File downloaded successfully`` apparaît comme indiqué par la flèche ci-dessous, alors le flashage a réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            Après le flashage, il est nécessaire de débrancher complètement la carte mère.
