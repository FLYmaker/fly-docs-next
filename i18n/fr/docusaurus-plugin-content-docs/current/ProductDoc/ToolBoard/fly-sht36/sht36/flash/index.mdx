# Instructions pour la mise à jour du firmware

    * Les SHT36/42 peuvent se connecter à l'ordinateur via une connexion CAN
    * La connexion CAN nécessite l'utilisation de bornes métalliques MX3.0 pour le câblage
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />

## Comment déterminer quel firmware vous avez besoin

    * Les SHT36/42 peuvent se connecter à l'ordinateur via une connexion CAM
    * <Button variant="contained" disableElevation href="./can">Mise à jour du firmware CAN</Button>
        * Avantages : Si vous flashez le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et cela supporte la connexion par bus.
        * Inconvénients : Peut causer des erreurs comme des timeouts de retour à zéro lorsqu'utilisé avec `KlipperScreen`.
        * Conseils : Si vous avez besoin d'utiliser CAN, assurez-vous que votre ordinateur le supporte et qu'il est équipé d'un UTOC ou d'une carte mère avec un firmware de pont.
