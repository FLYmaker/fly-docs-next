# Introduction au produit

    FLY SHT36/42 est une carte outil pour buse conçue par Guangzhou Mellow Electronics Technology Co., Ltd. pour les extrudeuses de machines d'impression 3D. En utilisant cette carte, vous pouvez remplacer le câblage complexe entre la carte mère et la tête d'impression par quatre fils, optimisant ainsi la disposition. SHT36 est adapté pour les moteurs pas à pas de 36, SHT42 pour ceux de 42. Elle intègre un transceiver CAN, un port USB, un driver TMC2209, un accéléromètre ADXL345, deux ventilateurs contrôlables et d'autres fonctions. Cette carte outil est compatible avec le firmware Klipper.

    * Compression extrême du volume, avec une conception de carte à 6 couches
    * Prend en charge la connexion par bus CAN, pour une transmission de données plus stable, moins de latence et une connexion plus fiable
    * Driver TMC2209 intégré, en mode UART, utilisable pour l'extrudeuse
    * Accéléromètre ADXL345 intégré
    * Supporte le PT100 (MAX31865)
    * Conception MOS de ventilateur remplaçable
    * Supporte les capteurs de nivellement BLTOUCH, PL08N, intègre des interrupteurs de fin de course XY
    * Supporte une entrée de tension de 12-24V
    * MCU : STM32F072
    * Communication : CANBUS, USB
    * Interfaces : ventilateurs contrôlables *2, RGB *1, fins de course *3, sonde *1, servo *1, tube chauffant *1, thermistance *1, PT100 *1

## Spécifications du produit

    1. Dimensions extérieures : 51.27mm*36.26mm (SHT36), 42mm*42mm (SHT42)
    2. Dimensions de montage : espacement des trous de 43.84mm, diamètre des trous de montage M3
    3. Processeur : STM32F072C8T6
    4. Tension d'entrée : DC12-24V 5A
    5. Courant maximum de sortie de la tige chauffante : 5A
    6. Courant maximum de sortie de l'interface du ventilateur : 1A
