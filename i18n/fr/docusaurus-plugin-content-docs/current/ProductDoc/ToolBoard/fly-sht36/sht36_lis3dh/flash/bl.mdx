# Micrologiciel Katapult

## Description du micrologiciel Katapult

    <CANBOOT />

    :::tip

    * SHT36 LIS3DH prend en charge CAN
    * Nous fournissons uniquement le micrologiciel CAN avec des vitesses de 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    Le bouton entouré est le `Bouton BOOT`

    :::

    * Débranchez complètement la carte mère, court-circuitez le bouton BOOT et branchez un câble de données avec la carte mère et votre appareil
    * Puis retirez le court-circuit sur le bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/boot.webp').default} size="50%" align="center" />

            * Voici la commande pour flasher le micrologiciel à une vitesse CAN de 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_500K.uf2
                ```
                        
                * Voici la commande pour flasher le micrologiciel à une vitesse CAN de 1M

                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_1M.uf2
                ```
