# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * SHT36 PRO prend en charge CAN
    * Ici, nous fournissons uniquement le firmware CAN avec des taux de 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    Le bouton encerclé est le `bouton BOOT`

    :::

    * Avec la carte mère complètement déconnectée, court-circuitez le bouton BOOT puis connectez un câble de données à la carte mère et à votre appareil
    * Ensuite, relâchez le court-circuit du bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="80%" align="center" />

            4. Flasher le firmware

            :::tip

            Vous n'avez besoin de flasher qu'un seul firmware

            :::

            * Voici la commande pour flasher le firmware avec un taux CAN de 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Voici la commande pour flasher le firmware avec un taux CAN de 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Si vous voyez le message ci-dessous, le flashage a réussi

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            Une fois le flashage terminé, vous devez couper l'alimentation de la carte mère une fois.

            :::
