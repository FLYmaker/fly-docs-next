# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip

    * Le SHT36 PRO prend en charge le CAN
    * Ici, nous ne fournissons que les firmwares CAN à des vitesses de 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    La partie entourée est le `bouton BOOT`

    :::

    * Avec la carte mère complètement débranchée, court-circuitez le bouton BOOT puis connectez un câble de données à la carte mère et à votre appareil
    * Ensuite, relâchez le bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="100%" align="center" />

            4. Flasher le firmware

            :::tip

            Il suffit de flasher l'un des firmwares

            :::

            * Voici la commande pour flasher le firmware à une vitesse CAN de 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Voici la commande pour flasher le firmware à une vitesse CAN de 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Si vous voyez le message suivant, le flashage a réussi

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            Après le flashage, il est nécessaire de débrancher complètement la carte mère.

            :::
