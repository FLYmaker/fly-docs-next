# Explications sur la gravure du firmware

    * Le SHT36 PRO peut se connecter à la machine hôte via une connexion CAN
    * La connexion CAN nécessite l'utilisation de `XT30(2+2)-F`
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La gravure du firmware de la carte mère nécessite l'utilisation de Katapult pour graver Klipper
* Si la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de procéder à la gravure du firmware Katapult à la fin, ce n'est qu'une précaution au cas où la gravure du firmware Katapult de la carte mère échouerait
* Vous pouvez rapidement appuyer deux fois sur la réinitialisation pour voir si un LED sur la carte mère commence à clignoter, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult
:::

## Compilation du firmware CAN

    * Le SHT36 PRO ne peut être connecté à la machine hôte que par le biais de la connexion CAN
    * <Button variant="contained" disableElevation href="./can">Gravure du firmware CAN</Button>
        * Avantage : Si le firmware `Katapult` est gravé, il est possible de mettre à jour le firmware sans toucher à la carte mère et de supporter la connexion par bus
        * Inconvénient : Peut provoquer des erreurs comme un dépassement de temps lors du retour à l'origine lorsqu'il est utilisé avec `KlipperScreen`.
        * Conseil : Si vous souhaitez utiliser CAN, assurez-vous que la machine hôte le supporte et nécessite l'utilisation d'UTOC ou d'une carte mère gravée avec le firmware de pont
