# Présentation du produit

    FLY-SHT36-PRO est une carte d'outils pour la buse de l'extrudeuse de l'imprimante 3D conçue par Guangzhou Mellow Electronic Technology Co., Ltd. En utilisant cette carte, vous pouvez remplacer les câbles complexes entre la carte mère et la tête d'impression par seulement quatre fils, optimisant ainsi le câblage. FLY-SHT36-PRO convient aux moteurs pas à pas de 36 pas, intégrant un émetteur-récepteur CAN, un port USB, un driver TMC5160, un capteur d'accélération ADXL345, deux ventilateurs contrôlables, ainsi que d'autres fonctionnalités.

    * Conception ultra-compacte avec une carte à 6 couches
    * Supporte la connexion par bus CAN pour une transmission de données plus stable, une latence réduite et une connexion plus fiable
    * Driver TMC5160 intégré pour l'extrudeuse
    * Capteur d'accélération ADXL345 intégré
    * Supporte PT100 (MAX31865)
    * Conception MOS pour ventilateur interchangeable
    * Supporte les capteurs de nivellement BLTOUCH, PL08N, et intègre des interrupteurs de fin de course XY
    * Supporte une entrée de tension de 12-24V
    * MCU : STM32F072
    * Communication : CANBUS, USB

## Paramètres du produit

    1. Supporte le mode CAN de Klipper, le mode RS232, et la commutation entre les modes RRFCAN
    2. Les interfaces de connexion utilisent un connecteur XT30 (2+2) supportant un courant de 15A, avec un pic de 30A
    3. MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    4. Tension d'entrée : DC12-24V 5A
    5. Courant de sortie maximum de la barre de chauffage : 5A
    6. Courant de sortie maximum de l'interface de ventilateur : 1A
