        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * La compilation est réussie si la sortie finale est :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Flashage du Firmware

    * Avant de flasher le firmware, connectez le panneau d'outils à l'UTOC ou à la carte mère avec le firmware de pontage à l'aide d'un câble de données XT30, ci-dessous un schéma de câblage de référence
    * Veuillez vérifier si les commutateurs sont bien positionnés
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * Assurez-vous que la résistance entre CAN H et CAN L, mesurée avec un multimètre en mode ohm, est d'environ `60Ω` lorsque l'appareil est hors tension. Si la résistance est de `120Ω`, ajoutez un jumper de `120Ω`. Si elle est d'environ `40Ω`, retirez le jumper.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.webp').default} size="50%" align="center" />
