# Instructions pour la mise à jour du firmware

    * La compilation du firmware et la mise à jour du firmware nécessitent l'utilisation d'un logiciel hôte
    * Le SHT36 V2 peut être connecté à l'hôte via CAN
    * Le CAN nécessite l'utilisation de `XT30(2+2)-F` pour la connexion
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La mise à jour du firmware de la carte mère nécessite de flasher Klipper via Katapult
* Si vous êtes sûr que la carte mère a déjà été flashée avec Katapult, vous n'avez pas besoin de flasher le firmware Katapult à la fin, c'est juste pour s'assurer que le firmware Katapult de la carte mère a été flashé avec succès
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère clignote, si c'est le cas, cela signifie que la carte mère est entrée dans le firmware Katapult
:::

## Compilation du firmware CAN

    * Le SHT36 V2 ne peut être connecté à l'hôte que par la méthode CAN
    * STM 103, veuillez consulter   <Button variant="contained" disableElevation href="./can103">Mise à jour du firmware CAN 103</Button>
    * APM 072 et STM 072, veuillez consulter   <Button variant="contained" disableElevation href="./can072">Mise à jour du firmware CAN 072</Button>
