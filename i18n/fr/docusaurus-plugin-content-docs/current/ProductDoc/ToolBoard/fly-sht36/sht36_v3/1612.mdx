# Utilisation d'Eddy

    :::info

    * Veuillez noter que l'utilisation d'un capteur de lit séparé est recommandée pour le Z-limite, il n'est pas recommandé d'utiliser le capteur de lit comme Z-limite
    * Lors de l'installation d'Eddy, la position de la base par rapport à la buse doit être au moins 1-2 mm au-dessus de la buse
    * Lors du réglage de la fréquence d'Eddy, la PCB de la bobine doit être éloignée du lit de chauffage d'au moins 5 mm
    * Veuillez ne pas tester avec le lit de chauffage à température ambiante, et ne pas allumer le lit pendant le test

    :::

## Configuration et instructions d'utilisation

    ### Configuration du capteur de lit

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Veuillez noter qu'il n'existe pas encore de module externe 1612, donc cette configuration ne peut pas être utilisée pour l'instant
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Configuration de la compensation de température

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### Configuration de bed_mesh

    * Dans [bed_mesh], le paramètre `horizontal_move_z` doit être défini à 2, afin que Eddy soit le plus proche possible de la plateforme lors de la numérisation

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
