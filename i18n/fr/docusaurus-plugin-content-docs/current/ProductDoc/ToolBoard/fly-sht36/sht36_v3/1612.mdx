# Utilisation de Eddy

    :::info

    * Veuillez noter que la numérisation du lit suggère l'utilisation d'une limite Z séparée, il n'est pas recommandé d'utiliser la numérisation du lit comme limite Z
    * Lors de l'installation de Eddy, la position de la partie inférieure par rapport à la buse doit être au moins 1 à 2 mm au-dessus de la buse
    * Lors de l'étalonnage de la fréquence de Eddy, le PCB de la bobine doit être à au moins 5 mm du lit chauffant
    * Veuillez ne pas tester avec le lit chauffant à température ambiante et ne pas allumer le lit chauffant pendant le test

    :::

## Configuration et instructions d'utilisation

    ### Configuration de la numérisation du lit

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Veuillez noter que pour l'instant, il n'existe pas de module externe 1612, donc cette configuration ne peut pas être utilisée pour le moment
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Configuration de la compensation de température

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### Configuration de bed_mesh

    * Dans [bed_mesh], il est suggéré de régler le paramètre `horizontal_move_z` à 2 pour que Eddy soit le plus proche possible de la plateforme lors de la numérisation

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
