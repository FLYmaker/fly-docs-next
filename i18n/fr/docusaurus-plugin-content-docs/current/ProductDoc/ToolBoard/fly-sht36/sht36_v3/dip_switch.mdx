# Introduction aux commutateurs embarqués

:::warning Points d'attention

En raison de la haute résolution des images, il est impossible de les voir clairement sur la page web. Veuillez **placer le curseur de la souris sur le schéma de câblage**, **cliquer avec le bouton droit de la souris**, et choisir **Enregistrer l'image sous** pour sauvegarder l'image sur votre bureau et la consulter ensuite !!!

:::


## Sélection du mode de fonctionnement

:::warning Points d'attention

Veuillez noter que le SHT36 V3 prend en charge trois modes de connexion. Il est nécessaire de modifier les commutateurs et de flasher le firmware pour que la connexion fonctionne correctement.
