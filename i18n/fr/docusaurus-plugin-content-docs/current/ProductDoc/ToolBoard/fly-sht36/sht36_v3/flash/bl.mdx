# Firmware Katapult

## Description du Firmware Katapult

    <CANBOOT />

    :::tip

    * Le SHT36_V3 prend en charge CAN et RS232
    * Seul le firmware CAN est fourni ici avec des vitesses de 500K et 1M
    :::

## Entrer en mode flash

    :::tip

    La flèche indique le bouton `BOOT`

    :::

    * Avec la carte mère complètement déconnectée, maintenez enfoncé le bouton BOOT, puis connectez un câble de données à la carte mère et à votre appareil
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="100%" align="center" />

            4. Flasher le firmware

            :::tip

            Vous n'avez besoin de flasher qu'un seul firmware

            :::

            * Voici la commande pour flasher le firmware à la vitesse CAN de 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Voici la commande pour flasher le firmware à la vitesse CAN de 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. Si le message suivant apparaît, le flashage est réussi

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            Après le flashage, vous devez complètement déconnecter la carte mère une fois.

            :::
