# Instructions de flashage du firmware

    * La compilation du firmware et le flashage du firmware nécessitent l'utilisation d'un logiciel de contrôle
    * Le SHT36 V3 peut être connecté au logiciel de contrôle via RS232 ou CAN
    * RS232 et CAN doivent être connectés en utilisant `XT30(2+2)-F`
    * Voici le style du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger À savoir
* Le flashage du firmware de la carte mère nécessite de flasher Klipper via Katapult
* Si vous êtes certain que la carte mère a déjà été flashée avec Katapult, il n'est pas nécessaire de flasher à nouveau le firmware Katapult, c'est juste une précaution au cas où le flashage du firmware Katapult de la carte mère aurait échoué
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si un LED sur la carte mère commence à clignoter, si c'est le cas, cela indique que la carte mère est entrée en mode firmware Katapult
:::

## Comment déterminer de quel firmware vous avez besoin

    * Le SHT36 V3 peut se connecter au logiciel de contrôle de la manière suivante
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantages : Si le firmware `Katapult` est installé, vous pouvez mettre à jour le firmware sans toucher à la carte mère, et il supporte la connexion par bus
        * Inconvénients : Peut causer des erreurs de timeout de retour à l'origine lorsqu'il est utilisé avec `KlipperScreen`.
        * Conseils : Assurez-vous que le logiciel de contrôle supporte CAN, et nécessite UTOC ou une carte mère avec un firmware de pont
    * <Button variant="contained" disableElevation href="./rs232">Firmware RS232</Button>
        * Avantages : Pas besoin de logiciels spéciaux pour l'utiliser, communication fiable et stable
        * Inconvénients : Pour mettre à jour le firmware, il faut entrer en mode DFU manuel ou double-cliquer sur RST pour entrer en mode `Katapult` pour mettre à jour le firmware
        * Conseils : Nécessite d'être utilisé avec G2T ou UTOR
