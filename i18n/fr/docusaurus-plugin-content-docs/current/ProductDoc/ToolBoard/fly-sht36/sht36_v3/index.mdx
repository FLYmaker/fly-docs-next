# 1. Introduction au produit

FLY SHT36 V3 est une carte d'outil de buse conçue par Guangzhou Mellow Electronic Technology Co., Ltd. pour la tête d'extrusion des imprimantes 3D. Avec cette carte d'outil, il est possible de remplacer le câblage complexe entre la carte mère et la tête d'impression par quatre fils, optimisant ainsi la disposition. La SHT36 V3 est adaptée aux moteurs pas à pas de 36, et intègre un émetteur-récepteur CAN, un émetteur-récepteur RS232, un port TypeC, un driver TMC2209, un capteur d'accélération LIS2DW, 2 ventilateurs contrôlables, et d'autres fonctions. Cette carte d'outil est compatible avec les firmwares Klipper et RRF.

## 1.1 Caractéristiques du produit

    * Bouton Boot préinstallé, les utilisateurs peuvent entrer en mode Boot via TypeC pour mettre à jour le firmware
    * Compatible avec les firmwares Klipper et RRF
    * Supporte les modes CAN, RS232, et RRF CAN,
    * En mode RS232, offre une vitesse de transmission plus élevée et une meilleure stabilité. En mode CAN, permet l'extension CAN avec une interface d'extension CAN prévue
    * Capteur de température supporté : le chip MAX31865 intégré est compatible avec les capteurs de température PT100 et PT1000.
    * Supporte deux ventilateurs contrôlables, avec une tension sélectionnable, remplaçable par une petite carte Mos pour faciliter la maintenance et le remplacement, avec un courant de sortie maximal de 1A
    * Capteur d'accélération LIS2DW et capteur LDC1612 intégrés
    * Interface RGB prévue, driver 2209 intégré, supporte plusieurs capteurs de nivellement tels que BLtouch, klicky, Voron Tap, PL08n, etc.
    * Supporte une tension d'entrée de 12-24V, un courant d'entrée de 15A, avec un port de barre chauffante agrandi, courant nominal de 10A

## 1.2 Nouvelles fonctionnalités

    * Supporte le basculement entre les modes Klipper CAN, RS232, et RRF CAN
    * Ajout du mode RS232, offrant une vitesse de transmission plus élevée, une meilleure stabilité, et une utilisation en débogage plus simple
    * Ajout du mode RRF CAN, peut basculer en tant que carte d'outil RRF, améliorant la compatibilité
    * Ajout d'une interface d'extension CAN, facilitant l'extension des appareils CAN
    * Ajout d'un interrupteur à bascule pour la résistance de terminaison, la résistance de terminaison de 120R peut être sélectionnée par un interrupteur sans nécessiter de cavalier
    * Ajout d'un câble d'alimentation/signal de 1 mètre, avec un design en torsade pour une meilleure résistance aux interférences

## 1.3 Paramètres du produit

    * MCU : Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Les interfaces de connexion utilisent un connecteur XT30 (2+2) supportant un courant de 15A, avec un pic à 30A
    * Firmware : Reprap/Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs intégrés : LIS2DW et LDC1612
    * Interface de barre chauffante : barre chauffante (HEAT0), 10A
    * Interfaces d'extension : RGB, EndStop, PT100/PT1000, port d'extension CAN XH2.54-2P
    * Interfaces de ventilateur : deux ventilateurs DC contrôlables
    * Driver de moteur : TMC2209 intégré
    * Capteur de température : un NTC de 100K ou PT1000 (TH0), un PT100/PT1000 optionnel
