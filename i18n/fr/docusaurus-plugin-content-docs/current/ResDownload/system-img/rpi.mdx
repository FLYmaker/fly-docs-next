# Image pour Raspberry Pi


  :::warning

  Précautions pour la carte SD :

  * La capacité doit être d'au moins `16GB` ou ne pas dépasser `128G` pour la carte TF
  * La vitesse de transfert de la carte TF doit être de `classe 10` ou supérieure, il est recommandé d'utiliser une carte TF de marque comme SanDisk

  :::

  :::danger À savoir

  * En raison de problèmes de serveur, veuillez ne pas mettre à jour le système ou Klipper à la légère
  * Image système compatible avec Raspberry Pi 3B/3B+/4B/CM4/5B
  * Cette image est basée sur `bookworm` pour l'installation de divers services
  * Cette image installe et active par défaut l'écran TFT, si vous n'utilisez pas l'écran TFT, vous devrez manuellement modifier `/boot/config.txt`

  :::

  :::info Important

  * Veuillez noter que le système téléchargé doit être décompressé, et non simplement renommé ou gravé directement !!!
  * Si vous ne le décompressez pas avant de graver, il ne démarrera pas !!!

  :::
