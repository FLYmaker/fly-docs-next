# Installation de FLY-Tools

    1. Récupérer la dernière version de Klipper

    ```bash
    cd ~/klipper
    git pull
    ```

    * Si vous avez modifié le code source local de Klipper et que le pull échoue, utilisez la commande suivante. (Cette opération supprimera les fichiers modifiés précédemment)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. Récupérer la dernière version de Katapult (anciennement Canboot)

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Installer le service Fly-Tools

        * Le processus d'installation dépend du fournisseur de service Git. Si le téléchargement reste bloqué, appuyez sur `CTRL+C` pour sortir et réessayer.
        * Github limite les requêtes à 60 par heure, le nombre pour Gitee est inconnu.
        * Les utilisateurs en Chine sont invités à utiliser Gitee pour obtenir le script d'installation.
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Obtenir le script d'installation via Github

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
