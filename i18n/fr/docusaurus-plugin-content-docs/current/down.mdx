# Téléchargement de ressources

:::tip

Veuillez sélectionner le type de ressource correspondant dans la barre de navigation à gauche pour télécharger.

:::

:::warning Attention

Lors du téléchargement de fichiers, veuillez prêter attention au nom du fichier et aux indications sur la page pour éviter de télécharger le mauvais fichier.

:::

:::danger À savoir

Certains fichiers peuvent être stockés sur Github ou dans un cloud, assurez-vous que votre réseau fonctionne correctement lors du téléchargement.

:::
