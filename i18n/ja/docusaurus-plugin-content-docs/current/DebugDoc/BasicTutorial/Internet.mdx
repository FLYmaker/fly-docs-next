# SSHを通じてネットワークに接続する


* システムが起動し、SSHを使用して上位機にシリアル接続されていることを確認してください。**MobaXterm_Personal**の使用を推奨します。
* 上位機に**M2WE**または**2.4G_WiFiモジュール**がインストールされていることを確認してください。**Lite2**と**MiniPad**の場合はモジュールは不要です。
* **2.4G_WiFiモジュール**、**Lite2**、**MiniPad**の3つは5G WiFiを使用できず、2.4G WiFiのみ使用可能であることに注意してください。
* ネットワークはデュアルバンド統合を使用しないでください。そうしないと正常に接続できません。
* アンテナが完全にインストールされていることを確認してください！！！

 1. SSHソフトウェアで**nmtui**コマンドを入力してエンターキーを押すと、以下の画面が表示されます。

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. エンターを押して**Activate a connection**を選択し、再度エンターを押します。

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. 接続したいネットワークを選択し、エンターを押してWiFiパスワードを入力し、**OK**を選択して接続を保存します。

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. WiFiに接続した後、キーボードから**Esc**キーを何度か押して現在の画面を終了し、**ip a**を入力してエンターキーを押します。

 * 図中の192.168.2.136が上位機のIPアドレスです。

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
