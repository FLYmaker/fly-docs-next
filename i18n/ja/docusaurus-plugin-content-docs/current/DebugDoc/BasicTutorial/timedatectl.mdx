# システムのタイムゾーンを変更する

    FLY上位機がFLY OS 3.3.4を使用している場合、FLY-Toolsを使って直接タイムゾーンを変更できます
    その他の上位機は以下の手順をご覧ください


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    `<TIMEZONE>`を希望するタイムゾーンコードに置き換えてください。タイムゾーンコードは[iana.org](https://www.iana.org/time-zones)のウェブサイトで検索できます。例えば、システムのタイムゾーンを北京（Asia/Shanghai）に変更したい場合は、以下のコマンドを入力します：

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    これで、システムのタイムゾーンが上海時間に変更されました。タイムゾーン設定を確認するには、以下のコマンドを使用してください：

```bash
date
```
