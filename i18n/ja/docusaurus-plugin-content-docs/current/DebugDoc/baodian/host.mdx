# 上位機編

## 新しく買った上位機を動かすにはどうすればいいですか

  1. まず、あなたの上位機が何のモデルか確認してください。Lite 2かPI V2など、どの上位機かを確認します。
  2. 以下のものを準備していますか
        * 書き込み用ソフトウェア
        * 対応するシステムイメージ
        * カードリーダー、TFカード、データ転送機能付きのTYPE-C

  システム書き込み前の注意事項:
        * メモリカードの容量は必ず`16GB`以上または`128G`以下のTFカードで、転送速度は`class10`以上であることを確認してください。サンディスクなどのブランドのTFカードを使用することをお勧めします。
        * システムを勝手に更新しないでください！！サーバーが海外にあるため、更新に問題が発生しやすいです。
        * できるだけ我々が推奨するソフトウェアを使用してください
        * できるだけLINUXの知識を学んでください。私は個人的に <Button variant="contained" disableElevation href="https://item.jd.com/12443890.html">《鳥哥のLinux私房菜 基礎学習篇》</Button> をお勧めします。

  3. FLYシステムイメージ <Button variant="contained" disableElevation href="/docs/category/システムイメージ">システムイメージ</Button>
  4. 必要なツール <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button>

## 各種ツールと対応するシステムイメージをダウンロードした後

  * システムを解凍し、解凍後のファイルの拡張子は`.img`になります
  * SDFormatterを使用してメモリカードを再フォーマットする必要があります
  * `win32DiskImager`を使用してシステムに書き込みます
  * `balenaetcher`でのシステム書き込みはお勧めしません
  * FLY上位機の場合は`FlyOS-Env.txt`を開いて設定する必要があります
  * システムを書き込み後、フォーマットしないでください！！！

## システムの起動

  * システムを書き込み後、上位機にシステムカードをインストールし、電源を入れて起動します
  * この時点でシステムは書き込みと設定が完了しています。データ転送機能付きのTYPE-CでPCに接続し、シリアル接続を使用してください
  * 他の上位機が正常にログを表示しない場合、2回エンターキーを押すか、システムを再書き込みしてください
  * 起動後、WIFIに接続できます


## M2WEとメモリカードの使用説明

  * 機械にM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
  * M2WEはSDカードを使用してシステムを書き込むことをお勧めします。USB経由でM2Wに書き込むと、ドライバーの互換性などの理由で書き込みに失敗する可能性があります。メモリカードを使用してM2WEに書き込むことをお勧めします。
  * ダウンロードしたシステムイメージが対応しているか確認してください。対応していないと起動できません。
  * Lite2とMINI PADはメモリカードのみ使用できます


## 日常使用の注意事項

  * 直接電源を切らないでください。正常にシャットダウンする必要があります。そうしないとデータが失われる可能性があります。
  * GitHudなどのサーバーが海外にあるため、自分で更新したりさまざまなプラグインをインストールするのはお勧めしません。
