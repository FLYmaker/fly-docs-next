# 外部ドライバーの使用ガイド

## 注意事項

:::danger 必知

* 本ガイドはKlipperによる制御を基にしています
* モジュールは共通陽極配線を使用しています

:::


## ドライバーの配線
    :::info

    * 以下の二つの方法から一つを選択してください

    :::
    * 第一の配線方法
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive.png').default} size="80%" align="center" />
    * 第二の配線方法
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive-1.png').default} size="80%" align="center" />

## ドライバーの細分化ジャンパー

    * ドライバーのジャンパーは不要！！！

## ドライバーの内蔵ディップスイッチ
    * ドライバーの内蔵ディップスイッチは`3200`を使用することを推奨します
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/3200.png').default} size="80%" align="center" />

## klipperの参考設定

    * 対応するドライバーの設定をコメントアウトする例えば`[tmc5160 stepper_x]`または`[tmc2209 stepper_x] `
    * `microsteps:`を`16`に設定
    * `step_pulse_duration: 0.000004`を追加
    * モーターの有効化: `enable_pin:!`の`!`を削除
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.png').default} size="50%" align="center" />

## ドライバーのパルス説明

    :::info

    * プリントモデルがずれる場合
    * モーターが振動し、異音やさまざまな問題が発生する場合
    * `step_pulse_duration: 0.000004`を変更することが可能です
    * この値を`0.000009`のように大きくする例えば`0.00001`や`0.00002`など

    :::
