以下はファームウェアのビルド方法を紹介します：
* SSHに接続した後、以下のコマンドを入力し、Enterキーを押します：

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* ここで` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のビルドデータとファームウェアを削除するもので、`make menuconfig`はファームウェアのビルドで、実行後下記の画面が表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* メニューの**Micro-controller Architecture**に入り、**STMicroelectronics STM32**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* メニューの**Processor model**に入り、**STM32F072**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**（ブートローダなし）**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/no.png').default} size="100%" align="Center" />

* **通信インターフェース**を選択し、**USB（PA11/PA12）**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb1.png').default} size="100%" align="Center" />
