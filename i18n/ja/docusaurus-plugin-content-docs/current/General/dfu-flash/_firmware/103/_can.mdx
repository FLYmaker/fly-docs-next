以下是如何编译固件的介绍：
* SSHに接続した後、以下のコマンドを入力してエンターを押します：

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* ここで、` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のコンパイルデータとファームウェアを削除し、`make menuconfig`はファームウェアをコンパイルします。実行後、以下の画面が表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* メニューの**Micro-controller Architecture**に入り、**STMicroelectronics STM32**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* メニューの**Processor model**に入り、**STM32F103**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/103/img/103.png').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**8KiB bootloader**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="Center" />

* **Communication interface**を選択し、**CAN bus (on PB8/PB9)**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.png').default} size="100%" align="Center" />
