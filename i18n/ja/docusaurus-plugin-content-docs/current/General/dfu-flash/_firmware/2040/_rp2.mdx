* BOOTをショートするか長押しし、type cで上位機に接続した後、`lsusb`を入力すると`2e8a:0003`が表示される
    * 現在のRP2040はファームウェアの書き込み状態にあり、後缀がuf2のファームウェアしか書き込めないことに注意してください。ファームウェアが書き込まれると消えます
    * `RPI-RP2`をアンマウントした後、BOOTをショートする必要はなく、BOOTボタンを押し続ける必要もありません
        <ImageView image={require('@site/docs/General/get-id/img/2e8a.webp').default} size="80%" align="Center" /> 
    * 以下のコマンドでファームウェアをRP2040に書き込みます

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
