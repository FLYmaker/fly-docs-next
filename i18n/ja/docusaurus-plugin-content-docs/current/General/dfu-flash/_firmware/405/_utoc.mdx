以下是固件编译的说明：
* SSHに接続した後、以下のコマンドを入力し、Enterキーを押してください：

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* ここで` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のコンパイルデータとファームウェアを削除し、`make menuconfig`はファームウェアをコンパイルします。実行後、以下の画面が表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* メニューの**Micro-controller Architecture**に入り、**STMicroelectronics STM32**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

画像を更新中
* メニューの**Processor model**に入り、**STM32F405**を選択し、Enterキーを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.png').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**32KiB bootloader**を選びます。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.png').default} size="100%" align="Center" />

* **Communication interface**を選択し、**USB to CAN bus bridge (USB on PA11/PA12)**を選びます。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/utoc.png').default} size="100%" align="Center" />
