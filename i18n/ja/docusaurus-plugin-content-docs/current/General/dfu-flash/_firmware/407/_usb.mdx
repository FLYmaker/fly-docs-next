以下はファームウェアのコンパイル方法を紹介します：
* SSHに接続後、以下のコマンドを入力してエンターを押します：

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* ここで` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のコンパイルデータとファームウェアを削除し、`make menuconfig`はファームウェアをコンパイルするものです。実行後、下記のような画面が表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択してエンターを押します

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* メニューの**Micro-controller Architecture**に入り、**STMicroelectronics STM32**を選択してエンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* メニューの**Processor model**に入り、**STM32F407**を選択してエンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/407.webp').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**32KiB bootloader**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/32kb.webp').default} size="100%" align="Center" />

* **Communication interface**を選択し、**USB (on PA11/PA12)**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/usb.webp').default} size="100%" align="Center" />
