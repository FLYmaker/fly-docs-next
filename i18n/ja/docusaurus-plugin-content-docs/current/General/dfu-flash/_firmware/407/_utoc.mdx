以下はファームウェアのコンパイル方法です：
* SSHに接続後、以下のコマンドを入力してEnterキーを押します：

        ```bash
        cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
        ```

* `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は前のビルドデータとファームウェアを削除するためのもので、`make menuconfig`はファームウェアをビルドします。実行後、以下の画面が表示されるはずです。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択してEnterキーを押します。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* **Micro-controller Architecture**メニューに入り、**STMicroelectronics STM32**を選択してEnterキーを押します。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* **Processor model**メニューに入り、**STM32F407**を選択してEnterキーを押します。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/407.png').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**32KiB bootloader**を選択します。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/32kb.png').default} size="100%" align="Center" />

* **Communication interface**を選択し、**USB to CAN bus bridge (USB on PA11/PA12)**を選択します。

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/utoc.png').default} size="100%" align="Center" />
