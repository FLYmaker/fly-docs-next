1. インストールする書き込みツール

* FLY上位機はインストールする必要はありません
* インストールが必要な場合：上位機がインターネットに接続できる必要があります
* 既にインストールされている場合はインストールする必要はありません

```bash
sudo apt update
sudo apt install dfu-util -y
```

2. 下記のコマンドを実行して接続が成功したかどうかを確認してください。

```bash
lsusb
```

* DFUデバイスが表示されない場合は**書き込みモード**に再び入ってください
* システムバージョンの問題で他の名前が表示される場合がありますが、`0483:df11`が表示されていれば書き込みモードに入っています

<ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.webp').default} size="100%" align="left" />


3. プレコンパイル済みファームウェアの取得

* ここにはプレコンパイル済みのファームウェアが含まれています
* 注意：上位機がインターネットに接続できる必要があります
* 上位機で下記のコマンドを実行してファームウェアパッケージをダウンロードしてください

```bash
cd && git clone https://github.com/FengHua741/FLY_Katapult.git
```

* 既にファームウェアパッケージをダウンロードしている場合は、更新後に残りの操作を行ってください

```bash
cd ~/FLY_Katapult && git pull
```
