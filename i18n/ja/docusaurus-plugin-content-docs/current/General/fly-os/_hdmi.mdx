## FLY上位機の使用ガイド

:::tip
ブラウザでIP:9999を開いてFLY-Toolsにアクセスしてください
:::

:::tip
FLYの上位機に搭載されているHDMIインターフェースはMicro HDMIです。HDMIケーブルを購入する際は注意してください！！！
:::

:::tip
FLYの上位機はFPC一線通に対応しているものは`FLY-PI-V2`,`FLY-C8`と`FLY-PRO-X10`です
:::

:::tip
FPCをHDMIに変換するモジュールを使用すると、ラズベリーパイなどの上位機とも互換性があります
:::

    * HDMIを使用するには下のオプションを有効にしてください
    * FLY Toolsに入った後
