# TFカードにシステムを書き込み

### 準備作業

    * まずソフトウェア`Win32DiskImager`をインストールする
    * システムを解凍し、解凍後のファイル名の拡張子は`.img`
    * `balenaetcher`を使用してシステムを書き込むことは推奨されません
    * TFカードをカードリーダーに挿入し、カードリーダーをコンピュータのUSBポートに挿入
    * TFカードにデータがある場合はバックアップしてください。システム書き込み後にシステムカードを外すことは推奨されません。M2WEに書き込む必要がある場合を除きます

### メモリーカードのフォーマット

    * `TFFormatter.exe`ソフトウェアを使用してTFカードをフォーマット
    * フォーマットしたいディスクを選択

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * または、Windowsに標準で備わっているディスク管理でパーティションを削除し、削除後に新たにパーティションを作成します。
