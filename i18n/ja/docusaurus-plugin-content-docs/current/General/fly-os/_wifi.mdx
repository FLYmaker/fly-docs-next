:::danger 注意事項

    * FLY上位機はM2WEまたは2.4G WIFIモジュールを使用する必要があります。それ以外の場合はWiFiを使用してください。
    * 中国語の名前が付いたWIFIや、2.4Gと5GのデュアルバンドWIFIはサポートされていないことに注意してください。
    * mini padまたはlite2の場合は、この警告を無視してください。

    :::

## 上位機への接続

    * BOOTディスクでWIFIの設定が失敗した場合は、次の設定を参照してWIFIに接続してください。シリアルポートで上位機に接続する必要があります。
    * まず、シリアルポートで[SSHに接続](/docs/DebugDoc/BasicTutorial/index.mdx "クリックでジャンプ")してください。
    * 以下のコマンドを実行します。パスワードの入力を求められる場合、パスワード`mellow`を入力してEnterキーを押してください。パスワードは表示されません。

    ```cfg
    sudo nmtui
    ```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui1.webp').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

    * `Activate a connection`を選択してEnterキーを押します。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

    * 次の画面に進み、キーボードの`↑ ↓`キーで接続したいWiFiを選択し、Enterキーを押してパスワードを入力し、接続が成功するのを待ちます。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />


        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.webp').default} size="80%" align="left" />

    * 接続が成功すると、次のように表示されます。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

    * ESCキーを押して終了し、コマンドラインに戻ります。`ip a`を入力すると、図中の`192.168.2.126`が上位機のIPアドレスです。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui7.webp').default} size="80%" align="left" />
