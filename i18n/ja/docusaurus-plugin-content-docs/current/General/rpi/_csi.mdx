## CSIの使用方法

```
* イメージの書き込みが完了したら、/boot/config.txtへアクセスし`camera_auto_detect=1`と記載されたセクションを探し、それを`camera_auto_detect=0`へ変更した後カメラのモデルに応じて設定を追加してください。
※以下は参考構成です

```bash
dtoverlay=imx219,cam1
```
