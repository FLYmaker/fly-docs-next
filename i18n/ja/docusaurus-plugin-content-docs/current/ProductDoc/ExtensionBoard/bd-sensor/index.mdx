# BDsensor

    BDセンサーは、3Dプリンタと連動して使用できる距離センサーで、リアルタイムでベッドの高さ調整とZ軸の調整が可能です。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.png').default} size="50%" align="left" />

## 機能

    * Z軸のリミットスイッチの代わり
    * Z軸を上げ下げする必要なくスキャン可能で、印刷速度を向上
    * XY移動スキャンで密度の高いベッドプローブが可能



## 接続

    * 電源：5V
    * ファームウェア：Klipper/Marlin
    * 接続：ツールボード/メインボード（BDセンサーはサーボポートとリミットポートを使用）
    * ホットベッド：普通のPEI、アルミ板ホットベッドなどの金属表面ホットベッド、磁石穴あきホットベッドと非金属表面ホットベッドはサポートされません


## 注意事項

    * Klipperはklipperと最新のBD-SENSORパッチをダウンロードし、メインボードのファームウェアを再書き込みする必要があります
    * デバッグ時は、PEIを傷つけないように緊急停止ボタンを手元に置いてください
    * ベッドスキャンの速度は200を推奨！
    * Z軸の原点復帰速度は5を超えないことを推奨します
