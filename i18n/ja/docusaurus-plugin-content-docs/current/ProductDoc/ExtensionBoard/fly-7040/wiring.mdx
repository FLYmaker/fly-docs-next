## 参照設定

    ```cfg
    [fan]
    ##    印刷冷却ファン 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## 起動時に自動回転する問題の解決方法

    :::info
    * ここでは簡単に説明します
    * ファームウェアをコンパイルする際にPWM信号ピンを追加し、その状態を反転させる
    * テスト用にSUPER8PROを使用、ピンはPF8を使用、ファームウェアコンパイル時に`GPIO pins to set at micro-controller startup`に`!PF8`を追加
    * 注意：`!`は英語の感嘆符を使用してください
    * 他のメインボードも同様です
    :::
