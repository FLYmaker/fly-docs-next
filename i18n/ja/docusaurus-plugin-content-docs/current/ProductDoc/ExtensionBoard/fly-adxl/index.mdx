# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.png').default} size="100%" align="left" />



## 製品特徴

    * 36電機などの場所に適用
    * Raspberry Pi、FLY PI、Orange Pi、主板などのSPIインターフェースに接続可能

## 製品仕様

    * 搭載インターフェース：ZH1.5 6Pプラグ
    * 通信方式：SPI
    * 動作電圧：5V

## 注意事項

    :::warning 注意事項

    * 動作電圧は5Vです、間違えて接続しないでください
    * 各種ファームウェアに対応していますが、SPI接続が可能かどうか確認してください

    :::
