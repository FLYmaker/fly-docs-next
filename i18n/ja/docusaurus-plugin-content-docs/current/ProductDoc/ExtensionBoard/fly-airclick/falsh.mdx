# ファームウェア更新と書き込み

    :::danger 注意事項

    * 出荷時に既にファームウェアがプリインストールされています

    :::

## ファームウェアのダウンロード

    * ファームウェアの説明
    * `wbtn-vX.X.X-X-XXXXXX-full.bin`は完全なファームウェアで、0x0アドレスから書き込む必要があり、書き込み後にすべての設定が保持されません
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin`は差分ファームウェアで、0x10000アドレスから書き込む必要があり、既存の設定に影響を与えずにファームウェアのみを更新します
    * `wbtn-vX.X.X-X-XXXXXX-www.bin`はUIファームウェアで、0x290000アドレスから書き込む必要があり、既存のファームウェアや設定に影響を与えずにウェブフロントエンドUIのみを更新します
    * <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/airclick.zip">ファームウェアパッケージのダウンロード</Button>

## ファームウェア更新
    * `AirClick`を電源オフにして、BOOTに従って`AirClick`のESP32書き込みポートをコンピューターに接続する必要があります
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. USB書き込みツールのダウンロードと解凍 <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">ダウンロード</Button>
    2. `flash_download_tool_3.9.4.exe`を開きます。注意：書き込みツールを開くとまずCMDウィンドウが表示され、その後書き込みツールのインターフェースが表示されます。CMDウィンドウを絶対に閉じてはいけません
    3. 書き込みツールで``ChipType:``に``ESP32-C3``を選択
    4. 書き込みツールで``WorkMode:``に``Develop``を選択
    5. 書き込みツールで``LoadMode:``に``USB``を選択
    6. 最後に``OK``をクリックし、下の図を参照してください
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. クリック後、下の画像が表示されます
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * 上記の画像の緑色の枠で囲まれた部分は画像と一致していることを確認してください、自主的に変更しないでください

    :::

    8. AirClickをUSB-Typecケーブルでコンピューターに接続
    9. 上記の画像の青い領域1ボタンをクリックして、書き込むファームウェアを選択
    10. 上記の画像の青い領域2に書き込みアドレスを入力
    11. 上記の画像の青い領域3でAirClickのシリアルポートを選択（表示されない場合は、AirClickの書き込みボタンを押し続けてからコンピューターに接続してください）
    12. 上記の画像の青い領域4で最大`921600`のボーレートを選択
    13. 上記の画像の青い領域5をクリックして書き込みを開始し、書き込み完了を待ってからAirClickのファームウェア更新が完了します（書き込み完了の表示が出るまで、接続を切断しないでください）
