# AirClick ワイヤレスボタン
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />



## 製品の特徴

    * 高度な互換性：Klipper機器用に設計されており、直接接続してプリセットされた指令を送信することができ、シームレスな互換性と操作体験を提供します。
    * カスタムマクロ指令：ユーザーは自身の需要に応じて指令をカスタマイズし、機器の操作をより柔軟で便利にすることができます。
    * 無線接続と有線接続の両方をKlipperにサポート
    * 無線接続：ESP32の設計により、直接Klipperにカスタム指令やマクロを送信し、ユーザーに高い柔軟性と拡張性を提供します。
    * 有線接続：STM32F072チップを使用して、ネットワーク接続がKlipperに接続できない場合でも、簡単な操作が可能です。合計で10個のカスタム指令またはマクロをサポート。
    * 使いやすさ：直感的なユーザーインターフェースにより、ユーザーは簡単にワイヤレスボタンを追加および管理でき、操作の利便性が大幅に向上します。

## 製品仕様

    * メインコントロールチップ：ESP32
    * 追加チップ：STM32F072
    * ESP32-WIFIを通じた指令送信：制限なし
    * STM32F072-IOを通じた指令送信:10個
    * 互換性：Klipper機器用に設計
    * 電源：5V

## 注意事項

    :::danger 注意事項

    * 本製品は5V電源で供給され、`type c`または`PH2.0-2P`端子を通じて電源供給が可能です
    * 本製品に搭載のアンテナはボタン信号の受信にのみ使用されます。WIFIに接続する必要がある場合は、アンテナを適切に取り付ける必要があります。アンテナ端子は`ipex3`です。
    * アンテナを適切に取り付けてください！！！そうしないと、信号が弱すぎて過熱が深刻になる可能性があります！！！
    * 長押しボタンを15秒以上押した場合、誤操作を防ぐため、システムが自動的にそれをシャットダウンします。

    :::
