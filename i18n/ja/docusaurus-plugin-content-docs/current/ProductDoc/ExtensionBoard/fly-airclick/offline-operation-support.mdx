# 有線接続の設定

:::danger 注意事項

* STM32F072にファームウェアを書き込んでKlipperに接続する必要があります。そうしないと有線接続操作がサポートされません。

:::

## コンパイルの開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押して **Save configuration** が表示されたら、`Y` キーを押してください。
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインインターフェースに戻ります。
    * `make -j4` を入力してコンパイルを開始します。時間がかかります。
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が表示されたらコンパイル成功です。

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 書き込みモードに入る

    * AirClickの電源を切った状態で、STM32のBOOTを押しながら、STM32の書き込みポートをType-Cケーブルで上位機に接続します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.webp').default} size="100%" align="center" />
