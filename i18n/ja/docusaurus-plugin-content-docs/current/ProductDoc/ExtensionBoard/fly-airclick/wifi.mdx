# Klipperへの接続

    :::info 重要な注意事項

    * 本製品に搭載されているアンテナは受信専用です。送信アンテナは自分で取り付ける必要があり、アンテナ端子は`ipex3`です。
    * アンテナをきちんと取り付けてください！！！さもないと信号が弱くて発熱がひどくなります！！！
    * 初回使用時は手動でAirClickのホットスポットに接続し、対応するWIFIに接続してください。

    :::


    :::info 重要な注意事項

    * AirClickはMoonrakerに接続した後、Klipper制御機に指令を送ります。
    * カスタムマクロ、緊急停止、Klipperの再起動、ファームウェアの再起動、システムのシャットダウンなどの機能を実現できます。

    :::

## AirClickのWIFI接続

    * AirClickのホットスポットに接続します。ホットスポット名は`FLY-WBTN-XXXXXX`です。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.png').default} size="50%" align="Center" />
    * ホットスポットに接続した後、ブラウザで[192.168.4.1](http://192.168.4.1/#/setting)にアクセスします。
    * WIFI設定画面が表示され、プリンターと同じルーターに手動で接続する必要があります。パスワードを入力して、下の青いボックスをクリックしてWIFIに接続します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.png').default} size="50%" align="Center" />
    * WiFiに接続すると、AirClickの接続後のIPアドレスが表示されます。下の青いボックスをクリックしてデバイスを再起動します。
        * 番号1で囲まれた部分がAirClickのIPアドレスです。
        * 番号2で囲まれた部分をコピーし、デバイスを再起動します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.png').default} size="80%" align="Center" />
    * 注意：対応するルーターに現在のネットワークを手動で切り替える必要があります。さもないとアクセスできません。

## 言語の変更
    * 右上の囲まれたアイコンをクリックして、必要な言語を選択します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-2.png').default} size="80%" align="Center" />

## Klipperへの接続

    :::info 重要な注意事項
    * AirClickはMoonrakerに接続した後、Klipper制御機に指令を送ります。
    * カスタムマクロ、緊急停止、Klipperの再起動、ファームウェアの再起動、システムのシャットダウンなどの機能を実現できます。
    :::

    * 左側のシステム設定をクリックします。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.png').default} size="80%" align="Center" />
    * 下の`Moonraker IP`を探し、KlipperマシンのIPアドレスを入力します。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-1.png').default} size="80%" align="Center" />
        * 番号1でKlipperホストのIPアドレスを入力します。
        * 番号2はデフォルトで`7125`です。ホストが他のポートを使用している場合は、対応するポートを入力してください。
        * 番号3で設定を保存します。
    * 正しく接続されると、ウェブページの上部にメッセージが表示されます。
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.png').default} size="80%" align="Center" />
