# 紹介

# LLLバッファーの使用

 	LLL消耗品バッファーは、3Dプリンターのリモートフィーディング用に設計されたデバイスで、消耗品の装填と取り外し操作を簡素化することを目的としています。このバッファーには、フィーディングとリトラクトのためにそれぞれ使用する2つのボタンが装備されており、ユーザーが電動でフィーディングとリトラクトを行うことができます。

### バッファー機能：

 	3Dプリンターのエクストルーダーがフィーディングを行う際、遠端の消耗品ラックが引っ張られると、バッファーはこの動作を検知し、自動的にノズル方向に一定量の消耗品を送り出します。引っ張りが発生するたびに、バッファーはこのフィード操作を繰り返します。これにより、エクストルーダーによる消耗品の引っ張りを効果的に減らし、エクストルーダーの推力の安定性を保つことができ、印刷品質の安定性を向上させます。

### 消耗品切れ検知機能：

 	バッファーには消耗品切れを検知するセンサーが内蔵されており、バッファー内に消耗品がないことを検知すると、3Dプリンターに印刷を一時停止する信号を送ります。この機能を使用するには、信号ピンを接続し、3Dプリンターの設定を変更する必要があります。一方、バッファー機能は設定を変更することなく直接使用できます。

 	これらの機能設計により、LLL消耗品バッファーは消耗品の操作を簡素化するだけでなく、3Dプリントプロセスの安定性と印刷品質を向上させます。



* 使用する際は消耗品を接続してから機能を使用してください。そうでない場合は、ボタンでモーターの動作を制御することができます。

## 設定参考

    ```cfg
    [filament_switch_sensor 消耗品切れ検知]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 消耗品切れ
    insert_gcode:
            M118 消耗品挿入
    ```
