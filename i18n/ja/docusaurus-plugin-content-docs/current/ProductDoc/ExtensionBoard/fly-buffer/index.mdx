# 紹介

# LLLバッファーの使用方法

    LLL消耗品バッファーは、3Dプリンターのリモートフィーディング設計に特化したデバイスで、消耗品の装填と取り外し操作を簡素化するために設計されています。このバッファーには、フィードインとフィードアウトのためにそれぞれ使用される2つのボタンが装備されており、ユーザーは電動でフィードインおよびフィードアウト操作を行うことができます。

### バッファー機能：

    3Dプリンターの押出機がフィーディングするとき、リモートの消耗品ラックに引っ張られると、バッファーはその動作を検知し、自動的にホットエンドの方向に消耗品を送り出します。引っ張りが発生するたびに、バッファーはこの送り操作を繰り返します。これにより、押出機が消耗品を引っ張る力が効果的に減少し、押出機の推力が安定して保たれ、印刷品質の安定性が向上します。

### 消耗品切れ検知機能：

    バッファーには消耗品切れ検知器が内蔵されており、消耗品がバッファー内にないことを検知すると、3Dプリンターにプリントを一時停止するシグナルを送信します。この機能を使用するには、シグナルピンを接続し、3Dプリンターの設定を変更する必要があります。一方、バッファー機能は設定変更なしで直接使用できます。

    これらの機能設計により、LLL消耗品バッファーは消耗品操作を簡素化するだけでなく、3Dプリントプロセスの安定性とプリント品質も向上させます。



* 使用する際には、消耗品を接続した後に機能を使用してください。そうでない場合、ボタンでモーターを制御するだけになります。

## 設定の参考

    ```cfg
    [filament_switch_sensor 断料检测]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 缺料
    insert_gcode:
            M118 上料
    ```
