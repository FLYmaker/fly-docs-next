# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.webp').default} size="40%" align="left" />


## 製品特點

    * CANおよび電源の1分4をサポートし、一度に4つのCANツールボードを拡張可能
    * 搭載のRJ11インターフェースは、CANプロトコルを使用するシステムとの接続と通信を目的として設計されています。RJ11はCANシステムにおいて標準インターフェースではありませんが、当社の製品はこれらをスムーズに接続する可能性を提供します。
    * 人に優しい設計により、プリント部品で直接レールに取り付け可能

## 製品仕様

    * 通信インターフェース：RJ11*1、XH2.54*5
    * 電源インターフェース：KF128-5.028-2P*5
    * 入力電圧：5V-24V
    * 互換性：RRF/Klipper/等CAN

## 注意事項

    :::warning 注意事項

    * 電源の入力はそのまま出力されることに注意してください。
    * 線順を間違えないように注意してください。
    * 拡張ボードをインストールする前に、上位機の電源を切ってください。
    * 湿気やほこりの多い環境での使用は避けてください。
    * 機器の取り付け時には、ツールボードの電圧仕様に注意し、損傷を避けてください。

    :::
