# FLY-G2T

    FLY-G2T多機能拡張ボードは、ラズベリーパイとFLY上位機の機能を拡張するために設計された高性能モジュールです。UART通信をサポートし、FPCインターフェースを統合しており、TFT-V2を外部接続できます。また、5Vファンの制御もサポートしています。これにより、工業自動化、3Dプリンタ、ホームオートメーション、ロボット制御など、人間とマシンのインターフェースや外部通信が必要な様々なアプリケーションでラズベリーパイのインタラクティブ性と可用性を強化する理想的なソリューションとなっています。

## 製品の特徴

    * 互換性：ラズベリーパイの複数のバージョンをサポートし、Zero/3B/3B+/4Bなどに対応。
    * UARTからRS232への変換：安定したシリアル通信を提供し、RS232インターフェースを介して他のハードウェアデバイスとデータ交換を行うことができます。
    * SPI容量タッチスクリーン：FPCインターフェースを介して接続されるTFT-V2容量スクリーンは、直感的なタッチ操作を提供し、グラフィックユーザーインターフェースが必要なアプリケーションに適しています。例：klipperscreenなど。
    * 5Vファンサポート：5Vファンインターフェースを統合し、ファンを接続して冷却を行い、上位機が長時間動作または高負荷状態でも安定性を保証します。
    * プラグアンドプレイ設計：拡張ボードはインストールが容易で、上位機に直接挿抜可能。追加の配線や複雑な設定は不要です。
    * コンパクトサイズ：拡張ボードのサイズは適切で、過度なスペースを占有せず、様々なプロジェクトに統合しやすい。

## 製品仕様

    * 通信インターフェース：UARTからRS232への変換
    * 表示インターフェース：SPIからFPCインターフェースへの変換
    * ファン電源：5V
    * 互換性：ラズベリーパイZero/3B/3B+/4B、FLY Pi V2など

## 注意事項

    :::warning 注意事項

    * RS232接続ツールボードを使用する際、上位機と`共地`にしてください！！！！！

    :::

    * 拡張ボードをインストールする前に、上位機の電源がオフであることを確認してください。
    * 湿気や埃が多い環境での拡張ボードの使用を避けてください。
    * 画面をインストールする際は、通電しないで抜き差しし、画面ケーブルの方向に注意してください。
    * ファンをインストールする際は、ファンの電流と電圧仕様に注意してください。
