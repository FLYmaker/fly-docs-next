# FLY_halo

## 1. ホームページ

    ヒートベッドまたはホットエンドをクリックし、画面上部から時計回りにスライドして目標温度に達したら、再度ヒートベッドまたはホットエンドをクリックして温度を設定します。

    MCU温度とホスト温度を表示する必要がある場合は、以下の設定を追加します。

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # MCUに関連付け（デフォルト）

    [temperature_sensor host]        # ホスト温度
    sensor_type: temperature_host     # ホストに関連付け
    ```


    :::warning 注意事項

    * 最高温度は設定ファイルの値を読み取ってください。

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.png').default} size="50%" align="left" />


## 2. 押出し

    * アイコンを長押ししてホットエンド温度、押出し長さ、押出し速度を設定できます。

    :::warning 注意事項

    * haloは単一のホットエンドの制御のみをサポートします。

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/Extrusion.png').default} size="50%" align="left" />

## 3. システムサービス

サービスをクリックすると、停止、再起動、起動ができます。

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/system-services.png').default} size="50%" align="left" />
