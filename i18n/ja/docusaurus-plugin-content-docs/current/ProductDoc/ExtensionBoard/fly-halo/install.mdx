## インストールガイド

## Voron Stealthburnerへのインストール

    * HaloのStealthburnerへのインストールには、私たちが提供するアダプターを使用する必要があります
    * まず私たちが提供する2つのSTLをダウンロードして自分でプリントしてください
        * [Stealthburner_main_body(Adapt_to_Halo).stl(ダウンロード)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(ダウンロード)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(ダウンロード)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### アダプターボードと接続ケーブルのインストール
    * 出線位置はあなたの好みに応じて配置できます。ファンの背面にぴったりつける必要はありません
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.png').default} size="50%" align="left" />


    :::warning 注意点

    * 出線色の説明**黒は負極GND**、**赤は正極VCCでDC12-24Vの電圧を受け入れます**、**白はGPIO47で詰まり検出に使用できます**、**黄色はGPIO48で印刷終了後の停止に使用できます**

    :::
