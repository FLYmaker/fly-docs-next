# Haloページ操作の紹介

## ページ構造

```bash
  ファイル印刷ページ   首页           押出ページ            システムサービス       設定               デバイス情報

|             |    |             |    |             |    |             |    |             |    |             |
| Print Files |<-->|     Home    |<-->|   Extrude   |<-->| Sys Service |<-->|   Setting   |<-->|  Dev Info   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blocking Det|
                   |             |
                   +-------------+
                     詰まり検出ページ
            (設定で詰まり検出を有効にするとこのページが表示されます)
```

## 押出ページ

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.webp').default} size="50%" align="left" />
    * 上図1の操作可能な領域と下記の番号に対応しています
    1. クリック：押出ヘッドの目標温度を1度上げる
    2. 長押し：押出温度設定に進む。環状バーをドラッグして温度を調整し、再度クリックで設定温度を確認
    3. クリック：押出ヘッドの目標温度を1度下げる
    4. クリック：押出
    5. 上下スクロール：押出または回収の長さを選択可能。単位はmm
    6. クリック：回収
    7. 長押し：上図2の押出速度設定に進む。上下スクロールで押出速度を選択し、単位はmm/s。選択後、周囲の空白領域をクリックして退出
