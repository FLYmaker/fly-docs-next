# 使用教程

## 注意事项

    * 初回使用時には必ずネットワークに接続してください。WIFIがインターネットに接続できない場合、haloは初回起動の設定を取得できません。
    * 設定ページの全ての設定は**保存設定**をクリックし、デバイスを再起動後、効果が発揮されます。
    * 堵料検知機能はデフォルトで無効になっており、設定で有効にし、再起動後にホーム画面からスクロールアップしてアクセスできます。
    * 印刷終了時に電源を切る機能はデフォルトで無効ですが、設定で有効にすることができます。
    * 全てのポップアップウィンドウで、印刷をキャンセルまたは一時停止するボタンは長押しが必要です。単押しでは無効です（誤操作を防ぐため）。
    * システムサービスページのHostサービスは上位機ホストです。
    * 公式ファームウェアを使用する際は、メイン制御チップやFLASHチップを変更しないでください。

    :::tip

    * 注意：`moonraker.conf`設定ファイルのサイズが4MBを超えると正常に使用できない可能性があります。異常が発生した場合は`moonraker.conf`のサイズを縮小してください。

    :::

## システムファームウェアの更新

    * haloバージョンのアップデートは、全量更新とOTA増量更新の2種類があります。
        * 全量更新：底層の大きなアップグレードが必要な場合、USBを使って全量ファームウェアパッケージを書き込みます。
        * 増量更新：通常のOTAアップグレード、デバイス情報ページで更新を確認できます。

## 携帯電話でスキャンしてWIFIを設定

    :::tip

    * 一部のモバイルデバイスでは、WiFiに接続するためのスキャン機能が存在しない場合があります。その場合、携帯電話のWIFI設定で**FLY-halo-xxxxxxxx**というホットスポットに接続してください。

    :::

    :::tip

    * ホットスポットに接続後、ウェブページが自動的にポップアップしない場合は、手動でブラウザを開いてリンクにアクセスしてください。

    :::

    :::tip

    * 注意：HALOは2.4GWiFiのみ使用可能です。

    :::

    :::tip

    * Haloの画面では隠しWIFIは検索できません。

    :::
