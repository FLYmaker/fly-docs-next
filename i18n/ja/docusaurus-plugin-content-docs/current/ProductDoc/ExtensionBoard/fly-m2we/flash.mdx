## システム焼き込みの準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上または128GB以下で、TFカードの転送速度はclass10以上である必要があります
    * 対応する<Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>をダウンロード
    * 必要な<Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼き込みツール</Button>をダウンロード

## システム焼き込み

:::info
M2WEは3つのモードをサポートしています。私はTFカードをM2WEに転送する方法とM2SD Burnerを使用してM2WEに焼き込む方法をお勧めします。
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムを焼き込むことをお勧めします。USB経由でM2Wに焼き込むと、ドライバーの互換性などの理由で失敗する可能性があります。直接[メモリーカードをM2WEに焼き込む](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)ことをお勧めします
* ダウンロードしたシステムイメージが対応しているかどうか注意してください。対応していない場合は起動できません
:::
