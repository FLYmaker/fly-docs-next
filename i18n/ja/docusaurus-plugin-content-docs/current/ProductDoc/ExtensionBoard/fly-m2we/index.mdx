# FLY-M2WE

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-m2we/img/m2we.png').default} size="40%" align="Center" />

## 注意事項

:::danger 注意事項

* M2WEは出荷時にFLYOSがプリインストールされており、直接起動可能です
* M2WEはFLYの製品でのみ使用可能。他のデバイスには対応していません
* M2WEのアンテナ端子は`IPEX1`です
* M2WEは2.4Gと5Gのデュアルバンド、中国語、特殊文字、デフォルトで非表示のWIFIに対応していません

:::
