# FLY VoltUp Power

    FLY-VoltUp Powerは高圧駆動用に設計された昇圧モジュールで、ユーザーは高圧電源を別途購入することなく高圧駆動を実現でき、3Dプリンターの設計時にスペースを節約するのに役立ちます。

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-power/img/power.webp').default} size="50%" align="left" />


## 製品特性

    * 出力電源48V/60V選択可能
    * ファン端子搭載、4010ファン固定用、入力電圧に応じて同じ電圧を出力
    * 二重電源インターフェース拡張：一つの電源インターフェースを提供し、ユーザーが同時に二つの電源出力を拡張できるようにし、48Vまたは60Vを個別に設定可能
    * 簡単な設置：標準サイズ設計で、さまざまな環境に適し、追加のドライバのインストールは不要。
    * コンパクトなサイズ：拡張ボードのサイズが適切で、過度なスペースを占有せず、さまざまなプロジェクトに統合しやすい。

## 製品仕様：

    * 電源インターフェース：12V/24V
    * 出力インターフェース：2路選択可能な電源 48V/60V  （ジャンパーピンの選択により）
    * ファンインターフェース：XH2.54 2P  VCC FAN 2つ


## 注意事項：

    * 設置前に電源を切断し、全ての機器をオフにしてください
    * 48V出力時、モーター電流は最大2Aまで設定可能で、60V出力時は最大1Aまで設定可能
    * 2つのVCC FANインターフェースの電圧は電源入力と一致し、出力電圧も電源電圧と同じ
    * モジュールには4010ファンの固定穴が予め用意されています。昇圧モジュールが発生する熱が大きいため、ファンを装備して放熱を行う必要があります。
    * 逆接続防止ガードチップ内蔵、顧客の操作ミスによる部品の破損を防ぐ。
    * 電圧選択ジャンパーピンを同時に挿入しないでください。部品の破損を避けるため。
    * 正極と負極を正しく接続し、部品の破損を避けてください。
