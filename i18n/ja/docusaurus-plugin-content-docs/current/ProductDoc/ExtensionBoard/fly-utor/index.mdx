# FLY-UTOR

    FLY-UTORは、高性能で多機能のインターフェース拡張ソリューションであり、ユーザーに追加のUSBポートと電源拡張機能を提供することを目的としています。また、UARTからRS232への通信をサポートします。この拡張ボードはコンパクトな設計で、設置が簡単で、追加のインターフェースと電源出力が必要なさまざまなアプリケーションに適しています。

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.png').default} size="100%" align="left" />


## 製品の特徴

    * 二つのUSBポート拡張：拡張ボードは二つの追加USBポートを提供し、USB 2.0規格をサポートし、さまざまなUSBデバイスとの互換性と高速データ転送を保証します。
    * UARTからRS232へのサポート：内蔵のUARTからRS232変換器が、ユーザーがシリアルデバイスをシステムに接続し、デバイスの通信能力を拡張することを便利にします。
    * 二つの電源ポート拡張：二つの電源ポートを提供し、ユーザーが同時に二つの外部電源を拡張することができます。
    * 安定した電源設計：拡張ボードは安定した電源回路を設計しており、接続されたデバイスの安定した動作を保証します。
    * 設置が簡単：標準サイズの設計を採用し、さまざまな環境に適しており、追加のドライバーのインストールは不要です。
    * コンパクトなサイズ：拡張ボードのサイズは合理的に設計されており、スペースを多く占有せず、さまざまなプロジェクトに組み込みやすいです。

## 製品仕様：

    * 使用可能なインターフェース：USB X2、UARTからRS232 X2、電源ポート X2
    * 電源ポート：12V/24拡張
    * 互換性：USBインターフェースを持つ上位機

## 注意事項：

    * インストール前に電源を切断し、全てのデバイスをオフにしてください。
    * マルチファンクションUSB拡張ボードをメインボードのUSBポートまたは他のUSB拡張インターフェースに挿入します。
    * 必要なUSBデバイスおよび/または電源を拡張ボードに接続します。
    * UARTからRS232機能を使用する場合は、対応するシリアルデバイスを接続します。
    * 電源を再接続し、デバイスを起動します。
    * システムは自動的に拡張ボードを検出し、設定を行うはずです。必要に応じて、操作後の設定注意事項を参照してください。
