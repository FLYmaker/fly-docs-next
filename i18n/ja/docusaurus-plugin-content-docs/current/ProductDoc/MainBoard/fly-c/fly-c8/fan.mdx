# 5Vファンの搭載

## C8は、メインコントローラーの冷却用に5Vの制御可能なインターフェースを搭載しています。接続方法は以下の図の通りです。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/corefan.webp').default} size="80%" align="center" />

## 参照設定

    ```cfg
    [mcu host]                      # FlyOSはLinux MCUをあなたのためにデフォルトで生成します。定義するだけで使用できます           
    serial: /tmp/klipper_host_mcu 

    [temperature_sensor FLY-π]      # ホストマシンの温度
    sensor_type: temperature_host


    [temperature_fan core_fan]      # ホストマシンの冷却ファン
    pin: host:gpiochip1/gpio103     # ホストマシンのファンピン
    max_power: 1.0
    sensor_type: temperature_host   # ホストマシンのコントローラー温度に設定
    control:watermark               # 制御方式
    target_temp: 48                 # ホストマシンの冷却ファン起動温度
    min_temp: 0                     # 最低温度、この温度以下でエラーが発生します
    max_temp: 90                    # 最高温度、この温度以上でエラーが発生します
    off_below: 0.10
    kick_start_time: 0.50
    max_speed: 0.8                  # 最大回転速度、フルパワー時の80%
    min_speed: 0.3                  # 最小回転速度、フルパワー時の30%
    ```
