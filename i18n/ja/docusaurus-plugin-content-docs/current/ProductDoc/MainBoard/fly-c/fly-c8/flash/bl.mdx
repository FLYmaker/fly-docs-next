# BLファームウェア

    :::tip

    * 出荷時に`ブートローダー`が既にインストールされているので、再インストールする必要はありません。

    :::

## 書き込みモードに入る


    * 書き込みモードに入る方法は2つあります
    * 1つ目：主板完全に電源を切った状態で、BOOTを長押ししながら、データ転送機能付きのケーブルの一方を主板に、もう一方をあなたのデバイスに接続します。WINDOWSに適しています。
    * 2つ目：データ転送機能付きのケーブルの一方を主板に、もう一方をあなたのデバイスに接続した後、BOOTを長押ししながらコマンドを入力します。上位機に適しています。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="100%" align="center" />

## ファームウェアの書き込み
    <Tabs>
        <TabItem value="win" label="Windows書き込みチュートリアル" default>

            * C8のブートローダーファームウェア <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/BL/GeminiV3.bin">ファームウェアを自動ダウンロード</Button>


            * ディップスイッチ`3`,`4`を右側に切り替え、下位機と上位機を内蔵USBで接続します。
            * ディップスイッチ`1`,`2`を左側に切り替えます。そうしないと下位機と上位機の接続に影響します。
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.png').default} size="100%" align="center" />
            <WIN />

        </TabItem>
        <TabItem value="linux" label="ホスト書き込みチュートリアル">

            * ディップスイッチ`1`,`2`を右側に切り替え、下位機と上位機を内蔵USBで接続します。
            * ディップスイッチ`3`,`4`を左側に切り替えます。そうしないと下位機と上位機の接続に影響します。
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.png').default} size="100%" align="center" />
            * この方法でディップスイッチを切り替えた後、下のUSBポートが下位機によって占有されるので、他のデバイスを接続しないでください。
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="50%" align="center" />

            * 上位機が正常にインターネットに接続できることを確認してください


                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
            * 下記のメッセージが表示されたら、下位機のBOOTボタンを長押ししてからエンターキーを押してください。

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="80%" align="center" />

            * 下記の画像が表示されたら、書き込みが成功し、BOOTボタンを放すことができます。
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/hid3.png').default} size="80%" align="center" />


        </TabItem>
