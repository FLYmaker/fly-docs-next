# ディップスイッチの紹介

:::tip
FLY Geminiはディップスイッチを介してホストマシンに接続します。ディップスイッチの位置が正しくない場合、正常に使用できません。
:::

:::danger 注意事項
ディップスイッチが指定された位置に設定されていないと、ホストマシンに正常に接続できないことに注意してください。
:::

## オンボードホストマシンへの接続

    * ディップスイッチ`1`,`2`を右側に切り替えて、下位機器を内蔵USBを介してホストマシンに接続します。
    * ディップスイッチ`3`,`4`を左側に切り替えます。そうしないと、下位機器とホストマシンの接続に影響を与えます。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.png').default} size="100%" align="center" />
    * この方法でディップスイッチを切り替えた後、下部のUSBポートは下位機器によって占有されます。ほかのデバイスを接続しないでください。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="100%" align="center" />


## 外部ホストマシンへの接続

    * ディップスイッチ`3`,`4`を右側に切り替えて、下位機器を内蔵USBを介してホストマシンに接続します。
    * ディップスイッチ`1`,`2`を左側に切り替えます。そうしないと、下位機器とホストマシンの接続に影響を与えます。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.png').default} size="100%" align="center" />
