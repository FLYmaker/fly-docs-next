# ファームウェアの書き込み説明

    
## どのファームウェアが必要かを判断する方法

    :::tip

    * 注意：あなたはどれか一つのファームウェアを書き込むだけでよいです

    :::
    
    * C8は以下の方法で上位機に接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * メリット：通信が信頼性が高く、安定している
        * デメリット：複数のUSBデバイスがあるとデータ伝送に影響を与える可能性がある
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * メリット：複数のデバイスを接続でき、USBポートを占有しない
        * デメリット：`KlipperScreen`と一緒に使用すると、ホーミングタイムアウトなどのエラーが発生する可能性がある。
