## システム書き込みの準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上、128GB以下で、転送速度がclass10以上である必要があります
    * 対応する<Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>をダウンロード
    * 必要な<Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button>をダウンロード

## システムの書き込み

:::info
TFカードへの書き込みとTFカードを介してM2WEへの書き込みはどちらか一つを選択してください
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEにはTFカードを使用してシステムをフラッシュすることをお勧めします。USBを介してM2Wに書き込むと、ドライバーの互換性などの理由で書き込みが失敗することがあります。直接[メモリカードからM2WEに書き込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することをお勧めします
* ダウンロードしたシステムイメージが対応しているかどうかを確認してください。対応していない場合は起動できません
:::
