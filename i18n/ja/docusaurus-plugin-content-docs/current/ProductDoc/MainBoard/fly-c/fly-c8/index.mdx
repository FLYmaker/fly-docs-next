## 製品の特徴

    * 32ビット、168MHzのARM Cortex-M0+シリーズSTM32F407VGT6マイクロコントローラを採用
    * 搭載TMCドライバのSPIおよびUART動作モード
    * 搭載DIAG機能ピン、シンプルなジャンパーの抜き差しだけで無限位帰位が可能
    * 数値制御ファン 24V、12V、5V電圧選択可能、降圧モジュールの購入の必要なし。
    * 新しいFlash書き込み方式、FLY-Toolsを直接使用してメインボードのファームウェアを更新可能
    * 多路電源管理、ポート、リミット、ドライバなどのショートでホストPCの動作に影響を与えない
    * RGBインターフェース予約（12V、5V）電圧選択可能
    * SPIインターフェース予約、加速度センサーを外部接続して加速度補正が可能
    * TFTスクリーンとHDMIスクリーンの一線通サポート
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグイン numpyとFLY-Toolsがプリインストールされており、複雑な操作やソースの変更が不要で初心者でも簡単に始められる（TFカードまたはM2WEの購入が必要）

## 製品仕様

### ホストPC
    * CPU：高性能全志H5チップ、4コア64ビットCortex-A53
    * GPU：高性能6コアMali 450、ピクセルフィルレート2.7gpixel/s以上
    * RAM：1GB DDR3（GPUと共有）
    * ROM：最大128GB SDCARD対応
    * 周辺機器：Spi x1、TFT X1、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN x1、3.5mmオーディオインターフェース X1
    * インターフェース：搭載M.2インターフェース、WIFI、eMMC等の拡張可能（非FLY π専用のM.2デバイスを接続しないでください）
    * 予約されたZH1.5制御可能5Vファンインターフェース、ホストPCの冷却に使用
    * TFTスクリーンとHDMIスクリーンのFPC一線通対応
    * CANブリッジインターフェース予約
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグイン numpyとFLY-Toolsがプリインストールされており、複雑な操作やソースの変更が不要で初心者でも簡単に始められる（TFカードまたはM2WEの購入が必要）

### 下位機

    * MCU: STM32F407VGT6
    * ファームウェア:  KLIPPER/RRF/Marlin
    * 12-24V直流電源対応、供給が安定、降圧モジュール不要
    * ヒートベッドポート最大出力電流:  15A
    * ヒーターロッドポート最大出力電流:  6A
    * ファンポート最大出力電流:  1A
    * ドライバ動作モード対応：SPI、UART、STEP/DIR
    * サポートドライバ：TMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988等外部対応
    * 拡張インターフェース：BLTouch（Servos、Probe）、RGB X2、UART X1、12864 X1
    * ファン: 二線式数値制御ファン X6 (24V、12V、5V)選択可能
    * 温度センサー：3路 NTC 100K
    * サポート機械構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
