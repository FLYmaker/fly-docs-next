# 上位機にSSH接続する


* シリアルポートまたはWIFIを使用してデバイスに接続できます
* **MobaXterm_Personal**または他の**SSH端末ツール**を用意してください
* SDカードまたはM2WEに**対応する上位機のシステム**を焼き込んでください
* **上位機が正常に起動しない場合は、ドライバ、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後に、ファームウェアのビルドが便利になるようにネットワークを介してSSHに接続してください！！**
* **ファームウェアをビルドする必要がある場合は、WIFIまたはLANケーブルを使用して上位機に接続するチュートリアルを参照してください**

        ## ネットワークIPを使用して上位機にSSH接続する

        * 上位機が接続されている**ルーターのバックエンドにログインしてください**
        * システムが**正常に起動していることを確認してください**
        * 上位機の**ユーザー名とパスワードを知っていることを確認してください**。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * マザーボードが起動し、ネットワークに接続されている場合、IPを使用してSSHに接続できます
        * MobaXtermを開き、以下の設定を行います。IPは前に確認したものを入力し、ルーターのバックエンドでも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここでは第4ステップで上位機のユーザー名を入力します**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初回接続時にはこのウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * パスワードの入力を求められます。上位機のパスワードを入力し、エンターキーを押してください

        **注意：ここで入力するパスワードは表示されません。入力後、直接エンターキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly`、パスワードは：`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
