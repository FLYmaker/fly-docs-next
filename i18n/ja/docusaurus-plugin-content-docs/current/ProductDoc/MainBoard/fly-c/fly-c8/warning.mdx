# マザーボードの注意事項

## 電源インジケータ
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.png').default} size="80%" align="center" />
    * TYPE-C電源のみで他のマザーボードの電源を接続しない場合、少なくとも2つのランプ`3.3V`と`5V`が点灯します
    * マザーボードの電源を接続した後、`3.3V`、`5V`、`VCC`のマザーボード電源ランプが点灯すべきです

### 問題のトラブルシューティング


    :::danger 重要

    * どんな外部デバイスも接続していない場合（ドライバー、リミットスイッチ、12864など）
    * TYPE-C電源のみで少なくとも`3.3V`が常に点灯します

    :::

    * マザーボードの電源とドライバーの電源を接続した後、`5V`のランプが点灯しない場合は、カスタマーサポートに連絡してください

## ドライバー

    * マザーボードは外部ドライバーをサポートします     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定ガイド</Button>

## リミットスイッチポート

    * 配線の順序に注意してください。三線リミットスイッチの場合、5V、GND、Sの3つを間違えるとマザーボードが故障する可能性があります
    * 二線リミットスイッチは直接SとGNDに接続します
    * PL08や他の近接スイッチは、シグナルポートにダイオードを外付けし、電圧を3.3V以下に下げる必要があります
