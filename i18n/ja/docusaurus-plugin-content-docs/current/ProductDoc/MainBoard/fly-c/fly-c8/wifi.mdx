# Wi-Fiに接続


## C8にアンテナを取り付ける

    * 赤い矢印が指している場所がWi-Fiアンテナの接続口です
    * Wi-Fiアンテナポートは`ipex1`です
    * アンテナをC8に取り付けないと、信号が弱くてインターネットに接続できない可能性があります
