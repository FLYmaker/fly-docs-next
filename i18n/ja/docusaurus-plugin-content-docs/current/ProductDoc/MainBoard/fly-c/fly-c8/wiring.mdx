## ドライバーのインストール

### マザーボードのドライバージャンパー

* **STEP/DIRモード。** このモードでは、ジャンパーピンを使用して細分を設定し、ポテンショメーターを調整して電流を調整します。最も一般的にはA4988ドライバーが使用され、この種のドライバーはメーカー提供の細分設定表を参照し、ジャンパーピンで細分を設定する必要があります。
* **UARTモード。** このモードで最も一般的に使用されるドライバーは：TMC2208、TMC2209、TMC2226などです。この種のドライバーチップはマスターコントローラーとUART非同期シリアル通信を行い、設定ファイルを変更することでドライバーの細分、動作電流、静音モードなどを設定できます。
* **SPIモード。** このモードで最も一般的に使用されるドライバーは：TMC5160、TMC2230などです。この種のドライバーチップはSPIでマスターコントローラーと同期シリアル通信を行い、同様に設定ファイルを変更することでドライバーの細分、動作電流、静音モードなどを設定できます。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/drive-dip.png').default} size="40%" align="center" />

----

### ドライバーのインストール

* ドライバーをインストールする前に、ドライバーやマザーボードに損傷を与えないために必要なチェックを行ってください。
* **FLYドライバー：** 無限位戻り機能を使用しない場合は、ダイポールスイッチを1の位置に設定してください；逆に、無限位戻り機能を使用する場合は、ダイポールスイッチをONの位置に設定してください。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.png').default} size="40%" align="center" />

    :::danger

    * **注意！！！ ドライバーモジュールをインストールする際、必ずENピンを左上角に挿入するようにしてください。そうしないとドライバーやマザーボードが損傷する可能性があります！！！ドライバーにヒートシンクをしっかりと貼り付けてください！！！**

    :::
