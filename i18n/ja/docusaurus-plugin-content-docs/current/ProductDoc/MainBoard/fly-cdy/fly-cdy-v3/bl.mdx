# ブートローダーファームウェア

## 書き込みモードへの進入

    :::tip
 
    右側が`BOOTボタン`、左側が`RESETボタン`

    :::

    * 書き込みモードに入る方法は2つあります
    * 第一種：基板が完全に電源オフの状態で、BOOTボタンを押しながら、データ転送可能なケーブルの一方を基板に、もう一方をデバイスに接続します
    * 第二種：データ転送可能なケーブルの一方を基板に、もう一方をデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押して、RSTボタンを離し、BOOTボタンを最後に離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/boot.png').default} size="50%" align="center" />
