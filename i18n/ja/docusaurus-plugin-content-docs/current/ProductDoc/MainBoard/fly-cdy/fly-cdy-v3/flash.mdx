# ファームウェアのコンパイルと書き込み

    :::tip

    * CDVはデフォルトでTFカードを使用してファームウェアを書き込みます
    * 容量が`32G`を超えないTFカードを準備してください
    * 無名のメモリカードは書き込みに失敗する可能性があるため、以下の推奨されるメモリカードを使用してファームウェアを更新してください
    * TFカードの転送速度は`class10`級以上が最適です。サンディスクなどのブランドのTFカードを使用することをお勧めします

    :::
