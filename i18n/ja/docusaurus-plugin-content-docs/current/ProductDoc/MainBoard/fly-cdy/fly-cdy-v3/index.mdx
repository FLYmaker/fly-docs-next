## 製品の特徴

    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ドライブ電圧の選択をサポート、デフォルト5V
    * RRFファームウェアをサポート

## 製品仕様

    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANバス接続をサポート、データ転送がより安定、遅延が少なく、接続がより安定
    * ファームウェア：Reprap/Klipper
    * ドライブサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160等
    * ドライブモードサポート：TMC：UART/SPI
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * repapファームウェアを使用すると、ESP32 WiFiモジュールの拡張が可能、1.6m/sをサポート
    * 独特なPWMファンMOS板設計、故障時は直接交換可能、顧客の修理難易度を低減
    * PCBは2OZ銅厚を採用、より大きな電流をサポート
    * 3つのADCインターフェース、最大3路の熱センサをサポート
    * 6つのIOインターフェース、レベリングとリミットに使用可能
    * 4つのヒータバインターフェース、最大4路のヒータバをサポート
