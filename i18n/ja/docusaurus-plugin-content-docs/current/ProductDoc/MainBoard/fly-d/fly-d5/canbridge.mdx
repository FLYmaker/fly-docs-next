# CANブリッジ

CANバスは、リアルタイムアプリケーション用のシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができます。世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内で異なるコンポーネント間の通信に使用され、重くて高価な配電ハーネスを置き換えるために使用されます。このプロトコルの堅牢性により、他の自動化や産業用アプリケーションにも拡大しています。CANプロトコルの特性には、完全なシリアルデータ通信の整合性、実時間サポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用してプリントヘッドをメインボードに接続する配線を減らします。以前は10本以上の線が必要でしたが、CANを使用すると4本の線だけで済み、配線量を大幅に減らし、布線が容易になります。
本章では、CANのブリッジ使用について簡単に概説します。

## ツールボードがメインボードにCANで接続される

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24vの電源線に接続してください。
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/wiring/canbridge.webp').default} size="80%" align="center" />
    * 120Ωのジャンパーを接続することを忘れないでください。
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/wiring/120.webp').default} size="80%" align="center" />

## メインボードのファームウェアの書き込み

* D5メインボードに書き込むファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください。
