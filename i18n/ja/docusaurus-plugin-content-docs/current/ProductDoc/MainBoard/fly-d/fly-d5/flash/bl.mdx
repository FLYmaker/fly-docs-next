# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D5はUSB接続を使用
    * D5は2024年9月1日から出荷時既にKatapultファームウェアがプリインストールされています
    :::

## 書き込みモードへの移行

    :::tip

    左が`BOOTボタン`、右が`RESETボタン`です

    :::

    * 書き込みモードに入る方法は2つあります
    * 第一の方法：主板の電源が完全に切れている状態で、BOOTボタンを押しながらデータ転送可能なケーブルを主板とデバイスに接続します
    * 第二の方法：データ転送可能なケーブルを主板とデバイスに接続した後、BOOTボタンを押しながらRESETボタンを押し、RESETボタンを離してからBOOTボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/img/boot.png').default} size="50%" align="center" />
