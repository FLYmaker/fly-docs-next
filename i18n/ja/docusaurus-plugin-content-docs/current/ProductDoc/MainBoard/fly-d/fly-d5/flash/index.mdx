# ファームウェアの書き込み説明

:::danger 知っておくべきこと
* メインボードのファームウェア書き込みにはKatapultを使用してKlipperを書き込みます
* メインボードにKatapultが既に書き込まれていることが確定している場合、最後のKatapultファームウェアの書き込みは不要です。それはメインボードのKatapultファームウェアが正しく書き込まれていない場合の予防措置です
* リセットボタンを2回素早く押して、メインボード上にLEDが点滅するかどうか確認できます。点滅する場合、メインボードがKatapultファームウェアに入っていることを示します
:::

## あなたが必要とするファームウェアをどうやって判断するか

    :::tip

    * 注意してください、あなたはそのうちの一つのファームウェアだけをフラッシュする必要があります

    :::
    
    * D5は以下の方法でホストマシンに接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性があり、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスを接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と共に使用すると、ホームポジションタイムアウト等のエラーが発生する可能性があります。
