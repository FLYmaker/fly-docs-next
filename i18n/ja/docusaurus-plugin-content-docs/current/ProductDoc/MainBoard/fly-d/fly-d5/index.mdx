# 紹介

> Mellow Fly-D5は、広州市マグネシウム・レンエレクトロニクス・テクノロジー有限公司のFly3Dチームが開発した高性能3Dプリンターボードで、最大5つのステッピングモーターをサポートします。


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/d5.png').default} size="60%" align="center" />

## **1. 製品の特徴**

* 32ビット、48MHzのARM Cortex-M0+シリーズのSTM32F072RBT6メインコントローラチップを採用
* CANバス接続をサポートし、データ伝送がより安定し、遅延が少なく、接続も安定
* ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
* DIAG機能ピンを搭載し、単純にジャンパキャップを抜き差しするだけで無限位置帰還が可能

## 2. 製品仕様

* MCU: STM32F072RBT6
* ファームウェア: Klipper/Marlin
* 12-24V直流電源サポート、電源供給がより安定し、降圧モジュール不要
* ドライバーの動作モードサポート：SPI、UART、STEP/DIR
* サポートドライバー：外付けTMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988等
* 拡張インターフェース：BLTouch（Servos、Probe）、UART X1、12864 X1
* ファン: 2線制御可能ファン X2、並列ファン X2、常時通電ファン X2
* 温度センサー：2路 NTC 100K
* 加熱インターフェース：2路制御可能インターフェースで10A対応
* サポートマシン構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
