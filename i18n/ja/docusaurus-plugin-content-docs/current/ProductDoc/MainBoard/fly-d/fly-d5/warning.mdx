# マザーボードの注意事項

## 電源インジケータ
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/power.webp').default} size="80%" align="center" />
    * TYPE-Cからの給電のみで他のマザーボードの電源を接続しない場合、少なくとも2つのランプ`3.3V`と`5V`が点灯
    * マザーボードの電源を接続すると、`3.3V`、`5V`、`VCC`のランプが点灯するはず

### 問題のトラブルシューティング


    :::danger 注意

    * 外付けデバイス（ドライバー、リミットスイッチ、12864等）がない場合
    * TYPE-Cからの給電だけで少なくとも`3.3V`と`5V`のLEDが常時点灯する

    :::

    * ランプが全く点灯しない場合は、ショートしている可能性があるので、カスタマーサポートに連絡してください
    * `3.3V`、`5V`のどちらかが点滅する場合は、ショートしている可能性があるので、カスタマーサポートに連絡してください

## ドライバー

    * マザーボードは外部ドライバーをサポート     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定ガイド</Button>

## リミットポート

    * 配線の順序に注意してください。三線リミットは5V、GND、Sの3つを間違えるとマザーボードが壊れる可能性があります
    * 二線であればSとGNDに直接接続
    * PL08や他の近接スイッチは、シグナルポートにダイオードを外付けして電圧を3.3V以下に下げる必要があります

## ファームウェア

    * ボタンの問題により、ファームウェアの更新にはKatapultファームウェアを使用
    * KatapultファームウェアをPCでフラッシュし、Katapultファームウェアを通じてklipperをフラッシュすることをお勧めします
