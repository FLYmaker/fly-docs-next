# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D7はUSB接続を使用
    * D7は2024年9月1日から出荷時にKatapultファームウェアがプリインストールされています

    :::

## 書き込みモードへの移行

    :::tip

    左側が`BOOTボタン`、右側が`RESETボタン`

    :::

    * 書き込みモードへの移行方法は2つあります
    * 第一の方法：電源を完全に切った状態で、BOOTボタンを押しながらデータ伝送用のケーブルを主板とデバイスに接続します
    * 第二の方法：データ伝送用のケーブルを主板とデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押し、RSTボタンを放してからBOOTボタンを放します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/boot.webp').default} size="50%" align="center" />
