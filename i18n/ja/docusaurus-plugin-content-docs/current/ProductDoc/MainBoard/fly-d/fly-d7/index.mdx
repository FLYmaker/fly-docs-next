# 紹介

> Fly-D7は、広州市メガレン電子科技有限公司のFly3Dマザーボード開発チームが提供する高性能3Dプリンターマザーボードで、最大7つのステッピングモーターをサポートします。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.webp').default} size="60%" align="center" />

## **1. 製品特性**

    * 32ビット、48MHzのARM Cortex-M0+シリーズSTM32F072RBT6メインコントローラチップを採用
    * CANバス接続をサポートし、データ伝送がより安定し、遅延が少なく、接続が安定
    * ドライバサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * 搭載DIAG機能ピン、シンプルなジャンパキャップの抜き差しで無限位帰位サポートを使用可能

## 2. 製品仕様

    * MCU: STM32F072RBT6
    * ファームウェア: Klipper/Marlin
    * 12-24V直流電源サポート、電源供給が安定し、降圧モジュール不要
    * ドライバ動作モードサポート：SPI、UART、STEP/DIR
    * サポートドライバ：外付けTMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988など
    * 拡張インターフェース：BLTouch（Servos、Probe）、UART X1
    * ファン: 2線制御ファン X2、パラレルファン X2、常時オンファン X2
    * 温度センサー：3路 NTC 100K
    * 加熱インターフェース：3路制御可能インターフェース、10Aサポート
    * サポートマシン構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
