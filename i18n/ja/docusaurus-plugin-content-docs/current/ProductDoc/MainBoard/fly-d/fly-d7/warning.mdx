# マザーボードの注意事項

## 電源インジケーターランプ
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/power.png').default} size="80%" align="center" />
    * TYPE-Cのみで電源を供給し、他のマザーボードの電源を接続しない場合は、少なくとも2つのランプが点灯する`3.3V`と`5V`
    * マザーボードの電源に接続した後は、マザーボードの電源ランプが点灯する`3.3V`、`5V`、`VCC`

### 問題のトラブルシューティング


    :::danger 注意

    * 外付けデバイスがない場合（ドライバー、リミットスイッチ、12864など）
    * TYPE-Cのみで電源供給すると、少なくとも`3.3V`と`5V`の2つのLEDが常に点灯する

    :::

    * ランプが点灯しない場合、ショートしている可能性があるため、カスタマーサポートに連絡してください
    * `3.3V`または`5V`のいずれかのランプが点滅する場合も、ショートしている可能性があるため、カスタマーサポートに連絡してください

## ドライバー

    * マザーボードは外部ドライバーをサポートしています     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定のチュートリアル</Button>

## リミットポート

    * 配線の順序に注意してください。3線式リミットスイッチの場合、5V、GND、Sを間違えるとマザーボードが破損する可能性があります
    * 2線式の場合は、SとGNDに直接接続してください
    * PL08や他の近接スイッチは、信号ポートにダイオードを外付けして電圧を3.3V以下に下げる必要があります
