# CANブリッジ

CANバスはリアルタイムアプリケーション向けのシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送し、世界中で最も広く利用されているフィールドバスの一つです。CANプロトコルは、自動車内で異なる部品間の通信に使用され、高価でかさばる配線束に代わるものです。このプロトコルの堅牢性により、その用途は他の自動化および産業用アプリケーションにまで広がっています。CANプロトコルの特徴には、完全性のあるシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、およびエラー検出機能が含まれます。
3Dプリンタでは、CANバスを使用することでプリントヘッドとメインボードの接続に必要な配線の数を減らします。従来は十数本の線が必要でしたが、CANを使用すると4本の線だけで済み、配線の数を大幅に減らし、配線難易度を低減します。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードがCANでメインボードに接続される

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24Vの電源線に接続してください
    
    * 120Ωのジャンパを接続してください
