# 板載周辺機器の紹介

    * ディップスイッチを右にスライドすると、Type-Cポートの横のUSBポートが有効になります
    * ディップスイッチを左にスライドすると、Type-Cポートの横のRS232ポートが有効になります
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/mode_dip.png').default} size="80%" align="center" />

## USBポートの紹介
    
    * カメラやメインボードなどの外部機器を接続できます
