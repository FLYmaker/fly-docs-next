# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D8はUSB接続を使用

    :::

## 焼き込みモードに入る

    :::tip

    左側が`RSTボタン`、右側が`BOOTボタン`

    :::

    * 焼き込みモードに入る方法は2種類
    * 第一の方法：メインボードが完全に電源が切れている状態で、BOOTボタンを押しながらデータ転送可能なケーブルをメインボードとあなたのデバイスに接続
    * 第二の方法：データ転送可能なケーブルをメインボードとあなたのデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押し、RSTボタンを離した後、最後にBOOTボタンを離す
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/flash/img/boot.png').default} size="50%" align="center" />
