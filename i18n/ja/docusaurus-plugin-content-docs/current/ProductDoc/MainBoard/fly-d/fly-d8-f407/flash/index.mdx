# ファームウェアの書き込み説明

:::danger 知っておくべきこと
* メインボードのファームウェアを書き込むには、Katapultを通じてKlipperを書き込む必要があります
* メインボードにKatapultが既に書き込まれている場合、後ろのKatapultファームウェアの書き込みは不要です。これは、メインボードのKatapultファームウェアが正常に書き込まれなかった場合の予防措置です
* リセットボタンを2回素早く押して、メインボード上にLEDが点滅するかどうか確認してください。点滅している場合は、メインボードがKatapultファームウェアに入っていることを示しています
:::

## どのファームウェアが必要かを判断する方法

    :::tip

    * 注意：あなたはその中の一種類のファームウェアを書き込むだけで大丈夫です

    :::
    
    * D8は以下の方法で上位機に接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性があり、安定している
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスを接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用する場合、ホームポジションのタイムアウトなどのエラーが発生する可能性があります。
