# 紹介

> Mellow Fly-D8は、広州市镁伦電子科技有限公司Fly3D基板開発チームが開発した高性能3Dプリンタ基板で、最大8つのステッピングモーターをサポートします。


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/d8.webp').default} size="80%" align="center" />

## 製品の特徴

    * 基板のリミットスイッチなどのインターフェースは5V供給で隔離保護が行われており、POWER電源を接続しないと電源が出力されません
    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407チップを採用
    * CANバスとRS232接続をサポートし、データ転送が安定し、遅延が少なく、接続が安定
    * 基板にはBOOTとRESETボタンが予約されており、ユーザーはUSBを通じてDFUモードでファームウェアを更新できます
    * 電源インターフェースには逆接続保護があり、DIY時に電源線を逆に接続して基板が焼けることを防ぎます
    * 2つのUSB2.0インターフェースを提供し、さまざまな機器の接続ニーズに対応するために、一つのUSBインターフェースはスイッチでRS232インターフェースに切り替え可能

# 製品の仕様

    * メインコントローラチップ：32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ファームウェア：Marlin 2.0/Klipper
    * ドライバモードサポート：TMC：UART/SPI
    * 8つのドライバサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ディスプレイ：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 自動レベリングセンサー対応：BLTouch、PL08Nなどの接近センサー、Klicky、Voron Tapなど
    * 入力電圧：12V/24V対応
    * 6つのリミットスイッチインターフェースで、正確な範囲制限と位置制御を実現
    * 2つのRGBインターフェースで、多彩な視覚効果を提供
    * 3つのヒータ管インターフェースで、エクストルーダーとヒートベッドに独立した制御を提供
    * 4つのサーミスタインターフェースで、最大4組の温度を同時に監視し、正確な温度制御を実現
    * USBインターフェース：USB2.0インターフェースを2つ拡張サポートし、一つのUSBはスイッチでRS232に切り替え可能
    * 2つのUSB2.0インターフェースを提供し、さまざまな機器の接続ニーズに対応するために、一つのUSBインターフェースはスイッチでRS232インターフェースに切り替え可能
