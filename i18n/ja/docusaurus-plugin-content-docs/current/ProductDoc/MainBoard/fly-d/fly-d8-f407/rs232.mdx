# RS232接続

RS232標準インターフェース（またEIA RS-232と呼ばれる）は、シリアル通信インターフェースの標準の一つであり、様々な機器間のデータ伝送に広く使用されています。その安定性と信頼性により、多くの機器の第一選択の通信方法となっています。このプロトコルは標準の5Vとは異なる信号レベルを使用して電線データを伝送し、信号干渉を最大限に減少させます。3Dプリンターシステムにおいて、RS232はCANに比べてより高いデータ伝送速度、より良い安定性、より簡単な設定、上位機との互換性が優れています。

## ツールボードがRS232でマザーボードに接続

    * ツールボードの電源線はRS232信号線に接続せず、12V-24vの電源線に接続してください

    * D8マザーボードファームウェアは再コンパイルの必要はありません
