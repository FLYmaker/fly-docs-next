# CANブリッジ

CANバスはリアルタイムアプリケーション用のシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができ、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内でさまざまな異なるコンポーネント間の通信に使用され、高価でかさばる配線束に取って代わります。このプロトコルの堅牢性により、他の自動化および産業用アプリケーションにその用途が拡大しています。CANプロトコルの特徴には、シリアルデータ通信の完全性、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシングおよびエラーチェック機能が含まれます。
3Dプリンタでは、プリントヘッドをメインボードに接続するための配線の数を減らすためにCANバスが使用されます。元々十数本の線が必要でしたが、CANを使用すると4本の線で済み、配線の数を大幅に減らし、配線作業を簡素化します。
本章では、CANのブリッジの使用について簡単に概説します。

## ツールボードがCANでメインボードに接続

    * ツールボードの電源線はCANポートに接続されるのではなく、12V-24vの電源線に接続されることに注意してください

    * 120Ωのジャンパーを接続することを忘れないでください

## メインボードのファームウェアの書き込み

* D8メインボードに書き込むファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください
