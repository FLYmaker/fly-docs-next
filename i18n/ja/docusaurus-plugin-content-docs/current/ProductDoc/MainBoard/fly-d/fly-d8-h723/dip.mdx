# オンセット周辺機器の紹介

    * ディップスイッチを右にすると、Type-Cポートの隣のUSBポートが開きます
    * ディップスイッチを左にすると、Type-Cポートの隣のRS232ポートが開きます
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/mode_dip.webp').default} size="80%" align="center" />

## USBポートの紹介
    
    * カメラやメインボードなどの外部デバイスを接続できます
