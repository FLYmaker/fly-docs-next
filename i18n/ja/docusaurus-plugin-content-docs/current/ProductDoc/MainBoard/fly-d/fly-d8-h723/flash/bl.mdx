# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D8はUSB接続を使用

    :::

## フラッシュモードに入る

    :::tip

    左側が`RSTボタン`、右側が`BOOTボタン`

    :::

    * フラッシュモードに入る方法は二つあります
    * 第一の方法：主板が完全に電源を切った状態で、BOOTボタンを押しながらデータ転送が可能なケーブルを主板とあなたのデバイスに接続
    * 第二の方法：データ転送が可能なケーブルを主板とあなたのデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押し、RSTボタンを離してからBOOTボタンを離す
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-h723/flash/img/boot.webp').default} size="50%" align="center" />
