# はじめに

> Mellow Fly-D8は、広州メロン電子科技有限公司のFly3Dマザーボード開発チームが開発した高性能3Dプリンターマザーボードで、最大8つのステッピングモーターをサポートします。

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-h723/img/d8.webp').default} size="80%" align="center" />

## 製品の特徴

    * マザーボードのリミットスイッチなどのインターフェースは5V電源供給で絶縁保護が施されており、POWER電源に接続しない限り電源は出力されません
    * 32ビット、550MHzのARM Cortex-M7シリーズSTM32H723を採用
    * CANバスとRS232接続をサポートし、データ転送がより安定し、遅延が少なく、接続も安定
    * マザーボードにはBOOTとRESETボタンが予約されており、ユーザーはUSBを通じてDFUモードに入り、ファームウェアを更新できます
    * 電源インターフェースには逆接続防止保護が施されており、DIYの際に電源線を逆に接続することで基板を焼くのを防ぎます
    * 2つのUSB2.0インターフェースを提供し、さらに多くの接続可能性を拡大しました。1つのUSBインターフェースはスイッチでRS232インターフェースに切り替えることができ、さまざまなデバイスの接続ニーズに対応

# 製品仕様

    * マイコン：32ビット、550MHzのARM Cortex-M7シリーズSTM32H723を採用
    * ファームウェア：Marlin 2.0/Klipper
    * ドライバーモードサポート：TMC：UART/SPI
    * 8つのドライバー対応：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 自動ベッドレベリングセンサー対応：BLTouch、PL08Nなど近接センサー、Klicky、Voron Tapなど
    * 入力電圧：12V/24Vサポート
    * 6つのリミットスイッチインターフェースで、正確な範囲制限と位置制御を実現
    * 2つのRGBインターフェースで、多彩なビジュアル効果を提供
    * 3つのヒータインターフェースで、エクストルーダとベッドを独立して制御
    * 4つの温度センサーインターフェースで、最大4つの温度を同時に監視し、正確な温度制御を実現
    * USBインターフェース：USB2.0インターフェースを2つ拡張し、そのうち1つのUSBはスイッチでRS232に切り替え可能
    * 2つのUSB2.0インターフェースを提供し、さらに多くの接続可能性を拡大しました。1つのUSBインターフェースはスイッチでRS232インターフェースに切り替えることができ、さまざまなデバイスの接続ニーズに対応
