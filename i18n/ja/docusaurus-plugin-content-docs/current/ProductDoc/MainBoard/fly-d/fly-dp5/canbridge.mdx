# CANブリッジ

CANバスはリアルタイムアプリケーション向けのシリアル通信プロトコルバスで、信号を伝送するためにツイストペアケーブルを使用し、世界中で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまなコンポーネント間の通信に使用され、重くて高価な配電線束に代わるものです。このプロトコルの堅牢性により、他の自動化や産業用途にも広がっています。CANプロトコルの特性には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、およびエラーチェック機能が含まれます。
3Dプリンタでは、プリントヘッドからメインボードへの配線数を減らすためにCANバスを使用します。以前は十数本の線が必要でしたが、CANを使用すると4本の線で済み、配線数を大幅に減らし、配線難易度を低減します。
本章では、CANのブリッジの使用について簡単に概説します。

## ツールボードでCANを使用してメインボードに接続する

    * ツールボードの電源線はCANポートに接続せず、12V-24Vの電源線に接続してください

* DP5メインボードに焼き込むファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください
