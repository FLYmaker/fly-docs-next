# Katapultファームウェア

## Katapultファームウェア説明

    <CANBOOT />

    :::tip

    * D5はUSB接続を使用

    :::

## 書き込みモードへの移行

    :::tip

    左が`BOOTボタン`、右が`RESETボタン`

    :::

    * 書き込みモードに入る方法は2つあります
    * 第一の方法：主板が完全に電源を切っている状態で、BOOTボタンを押しながらデータ転送可能なケーブルを主板とデバイスに接続
    * 第二の方法：データ転送可能なケーブルを主板とデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押して、RSTボタンを離し、最後にBOOTボタンを離す
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/flash/img/boot.png').default} size="50%" align="center" />

