# 紹介

> Mellow Fly-DP5は、広州市メロン電子科技有限公司のFly3Dチームが開発した高性能3Dプリンターボードで、最大5つのステッピングモーターをサポートします。


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.png').default} size="60%" align="center" />

## **1. 製品の特徴**

* 32ビット、48MHzのARM Cortex-M0+シリーズSTM32F072RBT6メインコントローラチップを採用
* CANバス接続をサポートし、データ伝送が安定し、遅延が小さく、接続が安定
* ドライバーサポート：DP2209とDP560
* 板上DIAG機能ピンを搭載、簡単なジャンパーキャップの抜き差しだけで無限位のホームポジションサポートが可能

## 2. 製品仕様

* MCU: STM32F072RBT6
* ファームウェア: Klipper/Marlin
* 12-24Vの直流電源をサポートし、供給が安定し、降圧モジュールが不要
* ドライバーの動作モードサポート：SPI、UART、STEP/DIR
* サポートドライバー：外部TMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988等
* 拡張インターフェース：BLTouch（Servos、Probe）、UART X1、12864 X1
* ファン: 二線制御ファン X2 、並列ファン X2 、常時ONファン X2
* 温度センサー：2路 NTC 100K
* 加熱インターフェース：2路制御インターフェース、10Aサポート
* サポートする機械構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
