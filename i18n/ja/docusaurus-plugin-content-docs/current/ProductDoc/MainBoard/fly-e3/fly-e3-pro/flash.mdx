# ファームウェアのコンパイルと書き込み

    :::tip

    * E3 ProはデフォルトでTFカードを使用してファームウェアを書き込みます
    * 容量が`32G`を超えないTFカードを準備してください
    * 無名メーカーのメモリカードは書き込みに失敗する可能性があるため、できるだけ下記の推奨メモリカードを使用してファームウェアを更新してください
    * TFカードの転送速度は`class10`級以上であることが望ましく、SanDiskなどのブランドのTFカードを使用することをお勧めします

    :::

## Klipperファームウェアのコンパイル

    <MakeOperate/>

    <USB />

    * `Q`キーを押して **Save configuration** が表示されたら、`Y`キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインに戻るはずです
    * `make -j4` と入力してコンパイルを開始します、時間が少しかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイルが成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
        
    * 下の図はコンパイル成功の例です

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make-done.png').default} size="50%" align="left" />
