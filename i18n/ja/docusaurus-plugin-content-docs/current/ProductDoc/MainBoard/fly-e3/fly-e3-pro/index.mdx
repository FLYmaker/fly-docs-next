    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ

## 製品仕様

    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ファームウェア：Reprap/Klipper
    * 搭載のTMC2209ドライバ5個
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 高圧INポート1個、自動ベッドレベリングセンサー対応：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * Repapファームウェアを使用する場合、ESP32 WiFiモジュールを使用し、1.6m/sをサポート
    * 独特のPWM Fan MOSボード設計で、損傷した場合に直接交換可能、顧客の修理難易度を低減、2pinファンインターフェース4個
    * PCBは2OZの銅厚を採用、大きな電流をサポート可能
    * ADCインターフェース2個、最大2路の熱敏抵抗をサポート
    * IOインターフェース4個、レベリングとリミットスイッチに使用可能
    * ヒーターロッドインターフェース3個、最大3路のヒーターロッドをサポート
