# CANブリッジ

CANバスはリアルタイムアプリケーション向けのシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができ、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内でさまざまな異なるコンポーネント間の通信に使用され、代わりに高価でかさばる配電線束を取り除きます。このプロトコルの堅牢性により、他の自動化および産業用アプリケーションへの用途が拡大しています。CANプロトコルの特徴には、完全性のあるシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、およびエラーチェック機能が含まれます。
3Dプリンターでは、プリントヘッドをメインボードに接続する配線の数を減らすためにCANバスが使用されます。従来は十数本の線が必要でしたが、CANを使用することで4本の線だけで済み、配線の数を大幅に減らし、配線の難易度を下げます。
本章では、CANのブリッジの使用について簡単に概要を説明します。

## ツールボードがメインボードにCANで接続される

    * ツールボードの電源線はCANポートに接続されるのではなく、12V-24vの電源線に接続されることに注意してください
