# BLファームウェア

    :::tip

    * 工場出荷時に既に`ブートローダー`がプリインストールされているため、再インストールする必要はありません。

    :::

## 書き込みモードへの移行

    :::tip

    右側が`BOOTボタン`、左側が`RESETボタン`です。

    :::

    * 書き込みモードへの移行方法は2通りあります。
    * 1つ目：完全に電源を切った状態で、BOOTボタンを押しながら、データ転送が可能なケーブルを一端をメインボードに、もう一端をあなたのデバイスに接続します。
    * 2つ目：データ転送が可能なケーブルを一端をメインボードに、もう一端をあなたのデバイスに接続した後、BOOTボタンを押し続け、RSTボタンを押してからRSTボタンを放し、最後にBOOTボタンを放します。
