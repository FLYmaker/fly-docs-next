## 製品仕様

    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANバス接続のサポートにより、データ伝送が安定し、遅延が少なく、接続が安定
    * ファームウェア：Reprap/Klipper
    * ドライバー対応：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバーモード対応：TMC：UART/SPI
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * repapファームウェアを使用すると、拡張ESP32 WiFiモジュールを使用でき、1.6m/sのサポート
    * 独自のPWM Fan MOSボード設計で、故障時には直接交換可能、顧客のメンテナンスの難易度を低減
    * PCBは2OZの銅厚を採用、大きな電流をサポート
    * 6つのADCインターフェース、最大で六路の熱敏サポート
    * 7つのIOインターフェース、調整とリミットに使用可能
    * 七つのヒートロッドインターフェース、最大で七路のヒートロッドをサポート
