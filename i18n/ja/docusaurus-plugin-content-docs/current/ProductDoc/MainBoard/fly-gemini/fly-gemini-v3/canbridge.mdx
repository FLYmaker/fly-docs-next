# CANブリッジ

CANバスはリアルタイムアプリケーション向けのシリアル通信プロトコルバスであり、双方向線を使用して信号を伝送し、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内でさまざまなコンポーネント間の通信に使用され、高価でかさばる配線束を置き換えます。このプロトコルの堅牢性により、他の自動化や産業用途にも拡張されています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの転送速度、11ビットのアドレッシング、およびエラーチェック機能が含まれます。
3Dプリンターでは、CANバスを使用してプリントヘッドからメインボードへの配線を減らし、元々10本以上の線が必要だったものを、CANを使用することで4本の線で済むようになり、配線の数を大幅に減らし、配線の難易度を下げます。
本章では、CANのブリッジ接続の使用について簡単に概要を説明します。

## ツールボードをCANでメインボードに接続

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24Vの電源線に接続してください
