# オンオフスイッチの紹介

:::tip
FLY Geminiはオンオフスイッチを介して上位機に接続します。位置が正しくない場合、正常に使用できません。
:::

:::danger 注意
オンオフスイッチが指定された位置に設定されていないと、上位機に正常に接続できませんのでご注意ください。
:::

## 内蔵上位機への接続

    * オンオフスイッチ`1`,`2`を右側に切り替えて、下位機と上位機を内蔵USBで接続します
    * オンオフスイッチ`3`,`4`を左側に切り替えないと、下位機と上位機の接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * この方法でオンオフスイッチを切り替えた後、下部のUSBポートは下位機によって占有されるため、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## 外部上位機への接続

    * オンオフスイッチ`3`,`4`を右側に切り替えて、下位機と上位機を内蔵USBで接続します
    * オンオフスイッチ`1`,`2`を左側に切り替えないと、下位機と上位機の接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
