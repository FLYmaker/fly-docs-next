# ファームウェアの書き込み説明

## どのファームウェアが必要かを判断する方法

    :::tip

    * 注意してください、必要なのはどれか一つのファームウェアだけです

    :::
    
    * Gemini V3は以下の方法で上位機と接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性が高く、安定している
        * 欠点：複数のUSBデバイスがデータ伝送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスを接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、原点復帰タイムアウトなどのエラーが発生する可能性があります。
