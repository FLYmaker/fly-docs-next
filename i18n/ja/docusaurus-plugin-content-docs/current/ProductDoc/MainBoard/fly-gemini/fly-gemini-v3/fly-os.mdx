## システム書き込みの準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上128GB以下で、TFカードの転送速度はclass10以上である必要があります
    * 対応する<Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>をダウンロード
    * 必要な<Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button>をダウンロード

## システムの書き込み

:::info
TFカードへの書き込みとTFカードを介してM2WEへの書き込みは、どちらか一方を選択して行ってください
:::

:::info
* 機械にM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEへの書き込みは、TFカードを使用してシステムをフラッシュすることをお勧めします。USB経由でのM2Wへの書き込みはドライバの互換性などの問題で失敗する可能性がありますので、[メモリカードを介してM2WEに書き込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を直接使用してください
* ダウンロードしたシステムイメージが対応しているかどうかを確認してください。対応していない場合、起動できません
:::
