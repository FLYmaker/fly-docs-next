# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.png').default} size="80%" align="center" />

## 製品の特徴
    * 搭載されたホストPCは、ラズベリーパイの代わりにホストPCとして動作し、1GBのオンボードDDR3メモリを備えています。
    * 5V電源保護の強化により、チップの損傷の可能性を低減します。
    * MCUチップのストレージカード設定が廃止され、ファームウェアの書き込みが容易になりました。
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyおよびFLY-Toolsが予めインストールされており、複雑な操作を必要とせず、初心者でも簡単に使い始めることができます（TFカードまたはM2WEの購入が必要です）。


## 製品仕様

    * CPU：高性能全志H5チップ、4コア64ビットCortex-A53
    * GPU：高性能6コアMali 450、ピクセルフィルレートが2.7gpixel/s以上
    * RAM：1GB DDR3（GPUと共有）
    * ROM：最大128GB SDCARD対応
    * 周辺機器：Spi x2、UART x2、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN x1
    * インターフェース：オンボードM.2インターフェース、WIFI、eMMC等の拡張が可能（専用プロトコル、FLY π以外のM.2デバイスには接続しないでください）
    * 12-24V直流電源対応、供給がより安定し、降圧モジュールが不要
    * 5V制御可能なファンインターフェースを予備、ホストPCの冷却に使用可能
    * CAN拡張インターフェースを予備
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyおよびFLY-Toolsが予めインストールされており、複雑な操作を必要とせず、初心者でも簡単に使い始めることができます（TFカードまたはM2WEの購入が必要です）
    * ドライバ: 高圧ドライバ*4
    * MCU: STM32F405
