# 上位機へのSSH接続


* 機器にシリアルポートまたはWIFIで接続することができます
* **MobaXterm_Personal**または他の**SSHターミナルツール**を事前に用意してください
* SDカードまたはM2WEに**対応する上位機のシステム**が書き込まれていることを確認してください
* **上位機が正常に起動しない場合は、ドライバー、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後に、ファームウェアのコンパイルが便利になるように、SSHにネットワーク接続してください！！**
* **ファームウェアのコンパイルが必要な場合は、WIFIまたは有線LANで上位機に接続するチュートリアルをご覧ください**

        ## ネットワークIPで上位機にSSH接続する

        * **ルーターのバックエンド**にログインしてください
        * システムが**正常に起動**していることを確認してください
        * 上位機の**ユーザー名とパスワード**を確認してください。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * 基板が起動し、ネットワークに接続されている場合、IPアドレスを使ってSSH接続することができます
        * MobaXtermを開き、以下の設定を行います。IPアドレスは先ほど確認したものを入力してください。またはルーターのバックエンドで確認することもできます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **注意：ここでは第4ステップで上位機のユーザー名を入力します**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * 初めて接続するとこのウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * パスワードの入力を求められます。上位機のパスワードを入力し、Enterキーを押してください

        **注意：ここで入力するパスワードは表示されません。入力後、直接Enterキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly` 、パスワードは：`mellow` です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />

        * ログイン成功
