# CANブリッジ

CANバスはリアルタイムアプリケーション用のシリアル通信プロトコルバスで、ツイストペアケーブルを使用して信号を伝送することができ、世界中で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまな部品間の通信に使用され、高価でかさばる配線束を代替します。このプロトコルの堅牢性により、他の自動化や産業用途に広がっています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、そしてエラーチェック能力が含まれます。
3Dプリンターでは、CANバスを使用してプリントヘッドとメインボード間の配線の数を減らします。元々は十数本の線が必要でしたが、CANを使用することで4本の線だけで済み、配線の数を大幅に減らし、配線作業の難易度を低減します。
本章では、CANブリッジの使用について簡単に概要を説明します。

## ツールボードがCANでメインボードに接続される

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24V電源線に接続してください。
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/canbridge.png').default} size="80%" align="center" />
    * 120Ωのジャンパーを接続することを忘れないでください。
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/120.png').default} size="80%" align="center" />

## メインボードのファームウェアを書き込む

* D5メインボードに書き込むファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください。
