# Katapultファームウェア

    :::tip

    * 工場出荷時に`katapult`が既にインストールされているので、再インストールする必要はありません。

    :::

## 書き込みモードへの移行

    :::tip

    囲まれた部分が`BOOTボタン`です。

    :::

    * 書き込みモードに移行するには、主板の電源を完全に切断した状態で、BOOTボタンを押し続け、データ転送用のケーブルの一方を主板に、他方をあなたのデバイスに接続してください。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/boot.webp').default} size="100%" align="center" />
