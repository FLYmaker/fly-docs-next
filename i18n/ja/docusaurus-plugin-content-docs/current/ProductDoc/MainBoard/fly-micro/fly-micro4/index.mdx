# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## 製品の特徴

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * CANブリッジ対応
    * 搭載4つの2209
    * 搭載 DIAG 機能ピン、シンプルなジャンパーハットの抜き差しで無限位復帰サポートが可能



## 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * ファームウェア:  Klipper
    * 12-24V直流電源対応、電源が安定し、降圧モジュールが不要
    * ドライバ動作モード対応：UART、STEP/DIR
    * 拡張インターフェース：BLTouch（サーボ、プローブ）、近接スイッチ X1
    * ファン: 制御可能な2線ファン X2 、常時ONファン X1
    * 温度センサー：2路 NTC 100K
    * 加熱インターフェース：2路制御インターフェース、10A対応
