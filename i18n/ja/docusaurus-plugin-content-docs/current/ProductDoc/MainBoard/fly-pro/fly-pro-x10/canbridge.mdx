# CANブリッジ

CANバスはリアルタイムアプリケーション用のシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができます。世界中で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまなコンポーネント間の通信に使用され、従来の高価でかさばる配電ハーネスを代替します。このプロトコルの堅牢性により、その用途は他の自動化および工業用途にまで拡大しています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、およびエラー検出機能が含まれます。
3Dプリンターでは、CANバスを使用してプリントヘッドとメインボードの接続線を減らすことができます。従来は十数本の線が必要でしたが、CANバスを使用することで、4本の線だけで済み、配線の数を大幅に減らし、配線作業の難易度を下げます。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードがCANでメインボードに接続

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24vの電源線に接続してください
