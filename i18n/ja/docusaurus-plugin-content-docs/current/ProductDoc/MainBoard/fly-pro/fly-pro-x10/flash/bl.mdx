# BLファームウェア

    :::tip

    * 出荷時には既に`bootloader`がプリインストールされているので、再度書き込みする必要はありません。

    :::

## 書き込みモードへの入り方


    * 書き込みモードに入る方法は二つあります
    * 第一の方法：基板が完全に電源を切った状態で、BOOTを短絡し、データ転送機能を持つケーブルの一端を基板に、もう一端をデバイスに接続します。
    * 第二の方法：データ転送機能を持つケーブルの一端を基板に、もう一端をデバイスに接続した後、BOOTを短絡し、RSTボタンを押してから離し、最後にBOOTのジャンパーを外します。

            4. ファームウェアの書き込み

            :::tip

            どちらか一方のファームウェアを書き込むだけで大丈夫です。

            * 以下はkatapultファームウェアの書き込みコマンドです。
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * 以下はbootloaderファームウェアの書き込みコマンドです。
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```
