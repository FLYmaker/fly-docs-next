## 製品特徴

    * 32ビット、550 MHzのARM Cortex-M7シリーズSTM32H723ZGT6マイコンを採用
    * マザーボードとコアボードはBTB接続方式を使用し、香橙派CM4やRaspberry Pi CM4などのコアボードに対応
    * マザーボードにはBOOTボタンを予備しており、ユーザーはDFU方式でマザーボードのブートローダーを更新可能
    * 10路の抜き差し式TPドライバを使用し、ジャンパーなしでドライバモードを切り替え可能
    * 板上にDIAG機能ピンを搭載し、簡単な抜き差しで無限位の帰位が可能
    * ドライバ電源はHVINとVIN電源選択をサポート
    * MicroSDカードでMCUファームウェアのアップグレードが可能、またはKlipperのmake flashコマンドでDFU経由でMCUファームウェアを更新可能
    * 6路の2線制御ファン、抜き差し可能なMOSサポート、2路の4線ファン、1路のVCCファン
    * 数控ファンは24V、12V、5Vの電圧選択が可能、外部のトランスモジュールの接続作業を省き、マザーボードの破損リスクを減少
    * 交換可能なヒューズを使用し、交換が容易；

## 製品仕様

    * MCU: STM32H723ZGT6
    * ドライバ電源:  VIN(12V-24V)、HVIN(12V-48V)
    * マザーボード電源:  VIN(12V-24V)
    * ヒートベッド電源:  BED IN(12V-24V)
    * 加熱インターフェース:  ヒートベッド(BED OUT)、ヒートスティック（HETA0、HETA1、HETA2、HETA3）
    * ヒートベッドポート最大出力電流: 15A
    * ヒートスティックポート最大出力電流: 6A
    * ファンポート最大出力電流: 1A
    * ドライバ動作モードサポート：SPI、UART、STEP/DIR
    * 拡張インターフェース：BLTouch（Servos、Probe）、UART X1、高圧リミットポート X1
