# マザーボードの注意事項

## 電源インジケータ
    * Type-C電源を使用し、他のマザーボード電源を接続しない場合、`SYS_5V`とSYS_3.3Vが点灯するべきです
    * マザーボード電源`HVIN`、`VIN`、`BOARO`、`POWER`に接続後、マザーボードの電源ランプが常時点灯すべきです

## ドライバー

    * TP5160とTP2209は名前が異なるだけで、ジャンパーなしでドライバーモードを選択できます
    * ドライバーポートにはTPドライバーのみをインストールでき、他のデバイスはインストールできません

## リミットポート
    * ProX10は5Vと12Vの電源選択をサポートしており、近接スイッチを使用する場合はダイオードをインストールして降圧する必要があります

## 上位機
    * ProX10は外部上位機の使用をサポートしており、ダイポールを選択してType-Cに接続すれば使用可能です

## ファームウェアの書き込み

    * マザーボードはTFカードを使用して書き込むことができますが、出荷時のデフォルトは`Katapult`です
    * TFカードを使用して書き込みを行う場合は、BLを再書き込みする必要があります
