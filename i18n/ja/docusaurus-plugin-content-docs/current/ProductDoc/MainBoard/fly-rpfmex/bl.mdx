# Katapultファームウェア



## 書き込みモードに入る

    :::tip

    囲まれた部分が`BOOTボタン`

    :::

    * 書き込みモードに入るには、メインボードが完全に電源を切った状態で、BOOTボタンを押しながら、データ転送用のケーブルの一方をメインボードに、もう一方をデバイスに接続します
