## 製品の特徴

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * 搭載4つの2209

## 製品のパラメータ

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * ファームウェア: Klipper
    * 12-24V直流給電に対応、給電が安定、降圧モジュール不要
    * ドライバーの動作モード：UART、STEP/DIR
    * 拡張インターフェース：制御可能なファン*2、RGB*1、リミットスイッチ*3
