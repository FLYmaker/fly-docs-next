# CANブリッジ

CANバスはリアルタイムアプリケーション用のシリアル通信プロトコルバスであり、ツイストペア線を使用して信号を伝送することができます。世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内の様々なコンポーネント間の通信に使用され、重くて高価な配線束を代替します。このプロトコルの堅牢性により、その用途は自動化や産業用途にまで広がっています。CANプロトコルの特性には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシング、そしてエラーチェック機能が含まれます。
3Dプリンターでは、CANバスを使用してプリントヘッドをマザーボードに接続する配線の数を減らします。以前は十数本の線が必要でしたが、CANを使用することで4本の線だけで済み、配線数を大幅に減らし、配線の難易度を下げます。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードがCANを使用してマザーボードに接続

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24vの電源線に接続することに注意してください

* SUPER5 PROマザーボードに書き込むファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください
