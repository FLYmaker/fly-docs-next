# BLファームウェア

    :::tip

    * 出荷時に`bootloader`が既にインストールされているので、再度インストールする必要はありません。

    :::

## 書き込みモードに入る

    :::tip

    右側が`BOOTジャンパー`、左側が`RESETボタン`

    :::

    * 書き込みモードに入る方法は2種類あります
    * 第一種：メインボードが完全に電源を切った状態で、BOOTボタンを押しながらデータ通信用のケーブルをメインボードとあなたのデバイスに接続します
    * 第二種：データ通信用のケーブルをメインボードとあなたのデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押し、RSTボタンを離してからBOOTボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/flash/img/boot.png').default} size="100%" align="center" />
