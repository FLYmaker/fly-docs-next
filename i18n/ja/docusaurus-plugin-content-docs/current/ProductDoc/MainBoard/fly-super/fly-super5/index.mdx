# 製品仕様

    * 32ビット 550 MHz の ARM Cortex-M7 シリーズ STM32H723ZGT6 メインコントロールチップを採用
    * ファームウェア：Marlin 2.0/Reprap/Klipper
    * ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライブモードサポート：TMC：UART/SPI
    * 5つの高圧ドライブ位置、5つの独立したモータードライブをサポート
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1つの高圧INポート、自動ベッドレベリングセンサー対応：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * オプションのリミットスイッチ電源：5V/12V/24V
    * repapファームウェアを使用することで、板載ESP32 WiFiモジュールを利用でき、1.6m/sをサポート
    * ユニークなPWMファンモジュール設計で、損傷した場合に直接交換可能、顧客の修理難易度を低減、2つの2pinファンインターフェースが4pin PWMファンをサポート
    * PCBは2OZの銅厚を採用し、より大きな電流をサポート
    * 3つのADCインターフェース、最大3つのサーミスタをサポート
    * 7つのIOインターフェース、ベッドレベリングとリミットスイッチに使用可能
    * 3つのヒートロッドインターフェース、最大3つのヒートロッドをサポート
