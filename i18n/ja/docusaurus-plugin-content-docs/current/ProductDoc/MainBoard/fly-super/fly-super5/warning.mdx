# マザーボードの注意事項

## 電源インジケーターランプ

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/power.webp').default} size="80%" align="center" />
    * TYPE-C電源のみで他のマザーボードの電源を接続しない場合、最低でも2つのランプ`3.3V`と`5V`が点灯する
    * マザーボードの電源を接続した後、マザーボードの電源ランプは`3.3V`、`5V`、`12V`が点灯するはずです
    * ドライバーの電源0-2、ドライバーの電源3-7、ヒートベッド電源を接続すると、マザーボード左下の電源インジケーターランプが正常に点灯する

### 問題のトラブルシューティング


    :::danger 注意

    * 外部デバイス（ドライバー、リミットスイッチ、12864など）が何も接続されていない場合
    * TYPE-C電源のみで少なくとも`3.3V`と`5V`の2つのLEDが常に点灯する

    :::

    * `3.3V`と`5V`が点灯しない場合は、マザーボードの5Vジャンパーを確認してください
    * ジャンパーを交換しても同じで、typecの正常な電源供給を確認した場合は、カスタマーサポートに連絡してください
    * `3.3V`、`5V`のいずれかのランプが点滅している場合、チップのショートがあることを示し、カスタマーサポートに連絡する必要があります
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/5v.webp').default} size="60%" align="center" />

## ドライバー
    * ドライバーへの電源供給は48Vを使用できますが、すべてのドライバーが48Vを使用する必要があります
    * マザーボードは外部ドライバーをサポートしています     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定のチュートリアル</Button>

## リミットポート

    * リミットポートが使用できない場合、リミットポートを交換するか、ドライバーのリミットピンを切り取ることができます
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.webp').default} size="80%" align="center" />




## ファームウェアの書き込み

    * マザーボードはTFカードを使用して書き込みを行います
    * 以前はチュートリアルの問題によりDFUを使用して書き込みを行っていたため、ファームウェアを更新する必要がある場合は <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/bl">BLを再書き込み</Button>する必要があります
