# BLファームウェア

    :::tip

    * 出荷時にはすでに`ブートローダ`がプリインストールされているので、再インストールは不要です。

    :::
## 書き込みモードへの移行


    * 書き込みモードへの移行方法は2通りあります
    * 第一の方法：メインボードが完全に電源を切った状態で、BOOTを短絡し、データ伝送機能のあるケーブルをメインボードとあなたのデバイスに接続します
    * 第二の方法：データ伝送機能のあるケーブルをメインボードとあなたのデバイスに接続した後、BOOTを短絡してRSTキーを押し、RSTキーを離してからBOOTジャンパーを外します
