    :::tip

    * Super8 PROはデフォルトでTFカードを使用してファームウェアを書き込みます
    * 容量が`32G`以下のTFカードを準備してください
    * 無名のメモリーカードは書き込み失敗の原因となる可能性があるため、以下の推奨メモリーカードを使用してファームウェアを更新してください
    * TFカードの転送速度は`class10`級またはそれ以上が望ましく、SanDisk等のブランドのTFカードを使用することをお勧めします

    :::

# ファームウェアの書き込み説明

    :::tip

    * Super8 Proは出荷時にTFカードのブートローダーが予め書き込まれているため、再度書き込む必要はありません

    :::

## どのファームウェアが必要かを判断する方法

    :::tip

    * 注意：あなたは一種類のファームウェアだけを書き込む必要があります

    :::
    
    * Super8 PROは以下の方法でホストマシンに接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * メリット：通信が安定しており、信頼性が高い
        * デメリット：複数のUSBデバイスがあるとデータ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * メリット：複数のデバイスを接続でき、USBポートを占有しません
        * デメリット：`KlipperScreen`と一緒に使用すると、ホーミングタイムアウトなどのエラーが発生する可能性があります。
