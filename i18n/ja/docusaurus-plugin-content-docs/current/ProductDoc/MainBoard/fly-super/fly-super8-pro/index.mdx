# 製品仕様

    * 32ビット550 MHzのARM Cortex-M7シリーズSTM32H723ZGT6マイコンを採用
    * ファームウェア：Marlin 2.0/Reprap/Klipper
    * ドライバサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバモードサポート：TMC：UART/SPI
    * 8つの高圧ドライバ位置、8つの独立したモータードライバ、5つのエクストルーダー、10つのPWMファンに対応
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1つの高圧INポート、自動ベッドレベリングセンサー対応：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * 選択可能なリミットスイッチ電源：5V/12V/24V
    * repapファームウェアを使用すると、搭載されたESP32 WiFiモジュールを利用でき、1.6m/sをサポート
    * 独特のPWM Fan MOS板設計で、故障時には直接交換可能、顧客のメンテナンス難易度を低減、2つの2pinファンインターフェースが4pin PWMファンをサポート
    * PCBは2OZの銅厚を採用し、大きな電流をサポート可能
    * 6つのADCインターフェース、最大6つのサーミスタをサポート
    * 8つのIOインターフェース、調平とリミットに使用可能
    * 5つのヒータースティックインターフェース、最大5つのヒータースティックをサポート
    * 2つのPWMインターフェース
