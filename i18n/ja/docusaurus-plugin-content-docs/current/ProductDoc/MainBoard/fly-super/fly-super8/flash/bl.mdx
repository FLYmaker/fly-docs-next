# BLファームウェア

    :::tip

    * 出荷時には既に`bootloader`がプリインストールされているので、再インストールする必要はありません。

    :::

## 書き込みモードに入る

    * 書き込みモードに入る方法は2種類あります。
    * 第一の方法：メインボードが完全に電源を切った状態で、BOOTを短絡し、データ転送対応のケーブルをメインボードとデバイスに接続します。
    * 第二の方法：データ転送対応のケーブルをメインボードとデバイスに接続した後、BOOTを短絡し、RSTキーを押し、RSTキーを離し、最後にBOOTのジャンパを外します。
