# 製品仕様


    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ファームウェア：Marlin 2.0/Reprap/Klipper
    * ドライバサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライブモードサポート：TMC：UART/SPI
    * 8つの高圧ドライバ位置、8つの独立したモータドライバサポート、5つの押出機、10つのPWMファン
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1つの高圧INポート、自動ベッドレベリングセンサー対応：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * リミットスイッチ電源選択：5V/12V/24V
    * repapファームウェアを使用すると、内蔵ESP32 WiFiモジュールを使用でき、1.6m/sをサポート
    * 独自のPWM Fan MOSボードデザイン、故障時には直接交換可能、顧客の修理を簡素化、2つの2ピンファンインターフェースは4ピンのPWMファンをサポート
    * PCBは2OZ銅厚を採用し、より大きな電流をサポート
    * 6つのADCインターフェース、最大6つのサーミスタサポート
    * 8つのIOインターフェース、レベリングとリミットに使用可能
    * 5つのヒータ棒インターフェース、最大5つのヒータ棒サポート
    * 2つのPMWインターフェース
