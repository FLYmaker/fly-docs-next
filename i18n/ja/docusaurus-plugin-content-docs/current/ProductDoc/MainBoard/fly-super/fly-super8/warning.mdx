# マザーボードの注意事項

## 電源インジケータ

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/power.png').default} size="80%" align="center" />
    * TYPE-Cからの電源供給のみで他のマザーボード電源に接続しない場合、最低でも2つのランプ`3.3V`と`5V`が点灯します
    * マザーボードの電源に接続した後、マザーボードの電源ランプは`3.3V`、`5V`、`12V`と点灯するはずです
    * ドライバ電源0-2、ドライバ電源3-7、ヒートベッド電源を接続した後、マザーボード左下の電源インジケータが正常に点灯します

### 問題のトラブルシューティング


    :::danger 注意

    * 外付けデバイスが何も接続されていない場合（ドライバ、リミットスイッチ、12864など）
    * TYPE-Cからの電源供給のみで、少なくとも`3.3V`と`5V`の2つのLEDが常時点灯します

    :::

    * `3.3V`と`5V`が点灯しない場合は、マザーボードの5Vジャンパを確認してください
    * ジャンパを交換しても同じで、typecが正常に電源供給されていることを確認した場合は、カスタマーサービスに連絡してください
    * `3.3V`、`5V`、`12V`のいずれかのランプが点滅している場合、チップのショートが発生している可能性があり、カスタマーサービスに連絡してください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/5v.png').default} size="60%" align="center" />

## ドライバ

    * ドライバ電源に注意してください。ドライバ電源は2系統に分かれます。第一路はドライバ3-7に電源を供給し、第二路は0-2に供給します
    * マザーボードは外部ドライバをサポートします     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定のチュートリアル</Button>

## リミットスイッチポート

    * 最後のドライバポート（Drive7）を除き、他のドライバポートのリミットスイッチは直接リミットスイッチポートに接続されています
    * リミットスイッチポートが使用できない場合、リミットスイッチポートを交換するか、ドライバのリミットスイッチピンを切り取ることができます
