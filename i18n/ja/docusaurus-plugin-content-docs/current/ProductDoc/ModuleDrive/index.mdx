# 導入

    * 一部のドライバに関する注意事項


## ドライバのインストール

    * ENが左上に合わせてインストールする必要があることに注意してください
    * FLYマザーボードのドライバポートのENピンは全て左上にあります
    * FLYのDP5、PRO X10、MMUはTPドライバのみ使用できます
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

## ドライバの設定

    * TPドライバと通常のドライバの設定方法は同じです
    * FLY-2240ドライバはUART接続をサポートしていますが、抵抗を手動で変更する必要があります
    * ドライバのサンプリング抵抗に注意が必要です。誤った設定はモーター、ドライバの破損につながる可能性があります
