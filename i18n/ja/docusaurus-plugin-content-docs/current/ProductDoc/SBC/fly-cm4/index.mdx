# 紹介

      Fly-CM4はRaspberry Pi 3Bと4Bと完全に互換性があり、特にRaspberry Pi CM4の拡張に改良されています。この拡張ボードは、3Dプリンターに最適化された一連の機能を提供します。12-24Vの直流電源供給により、3Dプリンターの精密な制御と連続運転に重要な安定した電力供給が保証されます。FLY-TFTとFly-HDMIスクリーンのFPC接続機能をサポートし、より便利な配線方法を提供します。Canポートを提供することで、上位機が直接Can接続で下位機と通信することができ、これは現在人気のある方法です。

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-cm4/img/cm4.png').default} size="80%" align="center" />

## 特徴

    * Raspberry Pi CM4コアボードをサポート
    * 12V-24V直流電源供給をサポート
    * UTOCモジュールを内蔵
    * TFTとHDMIの一線通をサポート
    * DSI/SPIスクリーンインターフェースをサポート
    * CSI/USBカメラインターフェースをサポート
    * 40PIN拡張インターフェース、Raspberry Piと互換
