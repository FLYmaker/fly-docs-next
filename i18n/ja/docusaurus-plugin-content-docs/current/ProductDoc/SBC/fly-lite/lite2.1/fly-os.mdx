## システムの書き込み前準備

    * カードリーダー
    * TFカード1枚、容量は必ず16GB以上または128GB以下のTFカードで、TFカードの転送速度はclass10レベル以上である必要があります
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-lite2">システムイメージ</Button>をダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button>をダウンロード

## システムの書き込み

:::info
* Lite 2.1はシステムをTFカードにのみ書き込むことができ、M2WEは使用できません
* システムの書き込みが完了したら、TFカードをLite 2.1に取り付ける必要があります
:::
