# 導入

    Fly-Pi-lite2.1は初心者向けのKlipper上位機です
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.png').default} size="80%" align="center" />

## 特徴

    * CPU：高性能の全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GBのSDCARDに対応
    * WIFI: 内蔵2.4G WiFi
    * 周辺機器：Spi x1，URAT x2, USB x2，HDMI x1
    * インターフェース：FPC予約、HDMI + USB FPC一体通
    * システムはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグイン numpyとFLY-Toolsがプリインストールされており、複雑な操作（TFカードの購入が必要）なしで初心者でも簡単に始められます


## 更新機能

    * TFT V1インターフェースをTFT V2にアップグレード
    * リセットとシャットダウンピンの追加
    * 電源インジケーターランプの追加
    * BOOTボタンの削除、リセットボタンに変更
    * 配線最適化
