# 画面接続と設定
:::info

* lite2.1はシステムにSSH接続して以下のコマンドを入力した後にFly Toolsの画面設定を使用できます
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Toolsは上位機がネットワークに接続された後に上位機の`IP+:9999`でアクセスする必要があります
* 例えば`192.168.6.110:9999`

:::
