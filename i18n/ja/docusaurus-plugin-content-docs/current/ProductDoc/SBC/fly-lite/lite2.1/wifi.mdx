# Wi-Fiへの接続

## LITE 2.1へのアンテナの取り付け

    * 赤い矢印が指す場所がWi-Fiアンテナインターフェース
    * Wi-Fiアンテナ口は`ipex1`
    * LITE 2.1にアンテナを取り付けないと、信号が弱くてインターネットに接続できない可能性があります

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.png').default} size="100%" align="left" />  

* Wi-Fiに正常に接続されていない場合は、まずアンテナが接続されているか確認し、その後下記の方法でWi-Fiに接続してください
