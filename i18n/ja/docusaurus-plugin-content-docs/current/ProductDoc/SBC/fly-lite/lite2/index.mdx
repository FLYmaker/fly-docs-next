# 紹介

    Fly-Pi-lite2は初心者向けのKlipperホストマシンです

## 特徴

    * CPU：高性能の全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GB SDCARDに対応
    * WIFI: 内蔵2.4G WiFi
    * 周辺機器：Spi x1、URAT x2, USB x2、HDMI x1
    * インターフェース：FPC予約、HDMI + USB FPC一線通
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグイン numpyとFLY-Toolsがプリインストールされており、ソースの変更など複雑な操作は不要、初心者でも簡単に始められます（TFカードが必要）
