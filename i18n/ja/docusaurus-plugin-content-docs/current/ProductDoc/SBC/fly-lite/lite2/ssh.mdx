# 上位機へのSSH接続


* 機器への接続はシリアルポートまたはWIFIを使用できます
* **MobaXterm_Personal**や他の**SSH端末ツール**を事前に準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼かれていることを確認してください
* **上位機が正常に起動しない場合は、ドライバ、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後に、SSHを介してネットワーク接続を行い、ファームウェアのコンパイルを容易にしてください！！**
* **ファームウェアのコンパイルが必要な場合は、WIFIまたはイーサネット接続を通じて上位機に接続するチュートリアルを参照してください**

        ## ネットワークIPを使用して上位機にSSH接続する

        * 上位機が接続されている**ルーターの管理画面にログイン**してください
        * システムが**正常に起動している**ことを確認してください
        * **上位機のユーザー名とパスワードを自分で把握**していることを確認してください。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * メインボードが起動しネットワークに接続されている場合、IPを使用してSSHに接続できます
        * MobaXtermを開き、以下の設定を行います。IPは前述の確認方法で、またはルーターの管理画面で確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここで入力するのは上位機のユーザー名です**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初回接続時、このウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        **注意：ここで入力するパスワードは表示されません。入力が終わったら直接エンターを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly` 、パスワードは：`mellow` です
