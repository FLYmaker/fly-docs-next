# WIFI接続

## LITE 2にアンテナを取り付ける

    * 赤い矢印が指す場所がWIFIアンテナのインターフェース
    * WIFIアンテナポートは`ipex1`
    * LITE 2にアンテナを取り付けないと、信号が弱くてインターネットに接続できない可能性があります

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* WIFIが正常に接続されていない場合、アンテナが接続されているか確認し、その後下記の方法でWIFIに接続してください
