# 加速度計の使用

    :::tip

    * Mini Padは、加速度計を直接接続するか、ツールボードなどに内蔵されている加速度計を使用することができます。

    :::

## 板載インターフェースへの加速度計の接続

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.png').default} size="80%" align="left" />

    * 参考設定

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # 例
    ```
