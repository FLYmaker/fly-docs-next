# 紹介

    Fly-MINI-Padは、エントリーレベルのKlipperホストマシンです

## 特徴

    * CPU：高性能の全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GBのSDカードをサポート
    * WIFI :  内蔵2.4G WiFi
    * 周辺機器：SPI x1、USB x2、Type-C x1
    * インターフェース：3つのXH2.54-2Pインターフェースがそれぞれサポート、POWER-INは最高24Vの電源入力、CAN-BUSはCANバス接続、LI-POは3.7Vで保護付きのリチウムポリマーバッテリーの電源供給/充電インターフェース
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグイン numpyとFLY-Toolsがプリインストールされており、ソースの変更などの複雑な操作は不要、初心者でも簡単に使い始められます（TFカードの購入が必要です）
    * 画面は静電容量式/抵抗式に分かれ、解像度は320*480
