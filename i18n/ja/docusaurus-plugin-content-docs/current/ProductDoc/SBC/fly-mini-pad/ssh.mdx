# ホストマシンにSSHで接続する


* 機器にシリアル接続またはWIFI接続を使用することができます
* **MobaXterm_Personal**または他の**SSH端末ツール**を用意してください
* SDカードまたはM2WEに**対応するホストマシンのシステム**を事前に焼き込んでください
* **ホストマシンが正常に起動しない場合は、ドライバー、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後にネットワークを介してSSHに接続してファームウェアをコンパイルすることをお勧めします！！**
* **ファームウェアをコンパイルする必要がある場合は、WIFIまたはLANケーブルでホストマシンに接続する方法を参照してください**

        ## ネットワークIPでホストマシンのSSHに接続する

        * ホストマシンが接続されている**ルーターの管理画面**にログインしてください
        * システムが**正常に起動していること**を確認してください
        * **ホストマシンのユーザー名とパスワード**を自分で確認してください。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * マザーボードが起動し、ネットワークに接続されている場合、IPアドレスを使用してSSHに接続することができます
        * MobaXtermを開き、以下の設定を行います。IPは前述で確認したものを入力してください。また、ルーターの管理画面でも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **注意：ここではホストマシンのユーザー名を入力します**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * 初めて接続するときは、このウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * パスワードの入力を求められたら、ホストマシンのパスワードを入力し、エンターキーを押してください

        **注意：ここで入力したパスワードは表示されません。入力後、直接エンターキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly`、パスワードは：`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />

        * ログイン成功
