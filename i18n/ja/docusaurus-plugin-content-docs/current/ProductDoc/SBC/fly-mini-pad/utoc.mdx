# 板載UTOCの使用

:::danger 注意

2つのUTOICを使用することは推奨されません。書き込みと接続が非常に面倒です。

:::
    * 板載UTOCを使用しない場合は、以下のコマンドを実行してください。実行しても何の通知もありません
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * 元に戻す方法。実行しても何の通知もありません
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * MINI Padには既にUTOCが搭載されています
