# Wi-Fiに接続

## アンテナをMINI PADに取り付ける

    * 赤い矢印が指している場所がWi-Fiアンテナのインターフェース
    * Wi-Fiアンテナポートは`ipex1`
    * アンテナをMINI PADに取り付けないと、信号が弱くてネットワークに接続できない可能性があります
