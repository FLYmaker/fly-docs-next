## システム焼き付け前の準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上128GB以下、TFカードの転送速度はclass10以上でなければならない
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>のダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼き付けツール</Button>のダウンロード

## システム焼き付け

:::info
TFカードへの焼き付けとTFカードを通じてM2WEへの焼き付けは、どちらか一方を選択してください
:::

:::info
* 機械にM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムを焼き付けることを推奨します。USBを通じてM2Wに焼き付けるとドライバーの互換性などの問題で焼き付けが失敗する可能性があります。直接[メモリーカードからM2WEに焼き付け](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することを推奨します
* ダウンロードしたシステムイメージが対応しているかどうかを確認してください。対応していない場合は起動できません
:::
