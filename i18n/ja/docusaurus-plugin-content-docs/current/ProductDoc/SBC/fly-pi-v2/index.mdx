# 紹介

        新しいFLY-Pi V2ホストマシンをリリースできてうれしく思います。このボードはラズベリーパイと同じ取り付けサイズで、ラズベリーパイの代わりにホストマシンとして動作することができます。1GBのオンボードDDR3メモリを備えており、Armbianをベースにしたカスタム最適化された完全なLINUXデスクトップを実行することができます。このボードはKlipperおよびRRFファームウェアと一緒に使用でき、SHTツールボードを使用している場合でもKlipperスクリーンを動作させることができます。
    このホストマシンを設計する目的は、顧客がKlipperファームウェアまたはRRFファームウェアをより簡単に使用できるようにすることです。LINUXシステムイメージを最適化し、3Dプリンターに適したものにし、インストール操作も簡素化しました。初心者でも少し基礎知識があればすぐに学んで使用することができます。他のホストマシンボードに比べて学習コストを節約し、3Dプリントの楽しみにより集中できるようにしました（SDカード付きのセットを購入すると、Klipperファームウェアがプリインストールされており、電源を入れるだけで使用でき、時間と手間を省きます）。

## 特徴

    * ラズベリーパイ3Bおよび4Bと同じ形状
    * CPU：高性能全志H5チップ、4コア64ビットCortex-A53
    * GPU：高性能6コアMali 450、ピクセルフィルレートは2.7gpixel/s以上
    * RAM：1GB DDR3（GPUと共有）(512+512)
    * ROM：最大128GBのSDCARD対応
    * 周辺機器：Spi x2、UART x2、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN（オンボードUSB-CAN変換モジュール、即ちオンボードUTOC）
    * インターフェース：オンボードM.2インターフェースで、WIFIやeMMCなどの拡張が可能です（独自プロトコル、FLY π専用でないM.2デバイスは接続しないでください）
    * 40ピンのインターフェース、ラズベリーパイと互換
    * 12-24Vの直流電源に対応、電源供給がより安定
    * 5VファンのZH1.5インターフェースを予約
    * システムにklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグイン numpyおよびFLY-Toolsがプリインストールされており、ソースの変更や複雑な操作は不要、初心者でも簡単に始められます（TFカードまたはM2WEの購入が必要）
