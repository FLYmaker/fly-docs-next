# 上位機へのSSH接続


* シリアルポートまたはWIFIを使用してデバイスに接続することができます
* **MobaXterm_Personal**やその他の**SSH端末ツール**を事前に準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼かれていることを確認してください
* **上位機が正常に起動しない場合は、ドライバー、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後にネットワークを介してSSHに接続し、ファームウェアをコンパイルしやすくしてください！！**
* **ファームウェアをコンパイルする必要がある場合は、WIFIまたはイーサネットで上位機に接続するチュートリアルをご覧ください**

        ## ネットワークIPで上位機のSSHに接続する

        * 上位機が接続されている**ルーターの管理画面にログイン**してください
        * システムが**正常に起動していることを確認**してください
        * **上位機のユーザー名とパスワードを知っていることを確認**してください。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * マザーボードが起動し、ネットワークに接続されている場合、IPを使用してSSHに接続できます
        * MobaXtermを開き、以下の設定を行い、IPは前述の確認結果やルーターの管理画面で確認したものを入力してください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **注意：ここで入力するのは上位機のユーザー名です**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * 初めて接続するとこのウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * パスワードの入力が求められます。上位機のパスワードを入力してエンターを押してください

        **注意：ここに入力するパスワードは表示されません。入力が完了したら直接エンターを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly`、パスワードは：`mellow`です
