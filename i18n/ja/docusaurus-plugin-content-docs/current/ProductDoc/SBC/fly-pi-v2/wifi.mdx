# WIFIに接続


## 天線をPI V2に取り付け

    * 赤い矢印が指す場所がWIFI天線インターフェース
    * WIFI天線口は`ipex1`
    * 天線をPI V2に取り付けないと、信号が弱くてインターネットに接続できない可能性があります
