## システム書き込み前準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上128GB以下で、転送速度はclass10以上である必要があります
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button> をダウンロード

## システム書き込み

:::info
TFカードへの書き込みとTFカードを介したM2WEへの書き込みは、いずれか一方を使用するだけです
:::

:::info
* 機械にM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムを書き込むことを推奨します。USBを介してM2Wに書き込むと、ドライバーの互換性などの理由で失敗する可能性がありますので、[メモリーカードを介したM2WEへの書き込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することをお勧めします
* ダウンロードしたシステムイメージが対応しているかどうか注意してください。対応していないと起動できません
:::
