# 上位機にSSH接続

* シリアルポートまたはWIFIを使用してデバイスに接続できます
* **MobaXterm_Personal**または他の**SSH端末ツール**を用意してください
* SDカードまたはM2WEに**対応する上位機のシステムを焼き込む**ようにしてください
* **上位機が正常に起動しない場合は、ドライバー、リミットスイッチ、ファンなどの周辺機器を取り外してください！！！**
* **最後にネットワーク経由でSSHに接続してファームウェアのビルドを簡単に行えるようにしてください！！**
* **ファームウェアのビルドが必要な場合は、WIFIまたはイーサネット接続の上位機のチュートリアルを参照してください**

        ## ネットワークIPを使用して上位機にSSH接続

        * 上位機が接続されている**ルーターの管理画面にログインしてください**
        * システムが**正常に起動している**ことを確認してください
        * 上位機の**ユーザー名とパスワードを知っている**ことを確認してください。FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * メインボードが起動してネットワークに接続されている場合、IPを使用してSSHに接続できます
        * MobaXtermを開き、以下の設定を行います。IPは先ほど確認したものを入力するか、ルーターの管理画面で確認してください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **注意：ここでは第4ステップで上位機のユーザー名を入力します**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * 初回接続ではこのウィンドウがポップアップします。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * パスワードの入力が求められます。上位機のパスワードを入力してリターンキーを押します

        **注意：ここで入力されたパスワードは表示されません。入力が完了したら直接リターンキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly` 、パスワードは：`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />
