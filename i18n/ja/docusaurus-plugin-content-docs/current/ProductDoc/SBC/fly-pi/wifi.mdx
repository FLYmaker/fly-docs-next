# WIFIに接続


## アンテナをPIに取り付ける

    * 赤い矢印が指す場所がWIFIアンテナのインターフェースです
    * WIFIアンテナポートは`ipex1`です
    * アンテナをPIに取り付けないと、信号が弱くてインターネットに接続できない可能性があります
