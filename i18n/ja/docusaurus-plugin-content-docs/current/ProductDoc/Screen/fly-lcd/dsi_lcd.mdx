# DSI-LCD5

    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-lcd/img/dsi_lcd5.webp').default} size="50%" align="left" />

## 製品紹介

    * 画面サイズ：120.7mm*76.0mm
    * 画面表示領域：108mm*66.2mm
    * 解像度：800*480
    * 画面タイプ：IPS
    * 通信方式：DSI/Mipi
    * 搭載インターフェース：15 pin DSI / 22pin Mipi
    * 安定動作：0°-60°C
    * タッチタイプ：Capacitive（キャパシティブ）
    * ロジック電圧：3.3V
    * 重量：135.5g

        <TabItem value="ラズベリーパイ5BがMipiインターフェースを使用" default>

        :::danger 注意

        * ラズベリーパイ5のDSIとCSIインターフェースは現在MIPIと改名された
        * MIPIインターフェースはCSIカメラやDSI画面を直接設定して使用可能
        * 金指の方向に注意してください

        :::

        * ラズベリーパイ5には現在2つのMIPIインターフェースがあり、正面から見ると1と0のマークが見えます

        * DSI画面の接続


            * 1番インターフェースを使用する場合、下記のように画面を設定してください

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * 0番インターフェースを使用する場合、下記のように画面を設定してください

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
