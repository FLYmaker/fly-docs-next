# DSI-LCD5

    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-lcd/img/dsi_lcd5.png').default} size="50%" align="left" />

## 製品概要

    * スクリーンサイズ：120.7mm*76.0mm
    * スクリーンの表示領域：108mm*66.2mm
    * 解像度：800*480
    * スクリーンタイプ：IPS
    * 通信方式：DSI/Mipi
    * 搭載インターフェース：15ピンDSI / 22ピンMipi
    * 動作安定性：0°-60°C
    * タッチタイプ：Capacitive（静電容量式）
    * ロジック電圧：3.3V
    * 重量：135.5g

        <TabItem value="ラズベリーパイ5BのMipiインターフェースの使用" default>

        :::danger 重要事項

        * ラズベリーパイ5のDSIおよびCSIインターフェースは現在、MIPIと改名されています
        * MIPIインターフェースは設定によりCSIカメラまたはDSIスクリーンとして直接使用できます
        * 指標の方向に注意してください

        :::

        * ラズベリーパイ5には現在2つのMIPIインターフェースがあり、正面から見ると1と0のマークが見えます

        * DSIスクリーン接続


            * 1番インターフェースを使用する場合は以下のようにスクリーンを設定します

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * 0番インターフェースを使用する場合は以下のようにスクリーンを設定します

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
