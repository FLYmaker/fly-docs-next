# 起動画面変更
:::tip
    厳密に`解像度`および`ファイル名`に従って画像を作成してください、そうしないとシステムは画像を認識しません

    作成した画像をSDカード/USBメモリのルートディレクトリに置き、画面が起動している状態で挿入し、アップグレードをクリックしてください
:::
## 起動LOGO変更

    * 画像要求`1024*600`
    * 名前要求`boot_logo.JPG`
    * 105KBを超えてはならない



## 最初の画面画像変更

    * 画像要求`736*563`
    * 名前要求`print.png`
    * 45KBを超えてはならない



## プリント画面画像変更

    * 画像要求`355*565`
    * 名前要求`print_smail.png`
    * 45KBを超えてはならない
