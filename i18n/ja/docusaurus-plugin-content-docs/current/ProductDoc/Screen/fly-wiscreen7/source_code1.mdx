# プロジェクト構造の紹介


    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide6.webp').default} size="80%" align="left" />



## 紹介
#### Logicフォルダ
* 私たちの99%のUIコードプログラミングはlogicフォルダ内で行われます。他のフォルダは無視してください
* gcodeLogic.ccはmain.cに似ており、99%のUIコードがここに存在します。他のファイルは特に注目する必要がありません

#### i18nフォルダ
* 各国語の翻訳テーブルを保存

#### resourcesフォルダ
* UIで使用する画像リソースを保存

#### uiフォルダ
* UIインターフェースファイルを保存、ダブルクリックで表示
* 左下の矢印の指す「小さな目」をダブルクリックすると、隠れたインターフェースを開くことができます
