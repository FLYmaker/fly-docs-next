##   更新
    * 1.ダウンロードしたファイルを`update.img`に名前を変更します。
    * 2.`update.img`ファイルをSDカード/USBメモリに入れてスクリーンに挿入します。スクリーンが起動している状態で挿入することができます。
    * 3.スクリーンを選択し、更新をクリックしてアップデートを完了させます。
