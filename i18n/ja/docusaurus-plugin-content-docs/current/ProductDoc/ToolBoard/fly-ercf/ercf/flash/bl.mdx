# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * ERCFはCAN、USBをサポート
    * ここではCANファームウェアを500Kと1Mの速度で提供します
    :::

## 書き込みモードに入る

    :::tip

    囲みは`BOOTボタン`

    :::

    * 書き込みモードに入るためには、完全に基板の電源を切った状態で、BOOTボタンを押しながらデータ転送が可能なケーブルを基板と上位機に接続します。
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.webp').default} size="100%" align="center" />

            4. ファームウェアの書き込み

            :::tip

            どれか一つのファームウェアを書き込むだけです

            :::

            * 以下は500K CAN速度のファームウェア書き込みコマンド
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * 以下は1M CAN速度のファームウェア書き込みコマンド
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * 以下はUSBのファームウェア書き込みコマンド
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. 下記のメッセージが表示されれば書き込み成功です

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            書き込み完了後、基板の電源を一度完全に切る必要があります。

            :::
