# USB ファームウェアのビルド

    :::tip

    * ERCF出荷時にはCAN接続のKatapultファームウェアがプリインストールされています
    * USBファームウェアの書き込みが必要な場合は、USB用のKatapultファームウェアを書き込む必要があります
    * 一度書き込んだ後は再度書き込む必要はありません
    * <Button variant="contained" disableElevation href="./bl">Katapultファームウェアの書き込み</Button>

    :::

## ビルドの開始

    <SSH />

    <MakeOperate/>

    <USB />

    * 起動時に設定するGPIOピンを選択し、「gpio17」と入力します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押し、「**Save configuration**」が表示されたら、`Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインに戻ります
    * `make -j4` を入力してビルドを開始します。少し時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されれば、ビルドが成功したことになります

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
