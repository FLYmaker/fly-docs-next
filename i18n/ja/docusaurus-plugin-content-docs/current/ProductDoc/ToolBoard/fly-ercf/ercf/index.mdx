# 製品紹介

    FLY-ERCFは、RP2040ベースの3Dプリンタ多色制御ボードで、主にEnraged Rabbitプロジェクトに使用されます。また、他の二軸ステッピングモータ制御にも使用できます。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.png').default} size="100%" align="left" />

## 製品特徴

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 2つのドライバースロット、UARTおよびSPIドライバーをサポート
    * オリジナルのERCF EASYボードと互換
    * 極めて小型化した4層ボードデザイン
    * 板載2.54MMデュアルロー7PIN拡張ポート、リミットスイッチ等に拡張可能
    * 12-24V電圧サポート
    * CANバスとRS232接続をサポート、データ伝送が安定、遅延が小さく、接続が安定
    * CANインターフェースはXT30（2+2）を使用、15A電流サポート

## 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * ファームウェア: Klipper
    * 入力電圧: DC12V-24V
    * インターフェース: 2.54MMデュアルロー7PIN拡張ポート、SERVO X1,ENDSTOP X2,EXTRA X1
    * ドライバー：UART/SPIおよびSTEP/DIRをサポート

    * 12-24V電圧サポート
    * CANバスをサポート、データ伝送が安定、遅延が小さく、接続が安定
    * CANインターフェースはXT30（2+2）を使用、15A電流サポート
