<Tabs>
    <TabItem value="can" label="CANBus UUIDの取得">

    * ID検索前に、XT30データケーブルを使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたマザーボードに接続します。以下は参考になる接続図です
    * ディップスイッチが正しい位置に設定されているか確認してください
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * CAN HとCAN Lが断電状態でテスターで測定した抵抗が約`60Ω`であることを確認してください。`120Ω`の場合は`120Ω`のジャンパーを接続する必要があります。`40Ω`の場合はジャンパーを取り外す必要があります
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.png').default} size="100%" align="center" />
        <GetCanbusUUID />

    </TabItem>

    * ディップスイッチが正しい位置に設定されているか確認してください
    <ImageView image={require('@site/docs/General/get-id/img/utor.png').default} size="80%" align="left" />
