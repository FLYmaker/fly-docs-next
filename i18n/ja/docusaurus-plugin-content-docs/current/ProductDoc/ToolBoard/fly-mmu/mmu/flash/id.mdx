# マザーボードIDの読み取り

    ページ翻訳時に設定を変更しないでください

    :::

    :::warning 注意事項

    ページ翻訳時に設定を変更しないでください

    :::

    :::danger 注意

    ページ翻訳時に設定を変更しないでください

    :::

<Tabs>
    <TabItem value="usb" label="USB IDの取得" default>
        
    * データ転送機能を持つデータケーブルを準備してください。特に長い場合は磁気リング付きのケーブルを使用することをお勧めします
    
        <GetUsbID />
        
    </TabItem>
    <TabItem value="can" label="CANBus UUIDの取得">

    * ID検索前に、XT30データケーブルを使用してツールボードをUTOCまたはブリッジファームウェアを書き込んだマザーボードに接続します。以下は参考接続図
    * ディップスイッチが適切な位置にあるか確認してください
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * CAN HとCAN Lを電源オフ状態でマルチメータで測定し、抵抗値が`60Ω`前後であることを確認してください。`120Ω`の場合は`120Ω`のジャンパを接続し、`40Ω`前後の場合はジャンパを取り除いてください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.png').default} size="50%" align="center" />
        <GetCanbusUUID />

    </TabItem>
    <TabItem value="rs232" label="RS232 IDの取得">

    * ID検索前に、XT30データケーブルを使用してツールボードをUTORまたはG2Tに接続します。以下は参考接続図
    * ディップスイッチが適切な位置にあるか確認してください
        <ImageView image={require('@site/docs/General/get-id/img/utor.png').default} size="80%" align="left" />              
        <GetRS232ID />

    </TabItem>
