# USB ファームウェアのコンパイル

    :::tip

    * MMUは出荷時にCAN接続のKatapultファームウェアがプリインストールされています
    * USBファームウェアを書き込む場合は、USB用のKatapultファームウェアを書き込む必要があります
    * 一度書き込み後、再書き込みは不要です
    * <Button variant="contained" disableElevation href="./bl">Katapultファームウェアの書き込み</Button>

    :::

## コンパイルの開始

    <SSH />

    <MakeOperate/>

    <USB />

    * GPIOピンの選択（マイクロコントローラ起動時に設定する）、入力：!PA15 注意：!は大文字です
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押して **Save configuration** が表示されたら、次に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインインターフェースに戻ったはずです
    * `make -j4` を入力してコンパイルを開始、時間がかかる場合があります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
