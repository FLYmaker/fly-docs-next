# 製品紹介

    Fly-MMUは多機能ツールボードで、高性能な制御基板です。3Dプリンターと自動化装置用に設計されており、強力なH723マイクロコントローラを統合し、最大4つのTMCステッピングモータードライバをサポートし、高圧駆動にも対応します。また、11個のリミットスイッチ入力があり、FPCを通じて16個のリミットを拡張できます。さらに、4つのサーボ制御出力を備えています。このツールボードはKlipperファームウェアと互換性があり、CANバスとRS232接続オプションを提供し、柔軟な通信と制御スキームを実現します。

## 製品特徴

    * 主制御には、32ビット550MHzのARM Cortex-M7シリーズSTM32H723主制御チップを使用
    * 4つのドライバ拡張をサポートし、最大48Vの高圧TP5160ドライバに対応
    * 複数のリミット拡張、基板に11個のリミットポートを搭載し、FPCを通じて16個のリミットポートを拡張可能
    * Klipperファームウェアを使用してUSB、CAN、RS232接続をサポート
    * CAN拡張ポートを予約、CANを使用する場合、他の機器にCAN拡張ポートで接続可能
    * CANバスとRS232接続をサポートし、データ転送がより安定し、遅延が少なく、接続が安定
    * CANインターフェースはXT30（2+2）を使用し、15Aの電流に対応

## 製品仕様

    * MCU: STM32H723
    * ファームウェア: Klipper CAN/RS232
    * 電源入力：HVIN:12V/48VとVIN:12V/24V
    * 拡張インターフェース：11個のリミット、4つのサーボ、1つのRGB、1つのCAN拡張ポート、FPC拡張ポートで16個のリミットを拡張可能
    * ドライバ：TP2209およびTP5160に対応し、他のFLYのTPドライバも使用可能
    * 12-24Vの電圧をサポート
    * CANバスとRS232をサポートし、データ転送がより安定し、遅延が少なく、接続が安定
    * CANインターフェースはXT30（2+2）を使用し、15Aの電流に対応
