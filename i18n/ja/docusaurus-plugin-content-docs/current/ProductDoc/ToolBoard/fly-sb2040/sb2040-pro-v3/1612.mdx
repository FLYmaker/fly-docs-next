# Eddyの使用方法

    :::info

    * ベッドスキャンには専用のZリミットを使用することを推奨します。ベッドスキャンをZリミットとして使用することはお勧めしません
    * Eddyをインストールする際、ノズルの位置はノズルの上に1-2mm以上離れている必要があります
    * Eddyの周波数を調整する際、コイルPCBはベッドから少なくとも5mm離す必要があります
    * ベッド全体が常温でテストしてください。テスト中はベッドをオンにしないでください

    :::

## 参照設定と使用説明

### ldc16121612デフォルト周波数変更
    この方法はsht36-v3/SB2040-v3またはSB2040-Pro-v3専用です
    Mobaxtermツールを使用してコマンドを実行できます

    ダウンロードURL:https://mobaxterm.mobatek.net/download.html
    
    [詳細についてはこちらを参考にしてください](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    最初のコマンドを実行し、Enterを押します

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    二つ目のコマンドを実行し、Enterを押します
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    置換が成功すると`LDC1612_FREQ = 40000000`を含む行が返されます。置換が成功したことになります。何も返されない場合は置換が失敗したことを示します。
    またはこのコマンドで成功したかどうかを確認できます
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   温かいアドバイス：変更が完了したらこのコマンドでklipperサービスを再起動し、再度eddyの周波数を校正してください
   ```cfg
   systemctl restart klipper
   ```
   eddyの校正コマンド
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### ベッドスキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意してください、現在外部1612モジュールは発売されていませんので、この設定は使用できません
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補償設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の`horizontal_move_z`パラメータは、プラットフォームをスキャンする際にEddyがプラットフォームにできるだけ近づくように2に設定することを推奨します

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
