# Eddyの使用

    :::info

    * 注意：ベッドスキャンには専用のZリミットを使用し、ベッドスキャンをZリミットとして使用しないことをお勧めします
    * Eddyをインストールする際、ノズルの位置に対して底部は少なくともノズルの上部1-2 mmにしてください
    * Eddyの周波数を較正する際、コイルPCBはベッドから少なくとも5mm離しておいてください
    * すべてのベッドを常温でテストし、テスト中はベッドを開かないでください

    :::

## 参照設定と使用説明

    ### ベッドスキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意：現在、外部1612モジュールは出ていないため、この設定は一時的に使用できません
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の`horizontal_move_z`パラメータは、ベッドスキャン時にEddyができるだけプラットフォームに近づくように、2に設定することをお勧めします

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
