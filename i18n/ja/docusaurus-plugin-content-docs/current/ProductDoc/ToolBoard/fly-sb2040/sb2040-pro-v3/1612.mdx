# Eddyの使用

    :::info

    * ベッドスキャンを行う際は、Zリミットを別途設定することをお勧めします。ベッドスキャンをZリミットとして使用することはお勧めしません。
    * Eddyを取り付ける際、ノズルより1-2mm上に位置するようにしてください。
    * Eddyの周波数を校正する際、コイルPCBはベッドから少なくとも5mm離れている必要があります。
    * ベッドをテストする際は、常温で行い、ベッドの電源は入れないでください。

    :::

## 設定と使用説明

    ### ベッドスキャンの設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 現在、外部1612モジュールは提供されていないため、この設定は使用できません。
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正の設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh]の`horizontal_move_z`パラメータを2に設定することをお勧めします。これにより、スキャン中にEddyがプラットフォームにより近づきます。

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
