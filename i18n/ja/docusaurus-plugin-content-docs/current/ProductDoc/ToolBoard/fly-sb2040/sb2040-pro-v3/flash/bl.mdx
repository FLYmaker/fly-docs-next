# Katapultファームウェア

## Katapultファームウェア説明

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3はCANとRS232をサポート
    * ここではCANファームウェアのみ提供され、500Kと1Mの速度があります
    :::

## 書き込みモードへの移行

    :::tip

    左側が`RESETボタン`で、右側が`BOOTボタン`

    :::

    * 書き込みモードへの移行方法は2つあります
    * 第一の方法：主板が完全に電源オフの状態で、BOOTボタンを押しながらデータ転送ケーブルの一端を主板に、もう一端をあなたのデバイスに接続します
    * 第二の方法：データ転送ケーブルの一端を主板に、もう一端をあなたのデバイスに接続した後、BOOTボタンを押しながらRSTボタンを押して、RSTボタンを離し、最後にBOOTボタンを離します
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
