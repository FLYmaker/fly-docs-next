# 1. 製品紹介

    FLY SB2040 PRO V3は、広州市メロン電子科技有限公司がVORON Stealthburberプリントヘッド用に開発したツールボードです。このツールボードを使用すると、主ボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SB2040はstealthburnerおよびafterburnerに適しており、CANトランスシーバー、RS232トランスシーバー、USBポート、TMC2240ドライバー、LIS2DW加速度センサー、3つの制御可能なファンなど、他の機能を統合したツールボードです。このツールボードはKlipperとRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/img/sb2040.webp').default} size="100%" align="center" />


## 1.1 製品特徴

    * ブートボタンを予約し、TypeCを通じてブートモードに入りファームウェアを更新可能
    * Klipperファームウェア、RRFファームウェアのサポート
    * CANモード、RS232モード、RRF CANモードのサポート
    * RS232モードでは、より高い伝送速度と安定性を備えています。CANモードでは、CAN拡張をサポートし、CAN拡張インターフェースを予約
    * 温度センサーサポート： 搭載されたMAX31865チップがPT100とPT1000温度センサーに互換
    * 搭載LIS2DW加速度センサーとLDC1612センサー
    * RGBインターフェースを予約し、2240ドライバーを搭載。klicky、Voron Tap、PL08nなどのさまざまなレベリングセンサーをサポート
    * 12-24V電圧、15A電流入力対応、加熱棒ポートを増大、定格電流10A

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRFCANモード間の切り替えをサポート
    * 新しいRS232モードを追加し、より高い伝送速度と安定性を備え、デバッグも簡単に
    * RRF CANモードを追加し、RRFツールボードに切り替えることが可能、適用性を強化
    * CAN拡張インターフェースを追加し、CANデバイスの拡張が容易
    * 終端抵抗のディップスイッチを追加し、120Rの終端抵抗をディップスイッチで選択可能、ジャンパーピンは不要
    * 1メートルの電源/信号接続ケーブルを追加し、接続ケーブルはツイストペア設計で、耐干渉性が強化

## 1.3 製品仕様


    * MCU：Raspberry Pi rp2040、Dual core ARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクタを使用し、15A電流、ピーク30Aをサポート
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * 搭載センサー：LIS2DWとオプションのldc1612およびmax31865
    * ヒータインターフェース: ヒーター（HEAT0）
    * 拡張インターフェース: RGB、EndStop、PT100/PT1000、XH2.54-2P CANポート
    * ファンインターフェース：1路DCファン、2路DCファンまたはPWMファン
    * モータードライバー：搭載TMC2240
    * 温度センサー：1路100K NTCまたはPT1000（TH0）、1路PT100/PT1000オプション
