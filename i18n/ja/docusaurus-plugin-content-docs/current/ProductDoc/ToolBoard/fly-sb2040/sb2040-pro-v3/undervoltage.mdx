# ドライバーの低電圧対策

## 方法一

    :::danger

    * 電源を交換する
    * 電源フィルターを追加する
    * モーターを交換して判断する
    * モーターの外郭をツールボードのGNDに接続する
    * エクストルーダードライバーの設定に`driver_TPFD: 0`を追加する

    :::

## 方法二
    * SSHで上位機に接続する
    * 以下のコマンドを入力する
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Klipperを更新するとこのステップが元に戻る可能性があります
    :::
