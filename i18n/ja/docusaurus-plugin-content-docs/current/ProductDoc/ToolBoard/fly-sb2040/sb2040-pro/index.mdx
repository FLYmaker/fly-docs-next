# 製品紹介

    FLY SB2040 PROは、広州メロン電子科技有限公司がVORON Stealthburberプリントヘッド用に開発したツールボードです。このツールボードを使用することで、プリントヘッドとメインボード間の複雑な配線を4本の線で置き換え、レイアウトを最適化できます。SB2040はstealthburnerおよびafterburnerに適しており、CANトランスシーバー、USBポート、TMC2240ドライバー、ADXL345加速度センサー、3つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperおよびRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/img/sb2040.png').default} size="100%" align="center" />


## 製品の特徴

    * Bootボタンを予約しており、ユーザーはTypeCを介してBootモードに入り、ファームウェアを更新できます
    * Klipperファームウェアをサポート
    * 温度センサーサポート： 内蔵MAX31865チップ、PT100互換
    * 内蔵ADXL345加速度センサー
    * RGBインターフェースを予約し、内蔵2240ドライバーを搭載しており、多種のレベリングセンサーに対応、例えばklicky、Voron Tap、PL08nなど
    * 12-24Vの電圧、15Aの電流入力に対応し、ヒーターポートを拡大、定格電流10A


## 製品仕様


    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクタを使用し、15Aの電流、ピーク30Aをサポート
    * ファームウェア：Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * 内蔵センサー：ADXL345とオプションのMAX31865
    * ヒーターインターフェース: ヒーター（HEAT0）
    * 拡張インターフェース: 制御可能なファン*3、RGB*1、リミットスイッチ*3、ヒートチューブ*1、サーモ*1
    * ファンインターフェース：3つの制御可能なファン
    * モータードライバー：内蔵TMC2240
