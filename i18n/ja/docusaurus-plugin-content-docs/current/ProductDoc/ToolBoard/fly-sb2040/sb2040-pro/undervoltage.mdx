# ドライバーのアンダーボルト解決方法

## 方法一

    :::danger

    * 電源を交換する
    * 電源フィルターを追加する
    * モーターの交換を判断する
    * モーターの外殻をツールボードのGNDに接続する
    * エクストルーダードライバーの設定に`driver_TPFD: 0`を追加する

    :::

## 方法二
    * SSHを通じてホストPCに接続する
    * 下記のコマンドを入力する
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Klipperを更新すると、この手順が元に戻る可能性があります
    :::
