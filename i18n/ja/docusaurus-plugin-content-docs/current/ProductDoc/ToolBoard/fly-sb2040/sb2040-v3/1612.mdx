# Eddyの使用方法

    :::info

    * ご注意ください。スキャンベッドは単独のZ制限を使用することをお勧めします。スキャンベッドをZ制限として使用することはお勧めしません。
    * Eddyをインストールする際、底部はノズルの上に少なくとも1-2mmの位置にする必要があります。
    * Eddyの周波数を調整する際、コイルPCBはヒートベッドから少なくとも5mm離れている必要があります。
    * すべてのヒートベッドが常温でテストされていることを確認してください。テスト中はヒートベッドを開かないでください。

    :::

## 参考設定と使用説明

    ### スキャンベッド設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * ご注意ください。現在、外部1612モジュールは出ていません。この設定は一時的に使用できません。
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補償設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の `horizontal_move_z` パラメータは、プラットフォームをスキャンする際に、Eddyができるだけプラットフォームに近づくように、2に設定することをお勧めします。

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
