# Katapultファームウェア

## Katapultファームウェア説明

    <CANBOOT />

    :::tip

    * SB2040_V3はCANとRS232をサポートしています
    * こちらではCANファームウェアを500Kと1Mの速度で提供しています
    :::

## 書き込みモードに入る

    :::tip

    左側が`リセットボタン`、右側が`ブートボタン`です

    :::

    * 書き込みモードに入る方法は2つあります
    * 第一種：基板が完全に電源が切れている状態で、BOOTボタンを押しながら、データ転送可能なデータケーブルの一端を基板に、もう一端をあなたのデバイスに接続します
    * 第二種：データ転送可能なデータケーブルの一端を基板に、もう一端をあなたのデバイスに接続した後、BOOTボタンを押し続け、RSTボタンを押してからRSTボタンを離し、最後にBOOTボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.webp').default} size="100%" align="center" />
