# 1. 製品紹介

    FLY SB2040 V3は、広州メロンエレクトロニクス科技有限公司がVORON Stealthburberプリントヘッド向けに開発したツールボードです。このツールボードを使用することで、プリントヘッドとメインボード間の複雑な配線を4本の線に置き換え、レイアウトを最適化できます。SB2040はstealthburnerとafterburnerに適しており、CANトランスシーバー、RS232トランスシーバー、USBポート、TMC2209ドライバー、LIS2DW加速度センサー、3つの制御可能なファンなど、他の機能も統合しています。このツールボードはKlipperおよびRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/sb2040.png').default} size="100%" align="center" />


## 1.1 製品の特徴

    * Bootボタンを予約し、ユーザーはTypeCを通じてBootモードに入ってファームウェアを更新できます
    * Klipperファームウェア、RRFファームウェアをサポート
    * CANモード、RS232モード、RRF CANモードをサポート
    * RS232モードでは、より高い転送速度と強力な安定性を備えています。CANモードでは、CAN拡張をサポートし、CAN拡張インターフェースを予約しています
    * 温度センサーサポート： 内蔵MAX31865チップ、PT100およびPT1000温度センサーと互換
    * 内蔵LIS2DW加速度センサーとLDC1612センサー
    * RGBインターフェースを予約し、内蔵2209ドライバー、さまざまな調平センサーをサポート、例えばklicky、Voron Tap、PL08nなど
    * 12-24V電圧、15A電流入力に対応し、ヒートロッドポートを増設、定格電流10A

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポート
    * 新しいRS232モードを追加し、より高い転送速度と強力な安定性、デバッグの簡便性を提供
    * 新しいRRF CANモードを追加し、RRFツールボードに切り替え可能、適用範囲を強化
    * CAN拡張インターフェースを追加し、CANデバイスの拡張が容易
    * 終端抵抗のディップスイッチを追加し、120Rの終端抵抗をディップスイッチで選択可能、ジャンパーは必要ありません
    * 1メートルの電源/信号接続線を追加し、接続線はツイストペア設計で、抗干渉性能が向上

## 1.3 製品仕様


    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクタを使用し、15A電流をサポート、ピーク30A
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * 論理電圧：DC3.3V
    * 内蔵センサー：LIS2DWとオプションのldc1612とmax31865
    * ヒートロッドインターフェース: ヒートロッド（HEAT0）
    * 拡張インターフェース: RGB、EndStop、PT100/PT1000、XH2.54-2P CANポート
    * ファンインターフェース：1つのDCファン、2つのDCファンまたはPWMファン
    * モータードライバー：内蔵TMC2209
    * 温度センサー：1つの100K NTCまたはPT1000（TH0）、1つのPT100/PT1000オプション
