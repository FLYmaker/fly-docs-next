# 製品紹介

    FLY SHT36/42は、広州メロン電子技術有限公司が3Dプリンターヘッドのために設計したノズルツールボードです。このツールボードを使用することで、メインボードとプリントヘッド間の複雑な配線を4本の線で置き換え、レイアウトを最適化できます。SHT36は36ステッピングモーターに適し、SHT42は42ステッピングモーターに適しています。CANトランスシーバー、USBポート、TMC2209ドライバー、ADXL345加速度センサー、2つの制御可能なファンなど、他の機能を統合したツールボードです。このツールボードはKlipperファームウェアに適しています。

    * 極限まで圧縮されたサイズ、6層基板設計を採用
    * CANバス接続をサポートし、データ転送がより安定し、遅延が少なく、接続が安定
    * 内蔵TMC2209ドライバー、UARTモードを使用し、押出機に使用可能
    * 内蔵ADXL345加速度センサー
    * PT100（MAX31865）サポート
    * 交換可能なファンモジュール設計
    * BLTOUCH、PL08Nなどの調平センサー対応、XYリミットスイッチ統合
    * 12-24V電圧入力対応 
    * MCU: STM32F072
    * 通信: CANBUS, USB
    * インターフェース：制御可能ファン*2、RGB*1、リミット*3、Probe*1、Servo*1、ヒーター*1、サーミスター*1、PT100*1

## 製品仕様

    1. 外形寸法：51.27mm*36.26mm（SHT36）、42mm*42mm（SHT42）
    2. 取り付け寸法：穴間隔43.84mm、M3取り付け孔径
    3. プロセッサー：STM32F072C8T6
    4. 入力電圧：DC12-24V 5A
    5. ヒーターの最大出力電流：5A
    6. ファンインターフェースの最大出力電流1A
