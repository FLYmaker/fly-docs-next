# 加速度計の使用


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Klipperの更新

    * Klipperは2024年11月13日にLIS3DH加速度計のサポートを追加しました。LIS3DH加速度計を使用するにはKlipperを更新する必要があります。
    * 上位機にネットワーク接続後、下記のコードを入力してください（ネットワークの問題で更新に失敗する場合があります）
        ```bash
        cd ~/klipper 
        git pull
        ```
## 設定の追加


    ```cfg
    #####################################################################
    #                       LIS3DH加速度計の設定（必要に応じて有効にしてください）
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  #ここに共振測定時のノズルの座標を設定します


```
