# 製品紹介

    FLY-SHT36-LIS3DHは、広州镁伦電子科技有限公司が3Dプリンタの押出ヘッド向けに設計したノズルツールボードです。このツールボードを使用すると、プリンタヘッドとメインボード間の複雑な配線を4本の線で置き換え、レイアウトを最適化できます。FLY-SHT36-LIS3DHは36ステップモータに適しており、CANトランスミッタ、USBポート、TMC2209ドライバ、LIS3DH加速度センサー、2つの制御可能なファンなどの他の機能を統合したツールボードです。このツールボードはKlipperファームウェアに適しています。

    * 体積圧縮が極端に小さく、6層ボード設計を採用
    * CANバス接続をサポートし、データ転送が安定し、遅延が少なく、接続が安定
    * 搭載TMC2209ドライバ、UARTモードを使用し、押出機に使用可能
    * 搭載ADXL345加速度センサー
    * PT100（MAX31865）サポート
    * 交換可能なファンモジュール設計
    * BLTOUCH、PL08Nなどのレベリングセンサーをサポート、XYリミットスイッチを統合
    * 12-24V電圧入力サポート 
    * MCU: STM32F072
    * 通信: CANBUS, USB
