# Katapultファームウェア

## Katapultファームウェア説明

    <CANBOOT />

    :::tip

    * SHT36 PROはCANに対応しています
    * ここでは、500Kと1Mの速度で提供されるCANファームウェアのみを提供します
    :::

## ファームウェア書き込みモードに入る

    :::tip

    囲みは`BOOTボタン`です

    :::

    * メインボードが完全に電源を切った状態で、BOOTボタンを短絡し、データ伝送用のケーブルの一方をメインボードに、もう一方をあなたのデバイスに接続します
    * 次に、BOOTの短絡を解除します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.webp').default} size="80%" align="center" />

            4. ファームウェアの書き込み

            :::tip

            どれか一つのファームウェアを書き込むだけで大丈夫です

            :::

            * 以下は500K CAN速度のファームウェア書き込みコマンドです
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * 以下は1M CAN速度のファームウェア書き込みコマンドです
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. 以下のメッセージが表示されれば書き込み成功です

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            書き込みが完了したら、メインボードを完全に電源を切る必要があります。

            :::

        </TabItem>
