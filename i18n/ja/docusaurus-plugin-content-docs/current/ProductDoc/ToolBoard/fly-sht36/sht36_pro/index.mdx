# 製品紹介

    FLY-SHT36-PROは、広州市メロン電子科技有限公司が3Dプリンターヘッド用に設計したノズルツールボードです。このツールボードを使用することで、メインボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化できます。FLY-SHT36-PROは36ステッピングモーターに適しており、CANトランシーバー、USBポート、TMC5160ドライバー、ADXL345加速度センサー、2つの制御可能なファンなど、他の機能を統合したツールボードです。

    * 体積が非常に小さく、6層基板設計を採用
    * CANバス接続をサポートし、データ伝送が安定し、遅延が少なく、接続が安定
    * 搭載されたTMC5160は押出機に使用可能
    * 搭載されたADXL345加速度センサー
    * PT100（MAX31865）をサポート
    * 交換可能なファンMOS設計
    * BLTOUCH、PL08Nなどの調整センサーをサポートし、XYリミットスイッチを統合
    * 12-24V電圧入力に対応
    * MCU: STM32F072
    * 通信: CANBUS, USB

## 製品仕様

    1. Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポート
    2. 接続インターフェースにはXT30(2+2)コネクタを使用し、15Aの電流をサポート、ピーク時には30A
    3. MCU：Raspberry Pi rp2040、Dual core ARM Cortex-M0+@133MHz
    4. 入力電圧：DC12-24V 5A
    5. ヒーター最大出力電流：5A
    6. ファンインターフェース最大出力電流1A
