# 072バージョンCANファームウェアのコンパイル

## コンパイルの開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * **GPIOピンのマイクロコントローラ起動時の設定**を選択し、入力：!PC13（大文字に注意）
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/072.png').default} size="100%" align="center" />
    * `Q`キーを押して**Save configuration**が表示されたら、`Y`キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインに戻ります
    * `make -j4`を入力してコンパイルを開始します。少し時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ファームウェアの書き込み

    * ファームウェアを書き込む前に、XT30データケーブルを使用してツールボードをUTOCまたはブリッジファームウェアを書き込んだメインボードに接続してください。以下は接続例です
    * ダイポールが正しい位置にあるかどうかを確認してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.png').default} size="80%" align="left" />
    * 電源を切った状態でCAN HとCAN Lの間の抵抗をテスターで測定してください。約`60Ω`の場合、`120Ω`のジャンパーを接続してください。約`40Ω`の場合、ジャンパーを取り外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.png').default} size="50%" align="center" />

    <CANFLASH />
