# 導入

## 製品導入

    FLY SHT36 V2は、広州マグネトロン電子科技有限公司が3Dプリンタの押出ヘッド用に設計したノズルツールボードです。このツールボードを使用することで、メインボードとプリントヘッド間の複雑な配線を4本の線で置き換え、レイアウトを最適化できます。SHT36 V2は36ステッピングモータに適しており、CANトランシーバ、USBポート、TMC2209ドライバ、ADXL345加速度センサー、2つの制御可能なファンなど、他の機能を統合したツールボードです。このツールボードはKlipperファームウェアに適しています。

    * 極限まで圧縮されたサイズ、6層基板設計を採用
    * CANバス接続をサポートし、データ転送がより安定、遅延が少なく、接続が安定
    * TMC2209ドライバを搭載し、UARTモードを使用、押出機に使用可能
    * ADXL345加速度センサーを搭載
    * PT100（MAX31865）サポート
    * 交換可能なファンモス設計
    * BLTOUCH、PL08Nなどのレベリングセンサーをサポートし、XYリミットスイッチを統合
    * 12-24V電圧入力対応 
    * MCU: STM32F072
    * 通信: CAN
    * BLTOUCH、PL08Nなどのレベリングセンサーをサポートし、XYリミットスイッチを統合

## 新機能

    * 4.7K抵抗（NTC熱電対用）または1.1K抵抗（PT1000用）の切り替えをサポート
    * CANインターフェースはMX3.0端子を使用せず、新しいコネクタが15A電流、ピーク30Aをサポート
    * 5/12/24Vファン電圧選択可能に
    * 内蔵NTC100k抵抗、倉温計測用
    * 電源逆接保護を追加
    * ADXL345のINT1をMCUに接続、特定のユーザーのレベリングに使用
    * ヒーティングロッドポートの拡大、定格電流10A
    * 高電圧入力リミットを追加
    * 主制御を国産MCUに変更
    * 磁気エンコーダー(as5047d)サポート、Klipperでの運動解析に使用（現在Klipperはサポートしていません）
    * 1メートルのCAN接続線を新たに追加、ユーザーが端子を正しく圧着できないことによる信号損失の問題を回避
    * 取り付け寸法：穴間距離43.84mm、M3取り付け穴径
    * プロセッサ：GD32F103（2022年10月28日以前に購入）、APM32F072（2022年10月28日以降に購入）
