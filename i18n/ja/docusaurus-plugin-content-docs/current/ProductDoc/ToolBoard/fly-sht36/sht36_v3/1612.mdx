# Eddyの使用

    :::info

    * 注意：ベッドスキャンには専用のZリミットスイッチを使用することをお勧めします。ベッドスキャンをZリミットスイッチとして使用しないでください
    * Eddyをインストールする際、ノズルの位置はノズルの上に少なくとも1-2mm必要です
    * Eddyの周波数を校正する際、コイルPCBはヒートベッドから少なくとも5mm離しておく必要があります
    * すべてのテストは常温で行い、テスト中はヒートベッドを開けないでください

    :::

## 参考設定と使用説明

    ### ベッドスキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意：現在、外部1612モジュールは発売されていないため、この設定はまだ使用できません
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] 内の`horizontal_move_z`パラメータは、ベッドスキャン時にEddyがプラットフォームにできるだけ近づくように、2に設定することをお勧めします

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
