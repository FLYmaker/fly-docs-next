# Eddy の使用

    :::info

    * ベッドスキャンには個別のZリミットの使用を推奨し、ベッドスキャンをZリミットとして使用することは推奨しません。
    * Eddy をインストールする際、ノズルの位置から下部はノズルの上1-2 mm以上にしてください。
    * Eddy 周波数のキャリブレーションを行う際、コイルPCBはベッドから少なくとも5mm離してください。
    * すべてのベッドテストは常温で行い、テスト中にベッドをオンにしないでください。

    :::

## 参照設定と使用説明

    ### ldc1612のデフォルト周波数変更
    この方法はsht36-v3/SB2040-v3またはSB2040-Pro-v3にのみ使用されます。
    Mobaxtermツールを使用してコマンドを実行できます。

    ダウンロードURL:https://mobaxterm.mobatek.net/download.html

    [詳細な使用方法はこちらをご参照ください](/docs/ProductDoc/SBC/fly-pi-v2/ssh)

    最初のコマンドを実行してEnterを押します。

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    2番目のコマンドを実行してEnterを押します。
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    置換が成功すると、`LDC1612_FREQ = 40000000`を含む行が返されます。成功していない場合は、何も返されません。
    または、このコマンドを使用して成功したかどうかを確認できます。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
cat /home/fly/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   注意：変更後、このコマンドを使用してklipperサービスを再起動し、Eddy 周波数を再キャリブレーションしてください。
   ```cfg
   systemctl restart klipper
   ```
   Eddy 周波数キャリブレーションコマンド
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### ベッドスキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意：現在、外部1612モジュールがリリースされていないため、この設定は一時的に使用できません。
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の `horizontal_move_z` パラメータは、ベッドスキャン時にEddyがプラットフォームにできるだけ近づくように、2に設定することをお勧めします。

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
