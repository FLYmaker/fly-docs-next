# Eddyの使用方法

    :::info

    * 床スキャンに専用のZリミットを使用することをお勧めします。床スキャンをZリミットとして使用することは推奨されません
    * Eddyをインストールする際、ノズルの位置はノズルの上部に少なくとも1-2 mmの余裕が必要です
    * Eddyの周波数を校正する際、コイルPCBは熱床から少なくとも5mm離してください
    * 熱床のテストは常温下で行い、テスト中は熱床を開かないでください

    :::

## 参考設定と使用説明

    ### 床スキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意：現在、外部1612モジュールは出ていません。この設定は一時的に使用できません
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の`horizontal_move_z`パラメータを2に設定することをお勧めします。プラットフォームをスキャンする際に、Eddyができるだけプラットフォームに近づくようにします

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
