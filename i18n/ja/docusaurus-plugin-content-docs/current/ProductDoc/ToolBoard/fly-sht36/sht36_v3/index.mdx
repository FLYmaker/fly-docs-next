# 1. 製品紹介

FLY SHT36 V3は、広州市メロンエレクトロニクス有限会社が3Dプリンターのエクストルーダーヘッドのために設計したノズルツールボードです。このツールボードを使用することで、プリンターヘッドとメインボード間の複雑な配線を4本の線で置き換え、レイアウトを最適化することができます。SHT36 V3は36ステップモータに対応しており、CANトランスシーバー、RS232トランスシーバー、TypeCポート、TMC2209ドライバー、LIS2DW加速度センサー、2つの制御可能なファンなどの機能を統合しています。このツールボードはKlipperおよびRRFファームウェアに対応しています。

    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/sht36.webp').default} size="100%" align="center" />


## 1.1 製品特徴

    * ブートボタンを予備し、ユーザーはTypeCを通じてブートモードに入り、ファームウェアを更新することができます
    * Klipperファームウェア、RRFファームウェアをサポート
    * CANモード、RS232モード、RRF CANモードをサポート
    * RS232モードでは、より高い伝送速度とより強い安定性を有しています。CANモードでは、CAN拡張をサポートし、CAN拡張インターフェースを予備しています
    * 温度センサーサポート： 板載MAX31865チップ、PT100およびPT1000温度センサーに対応
    * 2つの制御可能なファンをサポート、電圧は選択可能、小型モス板を交換可能、最大出力電流は1Aに達します
    * 板載LIS2DW加速度センサーとLDC1612センサー
    * RGBインターフェースを予備、板載2209ドライバー、多種のレベリングセンサーに対応、例えばBLtouch、klicky、Voron Tap、PL08n等
    * 12-24V電圧、15A電流入力に対応、ヒーティングロッドポートを拡大、定格電流10A

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポート
    * RS232モードを新たに追加、より高い伝送速度とより強い安定性を有し、デバッグも簡単
    * RRF CANモードを新たに追加、RRFツールボードに切り替え、適用性を強化
    * CAN拡張インターフェースを追加、CANデバイスの拡張を簡単に
    * 終端抵抗ダイポールスイッチを新たに追加、その終端抵抗120Rはダイポールで選択可能、ジャンパーが不要
    * 1メートルの電源/信号接続ケーブルを新たに追加、接続線はツイストペア設計を採用、耐ノイズ性能が強化

## 1.3 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクタを使用、15A電流をサポート、ピーク30A
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * 板載センサー：LIS2DWとLDC1612
    * ヒーティングロッドインターフェース: ヒーティングロッド（HEAT0）、10A
    * 拡張インターフェース: RGB、エンドストップ、PT100/PT1000、XH2.54-2P CAN拡張ポート
    * ファンインターフェース：2つのDC制御可能なファン
    * モータードライバー：板載TMC2209
    * 温度センサー：1路100K NTCまたはPT1000（TH0）、1路PT100/PT1000選択可能
