# Katapultファームウェア

## 書き込みモードに入る

    :::tip

    丸で囲まれているのが`BOOTボタン`です。

    :::

    * 書き込みモードに入るためには、完全に電源を切った状態で、BOOTボタンを押しながらデータ転送用のケーブルを一端はメインボードに、もう一端はあなたのデバイスに接続してください。
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-adxl/img/boot.webp').default} size="100%" align="center" />

            :::tip

            1つのファームウェアだけを書き込む必要があります。

            * 以下はファームウェアの書き込みコマンドです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/USBADXL/FLY_USBADXL.uf2
                ```

            5. 以下のメッセージが表示されれば書き込み成功です。
