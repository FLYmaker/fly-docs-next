# Katapultファームウェア


## 書き込みモードに入る

    :::tip

    囲みで示されているのが`BOOTボタン`です

    :::

    * 書き込みモードに入るには、メインボードが完全に電源オフの状態で、BOOTボタンを押しながら、データ転送機能を持つUSBケーブルの一方をメインボードに、もう一方をデバイスに接続します。
