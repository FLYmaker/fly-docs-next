# {props.name}システムダウンロード

:::warning

SDカードの注意事項:

* 容量は`16GB`以上`128GB`以下のTFカードでなければならない
* TFカードの転送速度は`class10`以上である必要があり、SanDisk等のブランドのTFカードを使用することを推奨

:::

:::danger 注意事項

* サーバーの問題により、システムまたはKlipperの更新は行わないでください
* FLY OSシステムをダウンロードしてください、オリジナルのArmbianは使用が難しく、インストールをお勧めしません

:::

:::info 重要な通知

* ダウンロードしたシステムは解凍が必要であり、直接拡張子を変更したり、直接書き込んだりしてはいけません！！！
* 解凍せずに書き込むと起動できません！！！

:::
