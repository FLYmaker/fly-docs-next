# {props.name}システムのダウンロード

:::warning

SDカードの注意事項:

* 容量は`16GB`以上、`128GB`以下のTFカードでなければならない
* TFカードの転送速度は`class10`級またはそれ以上である必要があります。サンディスクなどのブランドのTFカードを使用することをお勧めします

:::

:::danger 注意事項

* サーバーの問題により、システムやklipperを無作為に更新しないでください
* FLY OSシステムをダウンロードしてください。オリジナルのArmbianは使用が難しく、インストールを推奨しません

:::

:::info 重要な注意

* ダウンロードしたシステムは解凍が必要です。直接拡張子を変更したり、直接書き込みを行わないでください！！！
* 解凍せずに書き込むと起動できません！！！

:::
