# FLY-Toolsのインストール

    1. 最新バージョンのKlipperをプル

    ```bash
    cd ~/klipper
    git pull
    ```

    * ローカルのKlipperソースコードを変更した場合、プルが失敗する可能性があります。その場合は以下のコマンドを使用してください。 （この操作で以前変更したファイルが失われます）

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. 最新のKatapult（元Canboot）をプル

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Fly-Toolsサービスのインストール

        * インストールプロセスはGitサービスプロバイダに依存します。ダウンロードが止まった場合は`CTRL+C`で終了し、再度試してください
        * Githubの1時間あたりの最大リクエスト数は60回、Giteeは不明
        * 国内ユーザーはGiteeからインストールスクリプトを取得することをお勧めします
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Githubからインストールスクリプトを取得

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
