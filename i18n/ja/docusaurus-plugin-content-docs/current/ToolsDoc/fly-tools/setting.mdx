# 機能デモ

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.png').default} size="100%" align="left" />


    * ホスト名の変更をサポート
    * システムタイムゾーンの変更をサポート
    * 上位機40PIN拡張の有効化をサポート（PWMを有効化した後はシリアルポートで上位機に接続できなくなる可能性があることに注意。スクリーンを有効化した後、上位機の加速度ポートが使用できない場合があります）
    * マザーボードモデルの変更をサポート
    * Klipperウェブページの変更をサポート、FuiddとMainsailに対応
    * KlipperScreenの開閉をサポート
    * Klippyプロセスを指定のCPUコアで強制的に実行することをサポート、Klipperを選択したコアで実行することができます
