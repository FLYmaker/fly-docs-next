# 機能デモ

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/utils.gif').default} size="100%" align="left" />

    * Klipperのデフォルト負荷図の生成をサポート
    * MCU周波数図の生成をサポート
    * システム負荷図の生成をサポート
    * 温度曲線の生成をサポート
    * 周波数応答図の生成もサポートし、既存のファイルを直接図表にすることも可能


# `matplotlib`が不足している場合、以下のコマンドを実行してください

    ```bash
    ~/klippy-env/bin/pip install -v matplotlib
    ```
