# SSH를 통해 네트워크에 연결하기

* 시스템이 이미 시작되어 SSH를 통해 직렬 연결이 되어 있는지 확인하세요. **MobaXterm_Personal** 사용을 권장합니다.
* 상위 기기에 **M2WE** 또는 **2.4G_WiFi 모듈**이 설치되어 있는지 확인하세요. **Lite2** 및 **MiniPad**의 경우 모듈 설치가 필요 없습니다.
* **2.4G_WiFi 모듈**, **Lite2** 및 **MiniPad**는 2.4G WiFi만 사용할 수 있으며 5G WiFi는 사용할 수 없음을 주의하세요.
* 네트워크가 이중 주파수 결합을 사용하지 않도록 주의하세요, 그렇지 않으면 정상적으로 연결할 수 없습니다.
* 안테나가 이미 설치되었는지 확인하세요!!!

1. SSH 소프트웨어에서 **nmtui** 명령을 입력하고 엔터를 누르면 아래와 같은 화면이 나타납니다.

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

2. 엔터를 누르고 **Activate a connection**을 선택한 후 다시 엔터를 누릅니다.

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

3. 연결할 네트워크를 선택하고 엔터를 누른 후 WiFi 비밀번호를 입력한 다음 **OK**를 선택하여 연결을 저장합니다.

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

4. WiFi에 연결된 후, 키보드에서 **Esc** 키를 몇 번 눌러 현재 화면에서 나가고 **ip a**를 입력한 후 엔터를 누릅니다.

   * 192.168.2.136이 상위 기기의 IP 주소입니다.

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
