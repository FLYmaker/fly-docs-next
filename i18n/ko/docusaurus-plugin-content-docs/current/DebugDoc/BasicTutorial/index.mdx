# 상위기기 SSH 연결

:::danger

* 이 문서는 FLY 상위기기에만 적용됩니다. 다른 장치는 참고용입니다.
* 펌웨어 컴파일 및 번인 시, 네트워크 연결 튜토리얼을 참조하세요. 시리얼 연결 튜토리얼이 아닙니다.

:::

* 시리얼 또는 WIFI를 통해 장비에 연결할 수 있습니다.
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해주세요.
* SD카드 또는 M2WE에 **해당 상위기기의 시스템이 번인**되었는지 확인하세요.
* **상위기기가 정상적으로 시작되지 않으면 드라이브, 리미트 스위치, 팬 등 주변 장치를 제거하세요!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 편리하게 컴파일하세요!!**
* **펌웨어를 컴파일하려면 WIFI 또는 이더넷을 통해 상위기기에 연결하는 튜토리얼을 참조하세요.**

        ## 네트워크 IP를 사용하여 상위기기 SSH에 연결

        * 시스템이 **정상적으로 부팅**되었는지 확인하세요.
        * 상위기기가 연결된 **라우터 백엔드에 로그인**하세요. 상위기기의 IP 주소를 알지 못한다면.
        * **상위기기의 사용자 이름과 비밀번호를 알고 있어야 합니다**. FLY 시스템의 기본 사용자 이름은 **fly**이고, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 부팅되고 네트워크에 연결되어 있다면, 다음 방법으로 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 다음 설정을 진행하세요. IP는 앞서 확인한 IP를 입력하거나 라우터 백엔드에서 확인하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계는 상위기기의 사용자 이름을 입력하세요**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 처음 연결 시 이 창이 팝업되면 `Accept`를 클릭하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 표시되면 상위기기 비밀번호를 입력하고 엔터를 누르세요.

        **주의: 여기서 입력한 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요**

        FLY 시스템 기본값은 사용자 이름: `fly`, 비밀번호: `mellow`입니다.
