# 플러그인을 통해 LED 제어 확장하기

원문 문서

```cfg
https://github.com/julianschill/klipper-led_effect
```

## 설치

**MobaXterm_Personal** 등 **SSH 도구**를 사용하여 **WIFI**로 상위 기기에 연결하고 다음 사항을 확인하십시오:

1. **상위 기기에 Klipper 펌웨어가 설치되어 있는지 확인하세요**
2. **로그인한 사용자가 Klipper를 설치한 사용자여야 합니다**
3. **입력기가 영어인지 확인하세요**
4. **상위 기기가 장치를 정상적으로 검색할 수 있는지 확인하세요**
5. **위의 주의사항을 모두 준수하지 않으면 다음 단계로 진행할 수 없습니다**

이 모듈은 기존 Klipper 설치에 설치 스크립트를 사용하여 설치할 수 있습니다.

```cfg
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## 참조 구성

```cfg
[neopixel TP_led]
pin: PB7
chain_count: 25
# LED 수
# 램프 수
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
