# 상위기기편

## 새로 산 상위기기를 어떻게 작동시켜야 하나요

  1. 먼저 상위기기의 모델을 확인하세요. Lite 2, PI V2 등의 상위기기인지 확인합니다.
  2. 아래 항목들을 준비했는지 확인하세요
        * 소프트웨어를 구워넣기 위한 소프트웨어
        * 해당 시스템 이미지
        * 카드 리더기, TF 카드, 데이터 전송 기능이 있는 TYPE-C

  시스템을 구워넣기 전 주의사항:
        * 메모리 카드 용량은 반드시 `16GB` 이상, `128GB` 이하의 TF 카드여야 하며, 전송 속도는 `class10` 이상이어야 합니다. 샌디스크 등의 브랜드 TF 카드를 권장합니다.
        * 시스템을 무작정 업데이트하지 마세요!! 서버가 해외에 있어서 업데이트 문제가 쉽게 발생할 수 있습니다.
        * 최대한 저희가 추천하는 소프트웨어를 사용하세요.
        * 가능하면 LINUX 지식을 조금 배우시길 권장합니다. <Button variant="contained" disableElevation href="https://item.jd.com/12443890.html">《새로운 Linux 비공개 요리책 기초 학습편》</Button> 등을 추천드립니다.

  3. FLY 시스템 이미지 <Button variant="contained" disableElevation href="/docs/category/시스템이미지">시스템 이미지</Button>
  4. 필요한 도구 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">구워넣기 도구</Button>

## 각종 도구와 해당 시스템 이미지를 다운로드한 후

  * 시스템을 압축 해제합니다. 압축 해제 후 파일 확장자가 `.img`가 됩니다.
  * 메모리 카드를 SDFormatter로 다시 포맷해야 합니다.
  * `win32DiskImager`를 사용하여 시스템에 구워넣습니다.
  * `balenaetcher`를 사용하여 시스템을 구워넣는 것은 권장하지 않습니다.
  * FLY 상위기기의 경우 `FlyOS-Env.txt` 파일을 열어 구성해야 합니다.
  * 시스템을 구워넣은 후에는 절대 포맷하지 마세요!!!!!!

## 시스템 시작

  * 시스템을 구워넣은 후 상위기기에 시스템 카드를 설치하고 전원을 켜야 합니다.
  * 이때 시스템이 이미 구성되어 있습니다. 데이터 전송 기능이 있는 TYPE-C를 사용하여 컴퓨터에 연결하고 시리얼 연결을 사용하세요.
  * 다른 상위기기는 정상적으로 로그를 표시해야 합니다. 로그가 표시되지 않으면 엔터 키를 두 번 누르거나 시스템을 다시 설치하세요.
  * 시작 후 WIFI에 연결할 수 있습니다.


## M2WE와 메모리 카드 사용 설명

  * 기기에 M2WE와 메모리 카드가 설치되어 있으면 시스템은 기본적으로 메모리 카드에서 시작합니다.
  * M2WE는 SD 카드를 사용하여 시스템을 구워넣는 것을 권장합니다. USB를 통해 M2W에 구워넣으면 드라이버 호환성 등의 이유로 실패할 수 있습니다. 직접 메모리 카드를 사용하여 M2WE에 구워넣는 것을 권장합니다.
  * 다운로드한 시스템 이미지가 해당하는지 주의하세요. 그렇지 않으면 시스템이 시작되지 않습니다.
  * Lite2와 MINI PAD는 메모리 카드만 사용할 수 있습니다.


## 일상 사용 주의사항

  * 직접 전원을 끊지 말고 정상적으로 종료하세요. 그렇지 않으면 데이터 손실이 발생할 수 있습니다.
  * GitHud 등의 서버가 해외에 있으므로, 스스로 업데이트하거나 다양한 플러그인을 설치하는 것은 권장하지 않습니다.
