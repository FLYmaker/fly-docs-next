* BOOT 단축 또는 장시간 누르고 type c로 상위 컴퓨터에 연결한 후 `lsusb` 입력 시 `2e8a:0003`가 나타납니다.
* 현재 RP2040은 펌웨어가 플래시되는 상태에 있으며, 확장자가 uf2인 펌웨어만 플래시할 수 있습니다. 펌웨어가 플래시되면 사라집니다.
* `RPI-RP2`를 분리한 후에는 BOOT를 단축하거나 계속해서 BOOT 버튼을 누를 필요가 없습니다.
    <ImageView image={require('@site/docs/General/get-id/img/2e8a.png').default} size="80%" align="Center" /> 
* 아래 명령어를 통해 펌웨어를 RP2040에 플래시합니다.

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
