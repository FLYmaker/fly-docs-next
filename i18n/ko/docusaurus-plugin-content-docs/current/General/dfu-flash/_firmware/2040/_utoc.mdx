다음은 펌웨어를 컴파일하는 방법을 소개합니다:
* SSH에 연결한 후 아래 명령어를 입력하고 Enter를 누릅니다:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* 여기서 `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`는 이전에 컴파일된 데이터와 펌웨어를 삭제하고, `make menuconfig`는 펌웨어를 컴파일합니다. 실행 후 아래와 같은 화면이 나타납니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**을 선택하고 Enter를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* **Micro-controller Architecture** 메뉴로 들어가 **Raspberry Pi RP2040**을 선택하고 Enter를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.png').default} size="100%" align="Center" />

* **Bootloader offset**을 선택하고, **16KiB bootloader**를 선택합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.png').default} size="100%" align="Center" />

* **Communication interface**를 선택하고, **USB to CAN bus bridge**를 선택합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/utoc.png').default} size="100%" align="Center" />
