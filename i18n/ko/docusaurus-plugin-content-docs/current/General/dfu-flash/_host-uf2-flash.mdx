1. 설치 및 플래싱 도구

    * SSH에서 아래 명령어 입력
    * 첫 플래싱 시에만 실행, 이후에는 실행할 필요 없음
    ```bash
    cd ~/klipper/lib/rp2040_flash/ && make
    ```
    <ImageView image={require('@site/docs/General/dfu-flash/img/2040.png').default} size="100%" align="left" />

2. 아래 명령어를 실행하여 연결 여부 확인

    * `lsusb` 입력
    * `2e8a:0003` 장치가 없으면 **플래싱 모드**로 다시 진입 시도
    * 시스템 버전 문제로 다른 이름이 표시될 수 있지만 `2e8a:0003`가 있으면 플래싱 모드 진입

        <ImageView image={require('@site/docs/General/get-id/img/2e8a.png').default} size="80%" align="left" /> 

    :::warning 주의사항

    `2e8a:0003`이 있을 때 다음 단계로 진행

    :::

3. 사전 컴파일된 펌웨어 가져오기

    * 여기에는 사전 컴파일된 펌웨어가 포함되어 있습니다
    * 상위 기기가 인터넷에 연결될 수 있어야 설치 가능
    * 상위 기기에서 아래 명령어를 실행하여 펌웨어 패키지 다운로드

    ```bash
    cd && git clone https://github.com/FengHua741/FLY_Katapult.git
    ```

    * 이미 펌웨어 패키지를 다운로드한 경우 업데이트 후 나머지 작업 수행 권장

    ```bash
    cd ~/FLY_Katapult && git pull
    ```
