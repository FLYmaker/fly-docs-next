:::danger 주의 사항

* 시스템 호환성 문제로 인해 컴퓨터의 `USB2.0`을 사용하여 플래시
* 그래도 플래시가 되지 않으면 다른 호스트 컴퓨터를 사용하여 플래시

:::

1. 플래싱 도구 <Button variant="contained" disableElevation href="https://mellow-next.klipper.cn/docs/ResDownload/auxiliary_software/">다운로드 링크</Button>
2. `SetupSTM32CubeProgrammer`와 `STM32 DFU 드라이버`를 다운로드 필요
3. 앞서 컴파일한 펌웨어(klipper.bin) 또는 사전 컴파일된 `Katapult 펌웨어`를 컴퓨터의 임의 디렉토리로 복사
4. Type-C 데이터 케이블을 사용하여 메인보드를 컴퓨터에 연결, **DFU 플래싱 모드**로 진입했는지 확인
5. STM32CubeProgrammer에서 USB 모드 선택 후 새로고침, 연결
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-1.png').default} size="100%" align="center" />
6. 오류 창이 나타나지 않으면 연결 성공
7. 왼쪽 다운로드 아이콘 클릭 후 아래 몇 가지 선택
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-2.png').default} size="100%" align="center" />
8. 화살표가 가리키는 `Browse` 클릭 후 플래싱할 펌웨어 선택
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-3.png').default} size="100%" align="center" />
9. `Start address` 선택 후 `0x08000000` 입력
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-4.png').default} size="100%" align="center" />
10. 아래의 `Start automatic mode` 클릭하여 펌웨어 플래싱 시작
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-5.png').default} size="100%" align="center" />
