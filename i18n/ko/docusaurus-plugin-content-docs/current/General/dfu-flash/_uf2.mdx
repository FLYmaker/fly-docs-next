* 상위 컴퓨터가 정상적으로 인터넷에 연결되어 있는지 확인하십시오.
* 하위 컴퓨터가 상위 컴퓨터에 연결된 데이터 케이블이 데이터 전송 기능을 갖추고 있는지 확인하십시오.
* 상위 컴퓨터에 SSH로 연결한 후 `lsusb`를 입력하고 엔터를 누르면 다음 세 가지 상황이 발생할 수 있습니다.
    * 이미 인식된 USB 장치가 표시됩니다.
    * `ls` 명령어가 없다는 메시지가 나오면, 아래 명령어를 실행하여 설치할 수 있습니다.
        ```bash
        sudo apt-get install usbutils
        ```
    * 오류 메시지가 없고 다른 정보도 없는 경우(시스템 문제로 시스템을 교체해야 함)
* `lsusb`를 입력한 후 반응이 없으면, 이는 시스템 문제로 여기서는 해결할 수 없습니다. 정상적인 시스템으로 교체해야 합니다.
* 아래 이미지에 원으로 표시된 정보가 나타나면 다음 단계로 진행할 수 있습니다.
    <ImageView image={require('@site/docs/General/get-id/img/2e8a.png').default} size="80%" align="left" /> 
* `2e8a:0003`은 이번에 사용할 장치입니다. 이후의 메시지는 무시해도 됩니다. 시스템 문제로 인해 표시되지 않을 수 있거나 불완전하게 표시될 수 있습니다.
* 표시되지 않으면, 데이터 케이블이 제대로 연결되어 있는지 또는 메인보드가 Katapult 모드에 있는지(원래 이름은 CANBOOT) 직접 확인하십시오.
