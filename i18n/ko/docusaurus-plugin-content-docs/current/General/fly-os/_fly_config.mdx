# FLYOS_Env 사용법

    :::tip 팁

    * 3.3.4 시스템부터는 FLY-TOOLS를 사용하여 시스템 설정을 할 수 있으며, boot로 들어가서 설정을 수정할 필요가 없습니다.
    * 이전 시스템은 FLY-TOOLS를 직접 사용하여 설정을 지원하지 않습니다.
    * <Button variant="contained" disableElevation href="/docs/ToolsDoc/fly-tools/">FLY_TOOLS 사용 가이드</Button>

    :::
        
### WIFI 설정 설명

    :::danger 주의사항

    * WIFI 설정 방법이 복잡하여, 시리얼 연결을 통해 상위 컴퓨터에 접속한 후 WIFI 연결을 진행해야 합니다.

    :::
