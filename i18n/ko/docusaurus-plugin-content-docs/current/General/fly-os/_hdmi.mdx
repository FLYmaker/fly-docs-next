## FLY 상위 기기 사용 교재

:::tip
브라우저에서 IP:9999를 열어 FLY-Tools에 접속하세요.
:::

:::tip
FLY 상위 기기의 내장 HDMI 포트는 Micro HDMI입니다. HDMI 케이블을 구매할 때 주의하세요!!!
:::

:::tip
FLY의 상위 기기는 FPC 일선통을 지원하는 모델은 `FLY-PI-V2`, `FLY-C8` 그리고 `FLY-PRO-X10`입니다.
:::

:::tip
FPC를 HDMI 모듈로 변환하여 라즈베리 파이와 같은 상위 기기와 호환할 수 있습니다.
:::
