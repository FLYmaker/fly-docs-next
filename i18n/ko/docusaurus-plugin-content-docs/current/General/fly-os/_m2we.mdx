# M2WE에 USB를 통해 이미지 파일을 기록하는 방법

### 준비 작업

    * 먼저 `Win32DiskImager` 소프트웨어를 설치합니다.
    * `balenaetcher`를 사용하여 시스템을 기록하는 것은 권장하지 않습니다.
    * 시스템을 압축 해제한 후, 파일 이름의 확장자는 `.img`가 됩니다.
    * [M2WE BOOT 드라이버](/docs/ResDownload/auxiliary_software/ "클릭하면 이동합니다")를 다운로드합니다.

### 드라이버 설치

    * `Zadig` 소프트웨어를 엽니다.
    * 첫 번째 `Zadig`는 설치할 필요 없이 직접 실행할 수 있으며, 나머지 두 개는 설치가 필요합니다. 설치 과정에서는 모두 `다음` 또는 `Next`를 클릭하면 됩니다.
    * `Zadig`를 열고, 상단 메뉴 바의 `Options`에서 `List All Devices`를 체크합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/zadig.png').default} size="80%" align="left" />

:::

    * zadig를 엽니다

    <ImageView image={require('@site/docs/General/fly-os/img/zadig1.png').default} size="80%" align="left" />

    * 정상적으로 진행되면 zadig에 `Unknow Device #x` 또는 `USB Device(VID_1f3a_PID_efe8)`가 추가되며, 이를 선택합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/zadig2.png').default} size="80%" align="left" />

    * 장치를 선택한 후 USB ID가 위 그림과 일치하는지 확인하고, 일치하지 않으면 다른 장치를 선택합니다.
    * 위 그림의 두 번째 부분은 `WinUSB(vxxxxxxx)`만 선택할 수 있으며, 버전 번호는 중요하지 않습니다.
    * `Install Driver`를 클릭하여 드라이버를 설치합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/zadig3.png').default} size="80%" align="left" />

    * USB 드라이버 설치 성공 (드라이버는 한 번만 설치하면 됩니다. 나중에 다시 기록할 때는 드라이버를 다시 설치할 필요가 없습니다.)
    * 설치된 FLY-BOOT를 열고 `장치 확인`을 클릭합니다.
    * 이전 단계가 올바르게 수행되었다면 사용 가능한 장치가 발견되었다는 메시지가 표시됩니다.
    * `BOOT`를 시작하고, 정상적으로 진행되면 `BOOT`가 성공적으로 활성화되었다는 메시지가 표시됩니다.

### M2WE 포맷

    * `TFFormatter.exe` 소프트웨어를 사용하여 `BOOT` 디스크를 포맷합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * 또는 Windows 내장 디스크 관리자를 사용하여 파티션을 삭제하고, 파티션을 다시 생성합니다.
    * 이제 파일 탐색기를 열 수 있으며, 14.5GB의 USB 드라이브가 추가됩니다(16GB eMMC는 14.5GB로 표시됩니다).
    * 이제 Win32diskimager를 사용하여 시스템 이미지를 기록할 수 있으며, 드라이브 문자는 이 14.5GB USB 드라이브의 문자를 선택합니다.
