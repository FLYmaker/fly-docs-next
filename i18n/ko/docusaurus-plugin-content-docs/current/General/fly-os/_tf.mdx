# 시스템 TF 카드에 기록하기

### 준비 작업

    * 먼저 `Win32DiskImager` 소프트웨어를 설치하십시오.
    * 시스템을 압축 해제하여 `.img` 확장자가 있는 파일을 얻으십시오.
    * `balenaetcher`를 사용하여 시스템을 기록하는 것은 권장하지 않습니다.
    * TF 카드를 카드 리더기에 삽입하고, 카드 리더기를 컴퓨터의 USB 포트에 연결하십시오.
    * TF 카드에 데이터가 있는 경우, 시스템 기록 후 시스템 카드를 분리하지 않는 것이 좋습니다. M2WE에 기록할 필요가 있는 경우에만 TF 카드를 분리할 수 있습니다.

### 메모리 카드 포맷하기

    * `TFFormatter.exe` 소프트웨어를 사용하여 TF 카드를 포맷하십시오.
    * 포맷할 디스크를 선택하십시오.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />
