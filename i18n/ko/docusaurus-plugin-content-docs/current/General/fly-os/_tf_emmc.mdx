# TF 카드를 통해 M2WE에 기록하기

:::info
시스템 마이그레이션은 TF 카드에 있는 원래 이미지를 M2WE로 완전히 이전할 수 있습니다. 마이그레이션이 완료되면 M2WE의 내용이 원래 TF 카드의 내용과 일치하게 됩니다.
:::

    * M2WE를 설치한 후 전원을 켜서 시작합니다.
    * 먼저 시리얼을 통해 [SSH에 연결](/docs/DebugDoc/BasicTutorial/index.mdx "클릭하여 이동")합니다.
    * 아래 명령어를 실행합니다. 비밀번호를 입력하라는 메시지가 표시되면 `mellow`를 입력하고 엔터 키를 누릅니다. 비밀번호는 보이지 않습니다.

    ```cfg
    sudo nand-sata-install
    ```

    * `Boot from eMMC - system on eMMC` 옵션을 선택한 후 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_1.webp').default} size="80%" align="left" />

    * `This script will erase your eMMC. Continue?` 메시지가 나타나면 `Yes`를 선택하고 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_2.webp').default} size="80%" align="left" />

    * 파일 시스템 유형을 `ext4`로 선택하고 엔터 키를 누릅니다. (일반적으로 기본 옵션이므로 엔터 키만 누릅니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_3.webp').default} size="80%" align="left" />

    * 진행률 표시줄이 나타날 때까지 기다립니다. (2분 이상 기다려도 아래 이미지의 진행률 표시줄이 나타나지 않으면 `CTRL+C`를 눌러 작업을 종료한 후 장치를 재부팅하고 다시 시도합니다.)
    
    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_4.webp').default} size="80%" align="left" />
    * 시스템 마이그레이션 중 전원을 차단하거나 다른 작업을 수행하지 마십시오. 마이그레이션이 완료될 때까지 기다립니다.
    * 마이그레이션이 완료되면 엔터 키를 직접 누르고 장치가 완전히 꺼질 때까지 기다립니다.

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_5.webp').default} size="80%" align="left" />


    * 전원을 껀 후 TF 카드를 제거하고 전원을 다시 켜면 이제 eMMC에 있는 시스템으로 부팅할 수 있습니다.
