:::danger 알아야 할 사항

    * FLY 상위 컴퓨터는 M2WE 또는 2.4G WIFI 모듈을 사용해야 하며, 그렇지 않으면 WiFi를 사용해야 합니다.
    * 중국어 이름의 WIFI와 2.4G 및 5G 듀얼 밴드 WIFI는 지원되지 않으니 주의하세요.
    * mini pad 또는 lite2인 경우 이 경고를 무시하세요.

    :::

## 상위 컴퓨터 연결

    * BOOT 디스크에서 WIFI 설정이 실패한 경우, 이 설정을 참조하여 WIFI 연결을 시도하십시오. 시리얼 연결을 통해 상위 컴퓨터에 연결해야 합니다.
    * 먼저 시리얼을 통해 [SSH에 연결](/docs/DebugDoc/BasicTutorial/index.mdx "클릭하면 이동")하십시오.
    * 아래 명령을 실행하면 비밀번호 입력을 요구할 수 있습니다. `mellow`를 입력 후 엔터를 누르십시오. 비밀번호는 보이지 않습니다.

    ```cfg
    sudo nmtui
    ```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui1.png').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.png').default} size="80%" align="left" />

    * `Activate a connection`을 선택 후 엔터를 누릅니다.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.png').default} size="80%" align="left" />

    * 다음 화면으로 이동하면, 키보드의 `↑ ↓` 키를 사용하여 연결할 WiFi를 선택한 후 엔터를 누르고, 비밀번호를 입력하여 연결을 기다립니다.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.png').default} size="80%" align="left" />


        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.png').default} size="80%" align="left" />

    * 연결이 성공하면 다음과 같이 표시됩니다.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.png').default} size="80%" align="left" />

    * ESC 키를 눌러 명령 줄 인터페이스로 돌아가 `ip a`를 입력합니다. `192.168.2.126`는 상위 컴퓨터의 IP 주소입니다.
