:::tip

* 상위 컴퓨터가 정상적으로 네트워크에 연결되어 있는지 확인하십시오
* 펌웨어 컴파일 전에 네트워크를 통해 SSH를 사용하여 상위 컴퓨터에 로그인해야 합니다
* 상위 컴퓨터에 직렬 도구를 사용하여 로그인하지 마십시오

:::

## Klipper 펌웨어 컴파일 설정 페이지 작업 설명

    :::tip

    키보드 입력 모드가 반각 모드, 즉 영어 모드인지 확인하십시오

    :::

    1. 키보드의 위쪽 화살표 키 `↑`, 아래쪽 화살표 키 `↓`, 메뉴 항목을 선택하기 위해 커서를 위아래로 이동하는 데 사용됩니다
    2. 확인 키 `Enter` 또는 공백 키 `Space`, 메뉴를 선택하거나 하위 메뉴로 진입하는 데 사용됩니다
    3. 종료 키 `ESC`, 이전 메뉴로 돌아가는 데 사용됩니다
    4. `Q` 키는 Klipper 펌웨어 설정 페이지를 종료하는 데 사용됩니다
    5. `Y` 키는 Klipper 펌웨어 설정 페이지를 종료할 때, 프롬프트가 표시되면 `Y` 키를 눌러 구성 파일을 저장합니다

    :::warning

    설정 페이지 옵션이 적을 경우 먼저 `[ ] Enable extra low-level configuration options`을 선택하여 일부 숨겨진 옵션을 표시하십시오

    :::
