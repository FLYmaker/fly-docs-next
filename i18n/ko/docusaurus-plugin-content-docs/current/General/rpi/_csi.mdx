## CSI 사용 방법

    * 이미지 쓰기가 완료된 후, /boot/config.txt 파일에서 `camera_auto_detect=1`을 찾아 `camera_auto_detect=0`으로 수정하고, 카메라 모델에 따라 설정 추가
    * 아래는 참고 설정입니다.

    ```bash
    dtoverlay=imx219,cam1
    ```
