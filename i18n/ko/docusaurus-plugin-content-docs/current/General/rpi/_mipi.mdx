## 라즈베리 파이 5와 MIPI

:::tip
FLY_MIPI는 라즈베리 파이 5를 위해 특별히 설계된 어댑터 모듈로, DSI와 CSI 장치를 바로 사용할 수 있습니다.
:::

### 라즈베리 파이 5와 MIPI 연결

    :::danger 주의사항

    * 라즈베리 파이 5의 DSI와 CSI 인터페이스는 이제 MIPI로 이름이 변경되었습니다.
    * MIPI 인터페이스는 구성에 따라 CSI 카메라 또는 DSI 스크린을 직접 사용할 수 있습니다.
    * 핀의 방향을 주의하세요.

    :::

    * 라즈베리 파이 5에는 두 개의 MIPI 인터페이스가 있으며, 정면에서 보면 1과 0으로 표시되어 있습니다.

## DSI 스크린 연결

    * 1번 인터페이스를 사용하려면 아래와 같이 스크린을 구성합니다.

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * 0번 인터페이스를 사용하려면 아래와 같이 스크린을 구성합니다.

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## CSI 카메라 연결

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * 카메라 모델에 따라 설정을 해야 하며, 이 튜토리얼에서는 imx219를 기준으로 합니다.

    * 왼쪽 MIPI 인터페이스를 사용하려면 아래와 같이 카메라를 설정합니다.
    ```bash
    dtoverlay=imx219,cam1
    ```

    * 오른쪽 MIPI 인터페이스를 사용하려면 아래와 같이 카메라를 설정합니다.
    ```bash
    dtoverlay=imx219,cam0
    ```
