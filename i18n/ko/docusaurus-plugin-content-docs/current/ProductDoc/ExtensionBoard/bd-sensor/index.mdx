# BDsensor

    BD센서는 3D 프린터와 함께 사용할 수 있는 거리 센서로, 실시간으로 빠른 베드 평준화와 Z축 조정을 할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.png').default} size="50%" align="left" />

## 기능

    * Z축 리미트 대체
    * 스캔 시 Z축을 올리거나 내릴 필요가 없어 인쇄 속도 향상
    * XY 이동 스캔, 밀집된 네트 베드 탐지 가능

## 연결

    * 전원: 5V
    * 펌웨어: Klipper/Marlin
    * 연결: 툴보드/메인보드 (BD센서는 서보부와 리미트 포트를 사용)
    * 히트베드: 일반 PEI, 알루미늄 베드 등 금속 표면 히트베드, 자석 구멍이 뚫린 히트베드 및 비금속 표면 히트베드는 지원하지 않음

## 주의사항

    * Klipper는 klipper와 최신 BD-SENSOR 패치를 다운로드해야 하며, 메인보드 펌웨어를 다시 플래싱해야 합니다.
    * 디버깅 시 긴급 정지를 언제든지 트리거할 수 있는 곳에 두어 PEI를 긁지 않도록 주의하세요.
    * 베드 스캔 속도는 200으로 권장!
    * Z축 홈 속도는 5를 초과하지 않는 것이 좋습니다.
