## 연결

    * 메인보드의 PWM 포트에 연결, 다른 메인보드는 해당 문서를 참조하세요
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-7040/img/wiring.webp').default} size="80%" align="Center" />

## 참조 설정

    ```cfg
    [fan]
    ##    프린팅 냉각 팬 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## 부팅 시 자동 회전 해결 방법

    :::info
    * 여기서는 간단히 설명합니다
    * 펌웨어 컴파일 시 PWM 신호 핀을 추가하고 상태를 반전시킵니다
    * 테스트 시 SUPER8PRO 사용, 핀은 PF8 사용, 펌웨어 컴파일 시 `GPIO pins to set at micro-controller startup`에 `!PF8`를 추가하면 됩니다
    * `!`는 영어의 느낌표임에 주의하세요
    * 다른 메인보드도 동일합니다
    :::
