# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## 제품 특징

    * 36 모터 등에 적합
    * 라즈베리 파이, FLY PI, 오렌지 파이, 메인보드 등 SPI 인터페이스에 연결 가능

## 제품 사양

    * 내장 인터페이스: ZH1.5 6P 플러그
    * 통신 방식: SPI
    * 작동 전압: 5V

## 주의 사항

    :::warning 주의 사항

    * 작동 전압 5V, 잘못 연결하지 마세요
    * 다양한 펌웨어에 적합, SPI 연결을 사용할 수 있는지 확인하세요

    :::
