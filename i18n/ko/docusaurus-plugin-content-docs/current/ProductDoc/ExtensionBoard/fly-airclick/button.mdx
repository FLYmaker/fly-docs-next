# 무선 버튼 추가

    :::info 중요한 알림
    * 무선 버튼은 배터리를 설치해야 하며, 배터리 모델은 `CR927`입니다
    :::

    * 홈페이지로 돌아가서 그림의 빨간색 상자 안에 있는 `+`를 클릭하세요
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button.webp').default} size="80%" align="Center" />
    * 아래와 같은 화면이 나타나면, 배터리가 설치된 무선 버튼을 정상적으로 한 번 누르세요
    * 빨간색 상자 안에 자동으로 무선 버튼의 ID가 생성됩니다. 이 ID는 고유합니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-1.webp').default} size="80%" align="Center" />
    * 빨간색 상자 안에 버튼의 이름을 입력하세요
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-2.webp').default} size="80%" align="Center" />
    * 첫 번째 빨간색 상자를 클릭하면 두 번째 파란색 상자가 나타납니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-3.webp').default} size="80%" align="Center" />



    :::danger 참고사항

    * 사용자 정의 GCODE와 API 요청은 무선 연결을 통해 제어합니다
    * GPIO 트리거는 유선으로 제어합니다

    :::

## 사용자 정의 GCODE
    * 사용자 정의 `GCODE`는 빨간색 상자 안에 실행할 코드 또는 매크로 이름을 입력하면, AirClick이 자동으로 Klipper에 전송합니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/gcode.webp').default} size="80%" align="Center" />

## API 요청
    * 사용자 정의 매크로, 긴급 정지, Klipper 재시작, 펌웨어 재시작, 시스템 종료 등을 구현할 수 있습니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/api.webp').default} size="80%" align="Center" />

## GPIO 트리거
    * GPIO 트리거는 STM32F072와 Klipper 연결이 필요합니다
    * 펌웨어 설치 및 연결은 이 문서를 참조하세요 <Button variant="contained" disableElevation href="/docs/ProductDoc/ExtensionBoard/fly-airclick/offline-operation-support">펌웨어 설치 및 연결</Button>
    * 사용할 수 있는 IO는 `PB0`부터 `PB9`까지 총 열 개의 IO가 있으며, 펌웨어를 정상적으로 설치하고 Klipper에 연결하면 아래 참조 설정을 통해 정상적으로 구성할 수 있습니다
    
        ```cfg
        [mcu AirClick]
        serial: /dev/serial/by-id/usb-Klipper_stm32f072xb_AirClick-if00
        ### USB ID를 검색한 후 이 부분을 수정해야 합니다

        [gcode_button _test]
        pin: ^!AirClick:PB0
        press_gcode:
            M118 PB0

        [gcode_button _test1]
        pin: ^!AirClick:PB1
        press_gcode:
            M118 PB1

        [gcode_button _test2]
        pin: ^!AirClick:PB2
        ```
