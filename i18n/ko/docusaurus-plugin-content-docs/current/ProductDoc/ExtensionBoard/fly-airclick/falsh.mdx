# 펌웨어 업데이트 및 번인

    :::danger 주의사항

    * 출하 시 이미 펌웨어가 설치되어 있습니다

    :::

## 펌웨어 다운로드

    * 펌웨어 설명
    * `wbtn-vX.X.X-X-XXXXXX-full.bin`은 전체 펌웨어로, 0x0 주소에서 번인해야 하며, 번인 후 모든 설정이 유지되지 않습니다
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin`은 증분 펌웨어로, 0x10000 주소에서 번인해야 하며, 펌웨어만 업데이트하고 기존 설정에 영향을 주지 않습니다
    * `wbtn-vX.X.X-X-XXXXXX-www.bin`은 UI 펌웨어로, 0x290000 주소에서 번인해야 하며, 웹 페이지 프론트엔드 UI만 업데이트하고 기존 펌웨어 및 설정에 영향을 주지 않습니다
    * <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/airclick.zip">펌웨어 패키지 다운로드</Button>

## 펌웨어 업데이트
    * `AirClick`의 전원을 꺼야 하며, BOOT를 사용하여 `AirClick`의 ESP32 번인 포트를 컴퓨터와 연결해야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.png').default} size="80%" align="Center" />

    1. USB 번인 도구를 다운로드하고 압축을 해제합니다 <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">다운로드</Button>
    2. `flash_download_tool_3.9.4.exe`를 실행합니다. 주의: 번인 도구를 실행하면 먼저 CMD 창이 나타나고, 그 후 번인 도구 인터페이스가 나타납니다. CMD 창을 닫지 마세요
    3. 번인 도구에서 ``ChipType:``를 ``ESP32-C3``로 선택합니다
    4. 번인 도구에서 ``WorkMode:``를 ``Develop``로 선택합니다
    5. 번인 도구에서 ``LoadMode:``를 ``USB``로 선택합니다
    6. 마지막으로 ``OK``를 클릭하세요. 다음 이미지를 참조하십시오
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.png').default} size="50%" align="Center" />
    7. 클릭 후 다음 이미지가 나타납니다
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.png').default} size="50%" align="Center" />

    :::tip

    * 위 이미지의 녹색 박스 선택 영역은 이미지와 동일하게 유지해야 하며, 임의로 수정할 수 없습니다

    :::

    8. AirClick을 USB-Typec 데이터 케이블로 컴퓨터에 연결합니다
    9. 위 이미지의 파란색 영역 1 버튼을 클릭하여 번인할 펌웨어를 선택합니다
    10. 위 이미지의 파란색 영역 2를 클릭하여 번인 주소를 입력합니다
    11. 위 이미지의 파란색 영역 3을 클릭하여 AirClick의 시리얼 포트를 선택합니다 (없으면 AirClick의 번인 버튼을 누르고 컴퓨터에 연결합니다)
    12. 위 이미지의 파란색 영역 4에서 최대 `921600`의 보드레이트를 선택합니다
    13. 위 이미지의 파란색 영역 5를 클릭하여 번인을 시작합니다. 번인이 완료되면 AirClick의 펌웨어 업데이트가 완료됩니다 (번인 완료 메시지가 나타나기 전에는 AirClick의 데이터 케이블을 분리하지 마세요)
