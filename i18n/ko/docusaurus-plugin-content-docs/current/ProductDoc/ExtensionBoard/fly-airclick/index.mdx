# AirClick 무선 버튼
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.png').default} size="100%" align="Center" />



## 제품 특징

    * 높은 호환성: Klipper 기계를 위해 특별히 설계되어, Klipper 기계에 직접 연결하여 사전 설정된 명령을 보내고, 원활한 호환성과 사용 경험을 제공합니다.
    * 사용자 정의 매크로 명령: 사용자가 자신의 요구에 따라 명령을 사용자 정의하여 기계의 조작을 더 유연하고 편리하게 만들 수 있습니다.
    * 무선 연결과 유선 연결을 Klipper에 지원
    * 무선 연결: ESP32의 설계를 통해 Klipper에 직접 사용자 정의 명령이나 매크로를 보낼 수 있으며, 사용자에게 매우 높은 유연성과 확장성을 제공합니다.
    * 유선 연결: STM32F072 칩을 통해 Klipper에 네트워크가 연결되지 않는 경우에도 간단한 조작을 할 수 있습니다. 총 열 개의 사용자 정의 명령이나 매크로를 지원합니다.
    * 사용하기 쉬움: 직관적인 사용자 인터페이스를 갖추고 있어 사용자가 쉽게 무선 버튼을 추가하고 관리할 수 있어 조작 편의성을 크게 향상시킵니다.

## 제품 사양

    * 메인 컨트롤러 칩: ESP32
    * 추가 칩: STM32F072
    * ESP32-WIFI를 통해 명령 전송: 무제한 개수
    * STM32F072-IO를 통해 명령 전송: 10개
    * 호환성: Klipper 기계를 위해 특별히 설계됨
    * 전원: 5V

## 주의 사항

    :::danger 주의 사항

    * 이 제품은 5V 전원을 사용하며 `type c` 또는 `PH2.0-2P` 단자로 전원을 공급할 수 있습니다.
    * 이 제품의 내장 안테나는 버튼 신호를 수신하기 위한 용도로만 사용되며, WIFI 연결이 필요한 경우 안테나를 올바르게 설치해야 합니다. 안테나 단자는 `ipex3`입니다.
    * 안테나를 제대로 설치하십시오!!! 그렇지 않으면 신호가 너무 약해 과열될 수 있습니다!!!
    * 오작동을 방지하기 위해 버튼을 15초 이상 길게 누르면 시스템이 자동으로 꺼집니다.

    :::
