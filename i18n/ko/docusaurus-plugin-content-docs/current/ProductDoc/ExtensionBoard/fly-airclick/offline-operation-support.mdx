# 유선 연결 설정

:::danger 주의사항

* STM32F072에 펌웨어를 플래시한 후 Klipper에 연결해야 유선 연결 작업이 지원됩니다.

:::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` 키를 누르면 **Save configuration**이 나타납니다. 이때 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * 이제 설정이 저장되었고 명령행 인터페이스로 돌아갑니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 다소 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * 마지막으로 아래 내용이 출력되면 컴파일이 성공한 것입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 플래시 모드로 진입

    * AirClick 전원을 꺼진 상태에서 STM32의 BOOT를 누른 채로 STM32의 플래시 포트를 Type-C로 상위 컴퓨터와 연결합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.png').default} size="100%" align="center" />
