# Klipper 연결

    :::info 중요한 안내

    * 이 제품에 장착된 안테나는 수신용으로만 사용할 수 있습니다. 송신 안테나는 `ipex3` 단자에 직접 설치해야 합니다.
    * 안테나를 제대로 설치하십시오!!! 그렇지 않으면 신호가 너무 약해 심각한 발열이 발생할 수 있습니다!!!
    * 처음 사용 시 AirClick 핫스팟에 수동으로 연결한 후, 해당 WIFI에 연결해야 합니다.

    :::


    :::info 중요한 안내

    * AirClick은 Moonraker에 연결된 후 Klipper 컨트롤러에 명령을 보내 제어합니다.
    * 사용자 정의 매크로, 긴급 정지, Klipper 재시작, 펌웨어 재시작, 시스템 종료 등을 수행할 수 있습니다.

    :::

## AirClick WIFI 연결

    * AirClick의 핫스팟에 연결합니다. 핫스팟 이름은 `FLY-WBTN-XXXXXX`입니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.png').default} size="50%" align="Center" />
    * 핫스팟에 연결한 후 브라우저에서 [192.168.4.1](http://192.168.4.1/#/setting)에 접속합니다.
    * WIFI 설정 화면이 나타납니다. 프린터와 동일한 라우터에 연결하려면 비밀번호를 입력하고 아래 파란색 상자를 클릭하여 WIFI에 연결합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.png').default} size="50%" align="Center" />
    * WIFI에 연결되면 AirClick의 연결 후 IP 주소가 표시됩니다. 아래 파란색 상자를 클릭하여 장치를 재시작합니다.
        * 번호 1은 AirClick의 IP 주소를 표시합니다.
        * 번호 2는 IP를 복사하고 장치를 재시작하는 부분입니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.png').default} size="80%" align="Center" />
    * 현재 네트워크를 해당 라우터로 수동으로 전환해야 접속할 수 있습니다.

## 언어 변경
    * 오른쪽 상단에 있는 아이콘을 클릭하여 필요한 언어를 선택합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-2.png').default} size="80%" align="Center" />

## Klipper에 연결

    :::info 중요한 안내
    * AirClick은 Moonraker에 연결된 후 Klipper 컨트롤러에 명령을 보내 제어합니다.
    * 사용자 정의 매크로, 긴급 정지, Klipper 재시작, 펌웨어 재시작, 시스템 종료 등을 수행할 수 있습니다.
    :::

    * 왼쪽의 시스템 설정 아이콘을 클릭합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.png').default} size="80%" align="Center" />
    * 아래 `Moonraker IP`를 찾아 Klipper 장비의 IP 주소를 입력합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-1.png').default} size="80%" align="Center" />
        * 숫자 1에 Klipper 호스트의 IP 주소를 입력합니다.
        * 숫자 2는 기본적으로 `7125`로 설정되어 있습니다. 다른 포트를 사용하는 경우 해당 포트를 입력해야 합니다.
        * 숫자 3은 설정 저장 확인입니다.
    * 올바르게 연결되면 웹 페이지 상단에 연결 성공 메시지가 표시됩니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.png').default} size="80%" align="Center" />
