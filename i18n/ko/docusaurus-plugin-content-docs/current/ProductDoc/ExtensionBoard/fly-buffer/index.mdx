# LLL 버퍼 사용

 	LLL 소모품 버퍼는 3D 프린터의 원격 필라멘트 공급을 위해 설계된 장치로, 소모품의 장착 및 제거 작업을 간소화하는 것을 목표로 합니다. 이 버퍼에는 피딩과 리트랙트를 위한 두 개의 버튼이 장착되어 있어 사용자가 전동 피딩 및 리트랙트 작업을 수행할 수 있습니다.

### 버퍼 기능:

 	3D 프린터의 익스트루더가 필라멘트를 피딩할 때, 만약 원격 소모품 랙을 잡아당기면, 버퍼는 이 동작을 감지하고 자동으로 노즐 방향으로 필라멘트를 보냅니다. 매번 잡아당길 때마다 버퍼는 이 필라멘트 공급 작업을 반복합니다. 이를 통해 익스트루더가 소모품을 잡아당기는 것을 효과적으로 줄이고, 익스트루더의 추진력을 안정적으로 유지하여 인쇄 품질의 안정성을 높일 수 있습니다.

### 필라멘트 부족 감지 기능:

 	버퍼에는 필라멘트 부족 감지기가 내장되어 있어, 버퍼 내에 소모품이 없음을 감지하면 3D 프린터에 인쇄를 일시 정지하라는 신호를 보냅니다. 이 기능을 사용하려면 신호 핀을 연결하고 3D 프린터의 설정을 수정해야 합니다. 그러나 버퍼 기능은 설정 수정 없이 바로 사용할 수 있습니다.

 	이러한 기능 설계를 통해 LLL 소모품 버퍼는 소모품 작업을 간소화할 뿐만 아니라 3D 인쇄 과정의 안정성과 인쇄 품질을 향상시킵니다.

* 기능을 사용하기 전에 소모품을 연결해야 하며, 그렇지 않으면 버튼을 통해 모터를 제어할 수 있습니다.

## 설정 참조

    ```cfg
    [filament_switch_sensor 필라멘트 부족 감지]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 필라멘트 부족
    insert_gcode:
            M118 필라멘트 추가
    ```
