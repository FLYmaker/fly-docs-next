# LLL 버퍼 사용

    LLL 소모품 버퍼는 3D 프린터 원격 필라멘트 공급을 위해 설계된 장치로, 소모품의 적재와 제거 작업을 간소화하는 것을 목표로 합니다. 이 버퍼는 두 개의 버튼을 갖추고 있으며, 각각이 필라멘트 공급과 후퇴를 위해 사용됩니다. 이를 통해 사용자는 전동으로 필라멘트를 공급하거나 후퇴시킬 수 있습니다.

### 버퍼 기능:

    3D 프린터의 압출기가 필라멘트를 공급할 때, 원격 소모품 랙을 당기면 버퍼가 이 동작을 감지하고 자동으로 노즐 방향으로 필라멘트를 공급합니다. 매번 당길 때마다 버퍼는 이 공급 작업을 반복합니다. 이를 통해 압출기가 소모품을 당기는 것을 효과적으로 줄이고, 압출기의 추진력을 안정적으로 유지하여 인쇄 품질의 안정성을 높일 수 있습니다.

### 단선 감지 기능:

    버퍼에는 단선 감지기가 내장되어 있어서, 버퍼 내에 소모품이 없음을 감지하면 3D 프린터에 인쇄를 일시 정지하라는 신호를 보냅니다. 이 기능을 사용하려면 신호 핀을 연결하고 3D 프린터의 설정을 수정해야 합니다. 반면, 버퍼 기능은 설정을 수정하지 않고도 바로 사용할 수 있습니다.

    위의 기능 설계를 통해, LLL 소모품 버퍼는 소모품 작업을 단순화할 뿐만 아니라 3D 인쇄 과정의 안정성과 인쇄 품질을 향상시킵니다.



* 사용 시 주의 사항: 소모품을 연결한 후에 기능을 사용할 수 있습니다. 그렇지 않으면 버튼으로 모터를 제어할 수만 있습니다.

    ```cfg
            M118 재료 투입
    ```
