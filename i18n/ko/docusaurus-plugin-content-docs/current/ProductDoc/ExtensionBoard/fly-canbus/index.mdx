# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.png').default} size="40%" align="left" />


## 제품 특징

    * CAN 및 전원을 1:4로 분할하여 한 번에 네 개의 CAN 도구 보드를 확장할 수 있습니다.
    * 보드에 RJ11 인터페이스를 장착하여 CAN 프로토콜을 사용하는 시스템과 연결 및 통신하도록 설계되었습니다. RJ11이 CAN 시스템에서 표준 인터페이스가 아니지만, 우리 제품은 이를 원활하게 연결할 수 있는 가능성을 제공합니다.
    * 사용자 친화적인 설계로 인쇄물을 통해 직접 레일에 설치할 수 있습니다.

## 제품 사양

    * 통신 인터페이스: RJ11*1, XH2.54*5
    * 전원 인터페이스: KF128-5.028-2P*5
    * 입력 전압: 5V-24V
    * 호환성: RRF/Klipper/등 CAN

## 주의사항

    :::warning 주의사항

    * 전원 입력값이 얼마이면 그만큼 출력됩니다.
    * 선의 순서를 잘못 연결하지 마세요.
    * 확장 보드를 설치하기 전에 상위 기기의 전원을 꺼주세요.
    * 확장 보드를 습기나 먼지가 많은 환경에서 사용하지 마세요.
    * 장치를 설치할 때 도구 보드의 전압 사양에 주의하여 손상되지 않도록 하세요.

    :::
