# FLY-G2T

    FLY-G2T 다기능 확장 보드는 라즈베리 파이와 FLY 상위 기기 기능을 확장하기 위해 설계된 고성능 모듈입니다. UART 통신을 지원하며, FPC 인터페이스를 통합하여 TFT-V2를 외부에 연결할 수 있으며, 또한 5V 팬 제어를 지원합니다. 이로 인해 산업 자동화, 3D 프린터, 홈 자동화, 로봇 제어 또는 HMI와 외부 통신이 필요한 모든 경우에서 라즈베리 파이의 상호작용성과 사용성을 강화하는 이상적인 솔루션이 됩니다.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-g2t/img/g2t.png').default} size="60%" align="left" />

## 제품 특징

    * 호환성: 여러 버전의 라즈베리 파이와 호환, 라즈베리 파이 Zero/3B/3B+/4B 등.
    * UART to RS232: 안정적인 시리얼 통신을 제공하여 RS232 인터페이스를 통해 라즈베리 파이와 다른 하드웨어 장치 간에 데이터를 교환할 수 있게 합니다.
    * SPI 캐패시티브 터치 스크린: FPC 인터페이스를 통해 연결된 TFT-V2 캐패시티브 스크린으로 직관적인 터치 조작을 제공하며, 그래픽 사용자 인터페이스가 필요한 애플리케이션에 적합합니다. 예: klipperscreen 등.
    * 5V 팬 지원: 5V 팬 인터페이스를 통합하여 팬을 연결하여 장시간 작동하거나 고부하 상황에서도 상위 기기의 안정성을 보장합니다.
    * 플러그 앤 플레이 설계: 확장 보드는 설치가 쉬워 상위 기기에 직접 삽입 및 제거가 가능하며, 추가적인 배선이나 복잡한 설정이 필요 없습니다.
    * 콤팩트 사이즈: 확장 보드의 크기가 적절하여 프로젝트에 통합하기에 공간을 많이 차지하지 않습니다.

## 제품 사양

    * 통신 인터페이스: UART to RS232
    * 디스플레이 인터페이스: SPI to FPC 인터페이스
    * 팬 전원: 5V
    * 호환성: 라즈베리 파이 Zero/3B/3B+/4B, FLY Pi V2 등

## 주의사항

    :::warning 주의사항

    * RS232 연결 툴보드를 사용할 때는 반드시 상위 기기와 `공통 접지`를 해야 합니다!!!!!

    :::

    * 확장 보드를 설치하기 전에 상위 기기의 전원이 꺼져 있는지 확인하십시오.
    * 습기나 먼지 환경에서 확장 보드를 사용하지 마십시오.
    * 스크린을 설치할 때는 전원을 켜지 않은 상태에서 스크린 케이블 방향에 주의하여 설치하십시오, 손상되지 않도록 하세요.
    * 팬을 설치할 때는 팬의 전류와 전압 사양에 주의하여 설치하십시오, 손상되지 않도록 하세요.
