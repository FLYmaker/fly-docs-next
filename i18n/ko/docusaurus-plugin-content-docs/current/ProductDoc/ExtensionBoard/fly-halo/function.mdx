# FLY_halo

## 1.  홈페이지

    히트베드나 히트엔드를 클릭한 후, 화면 정상에서 시계방향으로 슬라이드하여 도달하고자 하는 온도까지 이동한 후 다시 히트베드나 히트엔드를 클릭하면 온도를 설정할 수 있습니다.

    MCU 온도와 호스트 온도를 표시하려면 다음 설정을 추가해야 합니다.

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # MCU와 연관(기본값)

    [temperature_sensor host]        # 호스트 온도
    sensor_type: temperature_host     # 호스트와 연관
    ```


    :::warning 주의사항

    * 최고 온도는 설정에서 읽어온 값에 따라 다릅니다.

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.png').default} size="50%" align="left" />
