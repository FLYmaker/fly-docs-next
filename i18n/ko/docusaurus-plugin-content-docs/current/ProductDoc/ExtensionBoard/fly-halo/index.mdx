# 제품 소개

    * 주요 컨트롤러는 ESP32-S3R8를 사용하며, ESP32보다 성능이 뛰어납니다. 8MB PSRAM과 16MB Flash가 장착되어 있습니다. 화면은 IPS 화면으로 해상도가 높고, 밝기와 시야각이 넓으며 색상이 세밀합니다. 정전식 터치 지원

## 제품 특징

    :::tip

    * 현재 총 6개의 페이지, 좌우 및 상하 슬라이딩으로 전환

    :::


    * 홈페이지（대기, 인쇄, 정지, 오류 등 상태 표시）

    * 압출（압출, 역압출, 압출 온도 제어）

    * 시스템 서비스（호스트, klipper, moonraker 등 서비스의 시작, 중지, 재시작 제어）

    * 설정（halo의 다양한 매개변수 설정에 사용, wifi, ip, 언어, 백라이트 밝기 등）

    * 장치 정보（halo 정보 표시, 펌웨어 버전. 업데이트 확인 등）

    * 소모품 감지: 재료 끊김 또는 막힘 감지, 소모품 길이 감지, 특정 길이의 막힘 후 자동 정지 및 GPIO47 상승（이 모든 기능은 기본적으로 비활성화되어 있으며, 설정에서 활성화 가능）

    * 인쇄 완료 후 전원 끄기: 인쇄 완료 후 설정된 시간 동안 지연 후 호스트 전원을 끄고 GPIO48 상승（이 모든 기능은 기본적으로 비활성화되어 있으며, 설정에서 활성화 가능）
