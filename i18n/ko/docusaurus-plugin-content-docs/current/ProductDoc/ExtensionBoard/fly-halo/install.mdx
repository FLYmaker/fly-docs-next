## Voron Stealthburner에 설치

    * Halo를 Stealthburner에 설치하려면 우리가 제공하는 어댑터를 사용해야 합니다
    * 먼저 우리가 제공하는 두 개의 STL 파일을 다운로드하여 직접 출력하세요
        * [Stealthburner_main_body(Adapt_to_Halo).stl(다운로드 클릭)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(다운로드 클릭)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(다운로드 클릭)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### 어댑터 보드 및 연결 케이블 설치
    * 출력 위치는 당신의 습관에 따라 설정할 수 있으며, 반드시 팬 뒷면에 붙일 필요는 없습니다
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />


    :::warning 주의사항

    * 출력 색상 설명 **검정색이 음극 GND**, **빨간색이 양극 VCC로 DC12-24V 전압을 받아들일 수 있음**, **흰색이 GPIO47로 재료 막힘 감지에 사용됨**, **노란색이 GPIO48로 인쇄 후 정지에 사용됨**

    :::
