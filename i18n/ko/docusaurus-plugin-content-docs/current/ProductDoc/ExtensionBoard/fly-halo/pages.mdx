# Halo 페이지 조작 소개

## 페이지 구조

```bash
  인쇄 파일 페이지           홈페이지             압출 페이지            시스템 서비스             설정              장치 정보
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Print Files |<-->|     Home    |<-->|   Extrude   |<-->| Sys Service |<-->|   Setting   |<-->|  Dev Info   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blocking Det|
                   |             |
                   +-------------+
                     막힘 감지 페이지
            (설정에서 막힘 감지를 활성화해야 이 페이지가 표시됩니다)
```

    2. 클릭: 열상 온도 설정으로 이동, 고리 바를 드래그하여 온도 조정, 영역 2를 다시 클릭하여 설정 온도를 확인

## 압출 페이지

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.png').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.png').default} size="50%" align="left" />
    * 위의 그림 1의 조작 가능한 영역은 아래 번호와 대응합니다
    1. 클릭: 압출 헤드 목표 온도 +1
    2. 길게 누르기: 압출 온도 설정으로 이동, 고리 바를 드래그하여 온도 조정, 다시 클릭하여 설정 온도를 확인
    3. 클릭: 압출 헤드 목표 온도 -1
    4. 클릭: 압출
    5. 위아래로 스크롤: 압출 또는 회수 길이를 선택, 단위 mm
    6. 클릭: 회수
    7. 길게 누르기: 위의 그림 2의 압출 속도 설정으로 이동, 위아래로 스크롤하여 압출 속도 선택, 단위 mm/s, 선택 후 주변 빈 영역을 클릭하여 나가기
