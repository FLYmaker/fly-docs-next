# 사용 가이드

## 주의 사항

    * 첫 사용 시 반드시 네트워크에 연결해야 합니다. WiFi가 인터넷에 연결되지 않으면, halo는 초기 설정 정보를 가져올 수 없습니다.
    * 설정 페이지의 모든 설정은 **설정 저장**을 클릭한 후 장치가 재부팅된 후에 적용됩니다.
    * 재료 막힘 감지 기능은 기본적으로 비활성화되어 있습니다. 설정에서 재료 막힘 감지를 활성화하고 재부팅한 후에야 홈 화면에서 위로 스와이프하여 진입할 수 있습니다.
    * 인쇄 후 자동 종료 기능은 기본적으로 비활성화되어 있으며, 설정에서 활성화할 수 있습니다.
    * 모든 팝업 창에서 인쇄 취소 또는 일시 정지를 필요로 하는 버튼은 반드시 길게 눌러야 작동하며, 단순 클릭은 무효합니다(오작동 방지).
    * 시스템 서비스 페이지에서 Host 서비스는 상위 기기의 호스트입니다.
    * 공식 펌웨어를 사용할 때는 주 컨트롤러 칩이나 FLASH 칩을 교체하지 마십시오.

    :::tip

    * 주의: `moonraker.conf` 설정 파일 크기가 4MB를 초과하면 정상적으로 사용할 수 없습니다. 비정상이 발생할 경우 `moonraker.conf` 설정 파일의 크기를 줄여주십시오.

    :::

## 시스템 펌웨어 업데이트

    * halo 버전 업데이트는 전체 업데이트와 OTA 증분 업데이트로 나뉩니다.
        * 전체 업데이트: 하위 레벨의 큰 업그레이드가 포함되며, USB를 통해 전체 펌웨어 패키지를 번인해야 합니다.
        * 증분 업데이트: 일반적인 OTA 업그레이드로, 장치 정보 페이지에서 업데이트를 확인할 수 있습니다.

## 스마트폰으로 QR 코드를 스캔하고 WIFI 설정

    :::tip

    * 일부 모바일 기기는 WiFi에 연결하기 위한 QR 코드 스캔 기능이 없을 수 있습니다. 이 경우, 핫스팟 이름이 **FLY-halo-xxxxxxxx**인 핫스팟에 WiFi 설정에서 직접 연결할 수 있습니다.

    :::

    :::tip

    * 핫스팟에 연결한 후 웹 페이지가 자동으로 뜨지 않으면, 브라우저를 열어 링크에 직접 접속해야 합니다.

    :::

    :::tip

    * HALO는 2.4GWiFi만 사용할 수 있습니다.

    :::

    :::tip

    * Halo 화면에서는 숨겨진 WiFi를 검색할 수 없습니다.

    :::
