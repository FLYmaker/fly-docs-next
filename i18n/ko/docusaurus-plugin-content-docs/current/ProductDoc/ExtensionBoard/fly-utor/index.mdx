# FLY-UTOR

    FLY-UTOR는 고성능, 다목적 인터페이스 확장 솔루션으로, 사용자에게 추가 USB 인터페이스와 전원 확장 기능을 제공하며, UART를 RS232 통신으로 변환하는 것을 지원합니다. 이 확장 보드는 설계가 간결하고 설치가 용이하여 추가 인터페이스와 전원 출력이 필요한 다양한 응용 분야에 적합합니다.

## 제품 특성

    * 이중 USB 포트 확장: 확장 보드는 두 개의 추가 USB 인터페이스를 제공하여 USB 2.0 표준을 지원하며, 다양한 USB 장치와의 호환성 및 고속 데이터 전송을 보장합니다.
    * UART에서 RS232로의 변환 지원: 내장된 UART에서 RS232 변환기를 통해 사용자가 직렬 장치를 시스템에 연결하고 장치의 통신 능력을 확장할 수 있습니다.
    * 이중 전원 인터페이스 확장: 두 개의 전원 인터페이스를 제공하여 사용자가 동시에 두 개의 외부 전원을 확장할 수 있도록 하여 추가 전원 요구를 충족시킵니다.
    * 안정적인 전원 설계: 확장 보드에는 연결된 장치가 안정적으로 작동할 수 있도록 안정적인 전원 회로가 설계되어 있습니다.
    * 쉬운 설치: 표준 크기 설계로 다양한 환경에 적합하며, 추가 드라이버 설치가 필요 없습니다.
    * 컴팩트 사이즈: 확장 보드의 크기가 적절하여 과도한 공간을 차지하지 않고 다양한 프로젝트에 통합하기 편리합니다.

## 제품 사양:

    * 사용 가능한 인터페이스: USB X2, UART에서 RS232로 X2, 전원 인터페이스 X2
    * 전원 인터페이스: 12V/24V 확장
    * 호환성: USB 인터페이스가 있는 상위 컴퓨터

## 주의사항:

    * 설치 전에 전원을 차단하고 모든 장치를 종료하십시오.
    * 멀티 USB 확장 보드를 메인보드 USB 포트나 다른 USB 확장 인터페이스에 삽입합니다.
    * 확장 보드에 필요한 USB 장치 및/또는 전원을 연결합니다.
    * UART에서 RS232 기능을 사용하려면 해당 직렬 장치를 연결합니다.
    * 전원을 다시 연결하고 장치를 시작합니다.
    * 시스템은 확장 보드를 자동으로 감지하고 설정해야 합니다. 필요 시 설정 관련 주의사항을 참조하십시오.
