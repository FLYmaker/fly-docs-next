# BL 펌웨어

    :::tip

    * 출하 시 이미 `bootloader`가 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::

## 번인 모드 진입


    * 번인 모드로 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 메인보드 전원이 완전히 꺼진 상태에서 BOOT 버튼을 길게 누르고 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다. WINDOWS에 적합합니다.
    * 두 번째 방법: 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 길게 누르고 명령어를 입력합니다. 호스트 머신에 적합합니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="100%" align="center" />

                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
