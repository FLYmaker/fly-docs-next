# 디프 스위치 소개

:::tip
FLY Gemini는 디프 스위치를 통해 상위 기기와 연결됩니다. 디프 스위치 위치가 맞지 않으면 정상적으로 사용할 수 없습니다.
:::

:::danger 참고사항
디프 스위치를 지정된 위치로 설정하지 않으면 상위 기기와 정상적으로 연결할 수 없습니다.
:::

## 온보드 상위 기기 연결

    * 디프 스위치 `1`, `2`를 오른쪽으로 이동하여 하위 기기를 내장 USB를 통해 상위 기기와 연결합니다.
    * 디프 스위치 `3`, `4`를 왼쪽으로 이동하지 않으면 하위 기기와 상위 기기 연결에 영향을 미칩니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.png').default} size="100%" align="center" />
    * 이 방법으로 디프 스위치를 조정한 후, 아래 USB 포트는 하위 기기에 의해 점유됩니다. 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="100%" align="center" />


## 외장 상위 기기 연결

    * 디프 스위치 `3`, `4`를 오른쪽으로 이동하여 하위 기기를 내장 USB를 통해 상위 기기와 연결합니다.
    * 디프 스위치 `1`, `2`를 왼쪽으로 이동하지 않으면 하위 기기와 상위 기기 연결에 영향을 미칩니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.png').default} size="100%" align="center" />
