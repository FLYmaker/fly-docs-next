## 시스템 번인 준비

    * 카드 리더기
    * TF 카드 한 장, 용량은 최소 16GB 이상 128GB 이하의 TF 카드, TF 카드의 전송 속도는 class10 이상이어야 합니다
    * 해당 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button> 다운로드
    * 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번인 도구</Button> 다운로드

## 시스템 번인

:::info
TF 카드 번인과 TF 카드를 통해 M2WE에 번인하는 것은 두 가지 중 하나만 사용하면 됩니다
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치되어 있으면, 시스템은 기본적으로 메모리 카드에서 부팅합니다
* M2WE는 TF 카드를 사용하여 시스템을 번인하는 것이 좋습니다. USB를 통해 M2W에 번인하는 경우 드라이버 호환성 등의 이유로 번인에 실패할 수 있으므로, [메모리 카드를 통해 M2WE에 번인](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 권장합니다
* 다운로드한 시스템 이미지가 일치하는지 확인하십시오. 일치하지 않으면 부팅할 수 없습니다
:::
