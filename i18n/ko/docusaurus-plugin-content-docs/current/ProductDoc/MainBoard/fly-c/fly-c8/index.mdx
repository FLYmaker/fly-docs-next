## 제품 특징

    * 32비트 168MHz 주파수의 ARM Cortex-M0+ 시리즈 STM32F407VGT6 메인 컨트롤러 칩 사용
    * TMC 드라이버의 SPI와 UART 작동 모드 내장
    * DIAG 기능 핀 내장, 단순한 점퍼 캡을 통한 무한 위치 복귀 사용 가능
    * 24V, 12V, 5V 전압 선택 가능한 숫자 제어 팬, 부스터 다운 모듈 구매 불필요
    * 새로운 플래시 펌웨어 업데이트 방식, FLY-Tools를 통해 메인보드 펌웨어 직접 업데이트 가능
    * 다중 전원 관리, 포트, 리미트, 드라이버 등이 단락되어도 상위 컴퓨터의 작동에 영향을 주지 않음
    * RGB 인터페이스 예약 (12V, 5V) 전압 선택 가능
    * SPI 인터페이스 예약, 외부 가속도 센서 연결을 통한 가속도 보상 가능
    * TFT 스크린과 HDMI 스크린 일선 통신 지원
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 진동 보정 플러그인 numpy와 FLY-Tools가 사전 설치되어 있어, 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 접근할 수 있음 (TF 카드 또는 M2WE 구매 필요)

## 제품 사양

### 상위 컴퓨터
    * CPU: 고성능 전지 H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 필링 레이트 2.7gpixel/s 이상
    * RAM: 1GB DDR3 (GPU와 공유)
    * ROM: 최대 128GB SDCARD 지원
    * 주변 장치: Spi x1, TFT X1, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1, 3.5mm 오디오 인터페이스 X1
    * 인터페이스: 메인보드 내장 M.2 인터페이스, WIFI, eMMC 등 확장 가능 (비공개 프로토콜, FLY π 전용이 아닌 M.2 장치 연결 금지)
    * 5V 제어 가능한 팬 인터페이스 예약, 상위 컴퓨터 냉각에 사용 가능
    * TFT 스크린과 HDMI 스크린 FPC 일선 통신 지원
    * CAN 브리지 인터페이스 예약
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 진동 보정 플러그인 numpy와 FLY-Tools가 사전 설치되어 있어, 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 접근할 수 있음 (TF 카드 또는 M2WE 구매 필요)

### 하위 컴퓨터

    * MCU: STM32F407VGT6
    * 펌웨어:  KLIPPER/RRF/Marlin
    * 12-24V 직류 전원 지원, 전원 공급 더 안정적, 부스터 다운 모듈 불필요
    * 히트베드 포트 최대 출력 전류: 15A
    * 히터 로드 포트 최대 출력 전류: 6A
    * 팬 포트 최대 출력 전류: 1A
    * 드라이버 작동 모드 지원: SPI, UART, STEP/DIR
    * 지원 드라이버: 외부 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등
    * 확장 인터페이스: BLTouch (Servos, Probe), RGB X2, UART X1, 12864 X1
    * 팬: 2선 숫자 제어 팬 X6 (24V, 12V, 5V 선택 가능)
    * 온도 센서: 3회로 NTC 100K
    * 지원 기계 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
