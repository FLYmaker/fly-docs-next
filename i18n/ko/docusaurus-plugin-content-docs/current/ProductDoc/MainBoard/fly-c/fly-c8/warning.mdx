# 메인보드 주의사항

## 전원 표시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.webp').default} size="80%" align="center" />
    * 타입-C 전원만 연결하고 다른 메인보드 전원을 연결하지 않았을 때 적어도 두 개의 램프가 켜져야 합니다. `3.3V`와 `5V`
    * 메인보드 전원을 연결한 후 메인보드 전원 램프는 `3.3V`, `5V`, `VCC`가 켜져야 합니다.

### 문제 해결


    :::danger 주의

    * 외부 장치가 전혀 연결되지 않은 상태에서（드라이버, 리미트 스위치, 12864 등）
    * 타입-C 전원만으로 최소한 `3.3V`가 항상 켜져야 합니다.

    :::

    * 메인보드 전원과 드라이버 전원을 연결한 후 `5V` 램프가 켜지지 않으면 고객 서비스에 연락해야 합니다.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다.     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 리미트 포트

    * 연결 순서에 주의하십시오. 삼선 리미트는 5V, GND, S 세 가지를 주의해서 연결해야 합니다. 잘못 연결하면 메인보드가 손상될 수 있습니다.
    * 양선은 S와 GND에 직접 연결하면 됩니다.
    * PL08와 다른 근접 스위치는 신호 포트에 다이오드를 외부에 연결하여 전압을 3.3V 이하로 낮춰야 합니다.
