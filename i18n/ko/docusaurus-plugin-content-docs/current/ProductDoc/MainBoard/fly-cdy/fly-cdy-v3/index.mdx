## 제품 특징

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * 드라이버 전압 선택 지원, 기본 5V
    * RRF 펌웨어 지원



## 제품 사양

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연 시간이 적으며 연결이 더 안정적
    * 펌웨어: Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 디스플레이: 직렬 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * repap 펌웨어를 사용하면 확장 ESP32 WiFi 모듈을 사용할 수 있으며 1.6m/s 지원
    * 독특한 PWM 팬 MOS 보드 설계로 손상 시 직접 교체 가능, 고객의 유지보수 난이도 감소
    * PCB는 2OZ 동판 두께를 채택하여 더 큰 전류를 지원할 수 있음
    * 3개의 ADC 인터페이스, 최대 3개의 서미스터 지원
    * 6개의 IO 인터페이스, 평준화 및 한계 설정에 사용 가능
    * 4개의 히터 바 인터페이스, 최대 4개의 히터 바 지원
