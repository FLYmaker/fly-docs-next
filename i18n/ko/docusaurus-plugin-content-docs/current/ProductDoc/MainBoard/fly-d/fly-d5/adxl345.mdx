## 참조 설정

    ```cfg
    [adxl345]
    cs_pin: PB12
    spi_bus: spi2 # 하드 SPI, 하드 SPI와 소프트 SPI 중 하나를 선택하세요. 기본값은 하드 SPI입니다.
    # spi_software_mosi_pin: PB15 # 소프트 SPI, 하드 SPI와 소프트 SPI 중 하나를 선택하세요. 기본값은 하드 SPI입니다.
    # spi_software_miso_pin: PB14
    # spi_software_sclk_pin: PB13

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # 일반적으로 히팅 베드의 중심, 기기의 실제 상황에 따라 수정하세요.
    ```
