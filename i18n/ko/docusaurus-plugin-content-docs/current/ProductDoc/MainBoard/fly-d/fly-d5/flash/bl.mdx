# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * D5는 USB 연결 사용
    * D5는 2024년 9월 1일 출고 시 이미 Katapult 펌웨어가 사전 설치되어 있음
    :::

## 플래시 모드 진입

    :::tip

    왼쪽은 `BOOT 버튼`, 오른쪽은 `RESET 버튼`

    :::

    * 플래시 모드로 진입하는 방법은 두 가지가 있습니다
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 케이블을 메인보드와 장치에 연결
    * 두 번째 방법: 데이터 전송이 가능한 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 채로 RST 버튼을 누르고, RST 버튼을 놓은 후 BOOT 버튼을 놓음
