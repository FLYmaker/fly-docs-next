# 펌웨어 번인 설명

:::danger 알아두어야 할 사항
* 메인보드 펌웨어 번인에는 Katapult를 통해 Klipper를 번인해야 합니다.
* 메인보드에 이미 Katapult가 번인되어 있음을 확신하면, 마지막에 있는 Katapult 펌웨어 번인은 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 번인되지 않았을 경우를 대비한 것입니다.
* 리셋 버튼을 두 번 빠르게 누른 후 메인보드 상에 LED가 깜빡이면, 메인보드가 Katapult 펌웨어로 들어갔음을 나타냅니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    :::tip

    * 주의: 당신은 단 하나의 펌웨어만 번인하면 됩니다.

    :::
    
    * D5는 다음과 같은 방법으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰성 있고 안정적입니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않습니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 시간 오류 등이 발생할 수 있습니다.
