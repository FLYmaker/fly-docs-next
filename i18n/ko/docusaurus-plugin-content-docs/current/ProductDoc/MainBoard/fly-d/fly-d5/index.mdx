# 소개

> Mellow Fly-D5는 광저우 마그네슘 멜로우 전자과학기술 유한공사 Fly3D 보드 개발팀이 출시한 고성능 3D 프린터 메인보드로, 최대 5개의 스테퍼 모터를 지원합니다.


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/d5.png').default} size="60%" align="center" />

## **1. 제품 특징**

* 32비트 48MHz 클럭의 ARM Cortex-M0+ 시리즈 STM32F072RBT6 메인 컨트롤러 칩 사용
* CAN 버스 연결 지원, 데이터 전송 더 안정적이고 지연 시간 줄이며 연결 더 안정적
* 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
* 보드에 DIAG 기능 핀 내장, 간단한 점퍼 캡 삽입만으로 무한정 홈 지원 사용 가능

## 2. 제품 사양

* MCU: STM32F072RBT6
* 펌웨어: Klipper/Marlin
* 12-24V DC 전원 지원, 전원 공급 더 안정적이고, 강압 모듈 필요 없음
* 드라이버 작동 모드 지원: SPI, UART, STEP/DIR
* 지원 드라이버: 외장 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등
* 확장 인터페이스: BLTouch(Servos, Probe), UART X1, 12864 X1
* 팬: 2선 제어 팬 X2, 병렬 팬 X2, 항상 켜진 팬 X2
* 온도 센서: 2채널 NTC 100K
* 히팅 인터페이스: 2채널 제어 인터페이스 10A 지원
* 지원 기계 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
