# CAN 브리징

CAN 버스는 실시간 응용을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호를 전송하기 위해 트위스트 페어를 사용할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신을 위해 사용되며, 이로 인해 비싸고 무거운 배선 다발을 대체합니다. 이 프로토콜의 견고성은 자동화 및 산업용 응용 분야로 확장되었습니다. CAN 프로토콜의 특성에는 직렬 데이터 통신의 완전성, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 능력이 포함됩니다.
3D 프린터는 CAN 버스를 사용하여 프린트 헤드와 메인 보드 간의 연결선 수를 줄입니다. 원래는 10여 개의 선이 필요했지만, CAN을 사용하면 네 개의 선만으로 충분하여 배선 수를 크게 줄이고 배선 난이도를 낮춥니다.
본 장에서는 CAN 브리징 사용에 대한 간략한 개요만 제공합니다.

## 도구 보드가 CAN으로 메인 보드와 연결됨

    * 도구 보드의 전원선은 CAN 포트에 연결되지 않고 12V-24V 전원선에 연결된다는 점에 주의하십시오.

    * 120Ω 점퍼를 연결해야 한다는 점에 주의하십시오.
