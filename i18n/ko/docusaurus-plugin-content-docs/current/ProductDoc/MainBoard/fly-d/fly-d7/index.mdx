# 소개

> Fly-D7은 광저우 마그넷론 전자 과학 기술 유한 회사 Fly3D 메인보드 연구 개발 팀에서 출시한 고성능 3D 프린터 메인보드로, 최대 7개의 스테퍼 모터를 지원합니다.


    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.png').default} size="60%" align="center" />

## **1. 제품 특징**

    * 32비트 주파수 48MHz의 ARM Cortex-M0+ 시리즈 STM32F072RBT6 주제어 칩을 채택
    * CAN 버스 연결을 지원하여 데이터 전송이 더 안정적이고 지연이 적으며 연결이 더 안정적임
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 보드 내장 DIAG 기능 핀을 통해 간단한 점퍼 케이프 연결만으로 무한 위치 복귀 지원

## 2. 제품 사양

    * MCU: STM32F072RBT6
    * 펌웨어: Klipper/Marlin
    * 12-24V DC 전원 지원, 더 안정적인 전원 공급, 승압 모듈 생략 가능
    * 드라이버 작동 모드 지원: SPI, UART, STEP/DIR
    * 지원 드라이버: 외장 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등
    * 확장 인터페이스: BLTouch(Servos, Probe), UART X1
    * 팬: 2선 제어 팬 X2, 병렬 팬 X2, 항상 켜진 팬 X2
    * 온도 센서: NTC 100K 3개
    * 히팅 인터페이스: 3개의 제어 가능 인터페이스, 10A 지원
    * 지원 기계 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
