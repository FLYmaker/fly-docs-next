# 메인보드 주의사항

## 전원 표시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/power.webp').default} size="80%" align="center" />
    * 타입-C 전원만 사용하고 다른 메인보드 전원을 연결하지 않은 경우 최소 두 개의 램프가 켜져야 합니다. `3.3V`와 `5V`
    * 메인보드 전원을 연결한 후 메인보드 전원 램프가 `3.3V`、`5V`、`VCC`로 켜져야 합니다.

### 문제 해결


    :::danger 주의

    * 외부 장치가 전혀 연결되지 않은 상태에서（드라이버, 한계 스위치, 12864 등）
    * 타입-C 전원만 사용하면 적어도 `3.3V`와 `5V` 두 개의 LED가 항상 켜져야 합니다.

    :::

    * 램프가 전혀 켜지지 않으면, 단락이 있을 가능성이 있으므로 고객 서비스에 문의하십시오.
    * `3.3V`와 `5V` 중 하나가 깜빡이면, 단락이 있을 가능성이 있으므로 고객 서비스에 문의하십시오.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다.     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 한계 스위치

    * 연결 순서에 주의하세요. 삼선 한계 스위치는 5V, GND, S 세 개의 선을 잘못 연결하면 메인보드가 손상될 수 있습니다.
    * 두 선이면 S와 GND에 직접 연결하면 됩니다.
    * PL08 및 기타 근접 스위치는 신호 포트에 다이오드를 외부에 연결하여 전압을 3.3V 이하로 낮춰야 합니다.

## 펌웨어

    * 버튼 문제로 인해 교육용으로 Katapult 펌웨어를 사용하여 펌웨어를 굽습니다.
    * 컴퓨터를 사용하여 Katapult 펌웨어를 굽고, Katapult 펌웨어를 통해 Klipper를 굽는 것이 좋습니다.
