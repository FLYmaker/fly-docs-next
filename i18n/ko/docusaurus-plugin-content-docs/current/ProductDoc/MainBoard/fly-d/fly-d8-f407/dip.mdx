# 온보드 주변 장치 소개

    * 딥 스위치를 오른쪽으로 누르면 Type-C 포트 옆의 USB 포트가 활성화됩니다.
    * 딥 스위치를 왼쪽으로 누르면 Type-C 포트 옆의 RS232 포트가 활성화됩니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/mode_dip.png').default} size="80%" align="center" />

## USB 포트 소개
    
    * 카메라와 같은 장치와 메인보드를 연결할 수 있습니다.
