## 번인 모드 진입

    :::tip

    왼쪽은`RST 버튼`, 오른쪽은`BOOT 버튼`

    :::

    * 번인 모드에 진입하는 방법은 두 가지가 있습니다
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누른 후 데이터 전송 가능한 데이터 케이블을 메인보드와 장치에 연결합니다
    * 두 번째 방법: 데이터 전송 가능한 데이터 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 상태에서 RST 버튼을 누릅니다. 그 후 RST 버튼을 먼저 놓고 BOOT 버튼을 놓습니다
