# 펌웨어 플래시 설명

:::danger 참고사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드에 이미 Katapult가 설치되어 있는지 확인되면 마지막의 Katapult 펌웨어 플래시는 필요 없습니다. 이것은 메인보드의 Katapult 펌웨어가 설치되지 않은 경우를 대비한 것입니다.
* 두 번 빠르게 리셋 버튼을 눌러 LED가 깜빡이면 메인보드가 Katapult 펌웨어에 진입한 것입니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    :::tip

    * 하나의 펌웨어만 설치하면 됩니다.

    :::
    
    * D8은 다음과 같은 방법으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰할 수 있습니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않습니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 등의 오류가 발생할 수 있습니다.
