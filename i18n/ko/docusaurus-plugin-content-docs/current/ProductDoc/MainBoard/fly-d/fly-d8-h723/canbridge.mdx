# CAN 브리징

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호를 전송하기 위해 트위스트 페어 케이블을 사용하며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신에 사용되며, 비용이 많이 들고 무거운 배선을 대체합니다. 이 프로토콜의 견고함 덕분에 자동화 및 산업 응용 분야에서도 사용됩니다. CAN 프로토콜의 특징은 직렬 데이터 통신의 완전성, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 능력을 포함합니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드와 메인보드 간의 배선 수를 줄입니다. 원래는 몇십 개의 선이 필요했지만, CAN을 사용하면 네 개의 선만으로 충분하여 배선 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리징 사용을 간략히 개괄합니다.

## 도구 보드가 CAN을 통해 메인보드에 연결

    * 도구 보드의 전원 선은 CAN 포트에 연결하는 것이 아니라, 12V-24v 전원 선에 연결해야 합니다.

    * 120Ω 점퍼를 연결해야 합니다.
