# 온보드 주변기기 소개

    * 딥 스위치를 오른쪽으로 누르면 Type-C 포트 옆의 USB 인터페이스가 활성화됩니다.
    * 딥 스위치를 왼쪽으로 누르면 Type-C 포트 옆의 RS232 인터페이스가 활성화됩니다.

## USB 포트 소개
    
    * 외부 카메라 및 메인보드 등의 장치를 연결할 수 있습니다.
