## 펌웨어 모드 진입

    :::tip

    왼쪽은 `RST 버튼`, 오른쪽은 `BOOT 버튼`

    :::

    * 펌웨어 모드 진입 방법에는 두 가지가 있습니다
    * 첫 번째: 메인보드 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누른 후 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다
    * 두 번째: 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 상태에서 RST 버튼을 누르고, RST 버튼을 먼저 놓은 후 BOOT 버튼을 놓습니다
