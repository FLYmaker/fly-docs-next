# 펌웨어 플래시 설명

:::danger 알아두어야 할 사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다
* 만약 메인보드에 이미 Katapult가 플래시되어 있는 경우, 마지막에 있는 Katapult 펌웨어 플래시는 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 플래시되지 않았을 경우를 대비한 것입니다
* 두 번 빠르게 리셋 버튼을 누르면 메인보드에 LED가 깜빡이는지 확인할 수 있습니다. 깜빡이면 메인보드가 Katapult 펌웨어에 진입한 것입니다
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    :::tip

    * 주의: 여러분은 이 중 하나의 펌웨어만 플래시하면 됩니다

    :::
    
    * D8은 다음 방식으로 상위 기기와 연결할 수 있습니다
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰할 수 있고 안정적입니다
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 시간 초과 등의 오류가 발생할 수 있습니다.
