# 소개

> Mellow Fly-D8는 광저우 맥론 전자 과학 기술 유한 회사의 Fly3D 보드 개발 팀이 출시한 고성능 3D 프린터 메인보드로 최대 8개의 스테퍼 모터를 지원합니다.


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-h723/img/d8.png').default} size="80%" align="center" />

## 제품 특징

    * 메인보드의 리미트 스위치 등 인터페이스는 5V 전원 공급을 위해 격리 보호를 했으며, POWER 전원을 연결하지 않으면 전원이 출력되지 않습니다
    * 32비트 550MHz 주파수의 ARM Cortex-M7 시리즈 STM32H723 메인 컨트롤러 칩을 채택했습니다
    * CAN 버스와 RS232 연결을 지원하여 데이터 전송이 더 안정적이고 지연 시간이 더 짧으며 연결이 더 안정적입니다
    * 메인보드에 BOOT 및 RESET 버튼이 예약되어 있어 사용자가 USB를 통해 DFU 모드로 펌웨어를 업데이트할 수 있습니다
    * 전원 인터페이스에 역접속 방지 보호가 있어, DIY 시 전원 선을 잘못 연결하여 보드가 타는 것을 방지합니다
    * 두 개의 USB 2.0 인터페이스를 제공하여 더 많은 연결 가능성을 확장했으며, 그 중 하나의 USB 인터페이스는 스위치를 통해 RS232 인터페이스로 전환할 수 있어 다양한 장비와의 연결 요구에 부합합니다

# 제품 사양

    * 메인 컨트롤러 칩: 32비트 550MHz 주파수의 ARM Cortex-M7 시리즈 STM32H723 메인 컨트롤러 칩을 채택했습니다
    * 펌웨어: Marlin 2.0/Klipper
    * 드라이브 모드 지원: TMC: UART/SPI
    * 8개의 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 디스플레이: 직렬 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 자동 베드 레벨링 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
    * 입력 전압: 12V/24V 지원
    * 6개의 리미트 스위치 인터페이스, 정밀한 범위 제한 및 위치 제어를 실현할 수 있습니다
    * 2개의 RGB 인터페이스, 다채로운 시각 효과를 제공할 수 있습니다
    * 3개의 히팅 튜브 인터페이스, 압출기와 히트 베드에 독립적인 제어를 제공합니다
    * 4개의 서미스터 인터페이스, 최대 네 그룹의 온도를 동시에 모니터링하여 정밀한 온도 제어를 실현할 수 있습니다
    * USB 인터페이스: USB 2.0 인터페이스를 두 개 지원하며, 그 중 하나는 스위치를 통해 RS232로 전환할 수 있습니다
    * 두 개의 USB 2.0 인터페이스를 제공하여 더 많은 연결 가능성을 확장했으며, 그 중 하나의 USB 인터페이스는 스위치를 통해 RS232 인터페이스로 전환할 수 있어 다양한 장비와의 연결 요구에 부합합니다
