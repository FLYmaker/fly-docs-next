# RS232 연결

RS232 표준 인터페이스(또는 EIA RS-232)는 일반적으로 사용되는 직렬 통신 인터페이스 표준 중 하나로, 다양한 장치 간의 데이터 전송에 널리 사용됩니다. 이 인터페이스의 안정성과 신뢰성 덕분에 많은 장치의 주요 통신 방식으로 선택됩니다. 이 프로토콜은 표준 5V와 다른 신호 레벨을 사용하여 데이터를 전송하므로, 신호 간섭을 최소화할 수 있습니다. 3D 프린터 시스템에서 RS232는 CAN에 비해 더 높은 데이터 전송 속도, 더 나은 안정성, 더 간단한 설정 및 상위 기기와의 더 좋은 호환성을 제공합니다.

## 툴보드가 메인보드에 RS232로 연결

    * 툴보드의 전원선은 RS232 신호선에 연결되지 않으며, 12V-24V 전원선에 연결됩니다.

    * D8 메인보드 펌웨어는 재컴파일할 필요가 없습니다.
