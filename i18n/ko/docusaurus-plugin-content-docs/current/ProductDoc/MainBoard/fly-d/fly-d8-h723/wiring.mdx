# PIN 단자 설명

:::warning 주의사항
이미지 해상도가 너무 높아 웹 페이지에서 확인할 수 없으므로, 마우스 커서를 연결도 위에 놓고 오른쪽 클릭하여 이미지를 데스크탑에 저장한 후 확인하세요!
