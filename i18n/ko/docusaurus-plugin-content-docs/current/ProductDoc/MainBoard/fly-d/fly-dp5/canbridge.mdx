# CAN 브리지

CAN 버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호 전송을 위해 트위스트 페어 케이블을 사용하며, 세계에서 가장 널리 사용되는 필드 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 구성 요소 간의 통신에 사용되어, 비싸고 무거운 전력 분배 케이블을 대체합니다. 이 프로토콜의 견고함 덕분에 자동화 및 산업 응용 분야에서도 활용되고 있습니다. CAN 프로토콜의 특징은 다음과 같습니다: 직렬 데이터 통신의 완전성 제공, 실시간 지원, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 능력.
3D 프린터에서 CAN 버스는 프린터 헤드를 메인보드에 연결하는 선을 줄이기 위해 사용됩니다. 원래는 수십 개의 선이 필요했으나, CAN을 사용하면 네 개의 선만으로 충분하여, 선의 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리지 사용에 대해 간략히 개요를 설명합니다.

## 도구 보드를 사용하여 메인보드에 CAN 연결

    * 도구 보드 전원선은 CAN 포트에 연결되지 않고, 12V-24V 전원선에 연결됩니다.

    * 120Ω 점퍼를 연결해야 합니다.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/wiring/120.png').default} size="80%" align="center" />

## 메인보드 펌웨어 업데이트

* DP5 메인보드에 `USB 브리지 CAN 펌웨어 설정`이 적용된 펌웨어를 확실히 설치하십시오.
