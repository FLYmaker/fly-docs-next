# 펌웨어 번인 설명

:::danger 주의사항
* 메인보드 펌웨어 번인은 Katapult를 통해 Klipper를 번인해야 합니다.
* 메인보드에 이미 Katapult가 번인되어 있는 경우, 마지막에 있는 Katapult 펌웨어 번인은 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 번인되지 않은 경우를 대비한 것입니다.
* 재빨리 리셋 버튼을 두 번 누른 후, 메인보드에 LED가 깜빡이면 메인보드가 Katapult 펌웨어로 진입한 것을 의미합니다.
:::

## 필요한 펌웨어를 어떻게 판단할까요?

    :::tip

    * 주의하세요, 한 가지 펌웨어만 번인하면 됩니다.

    :::
    
    * DP5는 다음과 같은 방식으로 상위 컴퓨터에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰성 있음
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있음
    * <Button variant="contained" disableElevation href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않음
        * 단점: `KlipperScreen`과 함께 사용할 경우, 홈 포지션 시간 초과 등의 오류가 발생할 수 있음.
