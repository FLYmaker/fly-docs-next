# 소개

> Mellow Fly-DP5는 광저우 마그네슘 멜로우 전자 기술 유한 회사 Fly3D 보드 개발팀에서 출시한 고성능 3D 프린터 메인보드로, 최대 5개의 스테핑 모터를 지원합니다.


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.png').default} size="60%" align="center" />

## **1. 제품 특징**

* 32비트 주파수 48MHz의 ARM Cortex-M0+ 시리즈 STM32F072RBT6 마이크로컨트롤러 칩 채택
* CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연 시간이 적으며 연결이 더 안정적입니다.
* 드라이버 지원: DP2209 및 DP560
* 온보드 DIAG 기능 핀, 단순히 점퍼 캡을 삽입하고 제거하여 무한 위치 복귀를 지원할 수 있습니다.

## 2. 제품 사양

* MCU: STM32F072RBT6
* 펌웨어: Klipper/Marlin
* 12-24V 직류 전원 지원, 전원 공급이 더 안정적이며 승압 모듈 필요 없음
* 드라이버 작동 모드 지원: SPI, UART, STEP/DIR
* 지원 드라이버: 외부 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등 지원
* 확장 인터페이스: BLTouch(서보, 프로브), UART X1, 12864 X1
* 팬: 2선 제어 팬 X2, 병렬 팬 X2, 항상 켜져 있는 팬 X2
* 온도 센서: NTC 100K 2개
* 히터 인터페이스: 10A 지원하는 2개의 제어 인터페이스
* 지원 기계 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
