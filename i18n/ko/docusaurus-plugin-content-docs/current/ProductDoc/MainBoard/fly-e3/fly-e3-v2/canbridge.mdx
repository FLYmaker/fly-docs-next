# CAN 브리징

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호를 전송하기 위해 트위스트 페어 케이블을 사용하며, 세계에서 가장 널리 사용되는 필드 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신에 사용되어, 비싸고 무거운 배선을 대체합니다. 이 프로토콜의 견고함은 다른 자동화 및 산업 애플리케이션으로의 사용을 확장시켰습니다. CAN 프로토콜의 특성에는 직렬 데이터 통신의 완전성, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 11비트의 주소 지정 및 오류 검출 능력이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드와 메인보드 간의 연결 선을 줄이기 위해 사용됩니다. 원래는 십여 개의 선이 필요했으나, CAN을 사용하면 네 개의 선만으로 충분하여 배선 수를 크게 줄이고 배선 난이도를 낮출 수 있습니다.
이 장에서는 CAN 브리징의 사용을 간략히 개괄합니다.

## 도구 보드를 사용하여 메인보드에 CAN 연결하기

    * 도구 보드의 전원선은 CAN 포트에 연결되지 않으며, 12V-24v 전원선에 연결됩니다.
