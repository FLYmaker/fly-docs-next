# BL펌웨어

    :::tip

    * 출하 시 이미 `bootloader`가 설치되어 있으니 다시 설치할 필요 없습니다.

    :::

## 번인 모드 진입

    :::tip

    오른쪽은 `BOOT 버튼`, 왼쪽은 `RESET 버튼`입니다.

    :::

    * 번인 모드로 진입하는 두 가지 방법이 있습니다.
    * 첫 번째: 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채 데이터 전송이 가능한 데이터 케이블을 메인보드에 연결합니다.
    * 두 번째: 데이터 전송이 가능한 데이터 케이블을 메인보드에 연결한 후, BOOT 버튼을 누른 채 RST 버튼을 누르고, RST 버튼을 놓은 후 마지막으로 BOOT 버튼을 놓습니다.
