## 제품 사양

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * CAN 버스 연결 지원, 데이터 전송 더 안정적, 지연 시간 더 짧음, 연결 더 안정적
    * 펌웨어: Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 디스플레이: 직렬 터치스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * repap 펌웨어 사용 시, 확장 ESP32 WiFi 모듈 사용 가능, 1.6m/s 지원
    * 독특한 PWM 팬 MOS 보드 설계, 손상 시 직접 교체 가능, 고객 유지보수 어려움 감소
    * PCB는 2OZ 구리 두께 사용, 더 큰 전류 지원 가능
    * 6개의 ADC 인터페이스, 최대 6개의 열전쌍 지원
    * 7개의 IO 인터페이스, 평준화 및 리미트 스위치에 사용 가능
    * 7개의 히터 바 인터페이스, 최대 7개의 히터 바 지원
