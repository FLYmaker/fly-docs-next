# 딥 스위치 소개

:::tip
FLY Gemini는 딥 스위치를 통해 호스트 컴퓨터에 연결됩니다. 딥 스위치 위치가 잘못되면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의
딥 스위치가 지정된 위치로 설정되어야 호스트 컴퓨터에 정상적으로 연결할 수 있습니다.
:::

## 온보드 호스트 컴퓨터 연결

    * 딥 스위치 `1`, `2`를 오른쪽으로 설정하여 하위 기기와 호스트 컴퓨터를 내장된 USB를 통해 연결합니다.
    * 딥 스위치 `3`, `4`를 왼쪽으로 설정하지 않으면 하위 기기와 호스트 컴퓨터의 연결에 영향을 미칩니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.png').default} size="50%" align="center" />
    * 이 방법에 따라 딥 스위치를 설정하면 하단 USB 포트가 하위 기기에 의해 점유됩니다. 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="50%" align="center" />


## 외부 호스트 컴퓨터 연결

    * 딥 스위치 `3`, `4`를 오른쪽으로 설정하여 하위 기기와 호스트 컴퓨터를 내장된 USB를 통해 연결합니다.
    * 딥 스위치 `1`, `2`를 왼쪽으로 설정하지 않으면 하위 기기와 호스트 컴퓨터의 연결에 영향을 미칩니다.
