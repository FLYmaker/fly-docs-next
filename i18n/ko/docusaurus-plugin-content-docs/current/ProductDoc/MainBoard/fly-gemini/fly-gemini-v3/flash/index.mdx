# 펌웨어 번인 설명

## 어떤 펌웨어가 필요한지 어떻게 판단할까요

    :::tip

    * 주의: 당신은 단 하나의 펌웨어만 번인하면 됩니다

    :::
    
    * Gemini V3는 다음과 같은 방법으로 상위 기기와 연결할 수 있습니다
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰할 수 있고 안정적
        * 단점: 여러 USB 장치가 데이터 전송에 영향을 줄 수 있음
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않음
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 등의 오류가 발생할 수 있음
