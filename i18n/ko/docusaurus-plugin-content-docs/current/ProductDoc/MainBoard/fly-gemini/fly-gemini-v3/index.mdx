# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.png').default} size="80%" align="center" />

## 제품 특징
    * 내장된 상위 컴퓨터로, 라즈베리 파이를 상위 컴퓨터로 대체할 수 있으며, 1GB의 내장 DDR3 메모리를 가지고 있습니다.
    * 5V 전원 보호 기능 강화로 칩 손상 가능성을 줄였습니다.
    * MCU 칩의 스토리지 카드 설정을 취소하여 펌웨어 업데이트가 더 용이해졌습니다.
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, resonance compensation 플러그인 numpy 및 FLY-Tools가 사전 설치되어 있어, 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 사용할 수 있습니다(별도로 TF 카드 또는 M2WE가 필요).

## 제품 사양

    * CPU: 고성능 Allwinner H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 필링 레이트 2.7gpixel/s 이상
    * RAM: 1GB DDR3(그래픽과 공유)
    * ROM: 최대 128GB SDCARD 지원
    * 주변 장치: Spi x2, UART x2, USB x4(USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1(100M), CAN x1
    * 인터페이스: 내장 M.2 인터페이스, WIFI, eMMC 등 확장 가능(비공개 프로토콜, FLY π 전용 M.2 장치가 아닌 것은 연결하지 마십시오)
    * 12-24V 직류 전원 지원, 전원 공급이 더 안정적이고, 전압 강하 모듈을 생략 가능
    * 상위 컴퓨터 냉각용으로 5V 제어 가능한 팬 인터페이스를 예비했습니다.
    * CAN 확장 인터페이스 예비
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, resonance compensation 플러그인 numpy 및 FLY-Tools가 사전 설치되어 있어, 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 사용할 수 있습니다(별도로 TF 카드 또는 M2WE가 필요).
    * 드라이버: 고전압 드라이버 4개
    * MCU: STM32F405
