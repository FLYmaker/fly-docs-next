# 상위기 SSH 연결


* 시리얼 포트 또는 WIFI를 통해 장치에 연결할 수 있습니다
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요
* SD 카드 또는 M2WE에 **해당 상위기 시스템**을 번인했는지 확인하세요
* **상위기가 정상적으로 시작되지 않으면 드라이버, 리미터, 팬 등 주변 장치를 제거하세요!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 편집하기 편리합니다!!**
* **펌웨어를 편집하려면 WIFI 또는 이더넷으로 상위기 연결 교습서를 참고하세요**

        ## 네트워크 IP를 사용하여 상위기 SSH 연결

        * 상위기가 연결된 **라우터 백엔드**에 로그인하세요
        * 시스템이 **정상적으로 시작**했는지 확인하세요
        * **상위기의 사용자 이름과 비밀번호를 알고 있는지** 확인하세요. FLY 시스템의 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다
        * 메인보드가 시작되고 네트워크에 연결되어 있으면 IP를 사용하여 SSH에 연결할 수 있습니다
        * MobaXterm을 열고 다음과 같이 설정합니다. IP는 앞서 확인한 IP를 입력하거나 라우터 백엔드에서 확인할 수 있습니다

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계에 상위기의 사용자 이름을 입력합니다**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 첫 연결 시 이 창이 팝업됩니다. `Accept`를 클릭하세요

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호 입력 프롬프트가 나타나면 상위기 비밀번호를 입력하고 엔터를 누릅니다

        **주의: 여기서 입력하는 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요**

        FLY 시스템의 기본 사용자 이름: `fly` , 비밀번호: `mellow`
