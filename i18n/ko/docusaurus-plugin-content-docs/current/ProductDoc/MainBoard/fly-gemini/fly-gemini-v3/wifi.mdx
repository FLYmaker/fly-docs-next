# WIFI 연결


## Gemini에 안테나 설치

    * 빨간색 화살표가 가리키는 곳이 WIFI 안테나 인터페이스입니다.
    * WIFI 안테나 포트는 `ipex1`입니다.
    * Gemini에 안테나를 설치하지 않으면 신호가 약해 인터넷 연결이 불가능할 수 있습니다.
