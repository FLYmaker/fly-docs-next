# CAN 브리징

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스로, 신호를 전송하기 위해 트위스트 페어 선을 사용하며, 세계에서 가장 널리 사용되는 필드 버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신에 사용되어 고가이고 무거운 배선 뭉치를 대체합니다. 이 프로토콜의 견고함 덕분에 다른 자동화 및 산업 응용 분야로 확장되었습니다. CAN 프로토콜의 특성에는 완전한 직렬 데이터 통신, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 동시에 11비트의 주소 지정 및 오류 검출 능력이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린터 헤드와 메인보드 간의 연결선 수를 줄입니다. 원래는 10여 개의 선이 필요했지만, CAN을 사용하면 4개의 선만으로도 충분하여 배선 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리징 사용에 대한 간략한 개요만을 다룹니다.

## 도구 보드를 사용하여 메인보드에 CAN 연결

    * 도구 보드의 전원선은 CAN 포트에 연결되는 것이 아니라 12V-24V 전원선에 연결됩니다.

    * 120Ω 점퍼를 연결해야 합니다.

## 메인보드 펌웨어 업데이트

* D5 메인보드에 'USB 브리징 CAN 펌웨어 설정'을 사용하여 펌웨어를 업데이트해야 합니다.
