# Katapult 펌웨어

    :::tip

    * 출하 시 이미 `katapult`가 사전 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::

## 번인 모드 진입

    :::tip

    원 안에 있는 것이 `BOOT 버튼`입니다.

    :::

    * 번인 모드로 진입하려면 메인보드의 전원을 완전히 차단한 상태에서 BOOT 버튼을 누르고 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결하세요.
