    <CAN />

    * GPIO 핀을 선택하여 마이크로컨트롤러가 시작할 때 설정, 입력: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` 키를 눌러 **Save configuration**이 나타나면 `Y` 키를 다시 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 구성 파일이 저장되고 명령줄 인터페이스로 돌아갑니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 좀 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래 내용이 출력되면 컴파일이 성공한 것입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
