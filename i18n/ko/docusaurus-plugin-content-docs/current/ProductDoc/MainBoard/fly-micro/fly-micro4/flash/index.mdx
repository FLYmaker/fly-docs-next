# 펌웨어 플래시 설명

:::danger 주의사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 만약 메인보드에 이미 Katapult가 플래시되어 있는 경우, 마지막 단계의 Katapult 펌웨어 플래시는 필요 없습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 플래시되지 않은 경우에 대한 예방책입니다.
* 메인보드가 Katapult 펌웨어에 들어갔는지 확인하려면, 리셋 버튼을 두 번 빠르게 눌러 메인보드 상의 LED가 깜빡이는지 확인하세요. 깜빡이면 Katapult 펌웨어에 들어갔다고 봅니다.
:::

## 필요한 펌웨어를 어떻게 판단하나요?

    :::tip

    * 주의: 당신은 하나의 펌웨어만 플래시하면 됩니다.

    :::
    
    * E3 V2는 다음과 같은 방법으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰할 수 있음
        * 단점: 여러 USB 장비가 있을 경우 데이터 전송에 영향을 줄 수 있음
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장비를 연결할 수 있으며 USB 포트를 차지하지 않음
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 포지션 타임아웃 등의 오류가 발생할 수 있음.
