# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## 제품 특징

    * MCU: Raspberry Pi rp2040, 듀얼 코어 ARM Cortex-M0+@133MHz
    * CAN 브리징 지원
    * 4개의 2209 내장
    * DIAG 기능 핀 내장, 간단한 점퍼 캡 연결만으로 무한위치 복귀 지원

## 제품 사양

    * MCU: Raspberry Pi rp2040, 듀얼 코어 ARM Cortex-M0+@133MHz
    * 펌웨어: Klipper
    * 12-24V DC 전원 지원, 더 안정적인 전원 공급, 전압 하강 모듈 생략
    * 드라이버 작동 모드 지원: UART, STEP/DIR
    * 확장 인터페이스: BLTouch(서보, 프로브), 근접 스위치 X1
    * 팬: 2선 제어 가능 팬 X2, 항상 켜진 팬 X1
    * 온도 센서: 2개의 NTC 100K
    * 히터 인터페이스: 2개의 제어 가능 인터페이스, 10A 지원
