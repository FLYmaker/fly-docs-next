# BL 펌웨어

    :::tip

    * 출고 시 `bootloader`가 이미 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::

## 펌웨어 모드 진입


    * 펌웨어 모드로 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 메인보드 전원이 완전히 꺼진 상태에서 BOOT를 단락시키고 데이터 전송 기능이 있는 데이터 케이블을 메인보드와 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송 기능이 있는 데이터 케이블을 메인보드와 장치에 연결한 후, BOOT를 단락시키고 RST 키를 누릅니다. 그런 다음 RST 키를 놓고 BOOT 점퍼를 분리합니다.

            4. 펌웨어 설치

            :::tip

            하나의 펌웨어만 설치하면 됩니다.

            :::


            * 아래는 katapult 펌웨어 설치 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * 아래는 bootloader 펌웨어 설치 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. 아래 그림의 화살표 부분에 ``File downloaded successfully``라는 내용이 나타나면 설치가 성공적으로 완료된 것입니다.
