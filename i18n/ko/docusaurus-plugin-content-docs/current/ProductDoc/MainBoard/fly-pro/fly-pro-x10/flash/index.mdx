# 펌웨어 플래싱 설명

    :::warning 주의사항

    * PRO X10에는 Katapult가 미리 플래시되어 있습니다.
    * BL을 한 번 플래시한 후에는 추가로 플래시할 필요가 없습니다.
    * PRO X10의 BL은 Katapult 펌웨어와 TF 카드로 플래시된 BL 펌웨어 두 가지가 있습니다.
    * 필요하다면 <Button variant="contained" disableElevation href="./bl">BL 펌웨어</Button>를 선택하여 다시 플래시할 수 있습니다.

    :::
    
    :::danger 알아두어야 할 사항
    * 메인보드 펌웨어는 Katapult를 통해 Klipper를 플래시해야 합니다.
    * 메인보드에 Katapult가 이미 플래시되었다면, 마지막에 있는 Katapult 펌웨어 플래싱은 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 플래시되지 않은 경우를 대비한 것입니다.
    * 리셋 버튼을 빠르게 두 번 누른 후 메인보드 상의 LED가 깜박이면 메인보드가 Katapult 펌웨어로 진입한 것을 의미합니다.
    :::

## 어떤 펌웨어가 필요한지 판단하는 방법

    :::tip

    * 한 가지 펌웨어만 플래시하면 됩니다.

    :::
    
    * Pro X10은 다음 방법으로 상위 기기와 연결할 수 있습니다:
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰성 있음
        * 단점: 여러 USB 장치가 있는 경우 데이터 전송에 영향을 줄 수 있음
    * <Button variant="contained" disableElevation href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않음
        * 단점: `KlipperScreen`과 함께 사용 시 홈 포지션 초과 등의 오류가 발생할 수 있음.
