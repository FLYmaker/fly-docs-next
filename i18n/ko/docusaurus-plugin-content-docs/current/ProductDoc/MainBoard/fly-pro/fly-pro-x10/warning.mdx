# 주의사항

## 전원 표시등
    * Type-C 전원을 사용하고 다른 메인보드 전원을 연결하지 않은 경우 `SYS_5V`와 SYS_3.3V가 켜져야 합니다.
    * 메인보드 전원 `HVIN`, `VIN`, `BOARO`, `POWER`에 연결하면 메인보드 전원 램프가 항상 켜져야 합니다.

## 드라이버

    * TP5160과 TP2209은 이름만 다르고 점퍼가 필요 없이 드라이버 모드를 선택할 수 있습니다.
    * 드라이버 포트에는 TP 드라이버만 설치할 수 있으며 다른 장치를 설치할 수 없습니다.

## 한계 포트
    * ProX10은 5V와 12V 전원 선택을 지원합니다. 근접 스위치를 사용할 경우 전압 강하를 위해 다이오드를 설치해야 합니다.

## 상위기기
    * ProX10은 외부 상위기기 사용을 지원하며, Type-C 연결을 선택한 후 사용할 수 있습니다.

## 펌웨어 번인

    * 메인보드는 TF 카드를 사용하여 번인할 수 있지만, 출하 시 기본값은 `Katapult`입니다.
    * TF 카드를 사용하여 번인하려면 BL을 다시 번인해야 합니다.
