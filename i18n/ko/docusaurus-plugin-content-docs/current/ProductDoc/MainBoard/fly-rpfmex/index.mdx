## 제품 사양

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * 펌웨어:  Klipper
    * 12-24V DC 전원 지원, 전원 공급이 더 안정적이며, 강압 모듈 필요 없음
    * 드라이버 작동 모드 지원：UART、STEP/DIR
    * 확장 인터페이스：제어 가능한 팬*2, RGB*1, 리미트 스위치*3
