# BL 펌웨어

    :::tip

    * 출하 시 이미 `부트로더`가 미리 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::

## 번인 모드 진입

    :::tip

    오른쪽이 `BOOT 점퍼`, 왼쪽이 `RESET 버튼`입니다.

    :::

    * 번인 모드로 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 메인보드의 전원을 완전히 차단한 상태에서, BOOT 버튼을 누른 상태로 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 상태에서 RST 버튼을 누르고, RST 버튼을 놓은 후 마지막으로 BOOT 버튼을 놓습니다.
