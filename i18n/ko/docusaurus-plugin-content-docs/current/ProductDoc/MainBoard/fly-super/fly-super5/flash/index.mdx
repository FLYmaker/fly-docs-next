    :::tip

    * Super5 Pro는 기본적으로 TF 카드를 사용하여 펌웨어를 번인합니다.
    * 용량이 `32G`를 초과하지 않는 TF 카드를 준비하세요.
    * 무명 메모리 카드는 번인이 실패할 수 있으므로, 가능한 아래 추천하는 메모리 카드를 사용하여 펌웨어를 업데이트하세요.
    * TF 카드의 전송 속도는 `class10` 등급 또는 그 이상이 좋으며, 샌디스크 등의 브랜드 TF 카드 사용을 권장합니다.

    :::

# 펌웨어 번인 설명

    :::tip

    * Super5 Pro는 출고 시 이미 TF 카드 bootloader가 번인되어 있어, 다시 번인할 필요가 없습니다.

    :::


## 어떤 종류의 펌웨어가 필요한지 어떻게 판단하나요?

    :::tip

    * 한 종류의 펌웨어만 번인하면 됩니다.

    :::

    * Super5 Pro는 다음과 같은 방법으로 상위 컴퓨터에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰할 수 있고 안정적입니다.
        * 단점: 여러 USB 장치가 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브릿지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않습니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 포지션 타임아웃 등의 오류가 발생할 수 있습니다.
