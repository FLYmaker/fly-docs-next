# 메인보드 주의사항

## 전원 표시등

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/power.png').default} size="80%" align="center" />
    * TYPE-C 전원만 공급하고 다른 메인보드 전원을 연결하지 않은 경우 최소한 두 개의 램프가 켜져야 합니다: `3.3V`와 `5V`
    * 메인보드 전원을 연결한 후 메인보드 전원 램프는 `3.3V`, `5V`, `12V`가 켜져야 합니다
    * 드라이버 전원0-2, 드라이버 전원3-7, 히트베드 전원을 연결한 후 메인보드의 왼쪽 하단 전원 표시등이 정상적으로 켜집니다

### 문제 확인


    :::danger 주의사항

    * 외부 장치 없이 (드라이버, 리미트 스위치, 12864 등)
    * TYPE-C 전원만으로 최소한 `3.3V`와 `5V` 두 개의 LED가 항상 켜져야 합니다

    :::

    * `3.3V`와 `5V`가 켜지지 않으면 메인보드 5V 점퍼를 확인하십시오
    * 점퍼를 교체한 후에도 동일하고 typec가 정상적으로 전원을 공급하는지 확인한 후 고객 서비스에 연락해야 합니다
    * `3.3V`, `5V` 중 하나의 램프가 깜빡이면 칩이 쇼트된 것을 의미하며 고객 서비스에 연락해야 합니다
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/5v.png').default} size="60%" align="center" />

## 드라이버
    * 드라이버 전원으로 48V를 사용할 수 있지만 모든 드라이버가 48V를 사용해야 합니다
    * 메인보드는 외부 드라이버를 지원합니다     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 리미트 포트

    * 리미트 포트가 작동하지 않으면 리미트 포트를 교체하거나 드라이버의 무한 리미트 단자를 잘라야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.png').default} size="80%" align="center" />




## 펌웨어 플래싱

    * 메인보드는 TF 카드를 사용하여 플래싱합니다
    * 이전에 교육 문제로 인해 DFU를 사용하여 플래싱했다면, 펌웨어를 업데이트하려면 <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/bl">BL을 다시 플래시</Button>해야 합니다
