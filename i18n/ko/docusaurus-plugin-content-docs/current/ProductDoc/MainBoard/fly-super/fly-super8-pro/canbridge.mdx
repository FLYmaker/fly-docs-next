# CAN 브리징

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호를 전송하기 위해 트위스트 페어 케이블을 사용하며, 전 세계에서 가장 널리 사용되는 필드버스 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신에 사용되어 고가이고 무거운 배선을 대체합니다. 이 프로토콜의 견고함 덕분에 다른 자동화 및 산업 응용 분야로도 확장되었습니다. CAN 프로토콜의 특성에는 직렬 데이터 통신의 완전성, 실시간 지원, 최대 1Mb/s의 전송 속도, 동시에 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린터 헤드가 메인보드에 연결되는 배선 수를 줄입니다. 원래 몇 십 개의 선이 필요했지만, CAN을 사용하면 단 네 개의 선만으로 가능해져 배선 수를 크게 줄이고 배선 작업을 용이하게 합니다.
이 장에서는 CAN 브리징 사용에 대한 간략한 개요만을 제공합니다.

## 도구 보드가 메인보드와 CAN 연결 사용

    * 도구 보드의 전원선은 CAN 포트에 연결되는 것이 아니라, 12V-24V 전원선에 연결됩니다.
