# BL 펌웨어

    :::tip

    * 공장 출고 시 이미 `bootloader`가 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::
## 부팅 모드 진입


    * 부팅 모드 진입 방법은 두 가지가 있습니다
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서, BOOT를 단락시키고 데이터 전송이 가능한 USB 케이블의 한쪽을 메인보드에, 다른 한쪽을 장치에 연결합니다
    * 두 번째 방법: 데이터 전송이 가능한 USB 케이블의 한쪽을 메인보드에, 다른 한쪽을 장치에 연결한 후, BOOT를 단락시키고 RST 버튼을 누릅니다. 그런 다음 RST 버튼을 놓고 BOOT 단락을 해제합니다.
