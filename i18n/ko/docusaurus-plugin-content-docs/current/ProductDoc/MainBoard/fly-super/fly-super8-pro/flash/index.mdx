    :::tip

    * Super8 PRO는 기본적으로 TF 카드를 사용하여 펌웨어를 번인합니다
    * 용량이 `32G`를 초과하지 않는 TF 카드를 준비하십시오
    * 저가형 메모리 카드는 번인 실패를 초래할 수 있으므로 가능한 아래 권장 메모리 카드를 사용하여 펌웨어를 업데이트하십시오
    * TF 카드의 전송 속도는 `class10` 등급 이상이 좋습니다. 샌디스크와 같은 브랜드의 TF 카드를 사용하는 것이 좋습니다

    :::

# 펌웨어 번인 설명

    :::tip

    * Super8 Pro는 출고 시 TF 카드 부트로더가 미리 설치되어 있어 다시 설치할 필요가 없습니다

    :::

## 필요한 펌웨어를 어떻게 판단하나요

    :::tip

    * 주의하십시오. 단 하나의 펌웨어만 번인하면 됩니다

    :::
    
    * Super8 PRO는 다음과 같은 방식으로 상위 컴퓨터에 연결할 수 있습니다
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰할 수 있고 안정적입니다
        * 단점: 여러 USB 장비가 데이터 전송에 영향을 줄 수 있습니다
    * <Button variant="contained" disableElevation href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장비를 연결할 수 있으며 USB 포트를 차지하지 않습니다
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 포지션 타임아웃 등의 오류가 발생할 수 있습니다.
