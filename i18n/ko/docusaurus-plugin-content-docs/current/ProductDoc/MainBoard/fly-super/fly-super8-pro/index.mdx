# 제품 사양

    * 32비트 주파수 550 MHz의 ARM Cortex-M7 시리즈 STM32H723ZGT6 메인 컨트롤러 칩 채용
    * 펌웨어: Marlin 2.0/Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 8개의 고압 드라이버 포지션, 8개의 독립 전동기 드라이브, 5개의 압출기, 10개의 PWM 팬 지원
    * 디스플레이: 직렬 터치스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1개의 고압 IN 포트, 자동 평판 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
    * 선택 가능한 리미트 스위치 전원: 5V/12V/24V
    * repap 펌웨어 사용 시, 내장된 ESP32 WiFi 모듈을 사용할 수 있으며, 1.6m/s를 지원
    * 독특한 PWM Fan MOS 보드 디자인으로, 손상 시 직접 교체 가능하여 고객의 유지 보수 난이도를 낮춤, 두 개의 2핀 팬 인터페이스가 4핀 PWM 팬을 지원
    * PCB는 2OZ 구리 두께를 사용하여 더 큰 전류를 지원
    * 6개의 ADC 인터페이스, 최대 6개의 서미스터 지원
    * 8개의 IO 인터페이스, 평판 및 리미트에 사용 가능
    * 5개의 히팅 로드 인터페이스, 최대 5개의 히팅 로드 지원
    * 2개의 PWM 인터페이스
