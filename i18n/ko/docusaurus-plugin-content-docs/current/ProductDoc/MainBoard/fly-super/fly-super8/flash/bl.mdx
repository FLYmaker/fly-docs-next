# BL 펌웨어

    :::tip

    * 출고 시 이미 `bootloader`가 미리 설치되어 있으므로 다시 설치할 필요가 없습니다.

    :::

## 번인 모드 진입


    * 번인 모드로 진입하는 방법에는 두 가지가 있습니다.
    * 첫 번째: 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 단자를 연결한 후 데이터 전송 기능이 있는 케이블의 한쪽 끝을 메인보드에, 다른 쪽 끝을 장치에 연결합니다.
    * 두 번째: 데이터 전송 기능이 있는 케이블의 한쪽 끝을 메인보드에, 다른 쪽 끝을 장치에 연결한 후 BOOT 단자를 연결하고 RST 키를 누른 다음 RST 키를 놓고 BOOT 점퍼를 분리합니다.
