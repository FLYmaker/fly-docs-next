    * TF 카드의 전송 속도는 `class10` 등급 이상이어야 하며, 샌디스크 등의 브랜드 TF 카드 사용을 권장합니다.

    :::

# 펌웨어 플래싱 설명


## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    :::tip

    * 펌웨어는 하나만 플래싱하면 됩니다.

    :::
    
    * Super8은 다음과 같은 방식으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰할 수 있고 안정적입니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브리징 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 오류 등이 발생할 수 있습니다.
