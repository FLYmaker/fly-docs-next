# 제품 사양


    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * 펌웨어: Marlin 2.0/Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 8개의 고압 드라이버 슬롯, 8개의 독립 모터 드라이버, 5개의 익스트루더, 10개의 PWM 팬 지원
    * 디스플레이: 직렬 터치스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1개의 고압 IN 포트, 자동 평면화 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
    * 선택 가능한 리미트 스위치 전원: 5V/12V/24V
    * repap 펌웨어 사용 시, 내장된 ESP32 WiFi 모듈을 사용하여 1.6m/s 지원
    * 독특한 PWM 팬 MOS 보드 디자인으로, 손상 시 직접 교체 가능하여 고객의 수리 난이도를 낮춤, 두 개의 2핀 팬 포트가 4핀 PWM 팬을 지원
    * PCB는 2OZ 구리 두께를 사용하여 더 큰 전류를 지원
    * 6개의 ADC 인터페이스, 최대 6개의 열 센서 지원
    * 8개의 IO 인터페이스, 평면화 및 리미트에 사용 가능
    * 5개의 히팅 로드 인터페이스, 최대 5개의 히팅 로드 지원
    * 2개의 PMW 인터페이스
