# 메인보드 주의사항

## 전원 표시등

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/power.webp').default} size="80%" align="center" />
    * TYPE-C 전원만으로 다른 메인보드 전원을 연결하지 않을 때 최소한 두 개의 램프 `3.3V`와 `5V`가 켜져야 합니다.
    * 메인보드 전원을 연결하면 메인보드 전원 램프는 `3.3V`、`5V`、`12V`가 켜져야 합니다.
    * 드라이버 전원0-2, 드라이버 전원3-7, 열침대 전원이 연결된 후에 메인보드 좌측 하단의 전원 표시등이 정상적으로 켜집니다.

### 문제 해결


    :::danger 주의사항

    * 외부 장치가 전혀 없는 경우(드라이버, 한계 스위치, 12864 등)
    * TYPE-C 전원만으로 최소한 `3.3V`와 `5V` 두 개의 LED가 항상 켜집니다.

    :::

    * `3.3V`와 `5V`가 켜지지 않으면, 메인보드 5V 점퍼를 확인하세요.
    * 점퍼를 교체한 후에도 동일하고, typec가 정상적으로 전원을 공급하는지 확인 후, 고객 서비스에 문의하세요.
    * `3.3V`、`5V`와 `12V` 중 하나의 램프가 깜빡이면, 칩이 단락되었다는 의미로 고객 서비스에 문의해야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/5v.webp').default} size="60%" align="center" />

## 드라이버

    * 드라이버 전원에 주의하세요. 드라이버 전원은 두 갈래로 나뉩니다. 첫 번째는 드라이버 3-7에 전원을 공급하고, 두 번째는 0-2에 공급합니다.
    * 메인보드는 외부 드라이버를 지원합니다.     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 한계 스위치 포트

    * 마지막 드라이버 포트(Drive7)만 한계 스위치 점퍼가 있습니다. 다른 드라이버 포트는 직접 한계 스위치 포트에 연결됩니다.
    * 한계 스위치 포트가 작동하지 않으면 한계 스위치 포트를 교체하거나 드라이버의 한계 스위치 핀을 잘라내세요.
