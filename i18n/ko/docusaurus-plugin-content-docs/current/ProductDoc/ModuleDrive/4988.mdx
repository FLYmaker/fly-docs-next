## Klipper 참조 설정
    * 4988은 `tmc5160 stepper_x` 또는 `tmc2209 stepper_x`와 같은 드라이버 구성을 필요로 하지 않으며, 구성 시 정상적으로 사용할 수 없습니다.
    * `microsteps:`를 `16`으로 설정하십시오
