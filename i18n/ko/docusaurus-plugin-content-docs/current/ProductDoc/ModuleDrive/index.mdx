# 소개

    * 몇 가지 드라이버 주의사항


## 드라이버 설치

    * EN이 왼쪽 상단에 정렬되어야 설치됨을 주의하세요
    * FLY 메인보드 드라이버 포트의 EN 핀은 모두 왼쪽 상단에 있습니다
    * FLY의 DP5, PRO X10, MMU는 TP 드라이버만 사용할 수 있습니다
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

## 드라이버 설정

    * TP 드라이버와 일반 드라이버의 설정 방법은 동일합니다
    * FLY-2240 드라이버는 UART 연결을 지원하지만, 저항을 수동으로 변경해야 합니다
    * 드라이버 샘플링 저항에 주의해야 합니다. 잘못된 설정은 모터나 드라이버를 손상시킬 수 있습니다
