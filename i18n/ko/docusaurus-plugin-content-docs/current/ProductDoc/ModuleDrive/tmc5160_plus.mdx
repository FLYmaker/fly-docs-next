# TMC 5160 Plus

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/5160plus.png').default} size="100%" align="center" />

## 기본 매개변수

    * 입력 전압 : 12-60V
    * 최대 전류 : 10A
    * 적용 가능 : Marlin/Klipper/RRF 펌웨어
    * 드라이버 모드 지원 : TMC : SPI
    * 마이크로스텝 보간 microPlyer가 있는 스텝/방향 인터페이스
    * 최대 해상도는 256 세분화
    * stealthChop2의 조용한 작업과 부드러운 동작 지원
    * 중간 주파수 공진 억제 지원
    * spreadCycle 고동적 모터 제어 촙퍼 지원
    * dcStep 로드 관련 속도 제어 지원
    * stallGuard2 고정밀 무센서 모터 로드 검출 지원
    * coolStep 전류 제어 지원, 최대 75%까지 절약 가능

## 메인보드 점퍼 및 드라이버 연결 설명

## 메인보드 참조 설정

    :::danger

    드라이버 샘플링 저항은 0.033이므로 잘못 설정하지 마십시오.

    :::


    <Tabs>
        <TabItem value="드라이버 EN 핀으로 RGB를 제어할 때" default>
        
        * 다이얼을 FAN 포트 쪽으로 돌리세요

        ```
        [stepper_x]
        step_pin: 
        dir_pin: 
        # enable_pin: !

        [tmc5160 stepper_x]
        cs_pin: 
        spi_software_mosi_pin: 
        spi_software_miso_pin: 
        spi_software_sclk_pin: 
        run_current: 1.0
        interpolate: False
        sense_resistor: 0.033
        stealthchop_threshold: 0
        driver_DISS2G: 1
        driver_DISS2VS: 1
        ```
        </TabItem>


        <TabItem value="메인보드 RGB 핀으로 RGB를 제어할 때" default>
        
        * 다이얼을 FAN 포트의 다른 방향으로 돌리세요

        ```
        [stepper_x]
        step_pin: 
        dir_pin: 
        enable_pin: !
        
        [tmc5160 stepper_x]
        cs_pin: 
        spi_software_mosi_pin: 
        spi_software_miso_pin: 
        spi_software_sclk_pin: 
        run_current: 1.0
        interpolate: False
        sense_resistor: 0.033
        stealthchop_threshold: 0
        driver_DISS2G: 1
        driver_DISS2VS: 1
        ```  

        </TabItem>

        <TabItem value="RGB 참조 설정" default>

        * 플러그인을 통해 드라이버 LED를 제어 <Button variant="contained" disableElevation href="/docs/DebugDoc/addon/led.mdx">클릭하면 이동합니다</Button>

        ```
        [neopixel TP_led]
        pin: 
        chain_count: 6
        # Number of LEDs
        # LED 개수
        color_order: GRB
        initial_RED: 0.4    
        initial_GREEN: 0.8
        initial_BLUE: 1
        initial_WHITE: 0.0
        #66CCFF 

        ```  

        </TabItem>

    </Tabs>

## 전류 경고

    :::danger

    TMC5160 Plus는 10A를 지원하지만, 과도한 전류는 모터를 손상시킬 수 있으며, 냉각을 강화해야 합니다.

    :::
