# TMC 5160 Pro V1.2

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/5160pro.webp').default} size="80%" align="center" />

## 기본 매개변수

    * 입력 전압: 24-48V
    * 최대 전류: 6A
    * 적합: Marlin/Klipper/RRF 펌웨어
    * 드라이브 모드 지원: TMC: SPI
    * 마이크로스텝 보간 microPlyer가 있는 스텝/방향 인터페이스
    * 최대 해상도 256 분할
    * stealthChop2 무음 작동 및 부드러운 동작 지원
    * 중간 주파수 공진의 공진 억제 지원
    * spreadCycle 고동적 모터 제어 초퍼 지원
    * dcStep 하중 관련 속도 제어 지원
    * stallGuard2 고정밀 비센서 모터 하중 감지 지원
    * coolStep 전류 제어 지원, 최대 75%까지 절약 가능

## 메인보드 점퍼 설명

    * TMC5160 Pro는 SPI 점퍼를 사용
    * FLY 드라이브의 SPI 핀은 왼쪽에서 아래로 네 번째 핀, CS 핀은 SPI 핀입니다. 메인보드가 해당 핀과 일치하지 않으면 SPI를 사용할 수 없습니다.

## TMC5160 Pro 참조 설정

    :::danger

    * 드라이브 샘플링 저항은 `sense_resistor: 0.033`입니다. 잘못 설정하지 마세요.
    * spi_bus와 spi_software는 둘 중 하나만 선택하세요.
    
    :::

### SPI 모드 참조 설정

    ```cfg
    [tmc5160 stepper_x]
    cs_pin:
    spi_bus:
    # spi_software_mosi_pin:
    # spi_software_miso_pin:
    # spi_software_sclk_pin: 
    run_current: 1.0                    # 모터 운전 전류 값
    interpolate: False                  # 256 마이크로스텝 보간을 켜거나 끕니다. (켜는 경우 True, 끄는 경우 False)
    sense_resistor: 0.033
    stealthchop_threshold: 0
    driver_DISS2G: 1
    driver_DISS2VS: 1
    ```
