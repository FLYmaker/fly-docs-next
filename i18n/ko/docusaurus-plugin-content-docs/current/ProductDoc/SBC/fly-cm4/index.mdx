# 소개

      Fly-CM4는 라즈베리 파이 3B와 4B와 완전히 호환되며, 특히 라즈베리 파이 CM4를 위한 확장을 개선했습니다. 이 확장 보드는 3D 프린터에 최적화된 다양한 기능을 제공합니다. 12-24V DC 전원 공급은 3D 프린터의 정밀 제어와 연속적인 운영에 매우 중요합니다. FLY-TFT와 Fly-HDMI 화면 FPC 연결 기능을 지원하여 보다 편리한 배선 방식이 제공됩니다. Can 포트를 통해 상위 컴퓨터와 하위 컴퓨터를 직접 연결할 수 있으며, 이는 현재 매우 인기 있는 방법입니다.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-cm4/img/cm4.png').default} size="80%" align="center" />

## 특징

    * 라즈베리 파이 CM4 코어 보드 지원
    * 12V-24V DC 전원 공급 지원
    * 내장 UTOC 모듈
    * TFT 및 HDMI 일선통 지원
    * DSI/spi 화면 인터페이스 지원
    * CSI/USB 카메라 인터페이스 지원
    * 40PIN 확장 인터페이스, 라즈베리 파이와 호환
