# 소개

    Fly-Pi-lite2.1은 입문용 Klipper 호스트 보드입니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.png').default} size="80%" align="center" />

## 특징

    * CPU: 고성능 Allwinner H3 칩, 4코어 64비트 Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: 최대 128GB SDCARD 지원
    * WIFI: 내장 2.4G WiFi
    * 주변 장치: Spi x1, URAT x2, USB x2, HDMI x1
    * 인터페이스: FPC 예약, HDMI + USB FPC 일체형
    * 시스템 사전 설치: klipper, moonraker, mainsail, fulidd, klipper-screen, 진동 보정 플러그인 numpy 및 FLY-Tools, 복잡한 소스 변경 없이 초보자도 쉽게 사용 가능 (TF 카드 구매 필요)


## 업데이트 기능

    * TFT V1 인터페이스를 TFT V2로 업그레이드
    * 리셋 및 전원 끄기 핀 추가
    * 전원 표시등 추가
    * BOOT 버튼 삭제, 리셋 버튼으로 교체
    * 배선 최적화
