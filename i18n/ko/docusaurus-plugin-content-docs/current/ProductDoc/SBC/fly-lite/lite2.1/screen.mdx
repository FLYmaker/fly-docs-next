# 화면 연결 및 설정
:::info

* lite2.1은 시스템에 SSH로 연결한 후 아래 명령어를 입력해야 Fly Tools 화면 설정을 사용할 수 있습니다.
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools는 상위 기기 네트워크 연결 후 상위 기기의 `IP+:9999`로 접근해야 합니다.
* 예: `192.168.6.110:9999`

:::
