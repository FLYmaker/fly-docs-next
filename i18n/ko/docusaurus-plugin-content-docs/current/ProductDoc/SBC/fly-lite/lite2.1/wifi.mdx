# WIFI 연결

## LITE 2.1에 안테나 설치하기

    * 빨간색 화살표가 가리키는 곳이 WIFI 안테나 포트입니다
    * WIFI 안테나 포트는 `ipex1`입니다
    * LITE 2.1에 안테나를 설치하지 않으면 신호가 약해 인터넷에 연결할 수 없을 수 있습니다

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.png').default} size="100%" align="left" />  

* WIFI에 정상적으로 연결되지 않는 경우 먼저 안테나가 제대로 연결되었는지 확인한 후 아래 방법으로 WiFi에 연결하십시오
