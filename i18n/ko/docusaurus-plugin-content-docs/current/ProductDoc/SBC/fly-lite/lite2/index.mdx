# 소개

    Fly-Pi-lite2는 입문용 Klipper 호스트 컴퓨터입니다.

## 특징

    * CPU: 고성능 Allwinner H3 칩, 4코어 64비트 Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: 최대 128GB SDCARD 지원
    * WIFI: 내장 2.4G WiFi
    * 주변 장치: Spi x1, URAT x2, USB x2, HDMI x1
    * 인터페이스: FPC 예약, HDMI + USB FPC 일선통
    * 시스템 사전 설치: klipper, moonraker, mainsail, fluid, klipper-screen, 공진 보정 플러그인 numpy 및 FLY-Tools. 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 사용 가능 (TF 카드 구매 필요)
