# WIFI 연결

## LITE 2에 안테나 설치

    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 단자입니다
    * WIFI 안테나 포트는 `ipex1`입니다
    * 안테나를 LITE 2에 설치하지 않으면 신호가 약해 인터넷 연결이 불가능할 수 있습니다

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* WIFI가 정상적으로 연결되지 않으면 먼저 안테나가 연결되어 있는지 확인한 후 아래 방법을 사용하여 WiFi에 연결하십시오
