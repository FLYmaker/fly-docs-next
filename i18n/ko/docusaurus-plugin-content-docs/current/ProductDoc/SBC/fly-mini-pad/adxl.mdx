# 가속도계 사용

    :::tip

    * Mini Pad는 내장된 인터페이스를 통해 가속도계를 직접 연결하거나 도구판 등에 내장된 가속도계를 사용할 수 있습니다.

    :::

## 내장 인터페이스에 외부 가속도계 연결

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.png').default} size="80%" align="left" />

    * 참조 구성

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # 예시
    ```
