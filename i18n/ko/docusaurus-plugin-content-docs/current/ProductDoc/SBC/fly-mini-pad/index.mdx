# 소개

    Fly-MINI-Pad는 입문용 Klipper 상위 컴퓨터입니다.

## 특징

    * CPU: 고성능 Allwinner H3 칩, 4코어 64비트 Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: 최대 128GB SDCARD 지원
    * WIFI: 내장 2.4G WiFi
    * 주변 장치: SPI x1, USB x2, Type-C x1
    * 인터페이스: 세 개의 XH2.54-2P 인터페이스가 각각 지원, POWER-IN 전원 공급은 최대 24V 입력 지원, CAN-BUS는 CAN 버스 연결 지원, LI-PO는 3.7V 보호 기능이 있는 리튬 배터리 전원 공급/충전 인터페이스 지원
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, 공진 보상 플러그인 numpy 및 FLY-Tools가 사전 설치되어 있어, 소스 변경 등 복잡한 작업 없이 초보자도 쉽게 사용할 수 있습니다(구매 시 TF 카드 필요)
    * 화면은 정전식/저항식 화면으로 나뉘며, 해상도는 320*480입니다.
