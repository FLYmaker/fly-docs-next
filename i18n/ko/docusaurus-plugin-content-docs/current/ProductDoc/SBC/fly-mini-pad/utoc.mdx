# UTOC 탑재 사용

:::danger 주의사항

두 개의 UTOIC 사용은 권장하지 않습니다. 번쩍이기와 연결이 매우 복잡해집니다.

:::
    * UTOC 탑재를 사용하지 않으려면 아래 명령을 실행하십시오. 실행 시 어떠한 메시지도 표시되지 않습니다.
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * 원래 상태로 복원하는 방법, 실행 시 어떠한 메시지도 표시되지 않습니다.
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * MINI Pad에는 이미 UTOC가 탑재되어 있습니다.
