# WIFI 연결

## MINI PAD에 안테나 설치하기

    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 포트입니다
    * WIFI 안테나 포트는 `ipex1`입니다
    * MINI PAD에 안테나를 설치하지 않으면 신호가 약해 인터넷 연결이 어려울 수 있습니다
