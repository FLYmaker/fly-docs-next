# 소개

        우리는 새로운 FLY-Pi V2 상위 컴퓨터의 출시를 기쁘게 알립니다. 이 보드는 라즈베리 파이와 동일한 설치 크기를 가지며, 상위 컴퓨터로 라즈베리 파이를 대체하여 실행할 수 있습니다. 1GB의 내장 DDR3 메모리를 가지고 있어 완전한 LINUX 데스크톱(Armbian 기반으로 최적화)을 실행할 수 있습니다. 이 보드는 Klipper와 RRF 펌웨어와 함께 사용할 수 있으며, SHT 툴 보드를 사용할 때 Klipper 스크린을 실행할 수도 있습니다.
    이 상위 컴퓨터를 설계한 목적은 고객이 Klipper 펌웨어나 RRF 펌웨어를 더 쉽게 사용할 수 있도록 하기 위함입니다. 우리는 LINUX 시스템 이미지를 3D 프린터 사용에 맞게 최적화하여 설치 및 운영을 더 간단하게 만들었으며, 초보자도 쉽게 배울 수 있습니다. 다른 상위 컴퓨터 보드에 비해 학습 비용을 절감하여 3D 프린팅의 즐거움에 더 집중할 수 있게 해줍니다(구매 시 SD 카드가 포함된 세트로 Klipper 펌웨어가 사전 설치되어 있어 전원을 켜면 바로 사용 가능, 시간과 수고를 절약할 수 있습니다).

## 특징

    * 라즈베리 파이 3B 및 4B와 동일한 형태
    * CPU: 고성능 올윈 H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 채우기 속도 2.7gpixel/s 이상
    * RAM: 1GB DDR3(그래픽과 공유)(512+512)
    * ROM: 최대 128GB SDCARD 지원
    * 주변 장치: Spi x2, UART x2, USB x4(USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1(100M), CAN (내장 USB to CAN 모듈, UTOC 내장)
    * 인터페이스: 내장 M.2 인터페이스, WIFI, eMMC 등 확장 가능 (개인화된 프로토콜, FLY π 전용 M.2 장치를 제외한 다른 장치 연결 금지)
    * 40핀 인터페이스, 라즈베리 파이와 호환
    * 12-24V DC 전원 지원, 전원 공급이 더 안정적
    * 5V 팬 ZH1.5 인터페이스 예약
    * 시스템 사전 설치 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 공진 보상 플러그인 numpy 및 FLY-Tools, 소스 변경 등 복잡한 작업이 필요 없어 초보자도 쉽게 접근할 수 있습니다(구매 시 TF 카드 또는 M2WE 필요)
