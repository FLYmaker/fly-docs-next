# 화면 연결 및 설정

:::info

* FLY-TOOLS는 상위 기기와 네트워크 연결 후 상위 기기의 `IP+:9999`를 입력하여 접속해야 합니다.
* 예를 들어 `192.168.6.110:9999`

:::
