# 소개

        우리는 새로운 FLY-π 메인보드를 발표하게 되어 기쁩니다. 이 보드는 상위 컴퓨터로서 라즈베리 파이를 대체할 수 있으며, 1GB의 온보드 DDR3 메모리를 탑재하여 Armbian 기반으로 최적화된 전체 LINUX 데스크탑을 실행할 수 있습니다. 이 보드는 Klipper 및 RRF 펌웨어와 함께 사용할 수 있으며, SHT 도구 보드를 사용할 때 Klipper 화면을 실행할 수도 있습니다.

이 상위 컴퓨터를 설계한 목적은 고객이 Klipper 펌웨어나 RRF 펌웨어를 더 쉽게 사용할 수 있도록 하는 것입니다. 우리는 LINUX 시스템 이미지를 3D 프린터에 더 적합하도록 최적화했으며, 설치 및 운영도 더 간단해졌습니다. 초보자도 기본 지식만으로 빠르게 학습할 수 있습니다. 다른 상위 컴퓨터 보드에 비해 학습 비용을 절감하여 3D 프린팅의 즐거움을 더 집중할 수 있게 해줍니다(구매 시 SD 카드가 포함된 패키지를 선택하면 Klipper 펌웨어가 사전 설치되어 있어 전원만 연결하면 바로 사용 가능하여 시간과 노력을 절약할 수 있습니다).

## 특징

    * CPU: 고성능 Allwinner H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 채우기 속도 2.7gpixel/s 이상
    * RAM: 1GB DDR3(그래픽과 공유)(512+512)
    * ROM: 최대 128GB SDCARD 지원
    * 주변장치: Spi x2, UART x2, USB x4(USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1(100M), CAN x1(CAN 모듈 별도 구매 필요)
    * 인터페이스: 온보드 M.2 인터페이스, WIFI, eMMC 등 확장 가능(전용 프로토콜, FLY π 전용이 아닌 M.2 장치를 연결하지 마십시오)
    * 40핀 인터페이스, 라즈베리 파이와 호환
    * 12-24V DC 전원 지원, 전원 공급이 더 안정적
    * 상위 컴퓨터 냉각을 위한 5V 제어 가능한 팬 인터페이스 예약
    * CAN 확장 인터페이스 예약
    * 시스템에는 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 공진 보상 플러그인 numpy 및 FLY-Tools가 사전 설치되어 있어 복잡한 소스 교체 작업 없이 초보자도 쉽게 사용할 수 있습니다(TF 카드 또는 M2WE 구매 필요)
