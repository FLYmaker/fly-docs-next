# 상위기기 SSH 연결

* 시리얼 포트 또는 WIFI를 사용하여 장치에 연결할 수 있습니다
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요
* SD 카드 또는 M2WE에 **해당 상위기기의 시스템**이 번인되었는지 확인하세요
* **상위기기가 정상적으로 부팅되지 않는 경우 드라이버, 리미트 스위치, 팬 등 주변 장치를 분리하세요!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 편리하게 컴파일하세요!!!**
* **펌웨어를 컴파일하려면 WIFI 또는 이더넷 케이블을 통해 상위기기에 연결하는 방법을 참조하세요**

        ## 네트워크 IP를 사용하여 상위기기 SSH에 연결

        * 상위기기가 연결된 **라우터 관리 페이지**에 로그인하세요
        * 시스템이 **정상적으로 부팅**되었는지 확인하세요
        * **상위기기의 사용자명과 비밀번호**를 알고 있어야 합니다. FLY 시스템의 기본 사용자명은 **fly**, 기본 비밀번호는 **mellow**입니다
        * 메인보드가 부팅되고 네트워크에 연결되어 있으면 IP를 사용하여 SSH에 연결할 수 있습니다
        * MobaXterm을 열고 다음과 같이 설정합니다. IP는 앞서 확인한 IP를 입력하거나 라우터 관리 페이지에서 확인할 수 있습니다

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **참고: 여기서 4단계에 입력하는 상위기기의 사용자명입니다**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * 첫 연결 시 다음 창이 팝업됩니다. `Accept`를 클릭하세요

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * 비밀번호를 요청하는 메시지가 나타나면 상위기기의 비밀번호를 입력하고 엔터를 누르세요

        **참고: 여기서 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르세요**

        FLY 시스템의 기본 사용자명은: `fly` , 비밀번호는: `mellow`입니다
