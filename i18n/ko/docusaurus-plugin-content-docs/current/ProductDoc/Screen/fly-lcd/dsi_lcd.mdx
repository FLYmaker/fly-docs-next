# DSI-LCD5

    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-lcd/img/dsi_lcd5.webp').default} size="50%" align="left" />

## 제품 소개

    * 화면 크기: 120.7mm*76.0mm
    * 화면 표시 영역: 108mm*66.2mm
    * 해상도: 800*480
    * 화면 유형: IPS
    * 통신 방식: DSI/Mipi
    * 온보드 인터페이스: 15 핀 DSI / 22핀 Mipi
    * 안정적인 작동: 0°-60°C
    * 터치 유형: Capacitive（정전식）
    * 논리 전압: 3.3V
    * 무게: 135.5g

        :::danger 주의사항

        * 라즈베리파이5의 DSI와 CSI 인터페이스는 이제 MIPI로 이름이 변경되었습니다.
        * MIPI 인터페이스는 CSI 카메라 또는 DSI 화면을 바로 사용할 수 있도록 설정할 수 있습니다.
        * 금손가락 방향에 주의하세요.

        :::

        * 라즈베리파이5에는 두 개의 MIPI 인터페이스가 있습니다. 정면에서 보면 1과 0으로 표시되어 있습니다.

        * DSI 화면 연결


            * 1번 인터페이스를 사용하려면 아래와 같이 화면을 설정해야 합니다.

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * 0번 인터페이스를 사용하려면 아래와 같이 화면을 설정해야 합니다.

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
