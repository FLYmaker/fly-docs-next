# 소개

    * 분리형 디자인, 다양한 코어 보드 DSI와 HDMI 지원
    * 다양한 크기, 4.3인치, 4인치, 7인치
    * 고해상도, 모든 디테일을 놓치지 않습니다
    * 10단계 밝기 조절 지원
    * 180° 화면 회전 지원

:::warning 주의사항
7인치 화면은 HDMI 코어 보드를 선택할 수 없으며, DSI 코어 보드는 정상적으로 제어합니다
:::
