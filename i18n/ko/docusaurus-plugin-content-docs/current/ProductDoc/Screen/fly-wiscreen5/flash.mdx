# 부팅 이미지 수정
:::tip
    `해상도` 및 `파일명`에 따라 정확히 이미지를 제작하십시오, 그렇지 않으면 시스템이 인식하지 못할 수 있습니다.

    제작된 이미지를 SD카드/USB의 루트 디렉토리에 넣고, 화면이 켜진 상태에서 삽입한 후 업그레이드 버튼을 클릭하십시오.
:::
## 부팅 로고 수정

    * 이미지 요구사항 `800*480`
    * 파일명 `boot_logo.JPG`
    * 용량이 `105KB`를 초과해서는 안 됩니다.




## 첫 화면 이미지 수정

    * 이미지 요구사항 `500*400`
    * 파일명 `print.png`
    * 용량이 `45KB`를 초과해서는 안 됩니다.



## 인쇄 화면 이미지 수정

    * 이미지 요구사항 `311*480`
    * 파일명 `print_smail.png`
    * 용량이 `45KB`를 초과해서는 안 됩니다.
