# 소개

      Fly_WiScreen은 광저우 마그넷론 전자 기술 유한 회사 Fly3D 메인보드 개발 팀이 출시한 고급 Klipper 제어 스크린으로, WiFi를 통해 3D 프린터를 제어할 수 있으며, SD 카드와 USB 드라이브를 통해 인쇄 파일을 전송할 수 있습니다. 스크린의 응답 속도가 매우 빨라 사용자에게 매우 좋은 경험을 제공합니다. 주종 모드 설정 지원, 다중 기기 동기화 작업이 사용자에게 기계 관리를 편리하게 합니다. 5V 전원 포트에는 충전 모듈을 연결하여 배터리 개조를 통해 이동성을 높일 수 있습니다.

## 제품 기능

    * 다른 Klipper 설치가 완료된 상위 기기와 WIFI 연결 지원
    * WIFI 검색 지원, WIFI 연결 후 상위 기기를 자동으로 검색 가능
    * 여러 대의 프린터를 제어하는 클러스터 컨트롤 지원
    * USB와 TF 카드를 통해 인쇄 파일 업로드 지원
    * 사용자 정의 매크로, 작업 디버깅이 더 편리

## 제품 사양

    * CPU: 전지 T113-S3, 듀얼 코어 CORTEX- A7 최대 1.2GHZ
    * 메모리: 128M DDR3 통합
    * 저장: 128MB Nand Flash, TF 카드 슬롯
    * 디스플레이: 800X480 캐패시티브 스크린
    * 외부 인터페이스: USB 2.0 포트 하나, TF 카드 슬롯 하나
    * WIFI 프로토콜: 802.11b/g/n
    * 스크린 타입: IPS
    * 스크린 크기: 5인치
    * 멀티터치: 지원
    * 정격 전압: 5V
