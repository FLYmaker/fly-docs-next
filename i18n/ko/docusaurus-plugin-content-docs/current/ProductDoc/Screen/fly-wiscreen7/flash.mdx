# 부팅 이미지 수정
:::tip
    `해상도`와 `파일명`에 맞춰 이미지를 제작하세요, 그렇지 않으면 시스템에서 인식하지 못할 수 있습니다.

    제작된 이미지를 SD카드/USB의 루트 디렉토리에 넣고, 화면이 켜진 상태에서 삽입 후 업그레이드 버튼을 클릭하세요
:::
## 부팅 로고 수정

    * 이미지 요구사항 `1024*600`
    * 파일명 `boot_logo.JPG`
    * 크기 `105KB`를 초과하지 말 것



## 첫 화면 이미지 수정

    * 이미지 요구사항 `736*563`
    * 파일명 `print.png`
    * 크기 `45KB`를 초과하지 말 것



## 인쇄 화면 이미지 수정

    * 이미지 요구사항 `355*565`
    * 파일명 `print_smail.png`
    * 크기 `45KB`를 초과하지 말 것
