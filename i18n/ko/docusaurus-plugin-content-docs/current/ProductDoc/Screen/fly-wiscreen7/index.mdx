# 소개

      Fly_WiScreen은 광저우 망렌 전자과학기술 유한회사 Fly3D 메인보드 개발팀이 출시한 고급 Klipper 제어 스크린으로, Wi-Fi와 이더넷 두 가지 모드로 3D 프린터를 제어할 수 있으며, SD 카드와 USB 드라이브를 통해 인쇄 파일을 전송할 수 있습니다. 스크린 응답 속도가 매우 빨라 사용자에게 매우 좋은 경험을 제공합니다. 주종 모드 설정이 가능하여 다중 장비 동기 작업을 통해 사용자가 기기를 쉽게 관리할 수 있습니다. 5V 전원 포트에 충전 모듈을 연결하여 배터리로 개조할 수 있어 이동이 편리합니다.

## 제품 기능

    * WIFI와 이더넷 연결을 통해 Klipper를 설치한 호스트 컴퓨터와 연결 지원
    * WIFI 검색 지원, WIFI에 연결 후 호스트 컴퓨터를 자동으로 검색할 수 있음
    * 다중 프린터 클러스터 제어 지원
    * USB 및 TF 카드를 통한 인쇄 업로드 지원
    * 사용자 정의 매크로를 통해 조작 및 디버깅이 더 편리함

## 제품 사양

    * CPU: 전지 T113-S3, 듀얼 코어 CORTEX-A7, 최대 1.2GHz
    * 메모리: 128M DDR3 통합
    * 저장소: 128MB Nand Flash, TF 카드 슬롯
    * 디스플레이: 1024X800 캐패시티브 스크린
    * 외부 인터페이스: USB 2.0 포트 1개, TF 카드 슬롯 1개
    * WIFI 프로토콜: 802.11b/g/n
    * 스크린 유형: IPS
    * 스크린 크기: 7인치
    * 멀티 터치: 지원
    * 정격 전압: 5V
