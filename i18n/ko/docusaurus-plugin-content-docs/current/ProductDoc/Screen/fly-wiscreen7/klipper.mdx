## Klipper 설정

### LED 설정

    ```cfg
    [led LED]
    white_pin:PA15
    ```

### ptc 설정

    ```cfg
    [heater_generic warehouse]
    heater_pin:E_CS
    sensor_type:ATC Semitec 104GT-2
    sensor_pin:PA1
    min_temp:-270
    max_temp:600
    control = pid
    pid_kp = 16.804
    pid_ki = 0.794
    pid_kd = 88.854
    ```

### FAN 설정

    ```cfg
    [fan_generic fan1]
    pin: FAN1
    ```


    ```cfg
    [fan_generic fan2]
    pin: FAN2
    ```

 

### 사용자 정의 Gcode

    * 화면을 아래로 내리면 사용자 정의 Gcode가 나타납니다.
    * 하나의 버튼으로 여러 Gcode를 보내는 형식:
    * 예시:

    ```cfg
    G91 
    G1 X10 F6000
    G90
    ```

    * 화면에 입력할 형식:(두 번째 줄부터 각 명령 앞에 \n 추가)

    ```cfg
    G91\nG1 X10 F6000\nG90
    ```



## 객체 제외 사용

    * moonraker를 교체해야 합니다.

    * 먼저 SSH에 연결하고, 인터넷에 정상적으로 연결할 수 있는지 확인하세요.

    1. moonraker 파일 백업

    ```bash
    mv ~/moonraker ~/moonraker-bak
    ```

    2. moonraker 서비스 중지

    ```bash
    sudo systemctl restart moonraker
    ```

    3. moonraker 가져오기

    ```bash
    git clone https://e.coding.net/g-ofpa1390/3D-Printers/moonraker-dev.git -b flylcd-dev moonraker
    ```

    4. 브라우저에서 상위 기기를 찾아 `moonraker.conf`를 추가하고, 시스템을 재부팅합니다.

    ```bash
    [network]
    ```

    5. 그런 다음 브라우저에 아래 주소를 입력하세요. 여기서 IP는 상위 기기의 IP로 대체해야 합니다.

    ```bash
    http://ip:7125/server/network/wifi/scan
    ```

* 이 단계가 완료되었음을 나타냅니다.
