# 프로젝트 구조 소개


    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide6.png').default} size="80%" align="left" />



## 소개
#### Logic 폴더
* 우리는 99%의 UI 코드 프로그래밍을 logic 폴더에서 수행하며, 다른 폴더는 신경 쓸 필요가 없다
* gcodeLogic.cc는 main.c와 유사하며, 99%의 UI 코드가 여기 존재하며, 다른 파일은 주의할 필요가 없다

#### i18n 폴더
* 각국의 언어 번역 테이블이 저장된 곳

#### resources 폴더
* UI에서 사용하는 이미지 자원이 저장된 곳

#### ui 폴더
* UI 화면 파일이 저장된 곳, 더블 클릭하여 열기
* 화면 하단 왼쪽 화살표가 가리키는 "작은 눈"을 더블 클릭하면 숨겨진 화면을 열 수 있음
