##   업데이트
    * 1. 다운로드한 파일을 `update.img`로 이름 변경
    * 2. `update.img` 파일을 SD카드/USB에 넣고 화면에 삽입, 화면이 켜진 상태에서도 삽입 가능.
    * 3. 화면을 선택하고 업그레이드를 클릭하면 업그레이드가 완료됩니다
