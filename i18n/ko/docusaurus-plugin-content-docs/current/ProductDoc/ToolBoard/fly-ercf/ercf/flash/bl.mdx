# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * ERCF는 CAN, USB를 지원합니다
    * 여기서는 CAN 펌웨어를 500K와 1M 속도로 제공합니다
    :::

## 번인 모드 진입

    :::tip

    원으로 표시된 부분이 `BOOT 버튼`입니다

    :::

    * 번인 모드로 진입하려면 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 데이터 케이블 한쪽을 메인보드에, 다른 한쪽을 호스트 컴퓨터에 연결하십시오
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.png').default} size="100%" align="center" />

            <Katapult />

            4. 펌웨어 번인

            :::tip

            한 개의 펌웨어만 번인하면 됩니다

            :::

            * 아래는 500K CAN 속도의 펌웨어 번인 명령어입니다
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 번인 명령어입니다
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * 아래는 USB의 펌웨어 번인 명령어입니다
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. 아래와 같은 메시지가 나타나면 번인이 성공한 것입니다

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            번인이 완료되면 메인보드를 완전히 전원을 껐다가 다시 켜야 합니다.

            :::
