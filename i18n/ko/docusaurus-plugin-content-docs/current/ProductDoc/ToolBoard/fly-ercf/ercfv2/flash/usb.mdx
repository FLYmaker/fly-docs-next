# USB 펌웨어 컴파일

    :::tip

    * ERCF는 출하 시 CAN 연결의 Katapult 펌웨어가 사전에 설치되어 있습니다.
    * USB 펌웨어 번인을 사용하려면 USB의 Katapult 펌웨어를 번인해야 합니다.
    * 번인 후 다시 번인할 필요가 없습니다.
    * <Button variant="contained" disableElevation href="./bl">Katapult 펌웨어 번인</Button>

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * 마이크로컨트롤러 시작 시 설정할 GPIO 핀 선택, 입력: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` 키를 눌러 **Save configuration**이 나타나면, `Y` 키를 눌러 저장
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령 줄 인터페이스로 돌아갔습니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 조금 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래의 내용이 출력되면 컴파일이 성공했습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
