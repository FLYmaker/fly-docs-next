# 제품 소개

    FLY-ERCF는 RP2040 기반의 3D 프린터 다색 제어 보드로, 주로 Enraged Rabbit 프로젝트에 사용됩니다. 또한 다른 두 축 스테퍼 모터 제어 상황에서도 사용할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.webp').default} size="100%" align="left" />

## 제품 특징

    * MCU: Raspberry Pi rp2040, 듀얼 코어 ARM Cortex-M0+@133MHz
    * CAN 모드, RS232 모드 지원
    * 두 개의 드라이버 위치, UART 및 SPI 드라이버 지원
    * 원본 ERCF EASY Board와 호환
    * 크기가 매우 작아 4층 보드 설계
    * 14PIN 확장 포트를 XHD2.54로 교체하여, 접속이 더 견고해져서 단선 문제를 방지
    * 12-24V 전압 지원
    * CAN 버스와 RS232 연결 지원, 데이터 전송이 더 안정적이고 지연이 적어 연결이 더 안정적
    * can 인터페이스는 XT30(2+2)를 사용하여, 15A 전류를 지원s

## 제품 사양

    * MCU: Raspberry Pi rp2040, 듀얼 코어 ARM Cortex-M0+@133MHz
    * 펌웨어: Klipper CAN/RS232
    * 입력 전압: DC12V-24V
    * 인터페이스: XHD2.54 이중 7PIN 확장 포트, SERVO X1, ENDSTOP X2, EXTRA X1
    * 드라이버: UART/SPI 및 STEP/DIR 지원
    * 12-24V 전압 지원
    * CAN 버스와 RS232 지원, 데이터 전송이 더 안정적이고 지연이 적어 연결이 더 안정적
    * can 인터페이스는 XT30(2+2)를 사용하여, 15A 전류를 지원s
