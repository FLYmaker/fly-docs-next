    <CAN />

    * GPIO 핀을 선택하여 마이크로컨트롤러 시작 시 설정, 입력: !PA15 주의: !는 대문자입니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can.png').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 이때 `Y` 키를 누릅니다
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령줄 인터페이스로 돌아갑니다
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 조금 걸립니다
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * 마지막으로 다음 내용이 출력되면 컴파일이 성공적으로 완료됩니다

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 펌웨어 번인

    * 펌웨어를 번인하기 전에 XT30 데이터 케이블을 사용하여 도구 보드를 UTOC 또는 브리지 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참고용 배선도입니다
    * DIP 스위치가 올바른 위치에 있는지 확인하십시오
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can_mode.png').default} size="90%" align="center" />
    * UTOC 연결 참고
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * 전원이 꺼진 상태에서 멀티미터로 CAN H와 CAN L의 저항값을 측정하여 `60Ω` 정도인지 확인합니다. `120Ω`라면 `120Ω` 점퍼를 연결해야 하고, `40Ω` 정도라면 점퍼를 제거해야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.png').default} size="50%" align="center" />
