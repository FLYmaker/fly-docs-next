# 펌웨어 플래시 설명

    * MMU는 RS232, CAN, USB 세 가지 연결 방식을 통해 상위 기기와 연결할 수 있습니다.
    * 그 중 RS232와 CAN은 `XT30(2+2)-F`를 사용하여 연결해야 하며, USB는 데이터 전송 기능이 있는 `TYPE-C` 데이터 케이블을 사용해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블 스타일입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드에 이미 Katapult가 플래시되어 있다면, 마지막에 있는 Katapult 펌웨어 플래시는 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 플래시되지 않은 경우를 대비한 것입니다.
* 리셋 버튼을 두 번 빠르게 누른 후 메인보드에 LED가 깜빡이면, 메인보드가 Katapult 펌웨어에 진입한 것을 의미합니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    * MMU는 다음과 같은 방식으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 플래시</Button>
        * 장점: `Katapult` 펌웨어를 플래시하면 메인보드를 건드리지 않고도 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하는지 확인하고, UTOC 또는 브리지 펌웨어가 플래시된 메인보드와 함께 사용해야 합니다.
    * <Button variant="contained" disableElevation href="./rs232">RS232 펌웨어 플래시</Button>
        * 장점: 특별한 소프트웨어가 필요 없으며, 통신이 신뢰할 수 있고 안정적입니다.
        * 단점: 펌웨어를 다시 플래시하려면 수동으로 DFU에 진입해야 하거나, RST를 두 번 눌러 `Katapult`에 진입하여 펌웨어를 업데이트해야 합니다.
        * 팁: G2T 또는 UTOR와 함께 사용해야 합니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어 플래시</Button>
        * 장점: Katapult 펌웨어가 플래시되면 메인보드를 건드리지 않고도 펌웨어를 업데이트할 수 있으며, 통신이 신뢰할 수 있고 안정적입니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
