# USB 펌웨어 컴파일

    :::tip

    * MMU는 출고 시 CAN 연결의 Katapult 펌웨어가 미리 설치되어 있습니다.
    * USB 펌웨어를 플래시하려면 USB의 Katapult 펌웨어를 플래시해야 합니다.
    * 한 번 플래시한 후에는 다시 플래시할 필요가 없습니다.
    * <Button variant="contained" disableElevation href="./bl">Katapult 펌웨어 플래시</Button>

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * 마이크로 컨트롤러 시작 시 설정할 GPIO 핀을 선택합니다. 입력: !PA15 (주의: !는 대문자입니다)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.png').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타납니다. 이때 `Y` 키를 눌러 저장합니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * 이제 구성 파일이 저장되었고 명령줄 인터페이스로 돌아갑니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 조금 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * 마지막에 다음과 같은 내용이 출력되면 컴파일이 성공적으로 완료된 것입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
