# 제품 소개

    Fly-MMU는 다기능 도구 보드로, 고성능 컨트롤 보드로서 3D 프린터 및 자동화 장비를 위해 설계되었습니다. 강력한 H723 마이크로컨트롤러를 통합하여 최대 4개의 TMC 스테퍼 모터 드라이버를 지원하며, 고압 구동을 지원하고, 11개의 리미트 스위치 입력이 내장되어 있으며 FPC를 통해 16개의 리미트 스위치를 확장할 수 있습니다. 또한, 4개의 서보 컨트롤 출력을 제공합니다. 이 도구 보드는 Klipper 펌웨어와 호환되며, CAN 버스와 RS232 연결 옵션을 제공하여 유연한 통신 및 컨트롤 방안을 실현합니다.

## 제품 특징

    * 주컨트롤러는 32비트 주파수 550MHz의 ARM Cortex-M7 시리즈 STM32H723 주컨트롤러 칩을 사용
    * 네 개의 드라이버 확장 지원, 최대 48V의 고압 TP5160 드라이버 지원
    * 여러 리미트 확장, 보드에 11개의 리미트 포트가 내장되어 있으며, FPC를 통해 16개의 리미트 포트를 확장할 수 있음
    * Klipper 펌웨어를 사용하여 USB, CAN 및 RS232 연결 지원
    * CAN 확장 포트 예약, CAN 사용 시 다른 장비를 CAN 확장 포트를 통해 연결 가능
    * CAN 버스와 RS232 연결 지원, 데이터 전송이 더 안정적이고 지연 시간이 적으며 연결이 더 안정적
    * CAN 인터페이스는 XT30(2+2)를 사용하며, 15A 전류를 지원

## 제품 사양

    * MCU: STM32H723
    * 펌웨어: Klipper CAN/RS232
    * 전원 입력: HVIN:12V/48V 및 VIN:12V/24V
    * 확장 인터페이스: 11개의 리미트, 4개의 서보, 1개의 RGB, 1개의 CAN 확장 포트 및 FPC 확장 포트를 통해 16개의 리미트 확장 가능
    * 드라이버: TP2209 및 TP5160 지원, 또한 다른 FLY의 TP 드라이버 사용 가능
    * 12-24V 전압 지원
    * CAN 버스와 RS232 지원, 데이터 전송이 더 안정적이고 지연 시간이 적으며 연결이 더 안정적
    * CAN 인터페이스는 XT30(2+2)를 사용하며, 15A 전류를 지원
