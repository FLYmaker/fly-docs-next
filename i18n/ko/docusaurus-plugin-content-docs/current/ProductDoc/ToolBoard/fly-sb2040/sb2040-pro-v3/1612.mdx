# Eddy 사용

    :::info

    * 스캔 베드를 사용할 때는 별도의 Z 제한을 사용하는 것이 좋으며, 스캔 베드를 Z 제한으로 사용하지 않는 것이 좋습니다.
    * Eddy를 설치할 때, 노즐 아래쪽에 최소 1-2 mm의 공간을 두어야 합니다.
    * Eddy 주파수를 보정할 때, 코일 PCB는 베드에서 최소 5mm 이상 떨어져 있어야 합니다.
    * 베드의 전체 온도가 상온에서 테스트를 진행하시고, 테스트 중에는 베드를 가동하지 마십시오.

    :::

## 참고 설정 및 사용 설명

    ### 스캔 베드 설정

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 현재 외부 1612 모듈이 출시되지 않았기 때문에, 이 설정은 사용할 수 없습니다.
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 온도 보상 설정

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh 설정

    * [bed_mesh]의 `horizontal_move_z` 매개변수는 2로 설정하는 것이 좋습니다. 이를 통해 스캔 플랫폼 시 Eddy가 플랫폼에 최대한 가까워집니다.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
