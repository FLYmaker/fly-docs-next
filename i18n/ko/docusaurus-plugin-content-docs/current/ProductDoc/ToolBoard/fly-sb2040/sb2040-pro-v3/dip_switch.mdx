# 온보드 디프 스위치 소개

:::warning 주의사항

이미지 해상도가 너무 높아 웹 페이지에서 볼 수 없으므로 **마우스 커서를 배선도 위에** 두고 **오른쪽 클릭**하여 **이미지로 저장**을 선택한 후, 이미지를 데스크탑에 저장하여 확인하십시오!!!

:::
