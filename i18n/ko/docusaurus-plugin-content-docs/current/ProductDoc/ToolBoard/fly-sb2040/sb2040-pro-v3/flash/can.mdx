    * 마이크로 컨트롤러 시작 시 설정할 GPIO 핀 선택, 입력: !gpio5 주의: !는 대문자입니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타납니다, 이때 `Y` 키를 눌러주세요
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령줄 인터페이스로 돌아갔을 것입니다
    * `make -j4`를 입력하여 컴파일을 시작합니다, 시간이 좀 걸립니다
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래와 같은 내용이 출력되면 컴파일이 성공한 것입니다

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 펌웨어 플래싱

    * 펌웨어를 플래싱하기 전에 XT30 데이터 케이블을 사용하여 도구판을 UTOC 또는 브리지 펌웨어가 플래싱된 메인보드에 연결하세요, 아래는 참고 연결도입니다
    * 주의: 점퍼 설정이 올바른 위치에 있는지 확인하세요
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * 전원이 꺼진 상태에서 멀티미터로 CAN H와 CAN L의 저항값이 `60Ω` 정도인지 확인하세요, 만약 `120Ω`라면 `120Ω` 점퍼를 켜야 하고, `40Ω` 정도라면 점퍼를 꺼야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.webp').default} size="100%" align="center" />
