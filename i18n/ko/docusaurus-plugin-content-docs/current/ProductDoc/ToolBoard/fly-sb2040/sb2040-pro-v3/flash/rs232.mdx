

## 플래싱 모드 진입

    :::tip

    왼쪽이 `RESET 버튼`, 오른쪽이 `BOOT 버튼`입니다

    :::

    * DFU 모드로 진입하는 두 가지 방법
    * 첫 번째: 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 데이터 케이블을 보드와 호스트에 연결합니다
    * 두 번째: 데이터 전송이 가능한 데이터 케이블을 보드와 호스트에 연결한 상태에서 BOOT 버튼을 누른 채로 RST 버튼을 누르고, RST 버튼을 놓은 후 BOOT 버튼을 놓습니다
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.png').default} size="100%" align="center" />
