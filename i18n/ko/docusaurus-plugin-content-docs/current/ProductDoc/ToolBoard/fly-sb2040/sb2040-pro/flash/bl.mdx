# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * SB2040_PRO는 CAN을 지원합니다
    * 여기서는 CAN 펌웨어를 500K와 1M 속도로 제공합니다
    :::

## 플래시 모드 진입

    :::tip

    동그라미로 표시된 것은 `BOOT 버튼`입니다

    :::

    * 메인보드의 전원을 완전히 차단한 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/boot.png').default} size="50%" align="center" />

            <Katapult />

            4. 펌웨어 플래시

            :::tip

            한 가지 펌웨어만 플래시하면 됩니다

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래시 명령어입니다
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래시 명령어입니다
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. 아래의 메시지가 나타나면 플래시가 성공적으로 완료된 것입니다

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            플래시가 완료되면, 메인보드의 전원을 완전히 한 번 껐다가 켜야 합니다.

            :::

        </TabItem>
    </Tabs>
