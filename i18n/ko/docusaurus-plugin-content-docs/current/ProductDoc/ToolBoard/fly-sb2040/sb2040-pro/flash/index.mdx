# 펌웨어 번인 설명

    * SB2040 PRO는 CAN 연결 방식으로 상위 컴퓨터에 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 선 스타일입니다.
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 참고사항
* 메인보드 펌웨어 번인은 Katapult를 통해 Klipper를 번인해야 합니다.
* 메인보드에 Katapult가 이미 번인되어 있는 것이 확실하다면 마지막에 있는 Katapult 펌웨어 번인은 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 번인되지 않았을 경우를 대비한 것입니다.
* 빠르게 두 번 리셋 버튼을 누른 후 메인보드에 LED가 깜빡이면 메인보드가 Katapult 펌웨어로 들어갔음을 나타냅니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하나요?

    * SB2040 PRO는 다음과 같은 방식으로 상위 컴퓨터에 연결할 수 있습니다.
    * [CAN 펌웨어](./can.mdx "클릭하면 이동")
        * 장점: `Katapult` 펌웨어를 설치하면 메인보드를 만지지 않고도 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 컴퓨터가 CAN을 지원하는지 확인하고, UTOC 또는 브리징 펌웨어가 설치된 메인보드와 함께 사용해야 합니다.


## Katapult 펌웨어 설명

    * SB2040 PRO는 기본적으로 CAN 연결 속도 1M의 Katapult 펌웨어를 포함하고 있습니다.
