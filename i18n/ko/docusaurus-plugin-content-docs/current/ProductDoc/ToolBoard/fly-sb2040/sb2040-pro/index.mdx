# 제품 소개

    FLY SB2040 PRO는 광저우 매로우 전자 기술 유한 회사에서 VORON Stealthburber 프린트 헤드를 위해 출시한 도구 보드로, 이 보드를 사용하면 기존 메인 보드와 프린트 헤드 사이의 복잡한 배선을 네 줄로 대체하여 레이아웃을 최적화할 수 있습니다. SB2040은 stealthburner와 afterburner에 적합하며, CAN 수신기, USB 포트, TMC2240 드라이버, ADXL345 가속도 센서, 세 개의 제어 가능한 팬 등의 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper와 RRF 펌웨어에 적합합니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/img/sb2040.png').default} size="100%" align="center" />


## 제품 특징

    * Boot 버튼을 예약하여, 사용자가 TypeC를 통해 Boot 모드에 들어가 펌웨어를 업데이트할 수 있음
    * Klipper 펌웨어 지원
    * 온도 센서 지원: 내장 MAX31865 칩, PT100 호환
    * 내장 ADXL345 가속도 센서
    * RGB 인터페이스 예약, 내장 2240 드라이버, 다양한 레벨링 센서 지원, 예: klicky, Voron Tap, PL08n 등
    * 12-24V 전압, 15A 전류 입력 지원, 히터 바 포트를 확대, 정격 전류 10A


## 제품 사양


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류 지원, 피크 30A
    * 펌웨어: Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 내장 센서: ADXL345 및 선택적 MAX31865
    * 히터 바 인터페이스: 히터 바(HEAT0)
    * 확장 인터페이스: 제어 가능한 팬 *3, RGB *1, 리미트 *3, 히터 파이프 *1, 온도 센서 *1
    * 팬 인터페이스: 세 개의 제어 가능한 팬
    * 모터 드라이버: 내장 TMC2240
