# 드라이브 언더볼트 해결 방법

## 방법 1

    :::danger

    * 전원 교체
    * 전원 필터 추가
    * 모터 교체 판단
    * 모터 외부를 도구 보드의 GND에 연결
    * 압출기 드라이브 설정에 `driver_TPFD: 0` 추가

    :::

## 방법 2
    * SSH를 통해 상위 기기에 연결
    * 아래 명령 입력
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Klipper 업데이트 후 이 단계가 복원될 수 있음
    :::
