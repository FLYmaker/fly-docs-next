# 온보드 딥스위치 소개

:::warning 주의사항

이미지 해상도가 너무 높아 웹 페이지에서 보기 어렵습니다. **마우스 커서를 연결도면 위에 놓고** **오른쪽 클릭**하여 **이미지 다른 이름으로 저장**을 선택하고, 이미지를 데스크탑에 저장한 후 확인하세요!!!

:::
