## 펌웨어 플래싱

    * 펌웨어를 플래싱하기 전에 XT30 데이터 케이블을 사용하여 도구 보드를 UTOC 또는 브리지 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참고로 연결도입니다.
    * DIP 스위치가 올바른 위치에 있는지 확인하세요.
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * 전원이 꺼진 상태에서 멀티미터로 CAN H와 CAN L의 저항값이 `60Ω` 정도인지 확인하십시오. `120Ω`이면 `120Ω` DIP 스위치를 켜야 하고, `40Ω` 정도이면 DIP 스위치를 꺼야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
