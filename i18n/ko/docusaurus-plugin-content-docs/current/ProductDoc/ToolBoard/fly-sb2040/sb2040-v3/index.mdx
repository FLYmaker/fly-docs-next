# 1. 제품 소개

    FLY SB2040 V3는 광저우 마겐 전자 기술 유한 회사가 VORON Stealthburber 프린트 헤드를 위해 출시한 도구 보드입니다. 이 도구 보드를 사용하면 기존의 복잡한 메인보드와 프린트 헤드 간의 연결을 네 줄의 선으로 대체하여 배치를 최적화할 수 있습니다. SB2040은 stealthburner와 afterburner에 적합하며, CAN 수신기, RS232 수신기, USB 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 세 개의 제어 가능한 팬 등의 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper와 RRF 펌웨어에 적합합니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/sb2040.png').default} size="100%" align="center" />


## 1.1 제품 특징

    * 부팅 버튼이 예약되어 있어, 사용자가 TypeC를 통해 부팅 모드로 들어가 펌웨어를 업데이트할 수 있습니다.
    * Klipper 펌웨어와 RRF 펌웨어를 지원합니다.
    * CAN 모드, RS232 모드, RRF CAN 모드를 지원합니다.
    * RS232 모드에서는 더 높은 전송 속도와 더 강한 안정성을 제공합니다. CAN 모드에서는 CAN 확장이 가능하며, CAN 확장 인터페이스가 예약되어 있습니다.
    * 온도 센서 지원: 내장된 MAX31865 칩으로 PT100 및 PT1000 온도 센서를 지원합니다.
    * 내장된 LIS2DW 가속도 센서와 LDC1612 센서
    * RGB 인터페이스 예약, 내장된 2209 드라이버로 klicky, Voron Tap, PL08n 등 다양한 평준화 센서를 지원합니다.
    * 12-24V 전압, 15A 전류 입력 지원, 히팅 로드 포트가 증가되어 정격 전류 10A

## 1.2 추가 기능

    * Klipper CAN 모드, RS232 모드, RRFCAN 모드 간의 전환을 지원합니다.
    * RS232 모드 추가, 더 높은 전송 속도와 더 강한 안정성, 디버깅 사용이 더 쉬워졌습니다.
    * RRF CAN 모드 추가, RRF 도구 보드로 전환 가능하여 적용 범위를 넓혔습니다.
    * CAN 확장 인터페이스 추가, CAN 장비 확장 용이
    * 종단 저항 스위치 추가, 종단 저항 120R을 점퍼 없이 스위치로 선택 가능
    * 1미터 전원/신호 연결선 추가, 연결선은 트위스트 설계로 방해 저항 성능이 더 강해졌습니다.

## 1.3 제품 사양


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류, 최대 30A를 지원합니다.
    * 펌웨어: Reprap/Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 내장 센서: LIS2DW와 선택 가능한 ldc1612 및 max31865
    * 히팅 로드 인터페이스: 히팅 로드（HEAT0）
    * 확장 인터페이스: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN 포트
    * 팬 인터페이스: 1개의 DC 팬, 2개의 DC 팬 또는 PWM 팬
    * 모터 드라이버: 내장된 TMC2209
    * 온도 센서: 100K NTC 또는 PT1000（TH0） 1개, PT100/PT1000 선택 가능 1개
