# 펌웨어 플래싱 설명

    * SB2040은 CAN 연결 방식을 사용하여 상위 컴퓨터와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블 스타일입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 플래싱은 Katapult를 통해 Klipper를 플래싱해야 합니다.
* 메인보드에 이미 Katapult가 설치되어 있는 것이 확실하다면 마지막에 있는 Katapult 펌웨어 플래싱은 필요하지 않습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 설치되지 않은 경우를 대비한 것입니다.
* 재설정 버튼을 두 번 빠르게 누르면 메인보드에 LED가 깜빡이기 시작하는지 확인하면, Katapult 펌웨어가 설치되었음을 나타냅니다.
:::

## 어떤 펌웨어가 필요한지 판단하는 방법

    * SB2040은 CAN 연결을 통해 상위 컴퓨터에만 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 플래싱</Button>
        * 장점: `Katapult` 펌웨어를 플래싱하면 메인보드에 손을 대지 않고 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 초과 시간 오류 등이 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 컴퓨터가 CAN을 지원하는지 확인해야 하며, UTOC 또는 브리지 펌웨어가 설치된 메인보드와 함께 사용해야 합니다.
