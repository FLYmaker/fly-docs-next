## 제품 소개

    FLY SB2040은 광저우 매그넷론 전자 기술 유한 공사가 VORON Stealthburber 프린트 헤드를 위해 출시한 노즐 도구 보드입니다. 이 도구 보드를 사용하면 프린트 헤드와 메인 보드 사이의 복잡한 연결을 네 줄의 선으로 대체하여 레이아웃을 최적화할 수 있습니다. SB2040은 36 스텝 모터에 적합하며, CAN 송수신기, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등의 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에 적합합니다.

    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연이 적으며 연결이 더 안정적입니다.
    * TMC2209 드라이버 탑재, UART 모드 사용, 압출기용으로 사용 가능
    * ADXL345 가속도 센서 탑재
    * BLTOUCH, PL08N 등의 레벨링 센서 지원, XY 리미트 스위치 통합
    * 12-24V 전압 입력 지원 
    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 4.7K 저항(ntc 서미스터용) 또는 1K 저항(PT1000용) 전환 지원 추가
    * can 인터페이스는 XT30(2+2)를 사용하여 15A 전류, 최대 30A 지원
    * 탑재된 NTC100k 저항 추가, 내부 온도 측정용
    * ADXL345의 INT1 및 INT2를 mcu에 연결하여 특정 사용자의 레벨링을 지원
    * 고압 입력 리미트 지원
    * 신호 손실 문제를 피하기 위해 3미터 CAN 연결선 제공
    * 2/3/4선 팬 지원
    * 분리형 디자인, Stealthburner 분리 용이
    * hartk의 Stealthburner Toolhead Board와 호환 (팬 보드를 교체할 필요 없음)
    * 인터페이스：제어 가능 팬 *3, RGB *1, 리미트 *3, 히팅 튜브 *1, 서미스터 *1

## 제품 사양

    * Klipper CAN 모드, RS232 모드, RRFCAN 모드 간 전환 지원
    * 연결 인터페이스는 XT30(2+2) 커넥터 사용, 15A 전류, 최대 30A 지원
    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 입력 전압：DC12-24V
    * 히팅 바 최대 출력 전류：5A
    * 팬 인터페이스 최대 출력 전류 1A
