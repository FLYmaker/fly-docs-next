# 펌웨어 번인 설명

    * SHT36/42는 CAN 연결 방식으로 상위 컴퓨터와 연결할 수 있습니다.
    * CAN은 MX3.0 금속 단자 압착선을 사용해야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.png').default} size="80%" align="left" />

## 어떤 종류의 펌웨어가 필요한지 어떻게 판단하나요?

    * SHT36/42는 CAM 방식으로 상위 컴퓨터와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 번인</Button>
        * 장점: `Katapult` 펌웨어를 번인하면 메인보드에 손을 대지 않고도 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 시간 초과 등의 오류가 발생할 수 있습니다.
        * 참고: CAN을 사용하려면 상위 컴퓨터가 CAN을 지원하는지 확인하고, UTOC 또는 브리지 펌웨어가 번인된 메인보드가 필요합니다.
