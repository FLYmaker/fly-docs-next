# 제품 소개

    FLY SHT36/42는 광저우 멜론 전자 기술 유한회사에서 3D 프린터 익스트루더 헤드를 위해 설계한 노즐 툴보드입니다. 이 툴보드를 사용하면 메인보드와 프린트 헤드 사이의 복잡한 배선을 네 줄로 대체하여 레이아웃을 최적화할 수 있습니다. SHT36은 36 스텝 모터에 적합하고, SHT42는 42 스텝 모터에 적합합니다. CAN 수신기, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등의 기능이 통합된 툴보드입니다. 이 툴보드는 Klipper 펌웨어에 적합합니다.

    * 크기 최적화, 6층 보드 설계 사용
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연이 적으며 연결이 더 안정적
    * TMC2209 드라이버 내장, UART 모드 사용, 익스트루더에 사용 가능
    * ADXL345 가속도 센서 내장
    * PT100 (MAX31865) 지원
    * 교체 가능한 팬 MOS 설계
    * BLTOUCH, PL08N 등 레벨링 센서 지원, XY 한계 스위치 통합
    * 12-24V 전압 입력 지원 
    * MCU: STM32F072
    * 통신: CANBUS, USB
    * 인터페이스: 제어 가능한 팬*2, RGB*1, 한계*3, Probe*1, Servo*1, 히팅 튜브*1, 서미스터*1, PT100*1

## 제품 사양

    1. 외형 치수: 51.27mm*36.26mm(SHT36), 42mm*42mm(SHT42)
    2. 설치 치수: 홀 간격 43.84mm, M3 설치 홀 직경
    3. 프로세서: STM32F072C8T6
    4. 입력 전압: DC12-24V 5A
    5. 히팅 바 최대 출력 전류: 5A
    6. 팬 인터페이스 최대 출력 전류 1A
