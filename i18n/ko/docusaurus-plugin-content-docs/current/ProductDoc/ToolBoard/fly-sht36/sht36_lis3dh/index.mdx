# 제품 소개

    FLY-SHT36-LIS3DH는 광저우 마그네슘렌 전자 기술 유한 회사가 3D 프린터 압출 헤드를 위해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면 주판과 프린트 헤드 사이의 복잡한 연결선을 네 줄로 대체할 수 있어 레이아웃을 최적화할 수 있습니다. FLY-SHT36-LIS3DH는 36 스테핑 모터에 적합하며 CAN 수신기, USB 포트, TMC2209 드라이버, LIS3DH 가속도 센서, 두 개의 제어 가능한 팬 등의 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에 적합합니다.

    * 체적을 극도로 압축, 6층 보드 설계
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적, 지연 시간이 더 짧고 연결이 더 안정적
    * 내장 TMC2209 드라이버, UART 모드 사용, 압출기용
    * 내장 ADXL345 가속도 센서
    * PT100(MAX31865) 지원
    * 교환 가능한 팬 MOS 디자인
    * BLTOUCH, PL08N 등의 레벨링 센서 지원, XY 리미트 스위치 통합
    * 12-24V 전압 입력 지원
    * MCU: STM32F072
    * 통신: CANBUS, USB
