# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * SHT36 PRO는 CAN을 지원합니다
    * 여기서는 CAN 펌웨어가 500K와 1M 속도로 제공됩니다
    :::

## 번인 모드로 진입

    :::tip

    원 안에 있는 것은 `BOOT 버튼`입니다

    :::

    * 메인보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 단락시키고 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다
    * 그런 다음 BOOT 단락을 해제합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="100%" align="center" />

            4. 펌웨어 번인

            :::tip

            단 하나의 펌웨어만 번인하면 됩니다

            :::

            * 아래는 500K CAN 속도의 펌웨어 번인 명령어입니다
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * 아래는 1M CAN 속도의 펌웨어 번인 명령어입니다
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```
