# 펌웨어 플래싱 설명

    * SHT36 PRO는 CAN 연결 방식으로 상위 기기와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블 스타일입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의
* 메인보드 펌웨어 플래싱은 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드에 이미 Katapult가 플래시되었다면 마지막에 있는 Katapult 펌웨어 플래싱을 할 필요가 없습니다. 이는 메인보드의 Katapult 펌웨어가 성공적으로 플래시되지 않은 경우에 대비한 것입니다.
* 빠르게 두 번 리셋을 누른 후 메인보드 상에 LED가 깜빡이면 메인보드가 Katapult 펌웨어로 들어갔음을 의미합니다.
:::

## CAN 펌웨어 컴파일

    * SHT36 PRO는 CAN 방식으로만 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 플래싱</Button>
        * 장점: `Katapult` 펌웨어를 플래시하면 메인보드를 건드리지 않고도 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용 시 리셋 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하는지 확인하고, UTOC 또는 브리지 펌웨어가 플래시된 메인보드와 함께 사용해야 합니다.
