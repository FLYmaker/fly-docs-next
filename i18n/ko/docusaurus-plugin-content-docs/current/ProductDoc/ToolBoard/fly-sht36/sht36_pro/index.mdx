# 제품 소개

    FLY-SHT36-PRO는 광저우 매그니파이 전자 과학 기술 유한회사가 3D 프린터 익스트루더 헤드를 위해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면, 기존의 메인보드와 프린트 헤드 사이의 복잡한 연결선을 네 개의 선으로 대체할 수 있어 레이아웃을 최적화할 수 있습니다. FLY-SHT36-PRO는 36 스텝 모터에 적합하며, CAN 수신기, USB 포트, TMC5160 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등 다양한 기능이 통합된 도구 보드입니다.

    * 크기 최소화, 6층 보드 설계 사용
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이며, 지연 시간이 더 짧고 연결이 더 안정적
    * 탑재된 TMC5160은 익스트루더에 사용 가능
    * 탑재된 ADXL345 가속도 센서
    * PT100(메이스테이트 31865) 지원
    * 교체 가능한 팬 MOS 디자인
    * BLTOUCH, PL08N 등 레벨링 센서 지원, XY 리미트 스위치 통합
    * 12-24V 전압 입력 지원 
    * MCU: STM32F072
    * 통신: CANBUS, USB

## 제품 사양

    1. Klipper CAN 모드, RS232 모드, RRFCAN 모드 간의 전환 지원
    2. 연결 인터페이스 XT30(2+2) 커넥터 사용, 15A 전류 지원, 최대 30A
    3. MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    4. 입력 전압: DC12-24V 5A
    5. 히터 막대 최대 출력 전류: 5A
    6. 팬 인터페이스 최대 출력 전류 1A
