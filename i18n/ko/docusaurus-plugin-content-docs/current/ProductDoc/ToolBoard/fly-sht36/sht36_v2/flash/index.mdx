# 펌웨어 번인 설명

    * 펌웨어 컴파일 및 펌웨어 번인 모두 상위 기기를 사용해야 합니다
    * SHT36 V2는 CAN 연결 방식으로 상위 기기와 연결할 수 있습니다
    * CAN 연결에는 `XT30(2+2)-F`를 사용해야 합니다
    * 아래는 `XT30(2+2)-F` 데이터 라인 스타일입니다
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger 참고사항
* 메인보드 펌웨어 번인에는 Katapult를 통해 Klipper를 번인해야 합니다
* 메인보드에 Katapult가 이미 번인되어 있으면 마지막에 있는 Katapult 펌웨어 번인을 할 필요가 없습니다, 이는 메인보드의 Katapult 펌웨어가 성공적으로 번인되지 않은 경우를 대비한 것입니다
* 두 번 빠르게 리셋을 누르면 메인보드 위에 LED가 깜빡이는지 확인할 수 있습니다, 깜빡이면 메인보드가 Katapult 펌웨어로 진입한 것을 의미합니다
:::
