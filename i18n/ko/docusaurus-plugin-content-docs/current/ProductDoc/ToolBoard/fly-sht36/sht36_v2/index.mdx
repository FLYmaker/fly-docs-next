# 소개

## 제품 소개

    FLY SHT36 V2는 광저우 마렌 전자과학기술 유한회사가 3D 프린터 압출 헤드를 위해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면, 원래 메인보드와 프린트 헤드 사이의 복잡한 배선을 네 개의 선으로 대체하여 레이아웃을 최적화할 수 있습니다. SHT36 V2는 36 스테핑 모터에 적합하며, CAN 수신기, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등 다양한 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에 적합합니다.

    * 최대한 압축된 크기, 6층 보드 설계 사용
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연 시간이 적으며 연결이 더 안정적입니다.
    * 내장 TMC2209 드라이버, UART 모드 사용, 압출기용으로 사용 가능
    * 내장 ADXL345 가속도 센서
    * PT100(MAX31865) 지원
    * 교환 가능한 팬 MOS 디자인
    * BLTOUCH, PL08N 등 레벨링 센서 지원, XY 한계 스위치 통합
    * 12-24V 전압 입력 지원 
    * MCU: STM32F072
    * 통신: CAN
    * BLTOUCH, PL08N 등 레벨링 센서 지원, XY 한계 스위치 통합

## 신규 기능

    * 4.7K 저항(ntc 서미스터용) 또는 1.1K 저항(PT1000용) 전환 지원 추가
    * can 인터페이스 MX3.0 단자 사용 중단, 새로운 커넥터는 15A 전류를 지원하며 최대 30A까지 가능
    * 5/12/24V 팬 전압 선택 가능 추가
    * 내장 NTC100k 저항 추가, 온도 측정용
    * 전원 역접 방지 기능 추가
    * ADXL345의 INT1을 mcu에 연결하여 일부 사용자의 레벨링에 사용
    * 히팅 로드 포트 크기 증가, 정격 전류 10A
    * 고전압 입력 한계 추가
    * 메인 컨트롤러를 국산 MCU로 교체
    * 마그네틱 인코더(as5047d) 지원, Klipper에서 동작 분석에 사용(현재 Klipper는 지원하지 않음)
    * 1미터 CAN 연결 케이블 추가, 사용자가 잘못된 터미널 압축으로 인한 신호 손실 문제를 피할 수 있도록
    * 설치 크기: 홀 간격 43.84mm, M3 설치 홀 지름
    * 프로세서: GD32F103(2022-10-28일 이전 구매), APM32F072(2022-10-28일 이후 구매)
