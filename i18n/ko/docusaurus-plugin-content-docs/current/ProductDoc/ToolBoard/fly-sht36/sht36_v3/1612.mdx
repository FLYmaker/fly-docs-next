# Eddy 사용

    :::info

    * 스캔 베드 사용 시 단독 Z 제한을 사용하는 것이 좋습니다. 스캔 베드를 Z 제한으로 사용하지 마세요.
    * Eddy를 설치할 때, 노즐 아래 부분은 노즐 위로 최소 1-2mm 위치해야 합니다.
    * Eddy 주파수를 보정할 때, 코일 PCB는 히트베드에서 최소 5mm 이상 떨어져 있어야 합니다.
    * 전체 히트베드를 상온에서 테스트하고, 테스트 중에는 히트베드를 켜지 마세요.

    :::

## 참조 설정 및 사용 설명서

    ### ldc16121612 기본 주파수 수정
    이 방법은 sht36-v3/SB2040-v3 또는 SB2040-Pro-v3에만 사용됩니다.
    Mobaxterm 도구를 사용하여 명령을 실행할 수 있습니다.

    다운로드 주소:https://mobaxterm.mobatek.net/download.html
    
    [자세한 사용 방법은 여기](/docs/ProductDoc/SBC/fly-pi-v2/ssh)

    첫 번째 명령을 실행하고 엔터를 누르세요

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    두 번째 명령을 실행하고 엔터를 누르세요
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    교체가 성공하면 `LDC1612_FREQ = 40000000`이 포함된 줄이 반환됩니다. 교체가 성공하지 않았다면 아무것도 반환되지 않습니다.
    또는 이 명령을 사용하여 성공 여부를 확인할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
cat /home/fly/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   참고: 수정 후 klipper 서비스를 재시작하고 Eddy 주파수를 다시 보정하세요.
   ```cfg
   systemctl restart klipper
   ```
   Eddy 보정 주파수 명령
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### 스캔 베드 설정

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 참고: 현재 외부 1612 모듈이 출시되지 않았으므로 이 설정은 사용할 수 없습니다.
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 온도 보정 설정

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh 설정

    * [bed_mesh]의 `horizontal_move_z` 매개변수를 2로 설정하여 스캔 플랫폼 시 Eddy가 가능한 한 플랫폼에 가깝게 만듭니다.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
