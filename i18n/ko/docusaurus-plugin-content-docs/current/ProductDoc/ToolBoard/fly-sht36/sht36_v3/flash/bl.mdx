# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * SHT36_V3는 CAN과 RS232를 지원합니다.
    * 여기서는 CAN 펌웨어를 500K와 1M 속도로 제공합니다.
    :::

## 부팅 모드 진입

    :::tip

    화살표가 가리키는 것이 `BOOT 버튼`입니다.

    :::

    * 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송이 가능한 데이터 케이블을 메인보드와 장치에 연결합니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.png').default} size="100%" align="center" />

            4. 펌웨어 플래시

            :::tip

            하나의 펌웨어만 플래시하면 됩니다.

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래시 명령어입니다.
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래시 명령어입니다.
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. 아래와 같은 메시지가 나타나면 플래시가 성공적으로 완료된 것입니다.

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />


            :::tip

            플래시가 완료되면 메인보드의 전원을 완전히 한 번 꺼야 합니다.

            :::
