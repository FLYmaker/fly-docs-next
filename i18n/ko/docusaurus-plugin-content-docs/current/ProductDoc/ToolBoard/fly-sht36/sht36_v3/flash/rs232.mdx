# RS232 펌웨어 컴파일

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <RS232 />

    * 마이크로컨트롤러 시작 시 설정할 GPIO 핀을 선택합니다. 입력: !gpio5 주의: !는 대문자입니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 이 때 `Y` 키를 눌러 저장합니다
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령줄 인터페이스로 돌아갑니다
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 조금 걸릴 수 있습니다
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래와 같은 내용이 출력되면 컴파일이 성공적으로 완료된 것입니다
    
        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```

    :::tip

    화살표가 가리키는 곳이 `BOOT 버튼`입니다

    :::
