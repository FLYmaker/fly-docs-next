# 1. 제품 소개

FLY SHT36 V3는 광저우 멜로우 전자과학기술 유한공사가 3D 프린터 익스트루더 헤드를 위해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면 원래 메인보드와 프린트 헤드 사이의 복잡한 배선을 네 개의 선으로 대체할 수 있어 레이아웃을 최적화할 수 있습니다. SHT36 V3는 36 스텝 모터에 적합하며, CAN 수신기, RS232 수신기, TypeC 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 2개의 제어 가능한 팬 등 다른 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper와 RRF 펌웨어에 적합합니다.

    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/sht36.webp').default} size="100%" align="center" />


## 1.1 제품 특징

    * 부팅 버튼이 예약되어 있어 사용자가 TypeC를 통해 부팅 모드로 들어가 펌웨어를 업데이트할 수 있습니다.
    * Klipper 펌웨어, RRF 펌웨어를 지원합니다.
    * CAN 모드, RS232 모드, RRF CAN 모드를 지원합니다.
    * RS232 모드에서는 더 높은 전송 속도와 더 강력한 안정성을 제공합니다. CAN 모드에서는 CAN 확장을 지원하며, CAN 확장 인터페이스를 예약합니다.
    * 온도 센서 지원: 보드 내에 MAX31865 칩이 탑재되어 있어 PT100 및 PT1000 온도 센서와 호환됩니다.
    * 두 개의 제어 가능한 팬을 지원하며, 전압 선택 가능, Mos 소형 보드를 교체할 수 있어 유지 보수 및 교체가 용이합니다. 최대 출력 전류는 1A까지 가능합니다.
    * 보드 내 LIS2DW 가속도 센서와 LDC1612 센서가 탑재되어 있습니다.
    * RGB 인터페이스가 예약되어 있으며, 보드 내 2209 드라이버가 장착되어 있어 다양한 레벨링 센서를 지원합니다. 예를 들어, BLtouch, klicky, Voron Tap, PL08n 등
    * 12-24V 전압, 15A 전류 입력 지원, 히터 포트 크기 증가, 정격 전류 10A

## 1.2 새로운 기능

    * Klipper CAN 모드, RS232 모드, RRF CAN 모드 간의 전환을 지원합니다.
    * 새로운 RS232 모드가 추가되어 더 높은 전송 속도, 더 강력한 안정성, 디버깅 사용도 더 간단해졌습니다.
    * RRF CAN 모드가 새롭게 추가되어 RRF 도구 보드로 전환할 수 있게 되었습니다.
    * CAN 확장 인터페이스가 새롭게 추가되어 CAN 장치를 쉽게 확장할 수 있습니다.
    * 종단 저항 다이얼 스위치가 새롭게 추가되어 종단 저항 120R을 다이얼 선택으로 설정할 수 있으며, 점퍼가 필요 없습니다.
    * 1미터 전원/신호 연결선이 새롭게 추가되어, 연결선은 트위스트 설계를 사용하여 간섭 방지 성능이 더 강해졌습니다.

## 1.3 제품 사양

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하며 15A 전류를 지원하고, 최대 30A 피크
    * 펌웨어: Reprap/Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 보드 내 센서: LIS2DW와 LDC1612
    * 히터 인터페이스: 히터(HEAT0), 10A
    * 확장 인터페이스: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN 확장 포트
    * 팬 인터페이스: 두 개의 DC 제어 가능 팬
    * 모터 드라이버: 보드 내 TMC2209
    * 온도 센서: 한 개의 100K NTC 또는 PT1000(TH0), 한 개의 PT100/PT1000 선택 가능
