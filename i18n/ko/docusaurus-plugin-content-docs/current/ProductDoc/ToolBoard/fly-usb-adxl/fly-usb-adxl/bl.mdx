# Katapult 펌웨어

## 부팅 모드 진입

    :::tip

    원 안에 있는 것이 `BOOT 버튼`입니다.

    :::

    * 부팅 모드로 진입하려면 메인보드의 전원을 완전히 차단한 상태에서, BOOT 버튼을 누른 채로 데이터 전송이 가능한 USB 케이블을 메인보드와 장치에 연결합니다.
