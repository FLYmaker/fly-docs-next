# {props.name} 시스템 다운로드

:::warning

SD 카드 주의사항:

* 용량은 `16GB` 이상 또는 `128GB` 이하의 TF 카드여야 합니다.
* TF 카드의 전송 속도는 `class10` 등급 이상이어야 하며, 샌디스크 등 브랜드의 TF 카드 사용을 권장합니다.

:::

:::danger 주의

* 서버 문제로 인해 시스템 또는 클리퍼를 무작위로 업데이트하지 마세요.
* FLY OS 시스템을 다운로드하세요. 원본 Armbian의 사용 난이도가 높아 설치를 권장하지 않습니다.

:::

:::info 중요한 알림

* 다운로드한 시스템은 압축을 해제해야 하며, 직접 확장자명을 변경하거나 직접 녹음하지 마세요!!!
* 압축을 해제하지 않고 녹음하면 부팅이 불가능합니다!!!

:::
