# 라즈베리 파이 이미지


  :::warning

  SD 카드 주의사항:

  * 용량은 `16GB` 이상 또는 `128GB` 이하의 TF 카드여야 합니다.
  * TF 카드의 전송 속도는 `class10` 급 이상이어야 하며, 샌디스크 등의 브랜드 TF 카드 사용을 권장합니다.

  :::

  :::danger 주의사항

  * 서버 문제로 인해 시스템이나 klipper를 무작정 업데이트하지 마세요.
  * 라즈베리 파이 3B/3B+/4B/CM4/5B에 적합한 시스템 이미지입니다.
  * 이 이미지는 `bookworm` 기반으로 다양한 서비스를 설치합니다.
  * 이 이미지는 기본적으로 TFT 스크린을 설치 및 활성화했습니다. TFT 스크린을 사용하지 않으려면 `/boot/config.txt` 파일을 수동으로 수정해야 합니다.

  :::

  :::info 중요한 팁

  * 다운로드한 시스템은 압축을 해제해야 하며, 직접 확장자만 변경하거나 직접 기록하지 마세요!!!
  * 압축을 해제하지 않고 기록하면 부팅이 불가능합니다!!!

  :::
