# 시스템 업데이트 로그

    * 이 시스템을 사용하는 중 문제가 발생하면 고객 지원 그룹에 피드백을 주세요.

            ## v2
                * 2022-1
                * 배포 버전을 Bullseye로 변경
                * 커널을 5.10.85로 업데이트
                * Klipper, Moonraker, Fluidd, Mainsail, KlipperScreen 사전 설치
                * Klipper는 Python3 환경 사용, 중국어 파일명 문제 해결
                * boot 파티션을 fat으로 변경, Windows 시스템에서 직접 설정 파일을 수정할 수 있음
                * 새로운 설정 시스템 FLY-Config 추가
                * CatSerial 추가, ```http://ip:9999```에 접속하면 연결된 모든 시리얼 장치를 직접 볼 수 있음

            ## v2.1

            ## v2.9.3
                * FLY-Π 적응
