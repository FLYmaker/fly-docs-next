# FLY-Tools 설치

    1. 최신 버전의 Klipper를 가져옵니다

    ```bash
    cd ~/klipper
    git pull
    ```

    * 로컬 Klipper 소스 코드를 수정하여 가져오기 실패 시, 다음 명령어를 사용하세요. (이 작업은 이전에 수정한 파일을 폐기합니다)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. 최신 Katapult(원래 Canboot)를 가져옵니다

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Fly-Tools 서비스 설치

        * 설치 과정은 Git 서비스 제공업체에 의존합니다. 다운로드 진행이 멈추면 `CTRL+C`를 눌러 종료 후 다시 시도하세요
        * Github는 시간당 최대 요청 횟수가 60회, Gitee는 알 수 없음
        * 국내 사용자는 Gitee를 통해 설치 스크립트를 가져오는 것이 좋습니다
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Github를 통해 설치 스크립트를 가져옵니다

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
