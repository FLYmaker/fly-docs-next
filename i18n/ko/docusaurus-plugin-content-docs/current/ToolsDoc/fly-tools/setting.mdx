# 기능 데모

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * 호스트 이름 수정 지원
    * 시스템 시간대 수정 지원
    * 상위기 40PIN 확장 활성화 지원 (PWM 활성화 후 시리얼 연결로 상위기 연결 불가, 스크린 활성화 후 상위기의 가속도 포트 사용 불가)
    * 메인보드 모델 수정 지원
    * Klipper 웹 페이지 수정 지원, Fuidd와 Mainsail 지원
    * KlipperScreen 켜기 또는 끄기 지원
    * Klippy 프로세스를 지정된 CPU 코어에서 강제 실행 지원, 선택한 코어에서 Klipper 실행 가능
