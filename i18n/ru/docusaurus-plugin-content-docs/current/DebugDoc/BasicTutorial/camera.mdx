# Добавление USB-камеры


:::danger Обратите внимание

* Необходимо установить сервис `crowsnest`, как на lite2, так и на mini pad, если система не установлена.
* Требуется камера с поддержкой Plug-and-Play, которая может использоваться в системах на базе Linux, Armbian и т.д. с Klipper без установки дополнительных драйверов.
* В данном руководстве используется последняя версия Klipper, crowsnest, fuidd. Если используется другой метод, необходимо изучить его самостоятельно.

:::

## Открытие конфигурации

:::tip

* Идентификаторы камер могут быть разными в зависимости от системы. Нам нужно определить ID, который не изменится.

:::

    * В Fuidd найдите файл конфигурации, затем откройте `crowsnest.conf`
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf.png').default} size="100%" align="center" />

    * Найдите `device: ` и удалите ID после него
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf1.png').default} size="100%" align="center" />

## Поиск ID камеры

:::tip

* Убедитесь, что камера подключена к верхней машине

:::

    * Нажмите на вкладку `DEVICES`, затем выберите `VIDEO` и нажмите обновить
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf2.png').default} size="100%" align="center" />
    * Найдите `path_by_id` и скопируйте строку после него
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf3.png').default} size="100%" align="center" />

## Добавление ID

    * Добавьте ID и сохраните конфигурацию
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf4.png').default} size="100%" align="center" />

## Включение камеры

    * Нажмите на настройки, выберите камеру, затем нажмите добавить камеру
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf5.png').default} size="100%" align="center" />
    * Введите имя по вашему выбору и нажмите сохранить
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf6.png').default} size="100%" align="center" />

## Проверка активации

    * Нажмите на панель управления, чтобы убедиться, что изображение с камеры отображается
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf7.png').default} size="100%" align="center" />
    * Если изображение не отображается, перезагрузите систему.
