# Подключение SSH к верхнему компьютеру

:::danger

* Этот документ применим только к верхнему компьютеру FLY, для других устройств он служит только для справки.
* Если вы собираетесь компилировать и записывать прошивку, пожалуйста, смотрите руководство по сетевому подключению, а не по подключению через последовательный порт.

:::

* Можно подключиться к устройству через последовательный порт или Wi-Fi.
* Заранее подготовьте **MobaXterm_Personal** или другой **SSH терминал**.
* Убедитесь, что на SD-карте или M2WE записана **соответствующая система для верхнего компьютера**.
* **Если верхний компьютер не запускается нормально, отсоедините драйверы, концевые выключатели, вентиляторы и другие периферийные устройства!!!**
* **Наконец, подключитесь через сеть к SSH для удобства компиляции прошивки!!!**
* **Для компиляции прошивки смотрите руководство по подключению верхнего компьютера через Wi-Fi или сетевой кабель**

        ## Использование сетевого IP для подключения к SSH верхнего компьютера

        * Убедитесь, что система **успешно запустилась**.
        * Пожалуйста, войдите в **админ-панель маршрутизатора**, к которому подключен верхний компьютер, если вы не знаете его IP-адреса.
        * Убедитесь, что вы знаете **имя пользователя и пароль верхнего компьютера**. По умолчанию для системы FLY имя пользователя **fly**, пароль **mellow**.
        * Если плата запущена и подключена к сети, вы можете подключиться к SSH следующим способом:
        * Откройте MobaXterm, настройте его как показано, введите IP, который вы проверили ранее, или найдите его в админ-панели маршрутизатора.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Обратите внимание: здесь вводится имя пользователя верхнего компьютера**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль верхнего компьютера и нажмите Enter.

        **Обратите внимание: пароль не отображается, введите его и нажмите Enter.**

        По умолчанию для системы FLY, имя пользователя: `fly`, пароль: `mellow`
