# Изменить часовой пояс системы

    Пожалуйста, обратите внимание, что если вы используете версию FLY-OS 3.3.4 на устройстве FLY, вы можете использовать FLY-Tools для изменения часового пояса непосредственно.
    Для других устройств пожалуйста, выполните следующие действия


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    Пожалуйста, замените `<TIMEZONE>` местным часовым поясом. Вы можете найти список доступных часовых поясов на сайте http://www.i18nguy.com/unicode/tz.html.
    Для этого документа выберите ваш часовой пояс, например UTC. В таком случае команда будет выглядеть следующим образом: ```sudo timedatectl set-timezone \+00\:00```

    Пример использования команды при изменении часового поясаutc Europe/Moscow:```sudo timedatectl set-timezoneEurope/Moscow```
    В случае изменения времени на устройстве с помощью этой команды, не забудьте также обновить текущее время в системе.
