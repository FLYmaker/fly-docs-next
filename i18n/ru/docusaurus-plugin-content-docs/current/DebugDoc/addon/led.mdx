# Расширение управления светодиодами через плагины

Оригинальный документ

```cfg
https://github.com/julianschill/klipper-led_effect
```

## Установка

Пожалуйста, используйте **MobaXterm_Personal** и другие **SSH-инструменты** для подключения через **Wi-Fi** к вашему главному компьютеру, и убедитесь в следующем:

1. **Убедитесь, что на главном компьютере установлена прошивка Klipper**
2. **Убедитесь, что вы вошли в систему под пользователем, который установил Klipper**
3. **Убедитесь, что ваша раскладка клавиатуры установлена на английскую**
4. **Убедитесь, что ваш главный компьютер может нормально обнаруживать устройства**
5. **Убедитесь, что все вышеуказанные предупреждения выполнены, иначе вы не сможете перейти к следующему шагу**

Этот модуль можно установить в существующую установку Klipper с помощью скрипта установки.

```cfg
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Пример конфигурации

```cfg
[neopixel TP_led]
pin: PB7
chain_count: 25
# Количество светодиодов
# Количество светодиодов
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
