# Глава о верхнем ПК

## Как запустить новый верхний ПК

  1. Сначала определите, какой у вас верхний ПК, Lite 2, PI V2 или другой.
  2. Есть ли у вас следующее:
        * Программное обеспечение для записи
        * Соответствующий образ системы
        * Картридер, карта TF, кабель TYPE-C с функцией передачи данных

  Важно перед записью системы:
        * Убедитесь, что карта памяти имеет объем не менее `16GB` и не более `128G`, и скорость передачи должна быть `class10` или выше; рекомендуется использовать карты TF бренда SanDisk и т.д.
        * Пожалуйста, не обновляйте систему без необходимости!! Сервер находится за границей, и обновление может вызвать проблемы.
        * По возможности используйте рекомендованное нами программное обеспечение.
        * Желательно иметь базовые знания по LINUX; я лично рекомендую <Button variant="contained" disableElevation href="https://item.jd.com/12443890.html">《鸟哥的Linux私房菜 基础学习篇》</Button> и т.д.

  3. Образ системы FLY <Button variant="contained" disableElevation href="/docs/category/系统镜像">Образ системы</Button>
  4. Необходимые инструменты <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Инструменты для записи</Button>

## После загрузки различных инструментов и соответствующих образов системы

  * Распакуйте систему, после распаковки файл будет иметь расширение `.img`
  * Необходимо использовать SDFormatter для повторного форматирования карты памяти
  * Используйте `win32DiskImager` для записи системы
  * Не рекомендуется использовать `balenaetcher` для записи системы
  * Если у вас верхний ПК FLY, вам нужно открыть `FlyOS-Env.txt` для настройки
  * После записи системы, пожалуйста, не форматируйте ее!!!!

## Запуск системы

  * После записи системы установите карту системы в верхний ПК и включите питание
  * Теперь система записана и настроена; подключите компьютер с помощью кабеля TYPE-C с функцией передачи данных, используя последовательное соединение
  * Обратите внимание, что другие верхние ПК обычно показывают загрузочные журналы; если их нет, нажмите Enter дважды или переустановите систему
  * После успешного запуска вы можете подключиться к Wi-Fi

## О M2WE и использовании карты памяти

  * Если в машине установлены M2WE и карта памяти, система по умолчанию загружается с карты памяти
  * Рекомендуется использовать карту SD для записи системы; запись на M2WE через USB может вызвать проблемы из-за несовместимости драйверов; рекомендуется использовать карту памяти для записи на M2WE
  * Обратите внимание на соответствие загружаемого образа системы; если он не соответствует, система не запустится
  * Обратите внимание, что Lite2 и MINI PAD могут использовать только карту памяти

## Важные моменты при ежедневном использовании

  * Пожалуйста, не отключайте питание напрямую, требуется корректное выключение, иначе возможна потеря данных
  * Поскольку серверы GitHud находятся за границей, не рекомендуется самостоятельно обновлять или устанавливать различные плагины
