# Нестабильность температуры при начале печати, слишком долгое ожидание

    * Этот макрос можно сразу добавить в конфигурацию
    * Этот макрос предназначен для нормального ожидания повышения температуры, но как только температура достигает целевого значения, сразу переходит к следующему шагу
    ```cfg title="printer.cfg"
    [gcode_macro M109]
    rename_existing: M109.1
    gcode:
        {% set s = params.S|float %}
        M104 {% for p in params %}{'%s%s' % (p, params[p])}{% endfor %}  ; Установка температуры экструдера
        {% if s != 0 %}
        TEMPERATURE_WAIT SENSOR=extruder MINIMUM={s} MAXIMUM={s+1}   ; Ожидание температуры экструдера (в пределах 1 градуса)
        {% endif %}	
    ```
