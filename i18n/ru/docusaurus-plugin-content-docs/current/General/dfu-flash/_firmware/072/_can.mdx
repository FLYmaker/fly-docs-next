Вот как компилировать прошивку:
* Подключившись к SSH, введите следующие команды и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Здесь `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а `make menuconfig` компилирует прошивку. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Войдите в меню **Micro-controller Architecture**, затем выберите **STMicroelectronics STM32** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Войдите в меню **Processor model**, выберите **STM32F072** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.png').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.png').default} size="100%" align="Center" />

* Выберите **Communication interface**, выберите: **CAN bus (on PB8/PB9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.png').default} size="100%" align="Center" />
