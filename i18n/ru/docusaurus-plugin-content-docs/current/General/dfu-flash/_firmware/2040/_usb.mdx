Вот как скомпилировать прошивку:
* После подключения к SSH введите следующую команду и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а `make menuconfig` компилирует прошивку. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Войдите в меню **Micro-controller Architecture** и выберите **Raspberry Pi RP2040**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* **Communication interface**, выберите: **USBSERIAL**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb.webp').default} size="100%" align="Center" />
