Вот как скомпилировать прошивку:
* Подключитесь через SSH и введите следующую команду, затем нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Команда `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а `make menuconfig` запускает процесс компиляции. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Включить дополнительные опции низкоуровневой конфигурации** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Архитектура микроконтроллера** и выберите **Raspberry Pi RP2040**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Выберите **Смещение загрузчика**, выберите: **16KiB загрузчик**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* Выберите **Интерфейс связи**, выберите: **Мост USB в CAN**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/utoc.webp').default} size="100%" align="Center" />
