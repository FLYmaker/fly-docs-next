Ниже описывается, как скомпилировать прошивку:
* Подключившись к SSH, введите следующую команду и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а `make menuconfig` компилирует прошивку. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Войдите в меню **Micro-controller Architecture**, затем выберите **STMicroelectronics STM32** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Войдите в меню **Processor model**, выберите **STM32F405** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.png').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.png').default} size="100%" align="Center" />

* **Communication interface**, выберите: **USB (on PA11/PA12)**

   <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/usb.png').default} size="100%" align="Center" />
