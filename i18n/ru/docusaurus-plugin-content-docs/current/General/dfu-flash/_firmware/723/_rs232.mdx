Вот как скомпилировать прошивку:
* Подключитесь к SSH и введите следующую команду, затем нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а `make menuconfig` компилирует прошивку. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.png').default} size="100%" align="Center" />

* Выберите **Включить дополнительные опции низкоуровневой конфигурации** и нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.png').default} size="100%" align="Center" />

* Войдите в меню **Архитектура микроконтроллера** и выберите **STMicroelectronics STM32**, затем нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.png').default} size="100%" align="Center" />

* Войдите в меню **Модель процессора** и выберите **STM32H723**, затем нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.png').default} size="100%" align="Center" />

* Выберите **Смещение загрузчика**, выберите: **Без загрузчика**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.png').default} size="100%" align="Center" />

* Выберите **Ссылка на тактовый генератор (кристалл 8 МГц)**, выберите: **Кристалл 25 МГц**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.png').default} size="100%" align="Center" />

* **Интерфейс связи**, выберите: **Последовательный (на USART1 PA10/PA9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/uart.png').default} size="100%" align="Center" />
