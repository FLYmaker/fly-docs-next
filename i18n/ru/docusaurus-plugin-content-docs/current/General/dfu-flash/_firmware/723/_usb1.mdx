Вот как скомпилировать прошивку:
* После подключения к SSH введите следующие команды и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Здесь `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, `make menuconfig` компилирует прошивку. После выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **STMicroelectronics STM32**, затем нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Перейдите в меню **Processor model**, выберите **STM32H723** и нажмите Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.webp').default} size="100%" align="Center" />

* Выберите **Clock Reference (8 MHz crystal)**, выберите: **25 MHz crystal**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* **Communication interface**, выберите: **USB (on PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/usb1.webp').default} size="100%" align="Center" />
