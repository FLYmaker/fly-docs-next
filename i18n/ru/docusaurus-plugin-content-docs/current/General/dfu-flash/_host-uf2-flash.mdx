1. Установка инструмента для прошивки

    * Введите следующие команды в SSH
    * Эту операцию нужно выполнить только один раз при первой прошивке, в дальнейшем повторять не нужно
    ```bash
    cd ~/klipper/lib/rp2040_flash/ && make
    ```
    <ImageView image={require('@site/docs/General/dfu-flash/img/2040.webp').default} size="100%" align="left" />

2. Выполните следующие команды для проверки успешности подключения.

    * Введите `lsusb`
    * Если устройство с ID `2e8a:0003` не отображается, попробуйте снова войти в **режим прошивки**
    * Из-за проблем с версией системы может отображаться другое имя, если присутствует `2e8a:0003`, это значит, что режим прошивки активирован

        <ImageView image={require('@site/docs/General/get-id/img/2e8a.webp').default} size="80%" align="left" /> 

    :::warning Внимание

    Продолжайте выполнение следующих шагов только если присутствует `2e8a:0003`.

    :::

3. Загрузка предварительно скомпилированной прошивки

    * Здесь находятся предварительно скомпилированные прошивки
    * Обратите внимание, что верхнее устройство должно иметь доступ в интернет для установки
    * Выполните следующую команду на верхнем устройстве для загрузки пакета прошивки

    ```bash
    cd && git clone https://github.com/FengHua741/FLY_Katapult.git
    ```

    * Если пакет прошивки уже был загружен, рекомендуется обновить его перед выполнением оставшихся действий

    ```bash
    cd ~/FLY_Katapult && git pull
    ```
