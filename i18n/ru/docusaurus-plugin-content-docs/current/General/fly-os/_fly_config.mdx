# Использование FLYOS_Env

    :::tip Подсказка

    * С версии системы 3.3.4 можно напрямую использовать FLY-TOOLS для конфигурации системы, без необходимости входа в boot для изменения конфигурации
    * Обратите внимание, что старые системы не поддерживают прямую конфигурацию с помощью FLY-TOOLS
    * <Button variant="contained" disableElevation href="/docs/ToolsDoc/fly-tools/">Руководство по использованию FLY_TOOLS</Button>

    :::
        
### Описание конфигурации WIFI

    :::danger Важно

    * Метод конфигурации WIFI сложен, требуется подключение через последовательный порт к верхнему уровню для настройки WIFI

    :::
