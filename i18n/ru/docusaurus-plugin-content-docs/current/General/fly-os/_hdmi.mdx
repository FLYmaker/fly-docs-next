## Руководство по использованию FLY верхнего компьютера

:::tip
Откройте в браузере ваш IP:9999 для доступа к FLY-Tools
:::

:::tip
На верхнем компьютере FLY встроенный интерфейс HDMI - это Micro HDMI, обратите внимание при покупке HDMI кабеля!!!
:::

:::tip
Верхний компьютер FLY поддерживает однолинейную связь FPC: `FLY-PI-V2`, `FLY-C8` и `FLY-PRO-X10`
:::

:::tip
Используя модуль FPC к HDMI, можно использовать верхний компьютер, например, Raspberry Pi
:::

    * Для использования HDMI необходимо включить соответствующую опцию для активации экрана
