# Запись на M2WE через подготовку USB

### Подготовительные работы

    * Установите программу `Win32DiskImager`
    * Не рекомендуется использовать `balenaetcher` для записи системы
    * Распакуйте систему, после распаковки файл будет иметь расширение `.img`
    * Скачайте [M2WE BOOT драйвер](/docs/ResDownload/auxiliary_software/ "Нажмите для перехода") 

### Установка драйвера

    * Откройте программу `Zadig`
    * Кроме первого `Zadig`, который можно запустить без установки, остальные два нужно установить. Во время установки просто нажимайте `Далее` или `Next`
    * В `Zadig` в меню `Options` выберите `List All Devices`

    <ImageView image={require('@site/docs/General/fly-os/img/zadig.png').default} size="80%" align="left" />

Убедитесь, что устройство подключено к компьютеру при нажатии кнопки
:::

    * Откройте установленный FLY-BOOT, нажмите `Проверить устройство`
    * Если предыдущие шаги выполнены правильно, будет выведено сообщение о найденном доступном устройстве
    * Нажмите на `Запустить BOOT`, если все в порядке, появится сообщение об успешном включении `BOOT`

### Форматирование M2WE

    * Используйте программу `TFFormatter.exe` для форматирования диска `BOOT`

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * Или используйте встроенное в Windows управление дисками для удаления раздела, после удаления создайте новый раздел.
    * Теперь в проводнике появится USB-накопитель на 14.5 ГБ (16 ГБ eMMC будет отображаться как 14.5 ГБ)
    * Таким образом можно использовать Win32diskimager для записи системного образа, выбрав диск с этим 14.5 ГБ USB-накопителем
