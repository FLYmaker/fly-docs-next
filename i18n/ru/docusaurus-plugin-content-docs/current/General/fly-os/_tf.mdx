# Запись системы на TF карту

### Подготовительные работы

    * Установите программу `Win32DiskImager`
    * Распакуйте систему, файл после распаковки должен иметь расширение `.img`
    * Не рекомендуется использовать `balenaetcher` для записи системы
    * Вставьте TF карту в кардридер, подключите кардридер к USB порту компьютера
    * Если на TF карте есть данные, сделайте резервную копию, после записи системы не рекомендуется извлекать карту, если только не нужно записать её на M2WE

### Форматирование карты памяти

    * Отформатируйте TF карту с помощью программы `TFFormatter.exe`
    * Выберите диск, который нужно отформатировать

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.png').default} size="80%" align="left" />

    * Или используйте встроенный в Windows диспетчер дисков для удаления и создания новых разделов.
