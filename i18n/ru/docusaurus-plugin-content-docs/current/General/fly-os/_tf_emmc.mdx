# Запись на M2WE через карту TF

:::info
Системная миграция может полностью перенести образ с карты TF в M2WE. После завершения миграции содержимое M2WE будет совпадать с содержимым карты TF.
:::

    * Установите M2WE и включите питание
    * Сначала подключитесь к SSH через последовательный порт [подключение к SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Нажмите, чтобы перейти") 
    * Выполните следующую команду, может потребоваться ввести пароль, введите `mellow` и нажмите Enter, обратите внимание, что пароль не виден

    ```cfg
    sudo nand-sata-install
    ```

    * Выберите опцию `Boot from eMMC - system on eMMC` и нажмите Enter (обычно это опция по умолчанию, просто нажмите Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_1.webp').default} size="80%" align="left" />

    * Когда появится сообщение `This script will erase your eMMC. Continue?`, выберите `Yes` и нажмите Enter (обычно это опция по умолчанию, просто нажмите Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_2.webp').default} size="80%" align="left" />

    * Выберите тип файловой системы `ext4` и нажмите Enter (обычно это опция по умолчанию, просто нажмите Enter)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_3.webp').default} size="80%" align="left" />

    * Терпеливо ждите, пока не появится индикатор выполнения (если в течение 2 минут индикатор не появится, нажмите `CTRL+C`, чтобы прервать операцию, затем перезагрузите устройство и попробуйте снова)
    
    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_4.webp').default} size="80%" align="left" />
    * Во время процесса миграции системы не отключайте питание и не выполняйте другие операции, пока миграция не будет завершена
    * После завершения миграции просто нажмите Enter и дождитесь полного выключения устройства

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_5.webp').default} size="80%" align="left" />


    * После выключения устройства извлеките карту TF, снова подключите питание и включите устройство, теперь оно может загружаться через систему в eMMC
