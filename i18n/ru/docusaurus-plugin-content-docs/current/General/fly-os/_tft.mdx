:::tip
Откройте в браузере ваш IP:9999 для доступа к FLY-Tools
:::

    * Для использования TFT нужно включить опцию ниже
    * Обратите внимание, что TFT-экраны бывают трех типов:
        * `FLY-TFT-V1`V1 Экран сопротивления, подходит для всех верхних панелей FLY
        * `FLY-TFT-V2-R` Экран сопротивления V2, подходит для C8 и PI-V2, а также для Raspberry Pi с 40 пинами, но требует использования вместе с G2T
        * `FLY-TFT-V2-C` Экран емкости V2, подходит для C8 и PI-V2, а также для Raspberry Pi с 40 пинами, но требует использования вместе с G2T
