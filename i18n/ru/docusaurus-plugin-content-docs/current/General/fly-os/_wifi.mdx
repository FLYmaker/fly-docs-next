

    :::danger Важно

    * FLY верхнеуровневой машине требуется использовать модуль M2WE или 2.4G WIFI, иначе используйте WiFi
    * Обратите внимание, что не поддерживаются WiFi с китайскими именами и двухдиапазонные WiFi 2.4G и 5G
    * Если у вас mini pad или lite2, проигнорируйте это предупреждение

    :::

## Подключение верхнеуровневой машины

    * Если не удается настроить WiFi в разделе BOOT, можно воспользоваться этой конфигурацией для подключения к WiFi, обратите внимание, что необходимо использовать последовательный порт для подключения к верхнеуровневой машине
    * Сначала используйте последовательный порт [подключитесь к SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Нажмите для перехода") 
    * Выполните следующую команду, может потребоваться ввести пароль, введите пароль `mellow` и нажмите Enter, обратите внимание, что пароль не виден

    ```cfg
    sudo nmtui
    ```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.png').default} size="80%" align="left" />

    * Выберите `Activate a connection` и нажмите Enter

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.png').default} size="80%" align="left" />

    * Перейдите на следующий экран, выберите WiFi, к которому хотите подключиться, с помощью клавиш `↑ ↓` на клавиатуре, нажмите Enter, введите пароль и дождитесь успешного подключения
