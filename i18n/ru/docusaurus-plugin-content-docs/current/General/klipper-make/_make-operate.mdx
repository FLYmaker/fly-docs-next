## Инструкция по работе с страницей конфигурации компиляции прошивки Klipper

    :::tip

    Убедитесь, что на клавиатуре включен полуполный режим ввода, то есть английский режим

    :::

    1. Клавиши `↑` и `↓` на клавиатуре используются для перемещения курсора вверх и вниз для выбора пунктов меню
    2. Клавиши `Enter` или `Space` используются для выбора пунктов меню или входа в подменю
    3. Клавиша `ESC` используется для возврата на предыдущий уровень меню
    4. Клавиша `Q` используется для выхода из страницы конфигурации прошивки Klipper
    5. Клавиша `Y` используется при выходе из страницы конфигурации прошивки Klipper, если есть подсказка, для сохранения файла конфигурации

    :::warning

    Если в конфигурационной странице мало опций, сначала выберите `[ ] Включить дополнительные низкоуровневые опции конфигурации`, чтобы отобразить некоторые скрытые опции

    :::
