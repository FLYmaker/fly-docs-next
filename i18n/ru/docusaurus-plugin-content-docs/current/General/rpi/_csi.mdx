## Методы использования CSI

    * После завершения записи образа, войдите в /boot/config.txt и найдите `camera_auto_detect=1`, измените его на `camera_auto_detect=0`, затем добавьте конфигурацию в зависимости от модели камеры
    * Ниже приведены примеры конфигурации

    ```bash
    dtoverlay=imx219,cam1
    ```
