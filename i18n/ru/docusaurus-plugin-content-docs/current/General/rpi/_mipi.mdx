## Подключение Raspberry Pi 5 к MIPI

:::tip
FLY_MIPI специально разработан для Raspberry Pi 5, позволяя напрямую использовать устройства DSI и CSI
:::

### Подключение Raspberry Pi 5 к MIPI

    :::danger Важно

    * Интерфейсы DSI и CSI на Raspberry Pi 5 теперь переименованы в MIPI
    * Интерфейс MIPI можно настроить для прямого использования камеры CSI или экрана DSI
    * Обратите внимание на направление контактной панели

    :::

    * Raspberry Pi 5 теперь имеет два интерфейса MIPI, если смотреть спереди, можно увидеть метки 1 и 0

## Подключение экрана DSI

    * Если используется интерфейс №1, необходимо настроить экран следующим образом

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * Если используется интерфейс №0, необходимо настроить экран следующим образом

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```
    * Схема подключения

## Подключение камеры CSI

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * Обратите внимание, что модель камеры должна соответствовать купленной вами, данное руководство использует камеру imx219 в качестве примера

    * Если используется левый интерфейс MIPI, настройте камеру следующим образом
    ```bash
    dtoverlay=imx219,cam1
    ```

    * Если используется правый интерфейс MIPI, настройте камеру следующим образом
    ```bash
    dtoverlay=imx219,cam0
    ```
    * Схема подключения
    <ImageView image={require('@site/docs/General/rpi/img/mipi_csi.webp').default} size="100%" align="left" />
