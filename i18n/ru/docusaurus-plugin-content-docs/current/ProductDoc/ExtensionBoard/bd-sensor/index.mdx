# BDsensor

    BDsensor — это датчик расстояния, который можно использовать с 3D-принтерами для быстрой калибровки уровня печатной платформы и настройки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.png').default} size="50%" align="left" />

## Функции

    * Замена концевика оси Z
    * Без необходимости перемещения оси Z при сканировании, повышение скорости печати
    * Сканирование при движении по осям XY, позволяет проводить интенсивное зондирование печатной платформы


## Подключение

    * Питание: 5V
    * Прошивка: Klipper/Marlin
    * Подключение: Плата инструментов/главная плата (BDsensor будет использовать один порт серво и концевик)
    * Печатная платформа: обычная PEI, алюминиевая платформа и другие металлические поверхности, не поддерживаются магнитные платформы с отверстиями и неметаллические поверхности


## Замечания

    * KLipper требует загрузки klipper и последнего патча BD-SENSOR, а также повторной прошивки платы
    * Во время отладки убедитесь, что аварийная остановка всегда доступна для активации, чтобы избежать повреждения PEI
    * Рекомендуемая скорость сканирования платформы — 200!
    * Скорость возвращения оси Z не должна превышать 5
