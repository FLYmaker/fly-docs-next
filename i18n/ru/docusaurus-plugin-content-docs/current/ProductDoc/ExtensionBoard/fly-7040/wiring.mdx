## Подключение

    * Подключите к порту PWM на основной плате, для других плат необходимо смотреть соответствующую документацию
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-7040/img/wiring.png').default} size="80%" align="Center" />

## Пример конфигурации

    ```cfg
    [fan]
    ##    Вентилятор охлаждения печати
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Решение проблемы с автоматическим вращением при включении

    :::info
    * Здесь приводится только краткое описание
    * При компиляции прошивки добавьте сигнальный пин PWM и инвертируйте его состояние
    * Тестировалось на SUPER8PRO, использовался пин PF8, в конфигурации прошивки выберите `GPIO pins to set at micro-controller startup` и добавьте `!PF8`
    * Обратите внимание, что `!` - это английский восклицательный знак
    * Аналогично для других плат
    :::
