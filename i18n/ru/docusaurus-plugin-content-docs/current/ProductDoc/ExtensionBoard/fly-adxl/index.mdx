# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## Характеристики продукта

    * Подходит для 36 моторов и других устройств
    * Может подключаться к Raspberry Pi, FLY PI, Orange Pi, материнской плате и другим устройствам с интерфейсом SPI

## Спецификации продукта

    * Интерфейс на плате: разъем ZH1.5 6P
    * Способ связи: SPI
    * Рабочее напряжение: 5В

## Предупреждения

    :::warning Предупреждения

    * Рабочее напряжение 5В, пожалуйста, не перепутайте
    * Подходит для различных прошивок, пожалуйста, убедитесь, что возможно использование SPI-соединения

    :::
