# Добавление беспроводной кнопки

    :::info Важное уведомление
    * Для работы беспроводной кнопки необходимо установить батарею, модель батареи `CR927`
    :::

    * Вернитесь на главную страницу и нажмите на `+` в красном прямоугольнике на изображении
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button.png').default} size="80%" align="Center" />
    * После появления следующего экрана нажмите на беспроводную кнопку с установленной батареей
    * В красном прямоугольнике автоматически появится уникальный ID беспроводной кнопки
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-1.png').default} size="80%" align="Center" />
    * Введите желаемое имя для кнопки в красном прямоугольнике
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-2.png').default} size="80%" align="Center" />
    * Нажмите на первый красный прямоугольник, появится второй синий прямоугольник
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/button-3.png').default} size="80%" align="Center" />



    :::danger Обратите внимание

    * Настройка пользовательского GCODE и вызов API осуществляются через беспроводное соединение
    * Активация GPIO осуществляется через проводное соединение

    :::

## Пользовательский GCODE
    * Вы можете ввести пользовательский `GCODE` в красный прямоугольник, AirClick автоматически отправит его на выполнение в Klipper
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/gcode.png').default} size="80%" align="Center" />

## Вызов API
    * Возможно создание пользовательских макросов, экстренной остановки, перезапуска Klipper, перезапуска прошивки, выключения системы и т.д.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/api.png').default} size="80%" align="Center" />

## Активация GPIO
    * Для активации GPIO необходимо подключить STM32F072 к Klipper
    * Прошивка и подключение описаны в этом документе <Button variant="contained" disableElevation href="/docs/ProductDoc/ExtensionBoard/fly-airclick/offline-operation-support">Прошивка и подключение</Button>
    * Доступны IO от `PB0` до `PB9`, всего десять. После прошивки прошивки и подключения к Klipper необходимо только правильно настроить, ниже приведен пример настройки
    
        ```cfg
        [mcu AirClick]
        serial: /dev/serial/by-id/usb-Klipper_stm32f072xb_AirClick-if00
        ### Нужно изменить это место после поиска USB ID

        [gcode_button _test]
        pin: ^!AirClick:PB0
        press_gcode:
            M118 PB0

        [gcode_button _test1]
        pin: ^!AirClick:PB1
        press_gcode:
            M118 PB1

        [gcode_button _test2]
        pin: ^!AirClick:PB2
        press_gcode:
            M118 PB2

        [gcode_button _test3]
        pin: ^!AirClick:PB3
        press_gcode:
            M118 PB3

            M118 PB9
        ```
