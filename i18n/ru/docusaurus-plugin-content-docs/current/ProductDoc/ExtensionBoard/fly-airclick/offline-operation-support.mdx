# Конфигурация кабельногого связей

:::danger Уведомление

* Для поддержки проводногого связей с Клиппером необходимо загрузить прошивку на STM32F072.

:::

## Начать компиляцию

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь должно быть сохранено и вы покидаете в интерфейсной строке команд
    * Введите `make -j4` для начала компиляции, это займет немного времени
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если компиляция успешна, будет отображено сообщение:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Перейти в режим загрузчика

    * При отключенном питании AirClick удерживайте кнопку BOOT и подключите разъем передачи STM32 через тип-C к верхнему компьютеру
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.png').default} size="100%" align="center" />
