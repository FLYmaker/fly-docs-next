# Подключение к Klipper

    :::info Важное уведомление

    * Встроенная антенна этого продукта может использоваться только для приема, для передачи необходимо установить антенну самостоятельно, разъем антенны - `ipex3`
    * Пожалуйста, убедитесь, что антенна установлена правильно!!! Иначе из-за плохого сигнала возможен сильный нагрев!!!
    * В первый раз использования необходимо вручную подключиться к точке доступа AirClick, затем подключиться к соответствующей сети Wi-Fi

    :::

    :::info Важное уведомление

    * AirClick подключается к Moonraker и отправляет команды для управления контроллером Klipper
    * Возможности включают: пользовательские макросы, экстренную остановку, перезагрузку Klipper, перезагрузку прошивки, выключение системы и т.д.

    :::

## Подключение AirClick к Wi-Fi

    * Подключитесь к точке доступа AirClick, название точки доступа - `FLY-WBTN-XXXXXX`
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * После подключения к точке доступа откройте браузер и перейдите по адресу [192.168.4.1](http://192.168.4.1/#/setting)
    * Появится интерфейс настройки Wi-Fi, вручную подключитесь к маршрутизатору, к которому подключен ваш принтер, введите пароль и нажмите на синюю кнопку ниже для подключения к Wi-Fi
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * После подключения к Wi-Fi будет отображен IP-адрес AirClick после подключения к Wi-Fi, нажмите на синюю кнопку внизу для перезагрузки устройства
        * Цифра 1 указывает на IP-адрес AirClick
        * Цифра 2 указывает на копирование IP и перезагрузку устройства
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * Обратите внимание, что вам нужно вручную переключить текущую сеть на соответствующий маршрутизатор, иначе вы не сможете получить доступ

## Изменение языка
    * Нажмите на значок, выделенный в правом верхнем углу, и выберите нужный язык
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-2.webp').default} size="80%" align="Center" />

## Подключение к Klipper

    :::info Важное уведомление
    * AirClick подключается к Moonraker и отправляет команды для управления контроллером Klipper
    * Возможности включают: пользовательские макросы, экстренную остановку, перезагрузку Klipper, перезагрузку прошивки, выключение системы и т.д.
    :::

    * Нажмите на настройки системы слева
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />
    * Найдите поле `Moonraker IP` и введите IP-адрес вашего Klipper
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-1.webp').default} size="80%" align="Center" />
        * Цифра 1 - введите IP-адрес хост-машины Klipper
        * Цифра 2 - по умолчанию `7125`, если хост-машина использует другой порт, введите соответствующий порт
        * Цифра 3 - подтвердите и сохраните настройки
    * После успешного подключения в верхней части веб-страницы появится уведомление
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />
