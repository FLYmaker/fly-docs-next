# Введение

# Использование буфера LLL

    Буфер расходных материалов LLL - это устройство, разработанное для удаленной подачи филамента в 3D-принтер, с целью упрощения загрузки и выгрузки материалов. Буфер оснащен двумя кнопками для подачи и отката филамента, что позволяет пользователю управлять процессом электрически.

### Функция буферизации:

    Когда экструдер 3D-принтера подает филамент, если он тянет за материалом на дальней стойке, буфер обнаруживает это действие и автоматически подает филамент в направлении горячего конца. При каждом рывке буфер повторяет эту операцию подачи. Это эффективно снижает натяжение филамента экструдером, поддерживая стабильность его толкающей силы, что повышает стабильность качества печати.

### Функция обнаружения обрыва филамента:

    В буфере встроен датчик обрыва филамента, который при обнаружении отсутствия материала в буфере отправляет сигнал 3D-принтеру для приостановки печати. Для использования этой функции необходимо подключить сигнальные контакты и изменить конфигурацию 3D-принтера. В то время как функция буферизации может использоваться сразу без изменения конфигурации.

    Благодаря этим функциям, буфер расходных материалов LLL не только упрощает работу с материалами, но и повышает стабильность процесса печати и качество печати.

* Обратите внимание, что для использования функций необходимо подключить филамент, иначе можно управлять мотором только с помощью кнопок
