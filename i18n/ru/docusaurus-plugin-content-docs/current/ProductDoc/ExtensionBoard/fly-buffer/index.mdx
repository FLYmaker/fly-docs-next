# Введение

# Использование LLL буфера

 	Буфер расходных материалов LLL предназначен для дистанционной подачи материалов в 3D-принтере, упрощая процесс загрузки и выгрузки материала. Буфер оснащен двумя кнопками для подачи и вытягивания материала, что позволяет пользователю выполнять электрическую подачу и вытягивание материала.

### Буферная функция:

 	Когда экструдер 3D-принтера подает материал, если материал зацепляется за удаленную стойку с материалами, буфер обнаруживает это действие и автоматически подает материал в направлении горячего конца. При каждом таком захвате буфер повторяет операцию подачи материала. Это эффективно уменьшает натяжение материала экструдером, обеспечивая стабильность давления экструдера, что повышает стабильность качества печати.

### Функция обнаружения обрыва материала:

 	В буфер встроен датчик обрыва материала, который отправляет сигнал 3D-принтеру на паузу печати, когда обнаруживается отсутствие материала в буфере. Для использования этой функции необходимо подключить сигнальные контакты и изменить конфигурацию 3D-принтера. Буферная функция работает без изменения конфигурации.

 	Благодаря этим функциональным возможностям, буфер расходных материалов LLL не только упрощает работу с материалами, но и повышает стабильность процесса 3D-печати и качество печати.



* Обратите внимание, что для использования функций необходимо подключить материал, в противном случае можно управлять работой мотора только через кнопки
