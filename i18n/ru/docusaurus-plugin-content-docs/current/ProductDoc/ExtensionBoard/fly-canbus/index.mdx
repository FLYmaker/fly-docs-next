# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.png').default} size="40%" align="left" />


## Характеристики продукта

    * Поддержка CAN и разделение питания на четыре, позволяя одновременно расширить четыре платы CAN-инструментов
    * На плате имеется интерфейс RJ11, предназначенный для подключения и общения с системами, использующими протокол CAN. Хотя RJ11 не является стандартным интерфейсом для CAN-систем, наш продукт предоставляет возможность их плавного соединения
    * Эргономичный дизайн, позволяющий легко устанавливать плату на направляющую с помощью печатных компонентов

## Технические спецификации

    * Коммуникационный интерфейс: RJ11*1, XH2.54*5
    * Питание: KF128-5.028-2P*5
    * Входное напряжение: 5В-24В
    * Совместимость: RRF/Klipper/и другие CAN

## Предупреждения

    :::warning Внимание

    * Обратите внимание, что напряжение на выходе будет таким же, как и на входе
    * Обратите внимание, не перепутайте порядок проводов
    * Убедитесь, что перед установкой расширения платы на верхнем уровне питание отключено.
    * Избегайте использования платы расширения в влажной или пыльной среде.
    * При установке оборудования обратите внимание на напряжение платы инструмента, чтобы избежать повреждений.

    :::
