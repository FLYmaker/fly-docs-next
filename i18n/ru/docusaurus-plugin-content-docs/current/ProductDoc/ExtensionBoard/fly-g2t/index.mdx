# FLY-G2T

    Многофункциональная плата расширения FLY-G2T предназначена для расширения функциональности Raspberry Pi и FLY верхнего уровня. Это высокопроизводительный модуль, поддерживающий UART-коммуникацию, с интегрированным FPC интерфейсом для внешнего подключения TFT-V2, а также поддерживающий управление вентилятором на 5В. Это делает его идеальным решением для улучшения интерактивности и удобства использования Raspberry Pi в различных приложениях, таких как промышленная автоматизация, 3D-печать, домашняя автоматизация, управление роботами или любые сценарии, требующие человеко-машинного интерфейса и внешней коммуникации.

## Характеристики продукта

    * Совместимость: поддержка различных версий Raspberry Pi, включая Raspberry Pi Zero/3B/3B+/4B и другие.
    * Преобразование UART в RS232: обеспечивает стабильную последовательную связь, позволяя Raspberry Pi обмениваться данными с другими устройствами через интерфейс RS232.
    * Экран емкостного сенсорного ввода SPI: через интерфейс FPC подключается к TFT-V2 емкостному экрану, предоставляя интуитивно понятное сенсорное управление, подходит для приложений, требующих графического пользовательского интерфейса, таких как klipperscreen и другие.
    * Поддержка вентилятора на 5В: встроенный интерфейс для вентилятора на 5В, позволяет подключать вентилятор для охлаждения, обеспечивая стабильность верхнего уровня при длительной работе или высокой нагрузке.
    * Дизайн с возможностью быстрого подключения и отключения: плата расширения спроектирована для легкой установки, просто вставляется и вынимается из верхнего уровня без необходимости дополнительных проводов или сложной настройки.
    * Компактные размеры: плата расширения имеет разумные размеры, не занимает много места, что упрощает интеграцию в различные проекты.

## Спецификации продукта

    * Интерфейс связи: UART в RS232
    * Дисплейный интерфейс: SPI в FPC интерфейс
    * Питание вентилятора: 5В
    * Совместимость: Raspberry Pi Zero/3B/3B+/4B, FLY Pi V2 и другие

## Важные замечания

    :::warning Важные замечания

    * При подключении инструментальной платы через RS232, убедитесь, что верхний уровень `объединен с землей`!!!!

    :::

    * Перед установкой платы расширения убедитесь, что верхний уровень отключен от источника питания.
    * Избегайте использования платы расширения во влажной или пыльной среде.
    * При установке экрана обращайте внимание на направление кабеля экрана и не подключайте его под напряжением, чтобы избежать повреждений.
    * При установке вентилятора проверьте его ток и напряжение, чтобы избежать повреждений.
