# FLY_halo

## 1. Главная страница

    Нажмите на термостол или экструдер, затем прокрутите по часовой стрелке с верхней части экрана, чтобы установить нужную температуру, и снова нажмите на термостол или экструдер.

    Если необходимо отобразить температуру микроконтроллера и температуру на хосте, добавьте следующую конфигурацию:

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # связано с микроконтроллером (по умолчанию)

    [temperature_sensor host]        # температура хоста
    sensor_type: temperature_host     # связано с хостом
    ```


    :::warning Внимание

    * Обратите внимание на максимальную температуру, установленную в конфигурации.

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.webp').default} size="50%" align="left" />
