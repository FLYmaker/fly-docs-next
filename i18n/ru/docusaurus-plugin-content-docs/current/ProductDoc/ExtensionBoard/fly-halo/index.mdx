# Введение в продукт

    * Главный контроллер использует ESP32-S3R8, производительность выше, чем у ESP32. Имеет 8MB PSRAM и 16MB Flash. Экран - IPS, высокое разрешение, яркость, широкие углы обзора, тонкая цветопередача. Поддерживает емкостное касание.

## Характеристики продукта

    :::tip

    * В настоящее время всего 6 страниц, переключение слайдами влево-вправо и вверх-вниз.

    :::


    * Главная (показывает режим ожидания, печать, остановка, ошибки и т.д.)

    * Экструзия (контроль экструзии, обратной экструзии, температуры экструзии)

    * Системные службы (контроль запуска, остановки, перезапуска хоста, klipper, moonraker и др.)

    * Настройки (для настройки различных параметров halo, wifi, ip, языка, яркости подсветки и т.д.)

    * Информация об устройстве (отображение информации о halo, версия прошивки, проверка обновлений и т.д.)

    * Детекция материала: обнаружение обрыва, засорения, длины материала, возможность автоматической остановки и поднятия GPIO47 на заданную длину засорения (все эти функции по умолчанию отключены, можно активировать в настройках)

    * Автоматическое выключение после печати: поддержка задержки выключения хоста и поднятия GPIO48 после завершения печати (все эти функции по умолчанию отключены, можно активировать в настройках)
