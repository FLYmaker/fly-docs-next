## Установка на Voron Stealthburner

    * Для установки Halo на Stealthburner требуются адаптеры, которые мы предоставляем
    * Сначала скачайте и распечатайте два STL-файла, которые мы предоставляем:
        * [Stealthburner_main_body(Adapt_to_Halo).stl(скачать)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(скачать)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(скачать)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Установка переходной платы и подключение кабелей
    * Позиция вывода кабеля может быть выбрана в соответствии с вашими предпочтениями, не обязательно прикреплять её к задней стороне вентилятора
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.png').default} size="50%" align="left" />


    :::warning Важное замечание

    * Обозначение цветов проводов: **черный - отрицательный (GND)**, **красный - положительный (VCC), приемлемое напряжение DC12-24V**, **белый - GPIO47 для обнаружения закупорки**, **желтый - GPIO48 для остановки после печати**

    :::
