# Введение в операции на странице Halo

## Структура страницы

```bash
  Страница печати файлов           Главная страница             Страница экструзии            Системные службы             Настройки              Информация об устройстве
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Print Files |<-->|     Home    |<-->|   Extrude   |<-->| Sys Service |<-->|   Setting   |<-->|  Dev Info   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blocking Det|
                   |             |
                   +-------------+
                     Страница обнаружения закупорки
            (Эта страница появится только если в настройках активировано обнаружение закупорки)
```

## Страница экструзии

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.png').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.png').default} size="50%" align="left" />
    * Область взаимодействия на рисунке 1 соответствует номерам ниже
    1. Однократное нажатие: Увеличение целевой температуры экструдера на 1
    2. Долгое нажатие: Вход в настройку температуры экструзии, регулировка температуры с помощью круговой шкалы, повторное нажатие подтверждает установленную температуру
    3. Однократное нажатие: Уменьшение целевой температуры экструдера на 1
    4. Однократное нажатие: Экструзия
    5. Прокрутка вверх/вниз: Выбор длины экструзии или ретракта, в мм
    6. Однократное нажатие: Ретракт
    7. Долгое нажатие: Вход в настройку скорости экструзии на рисунке 2, прокрутка для выбора скорости экструзии, в мм/с, после выбора нажмите на пустое пространство вокруг для выхода
