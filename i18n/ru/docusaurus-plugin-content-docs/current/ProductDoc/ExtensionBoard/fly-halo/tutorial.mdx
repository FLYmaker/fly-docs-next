# Руководство по использованию

## Меры предосторожности

    * При первом использовании необходимо подключение к интернету, если ваш WIFI не может подключиться к интернету, halo не сможет получить конфигурацию при первом включении.
    * Все настройки в странице настроек вступят в силу только после нажатия на кнопку **Сохранить настройки** и перезагрузки устройства.
    * Функция обнаружения застревания материала по умолчанию отключена, включите ее в настройках и перезагрузите устройство, чтобы получить доступ к ней с главной страницы, проведя пальцем вверх.
    * Функция автоматического выключения после печати по умолчанию отключена, ее можно включить в настройках.
    * Все кнопки, требующие отмены или приостановки печати в всплывающих окнах, должны быть зажаты, одиночное нажатие не будет действовать (для предотвращения случайных нажатий).
    * В разделе системных служб Host служба - это основной сервер верхнего уровня.
    * При использовании официальной прошивки не пытайтесь заменять главный микроконтроллер или чип FLASH.

    :::tip

    * Обратите внимание: если размер вашего файла конфигурации `moonraker.conf` превышает 4MB, вы можете столкнуться с проблемами в работе. Если возникают ошибки, уменьшите размер файла конфигурации `moonraker.conf`.

    :::

## Обновление прошивки системы

    * Обновление версии halo делится на полное обновление и инкрементное обновление OTA:
        * Полное обновление: связано с крупными обновлениями нижнего уровня, требуется использование USB для прошивки полного пакета прошивки.
        * Инкрементное обновление: стандартное OTA обновление, проверьте обновления на странице информации об устройстве.

## Использование телефона для сканирования QR-кода и настройки WIFI

    :::tip

    * Некоторые мобильные устройства могут не иметь функции сканирования QR-кода для подключения к WiFi, можно подключиться к точке доступа с именем **FLY-halo-xxxxxxxx** через настройки WIFI телефона.

    :::

    :::tip

    * Если после подключения к точке доступа не открылась веб-страница, откройте ссылку в браузере вручную.

    :::

    :::tip

    * Обратите внимание, что HALO может использовать только WiFi 2.4G.

    :::

    :::tip

    * Экран Halo не может найти скрытую сеть WIFI.

    :::
