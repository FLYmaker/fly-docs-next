## Подготовка к записи системы

    * Картридер
    * Карта TF емкостью не менее 16 ГБ и не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент для записи</Button>

## Запись системы

:::info
M2WE поддерживает три режима, я рекомендую использовать миграцию с TF-карты на M2WE и запись на M2WE через M2SD Burner
:::

:::info
* Если устройство оснащено M2WE и картой памяти, система загружается по умолчанию с карты памяти
* Рекомендуется использовать TF-карту для записи системы на M2WE, так как запись через USB на M2WE может не удаться из-за проблем совместимости драйверов, рекомендуется использовать [запись с карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие скачанного образа системы, иначе система не запустится
:::
