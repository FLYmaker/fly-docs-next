## Внимание

:::danger Внимание

* M2WE из коробки уже прошит FLYOS и готов к запуску
* M2WE можно использовать только с продуктами FLY, другие устройства не поддерживаются
* Антенное гнездо M2WE — это `IPEX1`
* M2WE не поддерживает объединение 2.4G и 5G, китайские символы, специальные символы, скрытые по умолчанию WIFI

:::
