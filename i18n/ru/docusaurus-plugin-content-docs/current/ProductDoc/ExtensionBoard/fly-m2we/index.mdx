## Важные замечания

:::danger Важные замечания

* M2WE уже предварительно прошит FLYOS и может быть запущен сразу после покупки
* M2WE может использоваться только с продуктами FLY и не поддерживает другие устройства
* Антенное гнездо M2WE — `IPEX1`
* M2WE не поддерживает одновременную работу в диапазонах 2.4G и 5G, китайские иероглифы, специальные символы, скрытые по умолчанию WIFI

:::
