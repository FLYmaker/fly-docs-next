# FLY VoltUp Power

    Модуль повышения напряжения FLY-VoltUp Power разработан специально для высоковольтного привода, пользователям не нужно покупать дополнительные высоковольтные источники питания для реализации высоковольтного привода, что помогает экономить место при проектировании 3D-принтера

## Характеристики продукта

    * Выходное напряжение 48V/60V на выбор
    * Встроенный разъем для вентилятора, а также крепление для вентилятора 4010, выходное напряжение равно входному
    * Расширение двойного источника питания: предоставляется один разъем питания, позволяющий пользователю одновременно расширить два выхода питания, причем можно отдельно настроить один из источников на выход 48V или 60V
    * Легкость установки: стандартные размеры для различных условий, не требуется дополнительная установка драйвера
    * Компактные размеры: размеры платы расширения разумны, не занимает много места, удобна для интеграции в различные проекты

## Технические характеристики:

    * Разъем питания: 12В/24В
    * Выходной разъем: два источника питания на выбор 48В/60В (выбор через перемычку)
    * Разъем для вентилятора: два XH2.54 2P VCC FAN


## Внимание:

    * Перед установкой отключите питание и выключите все устройства
    * При выходе 48В, максимальный ток двигателя можно установить на 2А, а при выходе 60В - на 1А
    * Напряжение на двух разъемах VCC FAN соответствует входному напряжению, выходное напряжение равно напряжению питания
    * Модуль предусматривает крепление для вентилятора 4010, так как модуль повышения напряжения выделяет большое количество тепла, обязательно используйте вентилятор для охлаждения
    * Встроенный чип защиты от обратной полярности и короткого замыкания, предотвращающий повреждение компонентов из-за ошибок пользователя
    * Не вставляйте одновременно перемычки выбора напряжения, чтобы избежать повреждения компонентов
    * Правильно подключите полярность, чтобы избежать повреждения компонентов
