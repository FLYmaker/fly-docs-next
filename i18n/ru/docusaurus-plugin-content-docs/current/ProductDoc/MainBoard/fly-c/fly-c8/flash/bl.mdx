# BL прошивка

    :::tip

    * Обратите внимание, что `bootloader` уже предустановлен на заводе, не нужно его перепрошивать

    :::

## Вход в режим прошивки


    * Существует два способа входа в режим прошивки
    * Первый способ: при полном отключении питания платы, долго удерживайте кнопку BOOT, затем подключите кабель передачи данных с одной стороны к плате, а с другой - к вашему устройству, подходит для WINDOWS
    * Второй способ: после подключения кабеля передачи данных с одной стороны к плате, а с другой - к вашему устройству, долго удерживайте кнопку BOOT и введите команду, подходит для хост-компьютера
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.png').default} size="100%" align="center" />

            * Переключите переключатели `3`, `4` влево, иначе это повлияет на подключение нижней платы к хост-компьютеру

                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
