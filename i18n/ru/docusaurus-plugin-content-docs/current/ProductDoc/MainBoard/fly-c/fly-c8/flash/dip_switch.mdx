# Введение в переключатели

:::tip
FLY Gemini подключается к верхнему компьютеру через переключатели, если положение переключателей неверно, то использовать его не получится
:::

:::danger Обратите внимание
Пожалуйста, обратите внимание, что переключатели должны быть установлены в указанное положение для нормального подключения к верхнему компьютеру
:::

## Подключение к встроенному верхнему компьютеру

    * Переключатели `1`,`2` переключите вправо, чтобы подключить нижний компьютер к верхнему компьютеру через встроенный USB
    * Переключатели `3`,`4` переключите влево, иначе это повлияет на подключение нижнего компьютера к верхнему
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.png').default} size="100%" align="center" />
    * После переключения переключателей таким образом, нижний USB-порт будет занят нижним компьютером, пожалуйста, не подключайте другие устройства
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="100%" align="center" />


## Подключение к внешнему верхнему компьютеру

    * Переключатели `3`,`4` переключите вправо, чтобы подключить нижний компьютер к верхнему компьютеру через встроенный USB
    * Переключатели `1`,`2` переключите влево, иначе это повлияет на подключение нижнего компьютера к верхнему
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.png').default} size="100%" align="center" />
