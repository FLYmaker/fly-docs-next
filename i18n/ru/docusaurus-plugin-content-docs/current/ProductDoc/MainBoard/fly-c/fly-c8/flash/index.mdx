# Инструкция по прошивке прошивки

    
## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только одну прошивку

    :::
    
    * C8 может быть подключен к верхнему уровню следующими способами:
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Плюсы: надежная связь, стабильность
        * Минусы: при наличии нескольких USB-устройств может влиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">Прошивка с USB-мостом</Button>
        * Плюсы: можно подключить несколько устройств без использования USB-портов
        * Минусы: может вызвать ошибки, такие как таймаут при использовании вместе с `KlipperScreen`.
