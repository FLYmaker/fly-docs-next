## Подготовка к записи системы

    * Картридер
    * TF-карта, объем которой должен быть не менее 16GB и не более 128GB, скорость передачи данных должна быть класса 10 или выше
    * Загрузка соответствующего <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образа системы</Button>
    * Загрузка необходимого <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмента для записи</Button>

## Запись системы

:::info
Запись на TF-карту и запись через TF-карту на M2WE требует только одного из этих методов.
:::

:::info
* Если на устройстве установлены M2WE и карта памяти, система по умолчанию запускается с карты памяти.
* Рекомендуется использовать TF-карту для записи системы на M2WE, так как запись через USB может не удаться из-за проблем с совместимостью драйверов. Рекомендуется использовать [запись на M2WE через карту памяти](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие загруженного образа системы, иначе система не запустится.
:::
