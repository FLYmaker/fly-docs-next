## Характеристики продукта

    * Использует 32-битный процессор ARM Cortex-M0+ серии STM32F407VGT6 с частотой 168MHz
    * На плате имеются драйверы TMC с режимами работы SPI и UART
    * На плате имеется вывод функции DIAG, позволяющий использовать бесконечный возврат домой простым переключением перемычки
    * Вентиляторы с числовым управлением на 24В, 12В, 5В на выбор, избавляя от необходимости покупки модулей понижения напряжения
    * Новый способ обновления прошивки через FLY-Tools
    * Многоуровневое управление питанием, короткое замыкание портов, концевиков, драйверов не влияет на работу верхнего уровня
    * Зарезервирован интерфейс RGB с поддержкой (12В, 5В) на выбор
    * Зарезервирован интерфейс SPI для подключения внешнего датчика ускорения для компенсации ускорения
    * Поддержка однолинейного соединения с экранами TFT и HDMI
    * Предустановлены klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, плагин компенсации резонанса numpy и FLY-Tools, нет необходимости в замене источника и сложных операциях, что упрощает использование для новичков (требуется покупка TF-карты или M2WE)

## Параметры продукта

### Верхний уровень
    * CPU: высокопроизводительный чип Allwinner H5, 4-ядерный 64-битный Cortex-A53
    * GPU: высокопроизводительный 6-ядерный Mali 450, скорость заполнения пикселей более 2.7 gpixel/s
    * ОЗУ: 1ГБ DDR3 (совместное использование с GPU)
    * ПЗУ: поддержка до 128ГБ SDCARD
    * Периферия: Spi x1, TFT x1, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1, 3.5мм аудио интерфейс x1
    * Интерфейсы: встроенный интерфейс M.2, позволяет расширить возможности WIFI, eMMC и т.д. (проприетарный протокол, не подключайте несовместимые с FLY π устройства M.2)
    * Зарезервирован один интерфейс ZH1.5 для контролируемого 5В вентилятора, используемого для охлаждения верхнего уровня
    * Поддержка однолинейного соединения с экранами TFT и HDMI через FPC
    * Зарезервирован интерфейс CAN для мостовой связи
    * Предустановлены klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, плагин компенсации резонанса numpy и FLY-Tools, нет необходимости в замене источника и сложных операциях, что упрощает использование для новичков (требуется покупка TF-карты или M2WE)

### Нижний уровень

    * МК: STM32F407VGT6
    * Прошивка: KLIPPER/RRF/Marlin
    * Поддержка питания постоянным током 12-24В, более стабильное питание, не требуется модуль понижения напряжения
    * Максимальный выходной ток порта нагревательной платформы: 15A
    * Максимальный выходной ток порта нагревательного стержня: 6A
    * Максимальный выходной ток порта вентилятора: 1A
    * Режимы работы драйвера: SPI, UART, STEP/DIR
    * Поддержка драйверов: внешние TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и т.д.
    * Расширенные интерфейсы: BLTouch (Servos, Probe), RGB x2, UART x1, 12864 x1
    * Вентиляторы: 6 штук двухпроводных числовых вентиляторов (поддержка 24В, 12В, 5В на выбор)
    * Датчики температуры: 3 канала NTC 100K
    * Поддержка структур машин: Cartesian, Delta, Kossel, Ultimaker, CoreXY
