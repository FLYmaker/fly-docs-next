# Подключение к SSH хост-машины


* Можно подключаться к устройству через последовательный порт или Wi-Fi
* Заранее подготовьте **MobaXterm_Personal** или другой **SSH-терминал**
* Убедитесь, что на SD-карте или M2WE записана **система для хост-машины**
* **Если хост-машина не запускается, снимите драйверы, ограничители, вентиляторы и т.д.!!!**
* **Наконец, подключитесь к SSH через сеть для удобства компиляции прошивки!!!**
* **Для компиляции прошивки смотрите руководство по подключению к хост-машине через Wi-Fi или сетевой кабель**

        ## Подключение к SSH хост-машины по IP-адресу

        * Войдите в **панель администратора маршрутизатора**
        * Убедитесь, что система **запущена корректно**
        * Убедитесь, что вы **знаете имя пользователя и пароль хост-машины**, по умолчанию для системы FLY имя пользователя **fly**, пароль **mellow**
        * Если плата запущена и подключена к сети, можно подключиться к SSH по IP-адресу
        * Откройте MobaXterm, настройте следующее, введя IP-адрес, который вы видели ранее, или проверьте его в панели администратора маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на шаге 4 введите имя пользователя хост-машины**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль хост-машины и нажмите Enter

        **Внимание: вводимый пароль не будет виден, после ввода сразу нажмите Enter**

        По умолчанию для системы FLY, имя пользователя: `fly`, пароль: `mellow`
