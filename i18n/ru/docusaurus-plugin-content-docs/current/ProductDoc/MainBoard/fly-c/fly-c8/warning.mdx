# Важные моменты о материнской плате

## Индикатор питания
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.webp').default} size="80%" align="center" />
    * При питании только через TYPE-C и без подключения других источников питания материнской платы, должны гореть по крайней мере два индикатора `3.3V` и `5V`
    * После подключения питания материнская плата должна иметь включенные индикаторы `3.3V`, `5V`, `VCC`

### Устранение неполадок


    :::danger Важно

    * В отсутствие подключенных устройств (драйверы, концевые выключатели, 12864 и т.д.)
    * Только через TYPE-C питание должно постоянно гореть `3.3V`

    :::

    * Если после подключения питания материнской платы и питания драйвера индикатор `5V` не горит, свяжитесь с поддержкой клиентов

## Драйверы

    * Материнская плата поддерживает внешние драйверы     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">Руководство по подключению и настройке</Button>

## Концевые выключатели

    * Обратите внимание на порядок подключения, для трехпроводных концевых выключателей важно правильно подключить 5V, GND, S, в противном случае возможен повреждение материнской платы
    * Если два провода, подключите S к GND напрямую
    * Для PL08 и других бесконтактных выключателей необходимо подключить диод к сигнальному входу для снижения напряжения до уровня ниже 3.3V
