# Подключение к Wi-Fi


## Установка антенны на C8

    * Красной стрелкой указано место для подключения Wi-Fi антенны
    * Разъем для Wi-Fi антенны - `ipex1`
    * Установите антенну на C8, иначе из-за слабого сигнала может быть невозможно подключиться к сети
