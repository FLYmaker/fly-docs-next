# Загрузчик прошивки

## Вход в режим прошивки

    :::tip
 
    Справа кнопка `BOOT`, слева кнопка `RESET`

    :::

    * Есть два способа войти в режим прошивки
    * Первый: при полном отключении питания платы, зажмите кнопку BOOT, затем подключите кабель данных к плате и к вашему устройству
    * Второй: после подключения кабеля данных к плате и к вашему устройству, зажмите кнопку BOOT, затем нажмите кнопку RST, затем отпустите кнопку RST и в последнюю очередь отпустите кнопку BOOT
