# Компиляция и прошивка прошивки

    :::tip

    * CDV по умолчанию использует карту TF для прошивки
    * Подготовьте карту TF емкостью не более `32G`
    * Некачественные карты памяти могут привести к сбоям при прошивке, поэтому желательно использовать рекомендуемые карты памяти для обновления прошивки
    * Скорость передачи данных карты TF должна быть `class10` или выше, рекомендуется использовать карты TF марки SanDisk и др.

    :::

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце отобразится следующий вывод, значит компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
        
    * На изображении ниже показан пример успешной компиляции
