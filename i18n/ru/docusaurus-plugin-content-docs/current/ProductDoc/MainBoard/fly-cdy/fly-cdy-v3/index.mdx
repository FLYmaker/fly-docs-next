# CDY V3_V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/cdy.png').default} size="80%" align="center" />

## Особенности продукта

    * 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Поддержка выбора напряжения питания, по умолчанию 5 В
    * Поддержка прошивки RRF

## Параметры продукта

    * 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Поддержка подключения по CAN-шине, более стабильная передача данных, меньшая задержка, более стабильное соединение
    * Прошивка: Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т.д.
    * Поддержка режимов работы драйверов: TMC: UART/SPI
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * При использовании прошивки repap можно использовать расширенный модуль ESP32 WiFi, поддерживает скорость 1.6 м/с
    * Уникальный дизайн платы MOS для вентилятора PWM, при повреждении можно легко заменить, снижая сложность обслуживания для клиентов
    * PCB с толщиной меди 2OZ, способен поддерживать больший ток
    * 3 интерфейса ADC, максимально поддерживает три термопары
    * 6 интерфейсов ввода/вывода, можно использовать для выравнивания и ограничения
    * 4 интерфейса нагревательных элементов, максимально поддерживает четыре нагревательных элемента
