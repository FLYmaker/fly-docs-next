## Справочная конфигурация

    ```cfg
    [adxl345]
    cs_pin: PB12
    spi_bus: spi2 # Жесткий SPI, выбирается один из двух вариантов: жесткий SPI или программный SPI. По умолчанию используется жесткий SPI
    # spi_software_mosi_pin: PB15 # Программный SPI, выбирается один из двух вариантов: жесткий SPI или программный SPI. По умолчанию используется жесткий SPI
    # spi_software_miso_pin: PB14
    # spi_software_sclk_pin: PB13

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # Обычно это центр горячей платы, пожалуйста, измените в соответствии с реальной ситуацией вашей машины
    ```
