# Шина CAN

Шина CAN - это последовательная коммуникационная протокол шина, используемая для приложений в реальном времени, которая может передавать сигналы с помощью кабеля с двойным витком и является одной из самых широко используемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобилях, заменяя дорогостоящие и громоздки электрические кабели. Прочность протокола позволяет его использовать в других автоматизированных и промышленных приложениях. Особенности протокола CAN включают целостную последовательную передачу данных, поддержку в реальном времени, скорость передачи до 1 Мбит/с, 11-битную адресацию и способность к проверке ошибок.
На 3D-принтерах шина CAN используется для уменьшения количества проводов, соединяющих печатную головку с материнской платой, вместо десятка проводов требуется только четыре, что значительно снижает количество проводов и сложность проводки.
В этой главе кратко описано использование шины CAN для мостования.

    * Обратите внимание, что линия питания инструментальной платы подключается не к порту CAN, а к линии питания 12V-24V
