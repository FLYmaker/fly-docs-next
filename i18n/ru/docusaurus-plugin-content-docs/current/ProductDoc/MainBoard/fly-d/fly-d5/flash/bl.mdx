# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * D5 подключается через USB
    * D5 с завода с 1 сентября 2024 года уже предварительно прошит прошивкой Katapult
    :::

## Вход в режим записи

    :::tip

    Слева находится кнопка `BOOT`, справа - кнопка `RESET`

    :::

    * Есть два способа войти в режим записи
    * Первый: при полном отключении питания главной платы, удерживайте кнопку BOOT, затем подключите один конец кабеля для передачи данных к главной плате, а другой - к вашему устройству
    * Второй: после подключения одного конца кабеля для передачи данных к главной плате, а другого - к вашему устройству, удерживайте кнопку BOOT, затем нажмите и отпустите кнопку RST, после чего отпустите кнопку BOOT
