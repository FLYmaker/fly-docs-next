# Внимание к материнской плате

## Индикатор питания
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/power.png').default} size="80%" align="center" />
    * При питании только через TYPE-C и без подключения других источников питания материнской платы должны светиться по крайней мере две лампы `3.3V` и `5V`
    * После подключения питания материнской платы должны светиться индикаторы `3.3V`, `5V`, `VCC`

### Решение проблем


    :::danger Внимание

    * При отсутствии каких-либо внешних устройств (драйверов, концевых выключателей, 12864 и т.д.)
    * При питании только через TYPE-C должны постоянно светиться по крайней мере две LED `3.3V` и `5V`

    :::

    * Если нет светящихся индикаторов, это означает короткое замыкание, необходимо связаться с службой поддержки
    * Если один из индикаторов `3.3V`, `5V` мигает, это означает короткое замыкание, необходимо связаться с службой поддержки

## Концевые выключатели

    * Обратите внимание на порядок подключения проводов, для трехпроводных концевых выключателей важен порядок 5V, GND, S, неправильное подключение может привести к повреждению материнской платы
    * Если подключать двухпроводные концевые выключатели, достаточно подключить S и GND
    * Для PL08 и других датчиков приближения необходимо подключить внешний диод к сигнальному порту для снижения напряжения до уровня ниже 3.3V
