## Вход в режим прошивки

    :::tip

    Слева находится кнопка `BOOT`, справа - кнопка `RESET`

    :::

    * Существует два способа войти в режим прошивки
    * Первый: При полностью отключенном питании платы, зажмите кнопку BOOT и подключите кабель с передачей данных к плате и вашему устройству
    * Второй: После подключения кабеля с передачей данных к плате и вашему устройству, зажмите кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST и в последнюю очередь отпустите кнопку BOOT
