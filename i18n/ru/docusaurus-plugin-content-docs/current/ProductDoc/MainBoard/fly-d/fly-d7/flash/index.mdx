# Инструкция по прошивке прошивки

:::danger Обратите внимание
* Прошивка основной платы требует прошивки Klipper через Katapult
* Если вы уверены, что на основной плате уже прошита прошивка Katapult, то последняя прошивка Katapult не требуется, это просто на случай, если прошивка Katapult не удалась
* Можно быстро дважды нажать кнопку сброса и посмотреть, загорится ли светодиод на плате, если да, то плата уже вошла в режим прошивки Katapult
:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только один тип прошивки

    :::
    
    * D7 может подключаться к верхнему компьютеру следующим образом
    * <Button variant="contained" disableElevation href="./usb">USB прошивка</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: если много USB устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB мостовая прошивка</Button>
        * Преимущества: можно подключить несколько устройств и не занимать USB порты
        * Недостатки: при совместном использовании с `KlipperScreen` может вызвать ошибки, такие как тайм-аут при возврате в исходное положение.
