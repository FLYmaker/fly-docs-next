# Введение

> Fly-D7 - это высокопроизводительная плата для 3D-принтера, выпущенная командой разработчиков Fly3D из Guangzhou Maile Technology Co., Ltd, поддерживающая до 7 шаговых двигателей.


    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.png').default} size="60%" align="center" />

## **1. Характеристики продукта**

    * Используется 32-битный микроконтроллер ARM Cortex-M0+ серии STM32F072RBT6 с частотой 48 МГц
    * Поддержка подключения по CAN-шине, более стабильная передача данных, меньшая задержка, более стабильное соединение
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * На плате имеются пины для функции DIAG, поддержка бесконечного позиционирования с помощью простого переключения перемычек

## 2. Параметры продукта

    * МК: STM32F072RBT6
    * Прошивка: Klipper/Marlin
    * Поддержка питания постоянного тока 12-24В, более стабильное питание, без необходимости в понижающих модулях
    * Режимы работы драйверов: SPI, UART, STEP/DIR
    * Поддержка драйверов: внешние TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и другие
    * Расширенные интерфейсы: BLTouch (Servos, Probe), UART X1
    * Вентиляторы: два управляемых двухпроводных вентилятора, два параллельных вентилятора, два постоянно работающих вентилятора
    * Датчики температуры: 3 канала NTC 100K
    * Разъемы для нагрева: 3 управляемых разъема с поддержкой до 10А
    * Поддержка типов машин: Cartesian, Delta, Kossel, Ultimaker, CoreXY
