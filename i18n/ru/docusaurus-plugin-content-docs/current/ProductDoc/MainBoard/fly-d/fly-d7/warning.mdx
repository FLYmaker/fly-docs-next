# Важные замечания по материнской плате

## Индикатор питания
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/power.png').default} size="80%" align="center" />
    * При питании только через TYPE-C и без подключения других источников питания материнской платы, должны гореть хотя бы два индикатора: `3.3V` и `5V`
    * После подключения питания материнской платы, индикаторы питания должны загореться: `3.3V`, `5V`, `VCC`

### Устранение неисправностей


    :::danger Важно

    * В отсутствие подключенных внешних устройств (драйверы, концевики, 12864 и т.д.)
    * Только при питании через TYPE-C, должны постоянно гореть `3.3V` и `5V` два светодиода

    :::

    * Если ни один индикатор не горит, это означает короткое замыкание, необходимо связаться с сервисным центром
    * Если один из индикаторов `3.3V` или `5V` мигает, это также указывает на короткое замыкание, необходимо связаться с сервисным центром

## Драйверы

    * Материнская плата поддерживает внешние драйверы     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">Руководство по подключению и настройке</Button>

## Концевики

    * Обратите внимание на порядок подключения, для трехпроводных концевиков важно правильно подключить 5V, GND, S, иначе возможен ущерб материнской плате
    * Для двухпроводных концевиков достаточно подключить S и GND
    * PL08 и другие датчики приближения требуют подключения диода на входе для снижения напряжения до 3.3V
