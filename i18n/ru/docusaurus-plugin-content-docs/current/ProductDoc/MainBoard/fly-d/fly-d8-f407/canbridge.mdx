# Бридж CAN

CAN-шина - это последовательная коммуникационная протокол шины, используемая для приложений в реальном времени, которая может передавать сигналы с использованием двойного витого провода и является одной из самых широко используемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобилях, заменяя дорогостоящие и громоздки электрические кабели. Робастность протокола позволяет его использовать в других автоматизированных и промышленных приложениях. Особенности протокола CAN включают целостную последовательную передачу данных, обеспечение поддержки в реальном времени, скорость передачи данных до 1 Мбит/с, одновременное наличие 11-битного адреса и способности к проверке ошибок.
На 3D-принтерах CAN-шина используется для уменьшения количества проводов, соединяющих печатную головку с материнской платой; раньше требовалось более десяти проводов, а с использованием CAN достаточно всего четырех, что значительно сокращает количество проводов и упрощает проводку.
В этой главе кратко описано использование бриджа CAN.

## Использование инструментальной платы для подключения к материнской плате через CAN

    * Обратите внимание, что линия питания инструментальной платы подключается не к порту CAN, а к линии питания 12V-24V
    
    * Обратите внимание, что необходимо подключить 120Ω переключатель
