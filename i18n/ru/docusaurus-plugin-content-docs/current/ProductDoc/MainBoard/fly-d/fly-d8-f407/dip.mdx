# Введение в платформенные периферийные устройства

    * Когда переключатель повернут вправо, открывается USB-порт рядом с портом Type-C
    * Когда переключатель повернут влево, открывается интерфейс RS232 рядом с портом Type-C

## Введение в USB-порт
    
    * Может подключать внешние камеры и другие устройства, такие как материнская плата
