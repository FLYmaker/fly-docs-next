## Вход в режим прошивки

    :::tip

    Слева находится кнопка `RST`, справа - кнопка `BOOT`

    :::

    * Есть два способа войти в режим прошивки
    * Первый: при полном отключении питания главной платы, удерживайте кнопку BOOT и подключите кабель с данными одним концом к главной плате, а другим - к вашему устройству
    * Второй: после подключения кабеля с данными одним концом к главной плате, а другим - к вашему устройству, удерживайте кнопку BOOT, затем нажмите и отпустите кнопку RST, после чего отпустите кнопку BOOT
