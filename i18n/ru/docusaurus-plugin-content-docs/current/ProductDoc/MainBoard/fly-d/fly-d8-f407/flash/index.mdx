# Инструкции по прошивке

:::danger Обратите внимание
* Для прошивки фирменного ПО основной платы необходимо использовать Katapult для прошивки Klipper
* Если вы уверены, что на плате уже установлена прошивка Katapult, то последняя прошивка Katapult не требуется, это просто на случай, если прошивка Katapult не удалась
* Можно быстро дважды нажать кнопку сброса и посмотреть, не начнет ли мигать светодиод на плате, если это произойдет, это означает, что плата уже перешла в режим прошивки Katapult
:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только один тип прошивки

    :::
    
    * D8 можно подключить к ПК следующими способами
    * <Button variant="contained" disableElevation href="./usb">Прошивка через USB</Button>
        * Плюсы: надежная связь, стабильность
        * Минусы: при наличии нескольких USB-устройств может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">Прошивка через USB-мост</Button>
        * Плюсы: можно подключить несколько устройств без занятия USB-порта
        * Минусы: возможны ошибки, связанные с таймаутами при использовании вместе с `KlipperScreen`.
