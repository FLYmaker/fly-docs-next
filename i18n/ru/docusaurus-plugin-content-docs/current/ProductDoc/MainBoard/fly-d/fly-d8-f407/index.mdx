# Описание

> Mellow Fly-D8 - это высокопроизводительная основная плата для 3D-принтеров, разработанная командой разработчиков основных плат Fly3D компании Guangzhou Melun Electronic Technology Co., Ltd., которая поддерживает до восьми шаговых двигателей.

## Особенности продукта

    * Интерфейсы ограничения хода и другие на плате защищены от изоляции питания 5V, если не подключить источник питания POWER, питание не будет выдаваться
    * Используется 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407
    * Поддерживается CAN шина и соединение RS232, передача данных более стабильна, задержка меньше, соединение более стабильно
    * На плате предусмотрены кнопки BOOT и RESET, пользователи могут войти в режим DFU через USB для обновления прошивки
    * Интерфейс питания имеет защиту от переполюсовки, чтобы избежать выгорания платы из-за неправильного подключения источника питания во время DIY
    * Предоставляются два интерфейса USB 2.0, что расширяет возможности подключения, один из USB интерфейсов может быть переключен на RS232 с помощью переключателя для соединения с различными устройствами

# Технические характеристики

    * Главный контрольный чип: 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Прошивка: Marlin 2.0/Klipper
    * Поддерживаемые режимы драйвера: TMC: UART/SPI
    * Поддержка восьми драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * Дисплей: серийный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Поддержка датчиков автоматического выравнивания стола: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и другие
    * Входное напряжение: поддерживается 12V/24V
    * Шесть интерфейсов ограничителей хода, что позволяет реализовать точное ограничение диапазона и контроль положения
    * Два интерфейса RGB, которые могут обеспечить разноцветные визуальные эффекты
    * Три интерфейса для нагревательных трубок, предоставляющих независимый контроль над экструдером и горячим столом
    * Четыре термисторных интерфейса, которые могут одновременно мониторить до четырех групп температур, обеспечивая точный контроль температуры
    * Интерфейсы USB: поддерживается расширение двух интерфейсов USB 2.0, один из USB может быть переключен на RS232 с помощью переключателя
    * Предоставляются два интерфейса USB 2.0, что расширяет возможности подключения, один из USB интерфейсов может быть переключен на RS232 с помощью переключателя для соединения с различными устройствами
