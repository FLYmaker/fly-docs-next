# Соединение RS232

Стандартный интерфейс RS232 (также известный как EIA RS-232) является одним из наиболее распространенных стандартов последовательной связи, широко используемых для передачи данных между различными устройствами. Его стабильность и надежность делают его предпочтительным способом связи для многих устройств. Протокол использует уровни сигналов, отличные от стандартных 5V, для передачи данных по проводам, что позволяет максимально уменьшить помехи сигнала. В системах 3D-принтеров RS232 по сравнению с CAN обладает более высокой скоростью передачи данных, лучшей стабильностью, более простой настройкой и лучшей совместимостью с верхним уровнем управления.
