# CAN-мост

CAN-шина - это последовательный коммуникационный протокол для реального времени, который может использовать витую пару для передачи сигналов и является одним из самых распространенных полевых шин в мире. Протокол CAN используется для коммуникации между различными компонентами автомобиля, заменяя дорогие и громоздкие жгуты проводов. Прочность этого протокола позволяет использовать его в других автоматизированных и промышленных приложениях. Характеристики протокола CAN включают целостную последовательную передачу данных, поддержку реального времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможность обнаружения ошибок.
В 3D-принтерах CAN-шина используется для уменьшения количества проводов, соединяющих печатную головку с основной платой. Вместо необходимости десятков проводов, с использованием CAN требуется всего четыре провода, что значительно уменьшает количество проводов и облегчает их прокладку.
В этой главе дается краткий обзор использования CAN-моста.

## Использование CAN для подключения инструментальной платы к основной плате

    * Обратите внимание, что питание инструментальной платы подключается не к CAN-порту, а к линии питания 12V-24v

    * Обратите внимание, что нужно установить перемычку на 120Ω
