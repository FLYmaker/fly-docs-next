## Вход в режим прошивки

    :::tip

    Слева кнопка `RST`, справа кнопка `BOOT`

    :::

    * Существует два способа входа в режим прошивки
    * Первый способ: при полностью отключенном питании платы, нажмите и удерживайте кнопку BOOT, затем подключите кабель передачи данных к плате и вашему устройству
    * Второй способ: после подключения кабеля передачи данных к плате и вашему устройству, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST и затем отпустите кнопку BOOT
