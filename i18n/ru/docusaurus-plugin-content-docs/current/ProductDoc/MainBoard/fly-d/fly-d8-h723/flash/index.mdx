# Инструкция по прошивке прошивки

:::danger Важно
* Для прошивки прошивки на материнскую плату необходимо использовать Katapult для прошивки Klipper
* Если вы уверены, что на материнской плате уже прошита Katapult, нет необходимости прошивать прошивку Katapult в конце, это просто на случай, если прошивка Katapult на материнской плате не удалась
* Можно быстро нажать кнопку сброса дважды и посмотреть, загорится ли светодиод на плате, если да, то это означает, что плата уже перешла в режим прошивки Katapult
:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только один из вариантов прошивки

    :::
    
    * D8 может быть подключен к компьютеру следующими способами
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: если несколько USB-устройств могут влиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">Прошивка с мостом USB</Button>
        * Преимущества: можно подключить несколько устройств и не занимать USB-порты
        * Недостатки: возможны ошибки, такие как таймаут при возврате в исходное положение, если используется `KlipperScreen`.
