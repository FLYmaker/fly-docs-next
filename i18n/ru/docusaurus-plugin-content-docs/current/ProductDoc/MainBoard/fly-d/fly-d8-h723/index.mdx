# Введение

> Mellow Fly-D8 - это высокопроизводительная плата для 3D-принтера, разработанная командой Fly3D компании Guangzhou Mellow Electronic Technology Co., Ltd., которая поддерживает до 8 шаговых двигателей.

## Характеристики продукта

    * Интерфейсы платы, такие как ограничители, защищены от короткого замыкания, если не подключен источник питания POWER, питание не будет подаваться
    * Используется 32-битный микроконтроллер ARM Cortex-M7 серии STM32H723 с частотой 550 МГц
    * Поддерживает подключение через шину CAN и RS232, обеспечивая более стабильную передачу данных, меньшую задержку и стабильное соединение
    * На плате предусмотрены кнопки BOOT и RESET, позволяющие пользователю через USB войти в режим DFU для обновления прошивки
    * Интерфейс питания имеет защиту от обратной полярности, предотвращающую повреждение платы при неправильном подключении источника питания во время DIY
    * Предоставлены два интерфейса USB2.0, расширяющие возможности подключения, один из которых может быть переключен на RS232 через переключатель для соответствия требованиям различных устройств

# Параметры продукта

    * Микроконтроллер: 32-битный микроконтроллер ARM Cortex-M7 серии STM32H723 с частотой 550 МГц
    * Прошивка: Marlin 2.0/Klipper
    * Поддерживаемые режимы драйверов: TMC: UART/SPI
    * Восемь поддерживаемых драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и др.
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Поддержка автоматических сенсоров выравнивания платформы: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и др.
    * Входное напряжение: поддержка 12В/24В
    * 6 интерфейсов для ограничителей, позволяющих реализовать точное ограничение диапазона и контроль положения
    * 2 интерфейса RGB для обеспечения разноцветных визуальных эффектов
    * 3 интерфейса для нагревательных элементов, обеспечивающие независимый контроль экструдера и нагревательной платформы
    * 4 интерфейса для термисторов, позволяющие одновременно отслеживать до четырех температурных зон для точного контроля температуры
    * Интерфейс USB: поддержка двух интерфейсов USB2.0, один из которых может быть переключен на RS232 через переключатель
    * Предоставлены два интерфейса USB2.0, расширяющие возможности подключения, один из которых может быть переключен на RS232 через переключатель для соответствия требованиям различных устройств
