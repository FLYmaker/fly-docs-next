# CAN-мост

CAN-шина - это последовательный коммуникационный протокол для реального времени, который может использовать витую пару для передачи сигналов и является одним из наиболее широко применяемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобиле, заменяя дорогостоящие и громоздкие кабели распределения. Благодаря своей надежности, протокол нашел применение и в других областях автоматизации и промышленности. Характеристики протокола CAN включают целостную последовательную передачу данных, поддержку реального времени, скорость передачи до 1 Мб/с, 11-битную адресацию и возможность обнаружения ошибок.
3D-принтеры используют CAN-шину для уменьшения количества проводов, соединяющих печатающую головку с основной платой. Вместо десятков проводов, с CAN достаточно использовать всего четыре, что значительно упрощает прокладку проводов.
В этой главе дается краткий обзор использования CAN-моста.
