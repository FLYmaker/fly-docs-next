## Вход в режим записи

    :::tip

    Слева кнопка `BOOT`, справа кнопка `RESET`

    :::

    * Есть два способа войти в режим записи:
    * Первый: при полностью отключенном питании платы, удерживая кнопку BOOT, подключите кабель с передачей данных к плате и устройству
    * Второй: после подключения кабеля с передачей данных к плате и устройству, удерживая кнопку BOOT, нажмите кнопку RST, затем отпустите RST и в последнюю очередь BOOT
