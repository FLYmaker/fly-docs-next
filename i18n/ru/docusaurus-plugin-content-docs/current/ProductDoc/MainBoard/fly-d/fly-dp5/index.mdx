# Введение

> Mellow Fly-DP5 — это высокопроизводительная плата для 3D-принтера, разработанная командой Fly3D компании Guangzhou Mellow Electronic Technology Co., Ltd., поддерживающая до 5 двигателей шагового типа.


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.png').default} size="60%" align="center" />

## **1. Характеристики продукта**

* Оснащена 32-битным процессором ARM Cortex-M0+ серии STM32F072RBT6 с частотой 48МГц
* Поддержка подключения по CAN-шине, обеспечивающая более стабильную передачу данных, меньшую задержку и стабильное соединение
* Поддержка драйверов: DP2209 и DP560
* На плате имеются пины для функции DIAG, позволяющие использовать бесконечное позиционирование с помощью простого переключения перемычек

## 2. Параметры продукта

* МК: STM32F072RBT6
* Прошивка: Klipper/Marlin
* Поддержка питания постоянного тока 12-24В, обеспечивающего более стабильное питание без необходимости использования понижающего модуля
* Режимы работы драйверов: SPI, UART, STEP/DIR
* Поддержка драйверов: внешние TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и др.
* Расширяемые интерфейсы: BLTouch (Servos, Probe), UART X1, 12864 X1
* Вентиляторы: двухпроводные управляемые вентиляторы X2, параллельные вентиляторы X2, постоянно работающие вентиляторы X2
* Датчики температуры: 2 канала NTC 100K
* Интерфейсы нагрева: 2 канала с поддержкой до 10А
* Поддерживаемые типы конструкций: Cartesian, Delta, Kossel, Ultimaker, CoreXY
