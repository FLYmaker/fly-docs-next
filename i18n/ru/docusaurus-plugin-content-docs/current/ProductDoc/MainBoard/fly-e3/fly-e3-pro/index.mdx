## Параметры продукта

    * 32-битный ARM Cortex-M4 серии 168 МГц, микросхема STM32F407ZGT6
    * Прошивка: Reprap/Klipper
    * На борту пять драйверов TMC2209
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 высоковольтный вход, поддерживает автоматический датчик выравнивания кровати: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
    * Используя прошивку repap, можно использовать модуль Wi-Fi ESP32, поддерживает 1.6 м/с
    * Уникальный дизайн платы PWM Fan MOS, при повреждении можно напрямую заменить, снижает сложность ремонта для клиентов, четыре 2-пиновых интерфейса вентиляторов
    * Печатная плата с толщиной меди 2OZ, способна поддерживать больший ток
    * 2 интерфейса АЦП, максимально поддерживает два термистора
    * 4 интерфейса ввода/вывода, могут использоваться для выравнивания и ограничения
    * 3 интерфейса нагревательных элементов, максимально поддерживает три нагревательных элемента
