# Мост CAN

Шина CAN (Controller Area Network) представляет собой последовательный протокол связи для реального времени, который может использовать витую пару для передачи сигналов и является одним из наиболее широко используемых полевых шин в мире. Протокол CAN используется для коммуникации между различными компонентами автомобиля, заменяя дорогие и громоздкие кабельные жгуты. Его надёжность позволяет использовать его в других автоматизированных и промышленных приложениях. Характеристики протокола CAN включают целостность последовательной передачи данных, поддержку в реальном времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможность обнаружения ошибок.
В 3D-принтерах шина CAN используется для уменьшения количества проводов, соединяющих печатающую головку с материнской платой. Вместо десятка проводов после внедрения CAN требуется всего четыре, что значительно уменьшает количество проводов и упрощает прокладку кабелей.
В этой главе будет дан краткий обзор использования моста CAN.
