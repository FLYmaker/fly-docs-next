# BL прошивка

    :::tip

    * Обратите внимание, что заводская прошивка `bootloader` уже установлена, поэтому повторная установка не требуется.

    :::

## Вход в режим прошивки

    :::tip

    Справа находится кнопка `BOOT`, слева - кнопка `RESET`.

    :::

    * Существует два способа входа в режим прошивки:
    * Первый способ: при полностью отключенном питании платы, зажмите кнопку BOOT и подключите кабель для передачи данных одним концом к плате, а другим - к вашему устройству.
    * Второй способ: после подключения кабеля для передачи данных одним концом к плате и другим - к вашему устройству, зажмите кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, а затем отпустите кнопку BOOT.
