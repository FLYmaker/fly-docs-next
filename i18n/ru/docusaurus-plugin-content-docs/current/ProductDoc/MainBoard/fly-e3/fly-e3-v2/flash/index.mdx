    :::tip

    * E3 V2 по умолчанию использует TF-карту для прошивки
    * Пожалуйста, приготовьте TF-карту емкостью не более `32G`
    * Использование некачественных карт памяти может привести к сбоям прошивки, рекомендуется использовать рекомендованные карты памяти для обновления прошивки
    * Скорость передачи данных TF-карты должна быть `class10` или выше, рекомендуется использовать карты SanDisk или других известных брендов

    :::

# Инструкция по прошивке

## Как определить, какая прошивка вам нужна

    :::tip

    * Обратите внимание, что вам нужно установить только одну из прошивок

    :::

    * E3 V2 можно подключить к верхнему компьютеру следующими способами:
    * <Button variant="contained" disableElevation href="./usb">Прошивка USB</Button>
        * Плюсы: надежная связь, стабильность
        * Минусы: если подключено несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">Прошивка USB-моста</Button>
        * Плюсы: можно подключить несколько устройств и не занимать USB-порт
        * Минусы: возможны ошибки, связанные с тайм-аутом при использовании вместе с `KlipperScreen`.
