## Параметры продукта

    * 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Поддержка подключения по шине CAN, более стабильная передача данных, меньшая задержка, более стабильное соединение
    * Прошивка: Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т.д.
    * Поддержка режимов драйверов: TMC: UART/SPI
    * Дисплей: серийный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 вход для высокого напряжения, поддержка автоматических датчиков выравнивания стола: BLTouch, PL08N и другие датчики близости, Klicky, Voron Tap и т.д.
    * При использовании прошивки repap, можно использовать расширенный модуль ESP32 WiFi, поддерживающий скорость до 1.6 м/с
    * Уникальный дизайн платы PWM Fan MOS, при повреждении можно легко заменить, уменьшая сложность ремонта для клиентов, четыре 2-pin интерфейса для вентиляторов
    * Печатная плата с толщиной меди 2OZ, способная поддерживать больший ток
    * 2 интерфейса ADC, поддерживают до двух термисторов
    * 4 интерфейса IO, могут использоваться для выравнивания и ограничения
    * 3 интерфейса для нагревательных стержней, поддерживают до трёх нагревательных стержней
