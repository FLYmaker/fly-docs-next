# Компиляция и прошивка микропрограммы

    :::tip

    * 407ZG по умолчанию использует карту TF для прошивки микропрограммы
    * Подготовьте карту TF емкостью не более `32G`
    * Карты памяти неизвестных брендов могут привести к сбоям при прошивке, по возможности используйте рекомендуемые карты памяти для обновления микропрограммы
    * Скорость передачи данных карты TF должна быть `class10` или выше, рекомендуется использовать карты SanDisk и других известных брендов

    :::
