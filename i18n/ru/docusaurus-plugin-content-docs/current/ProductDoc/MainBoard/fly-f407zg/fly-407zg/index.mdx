## Параметры продукта

    * 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Поддержка подключения по CAN-шине, передача данных более стабильная, меньшая задержка, более стабильное соединение
    * Прошивка: Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * Поддержка режимов драйвера: TMC: UART/SPI
    * Дисплей: серийный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Используя прошивку repap, можно использовать расширенный модуль WiFi ESP32, поддержка 1.6 м/с
    * Уникальный дизайн платы PWM Fan MOS, поврежденную можно легко заменить, что упрощает обслуживание клиентами
    * PCB с толщиной меди 2OZ, способная поддерживать больший ток
    * 6 интерфейсов АЦП, поддерживают до шести термисторов
    * 7 интерфейсов IO, могут использоваться для выравнивания и ограничения
    * Семь интерфейсов для нагревательных стержней, поддерживают до семи нагревательных стержней
