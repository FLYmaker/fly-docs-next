# ПРОШИВКА BL

    :::tip

    * Обратите внимание, что заводская прошивка `bootloader` уже установлена, и нет необходимости перепрошивать

    :::

## Вход в режим программирования


    * Есть два способа войти в режим программирования
    * Первый: при полностью отключенной плате, замкните BOOT и BOOT1, затем подключите кабель с передачей данных одной стороной к плате, а другой - к вашему устройству
    * Второй: после подключения кабеля с передачей данных одной стороной к плате, а другой - к вашему устройству, замкните BOOT и BOOT1, нажмите кнопку RST, затем отпустите кнопку RST и разомкните перемычку BOOT

            4. Прошивка прошивки

            :::tip

            Требуется прошить только одну прошивку

            :::


            * Ниже приведена команда прошивки bootloader
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/Gemini/GeminiV3.bin
                ```

            5. Если появится сообщение, указанное стрелкой на изображении ниже, ``File downloaded successfully``, то прошивка прошла успешно

            :::warning

            Пожалуйста, игнорируйте **ошибки после строки, указанной красной стрелкой**!!!
