# Программирование прошивки USB-to-CAN моста

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите `make -j4`, чтобы начать компиляцию, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводится следующее, значит компиляция завершилась успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Программирование прошивки

:::tip
Выполните следующую команду для программирования прошивки
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
