# Введение в переключатели

:::tip
FLY Gemini подключается к верхнему компьютеру через переключатели. Если положение переключателей неправильное, то устройство не будет работать корректно
:::

:::danger Важно
Обратите внимание, что переключатели должны быть установлены в указанные позиции для нормального подключения к верхнему компьютеру
:::

## Подключение к встроенному верхнему компьютеру

    * Переключатели `1`,`2` переместите вправо, чтобы соединить нижний компьютер с верхним через встроенный USB
    * Переключатели `3`,`4` переместите влево, иначе это повлияет на подключение нижнего компьютера с верхним
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.png').default} size="50%" align="center" />
    * После переключения переключателей в соответствии с этим методом, нижний USB порт будет занят нижним компьютером, не подключайте другие устройства
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.png').default} size="50%" align="center" />


## Подключение к внешнему верхнему компьютеру

    * Переключатели `3`,`4` переместите вправо, чтобы соединить нижний компьютер с верхним через встроенный USB
    * Переключатели `1`,`2` переместите влево, иначе это повлияет на подключение нижнего компьютера с верхним
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.png').default} size="50%" align="center" />
