# Инструкции по прошивке прошивки

## Как определить, какая вам нужна прошивка

    :::tip

    * Обратите внимание, что вам нужно прошить только один вид прошивки

    :::
    
    * Gemini V3 может подключаться к верхнему компьютеру следующими способами
    * <Button variant="contained" disableElevation href="./usb">USB прошивка</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: если несколько USB устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">Прошивка с USB мостом</Button>
        * Преимущества: можно подключить несколько устройств и не занимать USB порты
        * Недостатки: возможны ошибки, такие как тайм-аут возвращения на место при использовании вместе с `KlipperScreen`.
