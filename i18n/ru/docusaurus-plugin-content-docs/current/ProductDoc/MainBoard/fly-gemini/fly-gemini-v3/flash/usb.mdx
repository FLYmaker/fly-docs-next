# Компиляция прошивки через USB

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце вы увидите следующий вывод, если компиляция прошла успешно:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

:::tip
Выполните следующую команду для записи прошивки:
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
