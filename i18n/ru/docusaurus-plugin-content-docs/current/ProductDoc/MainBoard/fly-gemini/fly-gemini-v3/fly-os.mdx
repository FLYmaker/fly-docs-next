## Подготовка к записи системы

    * Картридер
    * TF карта, ёмкость должна быть не менее 16 ГБ и не более 128 ГБ, TF карта должна иметь скорость передачи данных class10 или выше
    * Скачать соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">системный образ</Button>
    * Скачать необходимое <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">ПО для записи</Button>

## Запись системы

:::info
Запись на TF карту и запись через TF карту на M2WE требуется только один из вариантов записи
:::

:::info
* Если на устройстве установлены M2WE и карта памяти, система по умолчанию запускается с карты памяти
* Рекомендуется использовать TF карту для записи системы на M2WE, так как запись через USB на M2W может не удаться из-за проблем совместимости драйверов, рекомендуется использовать [запись на M2WE через карту памяти](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Пожалуйста, обратите внимание на соответствие скачанного системного образа, иначе система не запустится
:::
