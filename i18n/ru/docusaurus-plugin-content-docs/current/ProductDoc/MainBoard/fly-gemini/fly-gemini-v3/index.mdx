# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.webp').default} size="80%" align="center" />

## Характеристики продукта
    * Встроенный верхний компьютер, может заменить Raspberry Pi в качестве верхнего компьютера, он имеет встроенную память DDR3 объемом 1GB.
    * Улучшенная защита источника питания 5V, уменьшает вероятность повреждения чипа.
    * Удалена настройка карты памяти для чипа MCU, что упрощает прошивку прошивки.
    * Система предустановлена с klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, плагином компенсации резонанса numpy и FLY-Tools, не требуется сложных операций, таких как замена источника, новички легко освоятся (необходимо приобрести карту TF или M2WE)


## Параметры продукта

    * CPU: Высокопроизводительный чип Allwinner H5, 4 ядра 64-битный Cortex-A53
    * GPU: Высокопроизводительный 6-ядерный Mali 450, скорость заполнения пикселей больше 2.7gpixel/s
    * RAM: 1GB DDR3 (совместно с GPU)
    * ROM: Поддержка до 128GB SDCARD
    * Периферия: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1
    * Интерфейсы: Встроенный интерфейс M.2, можно расширить WIFI, eMMC и т.д. (проприетарный протокол, не подключайте устройства M.2, не предназначенные для FLY π)
    * Поддержка питания постоянным током 12-24V, более стабильное питание, избавляет от необходимости использования модуля понижающего напряжения
    * Резервный интерфейс для управляемого вентилятора 5V, может использоваться для охлаждения верхнего компьютера
    * Резервный интерфейс для расширения CAN
    * Система предустановлена с klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, плагином компенсации резонанса numpy и FLY-Tools, не требуется сложных операций, таких как замена источника, новички легко освоятся (необходимо приобрести карту TF или M2WE)
    * Драйверы: Высоковольтные драйверы *4
    * MCU: STM32F405
