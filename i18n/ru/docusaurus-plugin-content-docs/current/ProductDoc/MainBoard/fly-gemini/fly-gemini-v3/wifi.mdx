# Подключение к Wi-Fi


## Установка антенны на Gemini

    * Место, указанное красной стрелкой, является интерфейсом антенны Wi-Fi
    * Разъем для антенны Wi-Fi - `ipex1`
    * Установите антенну на Gemini, иначе из-за слабого сигнала может быть невозможно подключиться к сети
