# CAN-мост

CAN-шина представляет собой последовательный коммуникационный протокол для реального времени, который может использовать витую пару для передачи сигналов и является одним из наиболее широко используемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобилях, заменяя дорогостоящие и громоздкие распределительные жгуты проводов. Его надёжность расширяет его применение на другие автоматизированные и промышленные задачи. Характеристики протокола CAN включают целостность последовательной передачи данных, поддержку реального времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможность обнаружения ошибок.
3D-принтеры используют CAN-шину для уменьшения количества проводов, подключаемых к печатной головке от основной платы. Вместо десятков проводов с использованием CAN требуется только четыре, что значительно уменьшает количество проводов и упрощает монтаж.
В этой главе даётся краткий обзор использования CAN-моста.

    * Обратите внимание, что питание инструментальной платы подключается не к CAN-порту, а к линии питания 12V-24V
