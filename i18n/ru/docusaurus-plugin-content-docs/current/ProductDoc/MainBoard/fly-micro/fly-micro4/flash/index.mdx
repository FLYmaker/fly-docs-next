# Инструкция по прошивке микропрограммы

:::danger Обратите внимание
* Для прошивки микропрограммы на материнскую плату необходимо использовать Katapult для прошивки Klipper
* Если вы уверены, что на материнской плате уже установлен Katapult, то нет необходимости прошивать его в последнюю очередь, это нужно только на случай, если прошивка Katapult не удалась
* Можно быстро дважды нажать кнопку сброса и посмотреть, загорится ли светодиод на плате, если да, то это означает, что плата перешла в режим Katapult
:::

## Как определить, какой тип микропрограммы вам нужен

    :::tip

    * Обратите внимание, вам нужно прошить только один из типов микропрограммы

    :::
    
    * E3 V2 может подключаться к верхнему компьютеру следующими способами:
    * <Button variant="contained" disableElevation href="./usb">USB-микропрограмма</Button>
        * Преимущества: надежная и стабильная связь
        * Недостатки: если подключено несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мостовая микропрограмма</Button>
        * Преимущества: можно подключить несколько устройств, не занимая USB-порты
        * Недостатки: возможны ошибки, такие как таймаут при возврате в исходное положение, при использовании вместе с `KlipperScreen`.
