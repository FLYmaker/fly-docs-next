# Компиляция прошивки USB

    :::tip

    * Micro4 поставляется с предустановленной прошивкой Katapult для USB-соединения, повторная запись не требуется

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/usb.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце вы увидите следующий вывод, то компиляция завершена успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
