## Характеристики продукта

    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Поддержка моста CAN
    * На борту 4 шт. 2209
    * На борту функциональные выводы DIAG, поддержка бесконечного возврата в исходное положение только через простое подключение/отключение перемычки



## Параметры продукта

    * MCU：Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Прошивка: Klipper
    * Поддержка питания постоянным током 12-24В, более стабильное питание, исключает необходимость в модуле понижения напряжения
    * Поддерживаемые режимы работы драйвера: UART, STEP/DIR
    * Расширяемые интерфейсы: BLTouch (Servos, Probe), концевой выключатель X1
    * Вентиляторы: два управляемых вентилятора с двумя проводами X2, постоянно включенный вентилятор X1
    * Датчики температуры: 2 канала NTC 100K
    * Интерфейсы нагрева: 2 канала управляемых интерфейсов, поддерживающих 10A
