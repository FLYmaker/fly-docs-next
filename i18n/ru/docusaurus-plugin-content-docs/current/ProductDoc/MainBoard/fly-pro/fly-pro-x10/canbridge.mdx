# Мост CAN

CAN-шина - это последовательный протокол связи, используемый для реального времени, который может использовать витую пару для передачи сигналов и является одним из наиболее широко применяемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобилях, заменяя дорогие и громоздкие распределительные жгуты проводов. Его надёжность позволила расширить его применение на другие области автоматизации и промышленные применения. Характеристики протокола CAN включают последовательную коммуникацию с полной целостностью данных, поддержку реального времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможность обнаружения ошибок.
На 3D-принтерах CAN-шина используется для уменьшения количества проводов между печатающей головкой и материнской платой. Если раньше требовалось несколько проводов, то с использованием CAN достаточно четырёх, что значительно уменьшает количество проводов и упрощает прокладку кабелей.
В данной главе дается краткий обзор использования моста CAN.

## Использование инструментальной платы для подключения к материнской плате через CAN

    * Обратите внимание, что питание инструментальной платы подключается не к порту CAN, а к линии питания 12V-24V
