# BL прошивка

    :::tip

    * Обратите внимание, что заводская прошивка `bootloader` уже установлена, перепрошивка не требуется

    :::

## Вход в режим прошивки


    * Существует два способа входа в режим прошивки
    * Первый: при полном отключении питания платы, замкнуть BOOT, затем подключить кабель с передачей данных к плате и устройству
    * Второй: после подключения кабеля с передачей данных к плате и устройству, замкнуть BOOT, нажать кнопку RST, затем отпустить RST и разомкнуть BOOT

            :::tip

            Требуется прошить только одну прошивку

            * Ниже приведена команда для прошивки katapult
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * Ниже приведена команда для прошивки bootloader
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. Если вы видите сообщение, указанное стрелкой на рисунке ниже ``File downloaded successfully``, прошивка завершена успешно
