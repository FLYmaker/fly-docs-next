# Инструкция по прошивке прошивки

    :::warning Внимание

    * В PRO X10 предустановлен Katapult
    * После прошивки BL повторная прошивка BL не требуется
    * В PRO X10 есть два типа BL: прошивка Katapult и прошивка BL для TF-карты
    * При необходимости можно выбрать и перепрошить <Button variant="contained" disableElevation href="./bl">BL прошивку</Button>

    :::
    
    :::danger Важно
    * Прошивка прошивки основной платы требует прошивки Klipper через Katapult
    * Если основная плата уже прошита Katapult, последняя прошивка Katapult не требуется, это только на случай, если прошивка Katapult не удалась
    * Можно быстро нажать на сброс дважды и посмотреть, загорится ли на плате один светодиод, это означает, что плата вошла в режим Katapult
    :::

## Как определить, какая прошивка вам нужна

    :::tip

    * Обратите внимание, вам нужно прошить только один тип прошивки

    :::
    
    * Pro X10 может подключаться к верхнему уровню следующим образом
    * <Button variant="contained" disableElevation href="./usb">USB прошивка</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: если несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мостовая прошивка</Button>
        * Преимущества: можно подключить несколько устройств и не занимать USB-порт
        * Недостатки: может вызвать ошибки, такие как тайм-ауты при использовании вместе с `KlipperScreen`.
