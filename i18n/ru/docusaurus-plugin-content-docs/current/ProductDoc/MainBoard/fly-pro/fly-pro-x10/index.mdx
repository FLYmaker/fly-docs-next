## Характеристики продукта

    * Используется 32-битный процессор ARM Cortex-M7 с частотой 550 МГц STM32H723ZGT6
    * Материнская плата и основная плата соединены через BTB, что позволяет использовать основные платы Orange Pi CM4 и Raspberry Pi CM4
    * На материнской плате предусмотрена кнопка BOOT, с помощью которой пользователь может обновлять загрузчик через DFU
    * Десять отключаемых портов для драйверов TP, без необходимости использования перемычек для переключения режимов драйверов
    * На плате есть выводы для функции DIAG, которые позволяют использовать бесконечное позиционирование, просто установив перемычку
    * Питание драйверов поддерживает выбор между HVIN и VIN
    * Возможность обновления прошивки MCU через карту MicroSD или через команду Klipper make flash с обновлением через DFU
    * Шесть двухпроводных управляемых вентиляторов, поддерживающих съемные MOS, два четырехпроводных вентилятора, один VCC вентилятор
    * Вентиляторы с цифровым управлением могут работать на напряжении 24V, 12V или 5V, что позволяет избежать необходимости подключения внешнего модуля преобразования напряжения и снижает вероятность повреждения материнской платы
    * Использование заменяемого предохранителя для удобства замены;

## Параметры продукта

    * MCU: STM32H723ZGT6
    * Питание драйверов: VIN(12V-24V), HVIN(12V-48V)
    * Питание материнской платы: VIN(12V-24V)
    * Питание термостола: BED IN(12V-24V)
    * Нагревательные интерфейсы: термостол (BED OUT), нагревательный стержень (HETA0, HETA1, HETA2, HETA3)
    * Максимальный выходной ток порта термостола: 15A
    * Максимальный выходной ток порта нагревательного стержня: 6A
    * Максимальный выходной ток порта вентилятора: 1A
    * Поддержка режимов работы драйверов: SPI, UART, STEP/DIR
    * Расширенные интерфейсы: BLTouch (Servos, Probe), UART X1, высоковольтный концевой выключатель X1
