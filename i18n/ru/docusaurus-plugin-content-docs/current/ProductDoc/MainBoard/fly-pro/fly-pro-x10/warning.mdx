# Внимание к материнской плате

## Индикатор питания
    * При использовании питания через type-c без подключения других источников питания материнской платы должны гореть `SYS_5V` и SYS_3.3V
    * После подключения питания материнской платы `HVIN`, `VIN`, `BOARO`, `POWER` индикатор питания материнской платы должен гореть постоянно

## Драйверы

    * TP5160 и TP2209 отличаются только названием, и для выбора режима драйвера не требуется перемычка
    * В порт драйвера можно установить только драйвер TP, другие устройства устанавливать нельзя

## Порт ограничения
    * ProX10 поддерживает выбор между 5V и 12V питанием, обратите внимание, если используете датчик приближения, необходимо установить диод для понижения напряжения

## Верхнее программное обеспечение
    * ProX10 поддерживает использование внешнего верхнего ПО, выберите подключение Type-c с помощью переключателя, чтобы использовать его

## Запись прошивки

    * Материнская плата поддерживает запись прошивки с помощью TF-карты, но по умолчанию установлен `Katapult`
    * Если необходимо использовать TF-карту для записи прошивки, необходимо перезаписать BL
