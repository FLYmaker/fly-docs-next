# FLY-Rpfmex

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-rpfmex/img/fly_rpfmex.webp').default} size="80%" align="center" />

## Особенности продукта

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Встроены 4 штуки 2209


## Параметры продукта

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Прошивка: Klipper
    * Поддержка питания постоянным током 12-24В, питание более стабильное, не требуется понижающий модуль
    * Режим работы драйвера: UART, STEP/DIR
    * Расширение интерфейсов: управляемые вентиляторы*2, RGB*1, концевые выключатели*3
