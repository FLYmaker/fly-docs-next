# CAN мост

CAN-шина - это последовательный коммуникационный протокол шины, используемый для реального времени, который может использовать витую пару для передачи сигналов и является одним из самых широко используемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобиле, заменяя дорогие и громоздкие жгуты проводов. Его надежность позволяет использовать его в других автоматизированных и промышленных приложениях. Характеристики протокола CAN включают целостную последовательную передачу данных, поддержку реального времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможности обнаружения ошибок.
3D-принтеры используют CAN-шину для уменьшения количества проводов, соединяющих печатную головку с основной платой. Раньше требовалось более десяти проводов, но с использованием CAN это количество сократилось до четырех, значительно упростив прокладку проводов.
В этой главе будет дан краткий обзор использования моста CAN.

## Использование инструментальной платы для подключения к основной плате через CAN

    * Обратите внимание, что питание инструментальной платы не подключается к порту CAN, а к линии питания 12V-24v.
