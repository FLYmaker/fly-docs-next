# BL прошивка

    :::tip

    * Обратите внимание, что на заводе уже предустановлен `bootloader`, нет необходимости его перепрошивать

    :::

## Вход в режим прошивки

    :::tip

    Справа находится `BOOT перемычка`, слева - `RESET кнопка`

    :::

    * Существует два способа входа в режим прошивки
    * Первый способ: при полном отключении питания материнской платы, удерживайте кнопку BOOT, затем подключите кабель с поддержкой передачи данных к материнской плате и вашему устройству
    * Второй способ: после подключения кабеля с поддержкой передачи данных к материнской плате и вашему устройству, удерживайте кнопку BOOT, затем нажмите кнопку RST, после чего отпустите RST и затем BOOT
