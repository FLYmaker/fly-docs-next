    :::tip

    * Super5 Pro по умолчанию использует карту TF для прошивки микропрограммы
    * Пожалуйста, подготовьте карту TF емкостью не более `32G`
    * Карты памяти неизвестных брендов могут привести к сбоям при прошивке, по возможности используйте рекомендуемые ниже карты памяти для обновления микропрограммы
    * Скорость передачи данных карты TF должна быть `class10` или выше, рекомендуется использовать карты SanDisk и других известных брендов

    :::

# Инструкции по прошивке микропрограммы

    :::tip

    * Super5 Pro при выходе с завода уже предварительно прошит загрузчиком TF-карты, повторная прошивка не требуется

    :::


## Как определить, какая микропрограмма вам нужна

    :::tip

    * Обратите внимание, вам нужно прошить только один из видов микропрограммы

    :::
    
    * Super5 Pro может подключаться к верхнему уровню следующими способами
    * <Button variant="contained" disableElevation href="./usb">USB микропрограмма</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: при наличии нескольких USB устройств может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мостовая микропрограмма</Button>
        * Преимущества: можно подключить несколько устройств и не занимает USB-порт
        * Недостатки: возможны ошибки, такие как таймаут при возврате в исходное положение, при совместном использовании с `KlipperScreen`.
