# Параметры продукта

    * Использует 32-битный процессор ARM Cortex-M7 с частотой 550 MHz STM32H723ZGT6
    * Прошивка: Marlin 2.0/Reprap/Klipper
    * Поддержка драйверов: A4988、LV8729、DRV8225、TMC2208/2209/5160 и др.
    * Поддержка режимов драйверов: TMC: UART/SPI
    * 5 высоковольтных позиций для драйверов, поддерживает 5 независимых двигателей
    * Дисплей: серийный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 высоковольтный вход, поддерживает автоматические датчики выравнивания: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
    * Выбор источника питания для концевых выключателей: 5V/12V/24V
    * При использовании прошивки Reprap можно использовать встроенный модуль WiFi ESP32, поддержка до 1.6м/с
    * Уникальный дизайн платы PWM Fan MOS, при повреждении можно легко заменить, уменьшая сложность ремонта для клиента, два разъема для вентиляторов на 2pin могут поддерживать вентиляторы на 4pin PWM
    * PCB с толщиной меди 2OZ, способен выдерживать больший ток
    * 3 интерфейса ADC, поддержка до трех термисторов
    * 7 интерфейсов IO, могут использоваться для выравнивания и концевых выключателей
    * 3 интерфейса для нагревательных элементов, поддержка до трех нагревательных элементов
