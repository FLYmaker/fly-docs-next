# CAN-мост

CAN-шина представляет собой серийный протокол связи для реального времени, который может использовать витую пару для передачи сигналов, являясь одной из наиболее широко используемых полевых шин в мире. Протокол CAN используется для связи между различными компонентами в автомобиле, что позволяет заменить дорогие и громоздкие электрические жгуты. Его надежность делает его применимым и в других областях автоматизации и промышленности. Характеристики протокола CAN включают в себя целостность данных серийной связи, поддержку реального времени, скорость передачи до 1Мб/с, а также 11-битную адресацию и возможность обнаружения ошибок.
3D-принтеры используют CAN-шину для уменьшения количества проводов, соединяющих печатающую головку с материнской платой. Вместо десятков проводов с использованием CAN достаточно всего четырех проводов, что значительно упрощает прокладку проводов.
В этой главе дается краткий обзор использования CAN-моста.

## Использование CAN для подключения инструментальной платы к основной плате

    * Обратите внимание, что питание инструментальной платы подключается не к порту CAN, а к линиям питания 12V-24V
