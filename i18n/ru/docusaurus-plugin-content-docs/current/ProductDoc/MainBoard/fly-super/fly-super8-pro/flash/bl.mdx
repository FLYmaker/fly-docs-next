# BL прошивка

    :::tip

    * Обратите внимание, что заводская установка `bootloader` уже выполнена, не нужно его перепрошивать.

    :::
## Вход в режим прошивки


    * Существует два способа входа в режим прошивки
    * Первый: полностью отключите питание платы, коротко замкните BOOT и подключите кабель с передачей данных к плате и вашему устройству
    * Второй: подключите кабель с передачей данных к плате и вашему устройству, коротко замкните BOOT, затем нажмите кнопку RST, отпустите RST и разомкните BOOT
