    :::tip

    * Super8 PRO по умолчанию использует карту TF для прошивки
    * Пожалуйста, приготовьте карту TF объемом не более `32G`
    * Низкокачественные карты памяти могут привести к сбоям при прошивке, по возможности используйте рекомендуемые ниже карты памяти для обновления прошивки
    * Скорость передачи данных карты TF должна быть `class10` или выше, рекомендуется использовать карты TF брендов SanDisk и т.д.

    :::

# Инструкция по прошивке

    :::tip

    * Super8 Pro изначально имеет предустановленный загрузчик на карте TF, повторная прошивка не требуется

    :::

## Как определить, какой тип прошивки вам нужен

    :::tip

    * Обратите внимание, что вам нужно прошить только один тип прошивки

    :::

    * Super8 PRO может подключаться к верхнему уровню следующими способами
    * <Button variant="contained" disableElevation href="./usb">Прошивка через USB</Button>
        * Плюсы: надежная связь, стабильность
        * Минусы: при множестве подключенных USB устройств могут возникнуть проблемы с передачей данных
    * <Button variant="contained" disableElevation href="./can">Прошивка через USB-мост</Button>
        * Плюсы: можно подключить несколько устройств и не занимает USB порт
        * Минусы: возможны ошибки, такие как тайм-ауты при калибровке при использовании вместе с `KlipperScreen`.
