# Параметры продукта

    * Используется 32-битный процессор ARM Cortex-M7 с частотой 550 MHz, STM32H723ZGT6
    * Прошивка: Marlin 2.0/Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * Режимы управления драйверами: TMC: UART/SPI
    * 8 высоковольтных слотов для драйверов, поддержка 8 независимых двигателей, 5 экструдеров, 10 ШИМ вентиляторов
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 высоковольтный вход, поддержка автоматического сенсора выравнивания: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
    * Выбор источника питания для концевых выключателей: 5В/12В/24В
    * При использовании прошивки repap, можно использовать встроенный модуль WiFi ESP32, поддержка скорости до 1.6м/с
    * Уникальный дизайн платы с ШИМ вентиляторами на MOSFET, при повреждении можно легко заменить, уменьшая сложность ремонта для клиентов; два интерфейса для вентиляторов с двумя контактами поддерживают 4-контактные ШИМ вентиляторы
    * Плата PCB с толщиной меди 2OZ, способна выдерживать больший ток
    * 6 интерфейсов АЦП, поддержка до шести термопар
    * 8 интерфейсов ввода-вывода, могут использоваться для выравнивания и концевых выключателей
    * 5 интерфейсов для нагревательных стержней, поддержка до пяти нагревательных стержней
    * 2 интерфейса ШИМ
