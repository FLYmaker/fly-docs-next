# Прошивка BL

    :::tip

    * Обратите внимание, что загрузчик уже предустановлен на заводе, повторная прошивка не требуется.

    :::

## Вход в режим прошивки

    * Существует два способа войти в режим прошивки:
    * Первый: при полностью отключенном питании материнской платы, замкните BOOT и подключите кабель данных к материнской плате и устройству.
    * Второй: после подключения кабеля данных к материнской плате и устройству, замкните BOOT, нажмите кнопку RST, затем отпустите RST и разомкните перемычку BOOT.
