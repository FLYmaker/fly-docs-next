    * Super8 по умолчанию использует карту TF для прошивки
    * Подготовьте карту TF емкостью не более `32G`
    * Карта памяти неизвестного бренда может привести к сбою прошивки, старайтесь использовать рекомендуемые ниже карты памяти для обновления прошивки
    * Скорость передачи данных карты TF должна быть не ниже `class10`, рекомендуется использовать карты TF бренда SanDisk и т.д.

# Инструкция по прошивке


## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, вам нужно прошить только один из вариантов прошивки

    :::
    
    * Super8 может подключаться к верхнему компьютеру следующими способами
    * <Button variant="contained" disableElevation href="./usb">USB прошивка</Button>
        * Преимущества: надежная связь, стабильность
        * Недостатки: если подключено несколько USB устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB мостовая прошивка</Button>
        * Преимущества: можно подключить несколько устройств и не занимать USB порт
        * Недостатки: возможны ошибки, такие как тайм-аут при использовании с `KlipperScreen` одновременно.
