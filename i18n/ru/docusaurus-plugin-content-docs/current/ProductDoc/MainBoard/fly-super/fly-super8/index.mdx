# Параметры продукта


    * 32-битный ARM Cortex-M4 серии 168 МГц, чип STM32F407ZGT6
    * Прошивка: Marlin 2.0/Reprap/Klipper
    * Поддержка драйверов: A4988、LV8729、DRV8225、TMC2208/2209/5160 и т.д.
    * Поддержка режима драйверов: TMC: UART/SPI
    * 8 высоковольтных драйверных позиций, поддержка 8 независимых двигателей, 5 экструдеров, 10 PWM вентиляторов
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 высоковольтный вход, поддержка автоматического сенсора выравнивания кровати: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
    * Выбор питания концевых выключателей: 5В/12В/24В
    * При использовании прошивки repap можно использовать встроенный модуль WiFi ESP32, поддержка до 1.6 м/с
    * Уникальный дизайн платы PWM Fan MOS, поврежденные элементы можно заменить напрямую, что упрощает ремонт для клиентов, два 2-pin интерфейса вентилятора могут поддерживать 4-pin PWM вентиляторы
    * Плата PCB с толщиной меди 2OZ, способна поддерживать больший ток
    * 6 интерфейсов АЦП, максимум поддержка шести термисторов
    * 8 интерфейсов ввода/вывода, могут использоваться для выравнивания и концевых выключателей
    * 5 интерфейсов для нагревательных элементов, максимум поддержка пяти нагревательных элементов
    * 2 интерфейса PWM
