# Расширение управления светодиодами через плагин

Необходимо подключиться через SSH к верхнему компьютеру и следовать инструкциям по установке и настройке по ссылке ниже:

```
https://github.com/julianschill/klipper-led_effect
```

## Установка

Используйте **MobaXterm_Personal** или другой **инструмент SSH** для подключения через **Wi-Fi** к вашему верхнему компьютеру, убедитесь в следующем:

1. **Убедитесь, что на верхнем компьютере установлена прошивка Klipper**
2. **Убедитесь, что вы вошли в систему под пользователем, который установил Klipper**
3. **Убедитесь, что ваша раскладка клавиатуры установлена на английскую**
4. **Убедитесь, что ваш верхний компьютер может нормально обнаруживать устройство**
5. **Убедитесь, что все вышеуказанные моменты выполнены, иначе невозможно будет продолжить**

Этот модуль можно установить в существующую установку Klipper с помощью скрипта.

```
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Пример конфигурации

```
[neopixel TP_led]
pin: PB7
chain_count: 25
# Количество светодиодов
# количество светодиодов
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
