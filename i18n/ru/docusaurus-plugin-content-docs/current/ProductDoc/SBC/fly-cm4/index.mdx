# Введение

      Fly-CM4 полностью совместим с Raspberry Pi 3B и 4B, и специально оптимизирован для Raspberry Pi CM4. Эта плата расширения предоставляет полный набор функций, оптимизированных для 3D-принтеров. Питание постоянным током 12-24В обеспечивает стабильность питания, что критически важно для точного управления и непрерывной работы 3D-принтера. Поддержка FLY-TFT и Fly-HDMI экрана с FPC-кабелем предоставляет более удобный способ прокладки кабелей. Наличие CAN-порта позволяет напрямую соединять верхний и нижний уровни, что является сейчас популярным способом.

## Характеристики

    * Поддержка ядра Raspberry Pi CM4
    * Поддержка питания постоянным током 12В-24В
    * Встроенный модуль UTOC
    * Поддержка TFT и HDMI однолинейного соединения
    * Поддержка интерфейсов экрана DSI/spi
    * Поддержка интерфейсов камеры CSI/USB
    * 40PIN интерфейс расширения, совместимый с Raspberry Pi
