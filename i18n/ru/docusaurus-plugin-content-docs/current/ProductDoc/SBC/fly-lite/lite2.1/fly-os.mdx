## Подготовка к записи системы

    * Картридер
    * Карта TF, емкость должна быть не менее 16 ГБ или не более 128 ГБ, скорость передачи данных должна быть class10 или выше
    * Загрузить соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-lite2">образ системы</Button>
    * Загрузить нужное <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">программное обеспечение для записи</Button>

## Запись системы

:::info
* Lite 2.1 может записывать систему только на TF-карту, M2WE не поддерживается
* После завершения записи системы, TF-карту необходимо установить на Lite 2.1
