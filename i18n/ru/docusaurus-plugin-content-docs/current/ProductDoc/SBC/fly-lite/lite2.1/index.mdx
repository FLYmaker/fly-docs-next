# Введение

    Fly-Pi-lite2.1 - это начальный уровень Klipper хост-машина
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.png').default} size="80%" align="center" />

## Характеристики

    * CPU: Высокопроизводительный чип Allwinner H3, 4-ядерный 64-битный Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Поддержка до 128GB SDCARD
    * WIFI: Встроенный 2.4G WiFi
    * Периферия: Spi x1, URAT x2, USB x2, HDMI x1
    * Интерфейсы: Зарезервированы FPC, HDMI + USB FPC в одной линии
    * Операционная система предустановлена klipper, moonraker, mainsail, fulidd, klipper-screen, плагин компенсации резонанса numpy и FLY-Tools, не требуется сложных операций по замене источников, что делает его доступным для новичков (необходима покупка TF-карты)


## Обновленные функции

    * Интерфейс TFT V1 обновлен до TFT V2
    * Добавлены контакты сброса и выключения
    * Добавлена индикаторная лампа включения питания
    * Удалена кнопка BOOT, заменена кнопкой сброса
    * Оптимизирована прокладка проводов
