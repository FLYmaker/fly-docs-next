# Подключение и настройка экрана
:::info

* Для использования Fly Tools на lite2.1 необходимо подключиться к системе через SSH и ввести следующую команду:
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Чтобы использовать Fly Tools, необходимо подключить верхний компьютер к сети и ввести IP-адрес верхнего компьютера:`IP+:9999`
* Например, `192.168.6.110:9999`

:::
