# Подключение и настройка экрана
:::info

* lite2.1 требует подключение по SSH к системе и ввод команды ниже перед использованием Fly Tools для настройки экрана
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools требует подключения к сети и ввода IP-адреса хоста с портом `:9999` для доступа
* Например `192.168.6.110:9999`

:::
