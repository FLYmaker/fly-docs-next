# Подключение к Wi-Fi

## Установка антенны на LITE 2.1

    * Место, указанное красной стрелкой, является интерфейсом антенны Wi-Fi
    * Разъем антенны Wi-Fi - `ipex1`
    * Установите антенну на LITE 2.1, иначе из-за слабого сигнала может быть невозможно подключиться к сети

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="100%" align="left" />  

* Если вы обнаружили, что Wi-Fi не подключается нормально, сначала проверьте, подключена ли антенна, а затем используйте приведенный ниже метод для подключения к Wi-Fi
