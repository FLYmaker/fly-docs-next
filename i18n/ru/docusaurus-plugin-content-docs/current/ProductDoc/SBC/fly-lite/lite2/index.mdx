# Введение

    Fly-Pi-lite2 - это начальный уровень Klipper хост-компьютер

## Характеристики

    * CPU: Высокопроизводительный чип Allwinner H3, 4 ядра, 64-битный Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Поддержка SDCARD до 128GB
    * WIFI: Встроенный 2.4G WiFi
    * Периферия: Spi x1, URAT x2, USB x2, HDMI x1
    * Интерфейсы: Зарезервированы FPC, HDMI + USB FPC для единой линии
    * Операционная система предустановлена с Klipper, Moonraker, Mainsail, Fulidd, Klipper-screen, плагинами компенсации резонанса numpy и FLY-Tools, не требует сложных операций замены источников, что упрощает процесс для новичков (необходима покупка TF-карты)
