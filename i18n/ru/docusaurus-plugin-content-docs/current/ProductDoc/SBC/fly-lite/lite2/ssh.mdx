# Подключение SSH к верхнему компьютеру


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Заранее подготовьте **MobaXterm_Personal** или другой **SSH-терминал**
* Убедитесь, что на SD-карте или M2WE записана **система для верхнего компьютера**
* **Если верхний компьютер не запускается нормально, отсоедините приводы, концевые выключатели, вентиляторы и другие периферийные устройства!!!**
* **В конце подключитесь через сеть к SSH для удобства компиляции прошивки!!**
* **Для компиляции прошивки ознакомьтесь с руководством по подключению верхнего компьютера через Wi-Fi или сетевой кабель**

        ## Подключение к SSH через сетевой IP

        * Войдите в **админку роутера**, к которому подключен верхний компьютер
        * Убедитесь, что система **запущена нормально**
        * Убедитесь, что **вы знаете имя пользователя и пароль верхнего компьютера**, по умолчанию в системе FLY имя пользователя **fly**, пароль **mellow**
        * Если плата уже запущена и подключена к сети, вы можете подключиться к SSH с помощью IP
        * Откройте MobaXterm и выполните следующие настройки, введите IP, который вы видели ранее, или проверьте в админке роутера

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Внимание: здесь на шаге 4 вводится имя пользователя верхнего компьютера**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль верхнего компьютера и нажмите Enter

        **Внимание: здесь вводимый пароль не отображается, просто введите и нажмите Enter**

        В системе FLY по умолчанию имя пользователя: `fly`, пароль: `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />
