# Подключение к WiFi

## Установка антенны на LITE 2

    * Красной стрелкой указано место для подключения WiFi антенны
    * Разъем для WiFi антенны - `ipex1`
    * Установите антенну на LITE 2, иначе возможны проблемы с подключением из-за слабого сигнала

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.png').default} size="100%" align="left" />  

* Если WiFi не подключается, сначала проверьте, подключена ли антенна, затем используйте методы ниже для подключения к WiFi
