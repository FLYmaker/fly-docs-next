# Использование акселерометра

    :::tip

    * Mini Pad может использовать встроенный интерфейс для прямого подключения акселерометра или использовать инструментальную плату с встроенным акселерометром

    :::

## Подключение акселерометра к встроенному интерфейсу

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.png').default} size="80%" align="left" />

    * Пример конфигурации

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # пример
    ```
