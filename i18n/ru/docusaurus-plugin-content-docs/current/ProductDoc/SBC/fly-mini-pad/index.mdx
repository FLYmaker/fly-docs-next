# Введение

    Fly-MINI-Pad - это вводный уровень верхнего компьютера Klipper

## Характеристики

    * CPU: Высокопроизводительный чип Allwinner H3, 4-ядерный 64-битный Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: Поддержка SD-карт до 128GB
    * WIFI : Встроенный 2.4G WiFi
    * Периферия: SPI x1, USB x2, Type-C x1
    * Интерфейсы: Три интерфейса XH2.54-2P поддерживают POWER-IN для питания до 24В, CAN-BUS для подключения по CAN-шине, LI-PO для питания/зарядки от защищённого литий-полимерного аккумулятора 3.7В
    * Система предустановлена с klipper, moonraker, mainsail, fulidd, klipper-screen, плагином компенсации резонанса numpy и FLY-Tools, без необходимости сложных операций по замене источников, что делает его более доступным для новичков (требуется приобретение карты TF)
    * Экран: емкостной/резистивный, разрешение 320*480
