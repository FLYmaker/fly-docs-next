# Подключение к хост-машине через SSH

* Можно подключиться к устройству через последовательный порт или Wi-Fi
* Заранее подготовьте **MobaXterm_Personal** или другой **SSH терминальный инструмент**
* Убедитесь, что на SD-карте или M2WE записана **система для соответствующей хост-машины**
* **Если хост-машина не запускается, отсоедините драйверы, ограничители, вентиляторы и другие периферийные устройства!!!**
* **В конце подключитесь к SSH через сеть для удобства компиляции прошивки!!**
* **Для компиляции прошивки смотрите руководство по подключению хост-машины через Wi-Fi или сетевой кабель**

        ## Подключение к хост-машине через SSH по IP-адресу

        * Войдите в **административную панель маршрутизатора**, к которому подключена хост-машина
        * Убедитесь, что система **успешно загружена**
        * Убедитесь, что **вы знаете имя пользователя и пароль хост-машины**, по умолчанию для системы FLY имя пользователя **fly**, пароль **mellow**
        * Если плата запущена и подключена к сети, можно подключиться к SSH по IP
        * Откройте MobaXterm и выполните следующие настройки, введите IP, который вы проверили ранее, или найдите его в административной панели маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Внимание: Здесь в шаге 4 введите имя пользователя хост-машины**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль хост-машины и нажмите Enter

        **Внимание: Введенный пароль не будет виден, после ввода просто нажмите Enter**

        По умолчанию для системы FLY имя пользователя: `fly`, пароль: `mellow`
