# Метод настройки

:::danger Обратите внимание
После использования этого метода вы не сможете подключать акселерометр к интерфейсу акселерометра mini pad.
:::

    * Откройте диск BOOT, найдите файл `armbianEnv.txt` и откройте его
    * Измените **uart2** на **uart3** в строке `overlays=usbhost2 usbhost3 uart2` и сохраните

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart.png').default} size="80%" align="center" />

## Метод подключения

    * Обратите внимание, что TX mini-pad должен быть подключен к RX материнской платы с прошивкой UART
    * Обратите внимание, что RX mini-pad должен быть подключен к TX материнской платы с прошивкой UART
    * Обратите внимание на общий провод
    * Иначе вы не сможете подключить материнскую плату с прошивкой UART

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart1.png').default} size="80%" align="center" />

## Пример конфигурации Klipper

    ```cfg
    [mcu]
    serial: /dev/ttyS3
    baud：250000
    #   Используемая скорость передачи. По умолчанию 250000
    restart_method:command
    ```
