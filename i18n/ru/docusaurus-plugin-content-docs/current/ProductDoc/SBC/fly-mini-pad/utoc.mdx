# Использование встроенного UTOC

:::danger Обратите внимание

Не рекомендуется использовать два UTOIC, прошивка и подключение будут очень сложными

:::
    * Если встроенный UTOC не нужен, выполните следующую команду, выполнение не даст никаких подсказок
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * Способ восстановления, выполнение не даст никаких подсказок
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * На MINI Pad уже установлен UTOC
