# Подключение к Wi-Fi

## Установка антенны на MINI PAD

    * Красная стрелка указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi - `ipex1`
    * Установите антенну на MINI PAD, иначе из-за слабого сигнала может быть невозможно подключиться к сети
