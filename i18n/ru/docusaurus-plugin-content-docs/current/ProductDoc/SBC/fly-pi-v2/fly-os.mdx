## Подготовка к записи системы

    * Картридер
    * Карта TF, емкость должна быть не менее 16 ГБ или не более 128 ГБ, скорость передачи данных должна быть class10 или выше
    * Скачать соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">системный образ</Button>
    * Скачать необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент для записи</Button>

## Запись системы

:::info
Запись на карту TF и запись через карту TF на M2WE требуется только один из методов записи
:::

:::info
* Если в устройстве установлены M2WE и карта памяти, система по умолчанию загружается с карты памяти
* Рекомендуется использовать карту TF для записи системы, так как запись через USB на M2W может не удаться из-за проблем с совместимостью драйверов, рекомендуется использовать [запись через карту памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие скачанного системного образа, если он не соответствует, система не запустится
:::
