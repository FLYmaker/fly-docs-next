# Подключение SSH к хосту


* Вы можете подключиться к устройству через последовательный порт или Wi-Fi
* Заранее подготовьте **MobaXterm_Personal** или другой **SSH-клиент**
* Убедитесь, что на SD-карте или M2WE записана **система для вашего хоста**
* **Если хост не запускается, отсоедините драйверы, концевики, вентиляторы и другие периферийные устройства!!!**
* **В конце подключитесь по сети к SSH для удобства компиляции прошивки!!**
* **Для компиляции прошивки смотрите руководство по подключению хоста через Wi-Fi или Ethernet**

        ## Подключение к хосту SSH через сетевой IP

        * Войдите в **административную панель роутера**, к которому подключен хост
        * Убедитесь, что система **успешно загрузилась**
        * Убедитесь, что вы **знаете имя пользователя и пароль хоста**, по умолчанию для системы FLY имя пользователя **fly**, пароль **mellow**
        * Если плата запущена и подключена к сети, вы можете подключиться к SSH через IP
        * Откройте MobaXterm и выполните следующие настройки, введите IP, который вы видели ранее, или найдите его в панели роутера

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Внимание: на шаге 4 введите имя пользователя хоста**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль хоста и нажмите Enter

        **Внимание: вводимый пароль не отображается, после ввода сразу нажмите Enter**

        По умолчанию для системы FLY имя пользователя: `fly`, пароль: `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.png').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.png').default} size="40%" align="center" />
