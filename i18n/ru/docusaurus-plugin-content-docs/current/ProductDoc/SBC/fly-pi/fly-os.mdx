## Подготовка к записи системы

    * Картридер
    * Карта TF емкостью не менее 16 ГБ или не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Загрузите соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Загрузите необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент для записи</Button>

## Запись системы

:::info
Запись на карту TF и запись через карту TF на M2WE требуется использовать только один из методов
:::

:::info
* Если в устройстве установлены M2WE и карта памяти, система по умолчанию загружается с карты памяти
* Рекомендуется использовать карту TF для обновления системы на M2WE, так как запись через USB может не удаться из-за проблем совместимости драйверов, рекомендуется использовать [запись через карту TF на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие загружаемого образа системы, иначе система не запустится
:::
