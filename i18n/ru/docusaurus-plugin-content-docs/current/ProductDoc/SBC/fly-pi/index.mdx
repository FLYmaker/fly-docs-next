# Введение

        Мы с радостью представляем совершенно новую материнскую плату FLY-π. Эта плата может заменить Raspberry Pi в качестве хост-компьютера, она оснащена 1 ГБ встроенной DDR3 памяти, что позволяет запускать полноценный рабочий стол LINUX (на базе оптимизированного Armbian). Эта плата может использоваться вместе с прошивками Klipper и RRF, а также может запускать экран Klipper при использовании инструментальной платы SHT.

Цель разработки этой хост-машины - упростить использование прошивок Klipper или RRF для клиентов. Мы оптимизировали образ системы LINUX для более удобного использования с 3D-принтерами, установка и работа стали проще, даже новички с небольшими знаниями смогут быстро освоиться. По сравнению с другими хост-платами это экономит время на обучение, позволяя вам сосредоточиться на удовольствии от 3D-печати (покупка комплекта с SD-картой с предустановленной прошивкой Klipper позволяет сразу начать работу, что экономит время и силы). Установка также проще, даже новички с небольшими знаниями смогут быстро освоиться. ## Характеристики

    * CPU: Высокопроизводительный чип Allwinner H5, 4-ядерный 64-битный Cortex-A53
    * GPU: Высокопроизводительный 6-ядерный Mali 450, скорость заполнения пикселей более 2.7gpixel/s
    * RAM: 1ГБ DDR3 (общий с GPU) (512+512)
    * ROM: Поддержка SD-карт до 128ГБ
    * Периферия: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100М), CAN x1 (требуется отдельная покупка модуля CAN)
    * Интерфейсы: Встроенный интерфейс M.2 для расширения WIFI, eMMC и т.д. (проприетарный протокол, не подключайте неспециализированные для FLY π устройства M.2)
    * 40-пиновый интерфейс, совместимый с Raspberry Pi
    * Поддержка питания постоянным током 12-24В, более стабильное питание
    * Предусмотрен интерфейс для контролируемого вентилятора на 5В для охлаждения хост-машины
    * Предусмотрен интерфейс расширения CAN
    * Система предварительно установлена с klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, плагином компенсации резонанса numpy и FLY-Tools, нет необходимости в сложных операциях по смене источников, новички легче освоятся (требуется покупка TF-карты или M2WE)
