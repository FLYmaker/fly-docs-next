# Подключение к SSH верхнего уровня

* Можно подключиться к устройству через последовательный порт или Wi-Fi
* Заранее подготовьте **MobaXterm_Personal** или другой **инструмент для SSH терминала**
* Убедитесь, что на SD-карте или M2WE записана **система для верхнего уровня**
* **Если верхний уровень не запускается, отключите все периферийные устройства, такие как драйверы, концевые выключатели, вентиляторы и т.д.!!!**
* **В конце подключитесь к SSH через сеть для удобства компиляции прошивки!!!**
* **Для компиляции прошивки смотрите руководство по подключению верхнего уровня через Wi-Fi или сетевой кабель**

        ## Использование IP-адреса для подключения к SSH верхнего уровня

        * Зайдите в **административную панель роутера**, к которому подключен верхний уровень
        * Убедитесь, что система **успешно запущена**
        * Убедитесь, что вы **знаете имя пользователя и пароль верхнего уровня**, по умолчанию для системы FLY имя пользователя **fly**, пароль **mellow**
        * Если плата уже запущена и подключена к сети, вы можете подключиться к SSH по IP
        * Откройте MobaXterm и настройте как показано ниже, введите IP-адрес, который вы нашли ранее или посмотрите в административной панели роутера

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.png').default} size="40%" align="center" />

        **Внимание: Здесь на четвертом шаге введите имя пользователя верхнего уровня**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.png').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.png').default} size="40%" align="center" />

        * Появится запрос на ввод пароля, введите пароль верхнего уровня и нажмите Enter

        **Внимание: Введенный пароль не будет виден, просто нажмите Enter после ввода**

        По умолчанию для системы FLY, имя пользователя: `fly`, пароль: `mellow`
