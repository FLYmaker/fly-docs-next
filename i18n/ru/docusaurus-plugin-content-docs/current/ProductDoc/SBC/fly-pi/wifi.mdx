# Подключение к Wi-Fi


## Установка антенны на PI

    * Красной стрелкой указано место для подключения антенны Wi-Fi
    * Разъем для антенны Wi-Fi - `ipex1`
    * Установите антенну на PI, иначе из-за слабого сигнала может быть невозможно подключиться к интернету
