    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-lcd/img/dsi_lcd5.webp').default} size="50%" align="left" />

## Описание продукта

    * Размер экрана: 120.7mm*76.0mm
    * Область отображения: 108mm*66.2mm
    * Разрешение: 800*480
    * Тип экрана: IPS
    * Способ связи: DSI/Mipi
    * Порты на плате: 15 pin DSI / 22pin Mipi
    * Стабильность работы: 0°-60°C
    * Тип сенсора: Capacitive（емкостной）
    * Логическое напряжение: 3.3V
    * Вес: 135.5g

        :::danger Важно знать

        * На Raspberry Pi 5 интерфейсы DSI и CSI теперь называются MIPI
        * Интерфейс MIPI можно настроить для прямого использования камеры CSI или экрана DSI
        * Обратите внимание на направление контактов

        :::

        * Теперь на Raspberry Pi 5 есть два интерфейса MIPI, если смотреть с лицевой стороны, можно увидеть маркировку 1 и 0

        * Подключение экрана DSI


            * Если используется интерфейс 1, необходимо настроить экран следующим образом

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Если используется интерфейс 0, необходимо настроить экран следующим образом

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
