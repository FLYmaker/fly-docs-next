## Введение в продукт

    * Размер экрана: 120.7мм*76.0мм
    * Область отображения: 108мм*66.2мм
    * Разрешение: 800*480
    * Тип экрана: IPS
    * Способ связи: DSI/Mipi
    * Бортовой интерфейс: 15 pin DSI / 22pin Mipi
    * Стабильная работа: 0°-60°C
    * Тип сенсора: Емкостной
    * Логическое напряжение: 3.3В
    * Вес: 135.5г

        :::danger Обратите внимание

        * Интерфейсы DSI и CSI Raspberry Pi 5 теперь называются MIPI
        * Интерфейс MIPI можно настроить для использования с камерой CSI или экраном DSI
        * Обратите внимание на направление контактных пластин

        :::

        * На Raspberry Pi 5 теперь есть два интерфейса MIPI, если смотреть спереди, можно увидеть обозначения 1 и 0

        * Подключение экрана DSI


            * Если используется интерфейс №1, необходимо настроить экран следующим образом:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Если используется интерфейс №0, необходимо настроить экран следующим образом:

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
