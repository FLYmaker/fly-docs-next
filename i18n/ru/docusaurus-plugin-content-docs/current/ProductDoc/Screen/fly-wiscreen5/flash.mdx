# Изменение изображения при загрузке
:::tip
    Пожалуйста, строго следуйте `разрешению` и `имени файла` при создании изображений, иначе система их не распознает

    Поместите созданные изображения в корневой каталог SD-карты/USB-накопителя, при включенном экране вставьте экран, нажмите на обновление
:::
## Изменение логотипа при загрузке

    * Требования к изображению `800*480`
    * Имя должно быть `boot_logo.JPG`
    * Не должно превышать `105KB`




## Изменение изображения на первой странице

    * Требования к изображению `500*400`
    * Имя должно быть `print.png`
    * Не должно превышать `45KB`



## Изменение изображения на странице печати

    * Требования к изображению `311*480`
    * Имя должно быть `print_smail.png`
    * Не должно превышать `45KB`
