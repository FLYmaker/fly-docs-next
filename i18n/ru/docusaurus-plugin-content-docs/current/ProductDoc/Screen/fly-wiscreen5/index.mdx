# Введение

      Fly_WiScreen - это высококачественный экран управления Klipper, выпущенный командой разработчиков платы Fly3D компании Guangzhou Magnesium Electronic Technology Co., Ltd. Экран позволяет управлять 3D-принтером через Wi-Fi, поддерживает передачу файлов для печати с SD-карты и USB-накопителя, обеспечивая пользователю очень быстрый отклик экрана и отличный опыт использования. Поддерживает настройку режима главного и подчиненного устройства для удобного управления несколькими машинами. Разъем для питания 5В может быть подключен к зарядному модулю для модификации батареи, что удобно для мобильного использования.

## Функции продукта

    * Поддержка подключения через Wi-Fi к другим хостам с установленным Klipper
    * Поддержка поиска по Wi-Fi, после подключения к Wi-Fi можно автоматически найти хост
    * Поддержка управления кластером нескольких принтеров
    * Поддержка загрузки файлов для печати через USB и TF-карту
    * Настраиваемые макросы для удобства операций и отладки

## Параметры продукта

    * Процессор: Allwinner T113-S3, двухъядерный CORTEX-A7, макс. 1.2 ГГц
    * Оперативная память: интегрированная DDR3 на 128 МБ
    * Хранилище: 128 МБ Nand Flash, слот для TF-карты
    * Дисплей: 800x480 емкостной экран
    * Внешние интерфейсы: один USB 2.0, один слот для TF-карты
    * Протокол Wi-Fi: 802.11b/g/n
    * Тип экрана: IPS
    * Размер экрана: 5 дюймов
    * Многоточечное касание: поддерживается
    * Номинальное напряжение: 5В
