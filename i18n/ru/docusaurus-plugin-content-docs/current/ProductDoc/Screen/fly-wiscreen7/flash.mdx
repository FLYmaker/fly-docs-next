# Изменение загрузочной картинки
:::tip
    Пожалуйста, строго следуйте указаниям по `разрешению` и `имени файла` при создании изображений, иначе система их не распознает.

    Поместите созданные изображения в корневой каталог SD-карты/флешки, вставьте её в экран, когда тот включен, и нажмите кнопку обновления.
:::
## Изменение логотипа при загрузке

    * Разрешение изображения должно быть `1024*600`
    * Имя файла должно быть `boot_logo.JPG`
    * Размер файла не должен превышать `105KB`



## Изменение изображения на первой странице

    * Разрешение изображения должно быть `736*563`
    * Имя файла должно быть `print.png`
    * Размер файла не должен превышать `45KB`



## Изменение изображения на странице печати

    * Разрешение изображения должно быть `355*565`
    * Имя файла должно быть `print_smail.png`
    * Размер файла не должен превышать `45KB`
