# Введение

      Fly_WiScreen — это высококлассный экран управления Klipper от команды разработчиков Fly3D компании Guangzhou Magnesium Electronics Technology Co., Ltd, который позволяет управлять 3D-принтером через Wi-Fi или Ethernet. Поддерживает передачу файлов для печати с SD-карты и USB-накопителя, экран имеет быстрый отклик, что обеспечивает отличный пользовательский опыт. Поддерживает режим мастер-слейв для удобного управления несколькими машинами. Порт питания 5В может быть использован для подключения зарядного модуля для модификации аккумулятора, что делает его удобным для мобильного использования.

## Функции продукта

    * Поддержка подключения к другим хостам с установленным Klipper через Wi-Fi и Ethernet
    * Поддержка поиска Wi-Fi, после подключения к Wi-Fi автоматически ищет хост
    * Поддержка управления кластером нескольких принтеров
    * Поддержка загрузки файлов для печати через USB и карту TF
    * Пользовательские макросы для удобства операций и отладки

## Параметры продукта

    * CPU: Allwinner T113-S3, двухъядерный CORTEX- A7, до 1.2GHz
    * Память: встроенная 128M DDR3
    * Хранилище: 128MB Nand Flash, слот для TF-карты
    * Дисплей: 1024x800 емкостной экран
    * Внешние интерфейсы: один USB 2.0, один слот для TF-карты
    * Wi-Fi протокол: 802.11b/g/n
    * Тип экрана: IPS
    * Размер экрана: 7 дюймов
    * Мультитач: поддерживается
    * Номинальное напряжение: 5V
