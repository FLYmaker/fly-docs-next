## Настройка Klipper

### Настройка светодиода

    ```cfg
    [led LED]
    white_pin:PA15
    ```

### Настройка ptc

    ```cfg
    [heater_generic warehouse]
    heater_pin:E_CS
    sensor_type:ATC Semitec 104GT-2
    sensor_pin:PA1
    min_temp:-270
    max_temp:600
    control = pid
    pid_kp = 16.804
    pid_ki = 0.794
    pid_kd = 88.854
    ```

### Настройка вентилятора

    ```cfg
    [fan_generic fan1]
    pin: FAN1
    ```


    ```cfg
    [fan_generic fan2]
    pin: FAN2
    ```

 

### Пользовательские G-code

    * После свайпа вниз на экране появляется пользовательский G-code,
    * Формат кнопки для отправки нескольких команд G-code:
    * Пример:

    ```cfg
    G91 
    G1 X10 F6000
    G90
    ```

    * Формат ввода на экране: (начиная со второй строки, перед каждой командой добавьте \n)

    ```cfg
    G91\nG1 X10 F6000\nG90
    ```



## Использование исключений объектов

    * Необходимо заменить moonraker

    * Сначала подключитесь по SSH и убедитесь, что у вас есть доступ в интернет

    1. Создайте резервную копию файлов moonraker

    ```bash
    mv ~/moonraker ~/moonraker-bak
    ```

    2. Остановите службу moonraker

    ```bash
    sudo systemctl restart moonraker
    ```

    3. Загрузите moonraker

    ```bash
    git clone https://e.coding.net/g-ofpa1390/3D-Printers/moonraker-dev.git -b flylcd-dev moonraker
    ```

    4. В браузере перейдите на верхний компьютер, найдите `moonraker.conf`, добавьте и перезагрузите систему

    ```bash
    [network]
    ```

    5. Затем введите в браузере адрес ниже, где IP нужно заменить на IP верхнего компьютера

    ```bash
    http://ip:7125/server/network/wifi/scan
    ```

* Этот шаг означает, что операция завершена
