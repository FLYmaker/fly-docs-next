## Введение
#### Папка Logic
* Мы программируем 99% кода пользовательского интерфейса в папке logic, другие папки можно игнорировать
* gcodeLogic.cc аналогично main.c,    99% кода пользовательского интерфейса находится здесь, другие файлы можно не обращать внимания

#### Папка i18n
* Хранит таблицы перевода на различные языки

#### Папка resources
* Хранит изображения, используемые в пользовательском интерфейсе

#### Папка ui
* Хранит файлы интерфейса пользователя, двойной клик для отображения
* Двойной клик по стрелке в левом нижнем углу, указывающей на "маленький глаз", позволяет открыть скрытые интерфейсы
