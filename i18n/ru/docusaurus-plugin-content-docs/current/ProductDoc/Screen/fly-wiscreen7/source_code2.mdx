## Настройка функциональности кнопки
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide10.gif').default} size="100%" align="left" />

* Использование функции для отправки одной команды G-code  
    ```cfg
    gcode_send2("","G28");
    ```
* Использование функции для отправки нескольких команд G-code за один раз:
    ```cfg
    gcode_send2("","G91\\nG1 X10 F3000\\nG90");
    ```
* Или используйте этот метод для отправки нескольких команд G-code за один раз:
    ```cfg
    gcode_send2("","G91");
    gcode_send2("","G1 X10 F3000");
    gcode_send2("","G90");
    ```
