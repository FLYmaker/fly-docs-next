##   Обновление
    * 1. Переименуйте загруженный файл в `update.img`
    * 2. Поместите файл `update.img` на SD-карту/USB-накопитель и вставьте его в экран, можно вставить при включенном экране.
    * 3. Выберите экран, нажмите на обновление, чтобы завершить обновление
