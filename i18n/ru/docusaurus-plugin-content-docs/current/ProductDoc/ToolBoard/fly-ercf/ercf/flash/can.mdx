# Компиляция прошивки CAN

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце появится следующее сообщение, если компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

    * Перед загрузкой прошивки подключите плату инструментов к UTOC или к основной плате с прошивкой моста, используя кабель XT30, ниже приведена схема подключения
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L при отключенном питании составляет около `60Ω`, если оно равно `120Ω`, установите перемычку `120Ω`, если около `40Ω`, удалите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/120.webp').default} size="50%" align="center" />
