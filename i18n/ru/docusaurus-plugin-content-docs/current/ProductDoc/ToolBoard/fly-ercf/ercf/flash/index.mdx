# Инструкция по прошивке прошивки

    * ERCF можно подключить к хосту через CAN
    * CAN необходимо подключить с помощью `XT30(2+2)-F`
    * Ниже представлен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Для прошивки прошивки на плату необходимо использовать Katapult для прошивки Klipper
* Если плата уже прошита Katapult, то нет необходимости выполнять последнюю прошивку Katapult, это просто на случай если прошивка Katapult на плате не удалась
* Можно быстро дважды нажать кнопку сброса и проверить, загорится ли светодиод на плате, если да, то плата уже перешла в режим Katapult
:::

## Как определить, какую прошивку вам нужно

    * ERCF можно подключить к хосту следующими способами
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN</Button>
        * Плюсы: Если прошить прошивку `Katapult`, можно обновлять прошивку без необходимости трогать плату, а также поддерживает подключение по шине
        * Минусы: Возможны ошибки, такие как тайм-ауты при возврате в исходное положение при использовании `KlipperScreen`.
        * Совет: Если вам нужно использовать CAN, убедитесь, что хост поддерживает CAN, и потребуется использовать UTOC или плату с прошитой прошивкой моста
    * <Button variant="contained" disableElevation href="./usb">Прошивка USB</Button>
        * Плюсы: Если прошить прошивку Katapult, можно обновлять прошивку без необходимости трогать плату, если используется SD-карта или DFU прошивка, все равно потребуется обновлять прошивку по старому методу, надежная связь, стабильность
        * Минусы: При наличии нескольких USB-устройств возможно влияние на передачу данных
