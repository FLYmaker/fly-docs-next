# Компиляция прошивки через USB

    :::tip

    * ERCF поставляется с предварительно прошитой прошивкой Katapult для подключения по CAN
    * Если требуется записать USB прошивку, нужно использовать USB версию Katapult
    * После записи прошивки повторная запись не требуется
    * <Button variant="contained" disableElevation href="./bl">Запись прошивки Katapult</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/usb.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * При успешной компиляции в конце появится следующее сообщение:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
