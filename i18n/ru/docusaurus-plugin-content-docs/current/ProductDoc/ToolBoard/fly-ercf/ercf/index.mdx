# Введение в продукт

    FLY-ERCF - это плата управления многокрасочного 3D принтера на базе RP2040, в основном используется для проекта Enraged Rabbit. Также может быть использована в других ситуациях с двумя осями шаговых двигателей.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.webp').default} size="100%" align="left" />

## Особенности продукта

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Два позиции для драйверов, поддержка UART и SPI драйверов
    * Совместимость с оригинальной ERCF EASY Board
    * Минимальный размер, использована 4-слойная плата
    * На борту 2.54MM двойной ряд 7PIN разъем для расширения, например, для концевых выключателей
    * Поддержка напряжения 12-24V
    * Поддержка подключения по CAN шине и RS232, передача данных более стабильная, задержка меньше, подключение стабильнее
    * Интерфейс CAN использует XT30 (2+2), поддержка тока 15A

## Параметры продукта

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Прошивка: Klipper
    * Входное напряжение: DC12V-24V
    * Интерфейсы: 2.54MM двойной ряд 7PIN разъем, SERVO X1, ENDSTOP X2, EXTRA X1
    * Драйверы: поддержка UART/SPI и STEP/DIR

    * Поддержка напряжения 12-24V
    * Поддержка CAN шины, передача данных более стабильная, задержка меньше, подключение стабильнее
    * Интерфейс CAN использует XT30 (2+2), поддержка тока 15A
