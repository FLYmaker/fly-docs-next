import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Переключение на Klipper

    * Потому что еще не объединено, нужно переключиться на Klipper, чтобы использовать

    ```bash
    sudo cp -r klipper klipperdev
    sudo rm -rf ~/klipper
    git clone https://github.com/FLYmaker/klipper.git
    sudo systemctl restart klipper
    ```

    * Пример конфигурации
    ```cfg
    #####################################################################
    #                       Конфигурация акселерометра LIS3DH (включите при необходимости)
    #####################################################################
    [lis2dw]
    is_lis3dh:True
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis2dw
    probe_points:
        125, 125, 20  #Здесь установите координаты, где будет находиться экструдер во время измерения резонанса


```
