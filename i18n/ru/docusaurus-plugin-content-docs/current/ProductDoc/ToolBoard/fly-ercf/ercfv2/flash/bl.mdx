# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * ERCF V2 поддерживает CANR, S232, USB
    * Здесь предоставляются только CAN прошивки на скорости 500K и 1M

    :::

## Вход в режим прошивки

    :::tip

    Кнопка `BOOT` обведена кружком

    :::

    * Для входа в режим прошивки нужно полностью отключить питание платы, затем зажать кнопку BOOT и подключить кабель данных к плате и компьютеру
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.png').default} size="100%" align="center" />

            4. Прошивка

            :::tip

            Нужно прошить только одну прошивку

            :::

            * Команда для прошивки прошивки с CAN скоростью 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_500K.uf2
                ```
                       
            * Команда для прошивки прошивки с CAN скоростью 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_1M.uf2
                ```
            * Команда для прошивки USB прошивки
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCFV2_CANBOOT_USB.uf2
                ```

            5. При появлении следующего сообщения прошивка успешна

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            После завершения прошивки необходимо полностью отключить питание платы.
