# Инструкция по прошивке прошивки

    * ERCF V2 может подключаться к верхнему уровню через RS232 и CAN
    * RS232 и CAN требуют использования `XT30(2+2)-F` для подключения
    * Ниже показан стиль кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Внимание
* Прошивка материнской платы требует использования Katapult для прошивки Klipper
* Если материнская плата уже прошита Katapult, последний шаг по прошивке Katapult можно пропустить, это предусмотрено на случай, если прошивка Katapult не удалась
* Быстро нажмите кнопку сброса дважды, чтобы проверить, загорится ли светодиод на плате, если да, то плата уже в режиме Katapult
:::

## Как определить, какую прошивку вам нужно

    * ERCF V2 может подключаться к верхнему уровню следующими способами
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN</Button>
        * Преимущество: если прошита прошивка `Katapult`, можно обновлять прошивку без необходимости физического доступа к плате, поддержка шинного подключения
        * Недостаток: возможны ошибки, такие как тайм-аут при возврате в исходное положение при использовании `KlipperScreen`.
        * Совет: если планируется использовать CAN, убедитесь, что верхний уровень поддерживает CAN, и требуется UTOC или плата с прошитым мостовым прошивкой
    * <Button variant="contained" disableElevation href="./rs232">Прошивка RS232</Button>
        * Преимущество: не требуется специальное программное обеспечение, надежная связь, стабильность
        * Недостаток: для обновления прошивки необходимо вручную войти в режим DFU или дважды нажать кнопку сброса, чтобы войти в `Katapult` для обновления прошивки
        * Совет: требуется использование G2T или UTOR
    * <Button variant="contained" disableElevation href="./usb">Прошивка USB</Button>
        * Преимущество: если прошита прошивка Katapult, можно обновлять прошивку без необходимости физического доступа к плате, надежная связь, стабильность
        * Недостаток: при наличии нескольких USB устройств возможно влияние на передачу данных
