# Компиляция прошивки USB

    :::tip

    * ERCF поставляется с предустановленной прошивкой Katapult для подключения по CAN
    * Для записи прошивки USB необходимо записать прошивку Katapult для USB
    * После записи повторная запись не требуется
    * <Button variant="contained" disableElevation href="./bl">Запись прошивки Katapult</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.png').default} size="100%" align="center" />
    * Нажмите `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4`, чтобы начать компиляцию, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводится следующее, значит компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
