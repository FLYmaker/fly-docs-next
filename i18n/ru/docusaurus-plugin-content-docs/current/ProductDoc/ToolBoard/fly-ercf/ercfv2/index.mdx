# Введение в продукт

    FLY-ERCF - это плата управления многоцветной 3D-принтером на базе RP2040, в основном используемая для проекта Enraged Rabbit. Она также может использоваться в других случаях управления двухосевыми шаговыми двигателями.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.png').default} size="100%" align="left" />

## Особенности продукта

    * MCU: Raspberry Pi rp2040, двухъядерный ARM Cortex-M0+@133MHz
    * Поддержка режима CAN, режима RS232
    * Два позиции для драйверов, поддержка UART и SPI драйверов
    * Совместимость с оригинальной платой ERCF EASY Board
    * Максимально компактный размер, используется 4-слойная конструкция платы
    * Замена 14-контактного разъема на XHD2.54 для более надежного соединения и предотвращения обрывов
    * Поддержка напряжения 12-24V
    * Поддержка подключения через CAN-шину и RS232, более стабильная передача данных, меньшие задержки, более стабильное соединение
    * Интерфейс CAN использует XT30 (2+2), поддерживает ток до 15A

## Параметры продукта

    * MCU: Raspberry Pi rp2040, двухъядерный ARM Cortex-M0+@133MHz
    * Прошивка: Klipper CAN/RS232
    * Входное напряжение: DC12V-24V
    * Интерфейсы: XHD2.54 двойной ряд 7-контактный разъем, SERVO X1, ENDSTOP X2, EXTRA X1
    * Драйверы: поддержка UART/SPI и STEP/DIR
    * Поддержка напряжения 12-24V
    * Поддержка подключения через CAN-шину и RS232, более стабильная передача данных, меньшие задержки, более стабильное соединение
    * Интерфейс CAN использует XT30 (2+2), поддерживает ток до 15A
