    :::

## Вход в режим загрузки

    :::tip

    Вверху находится кнопка `RESET`, внизу - кнопка `BOOT`

    :::

    * Существует два способа входа в режим загрузки
    * Первый: при полностью отключенной плате, зажмите кнопку BOOT и подключите кабель данных к плате и вашему устройству
    * Второй: после подключения кабеля данных к плате и вашему устройству, зажмите кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST и затем отпустите кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/boot.png').default} size="100%" align="center" />

            4. Загрузка прошивки

            :::tip

            Нужно загрузить только одну прошивку

            :::

            * Ниже приведена команда для загрузки прошивки с CAN скоростью 500K
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```
                       
            * Ниже приведена команда для загрузки прошивки с CAN скоростью 1M
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```

            * Ниже приведена команда для загрузки USB прошивки
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_USB.bin
                ```

            5. Если появится сообщение ``File downloaded successfully``, загрузка прошла успешно

            :::tip
