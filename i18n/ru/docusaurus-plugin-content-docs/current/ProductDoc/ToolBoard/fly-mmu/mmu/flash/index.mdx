# Инструкция по прошивке

    * MMU может подключаться к компьютеру тремя способами: через RS232, CAN и USB
    * Для подключения через RS232 и CAN используется разъем `XT30(2+2)-F`, для USB - кабель `TYPE-C` с функцией передачи данных
    * Ниже представлен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Важно
* Для прошивки платы необходимо через Katapult прошить Klipper
* Если плата уже прошита Katapult, то последний этап прошивки Katapult можно пропустить, это лишь на случай, если прошивка Katapult на плате не удалась
* Можно быстро дважды нажать на кнопку сброса и посмотреть, загорится ли на плате светодиод, это будет означать, что плата вошла в режим Katapult
:::

## Как определить, какую прошивку вам нужно

    * MMU может подключаться к компьютеру следующими способами:
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN</Button>
        * Преимущества: Если прошита прошивка `Katapult`, можно обновлять прошивку без физического доступа к плате, поддерживает шинное подключение
        * Недостатки: При использовании с `KlipperScreen` может вызвать ошибки, связанные с таймаутом возврата в исходное положение
        * Совет: Для использования CAN убедитесь, что компьютер поддерживает CAN, и используйте UTOC или плату с прошивкой моста
    * <Button variant="contained" disableElevation href="./rs232">Прошивка RS232</Button>
        * Преимущества: Не требует специального программного обеспечения, надежная и стабильная связь
        * Недостатки: Для обновления прошивки необходимо войти в режим ручного DFU или дважды нажать на RST, чтобы войти в `Katapult` для обновления прошивки
        * Совет: Требуется использование G2T или UTOR
    * <Button variant="contained" disableElevation href="./usb">Прошивка USB</Button>
        * Преимущества: Если прошита прошивка Katapult, можно обновлять прошивку без физического доступа к плате, если используется карта памяти или прошивка через DFU, то обновление производится по-старому, надежная и стабильная связь
        * Недостатки: При наличии нескольких USB-устройств может повлиять на передачу данных
