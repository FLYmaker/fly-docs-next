# Компиляция прошивки для USB

    :::tip

    * MMU изначально прошит фирменным CAN-соединенным Katapult
    * Если вы используете загрузку прошивки через USB, необходимо прошить USB версию Katapult
    * После прошивки повторная прошивка не требуется
    * <Button variant="contained" disableElevation href="./bl">Загрузка прошивки Katapult</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO pins to set at micro-controller startup, введите: !PA15 Обратите внимание, что ! в верхнем регистре
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце вы увидите следующие строки, если компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
