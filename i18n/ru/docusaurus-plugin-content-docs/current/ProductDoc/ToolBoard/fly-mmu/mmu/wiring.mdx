# Подключение материнской платы

    :::warning Примечание

    Из-за высокого разрешения изображений их невозможно рассмотреть в веб-браузере. Пожалуйста, **наведите курсор на схему подключения**, **щелкните правой кнопкой мыши** и выберите **Сохранить изображение как...**, чтобы сохранить изображение на рабочий стол для просмотра!!!

    :::
