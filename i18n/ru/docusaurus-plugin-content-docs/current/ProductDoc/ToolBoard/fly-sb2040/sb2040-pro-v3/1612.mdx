# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования кровати рекомендуется использовать отдельный Z-концевик, не рекомендуется использовать сканирование кровати как Z-концевик
    * При установке Eddy, дно должно быть на 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от нагревательной платы
    * Пожалуйста, убедитесь, что тестирование проводится при комнатной температуре, и не включайте нагревательную плату во время тестирования

    :::

## Рекомендуемая конфигурация и инструкции по использованию

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что на данный момент внешнего модуля 1612 нет, эта конфигурация пока не может быть использована
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация температурной компенсации

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Конфигурация bed_mesh

    * В параметре `horizontal_move_z` в [bed_mesh], рекомендуется установить значение 2, чтобы Eddy был как можно ближе к платформе при сканировании

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
