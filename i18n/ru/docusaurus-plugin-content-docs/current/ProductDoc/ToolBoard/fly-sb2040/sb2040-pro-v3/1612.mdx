# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования кровати рекомендуется использовать отдельный Z-ограничитель, не рекомендуется использовать сканирование кровати в качестве Z-ограничителя
    * При установке Eddy, нижняя часть должна находиться на высоте 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от кровати
    * Пожалуйста, не проводите тесты на нагретой кровати и не включайте её во время тестирования

    :::

## Справочная конфигурация и инструкции по использованию

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что в настоящее время нет внешнего модуля 1612, эта конфигурация временно не может быть использована
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация температурной компенсации

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Конфигурация bed_mesh

    * В [bed_mesh] параметр `horizontal_move_z` рекомендуется установить на 2, чтобы при сканировании платформы Eddy был как можно ближе к платформе

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
