# Введение в бортовые переключатели

:::warning Внимание

Из-за высокого разрешения изображения, на веб-странице его трудно разобрать. Пожалуйста, **поместите курсор мыши над схемой подключения**, **щелкните правой кнопкой мыши** и выберите **Сохранить изображение как**, чтобы сохранить изображение на рабочий стол и просмотреть его!!!

:::
