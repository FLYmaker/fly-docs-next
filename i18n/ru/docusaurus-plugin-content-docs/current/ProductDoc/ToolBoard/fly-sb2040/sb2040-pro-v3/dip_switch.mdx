# Введение в бортовые переключатели

:::warning Примечание

Из-за высокого разрешения изображений, просмотр на веб-странице может быть затруднен. Пожалуйста, **наведите курсор мыши на схему подключения**, **щелкните правой кнопкой мыши** и выберите **Сохранить изображение как**, чтобы сохранить изображение на рабочий стол для просмотра!!!

:::
