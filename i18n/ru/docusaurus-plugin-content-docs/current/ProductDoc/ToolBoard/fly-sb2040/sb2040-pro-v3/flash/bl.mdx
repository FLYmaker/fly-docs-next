# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3 поддерживает CAN и RS232
    * Здесь предоставляются только прошивки CAN с частотами 500K и 1M
    :::

## Вход в режим загрузки

    :::tip

    Слева находится кнопка `RESET`, справа - кнопка `BOOT`

    :::

    * Существует два способа входа в режим загрузки:
    * Первый: при полностью отключенном питании платы, нажмите и удерживайте кнопку BOOT, затем подключите кабель данных к плате и вашему устройству.
    * Второй: после подключения кабеля данных к плате и вашему устройству, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, после чего отпустите RST и затем BOOT.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.png').default} size="100%" align="center" />
