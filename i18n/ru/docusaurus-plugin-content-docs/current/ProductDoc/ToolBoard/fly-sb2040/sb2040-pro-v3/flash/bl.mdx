# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3 поддерживает CAN и RS232
    * Здесь предоставляются только прошивки CAN на скорости 500K и 1M
    :::

## Вход в режим прошивки

    :::tip

    Слева находится кнопка `RESET`, справа - кнопка `BOOT`

    :::

    * Существует два способа войти в режим прошивки
    * Первый: при полностью отключенном питании платы, нажмите и удерживайте кнопку BOOT, затем подключите кабель передачи данных к плате и вашему устройству
    * Второй: после подключения кабеля передачи данных к плате и вашему устройству, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST и затем отпустите кнопку BOOT
