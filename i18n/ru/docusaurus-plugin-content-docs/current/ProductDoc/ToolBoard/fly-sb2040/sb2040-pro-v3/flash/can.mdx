    * Выберите GPIO pins to set at micro-controller startup, введите: !gpio5 Обратите внимание, что ! - это заглавная буква
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/can.png').default} size="100%" align="center" />

    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * При успешной компиляции в конце вы увидите следующий вывод

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

    * Перед загрузкой прошивки подключите плату инструментов к UTOC или материнской плате с загруженной прошивкой моста, используя кабель XT30. Ниже приведена схема подключения
    * Обратите внимание на положение переключателей
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L в отключенном состоянии измеряется мультиметром и составляет около `60Ω`. Если оно равно `120Ω`, откройте переключатель `120Ω`. Если оно около `40Ω`, закройте переключатель
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.png').default} size="100%" align="center" />
