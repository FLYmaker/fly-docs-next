    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />

    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошитым мостовым прошивкой с помощью кабеля данных XT30, ниже представлена схема подключения
    * Обратите внимание на положение переключателя
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что CAN H и CAN L в отключенном состоянии измеряются мультиметром, сопротивление должно быть около `60Ω`, если оно равно `120Ω`, переключатель `120Ω` должен быть включен, если оно около `40Ω`, переключатель должен быть выключен
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.webp').default} size="100%" align="center" />
