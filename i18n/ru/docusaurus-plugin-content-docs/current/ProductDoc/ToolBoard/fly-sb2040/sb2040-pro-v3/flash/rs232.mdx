        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переход в режим прошивки

    :::tip

    Слева кнопка `RESET`, справа кнопка `BOOT`

    :::

    * Есть два способа войти в DFU:
    * Первый: полностью отключите питание платы, зажмите кнопку BOOT, затем подключите кабель передачи данных к плате и компьютеру
    * Второй: подключите кабель передачи данных к плате и компьютеру, зажмите кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, затем отпустите кнопку BOOT
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
