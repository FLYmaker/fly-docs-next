# Компиляция прошивки RS232

## Начало компиляции

    <SSH />

    <RS232 />

    * Выберите GPIO pins to set at micro-controller startup, введите: !gpio5 Обратите внимание, что ! - это заглавная буква
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/rs232.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь к командной строке
    * Введите `make -j4` для начала компиляции, это займет немного времени
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце будет выведено следующее сообщение, если компиляция прошла успешно:

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переход в режим загрузки

    :::tip

    Слева - кнопка `RESET`, справа - кнопка `BOOT`

    :::

    * Для перехода в режим DFU есть два способа:
    * Первый: при полном отключении питания платы, нажмите и удерживайте кнопку BOOT, затем подключите кабель с передачей данных к плате и компьютеру.
    * Второй: подключите кабель с передачей данных к плате и компьютеру, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите RST и затем отпустите BOOT.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.png').default} size="100%" align="center" />
