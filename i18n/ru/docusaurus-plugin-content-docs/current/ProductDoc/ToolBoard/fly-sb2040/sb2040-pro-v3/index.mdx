# 1. Введение в продукт

    FLY SB2040 PRO V3 - это плата, разработанная компанией Guangzhou Mellow Electronic Technology Co., Ltd. для использования с VORON Stealthburner. С помощью этой платы можно заменить сложную проводку между основной платой и печатающей головкой всего четырьмя проводами, что оптимизирует размещение. SB2040 подходит для stealthburner и afterburner, интегрируя в себя CAN-приемопередатчик, RS232-приемопередатчик, USB-порт, драйвер TMC2240, акселерометр LIS2DW, три управляемых вентилятора и другие функции. Эта плата совместима с прошивками Klipper и RRF.

## 1.1 Характеристики продукта

    * Предусмотрена кнопка Boot, позволяющая пользователю через TypeC войти в режим Boot для обновления прошивки
    * Поддержка прошивок Klipper и RRF
    * Поддержка режимов CAN, RS232 и RRF CAN,
    * В режиме RS232 обеспечивает более высокую скорость передачи данных и повышенную стабильность. В режиме CAN поддерживает расширение CAN с предусмотренным интерфейсом для расширения CAN
    * Поддержка температурных датчиков: встроенный чип MAX31865, совместимый с PT100 и PT1000.
    * Встроенные акселерометр LIS2DW и датчик LDC1612
    * Предусмотрен интерфейс RGB, встроенный драйвер 2240, поддержка различных датчиков нивелирования, например, klicky, Voron Tap, PL08n и т.д.
    * Поддержка напряжения 12-24V, тока до 15A, увеличенный порт для нагревательного элемента, номинальный ток 10A

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232 и RRF CAN
    * Добавлен режим RS232 с более высокой скоростью передачи данных и улучшенной стабильностью, а также упрощенным отладочным использованием
    * Добавлен режим RRF CAN, позволяющий переключить плату в режим RRF, что повышает совместимость
    * Добавлен интерфейс для расширения CAN, что упрощает добавление устройств CAN
    * Добавлен переключатель сопротивления терминала, его сопротивление 120R можно выбирать с помощью переключателя без необходимости использования перемычек
    * Добавлен кабель питания/сигнала длиной 1 метр, с двойной скруткой для повышения устойчивости к помехам

## 1.3 Параметры продукта


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Интерфейс подключения использует соединители XT30(2+2), поддерживающие ток до 15A, пиковый ток до 30A
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: LIS2DW и опционально ldc1612 и max31865
    * Интерфейс нагревательного элемента: Нагревательный элемент (HEAT0)
    * Расширенные интерфейсы: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN
    * Вентиляторы: один канал для DC вентилятора, два канала для DC или PWM вентиляторов
    * Драйвер двигателя: встроенный TMC2240
    * Датчики температуры: один канал для 100K NTC или PT1000 (TH0), один канал для опционального PT100/PT1000
