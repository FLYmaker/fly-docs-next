# 1. Введение в продукт

    FLY SB2040 PRO V3 - это инструментальная плата, выпущенная компанией Guangzhou Mellow Electronics Technology Co., Ltd. для VORON Stealthburber печатающей головки. Использование этой платы позволяет заменить множество проводов между основной платой и печатающей головкой всего на четыре, оптимизируя компоновку. SB2040 подходит для stealthburner и afterburner, включает в себя CAN приемопередатчик, RS232 приемопередатчик, USB-порт, драйвер TMC2240, акселерометр LIS2DW, три управляемых вентилятора и другие функции. Эта инструментальная плата совместима с прошивками Klipper и RRF.

## 1.1 Характеристики продукта

    * Имеется кнопка Boot, позволяющая пользователю через TypeC войти в режим Boot для обновления прошивки
    * Поддержка прошивок Klipper, RRF
    * Поддержка режимов CAN, RS232, RRF CAN,
    * В режиме RS232 обеспечивается более высокая скорость передачи данных и стабильность. В режиме CAN поддерживается расширение CAN, с предусмотренным интерфейсом для расширения CAN
    * Поддержка датчиков температуры: встроенный чип MAX31865, совместимый с датчиками температуры PT100 и PT1000.
    * Встроенный акселерометр LIS2DW и датчик LDC1612
    * Предусмотрены интерфейсы RGB, встроенный драйвер 2240, поддержка различных датчиков выравнивания, таких как klicky, Voron Tap, PL08n и др.
    * Поддержка напряжения 12-24V, входной ток 15A, увеличенный порт для нагревательного элемента, номинальный ток 10A

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232, RRF CAN
    * Добавлен режим RS232, обеспечивающий более высокую скорость передачи данных, большую стабильность и упрощенную отладку
    * Добавлен режим RRF CAN, позволяющий переключиться на инструментальную плату RRF, повышая совместимость
    * Добавлен интерфейс для расширения CAN, облегчающий расширение устройств CAN
    * Добавлен переключатель для выбора терминального резистора, терминальный резистор 120R можно выбрать через переключатель, без необходимости использования перемычек
    * Добавлен кабель питания/сигнала длиной 1 метр, кабель выполнен в двойной скрутке, что улучшает устойчивость к помехам

## 1.3 Параметры продукта


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Интерфейс подключения использует соединитель XT30(2+2), поддерживающий ток 15A, пиковое значение 30A
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: LIS2DW и опционально ldc1612 и max31865
    * Интерфейс нагревательного элемента: HEAT0
    * Расширенные интерфейсы: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN-порт
    * Интерфейс вентилятора: один DC вентилятор, два DC или PWM вентилятора
    * Драйвер двигателя: встроенный TMC2240
    * Датчик температуры: один 100K NTC или PT1000 (TH0), один PT100/PT1000 по выбору
