# Методы решения проблемы с недостаточным напряжением драйвера

## Метод один

    :::danger

    * Заменить источник питания
    * Добавить фильтр питания
    * Заменить двигатель для диагностики
    * Подключить корпус двигателя к земле на плате инструмента
    * В конфигурации экструдера добавить `driver_TPFD: 0`

    :::

## Метод два
    * Подключиться к хост-машине через SSH
    * Ввести нижеприведенную команду
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * После обновления klipper эти изменения могут быть отменены
    :::
