# Методы решения проблемы недостаточного напряжения драйвера

## Метод первый

    :::danger

    * Заменить источник питания
    * Добавить фильтр питания
    * Заменить двигатель для проверки
    * Подключить корпус двигателя к GND на плате инструментов
    * В конфигурации привода экструдера добавить `driver_TPFD: 0`

    :::

## Метод второй
    * Подключиться к верхнему компьютеру через SSH
    * Ввести команду ниже
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Обновление Klipper может отменить этот шаг
    :::
