# Компиляция прошивки CAN

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/can.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и выйти в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводится следующее, значит компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка устройства

    * Перед прошивкой подключите плату инструментов к UTOC или основной плате с прошивкой моста, ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.png').default} size="80%" align="left" />
    * Убедитесь, что в выключенном состоянии CAN H и CAN L имеют сопротивление около `60Ω`. Если сопротивление равно `120Ω`, подключите перемычку на `120Ω`. Если около `40Ω`, удалите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/120.png').default} size="50%" align="center" />
