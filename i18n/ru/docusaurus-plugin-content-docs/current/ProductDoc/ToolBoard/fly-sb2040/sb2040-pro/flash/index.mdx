# Инструкция по прошивке

    * SB2040 PRO можно подключать к хост-машине через CAN-соединение.
    * Для подключения CAN требуется использовать кабель `XT30(2+2)-F`.
    * Ниже представлен вид кабеля `XT30(2+2)-F`.
    <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Обратите внимание
* Прошивка основной платы требует использования Katapult для прошивки Klipper.
* Если вы уверены, что основная плата уже прошита Katapult, то нет необходимости выполнять последнюю прошивку Katapult, это лишь на случай, если прошивка Katapult на плате не удалась.
* Можно быстро нажать кнопку сброса дважды и проверить, загорится ли светодиод на плате, если да, то плата уже находится в режиме Katapult.
:::

## Как определить, какую прошивку вам нужно

    * SB2040 PRO можно подключать к хост-машине следующими способами:
    * [CAN-прошивка](./can.mdx "Нажмите для перехода")
        * Плюсы: Если прошита прошивка `Katapult`, можно обновлять прошивку без физического контакта с платой, а также поддерживается подключение через шину.
        * Минусы: Возможны ошибки, такие как тайм-аут возврата в исходное положение при использовании с `KlipperScreen`.
        * Совет: Если вы планируете использовать CAN, убедитесь, что ваша хост-машина поддерживает CAN, и вам потребуется UTOC или плата с прошитой прошивкой моста.


## Описание прошивки Katapult

    * SB2040 PRO по умолчанию поставляется с прошивкой Katapult для CAN-соединения со скоростью 1М.
