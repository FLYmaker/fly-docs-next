# Пояснение по прошивке прошивки

    * SB2040 PRO можно подключать к верхнему компьютеру через CAN
    * Для подключения по CAN требуется использование кабеля `XT30(2+2)-F`
    * Ниже приведен вид кабеля `XT30(2+2)-F`
    <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Важно
* Прошивка прошивки для основной платы должна быть сделана через прошивку Katapult для Klipper
* Если вы уверены, что на основной плате уже установлена прошивка Katapult, то последнюю прошивку Katapult устанавливать не нужно, это только на случай, если прошивка Katapult на плате не удалась
* Можно быстро нажать кнопку сброса дважды и посмотреть, загорится ли светодиод на плате, если да, то это означает, что плата перешла в режим Katapult
:::

## Как определить, какую прошивку вам нужно

    * SB2040 PRO можно подключать к верхнему компьютеру следующими способами
    * [CAN прошивка](./can.mdx "Нажмите, чтобы перейти")
        * Плюсы: Если прошита прошивка `Katapult`, можно обновлять прошивку без необходимости касаться платы, а также поддерживается подключение по шине
        * Минусы: Возможны ошибки, такие как тайм-ауты при возврате в исходное положение при использовании вместе с `KlipperScreen`.
        * Подсказка: Если необходимо использовать CAN, убедитесь, что верхний компьютер поддерживает CAN, и используйте UTOC или плату с прошитой прошивкой моста


## Пояснение по прошивке Katapult

    * SB2040 PRO по умолчанию оснащен прошивкой Katapult для подключения по CAN со скоростью 1М
