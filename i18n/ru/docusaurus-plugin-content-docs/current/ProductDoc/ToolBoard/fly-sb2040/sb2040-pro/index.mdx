# Введение в продукт

    FLY SB2040 PRO - это инструментальная плата, выпущенная Guangzhou Mellow Electronics Co., Ltd для VORON Stealthburber принтера, которая позволяет заменить множество проводов между основной платой и принтера на четыре кабеля, оптимизируя расположение. SB2040 подходит для stealthburner и afterburner, интегрируя CAN-трансивер, USB-порт, драйвер TMC2240, акселерометр ADXL345, три управляемых вентилятора и другие функции. Эта плата подходит для прошивок Klipper и RRF.

## Характеристики продукта

    * Предусмотрена кнопка Boot, пользователи могут войти в режим Boot через TypeC для обновления прошивки
    * Поддержка прошивки Klipper
    * Поддержка температурных датчиков: встроенный чип MAX31865, совместимый с PT100
    * Встроенный акселерометр ADXL345
    * Предусмотрен интерфейс RGB, встроенный драйвер 2240, поддержка различных сенсоров выравнивания, таких как klicky, Voron Tap, PL08n и т.д.
    * Поддержка напряжения 12-24В, тока 15А, увеличенный порт для нагревательного элемента, номинальный ток 10А


## Параметры продукта

    * MCU: Raspberry Pi rp2040, Двухъядерный ARM Cortex-M0+@133MHz
    * Разъемы для подключения используют XT30(2+2) соединители, поддерживающие ток до 15А, пиковый ток до 30А
    * Прошивка: Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: ADXL345 и опционально MAX31865
    * Интерфейс нагревательного элемента: нагревательный элемент (HEAT0)
    * Расширенные интерфейсы: управляемые вентиляторы*3, RGB*1, концевые выключатели*3, нагревательная трубка*1, термистор*1
    * Интерфейсы вентиляторов: три управляемых вентилятора
    * Драйвер двигателя: встроенный TMC2240
