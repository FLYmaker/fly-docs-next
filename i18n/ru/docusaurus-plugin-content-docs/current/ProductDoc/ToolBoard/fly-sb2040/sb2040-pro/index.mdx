# Введение в продукт

    FLY SB2040 PRO - это плата инструментов, выпущенная компанией Guangzhou Mellow Electronic Technology Co., Ltd. для печатающей головки VORON Stealthburner. Использование этой платы позволяет заменить сложную проводку между основной платой и печатающей головкой всего четырьмя проводами, оптимизируя расположение. SB2040 подходит для stealthburner и afterburner, она интегрирует CAN-приемопередатчик, USB-порт, драйвер TMC2240, акселерометр ADXL345, три управляемых вентилятора и другие функции. Эта плата совместима с прошивками Klipper и RRF.

## Характеристики продукта

    * Предусмотрена кнопка загрузки, пользователь может войти в режим загрузки через TypeC для обновления прошивки
    * Поддержка прошивки Klipper
    * Поддержка температурных датчиков: встроенный чип MAX31865, совместим с PT100
    * Встроенный акселерометр ADXL345
    * Предусмотрен интерфейс для RGB, встроенный драйвер 2240, поддержка различных датчиков выравнивания, таких как klicky, Voron Tap, PL08n и т.д.
    * Поддержка напряжения 12-24V, тока 15A, увеличенный порт для нагревательного стержня с номинальным током 10A


## Параметры продукта

    * MCU: Raspberry Pi rp2040, двойное ядро ARM Cortex-M0+@133MHz
    * Интерфейс подключения использует разъем XT30(2+2), поддерживает ток 15A, пиковый ток 30A
    * Прошивка: Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: ADXL345 и опционально MAX31865
    * Интерфейс нагревательного стержня: нагревательный стержень (HEAT0)
    * Расширенные интерфейсы: управляемые вентиляторы*3, RGB*1, концевые выключатели*3, нагревательный элемент*1, термопара*1
    * Вентиляторные интерфейсы: три управляемых вентилятора
    * Драйвер двигателя: встроенный TMC2240
