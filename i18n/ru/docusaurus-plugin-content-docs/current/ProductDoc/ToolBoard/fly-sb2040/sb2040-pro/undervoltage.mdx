# Методы решения проблемы с недостаточным напряжением на драйвере

## Метод один

    :::danger

    * Заменить источник питания
    * Добавить фильтр питания
    * Заменить двигатель для проверки
    * Подключить корпус двигателя к GND на плате управления
    * Добавить в конфигурацию драйвера экструдера `driver_TPFD: 0`

    :::

## Метод два
    * Подключиться к хосту через SSH
    * Ввести следующую команду
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * После обновления Klipper эти изменения могут быть отменены
    :::
