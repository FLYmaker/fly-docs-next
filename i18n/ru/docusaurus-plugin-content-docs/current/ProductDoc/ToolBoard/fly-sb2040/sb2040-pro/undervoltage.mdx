# Методы решения проблемы с недостаточным напряжением на драйвере

## Метод один

    :::danger

    * Заменить источник питания
    * Добавить фильтр источника питания
    * Заменить двигатель для проверки
    * Подключить корпус двигателя к GND на плате инструмента
    * В настройках драйвера экструдера добавить `driver_TPFD: 0`

    :::

## Метод два
    * Подключиться к хост-машине через SSH
    * Ввести ниже указанную команду
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * После обновления klipper эти изменения могут быть отменены
    :::
