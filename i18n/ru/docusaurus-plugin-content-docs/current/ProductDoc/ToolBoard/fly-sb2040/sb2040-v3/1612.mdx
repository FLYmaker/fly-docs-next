# Использование Eddy

    :::info

    * Обратите внимание, что рекомендуется использовать отдельное Z-ограничение для сканирования кровати, не рекомендуется использовать сканируемую кровать в качестве Z-ограничения
    * При установке Eddy, нижняя часть должна находиться как минимум на 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть удалена от нагревательной платформы как минимум на 5 мм
    * Пожалуйста, убедитесь, что вся нагревательная платформа находится при комнатной температуре во время тестирования, и не включайте платформу во время тестирования

    :::

## Пример конфигурации и инструкции по использованию

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что в настоящее время нет внешнего модуля 1612, поэтому данная конфигурация пока не может быть использована
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация компенсации температуры

        ```cfg
        [temperature_probe fly_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Конфигурация bed_mesh

    * В разделе [bed_mesh] параметр `horizontal_move_z` рекомендуется установить на 2, чтобы Eddy был как можно ближе к платформе при сканировании

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
