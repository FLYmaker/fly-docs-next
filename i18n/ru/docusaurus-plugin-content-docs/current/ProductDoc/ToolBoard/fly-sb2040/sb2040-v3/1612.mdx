# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования кровати рекомендуется использовать отдельный Z-лимит, не рекомендуется использовать сканирование кровати в качестве Z-лимита
    * При установке Eddy, положение нижней части относительно сопла должно быть на 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от нагревательной кровати
    * Пожалуйста, не тестируйте всю нагревательную кровать при комнатной температуре, и не включайте нагревательную кровать во время тестирования

    :::

## Рекомендуемая конфигурация и инструкция по использованию

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что в настоящее время нет внешнего модуля 1612, эта конфигурация пока не работает
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация температурной компенсации

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Конфигурация bed_mesh

    * В разделе [bed_mesh] параметр `horizontal_move_z` рекомендуется установить на 2, чтобы Eddy был как можно ближе к платформе при сканировании

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
