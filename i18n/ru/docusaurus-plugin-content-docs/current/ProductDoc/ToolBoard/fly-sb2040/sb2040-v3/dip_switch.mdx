# Введение в бортовые переключатели

:::warning Внимание

Из-за высокого разрешения изображений, их невозможно рассмотреть при просмотре на веб-странице. Пожалуйста, **переместите курсор мыши на схему подключения**, **нажмите правую кнопку мыши**, выберите **Сохранить изображение как**, сохраните изображение на рабочий стол и затем просмотрите его!!!

:::
