# Введение в использование бортовых переключателей

:::warning Внимание

Из-за высокого разрешения изображений их просмотр на веб-странице затруднен. Пожалуйста, **наведите курсор мыши на схему подключения**, **нажмите правую кнопку мыши** и выберите **Сохранить изображение как**, чтобы сохранить изображение на рабочем столе для просмотра!!!

:::
