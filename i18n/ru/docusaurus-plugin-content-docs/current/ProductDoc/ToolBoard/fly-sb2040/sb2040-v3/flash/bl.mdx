# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SB2040_V3 поддерживает CAN и RS232
    * Здесь предоставляются только прошивки CAN на скорости 500K и 1M
    :::

## Вход в режим прошивки

    :::tip

    Слева кнопка `RESET`, справа кнопка `BOOT`

    :::

    * Есть два способа войти в режим прошивки
    * Первый: при полном отключении питания платы, удерживайте кнопку BOOT, затем подключите кабель данных к плате и устройству
    * Второй: после подключения кабеля данных к плате и устройству, удерживайте кнопку BOOT, затем нажмите и отпустите кнопку RST, после чего отпустите кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.png').default} size="100%" align="center" />
