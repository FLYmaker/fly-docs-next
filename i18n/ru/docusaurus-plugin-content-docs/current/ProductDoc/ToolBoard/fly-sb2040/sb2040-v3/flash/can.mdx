        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце вы увидите следующий вывод, компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошитой прошивкой моста, используя кабель XT30
    * Обратите внимание на положение переключателей DIP
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании сопротивление между CAN H и CAN L измерено мультиметром и составляет около `60Ω`. Если оно равно `120Ω`, включите переключатель `120Ω`. Если оно около `40Ω`, выключите переключатель
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.png').default} size="100%" align="center" />
