        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/rs232.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводится следующее, компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Вход в режим загрузки

    :::tip

    Слева находится кнопка `RESET`, справа - кнопка `BOOT`

    :::

    * Есть два способа войти в режим DFU
    * Первый способ: при полном отключении платы от питания, нажмите и удерживайте кнопку BOOT, затем подключите кабель с поддержкой передачи данных сначала к плате, затем к компьютеру
    * Второй способ: подключите кабель с поддержкой передачи данных сначала к плате, затем к компьютеру, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, затем отпустите кнопку RST и в последнюю очередь отпустите кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.png').default} size="100%" align="center" />
