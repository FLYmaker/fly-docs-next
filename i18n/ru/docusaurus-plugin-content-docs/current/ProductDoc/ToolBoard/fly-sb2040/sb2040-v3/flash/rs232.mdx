## Вход в режим прошивки

    :::tip

    Слева кнопка `RESET`, справа кнопка `BOOT`

    :::

    * Существует два способа войти в режим DFU:
    * Первый: при полностью отключенной плате, нажмите и удерживайте кнопку BOOT, затем подключите кабель передачи данных к плате и компьютеру
    * Второй: когда кабель передачи данных уже подключен к плате и компьютеру, нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RST, после отпустите кнопку RST и затем BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.png').default} size="100%" align="center" />
