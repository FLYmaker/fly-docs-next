# 1. Введение в продукт

    FLY SB2040 V3 - это плата инструментов, разработанная компанией Guangzhou Mellow Electronics Technology Co., Ltd. для печатающей головки VORON Stealthburner. Используя эту плату, можно заменить сложную проводку между основной платой и печатающей головкой на четыре провода, оптимизируя компоновку. SB2040 подходит для stealthburner и afterburner, она интегрирует CAN-передатчик, RS232-передатчик, USB-порт, драйвер TMC2209, акселерометр LIS2DW, три управляемых вентилятора и другие функции. Эта плата совместима с прошивками Klipper и RRF.

## 1.1 Особенности продукта

    * Резервная кнопка Boot, пользователь может через TypeC войти в режим Boot для обновления прошивки
    * Поддерживает прошивки Klipper и RRF
    * Поддерживает режимы CAN, RS232, RRF CAN,
    * В режиме RS232 имеет более высокую скорость передачи и большую стабильность. В режиме CAN поддерживается расширение CAN, оставлен интерфейс для расширения CAN
    * Поддержка температурных датчиков: на борту чип MAX31865, совместим с датчиками температуры PT100 и PT1000.
    * На борту акселерометр LIS2DW и датчик LDC1612
    * Резервный интерфейс RGB, на борту драйвер 2209, поддерживает различные датчики выравнивания, например, klicky, Voron Tap, PL08n и т.д.
    * Поддержка входного напряжения 12-24В, тока 15А, увеличен порт для нагревательного элемента, номинальный ток 10А

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232, RRF CAN
    * Добавлен режим RS232 с более высокой скоростью передачи, большей стабильностью и упрощенным использованием для отладки
    * Добавлен режим RRF CAN, позволяющий переключаться на RRF инструментальную плату, увеличивая универсальность
    * Добавлен интерфейс для расширения CAN, упрощающий расширение устройств CAN
    * Добавлен переключатель терминального сопротивления, терминальное сопротивление 120R можно выбрать с помощью переключателя, без необходимости использования перемычек
    * Добавлен 1-метровый кабель питания/сигнала, кабель выполнен в двойном витом исполнении для лучшей устойчивости к помехам

## 1.3 Параметры продукта


    * МК: Raspberry Pi rp2040, двойное ядро ARM Cortex-M0+@133MHz
    * Интерфейс подключения использует разъем XT30(2+2) для поддержки тока до 15A, пиковый ток до 30A
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * На борту датчики: LIS2DW и опционально ldc1612 и max31865
    * Интерфейс нагревателя: Нагреватель (HEAT0)
    * Интерфейсы расширения: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN
    * Интерфейсы вентиляторов: один DC вентилятор, два DC или PWM вентилятора
    * Драйвер двигателя: на борту TMC2209
    * Датчик температуры: один 100K NTC или PT1000 (TH0), один опциональный PT100/PT1000
