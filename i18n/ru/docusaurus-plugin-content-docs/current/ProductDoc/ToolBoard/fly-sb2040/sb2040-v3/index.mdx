# 1. Введение в продукт

    FLY SB2040 V3 - это инструментальная плата, разработанная компанией Guangzhou Mellow Electronic Technology Co., Ltd. для VORON Stealthburner принтера. Используя эту плату, можно заменить сложные провода между основной платой и принтером на четыре провода, оптимизировав тем самым расположение. SB2040 подходит для stealthburner и afterburner, она интегрирует CAN приемопередатчик, RS232 приемопередатчик, USB порт, TMC2209 драйвер, LIS2DW акселерометр, три управляемых вентилятора и другие функции. Эта плата совместима с прошивками Klipper и RRF.

## 1.1 Характеристики продукта

    * Имеется кнопка Boot, пользователь может через TypeC войти в режим Boot для обновления прошивки
    * Поддержка прошивок Klipper, RRF
    * Поддержка режимов CAN, RS232, RRF CAN,
    * В режиме RS232 обеспечивается более высокая скорость передачи данных и стабильность. В режиме CAN поддерживается расширение CAN, предусмотрен интерфейс для расширения CAN
    * Поддержка температурных датчиков: встроенный чип MAX31865, совместимый с датчиками PT100 и PT1000.
    * Встроенные датчики ускорения LIS2DW и LDC1612
    * Предусмотрен интерфейс RGB, встроенный драйвер 2209, поддержка различных датчиков выравнивания, таких как klicky, Voron Tap, PL08n и т.д.
    * Поддержка напряжения 12-24В, тока 15А, увеличенный порт для нагревательного элемента, номинальный ток 10А

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232 и RRF CAN
    * Новый режим RS232 с более высокой скоростью передачи данных и стабильностью, упрощает отладку
    * Новый режим RRF CAN, позволяет переключиться на инструментальную плату RRF, расширяя функциональность
    * Добавлен интерфейс расширения CAN для удобства подключения дополнительных устройств CAN
    * Добавлен переключатель терминального резистора, терминальное сопротивление 120R можно выбрать с помощью переключателя, без необходимости пайки
    * Добавлен 1-метровый кабель питания/сигнала, использующий двойную скрутку для повышения помехозащищенности

## 1.3 Параметры продукта

    * MCU: Raspberry Pi rp2040, двуядерный ARM Cortex-M0+@133MHz
    * Интерфейсы подключения используют разъемы XT30(2+2), поддерживающие ток 15А, пиковое значение 30А
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: LIS2DW и опционально ldc1612 и max31865
    * Интерфейс нагревательного элемента: HEAT0
    * Интерфейсы расширения: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN
    * Интерфейсы вентиляторов: один DC вентилятор, два DC или PWM вентилятора
    * Драйвер двигателя: встроенный TMC2209
    * Датчики температуры: один 100K NTC или PT1000 (TH0), один PT100/PT1000 по выбору
