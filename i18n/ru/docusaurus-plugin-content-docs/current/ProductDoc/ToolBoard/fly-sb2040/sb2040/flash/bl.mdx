            4. Прошивка

            :::tip

            Необходимо прошить только одну прошивку

            :::

            * Ниже приведена команда для прошивки прошивки на скорости 500К CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * Ниже приведена команда для прошивки прошивки на скорости 1М CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. Если появляется следующее сообщение, прошивка успешна

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />

            :::tip

            После завершения прошивки необходимо полностью отключить питание платы.

            :::
