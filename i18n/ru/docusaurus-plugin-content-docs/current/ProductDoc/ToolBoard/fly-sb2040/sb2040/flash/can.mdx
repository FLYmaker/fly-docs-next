# Компиляция прошивки CAN

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/can.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце вы увидите следующий вывод, если компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или к материнской плате с прошитой прошивкой моста, используя кабель XT30, ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.png').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании сопротивление между CAN H и CAN L измеряется мультиметром и находится примерно на уровне `60Ω`. Если оно равно `120Ω`, вам нужно подключить перемычку `120Ω`, если оно около `40Ω`, перемычку нужно удалить
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/120.png').default} size="50%" align="center" />
