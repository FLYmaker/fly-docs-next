# Инструкции по прошивке микропрограммы

    * SB2040 можно подключить к верхнему компьютеру через CAN
    * Для подключения CAN требуется использовать `XT30(2+2)-F`
    * Вот как выглядит кабель `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Обратите внимание
* Прошивка микропрограммы на плате требует использования Katapult для прошивки Klipper
* Если уверены, что на плате уже установлена микропрограмма Katapult, нет необходимости выполнять последнюю прошивку микропрограммы Katapult, это лишь на случай, если прошивка не удалась
* Можно быстро нажать кнопку сброса дважды, чтобы проверить, начнет ли мигать светодиод на плате, если да, то плата уже в режиме Katapult
:::

## Как определить, какую микропрограмму вам нужно

    * SB2040 может быть подключен к верхнему компьютеру только через CAN
    * <Button variant="contained" disableElevation href="./can">Прошивка микропрограммы CAN</Button>
        * Плюсы: Если прошита микропрограмма `Katapult`, можно обновлять микропрограмму без необходимости физического доступа к плате, и поддерживается подключение по шине
        * Минусы: Возможны ошибки, такие как таймауты при возврате на исходную позицию при использовании с `KlipperScreen`.
        * Совет: Если вы планируете использовать CAN, убедитесь, что верхний компьютер поддерживает CAN, и вам потребуется UTOC или плата с прошитым мостом
