## Введение в продукт

    FLY SB2040 - это плата инструментов для экструдера, выпущенная компанией Guangzhou Magnesium Electron Technology Co., Ltd. для принтера VORON Stealthburber. Использование этой платы позволяет заменить сложную проводку между основной платой и экструдером на четыре провода, оптимизируя расположение. SB2040 подходит для шаговых двигателей на 36 шагов, интегрирует CAN-приемопередатчик, USB-порт, драйвер TMC2209, акселерометр ADXL345, два управляемых вентилятора и другие функции. Эта плата подходит для прошивки Klipper.

    * Поддерживает подключение по CAN-шине, обеспечивая более стабильную передачу данных, меньшую задержку и более надежное соединение
    * Встроенный драйвер TMC2209 в режиме UART, может использоваться для экструдера
    * Встроенный акселерометр ADXL345
    * Поддерживает датчики выравнивания, такие как BLTOUCH, PL08N, интегрированные концевые выключатели XY
    * Поддерживает входное напряжение 12-24В 
    * MCU: Raspberry Pi rp2040, двухъядерный ARM Cortex-M0+@133MHz
    * Добавлена поддержка переключения между резисторами 4.7K (для термистора ntc) и 1K (для PT1000)
    * Интерфейс CAN использует XT30 (2+2), поддерживает ток до 15A, пиковое значение 30A
    * Добавлен встроенный резистор NTC100k для измерения температуры камеры
    * Добавлены INT1 и INT2 акселерометра ADXL345 к микроконтроллеру для отдельных пользователей для выравнивания
    * Поддерживает высоковольтный входной ограничитель
    * В комплекте идет кабель CAN длиной 3 метра, чтобы избежать проблем с потерей сигнала из-за неправильного обжима
    * Поддерживает вентиляторы на 2/3/4 провода
    * Раздельный дизайн, Stealthburner легко разбирается
    * Совместим с платой инструментов Stealthburner от hartk (не нужно менять маленькую плату вентилятора)
    * Интерфейсы: управляемые вентиляторы*3, RGB*1, концевые выключатели*3, нагревательный элемент*1, термистор*1

## Параметры продукта

    * Поддерживает переключение между режимами CAN Klipper, RS232, RRFCAN
    * Соединительный интерфейс использует XT30(2+2) для поддержки тока до 15A, пиковое значение 30A
    * MCU: Raspberry Pi rp2040, двухъядерный ARM Cortex-M0+@133MHz
    * Входное напряжение: DC12-24V
    * Максимальный выходной ток нагревательного элемента: 5A
    * Максимальный выходной ток интерфейса вентилятора: 1A
