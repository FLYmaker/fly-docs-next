            <Katapult />

            4. Прошивка

            :::tip

            Необходимо прошить только одну прошивку

            :::

            * Ниже приведена команда для прошивки прошивки на скорости 500K CAN:
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Ниже приведена команда для прошивки прошивки на скорости 1M CAN:
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. Если появится сообщение ``File downloaded successfully`` как показано на изображении ниже, то прошивка прошла успешно

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.png').default} size="100%" align="center" />

            :::tip

            После завершения прошивки необходимо полностью отключить питание платы.

            :::
