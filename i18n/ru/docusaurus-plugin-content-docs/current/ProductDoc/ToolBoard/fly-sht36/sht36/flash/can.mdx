# Компиляция прошивки CAN

## Начало компиляции

    <SSH />

    <CAN />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующее сообщение, значит компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

    * Перед загрузкой прошивки подключите плату инструмента к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30, ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании измерение сопротивления между CAN H и CAN L с помощью мультиметра составляет около `60Ω`. Если оно равно `120Ω`, необходимо подключить перемычку `120Ω`, если около `40Ω`, перемычку нужно удалить
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.webp').default} size="100%" align="center" />
