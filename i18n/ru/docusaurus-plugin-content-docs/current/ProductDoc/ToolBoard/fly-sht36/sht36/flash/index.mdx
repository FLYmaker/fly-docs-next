# Инструкции по прошивке прошивки

    * SHT36/42 можно подключать к верхнему уровню через CAN
    * Для CAN необходимо использовать металлические клеммы MX3.0
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />

## Как определить, какая прошивка вам нужна

    * SHT36/42 можно подключать к верхнему уровню через CAM
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN</Button>
        * Плюсы: Если установлена прошивка `Katapult`, можно обновлять прошивку без необходимости касаться материнской платы, а также поддерживается подключение по шине.
        * Минусы: Возможны ошибки, такие как тайм-аут при возврате домой при использовании вместе с `KlipperScreen`.
        * Подсказка: Если вам нужно использовать CAN, убедитесь, что ваш верхний уровень поддерживает CAN, и вам потребуется UTOC или материнская плата с прошивкой моста.
