# Введение в продукт

    FLY SHT36/42 - это головка инструмента, разработанная компанией Guangzhou Mellow Electronic Technology Co., Ltd. для экструдера 3D-принтера. Используя эту плату инструмента, можно заменить сложные соединения между основной платой и экструдером на четыре провода, оптимизировав размещение. SHT36 подходит для шагового двигателя 36, SHT42 подходит для шагового двигателя 42. Она интегрирует в себя приемопередатчик CAN, USB-порт, драйвер TMC2209, акселерометр ADXL345, два контролируемых вентилятора и другие функции. Эта плата инструмента подходит для прошивки Klipper.

    * Экстремально компактные размеры, используется 6-слойная конструкция платы
    * Поддерживает подключение через шину CAN, обеспечивая более стабильную передачу данных, меньшую задержку и стабильное соединение
    * На борту драйвер TMC2209, используемый в режиме UART, для экструдера
    * На борту акселерометр ADXL345
    * Поддерживает PT100 (MAX31865)
    * Возможность замены MOSFET вентилятора
    * Поддерживает датчики выравнивания BLTOUCH, PL08N и интегрированные концевые выключатели X и Y
    * Поддерживает входное напряжение 12-24V 
    * MCU: STM32F072
    * Связь: CANBUS, USB
    * Интерфейсы: контролируемые вентиляторы*2, RGB*1, концевые выключатели*3, Probe*1, Servo*1, нагревательный элемент*1, терморезистор*1, PT100*1

## Параметры продукта

    1. Габариты: 51.27mm*36.26mm (SHT36), 42mm*42mm (SHT42)
    2. Размеры для установки: расстояние между отверстиями 43.84mm, диаметр отверстий под крепеж M3
    3. Процессор: STM32F072C8T6
    4. Входное напряжение: DC12-24V 5A
    5. Максимальный выходной ток нагревательного элемента: 5A
    6. Максимальный выходной ток на вентилятор: 1A
