# Использование акселерометра


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Обновление Klipper

    * Klipper поддерживает акселерометр LIS3DH с 13 ноября 2024 года. Для использования акселерометра LIS3DH необходимо обновить Klipper.
    * После подключения к хост-машине через сеть введите следующий код (обновление может не удаться из-за проблем с сетью):
        ```bash
        cd ~/klipper 
        git pull
        ```
## Добавление конфигурации


    ```cfg
    #####################################################################
    #                       Конфигурация акселерометра LIS3DH (включить при необходимости)
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  # Здесь установите координаты, где будет находиться экструдер при измерении резонанса


```
