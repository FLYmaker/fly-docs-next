    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: !gpio20 Обратите внимание, что ! - заглавная буква
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/can.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце вывода отображается следующее, то компиляция завершилась успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

    * Перед загрузкой прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30, ниже приведена схема подключения
    * Обратите внимание на положение переключателей
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании сопротивление между CAN H и CAN L измеряется мультиметром и находится в районе `60Ω`, если оно равно `120Ω`, подключите перемычку на `120Ω`, если около `40Ω`, удалите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/120.png').default} size="50%" align="center" />
