# Инструкции по прошивке

    * SHT36 LIS3DH может подключаться к ПК только через CAN
    * Для подключения CAN требуется использовать `XT30(2+2)-F`
    * Ниже представлен образец кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Важно
* Прошивка материнской платы требует использования Katapult для прошивки Klipper
* Если вы уверены, что на плате уже установлена прошивка Katapult, то нет необходимости выполнять последний шаг по прошивке Katapult, это только на случай, если прошивка Katapult на плате не удалась
* Вы можете быстро дважды нажать кнопку сброса и посмотреть, загорится ли светодиод на плате, если он загорится, это значит, что плата перешла в режим Katapult
:::

## Как определить, какую прошивку вам нужно

    * SHT36 LIS3DH может подключаться к ПК следующим образом
    * <Button variant="contained" disableElevation href="./can">Прошивка по CAN</Button>
        * Преимущества: если установлена прошивка `Katapult`, можно обновлять прошивку без физического доступа к плате, а также поддерживается подключение по шине
        * Недостатки: возможны ошибки, связанные с тайм-аутами при использовании `KlipperScreen`
        * Совет: если вы планируете использовать CAN, убедитесь, что ваше ПК поддерживает CAN и имеет подходящую плату UTOC или плату с прошивкой моста
