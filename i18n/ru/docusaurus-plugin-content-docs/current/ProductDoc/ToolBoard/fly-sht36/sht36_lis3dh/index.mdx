# Введение в продукт

    FLY-SHT36-LIS3DH - это экструдерная плата, разработанная компанией Guangzhou Mellow Electronics для 3D-принтеров. Использование этой платы позволяет заменить сложные соединения между основной платой и экструдером всего четырьмя проводами, оптимизируя компоновку. FLY-SHT36-LIS3DH подходит для 36-шаговых двигателей и включает в себя CAN-трансивер, USB-порт, драйвер TMC2209, акселерометр LIS3DH, два управляемых вентилятора и другие функции. Эта плата совместима с прошивкой Klipper.

    * Экстремально компактный дизайн, используется 6-слойная плата
    * Поддержка подключения по CAN-шине, более стабильная передача данных, меньшая задержка, более стабильное соединение
    * Встроенный драйвер TMC2209, использует режим UART, может быть использован для экструдера
    * Встроенный акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Заменяемый дизайн MOSFET для вентиляторов
    * Поддержка датчиков выравнивания BLTOUCH, PL08N и интеграция с концевыми выключателями XY
    * Поддержка входного напряжения 12-24В
    * MCU: STM32F072
    * Связь: CANBUS, USB
