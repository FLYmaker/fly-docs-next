# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SHT36 PRO поддерживает CAN
    * Здесь предоставляются только прошивки CAN на скорости 500K и 1M
    :::

## Вход в режим прошивки

    :::tip

    В кружке находится кнопка `BOOT`

    :::

    * Полностью отключите питание платы, закоротите кнопку BOOT, затем подключите кабель данных к плате и вашему устройству
    * Затем отпустите кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.png').default} size="100%" align="center" />

            4. Прошивка

            :::tip

            Нужно прошить только одну прошивку

            :::

            * Ниже приведена команда для прошивки прошивки на скорости CAN 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Ниже приведена команда для прошивки прошивки на скорости CAN 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Если появляется сообщение ниже, значит прошивка успешно завершена

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.png').default} size="100%" align="center" />
