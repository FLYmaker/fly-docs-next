# Компиляция CAN прошивки

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: !gpio13 Обратите внимание, что ! должно быть заглавным
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/can.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводятся следующие строки, то компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка прошивки

    * Перед прошивкой прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля данных XT30, ниже приведена схема подключения
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L при отключенном питании составляет около `60Ω`. Если сопротивление `120Ω`, то необходимо подключить перемычку `120Ω`, если сопротивление около `40Ω`, то перемычку нужно удалить
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/120.png').default} size="100%" align="center" />
