# Инструкции по прошивке прошивки

    * SHT36 PRO можно подключить к верхнему компьютеру через CAN
    * Для подключения CAN необходимо использовать `XT30(2+2)-F`
    * Ниже представлен образец кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Важно
* Для прошивки прошивки на плату необходимо использовать Katapult для прошивки Klipper
* Если плата уже прошита Katapult, то нет необходимости выполнять последний шаг по прошивке Katapult, это только на случай, если прошивка Katapult на плате не удалась
* Можно быстро нажать кнопку сброса два раза, чтобы проверить, загорится ли светодиод на плате, если да, то это означает, что плата находится в режиме Katapult
:::

## Компиляция прошивки CAN

    * SHT36 PRO может подключаться к верхнему компьютеру только через CAN
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN</Button>
        * Плюсы: Если прошита прошивка `Katapult`, можно обновлять прошивку без необходимости физического доступа к плате, а также поддерживает подключение через шину
        * Минусы: Возможны ошибки, такие как тайм-аут при гоминге при использовании с `KlipperScreen`
        * Совет: Если планируется использовать CAN, убедитесь, что верхний компьютер поддерживает CAN, и используйте UTOC или плату с прошитой прошивкой моста
