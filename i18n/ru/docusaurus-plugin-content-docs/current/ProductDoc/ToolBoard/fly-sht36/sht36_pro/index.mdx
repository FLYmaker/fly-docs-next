# Введение в продукт

    FLY-SHT36-PRO - это инструментальная плата для экструдера 3D-принтера, разработанная компанией Guangzhou Mellow Electronic Technology Co., Ltd. С помощью этой платы можно заменить сложную проводку между основной платой и печатающей головкой на четыре провода, оптимизировав расположение. FLY-SHT36-PRO подходит для шаговых двигателей на 36 шагов, она включает в себя CAN-приемопередатчик, USB-порт, драйвер TMC5160, акселерометр ADXL345, два управляемых вентилятора и другие функции.

    * Максимально компактный размер, используется шестислойная конструкция платы
    * Поддержка подключения по CAN-шине для более стабильной передачи данных, меньшей задержки и более стабильного соединения
    * На борту драйвер TMC5160 для экструдера
    * Встроенный акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Возможность замены вентиляторов с помощью MOSFET
    * Поддержка датчиков выравнивания BLTOUCH, PL08N, интегрированные концевые выключатели XY
    * Поддержка входного напряжения 12-24В
    * MCU: STM32F072
    * Связь: CANBUS, USB

## Параметры продукта

    1. Поддержка переключения между режимами Klipper CAN, RS232, RRFCAN
    2. Интерфейс подключения использует коннектор XT30(2+2), поддерживающий ток 15A, пиковое значение 30A
    3. MCU: Raspberry Pi rp2040, двуядерный ARM Cortex-M0+@133MHz
    4. Входное напряжение: DC12-24V 5A
    5. Максимальный выходной ток нагревателя: 5A
    6. Максимальный выходной ток для вентиляторов: 1A
