# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SHT36 V2 может использоваться только с подключением CAN
    * Здесь предоставляются прошивки CAN с частотами 500K и 1M

    :::


    :::warning Внимание

    * APMF072 можно прошивать только с помощью компьютера
    * STM32F103 можно обновлять прошивку только с помощью верхнего уровня

    :::


## Вход в режим прошивки

    :::tip

    В круге находится переключатель BOOT

    :::

    * STM32F103 не требует выполнения этого шага!!!
    * При полностью отключенной плате, нажмите и удерживайте кнопку BOOT, затем подключите кабель данных к плате и к вашему устройству

            4. Прошивка прошивки

            :::tip

            Необходимо прошить только один файл прошивки

            :::

            * Ниже приведена команда для прошивки прошивки с частотой CAN 500K
            ```bash
            sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36V2_072_500K.BIN
            ```
                       
            * Ниже приведена команда для прошивки прошивки с частотой CAN 1M
            ```bash
            sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36V2_072_1M.BIN
            ```


            5. Если появится сообщение, как указано стрелкой на изображении ниже, ``File downloaded successfully``, прошивка выполнена успешно

            <LS />

            * Скачайте заранее скомпилированную прошивку
            * Здесь собраны заранее скомпилированные прошивки
            * Обратите внимание, что для установки необходимо подключение к интернету
            * На верхнем уровне необходимо выполнить следующие команды для загрузки пакета прошивки

            ```bash
            cd && git clone https://github.com/FengHua741/FLY_Katapult.git
            ```

            * Если вы уже скачали пакет прошивки, рекомендуется обновить его перед выполнением оставшихся шагов

            ```bash
            cd ~/FLY_Katapult && git pull
            ```

            * Этот метод позволяет напрямую обновить прошивку Katapult на плате через CAN
            * Необходимо заменить `<Toolboard ID>` на CAN ID платы управления STM32F103
            * Ниже приведена команда для обновления прошивки с частотой CAN 1M

            ```bash
            ~/klippy-env/bin/python ~/klipper/lib/canboot/flash_can.py -f ~/FLY_Katapult/developer/SHT36/FLY_SHT36V2_103_1M.BIN  -u <Toolboard ID>
            ```

            * Ниже приведена команда для обновления прошивки с частотой CAN 500K
            ```bash
            ~/klippy-env/bin/python ~/klipper/lib/canboot/flash_can.py -f ~/FLY_Katapult/developer/SHT36/FLY_SHT36V2_103_500K.BIN  -u <Toolboard ID>
            ```
