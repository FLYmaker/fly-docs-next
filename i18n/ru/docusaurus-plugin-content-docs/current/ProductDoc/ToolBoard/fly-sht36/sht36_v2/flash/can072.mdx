# Компиляция прошивки CAN версии 072

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO pins to set at micro-controller startup**, введите: !PC13 Обратите внимание, что это заглавные буквы
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/072.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4`, чтобы начать компиляцию. Это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце выводится следующий текст, компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка

    * Перед прошивкой подключите плату инструментов к UTOC или материнской плате с прошитым мостом с помощью кабеля данных XT30, ниже приведена схема подключения
    * Обратите внимание на положение переключателей
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании сопротивление между CAN H и CAN L составляет около `60Ω`. Если оно равно `120Ω`, подключите перемычку `120Ω`, если около `40Ω`, удалите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.png').default} size="50%" align="center" />
