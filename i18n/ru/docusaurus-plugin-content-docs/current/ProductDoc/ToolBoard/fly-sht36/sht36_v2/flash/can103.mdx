# Компиляция прошивки CAN версии 103

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO pins to set at micro-controller startup**, введите: !PC13 Обратите внимание, что это заглавные буквы
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/103.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце вы увидите следующий вывод, что означает успешную компиляцию

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка прошивки

    * Перед прошивкой прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30, ниже приведена схема подключения
    * Обратите внимание на положение переключателя
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.png').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L при отключенном питании измеряется мультиметром около `60Ω`. Если сопротивление равно `120Ω`, подключите перемычку `120Ω`, если около `40Ω`, перемычку нужно снять
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.png').default} size="50%" align="center" />
