# Компиляция CAN прошивки версии 103

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO pins to set at micro-controller startup**, введите: !PC13 Обратите внимание, что это заглавные буквы
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/103.png').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * В конце выводится следующее сообщение, если компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка устройства

    * Перед прошивкой прошивки подключите плату инструментов к UTOC или материнской плате с прошитой прошивкой моста с помощью кабеля XT30, ниже приведена схема подключения
    * Обратите внимание, правильно ли установлен переключатель
        <ImageView image={require('@site/docs/General/get-id/img/utoc.png').default} size="80%" align="left" />
    * Убедитесь, что в отключенном состоянии сопротивление между CAN H и CAN L измеряется мультиметром и составляет около `60Ω`. Если сопротивление составляет `120Ω`, подключите перемычку `120Ω`. Если сопротивление составляет около `40Ω`, удалите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.png').default} size="50%" align="center" />
