# Инструкции по прошивке

    * Для компиляции и прошивки прошивки необходимо использовать ПК
    * SHT36 V2 может подключаться к ПК через CAN
    * CAN необходимо использовать с `XT30(2+2)-F`
    * Ниже представлен пример кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Обратите внимание
* Прошивка основной платы требует прошивки Klipper через Katapult
* Если вы уверены, что основная плата уже прошита Katapult, то нет необходимости прошивать Katapult в конце, это только на случай, если прошивка Katapult на плате не удалась
* Можно быстро нажать кнопку сброса дважды, чтобы проверить, не начнет ли мигать светодиод на плате, это будет означать, что плата находится в режиме Katapult
:::

## Компиляция CAN прошивки

    * SHT36 V2 может подключаться к ПК только через CAN
    * STM 103 смотрите <Button variant="contained" disableElevation href="./can103">103CAN прошивка</Button>
    * APM 072 и STM 072 смотрите <Button variant="contained" disableElevation href="./can072">072CAN прошивка</Button>
