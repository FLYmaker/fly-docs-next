# Инструкция по прошивке прошивки

    * Для компиляции и прошивки прошивки необходимо использовать ПО для верхнего уровня
    * SHT36 V2 может подключаться к ПО через CAN
    * Для подключения CAN используйте `XT30(2+2)-F`
    * Ниже приведен пример кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Обратите внимание
* Прошивка основной платы требует прошивки Klipper через Katapult
* Если основная плата уже прошита Katapult, нет необходимости повторно прошивать, это предусмотрено на случай, если прошивка Katapult не удалась
* Можно быстро нажать кнопку сброса дважды и посмотреть, загорится ли светодиод на плате, что будет означать, что плата перешла в режим Katapult
:::
