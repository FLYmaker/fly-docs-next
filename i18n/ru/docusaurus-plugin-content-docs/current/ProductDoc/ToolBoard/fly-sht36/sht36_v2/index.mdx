# Введение

## Введение в продукт

    FLY SHT36 V2 - это разработанная компанией Guangzhou Magnesium Electronic Technology Co., Ltd. инструментальная плата для экструдера 3D-принтера. Используя эту плату, можно заменить сложные соединения между основной платой и экструдером на четыре провода, оптимизируя расположение. SHT36 V2 подходит для шаговых двигателей на 36 шагов, она интегрирует приемопередатчик CAN, USB-порт, драйвер TMC2209, акселерометр ADXL345, два управляемых вентилятора и другие функции. Эта плата совместима с прошивкой Klipper.

    * Миниатюрный размер, использована 6-слойная плата
    * Поддержка подключения по шине CAN, более стабильная передача данных, меньшая задержка, более стабильное соединение
    * Встроенный драйвер TMC2209, работающий в режиме UART, подходит для экструдера
    * Встроенный акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Возможность замены вентилятора с MOS-дизайном
    * Поддержка датчиков выравнивания, таких как BLTOUCH, PL08N, встроенные концевые выключатели XY
    * Поддержка входного напряжения 12-24В 
    * MCU: STM32F072
    * Коммуникация: CAN
    * Поддержка датчиков выравнивания, таких как BLTOUCH, PL08N, встроенные концевые выключатели XY

## Новые функции

    * Добавлена поддержка переключения между резистором 4.7K (для NTC термистора) и резистором 1.1K (для PT1000)
    * CAN-интерфейс больше не использует разъем MX3.0, новый разъем поддерживает ток до 15А, пиковое значение 30А
    * Добавлена возможность выбора напряжения вентилятора 5/12/24В.
    * Добавлен встроенный резистор NTC100k для измерения температуры внутри корпуса
    * Добавлена защита от обратного подключения питания
    * Добавлен вывод INT1 акселерометра ADXL345 к микроконтроллеру для некоторых пользователей, использующих его для выравнивания.
    * Увеличен порт для нагревательного элемента, номинальный ток 10А
    * Добавлен ограничитель высокого входного напряжения
    * Заменен основной контроллер на отечественный MCU
    * Поддержка магнитного энкодера (as5047d), Klipper использует его для анализа движения (на данный момент Klipper не поддерживает)
    * Добавлен CAN-кабель длиной 1 метр, чтобы избежать проблем с потерей сигнала из-за неправильной обжимки пользователями
    * Размеры установки: расстояние между отверстиями 43.84 мм, диаметр отверстий M3
    * Процессор: GD32F103 (до 28 октября 2022 года), APM32F072 (после 28 октября 2022 года)
