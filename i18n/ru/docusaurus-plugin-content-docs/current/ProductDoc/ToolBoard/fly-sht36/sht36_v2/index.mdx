# Введение

## Введение в продукт

    FLY SHT36 V2 - это инструментальная плата для экструдера 3D-принтера, разработанная компанией Guangzhou Magnesium Electronic Technology Co., Ltd. Использование этой платы позволяет заменить сложные соединения между основной платой и экструдером на четыре провода, оптимизируя размещение. SHT36 V2 подходит для шаговых двигателей на 36 шагов, она интегрирует в себя приемник-передатчик CAN, USB-порт, драйвер TMC2209, акселерометр ADXL345, два управляемых вентилятора и другие функции. Эта плата совместима с прошивкой Klipper.

    * Экстремально компактный дизайн с использованием 6-слойной платы
    * Поддержка подключения по CAN-шине для более стабильной передачи данных, меньшей задержки и более надежного соединения
    * Встроенный драйвер TMC2209, работающий в режиме UART, подходит для экструдера
    * Встроенный акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Заменяемый дизайн MOSFET для вентилятора
    * Поддержка датчиков выравнивания, таких как BLTOUCH, PL08N, интегрированный XY концевой выключатель
    * Поддержка входного напряжения 12-24В 
    * MCU: STM32F072
    * Коммуникация: CAN
    * Поддержка датчиков выравнивания, таких как BLTOUCH, PL08N, интегрированный XY концевой выключатель

## Новые функции

    * Добавлена поддержка переключения между 4.7K резистором (для NTC термистора) или 1.1K резистором (для PT1000)
    * Интерфейс CAN больше не использует терминал MX3.0, новый разъем поддерживает ток до 15A, пиковый - 30A
    * Добавлена возможность выбора напряжения вентилятора 5/12/24V.
    * Добавлен встроенный NTC100k резистор для измерения температуры в камере
    * Добавлена защита от обратной полярности питания
    * Добавлено подключение INT1 акселерометра ADXL345 к MCU для выравнивания
    * Увеличен порт нагревательного элемента, номинальный ток 10A
    * Добавлен ограничитель высокого входного напряжения
    * Заменен главный контроллер на отечественный MCU
    * Поддержка магнитного энкодера (as5047d), используемого Klipper для анализа движения (пока Klipper не поддерживает)
    * Добавлен кабель CAN длиной 1 метр, чтобы избежать проблем с потерей сигнала из-за неправильного обжима коннекторов
    * Размеры установки: расстояние между отверстиями 43.84мм, диаметр отверстий M3
    * Процессор: GD32F103 (до 28 октября 2022 года), APM32F072 (после 28 октября 2022 года)
