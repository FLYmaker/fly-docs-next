# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования кровати рекомендуется использовать отдельный Z-концевик, не рекомендуется использовать сканирование кровати в качестве Z-концевика
    * При установке Eddy, его нижняя часть должна находиться минимум на 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от термостола
    * Пожалуйста, не проводите тесты при включенном термостоле, и убедитесь, что термостол находится при комнатной температуре

    :::

## Пример конфигурации и инструкция по использованию

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что в настоящее время внешний модуль 1612 не выпущен, данная конфигурация временно не может быть использована
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 0.8
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация температурной компенсации

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### Конфигурация bed_mesh

    * В параметре `horizontal_move_z` в [bed_mesh], рекомендуется установить значение 2, чтобы при сканировании платформы Eddy был как можно ближе к платформе

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
