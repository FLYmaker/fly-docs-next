# Введение в набор переключателей

:::warning Внимание

Из-за высокого разрешения изображений, на веб-страницах их трудно рассмотреть. Пожалуйста, поместите **курсор мыши над схемой подключения**, **щелкните правой кнопкой мыши** и выберите **Сохранить изображение как**, чтобы сохранить изображение на рабочий стол и просмотреть его там!!!

:::
