# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * SHT36_V3 поддерживает CAN и RS232
    * Здесь предоставляются только прошивки CAN с скоростью 500K и 1M
    :::

## Вход в режим прошивки

    :::tip

    Стрелка указывает на кнопку `BOOT`

    :::

    * При полном отключении питания платы, нажмите и удерживайте кнопку BOOT, затем подключите кабель для передачи данных к плате и вашему устройству

            4. Прошивка

            :::tip

            Необходимо прошить только один из файлов прошивки

            :::

            * Команда для прошивки прошивки со скоростью CAN 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Команда для прошивки прошивки со скоростью CAN 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. Если появится следующий экран, прошивка прошла успешно
