    <CAN />

    * Выберите GPIO pins to set at micro-controller startup, введите: !gpio5 Обратите внимание, что ! - это заглавная буква
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * При успешной компиляции в конце будет выведено следующее сообщение

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка микропрограмм
         
    * Перед прошивкой микропрограмм подключите плату инструментов к UTOC или материнской плате с прошитой микропрограммой моста с помощью кабеля XT30, ниже приведена схема подключения
    * Обратите внимание на положение переключателей
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что при отключении питания сопротивление между CAN H и CAN L составляет около `60Ω`. Если оно равно `120Ω`, необходимо подключить перемычку `120Ω`. Если сопротивление около `40Ω`, перемычку следует удалить
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/120.webp').default} size="100%" align="center" />
