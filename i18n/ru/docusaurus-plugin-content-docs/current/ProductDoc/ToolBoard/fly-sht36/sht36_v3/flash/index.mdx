# Инструкции по прошивке прошивки
    
    * Для компиляции и прошивки прошивки необходимо использовать ПК
    * SHT36 V3 можно подключить к ПК через RS232 или CAN
    * Для подключения RS232 и CAN необходимо использовать `XT30(2+2)-F`
    * Ниже представлен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.png').default} size="80%" align="left" />

:::danger Внимание
* Для прошивки прошивки на плате необходимо использовать Katapult для прошивки Klipper
* Если уверены, что на плате уже прошит Katapult, то нет необходимости в последнем шаге прошивки Katapult, это только на случай, если прошивка Katapult не удалась
* Можно быстро нажать кнопку сброса дважды и проверить, загорится ли светодиод на плате, если да, то это означает, что плата уже в режиме Katapult
:::

## Как определить, какая прошивка вам нужна

    * SHT36 V3 может подключаться к ПК следующим образом
    * <Button variant="contained" disableElevation href="./can">CAN прошивка</Button>
        * Плюсы: Если прошить прошивку `Katapult`, можно обновлять прошивку без физического доступа к плате, поддержка подключения по шине
        * Минусы: Возможны ошибки, такие как таймауты при калибровке при использовании `KlipperScreen`.
        * Подсказка: Для использования CAN убедитесь, что ПК поддерживает CAN, и потребуется UTOC или плата с прошивкой моста
    * <Button variant="contained" disableElevation href="./rs232">RS232 прошивка</Button>
        * Плюсы: Не требует специального ПО, надежная и стабильная связь
        * Минусы: Для обновления прошивки необходимо вручную войти в режим DFU или дважды нажать RST для входа в режим `Katapult`
        * Подсказка: Необходимо использовать G2T или UTOR
