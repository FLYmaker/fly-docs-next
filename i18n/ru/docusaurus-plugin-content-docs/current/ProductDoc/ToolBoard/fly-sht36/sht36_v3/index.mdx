# 1. Введение в продукт

FLY SHT36 V3 - это инструментальная плата для экструдера 3D-принтера, разработанная компанией Guangzhou Mellow Electronics Technology Co., Ltd. Использование этой платы позволяет заменить сложные соединения между основной платой и экструдером на четыре провода, оптимизируя расположение. SHT36 V3 подходит для шаговых двигателей с 36 шагами, в нее интегрированы CAN-приемопередатчик, RS232-приемопередатчик, TypeC порт, TMC2209 драйвер, LIS2DW акселерометр, 2 управляемых вентилятора и другие функции. Эта плата совместима с прошивками Klipper и RRF.

## 1.1 Характеристики продукта

    * Имеется кнопка Boot, пользователи могут через TypeC войти в режим Boot для обновления прошивки
    * Поддерживает прошивки Klipper, RRF
    * Поддерживает режимы CAN, RS232, RRF CAN
    * В режиме RS232 обеспечивает более высокую скорость передачи данных и стабильность. В режиме CAN поддерживает расширение CAN с предусмотренным интерфейсом для расширения CAN
    * Поддержка температурных датчиков: встроенный чип MAX31865, совместим с датчиками PT100 и PT1000.
    * Поддержка двух управляемых вентиляторов с возможностью выбора напряжения, заменяемые Mos модули для удобства обслуживания и замены, максимальный выходной ток до 1A
    * Встроенные акселерометр LIS2DW и датчик LDC1612
    * Предусмотрен интерфейс для RGB, встроенный драйвер 2209, поддержка различных датчиков для выравнивания, таких как BLtouch, klicky, Voron Tap, PL08n и т.д.
    * Поддержка напряжения 12-24В, тока 15А, увеличенный порт для нагревательного элемента, номинальный ток 10А

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232, RRF CAN
    * Добавлен режим RS232 для более высокой скорости передачи данных и стабильности, упрощающий отладку
    * Добавлен режим RRF CAN, позволяющий переключаться на инструментальную плату RRF, увеличивая совместимость
    * Добавлен интерфейс для расширения CAN, облегчающий подключение дополнительных устройств CAN
    * Добавлен переключатель для терминального резистора, терминальный резистор 120R можно выбрать с помощью переключателя, без необходимости использования перемычек
    * Добавлен 1-метровый кабель питания/сигнала с двойной скруткой для повышения устойчивости к помехам

## 1.3 Параметры продукта

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Используются соединители XT30(2+2) для подключения, поддерживающие ток 15A, пиковый ток 30A
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: LIS2DW и LDC1612
    * Интерфейс для нагревательного элемента: HEAT0, 10A
    * Расширенные интерфейсы: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN расширения
    * Вентиляторные интерфейсы: два DC управляемых вентилятора
    * Драйвер двигателя: встроенный TMC2209
    * Температурные датчики: один 100K NTC или PT1000 (TH0), один PT100/PT1000 по выбору
