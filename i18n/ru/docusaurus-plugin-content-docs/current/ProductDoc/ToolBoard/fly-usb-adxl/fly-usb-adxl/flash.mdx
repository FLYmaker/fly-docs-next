# Компиляция USB прошивки

    :::tip

    * USB акселерометр по умолчанию предварительно прошит с прошивкой Katapult для подключения по USB
    * Если прошивки нет, можно прошить самостоятельно
    * [Прошивка прошивки Katapult](./bl.mdx "Нажмите для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.png').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы выйдете в интерфейс командной строки
    * Введите `make -j4` для начала компиляции, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.png').default} size="50%" align="center" />
    * Если в конце будет выводиться следующий текст, то компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
