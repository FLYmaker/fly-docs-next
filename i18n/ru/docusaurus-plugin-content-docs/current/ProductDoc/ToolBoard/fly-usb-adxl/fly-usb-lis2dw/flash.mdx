# Компиляция прошивки USB

    :::tip

    * USB-акселерометры изначально прошиты прошивкой Katapult для USB-соединения
    * Если вы обнаружите, что прошивка отсутствует, вы можете загрузить её самостоятельно
    * [Загрузка прошивки Katapult](./bl.mdx "Нажмите для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />
    * Выберите GPIO pins to set at micro-controller startup, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/usb.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вы увидите следующий вывод, что означает успешную компиляцию

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
