# Загрузка системы {props.name}

:::warning

Внимание к SD-карте:

* Объем должен быть не менее `16GB` и не более `128GB` TF-карты
* Скорость передачи данных TF-карты должна быть `class10` или выше, рекомендуется использовать TF-карты бренда SanDisk и т.п.

:::

:::danger Важно знать

* Из-за проблем с сервером, пожалуйста, не обновляйте систему или Klipper без необходимости
* Пожалуйста, загрузите систему FLY OS, оригинальная версия Armbian сложна в использовании, установка не рекомендуется

:::

:::info Важное уведомление

* Обратите внимание, что загруженную систему нужно распаковать, а не просто изменить расширение или записать напрямую!!!
* Запись без распаковки не позволит системе загрузиться!!!

:::
