# Журнал обновлений системы

    :::info

    * В случае возникновения проблем в процессе использования системы, пожалуйста, сообщите в группу поддержки

    <Tabs>
        <TabItem value="H5" label="PI,PI V2,Gemini，C8 системы обновления">

            ## v2
                * 2022-1
                * Замена версии выпуска на Bullseye
                * Обновление ядра до 5.10.85
                * Предустановка Klipper, Moonraker, Fluidd, Mainsail, KlipperScreen
                * Klipper использует окружение Python3, решена проблема с именами файлов на китайском
                * Изменение раздела загрузки на FAT, позволяет напрямую изменять конфигурационные файлы в системе Windows
                * Добавление FLY-Config, новая система конфигурации
                * Добавление CatSerial, доступ по ```http://ip:9999``` позволяет напрямую просматривать все подключенные последовательные устройства

            ## v2.1
                * Исправление ошибки CatSerial без подключенных устройств

            ## v2.2
                * Добавление прошивки BL для серии плат Gemini, документация по прошивке смотрите

            ## v2.4
                * Исправление проблемы отсутствия некоторых сред для прошивки Klipper
                * Изменение шаблонов конфигурации
                * Добавление поддержки камеры mJpg
                * Добавление конфигурации пинов для плат V1 и V1.1. Переключение в FLY_Config

            ## v2.9
                * Поддержка Gemini v2
                * Улучшение поддержки CAN
                * Plug-and-Play для устройств CAN
                * Изменение модуля ядра gs_usb
                * Исправление проблемы невозможности загрузки после обновления ядра через apt update/apt upgrade

            ## v2.9.3
                * Адаптация для FLY-Π
                * Предустановка источников ядра и заголовочных файлов
                * Обновление окружения
                * Добавление публичного ключа для частного APT-репозитория
                * Установка правил PolicyKit для Moonraker
                * Исправление конфликтов правил udev
                * Исправление ошибок при прошивке Klipper из-за отсутствия библиотеки libusb
                * Исправление проблемы с загрузкой статических ресурсов FLY-Tools
                * Изменение модели платы по умолчанию на Gemini v2
                * Изменение по умолчанию serial в printer.cfg на /dev/ttyACM0
                * Изменение размера буфера отправки CAN на 1024

            ## v2.9.4
                * Отмена предустановки источников ядра и заголовочных файлов (будет распространяться в виде пакетов deb)
                * Исправление проблемы с невозможностью монтирования драйвера RTL88XXau

            ## v2.9.5
                * Добавление управления Wi-Fi сетями в KlipperScreen
                * Исправление других проблем

            ## v2.9.6
                * Поддержка модулей EMMC и WIFI
                * Отказ от использования mjpg-streamer, переход на crowsnest
                * Исправление других проблем

            ## v2.9.7
                * Исправление проблемы с невозможностью запуска KlipperScreen
                * Оптимизация драйвера Wi-Fi (решены проблемы с подключением, разрывом связи, нестабильностью и т.д.)
                * Оптимизация файловой системы

            ## v3
                * Поддержка платы FLY-Gemini v3
                * Добавление скрипта автоматической прошивки для платы Gemini v3

            ## v3.1
                * Поддержка платы FLY-Pi-V2
                * Обновление FLY-Tools
                * Скорость CAN по умолчанию 1M

            ## v3.1.1
                * Исправление автоматического расширения

            ## v3.3.3
                * Обновление FLY Tools
                * Обновление окружения
                * Обновление FLY-Config, переименован в FlyOS-Env, новая конфигурация

            ## v3.3.4
                * Оптимизация файловой системы
                * Добавление поворота экрана TFT
                * Добавление настройки пина выключения
                * Обновление всех системных компонентов
                * Поддержка изменения системных настроек через FLY TOOLS
