# Установка FLY-Tools

    1. Получите последнюю версию Klipper

    ```bash
    cd ~/klipper
    git pull
    ```

    * Если вы изменяли исходный код klipper локально и это привело к ошибке при получении, используйте следующую команду. (Эта операция удалит все ваши предыдущие изменения в файлах)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. Получите последнюю версию Katapult (ранее Canboot)

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Установка сервиса Fly-Tools

        * Процесс установки зависит от сервиса Git. Если загрузка застряла, нажмите `CTRL+C` и повторите попытку
        * Максимальное количество запросов в час для Github составляет 60, для Gitee неизвестно
        * Пользователям из Китая рекомендуется использовать Gitee для получения скрипта установки
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Получить скрипт установки через Github

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
