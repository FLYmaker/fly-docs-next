# Демонстрация функций

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * Поддержка изменения имени хоста
    * Поддержка изменения часового пояса системы
    * Поддержка активации 40PIN расширения верхнего уровня (обратите внимание, что после активации PWM невозможно использовать последовательное соединение с верхним уровнем, а после активации экрана возможно невозможно использовать порт ускорения верхнего уровня)
    * Поддержка изменения модели материнской платы
    * Поддержка изменения веб-страницы Klipper, поддержка Fuidd и Mainsail
    * Поддержка включения или выключения KlipperScreen
    * Поддержка принудительного запуска процесса Klippy на указанном ядре процессора, можно указать, на каком ядре будет работать Klipper
