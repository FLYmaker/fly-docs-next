# Загрузка ресурсов

:::tip

Пожалуйста, выберите соответствующий тип ресурсов для загрузки в левой навигационной панели.

:::

:::warning Обратите внимание

При загрузке файлов обращайте внимание на имя файла и подсказки на странице, чтобы избежать загрузки неправильных файлов.

:::

:::danger Обратите внимание

Некоторые файлы могут храниться на Github или облачном хранилище, убедитесь, что у вас есть стабильное интернет-соединение для загрузки.

:::
