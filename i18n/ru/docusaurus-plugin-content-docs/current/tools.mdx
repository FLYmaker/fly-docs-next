# Полезные инструменты

:::tip

Этот документ содержит различные полезные инструменты, связанные с адаптацией Klipper.

:::

:::warning Внимание

Некоторые инструменты, такие как `Парсер прошивки Klipper`, доступны онлайн, в то время как `Fly-Tools` требуют установки перед использованием. Пожалуйста, обратите внимание на различия.

:::

:::danger Важно

Если во время использования возникают проблемы, пожалуйста, своевременно отправляйте Issue.

:::
